Forms\n\n```csharp\nXfxControls.Init();\nglobal::Xamarin.Forms.Forms.Init(this, bundle);\n```\n\n*note: XfxEntry and XfxComboBox REQUIRES your app to use an AppCompat theme.*\n\n### iOS\n\nIn your AppDelegate, initialize XfxControls just before initializing Xamarin Forms\n\n```csharp\nXfxControls.Init();\nglobal::Xamarin.Forms.Forms.Init();\n```\n\n### Demos\n\n![](https://github.com/XamFormsExtended/Xfx.Controls/raw/master/resources/xfx.controls.ios.gif)\n![](https://github.com/XamFormsExtended/Xfx.Controls/raw/master/resources/xfx.controls.droid.gif)\n\n### Code\n\nDeclaration is exactly the same as a Xamarin.Forms.Entry, with some added properties\n\n#### XfxEntry\n\n```xml\n<!-- XfxEntry-->\n<xfx:XfxEntry Placeholder="Enter your name"\n              Text="{Binding Name}"\n              ErrorText="{Binding NameErrorText}" />\n```\n\nWhen the `ErrorText` property is set, the ErrorText will display, otherwise if it is null or empty, it\'s removed.\n\n#### XfxComboBox\n\n```xml\n<!-- XfxComboBox-->\n<xfx:XfxComboBox Placeholder="Enter your email address"\n                 Text="{Binding EmailAddress}"\n                 ItemsSource="{Binding EmailSuggestions}"\n                 SortingAlgorithm="{Binding SortingAlgorithm}"/>\n```\nThe XfxComboBox extends the XfxEntry and therefore also includes the `ErrorText` property.  \nBeyond that there is an `ItemsSource` property, `SelectedItem` property, and a `SortingAlgorithm` property.  \nThe first two are pretty self explanitory, but here\'s an example of how you can set the `SortingAlgorithm`\n\n```csharp\npublic class MyViewModel : INotifyPropertyChanged\n{\n\tpublic Func<string, ICollection<string>, ICollection<string>> SortingAlgorithm { get; } = (text, values) => values\n\t\t.Where(x => x.StartsWith(text, StringComparison.CurrentCultureIgnoreCase)))\n\t\t.OrderBy(x => x)\n\t\t.ToList();\n}\n```\n\n#### XfxCardView\n\n```xml\n<!-- XfxCardView -->\n<xfx:XfxCardView CornerRadius="3" Elevation="3">\n    <StackLayout Padding="0" Margin="0">\n        <Image Source="http://lorempixel.com/1024/200/abstract" HorizontalOptions="Fill" Aspect="Fill" />\n        <Label Margin="8">\n            <Label.Text>\n                Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse fringilla turpis turpis, id lobortis dolor vestibulum condimentum.\n            </Label.Text>\n        </Label>\n        <Button Text="Go to Main Page" Clicked="Button_OnClicked" Margin="8" />\n    </StackLayout>\n</xfx:XfxCardView>\n\n```\n\n## Contributions / Thanks\n\n - [@MarcBruins](https://github.com/MarcBruins)  for: [MBAutoComplete](https://github.com/MarcBruins/MBAutoComplete)\n - [@gshackles](https://github.com/gshackles) for: [FloatLabeledEntry](https://github.com/gshackles/FloatLabeledEntry)\n\n## License\n\nLicensed MIT, please review the [license](https://github.com/XamFormsExtended/Xfx.Controls/blob/master/LICENSE.md) file.\n'
