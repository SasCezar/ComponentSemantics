When this entry completed, it finds next entry in Parent layout and focus it. AdvancedEntry contains validation system inside it. You can set some properties to validate it and you can handle all your entries is validated or not with **FormView**.\nYou can set validation message and AnnotatinColor. Entry will automaticly display your message when it\'s not validated.\n</p>\n<h4>SAMPLE:</h4>\n\n```xaml\n <StackLayout Padding="30">\n            <Label Text="You can see FormView Below:"/>\n            <BoxView HeightRequest="1" Color="LightGray" />\n            <input:FormView IsValidated="{Binding IsValidated}">\n                \n                <input:AdvancedEntry \n                    Text="{Binding Email}"\n                    IsRequired="True"\n                    Title="Place your email below:"\n                    Annotation="Email"\n                    Placeholder="sample@domain.com"\n                    AnnotationColor="Accent"\n                    ValidationMessage="Please type a valid email address"\n                    IconImage="ic_email_black_24dp.png"\n                    MinLength="10"\n                    MaxLength="50"\n                    />\n\n                <input:AdvancedEntry \n                    Text="{Binding NameSurname}"\n                    IsRequired="True"\n                    Title="Place your name below:"\n                    Annotation="NameSurname"\n                    Placeholder="John Doe"\n                    AnnotationColor="Accent"\n                    ValidationMessage="Please type your name correctly"\n                    IconImage="ic_account_circle_black_24dp.png"\n                    MinLength="5"\n                    MaxLength="30"\n                    />\n\n                <input:AdvancedEntry \n                    Text="{Binding Phone}"\n                    IsRequired="True"\n                    Title="Place your phone number below:"\n                    Annotation="Phone"\n                    Placeholder="5439998877"\n                    AnnotationColor="Accent"\n                    ValidationMessage="Please type your phone number correctly"\n This view didn\'t created to static usage. You should Bind a model List as ItemSource, or if you don\'t use MVVM you can set in page\'s cs file like below. (You can override ToString method to fix display value or I\'ll add displayMember property soon.)</p>\n<h4>SAMPLE:</h4>\n\n```xaml\n<?xml version="1.0" encoding="utf-8" ?>\n<ContentPage xmlns="http://xamarin.com/schemas/2014/forms"\n             xmlns:x="http://schemas.microsoft.com/winfx/2009/xaml"\n             xmlns:local="clr-namespace:Sample.InputKit"\n             xmlns:input="clr-namespace:Plugin.InputKit.Shared.Controls;assembly=Plugin.InputKit"\n             x:Class="Sample.InputKit.MainPage">\n\n    <StackLayout Spacing="12" Padding="10,0" VerticalOptions="CenterAndExpand">\n\n        <input:SelectionView x:Name="selectionView" />\n\n    </StackLayout>\n</ContentPage>\n```\n\n```csharp\npublic partial class MainPage : ContentPage\n\t{\n\t\tpublic MainPage()\n\t\t{\n\t\t\tInitializeComponent();\n            selectionView.ItemSource = new[]\n            {\n                "Option 1","Option 2","Option 3","Option 4","Option 5","Option 6","Option 7","Option 8"\n            };\n\t\t}\n\t}\n```\n\n<a href="https://media.giphy.com/media/KXtC6oNnOgnJhvYecy/giphy.gif"><img src="https://media.giphy.com/media/KXtC6oNnOgnJhvYecy/giphy.gif" width="270" height="480" alt="Xamarin Forms SelectionView Enis Necipoglu" class="aligncenter size-medium" /></a>\n\n\nYou may use a object list as ItemSource, You can make this. Don\'t forget override **ToString()** method in your object.\n\nsample object:\n\n```csharp\n  public class SampleClass\n    {\n        public int Id { get; set; }\n        public string Name { get; set; }\n        public override string ToString() => Name;\n    }\n```\n\nUsage:\n\n```csharp\npublic partial class MainPage : ContentPage\n\t{\n\t\tpublic MainPage()\n\t\t{\n\t\t\tInitializeComponent();\n           selectionView.ItemSource = new[]\n            {\n                new SampleClass{ Name = "Option 1", Id = 1 },\n                new SampleClass{ Name = "Option 2", Id = 2 },\n                new SampleClass{ Name = "Option 3", Id = 3 },\n                new SampleClass{ Name = "Option 4", Id = 4 },\n                new SampleClass{ Name = "Option 5", Id = 5 },\n                new SampleClass{ Name = "Option 6", Id = 6 },\n                new SampleClass{ Name = "Option 7", Id = 7 },\n                new SampleClass{ Name = "Option 8", Id = 8 },\n            };\n\t\t}\n\t}\n```\n\n\n\n<h4>PROPERTIES:</h4>\n<ul>\n<li><strong>ItemSource:</strong> <em>(IList)</em> List of options</li>\n<li><strong>SelectedItem:</strong> <em>(object)</em> Selected Item from ItemSource</li>\n<li><strong>ColumnNumber:</strong> <em>(int)</em> Number of columng of this view</li>\n</ul>\n\n\n<br />\n<hr />\n\n# Did you like ?\n\n<a href="https://www.buymeacoffee.com/enisn" target="_blank"><img src="https://www.buymeacoffee.com/assets/img/custom_images/purple_img.png" alt="Buy Me A Coffee" style="height: auto !important;width: auto !important;" ></a>\n\nYour coffee keeps me awake while developing projects like this. \xf0\x9f\x91\x8d\xe2\x98\x95\n\n<hr />\n\n'
