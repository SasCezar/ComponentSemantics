b'## Firebase Push Notification Plugin for Xamarin iOS and Android\nSimple cross platform plugin for handling firebase push notifications.\n\n### Setup\n* Available on NuGet: http://www.nuget.org/packages/Plugin.FirebasePushNotification [![NuGet](https://img.shields.io/nuget/v/Plugin.FirebasePushNotification.svg?label=NuGet)](https://www.nuget.org/packages/Plugin.FirebasePushNotification/)\n* Install into your PCL project and Client projects.\n\n**Platform Support**\n\n|Platform|Version|\n| ------------------- | :------------------: |\n|Xamarin.iOS|iOS 8+|\n|Xamarin.Android|API 15+|\n\n### API Usage\n\nCall **CrossFirebasePushNotification.Current** from any project or PCL to gain access to APIs.\n\n## Features\n\n- Receive firebase push notifications\n- Subscribing/Unsubcribing to topics\n- Support for push notification category actions\n- Customize push notifications\n- Localization\n\n<p align="center">\n<img
