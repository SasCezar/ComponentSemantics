b'## [PureMVC](http://puremvc.github.com/) C# MultiCore Framework\n\nPureMVC is a lightweight framework for creating applications based upon the classic [Model-View-Controller](http://en.wikipedia.org/wiki/Model-view-controller) design meta-pattern. It supports [modular programming](http://en.wikipedia.org/wiki/Modular_programming) through the use of [Multiton](http://en.wikipedia.org/wiki/Multiton) Core actors instead of the [Singletons](http://en.wikipedia.org/wiki/Singleton_pattern) used in the [Standard](https://github.com/PureMVC/puremvc-csharp-standard-framework/wiki) Version.\n\n* [API Docs](http://puremvc.org/pages/docs/CSharp/multicore/)\n* [Unit Tests](http://puremvc.github.io/pages/images/screenshots/PureMVC-Shot-CSharp-Multicore-UnitTests.png)\n* [Discussion](http://forums.puremvc.org/index.php?board=20.0)\n* [MultiCore Version Overview Presentation](http://puremvc.tv/#P002)\n* [Legacy Implementation](https://github.com/PureMVC/puremvc-csharp-multicore-framework/tree/1.1.0)\n\n## Nuget Installation\n`dotnet add package PureMVC.Multicore`\n\n## Demos\n* [Employee Admin Microservice](https://github.com/PureMVC/puremvc-csharp-demo-microservice-employeeadmin/wiki)\n\n## Utilities\n* [Pipes](https://github.com/PureMVC/puremvc-csharp-util-pipes/wiki)\n\n## Platforms / Technologies\n* [C#](https://en.wikipedia.org/wiki/C_Sharp_(programming_language))\n* [Windows 10](https://en.wikipedia.org/wiki/Windows_10)\n* [Xamarin](https://en.wikipedia.org/wiki/Xamarin)\n* [Xbox](https://en.wikipedia.org/wiki/Xbox)\n* [Kinect](https://en.wikipedia.org/wiki/Kinect)\n* [HoloLens](https://en.wikipedia.org/wiki/Microsoft_HoloLens)\n\n## Status\nProduction -
