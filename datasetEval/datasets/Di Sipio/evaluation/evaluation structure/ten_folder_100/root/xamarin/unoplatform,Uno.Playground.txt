b"# Uno Playground\n\nThis is the code for published apps on stores (_Uno Gallery_) - Windows, Android & iOS\nand for the _Uno Playground_ <http://playground.platform.uno>.\n\n## Building it\n\nYou'll need Visual Studio 2017 v15.5+ to compile it. You also need those components:\n\n- Web Site development (aspnet core)\n- .NET Core cross-platform projects\n- Xamarin SDK (if you want to test iOS & Android)\n\n## UWP\n\nTo run the UWP project, simply select the project `Uno.UI.Demo.UWP` as starting\nproject.\n\n## Android & iOS\n\nTo run Xamarin versions, select the project `Uno.UI.Demo.Droid` or `Uno.UI.Demo.iOS`\nas starting project. For iOS, don't forget to pick emulator or device. You may need\nadditionnal provisionning configuration to run it on a device.\n\n## Wasm\n\nRun the project `Uno.UI.Demo.AspnetShell` - that's the `platform.uno` web site\nand go to the url `/Playground/index.html` once it's started (don't use the link\nin the site, it will bring you to the public version)."
