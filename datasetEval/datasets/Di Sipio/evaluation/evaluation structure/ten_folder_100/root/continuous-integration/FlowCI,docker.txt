b"# Install from Docker within 5 mins\n\n## Per-requirements\n\n- [Docker](https://docs.docker.com/install/) installed\n\n- [Docker-Compose](https://docs.docker.com/compose/install/) installed\n\n- Clone this 'docker' repo (make sure the scripts and docker compose file are available)\n\n    ```bash\n    git clone https://github.com/FlowCI/docker.git\n    ```\n\n## Start Service\n\n### 1. Start from command\n\n```bash\n## cd to docker dir which has been cloned\n##\n## example: cd ${HOME}/docker\n\n./start-server.sh {host} {email} {password}\n```\n\n- `{host}`: The host domain or ip address, find it by `ifconfig`.\n    > Hint: It doesn't work for `127.0.0.1` or `localhost`\n- `{email}` (Optional): Default admin email.\n    > `admin@flow.ci` will be default value if this argument not defined.\n- `{password}` (Optional): Default admin password. \n    > `123456` will be default value if this argument not dfined\n    \nLogin with admin email and password on `http://{host}:2015`\n\n![](https://github.com/FlowCI/docs/raw/master/v1.0/img/start_server.gif)\n\n\n Default Settings\n\n- Port `8080`: core service\n- Port `2015`: web\n- Port `27017`: database\n- Port `2181`: zookeeper\n- Port `5672` & `15672`: rabbitmq\n- where to store the data: `${HOME}/.flow.ci`\n\n> The default ports are exposed to host and data path can be changed from [server.yml](./server.yml) and [start-server.sh](./start-server.sh)\n\n### 2. Example\n\n```bash\n## Start with host, define admin email and password\n./start-server.sh 172.20.10.4 admin@flow.ci 1qaz@WSX\n```\n\n## Start Agent\n\n![](https://github.com/FlowCI/docs/raw/master/v1.0/img/start_agent.gif)\n\n### 1. Create Agent from admin page\n\n- Open web page: `http://{host}:2015/#/settings/agents` and click add\n  ![](https://github.com/FlowCI/docs/raw/master/v1.0/img/agent_add_click.png)\n- Fill in agent name.\n- Fill in agent tag and click '+' button to add.\n- Click 'save' button\n  ![](https://github.com/FlowCI/docs/raw/master/v1.0/img/agent_save_new.png)\n\n### 2. Start from command\n\n- Click 'copy' button to copy the token\n  ![](https://github.com/FlowCI/docs/raw/master/v1.0/img/agent_copy_token.png)\n\n- Start from command: `start-agent.sh {host} {token}`\n  - `{host}`: the host or ip address of server\n  - `{token}`: the agent token copied from admin page\n  - example:\n\n  ```bash\n  ./start-agent.sh 172.20.10.4 c2a957b7-5d09-4aa8-8d4f-90a0c2ee1392\n  ```\n\nDefault Settings\n\n- Port: the default port for ci server is `8080`\n- Where to store the data: default path is `${HOME}/.flow.ci.agent`\n\n> those settings could be changed from [start-agent.sh](./start-agent.sh)\n\n### 3. Per-install envrionments\n\n- git: `2.17.1`\n- java: openjdk `1.8.0_222`\n- mvn: `3.5.4`\n- nvm: `0.34.0`\n- node: `v10.16.3`\n- go: `1.12.9`"
