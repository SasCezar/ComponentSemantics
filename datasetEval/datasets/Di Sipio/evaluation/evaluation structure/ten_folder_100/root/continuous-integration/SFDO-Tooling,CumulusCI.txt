b'===============================\nCumulusCI\n===============================\n\n.. image:: https://img.shields.io/pypi/v/cumulusci\n           :target: https://pypi.org/project/cumulusci/\n           :alt: PyPI\n.. image:: https://img.shields.io/pypi/pyversions/cumulusci\n           :alt: PyPI - Python Version\n.. image:: https://img.shields.io/pypi/l/cumulusci\n           :alt: PyPI - License\n.. image:: https://readthedocs.org/projects/cumulusci/badge/?version=latest\n           :target: https://cumulusci.readthedocs.io/en/latest/?badge=latest\n           :alt: Documentation Status\n\nCumulusCI is a command line tool belt and set of reusable Python classes useful in the development and release process of building a Salesforce Managed Package application.\n\n\nKey Features\n------------\n\n* Out of the box, CumulusCI provides a complete best practice development and release process based on the processes used by Salesforce.org to build and release managed packages to thousands of users\n* Flexible and pluggable system for running tasks (single actions) and flows (sequences of tasks)\n* OAuth based org keychain allowing easy connection to Salesforce orgs and stored in local files using AES encryption\n\nFor more information, read the `full documentation`_.\n\n.. _`full documentation`: https://cumulusci.readthedocs.io/en/latest/\n\nIf you just want a quick intro, watch this screencast demo of using CumulusCI to configure a Salesforce project from a GitHub repository:\nhttps://asciinema.org/a/91555\n\nCumulusCI 1.0 (Ant based) Users, **PLEASE READ**\n================================================\n\nThe master branch now contains CumulusCI 2 which is not backwards compatible with the previous CumulusCI that was based on Ant.
