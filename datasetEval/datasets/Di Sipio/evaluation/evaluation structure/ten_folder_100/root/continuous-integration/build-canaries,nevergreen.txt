b"# Nevergreen\n\nNevergreen is a build monitor with attitude. Continually deliver via fast clear feedback.\n\n- Your builds should always be green. Nevergreen understands this and only shows you projects that have failed or are building\n- Nevergreen uses your local browser to store configuration. You only need to run it once to host different build monitors for every team\n- Nevergreen is fully responsive, it'll look just as good full screen or sharing the space with other dashboards\n- You can track as many CI servers as you like, allowing you to easily track dependent builds from other teams\n\n![Example Monitor page](doc/screenshot_monitor.png)\n\n![Example Tracking page](doc/screenshot_tracking.png)\n\n## How do I start monitoring my builds?\n\nYou can use [hosted Nevergreen](https://nevergreen.io) at `nevergreen.io` or\n[run Nevergreen locally](https://github.com/build-canaries/nevergreen/wiki/running-locally).\n\nOnce loaded:\n\n1. Add the URL of your CCTray XML feed on the Tracking page\n2. Select which projects you'd like to track\n3. Head to the Monitor page to see their current status!\n\n### How can I find my CCTray XML feed URL?\n\nMore information about how to get the CCTray XML feed for your CI server can be found on the \n[Nevergreen wiki finding your CCTray XML](https://github.com/build-canaries/nevergreen/wiki/find-cctray) page.\n\n### How can I get more help?\n\n- Nevergreen has contextual help, just look for the question mark icon\n- [The Nevergreen wiki](https://github.com/build-canaries/nevergreen/wiki)\n- [Tweet to @BuildCanaries](https://twitter.com/BuildCanaries) on Twitter\n- [Come chat to us on Slack](https://join.slack.com/t/nevergreen/shared_invite/enQtMzc4MDg5MTE2MDE2LWJjYjdiNDM4Y2U1YjBjZTdkZjFhZGQyOTM5ODQ2MzEwZDY4YjNiZDFmOTFlMGUwOWEyMjczNmUzNTlkZDk4ZmQ)\n\n## How can I fix a bug / add a feature / contribute?\n\nYou can run Nevergreen from source using the `./develop.sh` script. For more detailed information about running from\nsource please see the [contributing section of the wiki](https://github.com/build-canaries/nevergreen/wiki/contributing).\n\nPlease help us out by submitting a PR with any changes. We also keep a list of bugs / improvements / features using\n[GitHub issues](https://github.com/build-canaries/nevergreen/issues), if you're looking for some inspiration.\n\nYou can also help us out by taking this [short usage survey](https://build-canaries.github.io/2015/09/14/nevergreen-survey.html), thanks!\n\n## License\n\nCopyright \xc2\xa9 2014 - 2019 Build Canaries\n\nDistributed under the Eclipse Public License either version 1.0 or (at your option) any later version.\n"
