So flex.css\'s main purpose is to ensure\nthat every attribute can be supported by standard version\'s flex or old-version\'s -webkit-box.\ndue to autoprefixer compilation is used by flex.css,it will roll back to old-version\'s -webkit-box when standard\nflex is not supported by some browsers so the effect of layout will be the same.\nThen,here comes a magic effient layout tool of mobile terminal development ...\n```\n\n\n###  merits\n```\nConcise api, \nfamiliar attribute values, makes it easy for you to get started in using flex.css.\nIn html, the layout is bind with attributes, so it is seperated from css. In this way, it will be easier for you\nto maintain and modify your layout without modifying css.\n```\n\n\n### support\n```\nflex layout is split into three versions: old version: display: box; , transitional version: display:flexbox; ,\nand present standard version: display:flex; .\nAndroid\n2.3  began to support old version: display: -webkit-box;\n4.4 began to support standard version: display: flex;\nIOS\n6.1  began to support old version: display: -webkit-box;\n7.1 began to support standard version: display: flex;\nPC\nYou can use flex.css if you don\'t need to consider IE10-.\nflex.css is compatible with standard version and old version at the same time, so when a browser doesn\'t support standard version, it will roll back to old version.\n\n```\n![Alt text](https://github.com/lzxb/flex.css/raw/master/shot/caniuse.png)\n\n### use\n```html\n<!--\nAccording to what you need, include css files in the dist directory into your html\nflex.css should be matched by flex attribute\ndata-flex.css should be matched by data-flex attribute(used by React)\nIf you use webpack to package, after npm is installed, and ES6 compiler is deployed in your project,\nflex attribute matching can be implemented in this way:\nimport \'flex.css\';\ndata-flex attribute matching can be implemented in this way( used by React):\nimport \'flex.css/dist/data-flex.css\';\n -->\n<!-- flex attribute matching\xef\xbc\x8ca simple example of centering child element \xef\xbc\x9a -->\n  <div flex="main:center cross:center" style="width:500px;
