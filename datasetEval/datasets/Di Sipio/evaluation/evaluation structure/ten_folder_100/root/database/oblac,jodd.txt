You don\'t have to install anything,\nthe only prerequisites are [Git](http://help.github.com/set-up-git-redirect)\nand Java JDK.\n\n### Check out sources\n\nSimply clone **Jodd** Git repo:\n\n    git clone https://github.com/oblac/jodd.git jodd\n\n### Compile and test, build jars\n\nYou can build the Jodd project with:\n\n    gradlew build\n\nThis will build all jars and run all unit tests.\nTo skip the tests (for faster build), execute:\n\n    gradlew build -x test\n\n### Build full release with reports\n\nTo generate _full release_, including running integration tests and generating various reports,\nyou need [Docker](https://www.docker.com/) v1.12+.\n\n\tdocker-compose -f etc/docker-compose.yml up\n    gradlew clean release\n\nIntegration tests requires some infrastructure (like databases), hence Docker is\nused.\n\n### Install Jodd into your local Maven\n\n    gradlew install\n\n## :gift_heart: Contribute\n\nFeel free to [contribute](CONTRIBUTING.md)!
