https://apt.postgresql.org/pub/repos/apt/ACCC4CF8.asc | apt-key add -\n\nsudo apt install postgresql-X.Y-omnidb\n```\n\nFor more details, please check the OmniDB PostgreSQL debugger plugin\ndocumentation [here](https://omnidb.readthedocs.io/en/latest/en/23_debugger_plugin_installation.html).\n\n**IMPORTANT**: Currently OmniDB server is also available from PGDG repository,\nbut only working on Debian 10. But if you are developing for OmniDB, then you can install it\nfrom sources, with system-wide `pip` or user-wide `PyEnv`:\n\n\n## 3.1- On Debian >= 9 with `pip`\n\n```\nsudo apt install python3-pip\npip3 install pip --upgrade\npip3 install -r requirements.txt\n```\n\n## 3.2- On Debian/Ubuntu using `PyEnv`\n\n```\nsudo apt install git make build-essential libssl-dev zlib1g-dev libbz2-dev libreadline-dev libsqlite3-dev wget curl llvm libncurses5-dev libncursesw5-dev xz-utils\n\ngit clone https://github.com/pyenv/pyenv.git ~/.pyenv\necho \'export PYENV_ROOT="$HOME/.pyenv"\' >> ~/.bashrc\necho \'export PATH="$PYENV_ROOT/bin:$PATH"\' >> ~/.bashrc\necho \'eval "$(pyenv init -)"\' >> ~/.bashrc\nsource ~/.bashrc\n\npyenv install 3.6.5\ncd OMNIDB_FOLDER\npyenv local 3.6.5\n\npip install pip --upgrade\npip install -r requirements.txt\n```\n\n## 3.3- Cloning OmniDB repo\n\n```\ngit clone https://github.com/OmniDB/OmniDB\n```\n\n## 3.4- Running OmniDB from sources\n\nTo start OmniDB server, enter into `OmniDB/OmniDB` folder and type:\n\n```\npython omnidb-server.py -c omnidb.conf\n```\n'
