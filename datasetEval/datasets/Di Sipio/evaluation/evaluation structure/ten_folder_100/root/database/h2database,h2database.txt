b"# Welcome to H2, the Java SQL database. [![Build Status](https://travis-ci.org/h2database/h2database.svg?branch=master)](https://travis-ci.org/h2database/h2database)\n\n## The main features of H2 are:\n\n* Very fast, open source, JDBC API\n* Embedded and server modes; disk-based or in-memory databases\n* Transaction support, multi-version concurrency\n* Browser based Console application\n* Encrypted databases\n* Fulltext search\n* Pure Java with small footprint: around 2 MB jar file size\n* ODBC driver\n\nMore information: https://h2database.com\n\n## Downloads\n\n[Download latest version](https://h2database.com/html/download.html) or add to `pom.xml`:\n\n```XML\n<dependency>\n    <groupId>com.h2database</groupId>\n    <artifactId>h2</artifactId>\n    <version>1.4.200</version>\n</dependency>\n```\n\n## Documentation\n\n* [Tutorial](https://h2database.com/html/tutorial.html)\n* [SQL commands](https://h2database.com/html/commands.html)\n* [Functions](https://h2database.com/html/functions.html), [aggregate functions](https://h2database.com/html/functions-aggregate.html), [window functions](https://h2database.com/html/functions-window.html)\n* [Data types](https://h2database.com/html/datatypes.html)\n\n## Support\n\n* [Issue tracker](https://github.com/h2database/h2database/issues) for bug reports and feature requests\n* [Mailing list / forum](https://groups.google.com/forum/#!forum/h2-database) for questions about H2\n* ['h2' tag on Stack Overflow](https://stackoverflow.com/questions/tagged/h2) for other questions (Hibernate with H2 etc.)\n"
