b'Sequel Pro <img alt="Logo" src="https://sequelpro.com/images/logo.png" align="right" height="50">\n=======\n\nSequel Pro is a fast, easy-to-use Mac database management application for working with MySQL & MariaDB databases.\n\nYou can find more details on our website: [sequelpro.com](https://sequelpro.com)\n\n![Screenshot](https://sequelpro.com/images/browse.png)\n\nBuild Instructions\n=======\n\n * Install the [latest version of Xcode](https://itunes.apple.com/au/app/xcode/id497799835)\n * Install [GitHub for Mac](https://desktop.github.com/) (or [SourceTree](https://www.sourcetreeapp.com/), or [\xe2\x80\xa6](https://git-scm.com/downloads/guis))\n * Click "Clone in Desktop" on the right sidebar of our [GitHub page](https://github.com/sequelpro/sequelpro)\n * Open `sequel-pro.xcodeproj`\n * Click the `Run` button in the toolbar\n * If the above doesn\'t work, please file a [bug report](https://github.com/sequelpro/sequelpro/issues/new)\n\nContributing\n=======\n\nThe best way to help the project is to use our [test builds](https://sequelpro.com/test-builds) and report any issues (both bugs and missing features) in [the issue tracker](https://github.com/sequelpro/sequelpro/issues).
