b"# \xef\xa3\xbf m-cli\n\n:boom: **Swiss Army Knife for macOS !**\n\n## The difference\n\n**m-cli** differs from other mac command line tools in that:\n\n* Its main purpose is to manage administrative tasks and do it easier\n* It doesn't install 3rd party tools because it doesn't have dependencies\n* The installation is very easy and doesn't require intervention\n* It only uses macOS commands\n\n\n## Installation\n\n* **Using Homebrew:**\n\n  `brew install m-cli`\n\n* **Manual installation:**\n\n  `curl -fsSL https://raw.githubusercontent.com/rgcr/m-cli/master/install.sh | sudo sh`\n\n  You can also install it in a different path\n\n  `INSTALL_DIR=$HOME/.m-cli sh <(curl -fsSL https://raw.githubusercontent.com/rgcr/m-cli/master/install.sh)`\n\n  _Note: You need privileges_\n\n\n\n## Usage\n\n```\n\xef\xa3\xbf  Swiss Army Knife for macOS ! \xef\xa3\xbf\n\n\nusage:  m [OPTIONS] COMMAND [help]\n\n    OPTIONS\n        --update        update m-cli to the latest version\n        --uninstall     uninstall m-cli\n\n    COMMANDS:\n        help\n        battery\n        bluetooth\n        dir\n        disk\n        display\n        dns\n        dock\n        finder\n        firewall\n        flightmode\n        gatekeeper\n        hostname\n        info\n        itunes\n        localhost\n        lock\n        ntp\n        printer\n        network\n        nosleep\n        notification\n        restart\n        safeboot\n        screensaver\n        service\n        shutdown\n        sleep\n        timezone\n        touchbar\n        trash\n        update\n        user\n        volume\n        vpn\n        wallpaper\n        wifi\n```\n\n####  Battery:\n```\n    usage: m battery [ status | help ]\n\n    Examples:\n      m battery status    # get the battery status\n```\n\n#### Bluetooth:\n```\n    usage: m bluetooth [ status | on | enable | off | disable | help ]\n\n    Examples:\n      m bluetooth status    # bluetooth status\n      m bluetooth on        # turn on bluetooth\n      m bluetooth enable    # turn on bluetooth\n      m bluetooth off       # turn off bluetooth\n      m bluetooth disable   # turn off bluetooth\n```\n\n\n####  Disk:\n```\n    usage: m disk [ ls | list | info | fs | filesystems | ejectall | verify | repair | help ]\n\n    Examples:\n      m disk ls                                 # list disks\n      m disk list                               # list disks\n      m disk list /dev/disk0                    # list a specific disk\n\n      m disk fs                                 # list available filesystems for formatting\n      m disk filesystems                        # list available filesystems for formatting\n\n      m disk info /dev/disk0                    # display information\n\n      m disk ejectall                           # eject all mountable volumes\n\n      m disk verify volume /Volume/MyVol        # verify volume\n      m disk verify disk /dev/disk0             # verify disk\n\n      m disk repair volume /Volume/MyVol        # repair volume\n      m disk repair disk /dev/disk0             # repair disk\n\n      m disk format MS-DOS MYNAME /dev/disk2              # format the entire disk with a windows format (MS-DOS)\n      m disk format volume MS-DOS MYNAME /Volumes/myvol   # format the volume with a windows format (MS-DOS)\n\n      m disk reformat /Volumes/myvol            # reformat a volume\n      m disk rename CURRENTNAME NEWNAME         # rename a volume\n```\n\n#### Display:\n```\n    usage: m display [ status | help ]\n\n    Example:\n      m display status   # status of displays\n      m display help     # show usage\n```\n\n#### Dns:\n```\n    usage:  m dns [ flush | help ]\n\n    Examples:\n      m dns flush       # flushes local DNS\n```\n\n#### Dock:\n```\n    usage: m dock [ showdelay | autohide | magnification | position | addblankspace | addrecentitems | prune | help ]\n\n    Examples:\n      m dock showdelay x.x          # Changes how long the Dock takes to show up when auto-hide is enabled\n      m dock autohide YES           # Enable Dock's auto hide feature\n      m dock autohide NO            # Disable Dock's auto hide feature\n      m dock magnification YES      # Turn magnification on\n      m dock magnification NO       # Turn magnification off\n      m dock position BOTTOM        # Change Dock's position to the bottom of the screen\n      m dock position LEFT          # Change Dock's position to the left of the screen\n      m dock position RIGHT         # Change Dock's position to the right of the screen\n      m dock addblankspace          # Add a blank space (separator) to the Dock\n      m dock addrecentitems         # Add a stack containg your recent items to the Dock\n                                    #  (You can change the stack's type by right clicking on it)\n      m dock prune                  # Removes all apps from Dock\n```\n\n#### Dir:\n```\n    usage: m dir [ tree | size | delete | help ]\n\n    Examples:\n      m dir tree        # tree view of folders in the current path\n      m dir tree /path  # tree view of folders in a specific path\n\n      m dir delete empty          # delete empty folders recursively in the current path\n      m dir delete empty /path    # delete empty folders recursively in a specific path\n\n      m dir delete dsfiles        # delete .DS_Store files recursively in the current path\n      m dir delete dsfiles /path  # delete .DS_Store files recursively in a specific path\n\n      m dir size        # calculate current folder size\n      m dir size /path  # calculate folder size in a specific path\n```\n\n#### Finder:\n```\n    usage: m finder [ showhiddenfiles | showfileextensions | showdesktop | showpath | help  ]\n\n    Examples:\n      m finder showhiddenfiles           # get the current status\n      m finder showhiddenfiles YES       # show hidden files\n      m finder showhiddenfiles NO        # no show hidden files\n\n      m finder showextensions            # get the current status\n      m finder showextensions YES        # show all file extensions\n      m finder showextensions NO         # don't show all file extensions\n\n      m finder showdesktop               # get the current desktop status\n      m finder showdesktop YES           # enable the desktop\n      m finder showdesktop NO            # disable the desktop\n\n      m finder showpath YES              # show the current opened folder path on the top bar of the Finder window\n      m finder showpath NO               # show the current opened folder name on the top bar of the Finder window\n```\n\n#### Firewall:\n```\n    usage: m firewall [ status | enable | disable | list | add | remove | help ]\n\n    Examples:\n       m firewall status                # Show status\n       m firewall enable                # Enable firewall\n       m firewall disable               # Disable firewall\n       m firewall list                  # List applications handled by firewall\n       m firewall add /path/to/file     # Add app to firewall\n       m firewall remove /path/to/file  # Remove app from firewall\n```\n\n#### Flightmode:\n```\n    usage:  m flightmode [ on | off | help ]\n\n    Examples:\n      m flightmode on                      # turn flightmode on\n      m flightmode off                     # turn flightmode off\n```\n\n#### Gatekeeper:\n```\n    usage: m gatekeeper [ status | list | ls | enable | disable | create\xc2\xa0| help ]\n\n    Examples:\n      m gatekeeper status                               # gatekeeper status\n      m gatekeeper list                                 # list rules\n\n      m gatekeeper enable                               # enable gatekeeper\n      m gatekeeper disable                              # disable gatekeeper\n\n      m gatekeeper enable MYRULE                        # enable rule\n      m gatekeeper disable MYRULE                       # disable rule\n\n      m gatekeeper create  RULENAME /path/to/program    #  Create a rule for the application\n```\n\n#### Group:\n```\n    usage: m group [ list | ls | info | adduser | removeuser | ismember | help ]\n\n    Examples:\n      m group list                          # get list of groups\n      m group info mygroup                  # display group information\n\n      m group adduser myuser mygroup        # add an user to a specific group\n      m group removeuser myuser mygroup     # remove an user from a specific group\n\n      m group ismember myuser mygroup       # show if the user is a member of a specific group\n```\n\n#### Hostname:\n```\n    usage: m hostname [ help ]\n\n    Examples:\n      m hostname                # get the current hostname information (computername, hostname, localhostname and netbiosname)\n      m hostname newhostname    # set a new hostname (computername, hostname, localhostname, netbiosname)\n\n      m hostname help           # only shows this help\n```\n\n#### Info:\n```\n    usage: m info [ help ]\n\n    Examples:\n      m info        #  print macOS operating system version information\n```\n\n### Itunes:\n```\n   usage: m itunes [ status | play | pause | next | prev | mute | unmute | vol up | vol down | vol #| stop | quit | help ]\n\n    Examples:\n      m itunes status       # Show status\n      m itunes play         # Play track\n      m itunes pause        # Pause track\n      m itunes next         # Play next track\n      m itunes prev         # Play previous track\n      m itunes mute         # Mute iTunes\n      m itunes unmute       # Unmute iTunes\n      m itunes vol up       # Volume Up\n      m itunes vol down     # Volume Down\n      m itunes vol #        # Set volume level\n      m itunes stop         # Stop track\n      m itunes quit         # Quit iTunes\n```\n\n#### Localhost:\n```\n    usage: m localhost [ help ]\n\n    Examples:\n      m localhost ls                              # list current records in localhost\n      m localhost add 127.0.0.1 webpage.local     # add a new host to the localhost file\n      m localhost remove webpage.local            # remove a host from the localhost file\n```\n\n#### Lock:\n```\n    usage:  m lock [ help ]\n\n    Examples:\n      m lock      # lock session\n```\n\n#### Ntp:\n```\n    usage: m ntp [ status | enable | disable | set | help ]\n\n    Examples:\n      m ntp status                          # status of the network time service\n      m ntp enable                          # enable clock to use network time\n      m ntp disable                         # disable clock to use network time\n      m ntp set timehost1.net.sap.corp      # set network time server\n```\n\n#### Printer:\n```\n    usage: m printer [ settings | name | queue | drivers | web | help ]\n\n    Examples:\n      m printer settings    # Printer settings\n      m printer name        # Display printer names on system\n      m printer queue       # Display items in printer queue on system\n      m printer drivers     # Display all printer drivers\n      m printer web         # Enable and show web interface\n```\n\n#### Network:\n```\n    usage:  m network [ ls | list | location | help ]\n\n    Examples:\n      m network ls                          # list network interfaces\n      m network location                    # get current location\n      m network location ls                 # list locations\n      m network location create XYZ         # create a location\n      m network location delete XYZ         # delete a location\n      m network location switch XYZ         # switch location\n```\n\n#### Nosleep:\n```\n    usage: m nosleep [ until | help ]\n\n    Examples:\n      m nosleep until 3600            # no sleep until 3600 seconds\n      m nosleep until my_script.sh    # get the current status\n      m notification showcenter YES  # enable the notification center\n      m notification showcenter NO   # disable the notification center\n\n```\n\n#### Restart:\n```\n    usage:  m restart [ -f | --force | help ]\n\n    Examples:\n      m restart     # get the boot args\n      m safeboot enable     # enable safe boot\n      m safeboot disable    # disable safeboot\n```\n\n#### Screensaver:\n```\n    usage: m screensaver [ status | askforpassword | help ]\n\n    Examples:\n      m screensaver                         #  enable password requirement to unlock\n      m screensaver askforpassword NO       #  disable password requirement to unlock\n```\n\n#### Service:\n```\n    usage: m service [ --status-all | --list |  --ls | start | stop | load | unload | help ]\n\n\n    Examples:\n      m service --status-all                        # load a service\n      m service unload com.apple.sessionlogoutd     # unload a service\n```\n\n#### Shutdown:\n```\n    usage:  m  shutdown [-f | --force | help ]\n\n    Examples:\n      m shutdown     # list users\n      m user info demouser     # display user information\n\n      m user create            # create a user, it will ask you the below information\n                                    Username:\n                                    Full name:\n                                    Shell [/bin/bash]:\n                                    Password:\n\n      m user delete demouser   # delete user\n```\n\n#### Volume:\n```\n    usage:  m volume [ level(0-100) | change(-n|+n) | up | down | mute | unmute | ismute ]\n\n    Examples:\n      m volume 70     # set the volume to 70 %\n      m volume +5     # increase the volume by 5 (up to 100)\n      m volume -10    # decrease the volume by 5 (down to 0)\n      m volume up     # increase the volume by 6.25\n      m volume down   # decrease the volume by 6.25\n      m volume        #
