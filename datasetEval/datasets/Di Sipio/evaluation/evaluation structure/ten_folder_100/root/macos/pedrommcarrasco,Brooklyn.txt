b'![](https://github.com/pedrommcarrasco/Brooklyn/blob/master/Design/logo.jpg?raw=true)\n\n> Screen Saver by [Pedro Carrasco](https://twitter.com/pedrommcarrasco) based on the animations presented during **[Apple Special Event](https://www.youtube.com/watch?v=bfHEnw6Rm-4)** (October 30, 2018) from the Brooklyn Academy of Music, Howard Gilman Opera House.\n\n<p align="center">\n    <img src="https://github.com/pedrommcarrasco/Brooklyn/blob/master/Design/showcase.gif?raw=true" alt="Example"/>\n</p>\n\n![GitHub All Releases](https://img.shields.io/github/downloads/pedrommcarrasco/brooklyn/total.svg) [![Build Status](https://travis-ci.org/pedrommcarrasco/Brooklyn.svg?branch=master)](https://travis-ci.org/pedrommcarrasco/Brooklyn) ![GitHub contributors](https://img.shields.io/github/contributors/pedrommcarrasco/brooklyn.svg) [![apm](https://img.shields.io/apm/l/vim-mode.svg)](https://github.com/pedrommcarrasco/Brooklyn/blob/master/LICENSE)\n\n## Features \xe2\x9c\x85\n\n* Doesn\'t require internet connection\n* Homebrew support\n* Light & Dark theme\n* Select which animations you want to see\n* Loop each animation how many times you want\n* Make the animations\' order random\n\n<p align="center">\n    <img src="https://github.com/pedrommcarrasco/Brooklyn/blob/master/Design/preferenceMenu.png?raw=true" alt="Example"/>\n</p>\n\n## Installation \xf0\x9f\x93\xa6\n\nIndependently of how you install **Brooklyn**, please **close your System Preferences**\n\n### Manual :hand:\n\n1. [Click here to Download](https://github.com/pedrommcarrasco/Brooklyn/releases/download/2.0.1/Brooklyn.saver.zip)\n2. Open **Brooklyn.saver** (double click)\n3. `"Brooklyn.saver" can\'t be opened because it is from an unidentified developer` will appear, press `OK`\n4. Open `Preferences`\n5. Select `Security & Privacy`\n6. Select `General`\n7. On the bottom side, select `Open Anyway`\n\n### Homebrew \xf0\x9f\x8d\xba\n\n1. Open terminal\n2. Enter `brew cask install brooklyn`\n\n## Uninstallation \xf0\x9f\x97\x91\xef\xb8\x8f\n\n### Manual :hand:\n\n- Right-click on Brooklyn in System Preferences and select `Delete Brooklyn`, or\n- Delete `Brooklyn.saver` either in `/Library/Screen Savers` or `/Users/USERNAME/Library/Screen Savers`. \n\n### Homebrew \xf0\x9f\x8d\xba\n\n1. Open terminal\n2. Enter `brew cask uninstall brooklyn`\n\n## Compatibility \xf0\x9f\x94\xa7\n\nRequires OS X El Capitan (10.11) or above.\n\n## Support Brooklyn \xe2\x9d\xa4\xef\xb8\x8f\n\nHello there \xf0\x9f\x91\x8b\n\nI\xe2\x80\x99m Pedro, a Portuguese iOS Engineer since February 2017.
