`import` or commonjs `require`\nusing either lazy loading or a callback:\n\n```javascript\n// Make a new index, or open an existing one with this name\nimport si from \'search-index\'\n\n// "lazy load"- db may not be immediately initialized\ndb = si({ name: \'mySearchIndex\' })\n\n// ... or callback to be sure you have it in time\nsi({ name: \'myDB\' }, (err, db) => {\n  // db is guaranteed to be open and available\n})\n```\n\n#### Script tag method\n\nIn the `/dist` folder there is a file called\n`search-index.<version>.js` that can be used as a standalone in a\n`<script>` tag.
