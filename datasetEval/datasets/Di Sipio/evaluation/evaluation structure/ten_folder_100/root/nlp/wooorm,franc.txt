b'# ![franc][logo]\n\n[![Build Status][build-badge]][build-status]\n[![Coverage Status][coverage-badge]][coverage-status]\n\nDetect the language of text.\n\n## What\xe2\x80\x99s so cool about franc?\n\n1.  **franc** can support more languages<sup>(\xe2\x80\xa0)</sup> than any other\n    library\n2.  **franc** is packaged with support for [82][s], [187][m], or [401][l]\n    languages\n3.  **franc** has a CLI\n\n\xe2\x80\xa0 - Based on the [UDHR][], the most translated document in the world.\n\n## What\xe2\x80\x99s not so cool about franc?\n\n**franc** supports many languages, so make sure to pass it big documents,\nto get reliable results.\n\n## Installation\n\n[npm][]:\n\n```sh\nnpm install franc\n```\n\nThis installs the [`franc`][m] package, with support for 187 languages\n(languages which have 1 million or more speakers).  [`franc-min`][s] (82\nlanguages, 8m or more speakers) and [`franc-all`][l] (all 401 possible\nlanguages) are also available.  Finally, use `franc-cli` to install the\n[CLI][].\n\nBrowser builds for [`franc-min`][s], [`franc`][m], and [`franc-all`][l] are\navailable on [GitHub Releases][releases].\n\n## Usage\n\n```javascript\nvar franc = require(\'franc\')\n\nfranc(\'Alle menslike wesens word vry\') // => \'afr\'\nfranc(\'\xe0\xa6\x8f\xe0\xa6\x9f\xe0\xa6\xbf \xe0\xa6\x8f\xe0\xa6\x95\xe0\xa6\x9f\xe0\xa6\xbf \xe0\xa6\xad\xe0\xa6\xbe\xe0\xa6\xb7\xe0\xa6\xbe \xe0\xa6\x8f\xe0\xa6\x95\xe0\xa6\x95 IBM \xe0\xa6\xb8\xe0\xa7\x8d\xe0\xa6\x95\xe0\xa7\x8d\xe0\xa6\xb0\xe0\xa6\xbf\xe0\xa6\xaa\xe0\xa7\x8d\xe0\xa6\x9f\') // => \'ben\'\nfranc(\'Alle menneske er f\xc3\xb8dde til fridom\') // => \'nno\'\nfranc(\'\') // => \'und\'\nfranc(\'the\') // => \'und\'\n\n/* You can change what\xe2\x80\x99s too short (default: 10): */\nfranc(\'the\', {minLength: 3}) // => \'sco\'\n```\n\n###### `.all`\n\n```js\nconsole.log(franc.all(\'O Brasil caiu 26 posi\xc3\xa7\xc3\xb5es\'))\n```\n\nYields:\n\n```js\n[ [ \'por\', 1 ],\n  [ \'src\', 0.8797557538750587 ],\n  [ \'glg\', 0.8708313762329732 ],\n  [ \'snn\', 0.8633161108501644 ],\n  [ \'bos\', 0.8172851103804604 ],\n  ... 116 more items ]\n```\n\n###### `only`\n\n```js\nconsole.log(franc.all(\'O Brasil caiu 26 posi\xc3\xa7\xc3\xb5es\', {only: [\'por\', \'spa\']}))\n```\n\nYields:\n\n```js\n[ [ \'por\', 1 ], [ \'spa\', 0.799906059182715 ] ]\n```\n\n###### `ignore`\n\n```js\nconsole.log(franc.all(\'O Brasil caiu 26 posi\xc3\xa7\xc3\xb5es\', {ignore: [\'src\', \'glg\']}))\n```\n\nYields:\n\n```js\n[ [ \'por\', 1 ],\n  [ \'snn\', 0.8633161108501644 ],\n  [ \'bos\', 0.8172851103804604 ],\n  [ \'hrv\', 0.8107092531705026 ],\n  [ \'lav\', 0.810239549084077 ],\n  ... 114 more items ]\n```\n\n## CLI\n\nInstall:\n\n```bash\nnpm install franc-cli --global\n```\n\nUse:\n\n```text\nCLI to detect the language of text\n\nUsage: franc [options] <string>\n\nOptions:\n\n  -h, --help                    output usage information\n  -v, --version                 output version number\n  -m, --min-length <number>     minimum length to accept\n  -o, --only <string>           allow languages\n  -i, --ignore <string>         disallow languages\n  -a, --all                     display all guesses\n\nUsage:\n\n# output language\n$ franc "Alle menslike wesens word vry"\n# afr\n\n# output language from stdin (expects utf8)\n$ echo "\xe0\xa6\x8f\xe0\xa6\x9f\xe0\xa6\xbf \xe0\xa6\x8f\xe0\xa6\x95\xe0\xa6\x9f\xe0\xa6\xbf \xe0\xa6\xad\xe0\xa6\xbe\xe0\xa6\xb7\xe0\xa6\xbe \xe0\xa6\x8f\xe0\xa6\x95\xe0\xa6\x95 IBM \xe0\xa6\xb8\xe0\xa7\x8d\xe0\xa6\x95\xe0\xa7\x8d\xe0\xa6\xb0\xe0\xa6\xbf\xe0\xa6\xaa\xe0\xa7\x8d\xe0\xa6\x9f" | franc\n# ben\n\n# ignore certain languages\n$ franc --ignore por,glg "O Brasil caiu 26 posi\xc3\xa7\xc3\xb5es"\n# src\n\n# output language from stdin with only\n$ echo "Alle mennesker er f\xc3\xb8dt frie og" | franc --only nob,dan\n# nob\n```\n\n## Supported languages\n\n| Package | Languages | Speakers |\n| ------- | --------- | -------- |\n| [`franc-min`][s] | 82 | 8M or more |\n| [`franc`][m] | 187 | 1M or more |\n| [`franc-all`][l] | 401 | - |\n\n### Language code\n\nNote that franc returns [ISO 639-3][iso6393] codes (three letter codes).\n**Not** ISO 639-1 or ISO 639-2.  See also [GH-10][] and [GH-30][].\n\nTo get more info about the languages represented by ISO 639-3, you can use\n[langs][] or [iso-639-3][].\nTo map ISO 639-3 to ISO 639-1 codes, use [iso-639-3-to-1][],\nbut note that not all 639-3 codes can be represented in 639-1.\n\n## Ports\n\nFranc has been ported to several other programming languages.\n\n*   Elixir \xe2\x80\x94 [`paasaa`](https://github.com/minibikini/paasaa)\n*   Erlang \xe2\x80\x94 [`efranc`](https://github.com/G-Corp/efranc)\n*   Go \xe2\x80\x94 [`franco`](https://github.com/kapsteur/franco),\n    [`whatlanggo`](https://github.com/abadojack/whatlanggo)\n*   R \xe2\x80\x94 [`franc`](https://github.com/MangoTheCat/franc)\n*   Rust \xe2\x80\x94 [`whatlang-rs`](https://github.com/greyblake/whatlang-rs)\n\nThe works franc is derived from have themselves also been ported to other\nlanguages.\n\n## Derivation\n\nFranc is a derivative work from [guess-language][] (Python, LGPL),\n[guesslanguage][] (C++, LGPL), and [Language::Guess][language-guess]\n(Perl, GPL).
