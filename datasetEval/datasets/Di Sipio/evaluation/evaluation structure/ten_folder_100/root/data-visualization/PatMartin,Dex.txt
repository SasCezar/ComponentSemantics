b'<img src="https://raw.githubusercontent.com/PatMartin/Dex/master/images/github_docs/dexheader.png">\n\n# Dex\n\nDex : The data explorer is a powerful tool for data science.  It is written in Groovy and Java on top of JavaFX and offers the ability to:\n\n* Read in data from a variety of sources such files, programs and a variety of databases.\n* Transform the data in a powerful ways.\n* Apply powerful machine learning to the data via SMILE and R integration.\n* Visualize the data in over 50 distinct ways.\n* Output the data to a variety of databases and file formats.\n* Extend Dex from within via templates and internal scripting.  Save the extensions to reuse later.\n\n## Other Resources\n\n* [Main Site](http://dexvis.net)\n* [Blog](http://dexvis.wordpress.com)\n\n# Requirements\n\n - Java 8\n\n# Installation\n\nThere are 2 main ways to install Dex.\n\n1. Install via Git by following the instructions below.\n2. [Download a stable release](http://dexvis.net/doku.php?id=Download)\n\nThe packaged release tends to lag significantly behind the Git version.\n\n## Via Git\n\n### Download:\n\n```bash\ngit clone https://github.com/PatMartin/Dex.git\n```\n\n### Run\n\n```bash\ncd Dex\njava -jar Dex.jar\n```\n\n# Example Visualizations\n\nThe following are a small sample of the 50+ data visualizations Dex is
