b"Orange\n======\n\n[![Discord Chat](https://img.shields.io/discord/633376992607076354?style=plastic)](https://discord.gg/FWrfeXV)\n[![build: passing](https://img.shields.io/travis/biolab/orange3?style=plastic)](https://travis-ci.org/biolab/orange3)\n[![codecov](https://img.shields.io/codecov/c/github/biolab/orange3?style=plastic)](https://codecov.io/gh/biolab/orange3)\n\n[Orange] is a component-based data mining software. It includes a range of data\nvisualization, exploration, preprocessing and modeling techniques. It can be\nused through a nice and intuitive user interface or, for more advanced users,\nas a module for the Python programming language.\n\nThis is the latest version of Orange (for Python 3). The deprecated version of Orange 2.7 (for Python 2.7) is still available ([binaries] and [sources]).\n\n[Orange]: https://orange.biolab.si/\n[binaries]: https://orange.biolab.si/orange2/\n[sources]: https://github.com/biolab/orange\n\n\nInstalling with Miniconda / Anaconda\n------------------------------------\n\nOrange requires Python 3.6 or newer.\n\nFirst, install [Miniconda] for your OS. Create virtual environment for Orange:\n\n    conda create python=3 --name orange3\n\nIn your Anaconda Prompt add conda-forge to your channels:\n\n    conda config --add channels conda-forge\n\nThis will enable access to the latest Orange release. Then install Orange3:\n\n    conda install orange3\n\n[Miniconda]: https://docs.conda.io/en/latest/miniconda.html\n\nTo install the add-ons, follow a similar recipe:\n\n    conda install orange3-<addon name>\n\nSee specific add-on repositories for details.\n\nInstalling with pip\n-------------------\n\nTo install Orange with pip, run the following.\n\n    # Install some build requirements via your system's package manager\n    sudo apt install virtualenv build-essential python3-dev\n\n    # Create a separate Python environment for Orange and its dependencies ...\n    virtualenv --python=python3 --system-site-packages orange3venv\n    # ... and make it the active one\n    source orange3venv/bin/activate\n\n    # Install Qt dependencies for the GUI\n    pip install PyQt5 PyQtWebEngine\n\n    # Install Orange\n    pip install orange3\n\nStarting Orange GUI\n-------------------\n\nTo start Orange GUI from the command line, run:\n\n    orange-canvas\n    # or\n    python3 -m Orange.canvas\n\nAppend `--help` for a list of program options.\n"
