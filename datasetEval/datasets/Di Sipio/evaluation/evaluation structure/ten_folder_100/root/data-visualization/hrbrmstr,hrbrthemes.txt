It\xe2\x80\x99s a an extract/riff of\n[`hrbrmisc`](https://github.com/hrbrmstr/hrbrmisc) created by request.\n\nThe core theme: `theme_ipsum` (\xe2\x80\x9cipsum\xe2\x80\x9d is Latin for \xe2\x80\x9cprecise\xe2\x80\x9d) uses\nArial Narrow which should be installed on practically any modern system,\nso it\xe2\x80\x99s \xe2\x80\x9cfree\xe2\x80\x9d-ish. This font is condensed, has solid default kerning\npairs and geometric numbers. That\xe2\x80\x99s what I consider the \xe2\x80\x9cfont trifecta\xe2\x80\x9d\nmust-have for charts. An additional quality for fonts for charts is that\nthey have a diversity of weights. Arial Narrow (the one on most systems,\nanyway) does not have said diversity but this quality is not (IMO) a\n\xe2\x80\x9cmust have\xe2\x80\x9d.\n\nThe following functions are implemented/objects are exported:\n\nThemes:\n\n  - `theme_ipsum`: Arial Narrow\n  - `theme_ipsum_es`: Ccon Sans Condensed\n  - `theme_ipsum_rc`: Roboto Condensed\n  - `theme_ipsum_ps`: IBM Plex Sans font\n  - `theme_ipsum_pub`: Public Sans\n  - `theme_ipsum_tw`: Titilium Web\n  - `theme_modern_rc`: Roboto Condensed dark theme\n  - `theme_ft_rc`: Dark theme based on FT\xe2\x80\x99s dark theme (Roboto\n    Condensed)\n\nScales (that align with various themes):\n\n  - `scale_color_ipsum`: Discrete color & fill scales based on the ipsum\n    palette\n  - `scale_colour_ipsum`: Discrete color & fill scales based on the\n    ipsum palette\n  - `scale_fill_ipsum`: Discrete color & fill scales based on the ipsum\n    palette\n  - `scale_color_ft`: Discrete color & fill scales based on the FT\n    palette\n  - `scale_colour_ft`: Discrete color & fill scales based on the FT\n    palette\n  - `scale_fill_ft`: Discrete color & fill scales based on the FT\n    palette\n  - `scale_x_comma`: X & Y scales with opinionated presets for percent &\n    comma label formats\n  - `scale_x_percent`: X & Y scales with opinionated presets for percent\n    & comma label formats\n  - `scale_y_comma`: X & Y scales with opinionated presets for percent &\n    comma label formats\n  - `scale_y_percent`: X & Y scales with opinionated presets for percent\n    & comma label formats\n\nPalettes/Named Colors:\n\n  - `ipsum_pal`: A muted, qualitative color palette\n  - `ft_cols`: FT color palette\n  - `ft_pal`: A bright qualitative color palette\n  - `ft_text_col`: FT color palette\n\nFonts:\n\n  - `font_an`: Arial Narrow font name R variable aliases\n  - `font_es`: Econ Sans font name R variable aliases\n  - `font_es_bold`: Econ Sans font name R variable aliases\n  - `font_es_light`: Econ Sans font name R variable aliases\n  - `font_rc`: Roboto Condensed font name R variable aliases\n  - `font_rc_light`: Roboto Condensed font name R variable aliases\n  - `font_pub`: Public Sans font name R variable aliases\n  - `font_pub_bold`: Public Sans font name R variable aliases\n  - `font_pub_light`: Public Sans font name R variable aliases\n  - `font_pub_thin`: Public Sans font name R variable aliases\n  - `font_ps`: PlexSans font name R variable aliases\n  - `font_ps_light`: PlexSans font name R variable aliases\n  - `font_tw`: Titillium Web font name R variable aliases\n  - `font_tw_bold`: Titillium Web font name R variable aliases\n  - `font_tw_light`: Titillium Web font name R variable aliases\n\nR Markdown:\n\n  - `ipsum`: ipsum R markdown template\n  - `ipsum_pdf`: ipsum R markdown template for PDF output\n\nUtilities:\n\n  - `flush_ticks`: Makes axis text labels flush on the ends\n  - `ft_geom_defaults`: Change geom defaults from black to custom lights\n    for the FT theme\n  - `gg_check`: Spell check ggplot2 plot labels\n  - `import_econ_sans`: Import Econ Sans Condensed font for use in\n    charts\n  - `import_plex_sans`: Import IBM Plex Sans font for use in charts\n  - `import_roboto_condensed`: Import Roboto Condensed font for use in\n    charts\n  - `import_titillium_web`: Import Titillium Web font for use in charts\n  - `modern_geom_defaults`: Change geom defaults from black to white for\n    the modern theme\n  - `update_geom_font_defaults`: Update matching font defaults for text\n    geoms\n\n### Installation\n\n``` r\ninstall.packages("hrbrthemes", repos = "https://cinc.rud.is")\n# or\nremotes::install_git("https://git.rud.is/hrbrmstr/hrbrthemes.git")\n# or\nremotes::install_git("https://git.sr.ht/~hrbrmstr/hrbrthemes")\n#
