[License](#-license)\n\n## Collect statistics on the use of open source\n\nTOAST UI Chart applies Google Analytics (GA) to collect statistics on the use of open source, in order to identify how widely TOAST UI Chart is used throughout the world. It also serves as important index to determine the future course of projects. location.hostname (e.g. > \xe2\x80\x9cui.toast.com") is to be collected and the sole purpose is nothing but to measure statistics on the usage. To disable GA, use the following `usageStatistics` option when creating charts.\n\n```js\nvar options = {\n  ...\n  usageStatistics: false\n}\n\ntui.chart.barChart(container, data, options);\n```\n\nOr, include tui-code-snippet(**v1.5.0 or later**) and then immediately write the options as follows:\n\n```js\ntui.usageStatistics = false;\n```\n\n## \xf0\x9f\x8c\x8f Browser Support\n| <img src="https://user-images.githubusercontent.com/1215767/34348387-a2e64588-ea4d-11e7-8267-a43365103afe.png" alt="Chrome" width="16px" height="16px" /> Chrome | <img src="https://user-images.githubusercontent.com/1215767/34348590-250b3ca2-ea4f-11e7-9efb-da953359321f.png" alt="IE" width="16px" height="16px" /> Internet Explorer | <img src="https://user-images.githubusercontent.com/1215767/34348380-93e77ae8-ea4d-11e7-8696-9a989ddbbbf5.png" alt="Edge" width="16px" height="16px" /> Edge | <img src="https://user-images.githubusercontent.com/1215767/34348394-a981f892-ea4d-11e7-9156-d128d58386b9.png" alt="Safari" width="16px" height="16px" /> Safari | <img src="https://user-images.githubusercontent.com/1215767/34348383-9e7ed492-ea4d-11e7-910c-03b39d52f496.png" alt="Firefox" width="16px" height="16px" /> Firefox |\n| :---------: | :---------: | :---------: | :---------: | :---------: |\n| Yes | 8+ | Yes | Yes | Yes |\n\n## \xf0\x9f\x8c\x88 True Cross Browser Charts And It\'s Beautiful!\n\n| IE8 | CHROME |\n| --- | --- |\n| ![IE8](https://user-images.githubusercontent.com/35218826/37317347-aa1beb1e-26a7-11e8-860c-19274fbc1570.png) | ![CHROME](https://user-images.githubusercontent.com/35218826/37317349-aba19d08-26a7-11e8-997b-3a960c968238.png) |\n\n\n**No one likes legacy, but sometimes it matters.**  \nWe started TOAST UI Chart because we could not find a project supporting legacy browsers, maintained reliably, and free for commercial use.\n\n**The visuals of the charts in IE8 are exactly as it looks in Chrome.**  \nThe TOAST UI Chart guarantees an identical look for all browsers. \xf0\x9f\x92\xbe Install\n\nThe TOAST UI products can be installed by using the package manager or downloading the source directly.\nHowever, we highly recommend using the package manager.\n\n### Via Package Manager\n\nThe TOAST UI products are registered in two package managers, [npm](https://www.npmjs.com/) and [bower](https://bower.io/).\nInstall by using the commands provided by each package manager.\nWhen using npm, be sure [Node.js](https://nodejs.org) is installed in the environment.\n\n#### npm\n\n```sh\n$ npm install --save tui-chart # Latest version\n$ npm install --save tui-chart@<version> # Specific version\n```\n\n#### bower\n\n```sh\n$ bower install tui-chart # Latest version\n$ bower install tui-chart#<tag> # Specific version\n```\n\n\n### Via Contents Delivery Network (CDN)\nThe TOAST UI Chart is available over a CDN.\n\n- You can use cdn as below.\n\n```html\n<link rel="stylesheet" href="https://uicdn.toast.com/tui.chart/latest/tui-chart.min.css">\n<script src="https://uicdn.toast.com/tui.chart/latest/tui-chart.min.js"></script>\n```\n\n- The types of distribution files are classified into three types depending on whether or not they exist in the code of the dependency file.\n    1. `tui-chart.js` - This is not include dependency modules in the code.\n    2. `tui-chart-all.js` - Includes all dependency modules, including \'babel-polyfill\', in bundle file code\n    3. `tui-chart-polyfill.js` - This includes only babel-polyfill in code.\n\n- Within the download you\'ll find the following directories\n\n```\ntui.chart/\n\xe2\x94\x9c\xe2\x94\x80 latest\n\xe2\x94\x82  \xe2\x94\x9c\xe2\x94\x80 tui-chart.js\n\xe2\x94\x82  \xe2\x94\x9c\xe2\x94\x80 tui-chart.min.js\n\xe2\x94\x82 id="chart"></div>\n```\n\n### JavaScript\n#### Load\nThe TOAST UI Chart can be used by creating an instance with the constructor function. To access the constructor function, import the module using one of the three following methods depending on your environment.\n\n```javascript\nvar chart = tui.chart; /* namespace */\n```\n```javascript\nvar chart = require(\'tui-chart\'); /* CommonJS in Node.js */\n```\n```javascript\nimport chart from \'tui-chart\'; /* ES6 in Node.js */\n```\n\nFactory function needs three parameters: container, data, options\n\n* container: Wrapper HTML element that will contain the chart as a child\n* data: Numerical data the chart will be based on\n* options: functional options including legend, alignment, and tooltip visibilities\n\n``` javascript\nvar container = document.getElementById(\'chart-area\');\nvar data = {\n    category: [...], series: [...]\n};\nvar options = {\n    chart: {width: 700, height: 400}\n};\n\nchart.barChart(container, data, options);\n```\n\nRefer to [details](https://nhn.github.io/tui.chart/latest) for additional informations.\n\n##
