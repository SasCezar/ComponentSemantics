b"\n## mdp - A command-line based markdown presentation tool.\n\n![image](https://cloud.githubusercontent.com/assets/2237222/5810237/797c494c-a043-11e4-9dbd-959cab4055fa.gif)\n\n---\n\n***How to get started:***\n\nmdp needs the ncursesw headers to compile.\nSo make sure you have them installed:\n\n- on Raspbian (Raspberry Pi) you need `libncurses5-dev` and `libncursesw5-dev`\n\nNow download and install mdp:\n\n    $ git clone https://github.com/visit1985/mdp.git\n    $ cd mdp\n    $ make\n    $ make install\n    $ mdp sample.md\n\n- On Arch, you can use the existing [AUR package](https://aur.archlinux.org/packages/mdp-git/).\n- on Cygwin you can use the existing [package](https://cygwin.com/cgi-bin2/package-grep.cgi?grep=mdp.exe) from the setup program.\n- On Debian, you can use the existing [DEB package](https://tracker.debian.org/pkg/mdp-src), or run `apt-get install mdp`.\n- On FreeBSD, you can use the port [misc/mdp](http://www.freshports.org/misc/mdp).\n- On OS-X, use the existing [Homebrew Formula](http://brewformulas.org/Mdp) by running `brew install mdp`.\n- On Slackware, grab the SlackBuild here: (http://slackbuilds.org/apps/mdp/), or run `sbopkg -i mdp`.\n- On Ubuntu, you can use the existing [DEB package](https://launchpad.net/ubuntu/+source/mdp-src), or run `apt-get install mdp`.\n\nMost terminals support 256 colors only if the TERM variable is\nset correctly. To enjoy mdp's color fading feature:\n\n    $ export TERM=xterm-256color\n\n---\n\n***How to use it:***\n\nHorizontal rulers are used as slide separator.\n\nSupports basic markdown formating:\n\n- line wide markup\n    - headlines\n    - code\n    - quotes\n    - unordered list\n\n- in-line markup\n    - bold text\n    - underlined text\n    - code\n\nSupports headers prefixed by @ symbol.\n\n- first two header lines are displayed as title and author\n    in top and bottom bar\n\nReview sample.md for more details.\n\n---\n\n***Controls:***\n\n- h, j, k, l, Arrow keys,\n    Space, Enter, Backspace,\n    Page Up, Page Down - next/previous slide\n- Home, g - go to first slide\n- End, G - go to last slide\n- 1-9 - go to slide n\n- r - reload input file\n- q - exit\n\n\n---\n\n***How to debug it:***\n\nTo make a debug version of `mdp`, just type:\n\n    $ make DEBUG=1\n"
