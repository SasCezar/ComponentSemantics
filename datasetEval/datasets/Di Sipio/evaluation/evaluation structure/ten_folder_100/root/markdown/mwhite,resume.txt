b"# Resume [![Example](https://img.shields.io/badge/example-pdf-green.svg)](https://github.com/mwhite/resume/raw/master/resume.pdf)\n\nThis is a simple Markdown resum\xc3\xa9 template, LaTeX header, and pre-processing\nscript that can be used with [Pandoc](http://johnmacfarlane.net/pandoc/) to generate\nprofessional-looking PDF and HTML output.\n\nThe Markdown flavor supported is\n[Pandoc markdown](http://johnmacfarlane.net/pandoc/README.html#pandocs-markdown).\n\n## Dependencies and Installation\n\n* Pandoc >= 1.9 \n* Python >= 2.7\n* A Tex installation with pdflatex and the Tex Gyre Pagella font, and some\n  packages needed by pandoc.\n\n    On __Ubuntu__ you can get this by doing:\n\n    ```\n    $ sudo apt-get install texlive texlive-latex-extra tex-gyre lmodern\n    ```\n\n    On __macOS__, you could install [BasicTeX](http://tug.org/cgi-bin/mactex-download/BasicTeX.pkg).\n\n    If you run into problems while running `make`, you might need to run the following\n    commands first:\n\n    ```\n    $ sudo tlmgr --verify-repo=all update --self\n    $ sudo tlmgr --verify-repo=all install tex-gyre titlesec\n    ```\n\n## Usage\n\nClone the repo and create a new branch with a different Markdown file for your\nresum\xc3\xa9.\n\nTo generate PDF and HTML versions of each .md file in the directory:\n\n    $ make\n\nIn order to enable visually appealing display of contact information, the\nMarkdown is passed through a Python script that looks for contact details\nbeginning on the fourth line and moves them into a right-aligned, zero-height\nbox at the top of the document.  Lines with bullets (\xe2\x80\xa2) will be treated as\nseparate contact lines in the output.\n\nBy default, an image of your [Gravatar](http://www.gravatar.com) will be added\nto the HTML resum\xc3\xa9.  To avoid this:\n\n    $ GRAVATAR_OPTION=--no-gravatar make\n\n## Unicode characters\n\nThe default setup should handle most unicode characters.  If you still get\nerrors (such as for Chinese characters), install XeTeX and a font that has\nglyphs for the characters you need.\n\n    $ sudo apt-get install texlive-xetex ttf-wqy-zenhei\n\nModify `header.tex` to use the name of your preferred font, then run:\n\n    $ PANDOCARGS='--latex-engine=xelatex' make\n"
