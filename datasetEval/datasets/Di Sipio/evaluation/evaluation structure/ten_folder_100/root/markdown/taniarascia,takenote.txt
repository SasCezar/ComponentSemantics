What you see is what you paste. No WYSIWIG, no formatting pasted from the web, and no features you don\'t need or want.\n\n### Organized\n\nDrag-and-drop notes into categories, instantly search through notes, and pin your favorites to the top.\n\n### Beautiful\n\nBeautiful, clean design with light and dark themes.\n\n### Sync to GitHub\n\nIn progress!\n\n## Reviews\n\n> _"I think the lack of extra crap is a feature."_ \xe2\x80\x94 Craig Lam\n\n## Setup\n\n### Local authentication\n\nIn order to work on TakeNote locally, you must create your own local Auth0 application.\n\n- [Create Auth0 account](https://auth0.com/)\n- Create application with the following settings:\n    - Domain: .auth0.com\n    - Application Type: Single Page Application\n    - Allowed Callback URLs: http://localhost:3000\n    - Allowed Web Origins: http://localhost:3000\n    - Allowed Logout URLs: http://localhost:3000\n    - Allowed Origins (CORS): http://localhost:3000\n- Change config.json to:\n\n```\n{\n  "domain": "your-username.auth0.com",\n  "clientId": "your-client-id"\n}\n```\n\n- Go to Universal Login - Select "New".\n- Ensure "GitHub" permissions is the "Connections".\n\n\n### Install\n\n```bash\ngit clone git@github.com:taniarascia/takenote\ncd takenote\nnpm i\n```\n\n### Run\n\n```bash\nnpm start\n```\n\n### Run in Docker\n\n```bash\ndocker build -t takenote .\ndocker run -p 80:80 takenote\n```\n\n### Seed data\n\nTo seed the app with some test data, paste the contents of `seed.js` into your browser console.\n\n## Contributing\n\nTakeNote is an open source project, and contributions of any kind are welcome!
