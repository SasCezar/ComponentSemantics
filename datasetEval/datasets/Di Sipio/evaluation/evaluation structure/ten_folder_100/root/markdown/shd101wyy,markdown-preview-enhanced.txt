Thank you!\n\n## Introduction\n\nMarkdown Preview Enhanced is an extension that provides you with many useful functionalities such as automatic scroll sync, [math typesetting](https://shd101wyy.github.io/markdown-preview-enhanced/#/math), [mermaid](https://shd101wyy.github.io/markdown-preview-enhanced/#/diagrams?id=mermaid), [PlantUML](https://shd101wyy.github.io/markdown-preview-enhanced/#/diagrams?id=plantuml), [pandoc](https://shd101wyy.github.io/markdown-preview-enhanced/#/pandoc), PDF export, [code chunk](https://shd101wyy.github.io/markdown-preview-enhanced/#/code-chunk), [presentation writer](https://rawgit.com/shd101wyy/markdown-preview-enhanced/master/docs/presentation-intro.html), etc. A lot of its ideas are inspired by [Markdown Preview Plus](https://github.com/atom-community/markdown-preview-plus) and [RStudio Markdown](http://rmarkdown.rstudio.com/).\n\nFeel free to ask questions, post issues, submit pull request, and request new features.\n\nFor more information about this project and how to use this extension, please check out our documentation \xe2\xac\x87\xef\xb8\x8e\n\n## Keybindings\n\n> The <kbd>cmd</kbd> key for _Windows_ is <kbd>ctrl</kbd>.\n\n| Shortcuts                                   | Functionality              |\n| ------------------------------------------- | -------------------------- |\n| <kbd>ctrl-shift-m</kbd>                     | Toggle preview             |\n| <kbd>ctrl-shift-s</kbd>                     | Sync preview / Sync source |\n| <kbd>shift-enter</kbd>                      | Run Code Chunk             |\n| <kbd>ctrl-shift-enter</kbd>                 | Run all Code Chunks        |\n| <kbd>cmd-=</kbd> or <kbd>cmd-shift-=</kbd>  | Preview zoom in            |\n| <kbd>cmd--</kbd> or <kbd>cmd-shift-\\_</kbd> | Preview zoom out           |\n| <kbd>cmd-0</kbd>                            | Preview reset zoom         |\n| <kbd>esc</kbd>                              | Toggle sidebar TOC         |\n\n## Changelog\n\nNewest updates for the current minor version can be found at the [newest
