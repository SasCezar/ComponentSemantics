src="images/gifs/toggle-bold.gif" alt="toggle bold gif" width="282px">\n\n  ![check task list](images/gifs/keybinding-tasklist.gif) (check/uncheck task list)\n\n  See full key binding list in [keyboard shortcuts](#keyboard-shortcuts) section\n\n- **Table of contents**\n\n  ![toc](images/toc.png)\n\n  - The indentation rules (tab or spaces) of TOC will be the same of your current file (find it in the right bottom corner)\n\n  - To make TOC compatible with GitHub, you need to set option `githubCompatibility` to `true`\n\n  - Use `<!-- omit in toc -->` to ignore specific heading in TOC\n\n- **List editing**\n\n  ![on enter key 1](images/gifs/on-enter-key1.gif) (<kbd>Enter</kbd>)\n\n  ![on enter key 2](images/gifs/on-enter-key2.gif) (<kbd>Enter</kbd>)\n\n  ![on tab key](images/gifs/on-tab-key.gif) (<kbd>Tab</kbd>)\n\n  ![on backspace key](images/gifs/on-backspace-key.gif) (<kbd>Backspace</kbd>)\n\n  ![marker fixing](images/gifs/marker-fixing.gif) (auto fix ordered list markers)\n\n- **Print Markdown to HTML**\n\n  - Command `Markdown: Print current document to HTML`\n\n  - It\'s recommended to print the exported HTML to PDF with browser (e.g. Chrome) if you want to share your documents with others\n\n- **GitHub Flavoured Markdown**\n\n  - Table formatter\n\n    ![table-formatter](images/gifs/table-formatter.gif)\n\n    (Note that the keybinding is <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>I</kbd> on Linux)\n\n  - Task list\n\n- **Math**\n\n  ![math](images/math.png)\n\n- **Auto completions**\n\n  - Images\n\n    ![image paths](images/image-completions.png)\n\n  - Math functions\n\n    ![math completions](images/math-completions.png)\n\n  - Reference links\n\n    ![suggest ref link](images/gifs/suggest-ref-link-light.png)\n\n- **Others**\n\n  - Paste link on selected text\n\n    ![paste link](images/gifs/paste-link-light.gif)\n\n  - Override "Open Preview" keybinding with "Toggle Preview", which means you can close preview using the same keybinding (<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>V</kbd> or <kbd>Ctrl</kbd> + <kbd>K</kbd> <kbd>V</kbd>).\n\n## Available Commands\n\n- Markdown: Create Table of Contents\n- Markdown: Update Table of Contents\n- Markdown: Toggle code span\n- Markdown: Print current document to HTML\n- Markdown: Toggle math environment\n- Markdown: Toggle list\n\n## Keyboard Shortcuts\n\n<details>\n<summary>Table</summary>\n\n| Key                                               | Command                      |\n|
