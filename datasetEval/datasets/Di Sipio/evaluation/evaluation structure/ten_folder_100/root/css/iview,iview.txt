which you can use to quickly build an iView-based project.\n\nWe also provide a starter kit [iview-project](https://github.com/iview/iview-project) for you.\n\n### Install iView\n\nUsing npm:\n```\nnpm install iview --save\n```\n\nUsing a script tag for global use:\n\n```html\n<script type="text/javascript" src="iview.min.js"></script>\n<link rel="stylesheet" href="dist/styles/iview.css">\n```\n\nYou can find more info [on the website](https://www.iviewui.com/docs/guide/install-en).\n\n## Usage\n\n```vue\n<template>\n    <Slider v-model="value" range />\n</template>\n<script>\n    export default {\n        data () {\n            return {\n                value: [20, 50]\n            }\n        }\n    }\n</script>\n```\n\nUsing css via `import`:\n\n```js\nimport \'iview/dist/styles/iview.css\';\n```\n\n## Compatibility\n\n- Supports Vue.js 2.x\n- Supports Vue.js 1.x - [visit 1.0 docs](http://v1.iviewui.com/)\n- Supports SSR\n-
