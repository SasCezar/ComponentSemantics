tooltips made of pure CSS.\n</p>\n\n## Simple tooltips made of pure CSS\nBalloon.css lets you add tooltips to elements **without JavaScript** and in just a few lines of CSS.\n\nIt weights ~1.1kb minified (Brotli).\n\n<p align="center">\n  <img src="sample.gif" width="440" />\n</p>\n\n## Demo & Playground\n\nYou can play with Balloon.css here: https://jsfiddle.net/kuzmd942/\n\nYou also can check more examples at the website: https://kazzkiq.github.io/balloon.css/\n\n## Usage\n\n### Installation\n\n**Using npm:**\n```\nnpm install balloon-css\n```\n\nYou can then import the CSS file directly into your JS (using Webpack, etc):\n\n```js\n// Your index.js file, etc.\nimport \'balloon-css\';\n```\n\nor import the source file if using SASS/SCSS:\n\n```sass\n/* Your main.scss file, etc. */\n@import "~balloon-css/src/balloon"\n```\n\n**CDN version:**\n\nIf you don\'t want to use npm, there is a CDN option:\n\n```html\n<!-- This link will always fetch the latest version of Balloon.css -->\n<link rel="stylesheet" href="https://unpkg.com/balloon-css/balloon.min.css">\n```\n\n**Manually:**\nSimply download [`balloon.min.css`](https://raw.githubusercontent.com/kazzkiq/balloon.css/master/balloon.min.css) from this repo and add it to your HTML. e.g.\n\n```html\n<link rel="stylesheet" href="path/to/balloon.min.css">\n```\n\n### Positioning\nFor positioning, use `data-balloon-pos` attribute with one of the values: `up`, `down`, `left`, `right`, `up-left`, `up-right`, `down-left` or `down-right`:\n\n```html\n<button aria-label="Whats up!" data-balloon-pos="up">Hover me!</button>\n<button aria-label="Whats up!" data-balloon-pos="left">Hover me!</button>\n<button aria-label="Whats up!" data-balloon-pos="right">Hover me!</button>\n<button aria-label="Whats up!" data-balloon-pos="down">Hover me!</button>\n\n<button aria-label="Whats up!" data-balloon-pos="up-left">Hover me!</button>\n<button aria-label="Whats up!" data-balloon-pos="up-right">Hover me!</button>\n<button aria-label="Whats up!" data-balloon-pos="down-left">Hover me!</button>\n<button aria-label="Whats up!" data-balloon-pos="down-right">Hover me!</button>\n```\n\n### Showing tooltips programatically\nIf you want to show tooltips even when user interaction isn\'t happening, you can simply use the `data-balloon-visible` attribute:\n\n```html\n<button aria-label="Whats up!" data-balloon-pos="up" id="show-tip">Hover me!</button>\n<script>\n  const btn = document.getElementById(\'show-tip\')\n  btn.setAttribute(\'data-balloon-visible\', \'\')\n</script>\n```\n\n### Disabling animation\n\nIf for some reason you do not want animations in your tooltips, you can use the `data-balloon-blunt` attribute for that.\n\n```html\n<button data-balloon-blunt aria-label="No animation!" data-balloon-pos="up">No animation!</button>\n```\n\n### Customizing Tooltips\n\nBalloon.css exposes three CSS variables to make it easier to customize tooltips: `--balloon-color`, `--balloon-font-size` and `--balloon-move`. This way you can use custom CSS to make your own tooltip styles:\n\n```css\n/* Add this to your CSS */\n.tooltip-red {\n  --balloon-color: red;\n}\n\n.tooltip-big-text {\n  --balloon-font-size: 20px;\n}\n\n.tooltip-slide {\n  --balloon-move: 30px;\n}\n```\n\nIf you want to customize tooltips globally, use the `:root` selector:\n\n```css\n/* All tooltips would now be blue */\n:root {\n  --balloon-color: blue;\n}\n```\n\n### Glyphs and Icon Fonts\nYou can also add any HTML special character to your tooltips, or even use third-party Icon fonts:\n\n```html\n<button aria-label="HTML special characters: &#9787; &#9986; &#9820;" data-balloon-pos="up">Hover me!</button>\n<button aria-label="Emojis: \xf0\x9f\x98\x80 \xf0\x9f\x98\xac \xf0\x9f\x98\x81 \xf0\x9f\x98\x82 \xf0\x9f\x98\x83 \xf0\x9f\x98\x84 \xf0\x9f\x98\x85 \xf0\x9f\x98\x86" data-balloon-pos="up">Hover me!</button>\n```\n\nExample using [Font Awesome](https://fortawesome.github.io/Font-Awesome/):\n\n```html\n<button class="fa" aria-label="Font Awesome: &#xf030; &#xf133; &#xf1fc; &#xf03e; &#xf1f8;" data-balloon-pos="up">Hover me!</button>\n```\n\n### Contributing\nBalloon.css is mantained in SASS and LESS.
