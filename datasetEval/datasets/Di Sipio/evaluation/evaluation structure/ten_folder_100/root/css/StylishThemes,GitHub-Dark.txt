[Node.js](https://nodejs.org) 12 or greater, then install dependencies via `npm install` after which you can use the various commands available:\n\n- `npx grunt`: The default grunt task builds a custom theme using a provided `build.json` file. See the [wiki](https://github.com/StylishThemes/GitHub-Dark/wiki/Build) for details.\n- `npm run usercss`: Build a [Usercss](https://github.com/openstyles/stylus/wiki/Usercss) style.\n- `npx grunt themes`: Parse all theme files and combine them into minified styles in the `themes` directory.\n- `npm run clean`: Reformat `github-dark.css` to conform the style guide.\n- `npm run lint`: Run Stylelint on `github-dark.css`.\n- `npm run authors`: Regenerate the `AUTHORS` file based on git history.\n- `npm run imagemin`: Minify all images present in the `image` directory.\n- `npm run generate`: Regenerate auto-generated CSS rules based on GitHub\'s stylesheet. Run this regularly.\n- `npm run patch`: Increment the version on the patch version number, update headers and create a commit.\n- `npm run minor`: Increment the version on the minor version number, update headers and create a commit.\n- `npm run major`: Increment the version on the major version number, update headers and create a commit.\n- `npm run update`: Update and install dependencies.\n\nThanks to all that have [contributed](./AUTHORS) so far!\n\nAnd thanks for the shoutout on the [JS Party](https://changelog.com/jsparty/20#transcript-71) podcast!\n'
