b'# osmium\n\nOperating System written in Rust on RISC V(32im)\n\n## Rust version\n\n```\nrustc 1.34.0-nightly (f29b4fbd7 2019-01-31)\n```\n\n## Setup\n\n1. install rust/cargo.\n2. install riscv-toolchain\n3.\n\n```\nrustup component add rust-src\nrustup install nightly\ncargo install cargo-xbuild\n```\n\n4. `make setup` (install auxiliary tools(emulator and elf2bin translator). This part needs g++, Python3 and pipenv)\n\n\n## Build\n\n```\nmake build\n```\n\n## Run on the Emulator\n\n```\nmake run\n```\n'
