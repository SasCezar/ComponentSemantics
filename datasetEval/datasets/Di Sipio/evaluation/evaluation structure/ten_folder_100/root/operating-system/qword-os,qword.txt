b'# The qword kernel\n\n[![goto counter](https://img.shields.io/github/search/qword-os/qword/goto.svg)](https://github.com/qword-os/qword/search?q=goto)\n\nThe qword-OS kernel, a fully featured, capable x86_64 sex machine!\n\nGet a prebuilt qword distro at: https://ci.oogacraft.com/job/qword/lastSuccessfulBuild/artifact/qword.hdd\n\nAs said, this is just the kernel. To build the whole system, follow the\ninstructions at <https://github.com/qword-os/build>.\n\n## Build requirements\n\nThese are the tools needed for the build:\n- `git` (only for the initial download).\n- `bash`.\n- `make`.\n- `gcc`, 8 or higher.\n- `nasm`.\n- `objcopy`.\n- `qemu` (to test it).\n\n## Building\n\n```bash\n# Clone repo wherever you like and enter.\ngit clone https://github.com/qword-os/qword.git\ncd qword\n# Make the kernel, you can replace the 4 in -j4 with your number of cores + 1.\nmake clean && make -j4\n# Install in PREFIX, may need root permissions depending on the place.\nsudo make install PREFIX=..\n# Uninstall if needed in the specified PREFIX.\nsudo make uninstall PREFIX=..\n```\n'
