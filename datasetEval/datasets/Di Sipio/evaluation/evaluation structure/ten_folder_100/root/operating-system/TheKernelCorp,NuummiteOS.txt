b"# NuummiteOS\n\nAn OS written in [Crystal][crystal_home] as a Proof of Concept.   \nJoin our team on [Discord][discord] or [IRC][webchat] ( `#nuummite @ int0x10.com:6697 (SSL)` )\n\n## Contributing\n\nWe're always looking for help with our [Projects][projects]!\n\n## Building on Linux/WSL\n\n- Get an [i686-elf gcc cross-compiler][cross_cc] going\n- Install the [latest Crystal compiler][crystal_compiler]\n- Install the [latest version of SCons][scons]\n- Run `scons` to build\n- Run `scons test` to run Runtime tests.\n- Run `scons --qemu-curses` to build and run qemu\n\n## Troubleshooting\n\n**`xorriso : FAILURE : Cannot find path '/efi.img' in loaded ISO image`** `or`   \n**`grub-mkrescue: error: ``mformat`` invocation failed`**:\n\n- On Arch: `sudo pacman -Sy mtools`\n- On Debian/Ubuntu/WSL: `sudo apt-get install mtools`\n- On Fedora/RedHat/CentOS: `sudo yum install mtools`\n\n[cross_cc]: http://wiki.osdev.org/GCC_Cross-Compiler\n[crystal_home]: https://crystal-lang.org\n[crystal_compiler]: https://crystal-lang.org/docs/installation/index.html\n[discord]: https://discord.gg/nmESdX8\n[webchat]: http://int0x10.com/webchat?nick=&channels=nuummite%2Cprogramming%2C%23chat\n[scons]: http://scons.org/\n[projects]: https://github.com/TheKernelCorp/NuummiteOS/projects\n"
