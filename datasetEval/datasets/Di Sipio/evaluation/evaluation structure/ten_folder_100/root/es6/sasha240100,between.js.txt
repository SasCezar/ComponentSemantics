There are certain things that we were following while developed this library, we wanted to make it:\n\n- **Lightweight** :snowflake: JS Tweening library.\n > The library is only `9.08 Kb (3Kb gzip)`\n- **Performant** :zap: JS Tweening library. \n > It uses optimization patterns to speed up & smooth animation.\n- **Modern** :gem: JS Tweening library\n > The library is written in `ES6`, compiled to ES5 for global browsers support and provides `ES6 API`.\n\n# Install \n## With npm \n\n```\n$ npm install between.js\n```\n\n## Or fetch from CDN\n\n```\n<script src="https://rawgit.com/sasha240100/between.js/master/build/between.js"></script>\n```\n\n# Basic usage\n## Module\n```javascript\nimport Between from \'between.js\';\n\n// new Between(from, to).time(duration)\nnew Between(1, 10).time(1000)\n  .on(\'update\', (value) => { // This callback is executed in every frame\n      console.log(value);\n  });\n```\n\n## Or in HTML:\n\n```html\n<script src="./path/to/between.js"></script>\n<script>\n  new Between(1, 10).time(1000)\n    .on(\'update\', (value) => {\n        console.log(value);\n    });\n</script>\n```\n\n# API\n\n```js\n// Constructor\nnew Between(\n [Number|Object|Array] from, \n [Number|Object|Array] to\n)\n\n// Methods\n  .time([Number] duration) // Set duration\n  .loop([String] mode, [?Number] repeatTimes) // Set loop mode, if "repeatTimes" is falsy, treats as "endless"\n  .easing([Function] easing) // Set easing function\n  .on([String] eventName, [Function] callback) // Add event listener\n  .pause() // Pauses\n  .play() // Resumes\n\n// Getters\n  .isPaused // returns true if paused\n```\n\n\n\n> There is no need to "start" the tween. It is executed immediately once it was created.\n\n# Events\n\n```javascript\nimport Between from \'between.js\';\n\nnew Between(1, 10).time(1000)\n  .on(\'update\', (value) => {\n      console.log(value); \n  })\n  .on(\'start\', (value) => {\n      console.log(value);\n  })\n  .on(\'pause\', (value) => {\n      console.log(value); \n  })\n  .on(\'play\', (value) => {\n      console.log(value);\n  })\n  .on(\'complete\', (value) => {\n      console.log(value);\n  });\n```\n\n# Different values\n\n* Numbers\n* Arrays\n* Objects\n\n**Numbers**\n\n```javascript\nimport times\n\n```javascript\nimport Between from \'between.js\';\n\nnew Between(1, 10).time(4000)\n  .loop(\'repeat\', N)\n  .on(\'update\', (value, {times}) => {\n      console.log(value);\n      console.log(times);\n  });\n```\n\n[Example](http://between.js.org/loop-repeat.html)\n\nRepeat endless\n\n```javascript\nimport Between from \'between.js\';\n\nnew Between(1, 10).time(4000)\n });\n```\n\n[Example](http://between.js.org/loop-bounce.html)\n\n# Easing\n\n```javascript\nimport Between from \'between.js\';\nimport Easing from \'easing-functions\';\n\n// choose easing mode frome
