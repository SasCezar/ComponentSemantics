b'# Express ES2017 REST API Boilerplate\n[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg?style=flat-square)](http://makeapullrequest.com) [![npm version](https://badge.fury.io/js/express-rest-es2017-boilerplate.svg)](https://badge.fury.io/js/express-rest-es2017-boilerplate) [![Build Status](https://travis-ci.org/danielfsousa/express-rest-es2017-boilerplate.svg?branch=master)](https://travis-ci.org/danielfsousa/express-rest-es2017-boilerplate) [![Coverage Status](https://coveralls.io/repos/github/danielfsousa/express-rest-es2017-boilerplate/badge.svg?branch=master)](https://coveralls.io/github/danielfsousa/express-rest-es2017-boilerplate?branch=master)\n\nBoilerplate/Generator/Starter Project for building RESTful APIs and microservices using Node.js, Express and MongoDB\n\n## Features\n\n - No transpilers, just vanilla javascript\n - ES2017 latest features like Async/Await\n - CORS enabled\n - Uses [yarn](https://yarnpkg.com)\n - Express + MongoDB ([Mongoose](http://mongoosejs.com/))\n - Consistent coding styles with [editorconfig](http://editorconfig.org)\n - [Docker](https://www.docker.com/) support\n - Uses [helmet](https://github.com/helmetjs/helmet) to set some HTTP headers for security\n - Load environment variables from .env files with [dotenv](https://github.com/rolodato/dotenv-safe)\n - Request validation with [joi](https://github.com/hapijs/joi)\n - Gzip compression with [compression](https://github.com/expressjs/compression)\n - Linting with [eslint](http://eslint.org)\n - Tests with [mocha](https://mochajs.org), [chai](http://chaijs.com) and [sinon](http://sinonjs.org)\n - Code coverage with [istanbul](https://istanbul.js.org) and [coveralls](https://coveralls.io)\n - Git hooks with [husky](https://github.com/typicode/husky) \n - Logging with [morgan](https://github.com/expressjs/morgan)\n - Authentication and Authorization with [passport](http://passportjs.org)\n - API documentation generation with [apidoc](http://apidocjs.com)\n - Continuous integration support with [travisCI](https://travis-ci.org)\n - Monitoring with [pm2](https://github.com/Unitech/pm2)\n\n## Requirements\n\n - [Node v7.6+](https://nodejs.org/en/download/current/) or [Docker](https://www.docker.com/)\n - [Yarn](https://yarnpkg.com/en/docs/install)\n\n## Getting Started\n\n#### Clone the repo and make it yours:\n\n```bash\ngit clone --depth 1 https://github.com/danielfsousa/express-rest-es2017-boilerplate\ncd express-rest-es2017-boilerplate\nrm -rf .git\n```\n\n#### Install dependencies:\n\n```bash\nyarn\n```\n\n#### Set environment variables:\n\n```bash\ncp .env.example .env\n```\n\n## Running Locally\n\n```bash\nyarn dev\n```\n\n## Running in Production\n\n```bash\nyarn start\n```\n\n## Lint\n\n```bash\n# lint code with ESLint\nyarn lint\n\n# try to fix ESLint errors\nyarn lint:fix\n\n# lint and watch for changes\nyarn lint:watch\n```\n\n## Test\n\n```bash\n# run all tests with Mocha\nyarn test\n\n# run unit tests\nyarn test:unit\n\n# run integration tests\nyarn test:integration\n\n# run all tests and watch for changes\nyarn test:watch\n\n# open nyc test coverage reports\nyarn coverage\n```\n\n## Validate\n\n```bash\n# run lint and tests\nyarn validate\n```\n\n## Logs\n\n```bash\n# show logs in production\npm2 logs\n```\n\n## Documentation\n\n```bash\n# generate and open api documentation\nyarn docs\n```\n\n## Docker\n\n```bash\n# run container locally\nyarn docker:dev\n\n# run container in production\nyarn docker:prod\n\n# run tests\nyarn docker:test\n```\n\n## Deploy\n\nSet your server ip:\n\n```bash\nDEPLOY_SERVER=127.0.0.1\n```\n\nReplace my Docker username with yours:\n\n```bash\nnano deploy.sh\n```\n\nRun deploy script:\n\n```bash\nyarn deploy\n```\n\n## Tutorials\n -
