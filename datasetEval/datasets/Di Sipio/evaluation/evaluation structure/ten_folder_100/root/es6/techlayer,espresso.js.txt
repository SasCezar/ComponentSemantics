b"# Espresso.js\n\nEspresso.js is a tiny MVC library inspired by [Backbone](http://backbonejs.org) and [React](http://facebook.github.io/react/) with a focus on simplicity and speed.\n\nWe've aimed to bring the ideas of unidirectional data flow of [Flux](http://facebook.github.io/flux/docs/overview.html) to a simple, Backbone-style library.\n\n### Features\n\n- tiny, less than 500 lines and 3kb gzipped\n- zero dependencies\n- performance and memory focused\n- does not aim to support anything below IE10, but may work on older browsers using a [shim](https://github.com/termi/ES5-DOM-SHIM)\n\n### Documentation\n\nBackbone-style docs can be found [here](http://techlayer.com/espresso.js)\n\nAll documentation is automatically generated from `docs/index.md` using [Bocco](https://github.com/akrymski/bocco).\n\n### Getting Started\n\nIf you're using Browserify or Node/CommonJS, simply install the package:\n\n```$ sudo npm install --save espresso.js```\n\nAlternatively grab the [standalone version](https://raw.githubusercontent.com/techlayer/espresso.js/master/espresso.min.js?token=AAamF6ZPKrH6WZ5pN6wwM4QtQphAdmbLks5Ua2ecwA%3D%3D) that you can import with a `<script>` tag or checkout the [GitHub Repo](https://github.com/techlayer/espresso.js).\n\nNow dive in and check out the [To-Do Example](https://rawgit.com/techlayer/espresso.js/master/examples/todomvc/index.html) app.\n\n### Testing\n\n``` bash\n$ npm test\n```\n\n### Building\n\nIf you need to use this module outside a CommonJS environment, \nyou can build a standalone UMD module as follows:\n\n``` bash\n$ npm run build\n```\n\n"
