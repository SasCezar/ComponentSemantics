b'<p align="center">\n  <img src ="docs/bounty.PNG"/>\n</p>\n<p align="center" style="font-size: 1.2rem;">SVG library for transitioning numbers with motion blur</p>\n\n[![npm version](https://badge.fury.io/js/bounty.svg)](https://www.npmjs.com/package/bounty)\n[![Dependencies](https://david-dm.org/coderitual/bounty.svg)](https://david-dm.org/coderitual/bounty)\n[![npm downloads](https://img.shields.io/npm/dt/bounty.svg?maxAge=2592000)](https://www.npmjs.com/package/bounty)\n> JavaScript odometer or slot machine effect library for smoothly transitioning numbers with motion blur. Library uses functional approach and ES7 Function Bind Syntax. Internally based on SVG.\n\n<p align="center"><img src ="docs/logo.gif"/></p>\n\nSee the **[live version](https://coderitual.github.io/bounty/examples/)**.\n\n## Installation\nTo install the stable version:\n\n`npm install --save bounty`\n\n## Examples\nThe API is really simple and straigthforward:\n```js\nimport bounty from `bounty`;\n\nbounty({ el: \'.js-bounty\', value: \'\xc2\xa342,000,000\' });\n```\n\nYou can use it with other **options**:\n```js\nimport bounty from `bounty`;\n\nbounty({\n  el: \'.js-bounty\',\n  value: \'\xc2\xa342,000,000\',\n  initialValue: \'\xc2\xa3900,000\',\n  lineHeight: 1.35,\n  letterSpacing: 1,\n  animationDelay: 100,\n  letterAnimationDelay: 100\n});\n```\nIf you want to **cancel** the ongoing animation just call returned function:\n```js\nimport bounty from `bounty`;\n\nconst cancel = bounty({ el: \'.js-bounty\', value: \'\xc2\xa342,000,000\' });\ncancel();\n```\n\nLibrary is built using UMD thus the following usage in HTML is possible.\n\n```html\n<div class="js-bounty"></div>\n<script src="/bounty.js"></script>\n<script>\n  bounty.default({ el: \'.js-bounty\', value: \'\xc2\xa342,000,000\' })\n</script>\n```\n\nThe UMD build is also available on unpkg:\n\n```html\n<script src="https://unpkg.com/bounty@1.1.6/lib/bounty.js"></script>\n```\nYou can find the library on `window.bounty`.\n\n## That\'s it?\nYea! That\'s it. Other options like `font-family` and `font-size` are taken from **computed styles** so you can just style it like the other layers.\n```css\n.js-bounty {\n  font-size: 60px;\n  font-family: Roboto;\n  fill: #fff;\n  text-shadow: 1px 1px 5px rgba(0, 0, 0, 0.5);\n}\n```\n\n## How?\n\n If you\'re interested how it\'s made, see the **[presentation](http://slides.com/coderitual/odoo-js)**.\n\n## Roadmap\nThere is a work in progress to implement additional features:\n* [ ] `from` `to` API.\n* [ ] Full ASCII transition support.\n* [ ] Control animation.\n* [ ] Introduce Webcomponents API `<svg-bounty>`\n\n<p align="center"><img src ="docs/example2.gif"/></p>\n\n## License\nThe library is available under the MIT license.
