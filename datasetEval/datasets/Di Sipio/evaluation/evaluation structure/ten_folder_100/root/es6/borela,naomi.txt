installation\n\nClone the repository in your Sublime Text \xe2\x80\x9cPackages\xe2\x80\x9d directory:\n\n    git clone https://github.com/borela/naomi.git Naomi\n\nThe \xe2\x80\x9cPackages\xe2\x80\x9d directory is located at:\n\n* **OS X**: `~/Library/Application Support/Sublime Text 3/Packages`\n* **Linux**: `~/.config/sublime-text-3/Packages`\n* **Windows**: `%APPDATA%\\Sublime Text 3\\Packages`\n\n#### Installing using Package Control\n\n1. Install [Package Control](https://packagecontrol.io/installation).\n2. Run **Package Control: Install Package** command.\n3. Find and install the **Naomi** plugin.\n4. Restart Sublime Text.\n\n## Usage\n\nAfter installing the plugin and restarting Sublime, it will automatically use\nthe syntaxes contained in this package.\n\nYou can also set them manually through:\n\n  1.
