b"# Patchwork\n\n> A decentralized messaging and sharing app built on top of Secure Scuttlebutt (SSB).\n\n- Connect with friends without depending on any central servers.\n- Don't worry about spam, you only get messages from people you follow.\n- Use Patchwork online or offline, the data you need is stored on your device.\n- Sync messages with friends when you're on the same Wi-Fi network.\n- Keep secrets with private messages, which are *always* end-to-end encrypted.\n- Change and improve Patchwork however you'd like, it's free and open source.\n\n## Usage\n\n![Screenshot of Patchwork][screenshot]\n\nNew to Scuttlebutt? Join the network by connecting to a [pub][pub].\n\n1. Choose a pub from the [pub list][pub-list] and copy an invite code.\n2. Open Patchwork and select *Join Pub*.\n3. Paste the invite code and select *Redeem Invite*.\n\nYou're done! Check out `#new-people` to see who else has recently joined.\n\n## Installation\n\nMost people should **[download Patchwork for Windows, macOS, or Linux][gh-dl]**.\n\nAlternatively, you can install Patchwork with your favorite package manager.\n\n- **[npm][npm]:** `npm install --global ssb-patchwork`\n- **[yarn][yarn]:** `yarn global add ssb-patchwork`\n- **[yay][yay]:** `yay -S ssb-patchwork`\n\nBuilding from source? Check out [`INSTALL.md`][install] for more information.\n\n## Contributing\n\nCreate a [new issue][new-issue] to report problems or request features.
