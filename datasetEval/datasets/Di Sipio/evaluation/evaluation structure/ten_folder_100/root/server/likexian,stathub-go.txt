Just ONE binary files is needed for all of this.\n\n## DEMO\n\n![demo](doc/demo.png)\n\n## Feature\n\n- Powered by Golang\n- Only one binary files, collecting and displaying data\n- Easy deploy, no depends, no database required\n- SSL support, your domain support, secure and easy\n\n## Install\n\nYou shall choose a server for master, and install it following the below\n\n### Linux via curl\n\n    curl --insecure https://raw.githubusercontent.com/likexian/stathub-go/master/setup.sh | sh\n\n### Linux via wget\n\n    wget --no-check-certificate -O - https://raw.githubusercontent.com/likexian/stathub-go/master/setup.sh | sh\n\n### Open on your PC browser\n\nFor most systems, it will install and start after this, now you can view it via your PC browser.\n\nThe default url is\n\n    https://ip:15944\n\nThen enter the default password: likexian\n\n### Add a CLIENT (node)\n\nFollow the guide\n\n    https://ip:15944/help\n\n## FAQ\n\n- A client added, but no list on the page ?\n\n    Please refer to the client.log on the node, it may show what is wrong.\n\n- Can I use my domain instead of the https://ip:15944 ?\n\n    Yes, please add an A record on your dns server for one subdomain, then using https://subdomain.yourdomain:15944 .\n\n- Can I use https for visist ?\n\n    Sure, it is SSL by default, using self-signed cert.\n\n- Cal I use my SSL cert, not the self-signed one ?\n\n    Yes, please replace the cert in the cert dir, and it strong recommend that doing this.\n\n- Can I deploy it with my nginx ?\n\nSure, please add the folling config to nginx conf file.\n\n    location / {\n        proxy_pass https://127.0.0.1:15944;\n        proxy_set_header X-Real-IP $remote_addr;\n    }\n\n## Dependencies\n\n- [github.com/likexian/gokit](https://github.com/likexian/gokit)\n- [github.com/likexian/host-stat-go](https://github.com/likexian/host-stat-go)\n- [github.com/likexian/simplejson-go](https://github.com/likexian/simplejson-go)\n\n## LICENSE\n\nCopyright 2015-2019 Li Kexian\n\nLicensed under the Apache License 2.0\n\n## About\n\n- [Li Kexian](https://www.likexian.com/)\n\n## DONATE\n\n- [Help me make perfect](https://www.likexian.com/donate/)\n'
