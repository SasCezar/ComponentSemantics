b'elFinder\n========\n\n**WARNING: IF YOU HAVE OLDER (IN PARTICULAR 2.1.48 OR EARLIER) VERSIONS OF ELFINDER ON PUBLIC SERVERS, IT MAY CAUSE SERIOUS DAMAGE TO YOUR SERVER AND VISITED USER. YOU SHOULD UPDATE TO THE LATEST VERSION OR REMOVE IT FROM THE SERVER.**\n\n[![elFinder file manager for the Web](https://studio-42.github.io/elFinder/images/elFinderScr.png "elFinder file manager for the Web")](https://studio-42.github.io/elFinder/)\n\nelFinder is an open-source file manager for web, written in JavaScript using\njQuery UI. Creation is inspired by simplicity and convenience of Finder program\nused in Mac OS X operating system.\n\n[![Download [License](#license)\n\nBranches\n--------\n-  [master](https://github.com/Studio-42/elFinder/tree/master) - Main development branch\n-  [2.1-src](https://github.com/Studio-42/elFinder/tree/2.1-src) - 2.1 development branch, auto build to 2.1 on commit\n-  [2.1](https://github.com/Studio-42/elFinder/tree/2.1) - 2.1 nightly build branch\n\nFeatures\n--------\n * Usability like the MacOS Finder or Windows Explorer\n * Mobile friendly view for touch devices\n * All operations with files and folders on a remote server (copy, move,\n   upload, create folder/file, rename, etc.)\n * High performance server backend and light client UI\n * Multi-root support\n * Local file system, MySQL, FTP, Box, Dropbox, GoogleDrive and OneDrive volume storage drivers\n * Support AWS S3, Azure, Digital Ocean Spaces and more with [League\\Flysystem](https://github.com/barryvdh/elfinder-flysystem-driver) Flysystem driver\n * Cloud storage (Box, Dropbox, GoogleDrive and OneDrive) drivers\n * Background file/folder upload with Drag & Drop HTML5 support\n * Chunked file upload for large file\n * Upload directly to the folder\n * Upload form URL (or list)\n * List and Icons view\n * Keyboard shortcuts\n * Standard methods of file/group selection using mouse or keyboard\n * Move/Copy files with Drag & Drop\n * Drag & Drop to outside by starting drag with alt/option key press\n * Archives create/extract (zip, rar, 7z, tar, gzip, bzip2)\n * Rich context menu and toolbar\n * Quicklook, preview for common file types\n * Edit text files and images\n * "Places" for your favorites\n * Calculate directory sizes\n * Thumbnails for image, movie files\n * Easy to integrate with web editors (elRTE, CKEditor, TinyMCE)\n * Flexible configuration of access rights, upload file types, user interface\n   and other\n * Extensibility by event handling of backend and client side\n * Simple client-server API based on JSON\n * Supports custom information in info dialog\n * Configuable columns of list view\n * Supports custom CSS class function for the custom folder icon\n * Connector plugin\n     * [AutoRotate](https://github.com/Studio-42/elFinder/blob/2.1-src/php/plugins/AutoRotate/plugin.php) : Auto rotation on file upload of JPEG file by EXIF Orientation.\n     * [AutoResize](https://github.com/Studio-42/elFinder/blob/2.1-src/php/plugins/AutoResize/plugin.php) : Auto resize on file upload.\n     * [Normalizer](https://github.com/Studio-42/elFinder/blob/2.1-src/php/plugins/Normalizer/plugin.php) : UTF-8 Normalizer of file-name and file-path etc.\n     * [Sanitizer](https://github.com/Studio-42/elFinder/blob/2.1-src/php/plugins/Sanitizer/plugin.php) : Sanitizer of file-name and file-path etc.\n     * [Watermark](https://github.com/Studio-42/elFinder/blob/2.1-src/php/plugins/Watermark/plugin.php) : Print watermark on file upload.\n * For more details, see the [Changelog](https://github.com/Studio-42/elFinder/blob/master/Changelog)\n\nRequirements\n------------\n### jQuery / jQuery UI\n * jQuery 1.8.0+\n * jQuery UI 1.9.0+\n    * Required: draggable, droppable, resizable, selectable, button and slider\n    * Recommend: sorter (To make sortable the List column and the Places)\n\n**However, we recommend newest version.**\n\n### Client\n * Modern browsers both of desktop or mobile. elFinder was tested in newest Chrome, Edge, Firefox, IE and Opera\n     - **Caution**: Web App mode ("apple-mobile-web-app-capable" meta tag) on iOS is not work perfectly in elFinder\n\n### Server\n * Any web server\n * PHP 5.2+ (Recommend PHP 5.4 or higher) And for thumbnails - GD / Imagick module / convert(imagemagick) require\n     * Recommend PHP 7.1 or higher to supports non-ASCII character of file path/name on the Windows server\n\nInstallation\n------------\n### Builds (compressed)\n 1. Download and unzip one of the [builds](#downloads) below to your PHP server\n 2. Rename `/php/connector.minimal.php-dist` to `/php/connector.minimal.php`\n 3. Load `/elfinder.html` in your browser to run elFinder\n\n### Source (uncompressed)\n 1. Clone this repository to your PHP server\n\n      ```\n      $ git clone https://github.com/Studio-42/elFinder.git\n      ```\n\n 2. Rename `/php/connector.minimal.php-dist` to `/php/connector.minimal.php`\n 3. Load `/elfinder.src.html` in your browser to run elFinder\n\n### Installer\n - [Setup elFinder 2.1.x nightly with Composer](https://github.com/Studio-42/elFinder/tree/gh-pages/tools/installer/setup_with_composer)\n\nDownloads\n------------\n**Stable releases** ([Changelog](https://github.com/Studio-42/elFinder/blob/master/Changelog))\n + [elFinder 2.1.50](https://github.com/Studio-42/elFinder/archive/2.1.50.zip)\n + [elFinder 2.0.9](https://github.com/Studio-42/elFinder/archive/2.0.9.zip) (deprecated)\n\n**Nightly builds**\n + [elFinder 2.1.x (Nightly)](https://github.com/Studio-42/elFinder/archive/2.1.zip)\n\nDemo sites\n------------\n**2.1.x Nightly**\n + https://studio-42.github.io/elFinder/ (with CORS)\n + https://hypweb.net/elFinder-nightly/demo/2.1/\n\nFAQs\n------------\n\n### Should I use elFinder builds (compressed) or source (uncompressed)?\n\nFor debugging and development, use the [source](#source-uncompressed).
