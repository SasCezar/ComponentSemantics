b"# Ajax Contact Form\n\nA Simple Ajax Contact Form developed in PHP with HTML5 Form validation and pure JavaScript.\n\n## Demo\n\nView [demo here](http://www.pinceladasdaweb.com.br/blog/uploads/ajax-contact-form/).\n\n## Download\n\nYou can download the latest version or checkout all the releases [here](https://github.com/pinceladasdaweb/Ajax-Contact-Form/releases).\n\n## Requirements\n\n* PHP >=5.3\n\n## How to use?\n\nOpen the config.php [`config.php`](contact-form/config/config.php) file and fill with your informations.\n\n```php\n<?php\n\nreturn [\n    'subject' => [\n        'prefix' => '[Contact Form]'\n    ],\n    'emails' => [\n        'to'   => '', // Email to receive emails via the form.\n        'from' => ''  // A valid email where the domain should be the same when the form is hosted.\n    ],\n    'messages' => [\n        'error'   => 'There was an error sending, please try again later.',\n        'success' => 'Your message has been sent successfully.',\n        'validation' => [\n            'emptyname'    => 'Name is required.',\n            'emptyemail'   => 'Email is invalid.',\n            'emptysubject' => 'Subject is required.',\n            'emptymessage' => 'Message is required.'\n        ]\n    ],\n    'fields' => [\n        'name'     => 'Name',\n        'email'    => 'Email',\n        'phone'    => 'Phone',\n        'subject'  => 'Subject',\n        'message'  => 'Message',\n        'btn-send' => 'Send'\n    ]\n];\n```\n\n## Browser
