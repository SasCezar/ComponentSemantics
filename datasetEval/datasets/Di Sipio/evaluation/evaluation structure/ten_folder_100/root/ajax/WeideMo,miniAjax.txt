method\n    data: { name: "WeideMo", age: 26 },         //request params\n    dataType: "json",                           //received dataType\n    success: function (response, xml) {\n        // do timeout\n})\n```\n\n### ready\n\n```javascript\nready(function(){\n    // do something when page ready\n})\n```\n\n## Fetures\n\n### High coverage of browsers\n\nAlmost support all of the browsers,
