b'# Pixel Wave Page Transition\n![Pixel Wave Animation Preview](/dev/img/pixelWave.gif "Pixel Wave Animation Preview")\n![Pixel Wave Animation Mobile](/dev/img/pixelWaveMobile1.gif "Pixel Wave Animation Mobile")\n\n## Install\nAt the bottom of your body:\n```javascript\n<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/1.20.3/TweenMax.min.js"></script>\n<script src="js/PixelWave-min.js"></script>\n\nvar pixelwave = new PixelWave({\n    // options like:\n    // canvasTop: 0\n});\n```\n## Run Animation\n```javascript\npixelwave.start();\n```\n\n## Options\nWhen starting the wave you can add callback functions to get executed at certain times:\n```javascript\npixelwave.start([runsAtStart], [runsAtMiddle], [runsAtEnd]);\n```\n\nYou can pass these options while you init the wave:\n\n| Description                                                                                        | Name                     | Default                      |\n|----------------------------------------------------------------------------------------------------|--------------------------|------------------------------|\n| Set width of Pixel                                                                                 | xSize                    | 48                           |\n| Set height of Pixel                                                                                | ySize                    | 48                           |\n| How many steps there are between full fill and the beginning of the wave. Add as many as you like. | steps                    | [0.1, 0.1, 0.3, 0.5, 0.5, 1] |\n| Color of the wave                                                                                  | color                    | #305EFF                      |\n| Time in s the screen needs to fill.                                                                | speedIn                  | 0.7                          |\n| Time in s the screen needs to empty.                                                               | speedOut                 | 0.7                          |\n| Position top of the canvas element                                                                 | canvasTop                | 0                            |\n| Position left of the canvas element                                                                | canvasLeft               | 0                            |\n| Sets the z-Index for the canvas element                                                            | zIndex                   | 99999                        |\n| This is a function that sets a given size for different devices. (*add values)                     | autoCalculateSquaresSize | false                        |\n| Add a delay in s before animation out runs.                                                        | delayMiddle              | 0                            |\n\n## Full Example with options\n\n```javascript\n<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/1.20.3/TweenMax.min.js"></script>\n<script src="js/PixelWave-min.js"></script>\n\nvar pixelwave = new PixelWave({\n    canvasTop: 50, //for a header\n    speedIn: 3,\n    speedOut: 1,\n    autoCalculateSquaresSize: true,\n    color: \'#000000\'   \n});\n\nfunction start () {console.log(\'start\')}\nfunction middle () {console.log(\'middle\')}\nfunction end () {console.log(\'end\')}\n\ndocument.getElementById(\'startImage\').addEventListener(\'click\', function() {\n    pixelwave.start(start, middle, end);\n})\n```\n### Dependencies\nGreensockets amazing [gsap](https://greensock.com/gsap). \n### Contribution\nIf anyone wants to contribute, just fork the project or write me at vinzenz@sansho.studio :)\n'
