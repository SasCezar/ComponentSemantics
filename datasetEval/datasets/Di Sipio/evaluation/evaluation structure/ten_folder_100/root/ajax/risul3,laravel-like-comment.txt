b'[![Latest Stable Version](https://poser.pugx.org/risul/laravel-like-comment/v/stable)](https://packagist.org/packages/risul/laravel-like-comment)\n[![Total Downloads](https://poser.pugx.org/risul/laravel-like-comment/downloads)](https://packagist.org/packages/risul/laravel-like-comment)\n[![License](https://poser.pugx.org/risul/laravel-like-comment/license)](https://packagist.org/packages/risul/laravel-like-comment)\n\nLaravel like comment\n=====================\n\nlaravel-like-comment is an ajax based like and commenting system for laravel. Which can be used with anything like page, image, post, video etc. User needs to be loged in to be able to like or comment.\n\n## Features\n* Like\n* Dislike\n* Comment\n* Comment voting\n* User avatar in comment\n\n## Demo\n[Try it](http://risul.herokuapp.com/laravel-like-comment)\n\n[Watch](https://www.youtube.com/watch?v=06kcpsnN-bo)\n\n## Installation\n\nRun\n```bash\ncomposer require risul/laravel-like-comment\n```\n\n## Configuration\nAdd \n``` \nrisul\\LaravelLikeComment\\LikeCommentServiceProvider::class\n```\nin your ```service providerr``` list.\n\nTo copy views and migrations run \n``` \nphp artisan vendor:publish\n```\n\nRun\n```\nphp artisan migrate\n```\nIt will create like and comment table.\n\nAdd this style links to your view head\n```html\n    <link href="//cdnjs.cloudflare.com/ajax/libs/semantic-ui/2.2.2/components/icon.min.css" rel="stylesheet">\n    <link href="//cdnjs.cloudflare.com/ajax/libs/semantic-ui/2.2.2/components/comment.min.css" rel="stylesheet">\n    <link href="//cdnjs.cloudflare.com/ajax/libs/semantic-ui/2.2.2/components/form.min.css" rel="stylesheet">\n    <link href="//cdnjs.cloudflare.com/ajax/libs/semantic-ui/2.2.2/components/button.min.css" rel="stylesheet">\n    <link href="{{ asset(\'/vendor/laravelLikeComment/css/style.css\') }}" rel="stylesheet">\n```\n\nAdd jquery and script\n```html\n    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.1.0/jquery.min.js"></script>\n    <script src="{{ asset(\'/vendor/laravelLikeComment/js/script.js\') }}" type="text/javascript"></script>\n```\n\n\nIn `config/laravelLikeComment.php` add user model path\n```php\n\'userModel\' => \'App\\User\'\n```\n\nAdd following code in your user model \n```php\n    /**\n     * Return the user attributes.\n\n     * @return array\n     */\n    public static function getAuthor($id)\n    {\n        $user = self::find($id);\n        return [\n            \'id\'     => $user->id,\n            \'name\'   => $user->name,\n            \'email\'  => $user->email,\n            \'url\'    => \'\',  // Optional\n            \'avatar\' => \'gravatar\',  // Default avatar\n            \'admin\'  => $user->role === \'admin\', // bool\n        ];\n    }\n```\n\n## Usage\nAdd this line at where you want to integrate Like\n```php\n@include(\'laravelLikeComment::like\', [\'like_item_id\' => \'image_31\'])\n```\n`like_item_id:` This is the id of the item,page or model for which the like will be used\n\nAdd this line where you want to integrate Comment\n```php\n@include(\'laravelLikeComment::comment\', [\'comment_item_id\' => \'video_12\'])\n```\n```comment_item_id:``` This is the id of the item, page, or model for which the comment will be used\n'
