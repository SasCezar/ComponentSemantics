b"Open Source Social Network [5.1 LTS]\n======================================\n[![Open Source Love](https://badges.frapsoft.com/os/v2/open-source.svg?v=102)](https://www.opensource-socialnetwork.org/)\n[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg?style=flat-square)](https://www.opensource-socialnetwork.org/)\n[![Build Status](https://travis-ci.org/opensource-socialnetwork/opensource-socialnetwork.svg?branch=v5.x)](https://travis-ci.org/opensource-socialnetwork/opensource-socialnetwork)\n[![Download Latest](https://img.shields.io/badge/Download-Latest%20Version-blue.svg)](https://www.opensource-socialnetwork.org/download)\n[![DeepScan grade](https://deepscan.io/api/teams/4774/projects/6525/branches/54923/badge/grade.svg)](https://deepscan.io/dashboard#view=project&tid=4774&pid=6525&bid=54923)\n\nOpensource-Socialnetwork (OSSN) is a social networking software written in PHP. It allows you to make a social networking website and helps your members build social relationships, with people who share similar professional or personal interests. It is available in 10 international languages.\n\nOSSN is released under the ***Open Source Social Network License (OSSN LICENSE)***\n\nKernighan & Ritchie (K&R Variant 1TBS) indentation standard is used for the OSSN.\n\nLanguages\n==========\n* English\n* German\n* French\n* Romanian\n* Portuguese\n* Turkish\n* Spanish\n* Italian\n* Russian\n* Dutch\n* Hebrew\n\nFront-End Features\n===================\n* User Registration\n* User Login\n* Profile \n* Profile Photo\n* Profile Cover\n* Add/Remove Friends\n* Live Chat\n* Wall posts\n* Photos\n* Ads\n* Groups\n* Tag friends in posts\n* User block system\n* User poke system\n* Ajax Comments\n* Ajax Likes\n* Ajax Photos in comments\n* Group cover photos\n* Repostion Profile/Group cover\n* Notifications\n* Friend Requests\n* Chat Bar\n* Invite Friends\n* Embed Videos\n* Smilies\n* SitePages (terms, privacy, about)\n* Site Search\n* Reset Password\n* Newsfeed page\n* Post Edit\n* Comment Edit\n* Mobile Friendly\n* A photo gallery view.\n* Emojii Support\n\nBackend Features\n=================\n\n* Admin Dashboard for site overview\n* Online users count (male/female) graph\n* Total site users count (by months) graph\n* Update Notification\n* Add User\n* Remove User\n* Edit User\n* Ads Manager\n* Site Cache Settings\n* Site Basic Settings\n* Unvalidated users\n* Manually validate unvalidated users\n* and much more components settings\n\nPrerequisite\n=============\n\nhttps://www.opensource-socialnetwork.org/wiki/view/706/how-to-install-open-source-social-network\n\nDirectory Permissions\n============\nOSSN must be installed into a directory writable by the webserver, or it will not function properly. For most Linux servers running Apache v2, making the parent directory writable is very simple:\n\n```\nchgrp www-data /path/to/ossn_parent_directory\nchmod g+w /path/to/ossn_parent_directory\n``` \n\nIf you are not running Apache v2 or are using a different type of webserver, you'll need to read your system's documentation or speak to your server administrator before proceeding.\n\nInstallation\n============\nhttps://www.opensource-socialnetwork.org/wiki/view/706/how-to-install-open-source-social-network\n\nDEMO\n====\nhttp://demo.opensource-socialnetwork.org/\n\nUPGRADE\n========\nhttps://www.opensource-socialnetwork.org/wiki/view/708/how-to-upgrade-ossn\n\n\nCopyright 2014-2016 Informatikon Technologies (informatikon.com)\n\nCopyright 2016-2019 SOFTLAB24 (https://www.softlab24.com/)\n\nCopyright 2014-2019 Open Source Social Network (https://www.opensource-socialnetwork.org/)\n"
