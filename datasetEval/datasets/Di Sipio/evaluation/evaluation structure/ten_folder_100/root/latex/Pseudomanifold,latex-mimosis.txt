b'# latex-mimosis: A minimal & modern template for your thesis\n\nThis repository contains a minimal & modern LaTeX template for\ndissertations and other university documents.\n\nFor the impatient or curious: [this is what the template looks\nlike](Thesis.pdf).\nYou may also want to take a look at my [my Ph.D. dissertation](http://bastian.rieck.me/research/Dissertation_Rieck_2017.pdf), which uses a predecessor of this template.\n\n# Users\n\nBefore going over the details of this template, why not look at how it\nlooks in practice? The following documents have been typeset with this\ntemplate&nbsp;(or a slightly modified variant of it):\n\n- S. Almasian, [Learning Joint Vector Representations of Words and Named Entities](https://github.com/satya77/Thesis_Entity_Embeddings/blob/master/MasterThesis_SatyaAlmasian.pdf), M.Sc.&thinsp;thesis, Heidelberg University, 2018\n- K. Hanser, [Visualization of Coherence in Meteorological Data](https://github.com/hanserK/master_thesis/blob/master/Thesis_Karsten_Hanser.pdf), M.Sc.&thinsp;thesis, Heidelberg University, 2018\n- P. Jung, [On the Frame of Reference in Flow Visualization](https://github.com/JungStar/master_thesis/blob/master/Thesis.pdf), M.Sc.&thinsp;thesis, Heidelberg University, 2019\n- B. Rieck, [Persistent Homology in Multivariate Data Visualization](http://archiv.ub.uni-heidelberg.de/volltextserver/22914/1/Dissertation.pdf), Ph.D.&thinsp;thesis, Heidelberg University, 2017\n\n# Advantages\n\nThis template aims to be&hellip;\n- clean: no LaTeX trickery\n- minimal: no unnecessary adjustments and decorations\n- modern: typographically pleasing\n\nIt is specifically suited for the European education system because it\nuses A4 paper size by default&mdash;this can be easily adjusted to fit\nyour personal needs, though&nbsp;(see below).\n\nThe class is based on [`KOMA-script`](komascript.de), so it should be\nflexible enough to suit virtually any purpose.\n\n# How to use\n\n- Clone this repository\n- Copy the file `mimosis.cls` into your document directory\n- Add `\\documentclass{mimosis}` to your document preamble\n- Optionally copy the file `Thesis.tex` and the files in `Sources` as\n  a starting point\n- Use `latexmk` to build the document using `pdflatex`\n- Write a nice thesis in LaTeX\n\n# How to customize\n\nThe template is based on the excellent [`KOMA-script`](https://ctan.org/pkg/koma-script)\nclass. You can thus change the appearance of many things quite easily.\nFor example, if you want the thesis to use the `letter` paper format,\njust add\n\n    \\KOMAoptions{paper=letter}\n\nin the preamble of the document and recompile.\n\n# Example\n\nThe repository comes with an example file called `Thesis.tex`. Please\ntake a look at this file in order for more detailed instructions about\nhow to use the class.\n\nIt is recommended to use `latexmk` to build your LaTeX documents. Your\ndistribution might already have this command. If so, you can use\n\n    latexmk\n\nin the main directory of this repository in order to build the example\nfile.\n\n## Required packages for the class\n\nThe template uses various LaTeX packages that you should install using\nyour favourite LaTeX distribution. Some distributions already do this\nautomatically when you compile the document for the first time. Others\nrequire manual updates. Please refer to the documentation of your LaTeX\ndistribution for more details.\n\nHere is a list of packages that you need&nbsp;(I am using the package\nname as specified on CTAN):\n\n- [`amsmath`](https://ctan.org/pkg/amsmath)\n- [`amsthm`](https://ctan.org/pkg/amsthm)\n- [`booktabs`](https://ctan.org/pkg/booktabs)\n- [`csquotes`](https://ctan.org/pkg/csquotes)\n- [`dsfont`](https://ctan.org/pkg/dsfont)\n- [`glossaries`](https://ctan.org/pkg/glossaries)\n- [`graphicx`](https://ctan.org/pkg/graphicx)\n- [`fontspec`](https://ctan.org/pkg/fontspec)&nbsp;(only for LuaTeX and XeTeX users)\n- [`ifluatex`](https://ctan.org/pkg/ifluatex)\n- [`ifpdf`](https://ctan.org/pkg/ifpdf)\n- [`ifxetex`](https://ctan.org/pkg/ifxetex)\n- [`inputenc`](https://ctan.org/pkg/inputenc)&nbsp;(only for pdfTeX users)\n- [`koma-script`](https://ctan.org/pkg/koma-script)\n- [`makeidx`](https://ctan.org/pkg/makeidx)\n- [`paralist`](https://ctan.org/pkg/paralist)\n- [`setspace`](https://ctan.org/pkg/setspace)\n- [`siunitx`](https://ctan.org/pkg/siunitx)\n- [`subcaption`](https://ctan.org/pkg/subcaption)\n- [`xcolor`](https://ctan.org/pkg/xcolor)\n- [`xspace`](https://ctan.org/pkg/xspace)\n\n## Required packages for the example document\n\nTypesetting the example document requires an additional set of packages.\nFeel free to remove them, though&mdash;they are only used for showcasing\nhow a real document might look like.\n\n- [`biblatex`](https://ctan.org/pkg/biblatex)\n- [`bookmark`](https://ctan.org/pkg/bookmark)\n- [`etoolbox`](https://ctan.org/pkg/etoolbox)\n- [`hyperref`](https://ctan.org/pkg/hyperref)\n- [`metalogo`](https://ctan.org/pkg/metalogo)\n\n\nFor pdfTeX users:\n\n- [`ebgaramond`](https://ctan.org/pkg/ebgaramond)\n- [`sourcecodepro`](https://ctan.org/pkg/sourcecodepro)\n\nFor LuaTeX or XeTeX users:\n\n- The Minion Pro font; please use your favourite search engine for this\n  or change the line `\\setmainfont{Minion Pro}` in the preamble to\n  another font&nbsp;(or leave it out entirely)\n\n# License\n\nThe template uses the MIT license.
