b'# limecv <img src="src/images/limecv-icon.png" alt="limecv project icon" width="40px"/>\n\n[![Build Status](https://travis-ci.org/opieters/limecv.svg?branch=master)](https://travis-ci.org/opieters/limecv)\n\n`limecv` is a document class to typeset curriculum vit\xc3\xa6 in (Xe/Lua)LaTeX.\n\nThe design of this document class is detailed in [a series of blog posts][blog].  \n\nExample CV:\n\n<div>\n  <img src="images/cv.png" alt-="CV example" width="300px"/>\n  <img src="images/cover_letter.png" alt-="cover letter example" width="300px"/>\n</div>\n\n`limecv` offers support for a bio/profile, contact details, language skills, (general) skills, projects, education, experience and references. It also features a custom cover letter design. \n\n## Installation\n\nThe package is [available from CTAN](https://ctan.org/pkg/limecv), so you should use a package manager to install this package.\n\nAfter installing this package, make sure you have the following fonts installed:\n\n* [FontAwesome][FA]\n* [Fira][fira] (only needed when using XeLaTeX/LuaLaTeX)\n\nThe following packages need to be installed on your system. They are all available on CTAN: \n  `l3kernel`, \n  `l3packages`, \n  `listings`, \n  `pgf`, \n  `tools`,\n  `graphics`, \n  `xkeyval`, \n  `hyperref`,\n  `xcolor`, \n  `cleveref`, \n  `etoolbox`, \n  `oberdiek`,\n  `ifxetex`, \n  `ifluatex`, \n  `tools`, \n  `url`, \n  `parskip`, \n  `xstring`, \n  `fontspec` (XeLaTeX and LuaLaTeX only),\n  `fontawesome`,\n  `lipsum`,\n  `zapfding`,\n  `luaotfload`, (LuaLaTeX only)\n  `cjk`,\n  `xecjk`,\n  `fandol` and (final 3 for Asian fonts, not needed for LaTeX)\n  `xdvipdfmx`\n\n\n## Documentation\n\nThe class documentation can be found on [CTAN](http://mirrors.ctan.org/macros/latex/contrib/limecv/limecv.pdf).\n\nAlternatively, the documentation can also be created by running `xelatex limecv.dtx`.\n\n## Examples\n\nTo compile the examples in `/examples/`, install the package from CTAN or compile from source by running: `xelatex limecv.ins` and copy the resulting `limecv.cls` file to the `/examples/` folder.\n\n## Licence\n\nLPPL v1.3c See LICENSE file.\n\nDefault profile picture courtesy of Twitter. \n\n\xc2\xa9 2017 - 2018 Olivier Pieters\n\n[blog]: https://olivierpieters.be/blog/archive/tag/limecv\n[FA]: http://fontawesome.io\n[fira]: https://github.com/mozilla/Fira\n[docs]: http://ctan.org/tex-archive/macros/generic/limecv/limecv.pdf\n'
