b'# TeXstudio [![Build Status](https://travis-ci.org/texstudio-org/texstudio.svg?branch=master)](https://travis-ci.org/texstudio-org/texstudio) [![Codacy Badge](https://api.codacy.com/project/badge/Grade/2b708b233294458288e60712d603f8bc)](https://www.codacy.com/app/sunderme/texstudio?utm_source=github.com&amp;utm_medium=referral&amp;utm_content=texstudio-org/texstudio&amp;utm_campaign=Badge_Grade) [![Coverity Scan Build Status](https://img.shields.io/coverity/scan/5228.svg)](https://scan.coverity.com/projects/texstudio)\n\n\nTeXstudio is a fully featured LaTeX editor. Our goal is to make writing LaTeX documents as easy and comfortable as possible. Some of the outstanding features of TeXstudio are an integrated pdf viewer with (almost) word-level synchronization, live inline preview, advanced syntax-highlighting, live checking of references, citations, latex commands, spelling and grammar. Find out more at our website.\n\n## Features \n\n\n### Comfortable Editing\n \n\n* Multi-Cursors  \n\n\nEdit multiple positions at the same time. Vertical blocks can even be copied and pasted.\n\n<p align="center">\n<img alt="Multi-Cursors" src="https://texstudio.org/images/block-cursor.png"> </p>\n\n* Auto completion \n\n\nSuggests possible LaTeX commands as you type. There is a tooltip help explaining the command.\n\nCompletion works also for references and bibliography entries.\n\n<p align="center">\n<img alt="Auto completion" src="https://texstudio.org/images/autocompletion-help.png"> </p>\n\n* More than 1000 mathematical symbols \n\n\nQuick access to more than 1000 mathematical symbols.\n\n<p align="center"><img alt="Symbols" src="https://texstudio.org/images/symbols.png"> </p>\n\n* Bookmarks \n\n\nUse bookmarks too keep references to important places in your text and to navigate there quickly.\n\nTooltips show a preview of the code position.\n\n<p align="center"><img alt="Bookmarks" src="https://texstudio.org/images/bookmarks.png"> </p>\n\n* Link overlays \n\n\nFilenames, references and citations turn into links when moving the mouse over them while holding <Ctrl>.\n\nAlso here, tooltips show a preview of the code position.\n\n<p align="center"><img alt="Link overlays" src="https://texstudio.org/images/link-overlay.png"> </p>\n\n* Assistants for images, tables, formulas, ... \n\n\nGenerate blocks of code using assistants without the need for detailed LaTeX knowledge.\n\n<p align="center"><img alt="assistants" src="https://texstudio.org/images/assistants.png"> </p>\n\n* Drag & drop support for images \n\nWhen you drag an image on the editor the image assistant opens. From there it\'s just one more click to have the code for the image inclusion. Of course, we try to make the paths relative. This also works with multiple images.\n\n<p align="center"><img alt="drag & drop" src="https://texstudio.org/images/dragdropimages.png"> </p>\n\n* Table-formatting \n\n\nYou can copy / paste / insert new table columns with a single click.\n\nThe table-autoformatter aligns the table code.\n\n<p align="center"><img alt="table formatting" src="https://texstudio.org/images/table-formating.png"> </p>\n\n### Keep The Overview\n\n\n*  Structure view\n*  Code folding\n*  Advanced syntax highlighting\n*  Interactive spellchecker\n*  Interactive grammar checker\n*  Interactive reference checker\n*  Clear display of LaTeX errors and warnings (in editor and as list)\n\n<p align="center"><img alt="Inline Checking" src="https://texstudio.org/images/inlineChecking_zoom.png"> </p>\n\n### Advanced build system\n\n\n* Built-in support for various LaTeX compilers, index, bibliography and glossary tools, Latexmk, and many more\n* Automatic detection of the need for multiple LaTeX runs\n* Run any program you like\n* Completely customizable for the creation of the complete document\n\n<p align="center"><img alt="Error Highlighting" src="https://texstudio.org/images/errorHighlighting_zoom.png"> </p>\n\n### View The Result\n\n\n* Integrated PDF viewer with (almost) word-level syncing\n* Live-updating inline preview for formulas and code segments\n* Tooltip preview for included images\n\n\n### Easy Setup\n\n\n* Installers or packages available for Windows, Linux and Mac OS X\n* Portable USB version available\n* Automatic detection of MikTeX, TeX Live, Ghostscript and Standardlatex\n* SVN support\n\n## Participate\n\nAs an open source project TeXstudio strongly benefits from an active community.\n\n### Use and spread the word\n \nThe easiest way to help on the development with TeXstudio is to use it! Furthermore, if you like TeXstudio, tell all your friends and colleagues about it.\n\nIn particular we are looking for people who want to try the most recent features before they are officially released. If you are interested download and subscribe to [release candidates and development snapshots](https://texstudio.org/#snapshots).\n\n### Give Feedback\n\nUser feedback is highly welcome. Depending on the type of feedback you may use different channels to communicate.\n\n*  Feature requests,bug reports and questions can be placed in the [issue tracker](https://github.com/texstudio-org/texstudio/issues). \nPlease set a label whether this is a feature request/enhancement, a question or a bug report. \nIn case of a bug report, please state version and operation system and a description which allows to reproduce the issue on another computer.\n\n### Contribute\n\nYou do not necessarily have to be a programmer to help and make TeXstudio even better. There are many ways to contribute.\n\n* Write cwl-files for your favorite package.\n* [Translate](https://www.transifex.com/texstudio/texstudio/) TeXstudio into a new language or update an existing translation.\n* Design icons.\n* Help on improving our website.\n* Update or extend the [user manual](http://texstudio.sourceforge.net/manual/current/usermanual_en.html) or [the wiki](https://github.com/texstudio-org/texstudio/wiki).\n* Write tutorials or create screencasts.\n* Package TeXstudio for your favorite distribution or platform.\n* TeXstudio is written in C++/Qt. Programmers are welcome to implement new features or fix bugs. Code development take place on github. Pull-requests, direct patches or changed files are welcome.\n\nIt is a good idea to announce your plans on the texstudio issue list. So everybody knows what\'s going on around TeXstudio and there is no duplicate work. For questions how you can contribute contact one of the developers.\n\n\n[TeXstudio - A LaTeX Editor Web Site](http://www.texstudio.org)\n'
