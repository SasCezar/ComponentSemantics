content.\n\n\n## Contributing Guide\n\n### Install\n\n```shell\nyarn\n```\n\nThe installation failed in `Windows`, can try this command:\n\n```shell\nyarn --ignore-platform\n```\n\n### Dev Tools Extension\n\n```shell\ncp ./config/devconfig.example.json ./config/devconfig.json\n```\n\nEdit `./config/devconfig.json`, add Chrome extensions, for example:\n\n```json\n{\n    "extensions": [\n        "/Users/xxx/Library/Application Support/Google/Chrome/Default/Extensions/lmhkpmbekcpmknklioeibfkpmmfibljd/2.15.3_0",\n        "/Users/xxx/Library/Application Support/Google/Chrome/Default/Extensions/fmkadmapgofadopljbjfkapdkoienihi/3.4.0_0"\n    ]\n}\n```\n\n### Run Main Process\n\n```shell\nnpm run dev:main\n```\n\n### Run Renderer Process\n\n```shell\nnpm run dev:renderer\n```\n\n### Build\n\n```shell\nnpm run build:all|main|renderer\n```\n\n### Package\n\n```\nnpm run packager:mac|win|win:64|linux\n```\n\n### Thanks\n\nThanks
