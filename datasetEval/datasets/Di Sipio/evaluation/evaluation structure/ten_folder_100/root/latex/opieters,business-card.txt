b'Business Card\n=============\n\nExample:\n\n<div>\n    <img src="images/front.png" alt-="front side business card" width="200px"/>\n    <img src="images/back.png" alt-="back side business card" width="200px"/>\n</div>\n\nHow this business card was designed, is explained in [this blog post](https://olivierpieters.be/blog/2017/02/11/designing-a-business-card-in-latex).\n\nRequirements\n------------\n\n* Recent TeX installation (tested on a 2017 one)\n* XeLaTeX\n* [Font Awesome](https://github.com/xdanaux/fontawesome-latex)\n* [Fira Sans](https://github.com/mozilla/Fira)\n\nIt is also possible to use [this Docker container](https://hub.docker.com/r/accupara/business-cards/):\n\n```shell\ndocker run \\\n    --rm -it \\\n    -v `pwd`:/tmp/src accupara/business-cards \\\n    /bin/bash -c \'cd /tmp/src/src ; xelatex front.tex;\'\n```\n\nBuilding Documents\n------------------\n\nBuild the front and back sides with XeLaTeX:\n\n```shell\nxelatex src/front.tex\nxelatex src/back.tex\n```\n\nSVG Files\n---------\n\nLaTeX is not equipped to handle SVG files directly. A conversion to a PDF file is needed. This can be done using an external tool such as Inkscape:\n\n```shell\ninkscape --without-gui --export-area-drawing --file=logo.svg --export-pdf=logo.pdf\n```\n\nLicense\n-------\n\nGNU GPLv3.
