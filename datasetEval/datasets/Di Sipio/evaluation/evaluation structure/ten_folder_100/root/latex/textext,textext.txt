b'Status master: |Build Status Master|, Status develop: |Build Status Develop|\n\nTexText - A TeX extension for Inkscape\n======================================\n\nTexText is a Python plugin for the vector graphics editor\n`Inkscape <http://www.inkscape.org/>`__ providing the possibility to add\nand re-edit LaTeX generated SVG elements to your drawing.\n\n.. figure:: docs/source/images/textext-with-inkscape.png\n    :alt: TexText dialog with Inkscape\n\nKey features\n------------\n\n-  Windows/ Linux/ MacOS support\n-  LaTeX generated SVG elements can be re-edited later\n-  Multi-line editor with syntax highlighting (see `\xc2\xa7 <usage-dialog-overview_>`_)\n-  Compilation with **PdfLaTeX**, **XeLaTeX** or **LuaLaTex** (see `\xc2\xa7 <usage-tex-compilers_>`_)\n-  Interoperable scaling in TexText and Inkscape (see `\xc2\xa7 <usage-scaling_>`_)\n-  Font size match with Inkscape text (see `\xc2\xa7 <usage-font_>`_)\n-  Customizable TeX preamble (additional packages, parskip, parindent, etc.) (see `\xc2\xa7 <usage-preamble-file_>`_)\n-  Colorization via TeX commands/Inkscape is kept after re-editing (see `\xc2\xa7 <usage-colorization_>`_)\n-  Alignment anchor of the produced output (see `\xc2\xa7 <usage-alignment_>`_)\n-  Preview images  (see `\xc2\xa7 <usage-preview_>`_)\n-  Compatibility with TexText down to version 0.4.x\n\nDownload\n--------\n\nDownload of the most recent version (and older ones): https://github.com/textext/textext/releases\n\n\nDocumentation\n-------------\n\nDocumentation hosted at https://textext.github.io/textext\nIt contains `installation <installation-toc_>`_ and `usage <usage-toc_>`_ instructions\n\n\nHistory\n-------\n\nThis repository continues the development of the plugin which took place\nat https://bitbucket.org/pitgarbe/textext until January 2018.\nOriginally, TexText had been developed by `Pauli\nVirtanen <http://www.iki.fi/pav/software/textext/>`__ based on the\nplugin InkLaTeX written by Toru Araki.\n\n.. |Build Status Master| image:: https://travis-ci.com/textext/textext.svg?branch=master\n   :target: https://travis-ci.com/textext/textext\n\n.. |Build
