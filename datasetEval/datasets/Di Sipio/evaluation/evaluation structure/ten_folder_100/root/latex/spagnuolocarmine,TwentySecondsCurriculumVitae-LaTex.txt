b'Twenty Seconds Curriculum Vitae in LaTex\n========================================\n\n[![Donate](https://img.shields.io/badge/PayPal-Donate%20to%20Author-blue.svg)](https://www.paypal.me/CarmineSpagnuolo)\n\n# Curricula Vitae - R\xc3\xa9sum\xc3\xa9s\nA curriculum vitae, otherwise known as a CV or r\xc3\xa9sum\xc3\xa9, is a document used by individuals to communicate their work history, education and skill set. This is a style template for your curriculum written in LaTex. The main goal of this template is to provide a curriculum that is able to survive to the r\xc3\xa9sum\xc3\xa9s screening of "twenty seconds".\n\n_The author assumes no responsibility for the topicality, correctness, completeness or quality of the information provided and for the obtained r\xc3\xa9sum\xc3\xa9s._\n\nThis is designed for computer scientists but there is no limitation to use it for r\xc3\xa9sum\xc3\xa9s in other disciplines.\n\n### If you like this curriculum, please don\'t forget to leave a star, to help the development and improvement.\n\n## The basic idea is KISS - Keep It Simple, Stupid.\n\nIn a nutshell _**"It is vain to do with more what can be done with fewer"**_ -- Occam\'s razor --\n\n* **This template has been designed to create a "one-page" r\xc3\xa9sum\xc3\xa9 is therefore not suitable to create curriculum of more than one-page.** \n\n* _Please do not try to create curriculum more than one-page._ \n\n### How to describe your experiences?\n\nThere are many theories about the r\xc3\xa9sum\xc3\xa9 screening process of "Big" companies.\nResume screeners and the interviewers look in your r\xc3\xa9sum\xc3\xa9 for:\n\n- Are you smart?\n- Can you code (act for what you apply)?\n\nAnyway according to the guidelines of this template you should use a really simple form to describe each items in your r\xc3\xa9sum\xc3\xa9: \n\n\tAccomplished <X> by implementing <Y> which led to <Z>\n\nHere\'s an examples:\n\t\n\tReduced object rendering time by 75% by applying Floyd\'s algorithm, leading to a 10% reduction in system boot time.\n\t\n-- _Cracking the Coding Interview, Book, Gayle Laakmann Mcdowell_ --\n\n## Toy R\xc3\xa9sum\xc3\xa9\n***\n![sample r\xc3\xa9sum\xc3\xa9](https://github.com/spagnuolocarmine/TwentySecondsCurriculumVitae-LaTex/raw/master/Twenty-Seconds_cv.jpg)\n***\n\n# Build \nThis guide walks you to build your r\xc3\xa9sum\xc3\xa9.\n\nBuild requirements:\n\n* LaTex installation.\n\t* additionals packages:\t \t\n\t\t- ClearSans, fontenc\n\t\t- tikz\n\t\t- xcolor\n\t\t- textpos\n\t\t- ragged2e\n\t\t- etoolbox\n\t\t- ifmtarg\n\t\t- ifthen\n\t\t- pgffor\n\t\t- marvosym\n\t\t- parskip\n\n### Build through GNU Make command\nClean your project r\xc3\xa9sum\xc3\xa9.\n\t\n\tmake clean\n\t\nBuild your project r\xc3\xa9sum\xc3\xa9.\n\n\tmake all\n\t\n-- _Alternately you can build through your favorite LaTex editor._ --\n\n# Environment style and list of commands\n\nThe style is divided in two parts. The former is the left side bar: that contains personal information, profile picture, and information about your professional skills. The second part is the body that should be contains details about your academic studies, professional experiences and all the information that you want (remember the KISS principle).\n\n### Profile environment\nThese are the command to set up the profile information.\n\n* Set up the image profile.\n\t\n\t\t\\profilepic{paht_name}\n* Set up your name.\n\t\n\t\t\\cvname{your name}\n* Set up your job profile.\n\t\n\t\t\\cvjobtitle{your job title}\n* Set up your date of birth.\n\t\n\t\t\\cvdate{date}\t\n* Set up your address.\n\t\n\t\t\\cvaddress{address}\t\t\n* Set up your telephone number.\n\t\n\t\t\\cvnumberphone{phone number}\n* Set up your email.\n\t\n\t\t\\cvmail{email address}\n* Set up your personal home page.\n\t\n\t\t\\cvsite{home page address}\n* Set up a brief description of you.\n\t\n\t\t\\about{brief description}\n* Set up the skills with chart style. Each skill must is a couple `{name/value}`, where the value is a floating point value between `0` and `6`. This is an agreement for the graphics issues, the `0` correspond to a Fundamental awareness while `6` to a Expert awareness level.\n\t\n\t\t\\skills{{name skill1/5.8},{name skill2/4}} \n* Set up the skills with text style.\n\t\n\t\t\\skillstext{{name skill1/5.8},{name skill2/4}} \n\nTo create the profile use the command:\n\n\t\\makeprofile\n\n### Body environment\nThe body document part is composed by sections.\nIn the sections you can put two kinds of list items.\n\nThe first (_Twenty items environment_) intends a list of detailed information with four part: **Data** -- **Title** -- **Place** -- **Description**. \n\nThe second (_Twenty items short environment_) intends a fewer informationinformation (you can customize this list more easily): **Data** -- **Description**.\n#### Sections\n* Set up a new section in the body part.\n\t\t\n\t\t\\section{sction name}\n\n\n#### Twenty items environment\n```\n\\begin{twenty}\n  \\twentyitem\n    {year}\n    {title}\n    {place}\n    {description}\n\\end{twenty}\n```\n\n#### Twenty items short environment\n```\n\\begin{twentyshort}\n  \\twentyitemshort\n    {year}\n    {description}\n\\end{twentyshort}\n```\n### Other commands\nThere other two fun command: \\icon and \\round; that enables to wrap the text in oval shape.\n\n```\n\t\\icon{text}\n\t\\round{text, color}\n```\n'
