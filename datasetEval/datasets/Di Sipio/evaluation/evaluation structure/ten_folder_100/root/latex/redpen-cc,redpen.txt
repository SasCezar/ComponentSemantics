RedPen automates the verifications of input documents written in natural languages\n\n\n<p align="center">\n   <a href="http://redpen.cc"><img src="logo/redpen-logo.png"/></a>\n</p>\n\nSee the detail usage in the RedPen [home page](http://redpen.cc/).\n\nFeatures\n=========\n\n- Language independent\n- Highly customizable\n- Multi platform (MacOS, Windows, and Linux)\n- Support various markup text formats (Wiki, Markdown, AsciiDoc, LaTeX, Re:VIEW, reStructuredText)\n- [Plugin system: users can write their extensions with JavaScript](http://blog.redpen.cc/2015/09/08/writing-extension-with-javascript/)\n- [Annotation based error suppression](https://blog.redpen.cc/2016/06/14/introduction-of-annotation-for-suppressing-errors-from-document-checking-tool-redpen/)\n- [Practical REST API and UI](http://redpen.herokuapp.com/)\n\n<p
