man page will be at neph.1, binary will be at bin/neph\n```\nIf you don\'t have a previous version of `neph` installed:\n```bash\n$ git clone https://github.com/tbrand/neph\n$ cd neph\n$ shards build    # Now executable binary is located at bin/neph\n$ bin/neph man    # Generate man page. It will be located at neph.1\n```\n\n## Usage\n\nAll features for neph.yaml is written in [sample/neph.yaml](https://github.com/tbrand/neph/blob/master/sample/neph.yaml).
