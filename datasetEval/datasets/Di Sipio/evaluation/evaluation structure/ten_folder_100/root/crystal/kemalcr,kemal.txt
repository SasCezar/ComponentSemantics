b'\n[![Kemal](https://avatars3.githubusercontent.com/u/15321198?v=3&s=200)](http://kemalcr.com)\n\n# Kemal\n\nLightning Fast, Super Simple web framework.\n\n[![Build Status](https://travis-ci.org/kemalcr/kemal.svg?branch=master)](https://travis-ci.org/kemalcr/kemal)\n[![Join the chat at https://gitter.im/sdogruyol/kemal](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/sdogruyol/kemal?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)\n\n# Super Simple \xe2\x9a\xa1\xef\xb8\x8f\n\n```ruby\nrequire "kemal"\n\n# Matches GET "http://host:port/"\nget "/" do\n  "Hello World!"\nend\n\n# Creates a WebSocket handler.\n# Matches "ws://host:port/socket"\nws "/socket" do |socket|\n  socket.send "Hello from Kemal!"\nend\n\nKemal.run\n```\n\nStart your application!\n\n```\ncrystal src/kemal_sample.cr\n```\nGo to *http://localhost:3000*\n\nCheck [documentation](http://kemalcr.com) or [samples](https://github.com/kemalcr/kemal/tree/master/samples) for more.\n\n# Installation\n\nAdd this to your application\'s `shard.yml`:\n\n```yaml\ndependencies:\n  kemal:\n    github: kemalcr/kemal\n```\n\nSee also [Getting Started](http://kemalcr.com/guide/).\n\n# Features\n\n- Support all REST verbs\n- Websocket support\n- Request/Response context, easy parameter handling\n- Middleware support\n- Built-in JSON support\n- Built-in static file serving\n- Built-in view templating via [Kilt](https://github.com/jeromegn/kilt)\n\n# Documentation\n\nYou can read the documentation at the official site [kemalcr.com](http://kemalcr.com)\n\n## Thanks\n\nThanks to Manas for their awesome work on [Frank](https://github.com/manastech/frank).\n'
