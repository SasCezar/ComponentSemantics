b'# Hedron\n\nHedron is an easy-to-use, OOP Crystal GUI library, that allows you to do the following:\n\n- Create your own UI classes ("widgets") that fully integrate with existing classes\n- A markup language (with extension `.hdml`), which makes creating UI easy\n- Full integration with custom widgets and HDML, so you can inject your own widgets into your markup\n\nHedron is under active development, so check back often!\n\n## Installation\n\n1. Follow the instructions for installation laid out in [andlabs/libui](https://github.com/andlabs/libui).\n2. Copy the compiled files from step 1 (i.e. files in `build/out`) to `/usr/lib` for OSX and Linux users.\n3. Go to your `shard.yml` file, and enter this in:\n\n```yaml\ndependencies:\n  hedron:\n    github: hedron-crystal/hedron\n    version: 0.1.0\n```\n\n## Acknowledgement\n\n- [andlabs/libui](https://github.com/andlabs/libui), for providing the basic UI elements\n- [Fusion/libui.cr](https://github.com/Fusion/libui.cr), for originally providing the libui bindings\n\n## Contributors\n\n- [Qwerp-Derp](https://github.com/Qwerp-Derp) Hanyuan Li - creator, maintainer\n'
