b'# ruby-install\n\n[![Build Status](https://travis-ci.org/postmodern/ruby-install.svg?branch=master)](https://travis-ci.org/postmodern/ruby-install)\n\nInstalls [Ruby], [JRuby], [Rubinius], [TruffleRuby] or [mruby].\n\n## Features\n\n* Supports installing arbitrary versions.\n* Supports downloading the latest versions and checksums from [ruby-versions].\n* Supports installing into `/opt/rubies/` for root and `~/.rubies/` for users\n  by default.\n* Supports installing into arbitrary directories.\n* Supports downloading from arbitrary URLs.\n* Supports downloading from mirrors.\n* Supports downloading/applying patches.\n* Supports specifying arbitrary `./configure` options.\n* Supports downloading archives using `wget` or `curl`.\n* Supports verifying downloaded archives using `md5sum`, `md5` or `openssl md5`.\n* Supports installing build dependencies via the package manager:\n  * [apt]\n  * [dnf]\n  * [yum]\n  * [pacman]\n  * [zypper]\n  * [pkg]\n  * [macports]\n  * [brew]\n* Has tests.\n\n## Anti-Features\n\n* Does not require updating every time a new Ruby version comes out.\n* Does not require recipes for each individual Ruby version or configuration.\n* Does not support installing trunk/HEAD.\n\n## Requirements\n\n* [bash] >= 3.x\n* [wget] > 1.12 or [curl]\n* `md5sum`, `md5` or `openssl md5`.\n* `tar`\n* `bzip2`\n* `patch` (if `--patch` is specified)\n* [gcc] >= 4.2 or [clang]\n\n## Synopsis\n\nList supported Rubies and their major versions:\n\n    $ ruby-install\n\nList the latest versions:\n\n    $ ruby-install --latest\n\nInstall the current stable version of Ruby:\n\n    $ ruby-install ruby\n\nInstall the latest version of Ruby:\n\n    $ ruby-install --latest ruby\n\nInstall a stable version of Ruby:\n\n    $ ruby-install ruby 2.3\n\nInstall a specific version of Ruby:\n\n    $ ruby-install ruby 2.2.4\n\nInstall a Ruby into a specific directory:\n\n    $ ruby-install --install-dir /path/to/dir ruby\n\nInstall a Ruby into a specific `rubies` directory:\n\n    $ ruby-install --rubies-dir /path/to/rubies/ ruby\n\nInstall a Ruby into `/usr/local`:\n\n    $ ruby-install --system ruby 2.4.0\n\nInstall a Ruby from an official site with directly download:\n\n    $ ruby-install -M https://ftp.ruby-lang.org/pub/ruby ruby 2.4.0\n\nInstall a Ruby from a mirror:\n\n    $ ruby-install -M http://www.mirrorservice.org/sites/ftp.ruby-lang.org/pub/ruby ruby 2.0.0-p645\n\nInstall a Ruby with a specific patch:\n\n    $ ruby-install -p https://raw.github.com/gist/4136373/falcon-gc.diff ruby 1.9.3-p551\n\nInstall a Ruby with a specific C compiler:\n\n    $ ruby-install ruby 2.4.0 -- CC=gcc-4.9\n\nInstall a Ruby with specific configuration:\n\n    $ ruby-install ruby 2.4.0 -- --enable-shared --enable-dtrace CFLAGS="-O3"\n\nInstall a Ruby without installing dependencies first:\n\n    $ ruby-install --no-install-deps ruby 2.4.0\n\nUninstall a Ruby version:\n\n    $ rm -rf ~/.rubies/ruby-2.4.0\n\n### Integration\n\nUsing ruby-install with [RVM]:\n\n    $ ruby-install --rubies-dir ~/.rvm/rubies ruby 2.4.0\n\nUsing ruby-install with [rbenv]:\n\n    $ ruby-install --install-dir ~/.rbenv/versions/2.4.0 ruby 2.4.0\n\nruby-install can even be used with [Chef].\n\n## Install\n\n    wget -O ruby-install-0.7.0.tar.gz https://github.com/postmodern/ruby-install/archive/v0.7.0.tar.gz\n    tar -xzvf ruby-install-0.7.0.tar.gz\n    cd ruby-install-0.7.0/\n    sudo make install\n\n### PGP\n\nAll releases are [PGP] signed for security. Instructions on how to import my\nPGP key can be found on my [blog][1]. To verify that a release was not tampered\nwith:\n\n    wget https://raw.github.com/postmodern/ruby-install/master/pkg/ruby-install-0.7.0.tar.gz.asc\n    gpg --verify ruby-install-0.7.0.tar.gz.asc ruby-install-0.7.0.tar.gz\n\n### Homebrew\n\nruby-install can also be installed with [homebrew]:\n\n    brew install ruby-install\n\nOr the absolute latest ruby-install can be installed from source:\n\n    brew install ruby-install --HEAD\n\n### Arch Linux\n\nruby-install is already included in the [AUR]:\n\n    yaourt -S ruby-install\n\n### Fedora Linux\n\nruby-install is available on [Fedora Copr](https://copr.fedorainfracloud.org/coprs/postmodern/ruby-install/).\n\n### FreeBSD\n\nThere is a [FreeBSD port] of ruby-install which can be copied into the local\nports tree.\n\n## Known Issues\n\n### Ruby\n\nRubies older than 1.9.3-p429 will not compile with [Clang][clang] and require\n[GCC][gcc] >= 4.2. Normally, Linux and BSD systems will already have GCC\ninstalled. OS X users can install GCC via [homebrew]:\n\n    brew tap homebrew/versions\n    brew install gcc49\n\nAnd run ruby-install again:\n\n    ruby-install ruby 2.4.0 -- CC=gcc-4.9\n\n### Rubinius\n\n[Rubinius] 1.x and 2.0.x is not supported due to multiple bugs.\nInstead, please use Rubinius >= 2.1.x.\n\n## Alternatives\n\n* [RVM]\n* [ruby-build]\n\n## Endorsements\n\n> I like the approach you\'re taking.
