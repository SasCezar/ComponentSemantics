b'# Dotfiles\nColourful & robust configuration files and utilities for Mac, Linux and BSD. Installation is done with a simple command:\n\n```sh\ncurl -L https://git.io/pmdot | sh\n```\n\nThe short URL expands to GitHub-hosted `install.sh`, which can be easily audited.\n\n## Usage\n\n- **MacOS:** Ensure you have XCode or dev tools. It can be downloaded from the app store.\n    - Optionally `sh etc/bootstrap-macos.sh`\n- **Linux and BSD:** Ensure you have `git` and `zsh` installed.\n- **MacOS terminal:** `terminal/monokai_pro_spectrum.terminal` (Settings -> Profiles -> Press "gear" -> Import).\n- **Git:** Don\'t forget to adjust `home/.gitconfig` or you\'ll have improper commit author\n\n## Features\n\n![](https://user-images.githubusercontent.com/574696/61765243-eb19dc00-ade4-11e9-8d16-5a402a0fdfec.png)\n![](https://user-images.githubusercontent.com/574696/61765242-eb19dc00-ade4-11e9-8db0-ac607e1eed8a.png)\n\n* **No external dependencies!** Great, when compared to oh-my-zsh.\n* Auto-completion\n* Syntax highlighting\n* Useful utilities:\n    * `ff file-name-or-pattern` - fast recursive search for a file name in directories.\n    * `tarbz2`, `untarbz2` - best archive compression. Utilizes parallel `pbzip2` when available.\n    * `extract archive.tar.bz` \xe2\x80\x94 unpack any archive (supports many extensions)\n    * `ram safari` \xe2\x80\x94 show app RAM usage\n    * `loc py coffee js html css` \xe2\x80\x94 count lines of code\n    * `curl http://site/v1/api.json | json` - pretty-print JSON\n* `git-extras` - useful git functions, defined in `home/.gitconfig`:\n    * Opinionated `git log`, `git graph`\n    * `gcp` for fast `git commit -m ... && git push`\n    * `git sign` for PGP-signed git\n    * `git cleanup` \xe2\x80\x94 clean up merged git branches. Very useful if\n    you\xe2\x80\x99re doing github pull requests in topic branches.\n    * `git summary` \xe2\x80\x94 outputs commit email statistics.\n    * `git release` and `git sign-release` \xe2\x80\x94 commit and tag the commit. Publishes to NPM for node projects.\n    * `git url` - opens GitHub repo for current git repo.\n    * `git-changelog`, `git-setup` etc.\n* `etc` \xe2\x80\x94 MacOS fine tuning\n* `vscode` \xe2\x80\x94 Sublime Text theme & settings\n* Sets terminal tab and window title to current directory\n* [homesick](https://github.com/technicalpickles/homesick) /\n  [homeshick](https://github.com/andsens/homeshick)-compatible\n\n## Not included\n\n- `diff-so-fancy`: solid alternative to `git diff`\n\n## License\n\nThe MIT License (MIT)\n\nCopyright (c) 2019 Paul Miller [(paulmillr.com)](https://paulmillr.com)\n\nPermission is hereby granted, free of charge, to any person obtaining a copy\nof this software and associated documentation files (the \xe2\x80\x9cSoftware\xe2\x80\x9d), to deal\nin the Software without restriction, including without limitation the rights\nto use, copy, modify, merge, publish, distribute, sublicense, and/or sell\ncopies of the Software, and to permit persons to whom the Software is\nfurnished to do so, subject to the following conditions:\n\nThe above copyright notice and this permission notice shall be included in\nall copies or substantial portions of the Software.\n\nTHE SOFTWARE IS PROVIDED \xe2\x80\x9cAS IS\xe2\x80\x9d, WITHOUT WARRANTY OF ANY KIND, EXPRESS OR\nIMPLIED, INCLUDING BUT NOT LIMITED TO THE WARRANTIES OF MERCHANTABILITY,\nFITNESS FOR A PARTICULAR PURPOSE AND NONINFRINGEMENT.
