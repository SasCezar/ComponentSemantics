b"# Hanami :cherry_blossom:\n\nThe web, with simplicity.\n\n## Frameworks\n\nHanami is a **full-stack** Ruby web framework.\nIt's made up of smaller, single-purpose libraries.\n\nThis repository is for the full-stack framework,\nwhich provides the glue that ties all the parts together:\n\n* [**Hanami::Model**](https://github.com/hanami/model) - Persistence with entities, repositories and data mapper\n* [**Hanami::View**](https://github.com/hanami/view) - Presentation with a separation between views and templates\n* [**Hanami::Controller**](https://github.com/hanami/controller) - Full featured, fast and testable actions for Rack\n* [**Hanami::Validations**](https://github.com/hanami/validations) - Validations mixin for Ruby objects\n* [**Hanami::Router**](https://github.com/hanami/router) - Rack compatible HTTP router for Ruby\n* [**Hanami::Helpers**](https://github.com/hanami/helpers) - View helpers for Ruby applications\n* [**Hanami::Mailer**](https://github.com/hanami/mailer) - Mail for Ruby applications\n* [**Hanami::Assets**](https://github.com/hanami/assets) - Assets management for Ruby\n* [**Hanami::CLI**](https://github.com/hanami/cli) - Ruby command line interface\n* [**Hanami::Utils**](https://github.com/hanami/utils) - Ruby core extensions and class utilities\n\nThese components are designed to be used independently or together in a Hanami application.\n\n## Status\n\n[![Gem Version](https://badge.fury.io/rb/hanami.svg)](https://badge.fury.io/rb/hanami)\n[![TravisCI](https://travis-ci.org/hanami/hanami.svg?branch=master)](https://travis-ci.org/hanami/hanami)\n[![Test Docs](http://inch-ci.org/github/hanami/hanami.svg)](http://inch-ci.org/github/hanami/hanami)\n\n## Installation\n\n__Hanami__ supports Ruby (MRI) 2.3+\n\n```shell\ngem install hanami\n```\n\n##
