b'<p align="center">\n  <img src="https://github.com/bitwarden/brand/blob/master/screenshots/apps-combo-logo.png" alt="Bitwarden" />\n</p>\n<p align="center">\n  <a href="https://ci.appveyor.com/project/bitwarden/server/branch/master" target="_blank">\n    <img src="https://ci.appveyor.com/api/projects/status/n4ntjp7ag1uwxckd/branch/master?svg=true" alt="appveyor build" />\n  </a>\n  <a href="https://hub.docker.com/u/bitwarden/" target="_blank">\n    <img src="https://img.shields.io/docker/pulls/bitwarden/api.svg" alt="DockerHub" />\n  </a>\n  <a href="https://gitter.im/bitwarden/Lobby" target="_blank">\n    <img src="https://badges.gitter.im/bitwarden/Lobby.svg" alt="gitter chat" />\n  </a>\n</p>\n\n-------------------\n\nThe Bitwarden Server project contains the APIs, database, and other core infrastructure items needed for the "backend" of all bitwarden client applications.\n\nThe server project is written in C# using .NET Core with ASP.NET Core. The database is written in T-SQL/SQL Server. The codebase can be developed, built, run, and deployed cross-platform on Windows, macOS, and Linux distributions.\n\n## Build/Run\n\n### Requirements\n\n- [.NET Core 2.x SDK](https://www.microsoft.com/net/download/core)\n- [SQL Server 2017](https://docs.microsoft.com/en-us/sql/index)\n\n*These dependencies are free to use.*\n\n### Recommended Development Tooling\n\n- [Visual Studio](https://www.visualstudio.com/vs/) (Windows and macOS)\n- [Visual Studio Code](https://code.visualstudio.com/) (other)\n\n*These tools are free to use.*\n\n### API\n\n```\ncd src/Api\ndotnet restore\ndotnet build\ndotnet run\n```\n\nvisit http://localhost:5000/alive\n\n### Identity\n\n```\ncd src/Identity\ndotnet restore\ndotnet build\ndotnet run\n```\n\nvisit http://localhost:33657/.well-known/openid-configuration\n\n## Deploy\n\n<p align="center">\n  <a href="https://hub.docker.com/u/bitwarden/" target="_blank">\n    <img src="https://i.imgur.com/SZc8JnH.png" alt="docker" />\n  </a>\n</p>\n\nYou can deploy Bitwarden using Docker containers on Windows, macOS, and Linux distributions. Use the provided PowerShell and Bash scripts to get started quickly. Find all of the Bitwarden images on [Docker Hub](https://hub.docker.com/u/bitwarden/).\n\nFull documentation for deploying Bitwarden with Docker can be found in our help center at: https://help.bitwarden.com/article/install-on-premise/\n\n### Requirements\n\n- [Docker](https://www.docker.com/community-edition#/download)\n- [Docker Compose](https://docs.docker.com/compose/install/) (already included with some Docker installations)\n\n*These dependencies are free to use.*\n\n### Linux & macOS\n\n```\ncurl -s -o bitwarden.sh \\\n    https://raw.githubusercontent.com/bitwarden/server/master/scripts/bitwarden.sh \\\n    && chmod +x bitwarden.sh\n./bitwarden.sh install\n./bitwarden.sh start\n```\n\n### Windows\n\n```\nInvoke-RestMethod -OutFile bitwarden.ps1 `\n    -Uri https://raw.githubusercontent.com/bitwarden/server/master/scripts/bitwarden.ps1\n.\\bitwarden.ps1 -install\n.\\bitwarden.ps1 -start\n```\n\n## Contribute\n\nCode contributions are welcome!
