b'**This project is no longer active developement, please see [CoolQLCool](https://www.github.com/dinubs/coolqlcool) for a similar more active project ([ref](https://github.com/dinubs/jam-api/issues/24))**\n\n# Jam API\nJam API is a service that allows you to turn any site into a JSON accessible api using CSS selectors. To get started simply run a post request to https://www.jamapi.xyz with formdata of "url" and "json_data", here\'s an example of what your data should look like:\n```json\n{\n  "title": "title",\n  "logo": ".nav-logo img",\n  "paragraphs": [{ "elem": ".home-post h1", "value": "text"}],\n  "links": [{"elem": ".home-post > a:first-of-type", "location": "href"}]\n}\n```\nUsing API you can simply generate JSON data from any website.\n\n## Code samples\n### nodejs\n```js\nconst request = require(\'request\');\nrequest.post(\'https://www.jamapi.xyz/\', {form: {url: \'http://www.gavin.codes/\', json_data: \'{"title": "title"}\'}}, function(err, response, body) {\n  console.log(body);\n})  \n```\n### Javascript\n```js\nfetch(\'https://www.jamapi.xyz\', {\n    method: \'POST\',\n    headers: {\n      \'Accept\': \'application/json\',\n      \'Content-Type\': \'application/json\'\n    },\n    body: JSON.stringify({\n      url: \'http://www.gavin.codes/\',\n      json_data: \'{"title": "title"}\'\n    })\n  }).then(function(response) {\n    return response.json();\n  }).then(function(json) {\n    document.body.innerHTML = json;\n  });  \n```\n### Ruby\n```ruby\nrequire \'httparty\'\nresponse = HTTParty.post("https://www.jamapi.xyz/",\n  :body => { "url" => "http://www.gavin.codes/", "json_data" => "{\'title\': \'title\'}"})  \nputs response.to_json\n```\n### Python\n```python\nimport requests\npayload = {\'url\': \'http://www.gavin.codes/\', \'json_data\': \'{"title": "title"}\'}\n\nr = requests.post("https://www.jamapi.xyz", data=payload)\nprint(r.json())\n```\n### curl\n```shell\ncurl -X POST \\\n  -F \'url=http://www.gavin.codes/\' \\\n  -F \'json_data={"title":"title"}\' \\\n  https://www.jamapi.xyz\n```\n\n## Features\nWill auto pull the img src on corresponding elements, will auto pull the href from links. If passing JSON, you must provide a "elem" property, and then the element attributes you want. When you pass an array with JSON you\'ll get a structure that looks as follows:\n```json\n[\n  {\n      "index": 0,\n      "value": {\n          "value": "Porter Robinson \xe2\x80\x93 Sad Machine (Cosmo\xe2\x80\x99s Midnight Remix)"\n      }\n  },\n  {\n      "index": 1,\n      "value": {\n          "value": "Listen to Rachel Platten\xe2\x80\x99s \xe2\x80\x9cStand By You\xe2\x80\x9d"\n      }\n  }]\n```\nAll the attributes you provide as JSON will be put inside of the value property, and the index property is to be able to track what index it ocurred in the DOM. I nested JSON values into it\'s own so that you can still have an "index" property returned and not run into issues.\n\n## How it works\nMain power of the program is in `services/html_to_json.js`. Start site with `node index` after doing `npm install`.\n\nSuggested node version is at least `4.2.2`\n'
