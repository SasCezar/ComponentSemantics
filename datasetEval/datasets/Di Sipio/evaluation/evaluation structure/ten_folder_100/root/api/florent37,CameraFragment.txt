b'# CameraFragment\n\n[![Android Arsenal](https://img.shields.io/badge/Android%20Arsenal-CameraFragment-brightgreen.svg?style=flat)](https://android-arsenal.com/details/1/5089)\n[![CircleCI](https://circleci.com/gh/florent37/CameraFragment/tree/master.svg?style=svg)](https://circleci.com/gh/florent37/CameraFragment/tree/master)\n\nA simple easy-to-integrate Camera Fragment for Android\n\nCameraFragment preview directly the camera view, and provides a easy API to capture or manage the device\n\nYou can setup your own layout and control the camera using CameraFragment\n\n<a href="https://goo.gl/WXW8Dc">\n  <img alt="Android app on Google Play" src="https://developer.android.com/images/brand/en_app_rgb_wo_45.png" />\n</a>\n\n# CameraKit\n\nThis library works, but on some devices... errors happens, I don\'t have time to maintain compatibility with all device\nA community group was created, they created CameraKit, don\'t hesitate to try it !\n\nhttps://github.com/CameraKit/camerakit-android\n\n# Setup\n\n[![png](https://raw.githubusercontent.com/florent37/CameraFragment/master/media/sample.png)](https://github.com/florent37/CameraFragment)\n\n```java\n//you can configure the fragment by the configuration builder\nCameraFragment cameraFragment = CameraFragment.newInstance(new Configuration.Builder().build());\n\ngetSupportFragmentManager().beginTransaction()\n                .replace(R.id.content, cameraFragment, FRAGMENT_TAG)\n                .commit();\n```\n\n## Actions\n\nYou can directly take a photo / video with\n```java\ncameraFragment.takePhotoOrCaptureVideo(callback);\ncameraFragment.takePhotoOrCaptureVideo(callback, directoryPath, fileName);\n```\n\n[![gif](https://raw.githubusercontent.com/florent37/CameraFragment/master/media/take_photo.gif)](https://github.com/florent37/CameraFragment)\n\nFlash can be enable / disabled ( `AUTO` / `OFF` / `ON` ) with\n\n```java\ncameraFragment.toggleFlashMode();\n```\n\n[![gif](https://raw.githubusercontent.com/florent37/CameraFragment/master/media/switch_flash.gif)](https://github.com/florent37/CameraFragment)\n\nCamera Type can be modified ( `BACK` / `FRONT` ) with\n\n```java\ncameraFragment.switchCameraTypeFrontBack();\n```\n\n[![gif](https://raw.githubusercontent.com/florent37/CameraFragment/master/media/switch_camera.gif)](https://github.com/florent37/CameraFragment)\n\nCamera action ( `PHOTO` / `VIDEO` ) can be modified with\n\n```java\ncameraFragment.switchActionPhotoVideo();\n```\n\n[![gif](https://raw.githubusercontent.com/florent37/CameraFragment/master/media/switch_action.gif)](https://github.com/florent37/CameraFragment)\n\nAnd you can change the captured photo / video size with\n\n```java\ncameraFragment.openSettingDialog();\n```\n\n[![gif](https://raw.githubusercontent.com/florent37/CameraFragment/master/media/settings.gif)](https://github.com/florent37/CameraFragment)\n\n# Listeners\n\n## Result\n\nGet back the result of the camera record / photo in the `CameraFragmentResultListener`\n\n```java\ncameraFragment.setResultListener(new CameraFragmentResultListener() {\n       @Override\n       public void onVideoRecorded(byte[] bytes, String filePath) {\n                //called when the video record is finished and saved\n\n                startActivityForResult(PreviewActivity.newIntentVideo(MainActivity.this, filePath));\n       }\n\n       @Override\n       public void onPhotoTaken(byte[] bytes, String filePath) {\n                //called when the photo is taken and saved\n\n                startActivity(PreviewActivity.newIntentPhoto(MainActivity.this, filePath));\n       }\n});\n```\n\n## Camera Listener\n\n```java\ncameraFragment.setStateListener(new CameraFragmentStateListener() {\n\n    //when the current displayed camera is the back\n    void onCurrentCameraBack();\n    //when the current displayed camera is the front\n    void onCurrentCameraFront();\n\n    //when the flash is at mode auto\n    void onFlashAuto();\n    //when the flash is at on\n    void onFlashOn();\n    //when the flash is off\n    void onFlashOff();\n\n    //if the camera is ready to take a photo\n    void onCameraSetupForPhoto();\n    //if the camera is ready to take a video\n    void onCameraSetupForVideo();\n\n    //when the camera state is "ready to record a video"\n    void onRecordStateVideoReadyForRecord();\n    //when the camera state is "recording a video"\n    void onRecordStateVideoInProgress();\n    //when the camera state is "ready to take a photo"\n    void onRecordStatePhoto();\n\n    //after the rotation of the screen / camera\n    void shouldRotateControls(int degrees);\n\n    void onStartVideoRecord(File outputFile);\n    void onStopVideoRecord();\n});\n```\n\n## Text\n\nCameraFragment can ping you with the current record duration with `CameraFragmentTextListener`\n\n# Widgets\n\nCameraFragment comes with some default views\n\n`RecordButton`, `MediaActionSwitchView`, `FlashSwitchView`, `CameraSwitchView`, `CameraSettingsView`\n\n[![png](https://raw.githubusercontent.com/florent37/CameraFragment/master/media/buttons.png)](https://github.com/florent37/CameraFragment)\n\n# Download\n\n<a
