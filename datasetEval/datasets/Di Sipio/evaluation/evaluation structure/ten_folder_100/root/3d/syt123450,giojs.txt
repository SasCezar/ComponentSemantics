What makes Gio.js different is that it is **simple** to use Gio.js to **customize** a 3D data visualization model in a declarative way, add your own data, and integrate it into your own **modern** web application.\n\nThis library is inspired by the <a href="https://github.com/dataarts/armsglobe">Arms Trade Visualization project</a> developed by Michael Chang and presented during Google Ideas INFO 2012. See <a href="http://mflux.tumblr.com/post/28367579774/armstradeviz">original post</a>. With Gio.js, it is easy to reproduce this fantastic data visualization model, and integrate it into web application.\n\n* **Simplicity** -- Create a 3D global data visualization model with your own data in just four lines of javascript code\n* **Customization** -- Design your own globe style with easy to use APIs in Gio.js\n* **Modernization** -- Build a responsible, fully interactive, rich 3D frontend application using Gio.js\n\n<!-- [END motivation] -->\n\n<!-- [START getstarted] -->\n## Getting Started\n\n### Installation\n- Option 1: \\<script\\> tag\n\nInclude Three.js dependency:\n```html\n<script src="three.min.js"></script>\n```\nInclude local Gio.js library\n```html\n<script src="gio.min.js"></script>\n```\nor through CDN\n```html\n<script src="https://threejs.org/build/three.min.js"></script>\n<script src="https://cdn.jsdelivr.net/npm/giojs@2.2.1/build/gio.min.js"></script>\n```\n- Option 2: npm\n```bash\nnpm install giojs --save\n```\n- Option 3: yarn\n```bash\nyarn add giojs\n```\n### Usage\n\nAfter including "three.min.js" and "gio.min.js" in your html, create a `div` to render the 3D Gio globe: (the usage of Gio.js is a little bit different in WeChat mini game, checkout this [demo](https://github.com/syt123450/giojs-wechat-minigame-demo) to know more about it)\n\n```html\n<!DOCTYPE HTML>\n<html>\n<head>\n\n  <!-- include three.min.js library-->\n  <script src="three.min.js"></script>\n\n  <!-- include gio.min.js library-->\n  <script src="gio.min.js"></script>\n\n</head>\n<body>\n\n  <!-- container to draw 3D Gio globe-->\n  <div id="globalArea"></div>\n\n</body>\n</html>\n\n```\nTo initialize and render the 3D Gio globe:\n\n```html\n\n<script>\n\n    // get the container to hold the IO globe\n    var container = document.getElementById( "globalArea" );\n\n    // create controller for the IO globe, input the container as the parameter\n    var controller = new GIO.Controller( container );\n\n    /**\n    * use addData() API to add the the data to the controller\n    * know more about data format, check out documentation about data: http://giojs.org/html/docs/dataIntro.html\n    * we provide sample data for test, get sample data from: https://github.com/syt123450/giojs/blob/master/examples/data/sampleData.json\n    */\n    controller.addData( data );\n\n    // call the init() API to show the IO globe in the browser\n    controller.init();\n\n</script>\n```\nIf everything goes well, you shall see [this](http://giojs.org/examples/00_hello_world(simplest).html). For more fancier usage of Gio.js, check out <a href="http://giojs.org/html/docs/index.html">Gio.js API document</a>\n\nClick the codepen logo to try it in Codepen: &nbsp;&nbsp;<a target="_blank" href="https://codepen.io/syt123450/pen/VXNdgM"><img width=50 height=50 src="https://raw.githack.com/syt123450/giojs/master/assets/readme/codepen.png"></a>\n\n<!-- [END getstarted] -->\n\n<!-- [START documentation] -->\n\n## <div id="intro2">Gio.js 2.0 Introduction</div>\n\nAfter Gio.js 1.0 released, developers propose many cool and constructive advise. Gio.js implements most of these features. Gio.js 2.0 comes with many big features such as:\n\n* Data group and switchDataSets API to switch between different data sets [<a href="http://giojs.org/html/docs/dataAdd.html#group">introduction</a>]\n\n* Provide Three.js programming interface [<a href="http://giojs.org/html/docs/interfaceThree.html">introduction</a>]\n\n* Provide Stats.js programming interface [<a href="http://giojs.org/html/docs/interfaceStats.html">introduction</a>]\n\n* Support export data to continent [<a href="http://giojs.org/html/docs/dataContinent.html">introduction</a>]\n\n* Provide input data check [<a href="https://github.com/syt123450/giojs/issues/86">issue</a>]\n\n* Support WeChat mini program [<a href="https://github.com/syt123450/giojs-wechat-minigame-demo">demo</a>]\n\n* New clearData [<a href="http://giojs.org/html/docs/dataClear.html">introduction</a>]\n\n* New closeLiveLoader [<a href="http://giojs.org/html/docs/dataLiveLoad.html">introduction</a>]\n\n* Transparent Background [<a href="http://giojs.org/html/docs/designTransparent.html">introduction</a>]\n\n* Auto Rotation [<a href="https://giojs.org/html/docs/designRotation.html">introduction</a>]\n\n## Examples\n\nGio.js has many representative demos to show how to work with Gio.js, there are mainly three example `configure`(json)](http://giojs.org/html/docs/configureAPI.html)** \n\nConfigure the controller.\n\n**[\xe2\x9e\xa1 `setInitCountry(ISOCode, default: \'CN\')`](http://giojs.org/html/docs/designInitCountry.html)**\n\nSet the country initially selected.\n\n**[\xe2\x9e\xa1 `lightenMentioned(boolean, default: \'false\')`](http://giojs.org/html/docs/designLightenMen.html)**\n\nMakes the mentioned country brighter than those not mentioned in the input dataset.\n\n**[\xe2\x9e\xa1 `disableUnmentioned(boolean, default: \'false\')`](http://giojs.org/html/docs/designDisableUnmen.html)**\n\nDisables an unmentioned country and makes it unselectable.\n\n**[\xe2\x9e\xa1 `showInOnly(boolean, default: \'false\')`](http://giojs.org/html/docs/designShowOnly.html)**\n\nDisplays only the line with data flowing into the selected country.\n\n**[\xe2\x9e\xa1 `showOutOnly(boolean, default: \'false\')`](http://giojs.org/html/docs/designShowOnly.html)**\n\nDisplays only the line with data flowing out of the selected country.\n\n**[\xe2\x9e\xa1 `addHalo(string)`](http://giojs.org/html/docs/designHalo.html)**\n\nEnables the halo around the earth. When enabling halo, the color of halo also can be specified.\n\n**[\xe2\x9e\xa1 `removeHalo()`](http://giojs.org/html/docs/designHalo.html)**\n\nDisables the halo around the earth.\n\n**[\xe2\x9e\xa1 `enableStats()`](http://giojs.org/html/docs/designStats.html)**\n\nDisplays stats on the upper-left corner of the scene.\n\n**[\xe2\x9e\xa1 `disableStats()`](http://giojs.org/html/docs/designStats.html)**\n\nHides stats on the upper-left corner of the scene.\n\n**[\xe2\x9e\xa1 `setTransparentBackground()`](http://giojs.org/html/docs/designTransparent.html)**\n\nMake default background transparent.\n\n**[\xe2\x9e\xa1 `setAutoRotation()`](https://giojs.org/html/docs/designRotation.html)**\n\nMake globe rotate automatically.\n\n**[\xe2\x9e\xa1 `setStyle(string)`](http://giojs.org/html/docs/colorStyle.html)**\n\nSets the color scheme of the 3D globe by name.\n\n**[\xe2\x9e\xa1 `setSurfaceColor(string, default: \'#ffffff\')`](http://giojs.org/html/docs/colorSurface.html)**\n\nSets the color of the surface the 3D globe.\n\n**[\xe2\x9e\xa1 `setSelectedColor(string, default: \'#ffffff\')`](http://giojs.org/html/docs/colorSelected.html)**\n\nSets color of the country in selected state.\n\n**[\xe2\x9e\xa1 `setExportColor(string, default: \'#DD380C\')`](http://giojs.org/html/docs/colorExport.html)**\n\nSets the color of out line.\n\n**[\xe2\x9e\xa1 `setImportColor(string, default: \'#154492\')`](http://giojs.org/html/docs/colorImport.html)**\n\nSets the color of in line.\n\n**[\xe2\x9e\xa1 `setHaloColor(string, default: \'#ffffff\')`](http://giojs.org/html/docs/colorHalo.html)**\n\nSets the color of the Halo.\n\n**[\xe2\x9e\xa1 `setBackgroundColor(string, default: \'#000000\')`](http://giojs.org/html/docs/colorBackground.html)**\n\nSets the color of the background.\n\n**[\xe2\x9e\xa1 `adjustOceanBrightness(float, default: 0.5)`](http://giojs.org/html/docs/colorOceanBr.html)**\n\nSets the brightness of the ocean.\n\n**[\xe2\x9e\xa1 `adjustRelatedBrightness(float, default: 0.5)`](http://giojs.org/html/docs/colorRelatedBr.html)**\n\nSets the brightness of the related country.\n\n**[\xe2\x9e\xa1 `adjustMentionedBrightness(float, default: 0.5)`](http://giojs.org/html/docs/colorMenBr.html)**\n\nSets the brightness of the mentioned country.\n\n**[\xe2\x9e\xa1 `addData(json)`](http://giojs.org/html/docs/dataAdd.html)**\n\nLoads data to the controller of Gio and overrides previous data synchronously.\n\n**[\xe2\x9e\xa1 `clearData()`](http://giojs.org/html/docs/dataClear.html)**\n\nClear data in the globe, and remove spline system and moving particle.\n\n**[\xe2\x9e\xa1 `switchDataSet()`](http://giojs.org/html/docs/dataSwitchSet.html)**\n\nswitch data set when a data group is added.\n\n**[\xe2\x9e\xa1 `addDataAsync(url, callback)`](http://giojs.org/html/docs/dataAddAsync.html)**\n\nLoads data to the controller of Gio and overrides previous data asynchronously.\n\n**[\xe2\x9e\xa1 `liveLoad(url, callback, duration)`](http://giojs.org/html/docs/dataLiveLoad.html)**\n\nLoads data to the controller periodically.\n\n**[\xe2\x9e\xa1 `closeLiveLoader()`](http://giojs.org/html/docs/dataLiveLoad.html)**\n\nClose live load function.\n\n**[\xe2\x9e\xa1 `getScene()`](http://giojs.org/html/docs/interfaceThree.html)**\n\nGet the reference of three.js scene object.\n\n**[\xe2\x9e\xa1 `getStatsObject()`](http://giojs.org/html/docs/interfaceStats.html)**\n\nGet the reference of Stats.js panel object.\n\n**[\xe2\x9e\xa1 `onCountryPicked(callback)`](http://giojs.org/html/docs/callbackPicked.html)**\n\nFired when selected country changes.\n\n**[\xe2\x9e\xa1 `switchCountry(IsoCode)`](http://giojs.org/html/docs/advancedSwitch.html)**\n\nChange currently selected country.\n\n## Documentation\n\n- For quick start, checkout [Getting
