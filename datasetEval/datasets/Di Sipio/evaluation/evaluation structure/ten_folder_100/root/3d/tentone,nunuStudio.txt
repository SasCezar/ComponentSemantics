b'<img src="https://raw.githubusercontent.com/tentone/nunuStudio/master/docs/img/github/logo.png">\r\n\r\n[![GitHub version](https://badge.fury.io/gh/tentone%2FnunuStudio.svg)](https://badge.fury.io/gh/tentone%2FnunuStudio)\r\n[![GitHub issues](https://img.shields.io/github/issues/tentone/nunuStudio.svg)](https://github.com/tentone/nunuStudio/issues) [![GitHub stars](https://img.shields.io/github/stars/tentone/nunuStudio.svg)](https://github.com/tentone/nunuStudio/stargazers)\r\n\r\n - nunustudio is an open source 3D VR game engine for the web it allows designers and web developers to easily develop 3D experiences that can run directly in a web page or be exported as Desktop applications.\r\n - It has a fully featured visual editor, supports a wide range of file formats, the tools are open source and completely free to use for both personal and commercial usage, it is powered by open web APIs like WebGL, WebVR and WebAudio.\r\n - Visual scene editor, code editor, visual tools to edit textures, materials, particle emitters, etc and a powerful scripting API that allows the creation of complex applications.\r\n\r\n\r\n\r\n## Web Editor\r\n - There is a fully featured web version of the editor available at https://nunustudio.org/editor\r\n - The web version was tested with Firefox, Chrome and Microsoft Edge, mobile browsers are not supported.\r\n\r\n<img src="https://raw.githubusercontent.com/tentone/nunuStudio/master/docs/img/github/web.png">\r\n\r\n## Documentation\r\n - Documentation for the scripting API is available on the nunu webpage https://nunustudio.org/docs\r\n    - nunuStudio was documented using YUIDocs\r\n - Tutorials available on the project page at <https://nunustudio.org/learn.html>\r\n\r\n\r\n\r\n## Screenshots\r\n<img src="https://raw.githubusercontent.com/tentone/nunuStudio/master/docs/img/github/2.png"><img src="https://raw.githubusercontent.com/tentone/nunuStudio/master/docs/img/github/3.png">\r\n<img src="https://raw.githubusercontent.com/tentone/nunuStudio/master/docs/img/github/4.png"><img src="https://raw.githubusercontent.com/tentone/nunuStudio/master/docs/img/github/1.png">\r\n<img src="https://raw.githubusercontent.com/tentone/nunuStudio/master/docs/img/github/5.png"><img src="https://raw.githubusercontent.com/tentone/nunuStudio/master/docs/img/github/6.png">\r\n\r\n\r\n\r\n## Features\r\n\r\n- Visual application editor\r\n  - Drag and drop files directly into the project\r\n  - Manage project resources\r\n  - Edit material, textures, shaders, code, etc\r\n- Built on three.js rendering library\r\n  - Real time lighting and shadow map support\r\n  - three.js code can be used inside nunuStudio scripts without the need for THREE prefix\r\n  - Wide range of file formats supported\r\n- NWJS and Cordova exports for desktop and mobile deployment\r\n- Physics engine (cannon.js)\r\n- Compatible with WebVR / WebXR\r\n\r\n\r\n\r\n## Installation\r\n\r\n- nunuStudio is intended to run with NWJS on the desktop (Linux and Windows are supported)\r\n  - Download last version from releases in the github page\r\n- To run the development version of nunuStudio\r\n- There a web version available on the project webpage\r\n  - The web version cannot export desktop and mobile projects\r\n\r\n\r\n\r\n## Building\r\n- nunuStudio uses a custom solution for code management based of an include function\r\n- To build nunuStudio Java and NodeJS are required\r\n  - Javascript is optimized and minified using Google closure\r\n  - Documentation generation uses YuiDocs\r\n- The building system generates minified builds for the runtime and for the editor\r\n- To build nunu editor, runtime and documentation, run "npm run build"\r\n- The build system is compatible with windows, linux and macos.\r\n\r\n\r\n\r\n## Libraries\r\n- nunuStudio is built on top of a number of open source projects\r\n  - NWJS ([nwjs.io](https://nwjs.io))\r\n  - three.js ([github.com/mrdoob/three.js](https://github.com/mrdoob/three.js))\r\n  - Cannon.JS ([schteppe.github.io/cannon.js](https://schteppe.github.io/cannon.js))\r\n  - opentype ([opentype.js.org](https://opentype.js.org))\r\n  - SPE ([github.com/squarefeet/ShaderParticleEngine](https://github.com/squarefeet/ShaderParticleEngine))\r\n  - JSColor ([jscolor.com](http://jscolor.com))\r\n  - CodeMirror ([codemirror.net](https://codemirror.net))\r\n  - Spine Runtime (<https://github.com/EsotericSoftware/spine-runtimes>)\r\n  - LeapJS ([github.com/leapmotion/leapjs](https://github.com/leapmotion/leapjs))\r\n  - JSHint ([jshint.com](https://jshint.com))\r\n  - YUIDocs ([yui.github.io/yuidoc](https://yui.github.io/yuidoc))\r\n\r\n\r\n\r\n## Runtime\r\n- nunuStudio apps are meant to be used inside web pages\r\n- To embed applications made inside nunuStudio in HTML pages the following code can be used\r\n\r\n```html\r\n<html>\r\n\t<head>\r\n\t\t<script src="nunu.min.js"></script>\r\n\t</head>\r\n\t<body onload="NunuApp.loadApp(\'pong.nsp\', \'canvas\')">\r\n\t\t<canvas width="800" height="480" id="canvas"></canvas>\r\n\t</body>\r\n</html>\r\n```\r\n\r\n\r\n\r\n## License\r\n\r\n- nunuStudio uses a MIT license that allow for comercial usage of the platform without any cost.\r\n- The license is available on the project GitHub page\r\n'
