b'[NEON](http://ne-on.org): Nette Object Notation\n===============================================\n\n[![Downloads this Month](https://img.shields.io/packagist/dm/nette/neon.svg)](https://packagist.org/packages/nette/neon)\n[![Build Status](https://travis-ci.org/nette/neon.svg?branch=master)](https://travis-ci.org/nette/neon)\n[![Coverage Status](https://coveralls.io/repos/github/nette/neon/badge.svg?branch=master)](https://coveralls.io/github/nette/neon?branch=master)\n[![Latest Stable Version](https://poser.pugx.org/nette/neon/v/stable)](https://github.com/nette/neon/releases)\n[![License](https://img.shields.io/badge/license-New%20BSD-blue.svg)](https://github.com/nette/neon/blob/master/license.md)\n\n\nIntroduction\n------------\n\nNEON is a human-readable data serialization language. It is commonly used for configuration files, but could be used in many applications where data is being stored\n\nNEON is very similar to YAML.The main difference is that the NEON supports "entities" (so can be used e.g. to parse phpDoc annotations) and tab characters for indentation.\nNEON syntax is a little simpler and the parsing is faster.\n\nDocumentation can be found on the [website](https://doc.nette.org/neon).\n\nIf you like Nette, **[please make a donation now](https://nette.org/donate)**. Thank you!\n\n\nNEON language\n-------------\n\nTry NEON [in sandbox](https://ne-on.org)!\n\nExample of NEON code:\n\n```\n# my web application config\n\nphp:\n\tdate.timezone: Europe/Prague\n\tzlib.output_compression: yes  # use gzip\n\ndatabase:\n\tdriver: mysql\n\tusername: root\n\tpassword: beruska92\n\nusers:\n\t- Dave\n\t- Kryten\n\t- Rimmer\n```\n\nInstallation\n------------\n\nThe recommended way to install is via Composer:\n\n```\ncomposer require nette/neon\n```\n\nIt requires PHP version 7.1 and supports PHP up to 7.4.\n\n\nUsage\n-----\n\n`Nette\\Neon\\Neon` is a static class for encoding and decoding NEON files.\n\n`Neon::encode()` returns $value encoded into NEON. Flags accepts Neon::BLOCK which formats NEON in multiline format.\n\n```php\nuse Nette\\Neon\\Neon;\n$neon = Neon::encode($value); // Returns $value encoded in NEON\n$neon = Neon::encode($value, Neon::BLOCK); // Returns formatted $value encoded in NEON\n```\n\n`Neon::decode()` converts given NEON to PHP value:\n\n```php\n$value = Neon::decode(\'hello: world\'); // Returns an array [\'hello\' => \'world\']\n```\n\nBoth methods throw `Nette\\Neon\\Exception` on error.\n\n\nEditors plugins\n---------------\n\n- NetBeans IDE has built-in support\n- [PhpStorm](https://plugins.jetbrains.com/plugin/7060?pr)\n- [Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=Kasik96.latte)\n- [Emacs](https://github.com/Fuco1/neon-mode)\n\n\nOther languages\n---------------\n\n- [Neon for Javascript](https://github.com/matej21/neon-js)\n- [Neon for Python](https://github.com/paveldedik/neon-py)\n'
