[here](https://github.com/juliosueiras/terraform-lsp/releases)\n\n## Building\n\n### Native\n\n- download go modules `GO111MODULE=on go mod download`\n- run make or `go build`\n\n### Nixpkgs\n\n- install nixpkgs\n- `nix-build`\n\n## Features\n\n- Variables complex completion(infinite nesting type)\n- Provider Config completion\n- Resource(with infinite block, looking at you kubernetes provider ;) ) completion\n- Data source completion\n- Dynamic Error Checking(Terraform and HCL checks)\n- Communication using provider binary(so it will support any provider as long as is built with terraform 0.12 sdk)\n- Module nesting(inifinte as well) variable completion\n\n## Todo\n\nAll Todos are listed [here](Todo.md)\n\n## LSP Support Table\n\n| Feature | Description | Status
