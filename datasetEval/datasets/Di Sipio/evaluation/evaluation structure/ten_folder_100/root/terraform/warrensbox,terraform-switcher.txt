b'[![Build Status](https://travis-ci.org/warrensbox/terraform-switcher.svg?branch=master)](https://travis-ci.org/warrensbox/terraform-switcher)\n[![Go Report Card](https://goreportcard.com/badge/github.com/warrensbox/terraform-switcher)](https://goreportcard.com/report/github.com/warrensbox/terraform-switcher)\n[![CircleCI](https://circleci.com/gh/warrensbox/terraform-switcher/tree/master.svg?style=shield&circle-token=55ddceec95ff67eb38269152282f8a7d761c79a5)](https://circleci.com/gh/warrensbox/terraform-switcher)\n\n# Terraform Switcher\n\n<img style="text-allign:center" src="https://s3.us-east-2.amazonaws.com/kepler-images/warrensbox/tfswitch/smallerlogo.png" alt="drawing" width="120" height="130"/>\n\n<!-- ![gopher](https://s3.us-east-2.amazonaws.com/kepler-images/warrensbox/tfswitch/logo.png =100x20) -->\n\nThe `tfswitch` command line tool lets you switch between different versions of [terraform](https://www.terraform.io/).\nIf you do not have a particular version of terraform installed, `tfswitch` will download the version you desire.\nThe installation is minimal and easy.\nOnce installed, simply select the version you require from the dropdown and start using terraform.\n\nSee installation guide here: [tfswitch installation](https://warrensbox.github.io/terraform-switcher/)\n\n## Installation\n\n`tfswitch` is available for MacOS and Linux based operating systems.\n\n### Homebrew\n\nInstallation for MacOS is the easiest with Homebrew. [If you do not have homebrew installed, click here](https://brew.sh/).\n\n\n```ruby\nbrew install warrensbox/tap/tfswitch\n```\n\n### Linux\n\nInstallation for other linux operation systems.\n\n```sh\ncurl -L https://raw.githubusercontent.com/warrensbox/terraform-switcher/release/install.sh | bash\n```\n\n### Install from source\n\nAlternatively, you can install the binary from source [here](https://github.com/warrensbox/terraform-switcher/releases)\n\n## How to use:\n### Use dropdown menu to select version\n<img src="https://s3.us-east-2.amazonaws.com/kepler-images/warrensbox/tfswitch/tfswitch.gif" alt="drawing" style="width: 180px;"/>\n\n1.  You can switch between different versions of terraform by typing the command `tfswitch` on your terminal.\n2.  Select the version of terraform you require by using the up and down arrow.\n3.  Hit **Enter** to select the desired version.\n\nThe most recently selected versions are presented at the top of the dropdown.\n\n### Supply version on command line\n<img src="https://s3.us-east-2.amazonaws.com/kepler-images/warrensbox/tfswitch/tfswitch-v4.gif" alt="drawing" style="width: 170px;"/>\n\n1. You can also supply the desired version as an argument on the command line.\n2. For example, `tfswitch 0.10.5` for version 0.10.5 of terraform.\n3. Hit **Enter** to switch.\n\n### See all versions including beta, alpha and release candidates(rc)\n<img src="https://s3.us-east-2.amazonaws.com/kepler-images/warrensbox/tfswitch/tfswitch-v5.gif" alt="drawing" style="width: 170px;"/>\n\n1. Display all versions including beta, alpha and release candidates(rc). \n2. For example, `tfswitch -l` or `tfswitch --list-all` to see all versions.\n3. Hit **Enter** to select the desired version.\n\n\n### Use .tfswitch.toml file  (For non-admin - users with limited privilege on their computers)\nThis is similiar to using a .tfswitchrc file, but you can specify a custom binary path for your terraform installation\n\n<img src="https://s3.us-east-2.amazonaws.com/kepler-images/warrensbox/tfswitch/tfswitch-v7.gif" alt="drawing" style="width: 170px;"/>   \n<img src="https://s3.us-east-2.amazonaws.com/kepler-images/warrensbox/tfswitch/tfswitch-v8.gif" alt="drawing" style="width: 170px;"/>\n\n1. Create a custom binary path. Ex: `mkdir /Users/warrenveerasingam/bin` (replace warrenveerasingam with your username)\n2. Add the path to your PATH. Ex: `export PATH=$PATH:/Users/warrenveerasingam/bin` (add this to your bash profile or zsh profile)\n3. Pass -b or --bin parameter with your custom path to install terraform. Ex: `tfswitch -b /Users/warrenveerasingam/bin/terraform 0.10.8 `\n4. Optionally, you can create a `.tfswitch.toml` file in your terraform directory.\n5. Your `.tfswitch.toml` file should look like this:\n```\nbin = "/Users/warrenveerasingam/bin/terraform"\nversion = "0.11.3"\n```\n4. Run `tfswitch` and it should automatically install the required terraform version in the specified binary path\n\n### Use .tfswitchrc file\n<img src="https://s3.us-east-2.amazonaws.com/kepler-images/warrensbox/tfswitch/tfswitch-v6.gif" alt="drawing" style="width: 170px;"/>\n\n1. Create a `.tfswitchrc` file containing the desired version\n2. For example, `echo "0.10.5" >> .tfswitchrc` for version 0.10.5 of terraform\n3. Run the command `tfswitch` in the same directory as your `.tfswitchrc`\n\n\n**Automatically switch with bash**\n\nAdd the following to the end of your `~/.bashrc` file:\n(Use either `.tfswitchrc` or `.tfswitch.toml`)\n\n```\ncdtfswitch(){\n  builtin cd "$@";\n  cdir=$PWD;\n  if [ -f "$cdir/.tfswitchrc" ]; then\n    tfswitch\n  fi\n}\nalias cd=\'cdtfswitch\'\n```\n\n**Automatically switch with zsh**\n\nAdd the following to the end of your `~/.zshrc` file:\n\n```\nload-tfswitch() {\n  local tfswitchrc_path=".tfswitchrc"\n\n  if [ -f "$tfswitchrc_path" ]; then\n    tfswitch\n  fi\n}\nadd-zsh-hook chpwd load-tfswitch\nload-tfswitch\n```\n> NOTE: if you see an error like this: `command not found: add-zsh-hook`, then you might be on an older version of zsh (see below), or you simply need to load `add-zsh-hook` by adding this to your `.zshrc`:\n>    ```\n>    autoload -U add-zsh-hook\n>    ```\n\n*older version of zsh*\n```\ncd(){\n  builtin cd "$@";\n  cdir=$PWD;\n  if [ -f "$cdir/.tfswitchrc" ]; then\n    tfswitch\n  fi\n}\n```\n\n## Additional Info\n\nSee how to *upgrade*, *uninstall*, *troubleshoot* here:[More info](https://warrensbox.github.io/terraform-switcher/additional)\n\n\n## Issues\n\nPlease open  *issues* here: [New Issue](https://github.com/warrensbox/terraform-switcher/issues)\n'
