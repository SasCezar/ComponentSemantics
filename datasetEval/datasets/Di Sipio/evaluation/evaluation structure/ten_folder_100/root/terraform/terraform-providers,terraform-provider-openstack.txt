b'Terraform OpenStack Provider\n============================\n\n- Website: https://www.terraform.io\n- [![Gitter chat](https://badges.gitter.im/hashicorp-terraform/Lobby.png)](https://gitter.im/hashicorp-terraform/Lobby)\n- Mailing list: [Google Groups](http://groups.google.com/group/terraform-tool)\n\n<img src="https://cdn.rawgit.com/hashicorp/terraform-website/master/content/source/assets/images/logo-hashicorp.svg" width="600px">\n\nMaintainers\n-----------\n\nThis provider plugin is maintained by:\n\n* Andrei Ozerov ([@ozerovandrei](https://github.com/ozerovandrei))\n* Gavin Williams ([@fatmcgav](https://github.com/fatmcgav))\n* Joe Topjian ([@jtopjian](https://github.com/jtopjian))\n\nRequirements\n------------\n\n-\t[Terraform](https://www.terraform.io/downloads.html) 0.12.x\n-\t[Go](https://golang.org/doc/install) 1.12 (to build the provider plugin)\n\nBuilding The Provider\n---------------------\n\nClone repository to: `$GOPATH/src/github.com/terraform-providers/terraform-provider-openstack`\n\n```sh\n$ mkdir -p $GOPATH/src/github.com/terraform-providers; cd $GOPATH/src/github.com/terraform-providers\n$ git clone https://github.com/terraform-providers/terraform-provider-openstack\n```\n\nEnter the provider directory and build the provider\n\n```sh\n$ cd $GOPATH/src/github.com/terraform-providers/terraform-provider-openstack\n$ make build\n```\n\nUsing the provider\n----------------------\nPlease see the documentation at [terraform.io](https://www.terraform.io/docs/providers/openstack/index.html).\n\nOr you can browse the documentation within this repo [here](https://github.com/terraform-providers/terraform-provider-openstack/tree/master/website/docs).\n\nDeveloping the Provider\n---------------------------\n\nIf you wish to work on the provider, you\'ll first need [Go](https://golang.org) installed on your machine (version 1.12+ is *required*). You\'ll also need to correctly setup a [GOPATH](https://golang.org/doc/code.html#GOPATH), as well as adding `$GOPATH/bin` to your `$PATH`.\n\nTo compile the provider, run `make build`. This will build the provider and put the provider binary in the `$GOPATH/bin` directory.\n\n```sh\n$ make build\n...\n$ $GOPATH/bin/terraform-provider-openstack\n...\n```\n\nFor further details on how to work on this provider, please see the [Testing and Development](https://www.terraform.io/docs/providers/openstack/index.html#testing-and-development) documentation.\n\nThank You\n---------\n\nWe\'d like to extend special thanks and appreciation to the following:\n\n### OpenLab\n\n<a href="http://openlabtesting.org/"><img src="assets/openlab.png" width="600px"></a>\n\nOpenLab is providing a full CI environment to test each PR and merge for a variety of OpenStack releases.\n\n### VEXXHOST\n\n<a href="https://vexxhost.com/"><img src="assets/vexxhost.png" width="600px"></a>\n\nVEXXHOST is providing their services to assist with the development and testing of this provider.\n\n\n'
