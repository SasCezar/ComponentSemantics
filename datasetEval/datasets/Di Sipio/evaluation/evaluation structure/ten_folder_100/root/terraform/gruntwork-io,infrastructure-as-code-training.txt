b'# Infrastructure-as-code Training\n\nThis repo contains materials for learning how to use infrastructure-as-code practices, including:\n\n1. An [example Ruby on Rails app](/example-rails-app) used for demonstration purposes.\n1. A [Packer example](/packer-example) that shows how to create an AMI that has Ruby on Rails installed and contains\n   the code from the example Rails app.\n1. A [Docker example](/docker-example) that shows how to create a Docker container that has Node.js installed and\n   runs a simple Node "Hello, World" web server.\n1. A [basic Terraform example](/terraform-example-basic) that shows an intro to Terraform.\n1. A [full Terraform example](/terraform-example-full) that shows more advanced Terraform usage, including how to take\n   the AMI created by the Packer example, deploy it on AWS, and run the example Rails app.\n\nNote: all the code in this repo is used only for demonstration and teaching purposes and should not be used in\nproduction. If you need help using Packer, Terraform, or any other DevOps tools in production, reach out to us at\n[Gruntwork](http://www.gruntwork.io/).\n'
