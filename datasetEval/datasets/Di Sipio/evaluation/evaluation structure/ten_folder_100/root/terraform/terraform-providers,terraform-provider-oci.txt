b'## NOTICE  \n**The OCI Terraform Provider is now available for automatic download through the Terraform Provider Registry. \nFor more information on how to get started view the [documentation](https://www.terraform.io/docs/providers/oci/index.html) \nand [setup guide](https://www.terraform.io/docs/providers/oci/guides/version-3-upgrade.html).**\n\n\nTerraform Provider for Oracle Cloud Infrastructure\n==================\n\n- [Documentation](https://www.terraform.io/docs/providers/oci/index.html)\n- [Examples](https://github.com/terraform-providers/terraform-provider-oci/tree/master/examples)\n- [OCI forums](https://cloudcustomerconnect.oracle.com/resources/9c8fa8f96f/summary)\n- [Github issues](https://github.com/terraform-providers/terraform-provider-oci/issues)\n- [Troubleshooting](https://www.terraform.io/docs/providers/oci/guides/troubleshooting.html)\n\n[![wercker status](https://app.wercker.com/status/666d2ee10f45dde41189bb03248aadf9/s/master "wercker status")](https://app.wercker.com/project/byKey/666d2ee10f45dde41189bb03248aadf9)\n\n\nRequirements\n------------\n\n- [Terraform](https://www.terraform.io/downloads.html) v0.10.1 or greater\n- [Go](https://golang.org/doc/install) 1.11.4 (recommended)\n\nNote: You may use any version 1.8 or above to build the provider. However, the `goimports`, `go vet`, and `gofmt` code checks will only pass when using version 1.11.\n\n\nBuilding the Provider\n---------------------\n\nClone repository to: `$GOPATH/src/github.com/terraform-providers/terraform-provider-oci`\n\n```sh\n$ mkdir -p $GOPATH/src/github.com/terraform-providers; cd $GOPATH/src/github.com/terraform-providers\n$ git clone git@github.com:terraform-providers/terraform-provider-oci\n```\n\nEnter the provider directory and build the provider\n\n```sh\n$ cd $GOPATH/src/github.com/terraform-providers/terraform-provider-oci\n$ make build\n```\n\n\nUsing the Provider\n----------------------\nIf you\'re building the provider, follow the instructions to [install it as a plugin.](https://www.terraform.io/docs/plugins/basics.html#installing-a-plugin) \nAfter placing it into your plugins directory,  run `terraform init` to initialize it and begin using Terraform with the Oracle Cloud Infrastructure provider.\n\n\nTroubleshooting the Provider\n----------------------\n\nSee [verbose logging](https://www.terraform.io/docs/providers/oci/guides/troubleshooting.html#verbose-logging-for-oci-terraform-provider) for the details.\n\nDeveloping the Provider\n---------------------------\n\nTo add features to the provider, install [Go](http://www.golang.org) and configure your your [GOPATH](http://golang.org/doc/code.html#GOPATH)\n\nCompile the provider by running `make build`. The provider binary will output to your `$GOPATH/bin` directory, make sure this has been added to your `$PATH`.\n\n```sh\n$ make build\n```\n\nTo test the provider run `make testacc`.\n\n```sh\n$ make testacc\n```\n\n> **Note:** The tests run against live OCI service APIs, you will need to configure environment variables with valid credientials as shown in the [documentation](https://www.terraform.io/docs/providers/oci/index.html).\n'
