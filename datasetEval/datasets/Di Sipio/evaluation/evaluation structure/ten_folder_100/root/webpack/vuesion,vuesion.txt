Run `npx @vuesion/create [project-name]` for example `npx @vuesion/create my-app`\n2. Change directory `cd my-app`\n3. Run `npm run dev`\n\n## Via Git\n\n1. Download or clone (`git clone https://github.com/vuesion/vuesion`) the repository\n2. Change directory `cd vuesion`\n3. Install dependencies: `npm install`\n4. Run `npm run dev`\n\n# Sponsors\n\n<a href="https://careers.bcgdv.com/locations/berlin" title="BCG Digital Ventures">\n  <img src="https://user-images.githubusercontent.com/1667598/38944976-89d5c03c-4335-11e8-92f4-910049c2166a.jpeg" alt="BCG Digital Ventures" />\n</a>\n\n# Support this project\n\n- [Give a :star:!](https://github.com/vuesion/vuesion/stargazers)\n- [![Tweet](https://img.shields.io/twitter/url/http/shields.io.svg?style=flat)](https://twitter.com/intent/tweet?text=Vuesion%20an%20enterprise%20ready%20boilerplate%20for%20isomorphic,%20progressive%20web%20apps%20with%20Vue.JS&url=https://github.com/vuesion/vuesion&via=vuesion1&hashtags=Vuesion,VueJS,SEO,Enterprise) it!\n- Join <a href="https://slack-vuesion.herokuapp.com/" target="_blank"><img src="https://home-assistant.io/images/supported_brands/slack.png" height="20px" /></a> or <a href="https://discord.gg/59x5cg2" target="_blank"><img src="https://cdn0.iconfinder.com/data/icons/square-logo-buttons/512/discord-1-128.png" height="20px" /></a>!\n- [Contribute!](https://github.com/vuesion/vuesion/blob/master/.github/CONTRIBUTING.md)\n- [Fund an Issue!](https://issuehunt.io/r/vuesion/vuesion)\n\n# Contribute\n\nContributions are always welcome!
