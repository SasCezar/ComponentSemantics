Why does search for package X throw `MissingDependencyError` ?\n\nThis error is thrown if a package `require`s a dependency without adding it in its dependencies or peerDependencies list. In the absence of such a definition, we cannot reliably report the size of the package - since we cannot resolve any information about the package.\n\nIn such a case, it\'s best to report an issue with the package author asking the missing package to be added to its `package.json`\n\n#### 2.
