Django\n\n:License: MIT\n\nBootcamp is an open source **enterprise social network** of open purpose, on which you can build for your own ends.\n\nThe project has four basic apps:\n\n* News (A Twitter-like microblog)\n* Articles (A collaborative blog)\n* Question & Answers (A Stack Overflow-like platform)\n* Messeger (A basic chat-a-like tool for asynchronous communication.)\n\nTechnology
