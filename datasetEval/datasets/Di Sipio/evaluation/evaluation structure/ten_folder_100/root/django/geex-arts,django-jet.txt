or\n    easy_install django-jet\n\n* Add 'jet' application to the INSTALLED_APPS setting of your Django project settings.py file (note it should be before 'django.contrib.admin'):\n\n.. code:: python\n\n    INSTALLED_APPS = (\n        ...\n        'jet',\n        'django.contrib.admin',\n    )\n        \n* Make sure ``django.template.context_processors.request`` context processor is enabled in settings.py (Django 1.8+ way):\n\n.. code:: python\n\n    TEMPLATES = [\n        {\n            'BACKEND': 'django.template.backends.django.DjangoTemplates',\n            'DIRS': [],\n            'APP_DIRS': True,\n            'OPTIONS': {\n                'context_processors': [\n                    ...\n                    'django.template.context_processors.request',\n                    ...\n                ],\n            },\n        },\n    ]\n\n.. warning::\n    Before Django 1.8 you should specify context processors different way.
