b"*********************\nDjango Simple Captcha\n*********************\n\n.. image:: https://travis-ci.org/mbi/django-simple-captcha.png?branch=master\n  :target: http://travis-ci.org/mbi/django-simple-captcha\n\n\nDjango Simple Captcha is an extremely simple, yet highly customizable Django application to add captcha images to any Django form.\n\n.. image:: http://django-simple-captcha.readthedocs.io/en/latest/_images/captcha3.png\n\nFeatures\n++++++++\n\n* Very simple to setup and deploy, yet very configurable\n* Can use custom challenges (e.g. random chars, simple maths, dictionary word, ...)\n* Custom generators, noise and filter functions alter the look of the generated image\n* Supports text-to-speech audio output of the challenge text, for improved accessibility\n* Ajax refresh\n\nRequirements\n++++++++++++\n\n* Django 1.8+\n* A recent version of the Pillow compiled with FreeType support\n* Flite is required for text-to-speech (audio) output, but not mandatory\n\nPython 3 compatibility\n++++++++++++++++++++++\n\nThe current development version supports Python3 via the `six <https://pypi.python.org/pypi/six>`_ compatibility layer.\nYou will need to install `Pillow <https://github.com/python-imaging/Pillow>`_ because PIL doesn't support Python3 yet.\n\n\nDocumentation\n+++++++++++++\n\nRead the `documentation online <http://django-simple-captcha.readthedocs.org/en/latest/>`_.\n"
