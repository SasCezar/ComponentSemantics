Running\n- Prerequisites\n  - Node\n  - MySQL (or Postgres / Sqlite / MSSQL)\n- Clone repo `git clone git@github.com:atulmy/crate.git crate`\n- Switch to `code` directory `cd code`\n- Configurations\n  - Modify `/api/src/config/database.json` for database credentials\n  - Modify `/api/.env` for PORT (optional)\n  - Modify `/web/.env` for PORT / API URL (optional)\n  - Modify `/mobile/src/setup/config.json` for API URL (tip: use `ifconfig` to get your local IP address)\n  - Add your local IP address in: `code/mobile/android/app/src/debug/res/xml/react_native_config.xml` for Cleartext Traffic (API level 28+). Read more [here](https://facebook.github.io/react-native/docs/integration-with-existing-apps#network-security-config-api-level-28).\n- Setup\n  - API: Install packages and database setup (migrations and seed) `cd api` and `npm run setup`\n  - Webapp: Install packages `cd web` and `npm install`\n  - Mobile: Install packages `cd mobile` and `npm install`\n- Development\n  - Run API `cd api` and `npm start`, browse GraphiQL at http://localhost:8000/\n  - Run Webapp `cd web` and `npm start`, browse webapp at http://localhost:3000/\n  - Run Mobile `cd mobile` and `npm start`, browse mobile on either emulator or using Expo on your mobile phone\n- Production\n  - Run API `cd api` and `npm run start:prod`, creates an optimized build in `build` directory and runs the server\n  - Run Webapp `cd web` and `npm run start:prod`, creates an optimized build in `build` directory and runs the server\n\n## Multi-package automation\n- New developers are advised to run through the above 'setup and running' process before reading further.\n- Optional multi-package automation for faster setup and easier dev environment initiation.\n- No need to cd to sub-folders unless working with mobile or running a production build.\n- Once Node, MySQL, repo clone and configuration are setup correctly\n    - Switch to `code` directory `cd code`\n    - Setup\n        - Setup API, Webapp and Mobile with a single command `npm run setup`\n    - Development\n        - Run API and Webapp `npm start`, browse GraphiQL at http://localhost:8000/ and Webapp at http://localhost:8000/\n        - Run API alone `npm start:api`, browse GraphiQL at http://localhost:8000/\n        - Run Webapp alone `npm start:web`, browse webapp at http://localhost:3000/\n\n## Resources and Inspirations\n- \xe2\x9c\x8d\xef\xb8\x8f Opinionated project architecture for Full-Stack JavaScript Applications -
