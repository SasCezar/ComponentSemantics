started. \nMake sure you have Xcode installed from the App Store. \nThen run the following command to install Xcode\'s command line tools, if you don\'t have that yet\n```sh\nxcode-select --install\n```\n\nInstall [`Bundler`](https://bundler.io) for managing Ruby gem dependencies\n```sh\n[sudo] gem install bundler\n```\n\nInstall [Brew](https://github.com/Homebrew/brew) package manager for macOS\n```sh\n/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"\n```\n\nInstall [`Node`](https://nodejs.org/en/) (required to install [Apollo](https://github.com/apollographql/apollo-ios))\n```sh\nbrew install node\n```\n\nThe following commands will set up SwiftHub\n```sh\ncd SwiftHub\nbundle install\nbundle exec fastlane setup\n```\n\nTo update all tools and pods\n```sh\nbundle exec fastlane update\n```\n\nAlrighty! We\'re ready to go!\n\n## Documentation\nYou can generate the API docs locally. Run `jazzy` from the root of this repository. This requires installation of [jazzy](https://github.com/realm/jazzy/). You will find the output in `docs/`. You can set options for your project\xe2\x80\x99s documentation in a configuration file, [.jazzy.yaml](https://github.com/khoren93/SwiftHub/blob/master/.jazzy.yaml) by default.\n\n## Debugging\n[Flex](https://github.com/Flipboard/FLEX) debugging tool has been integrated in this application. To enable it, just swipe right anywhere in the application.\nThere are also included debugging [Hero](https://github.com/HeroTransitions/Hero) animations. To use it, swipe right with two fingers. Repeat this to disable.\n\n## Fastlane\n\n[Fastlane](https://fastlane.tools) automates common development tasks - for example bumping version numbers, running tests on multiple configurations, or submitting to the App Store. You can list the available lanes (our project-specific scripts) using `bundle exec fastlane lanes`. You can list available actions (all actions available to be scripted via lanes) using `bundle exec fastlane actions`.
