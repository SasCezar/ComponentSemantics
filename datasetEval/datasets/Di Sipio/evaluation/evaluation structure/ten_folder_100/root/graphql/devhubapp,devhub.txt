GitHub Dashboard), a great way to discover interesting repositories;\n- [x] **Save for later**: Save any activity or notification for later, so you don\'t forget to get back to them;\n- [x] **Keyboard shortcuts**: Do anything using only the keyboard ([see shortcuts below](#keyboard-shortcuts));\n- [x] **Theme support**: Choose between 6 light or dark themes;
