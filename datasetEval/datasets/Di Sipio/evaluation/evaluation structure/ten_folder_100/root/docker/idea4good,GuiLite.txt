code, could run on **all platforms**(e.g, iOS/macOS/WathOS, Android, Linux, Windows, Docker and MCU without OS), and work with **many languages**(e.g. Swift, Java, Javascript, C#, Golang).\n- \xe2\x9c\x82\xef\xb8\x8fSmall: 5,000 lines of C++ code, could be merged into 1 header & 1 source file(GuiLite.h/GuiLite.cpp) - Easy to [build & port](documents/HowToBuild.md)\n- \xe2\x9a\xa1Fast: Render GUI within one invoking, independent of any OS and any third party library -  Make GUI run quickly and smoothly\n- \xf0\x9f\x92\x89Inject: Run inside Qt/MFC/Winform/Cocoa/Web - Keep Qt/MFC code reusable, and speed it up by local GuiLite-optimization\n- \xe2\x9a\x99\xef\xb8\x8f\xef\xb8\x8fHardware Minimum Requirment:\n\n| Processor | Disk/ROM space | Memory |\n| --- | --- | --- |\n| 24 MHZ |
