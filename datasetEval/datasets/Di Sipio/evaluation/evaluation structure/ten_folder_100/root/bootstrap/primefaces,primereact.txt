PrimeReact\n\nPrimeReact is a rich set of open source UI Components for React.\n\n[![PrimeReact Logo](https://www.primefaces.org/wp-content/uploads/2017/09/primereact-transparent-250.png "PrimeReact")](https://www.primefaces.org/primereact)\n\nSee [PrimeReact homepage](https://www.primefaces.org/primereact) for live showcase and documentation.\n\n\n<p align="center">\n  <a href="https://bitsrc.io/primefaces/primereact"><img src="https://i.imagesup.co/images2/0__05c8234cb5e0df.jpg"></a>\n</p>\n\n[Components demo](https://bitsrc.io/primefaces/primereact)\n\n## Download\n\nPrimeReact is available at npm, if you have an existing application run the following command to download it to your project.\n\n```\nnpm install primereact --save\nnpm install primeicons --save\n```\n\n## Import\n\n```javascript\n//import {ComponentName} from \'primereact/{componentname}\';\nimport {Dialog} from \'primereact/dialog\';\nimport {Accordion,AccordionTab} from \'primereact/accordion\';\n```\n\n## Dependencies\n\nMajority of PrimeReact components (95%) are native and there are some exceptions having 3rd party dependencies such as Google Maps for GMap.\n\nIn addition, components require PrimeIcons for icons, classNames package to manage style classes and react-transition-group for animations.\n\n```json\ndependencies: {\n    "react": "^16.0.0",\n    "react-dom": "^16.0.0",\n    "react-transition-group": "^2.2.1",\n    "classnames": "^2.2.5",\n    "primeicons": "^2.0.0"\n}\n```\n\n## Styles\nThe css dependencies are as follows, note that you may change the theme with another one of your choice.\n\n```\nprimereact/resources/themes/nova-light/theme.css\nprimereact/resources/primereact.min.css\nprimeicons/primeicons.css\n```\n\nIf you are using a bundler such as webpack with a css loader you may also import them to your main application component, an example from create-react-app would be.\n\n```javascript\nimport \'primereact/resources/themes/nova-light/theme.css\';\nimport \'primereact/resources/primereact.min.css\';\nimport \'primeicons/primeicons.css\';\n```\n\n## QuickStart\n\nAn [example application](https://github.com/primefaces/primereact-quickstart) based on create-react-app is available at github.\n\n## TypeScript\n\nTypescript\nTypescript is fully supported as type definition files are provided in the npm package of PrimeReact. A sample [typescript-primereact application](https://github.com/primefaces/primereact-typescript-quickstart) is available as well at github.\n'
