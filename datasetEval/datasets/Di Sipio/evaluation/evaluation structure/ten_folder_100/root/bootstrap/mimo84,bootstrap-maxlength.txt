b'# [Bootstrap MaxLength](http://mimo84.github.com/bootstrap-maxlength/) [![Build Status](https://travis-ci.org/mimo84/bootstrap-maxlength.png?branch=master)](https://travis-ci.org/mimo84/bootstrap-maxlength)\n\n\nThis plugin integrates by default with Twitter bootstrap using badges to display the maximum length of the field where the user is inserting text.\nThis plugin uses the HTML5 attribute "maxlength" to work.\n\nThe indicator badge shows up on focusing on the element, and disappears when the focus is lost.\n\n## Configurable options\n\n * **alwaysShow**: if true the threshold will be ignored and the remaining length indication will be always showing up while typing or on focus on the input. Default: false.\n * **threshold**: this is a number indicating how many chars are left to start displaying the indications. Default: 10.\n * **warningClass**: it\'s the class of the element with the indicator. By default is the bootstrap "label label-success" but can be changed to anything you\'d like.\n * **limitReachedClass**: it\'s the class the element gets when the limit is reached. Default is "label label-important label-danger" (to support Bootstrap 2 and 3).\n * **separator**: represents the separator between the number of typed chars and total number of available chars. Default is "/".\n * **preText**: is a string of text that can be outputted in front of the indicator. preText is empty by default.\n * **postText**: is a string outputted after the indicator. postText is empty by default.\n * **showMaxLength**: if false, will display just the number of typed characters, e.g. will not display the max length. Default: true.\n * **showCharsTyped**: if false, will display just the remaining length, e.g. will display remaining length instead of number of typed characters. Default: true.\n * **placement**: is a string, define where to output the counter. Possible values are: **bottom** ( *default option* ), **left**, **top**, **right**, **bottom-right**, **top-right**, **top-left**, **bottom-left** and **centered-right**.\n *  **appendToParent**: appends the maxlength indicator badge to the parent of the input rather than to the body.\n * **message**: an alternative way to provide the message text, i.e. \'You have typed %charsTyped% chars, %charsRemaining% of %charsTotal% remaining\'. %charsTyped%, %charsRemaining% and %charsTotal% will be replaced by the actual values. This overrides the options separator, preText, postText and showMaxLength. Alternatively you may supply a function that the current text and max length and returns the string to be displayed. For example, function(currentText, maxLength) { return \'\' + Math.ceil(currentText.length / 160) + \' SMS Message(s)\'; }\n * **utf8**: if true the input will count using utf8 bytesize/encoding.  For example: the \'\xc2\xa3\' character is counted as two characters.\n * **showOnReady**: shows the badge as soon as it is added to the page, similar to alwaysShow\n * **twoCharLinebreak**: count linebreak as 2 characters to match IE/Chrome textarea validation.\n * **customMaxAttribute**: String -- allows a custom attribute to display indicator without triggering native maxlength behaviour.  Ignored if value greater than a native maxlength attribute. \'overmax\' class gets added when exceeded to allow user to implement form validation.\n * **allowOverMax**: Will allow the input to be over the customMaxLength. Useful in soft max situations.\n * **placement**: is a string, object, or function, to define where to output the counter.\n   * Possible string values are: **bottom** ( *default option* ), **left**, **top**, **right**, **bottom-right**, **top-right**, **top-left**, **bottom-left** and **centered-right**. Are also available : **bottom-right-inside** (like in Google\'s material design, **top-right-inside**, **top-left-inside** and **bottom-left-inside**.\n   * Custom placements can be passed as an object, with keys **top**, **right**, **bottom**, **left**, and **position**. These are passed to $.fn.css.\n   * A custom function may also be passed. This method is invoked with the {$element} Current Input, the {$element} MaxLength Indicator, and the Current Input\'s Position {bottom height left right top width}.\n\n\n## Events\n\n* **maxlength.reposition** on an input element triggers re-placing of its indicator. Useful if textareas are resized by an external trigger.\n* **maxlength.shown** is triggered when the indicator is displayed.\n* **maxlength.hidden** is triggered when the indicator is removed from view.\n\n## Examples\n\nBasic implementation:\n```javascript\n$(\'input[maxlength]\').maxlength();\n```\n\nChange the threshold value:\n```javascript\n$(\'input.className\').maxlength({\n    threshold: 20\n});\n```\n\nAn example with some of the configurable options:\n```javascript\n$(\'input.className\').maxlength({\n    alwaysShow: true,\n    threshold: 10,\n    warningClass: "label label-info",\n    limitReachedClass: "label label-warning",\n    placement: \'top\',\n    preText: \'used \',\n    separator: \' of \',\n    postText: \' chars.\'\n});\n```\n\nThe same example using the message option:\n\n```javascript\n$(\'input.className\').maxlength({\n    alwaysShow: true,\n    threshold: 10,\n    warningClass: "label label-info",\n    limitReachedClass: "label label-warning",\n    placement: \'top\',\n    message: \'used %charsTyped% of %charsTotal% chars.\'\n});\n```\n\nAn example allowing user to enter over max characters. __NOTE: you cannot have the maxlength attribute on the input element. You will need to provide the customMaxAttribute attribute__\n\nSample HTML element:\n```html\n<textarea class="form-control" id="xyz" name="xyz" maxlength="10"></textarea>\n```\n\n```javascript\n// Setup maxlength\n$(\'.form-control\').maxlength({\n\talwaysShow: true,\n\tvalidate: false,\n\tallowOverMax: true,\n    customMaxAttribute: "90"\n});\n\n// validate form before submit\n$(\'form\').on(\'submit\', function (e) {\n\t$(\'.form-control\').each(\n\t\tfunction () {\n\t\t\tif ($(this).hasClass(\'overmax\')) {\n\t\t\t\talert(\'prevent submit here\');\n\t\t\t\te.preventDefault();\n\t\t\t\treturn false;\n\t\t\t}\n\t\t}\n\t);\n});\n```\n\nAn example of triggering a `maxlength.reposition` event whenever an external autosize plugin resizes a textarea:\n```javascript\n$(\'textarea\').on(\'autosize:resized\', function() {\n    $(this).trigger(\'maxlength.reposition\');\n});\n```\n'
