Pick an outstanding task, let us know what you are working on and fire away with any questions.\n\nThe package is made up of 2 main folders:\n\n- /src contains all the Tabler React components\n- /example is our [create-react-app](https://github.com/facebook/create-react-app/) based demo website\n\nTo setup and run a local copy:\n\n1.  Clone this repo with `git clone https://github.com/tabler/tabler-react`\n2.  Run `yarn install` in the root folder\n3.  Run `yarn install` in the example folder\n4.  In seperate terminal windows, run `yarn start` in the root and example folders.\n\nYou should now be up and running with live browser reloading of the example website while you work on Tabler React components in the /src folder.\n\nWhen you\'re done working on your changes, submit a PR with the details and include a screenshot if you\'ve changed anything visually.\n\n## License\n\nMIT \xc2\xa9 [jonthomp](https://github.com/jonthomp), [AaronCoplan](https://github.com/AaronCoplan) and [the contributors](https://github.com/tabler/tabler-react/graphs/contributors).\n'
