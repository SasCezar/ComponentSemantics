repo\n$ git clone https://github.com/coreui/coreui-free-vue-admin-template.git CoreUI-Vue\n\n# go into app\'s directory\n$ cd CoreUI-Vue\n\n# install app\'s dependencies\n$ npm install\n```\n\n##
