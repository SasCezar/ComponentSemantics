b'\nSocial Buttons for Bootstrap\n============================\n[![npm version](https://badge.fury.io/js/bootstrap-social.svg)](https://badge.fury.io/js/bootstrap-social)\n[![Bower version](https://badge.fury.io/bo/bootstrap-social.svg)](https://badge.fury.io/bo/bootstrap-social)\n\nSocial Buttons made in pure CSS based on\n[Bootstrap](http://twbs.github.io/bootstrap/) and\n[Font Awesome](http://fortawesome.github.io/Font-Awesome/)!\n\n[Check the live demo!](http://lipis.github.io/bootstrap-social)\n\nInstallation\n------------\n\nInclude the `bootstrap-social.css` or `bootstrap-social.less` in your project, or\ninstall it through [Bower](http://bower.io/):\n\n    bower install bootstrap-social\n\nAvailable classes\n-----------------\n - `btn-adn`\n - `btn-bitbucket`\n - `btn-dropbox`\n - `btn-facebook`\n - `btn-flickr`\n - `btn-foursquare`\n - `btn-github`\n - `btn-google`\n - `btn-instagram`\n - `btn-linkedin`\n - `btn-microsoft`\n - `btn-odnoklassniki`\n - `btn-openid`\n - `btn-pinterest`\n - `btn-reddit`\n - `btn-soundcloud`\n - `btn-tumblr`\n - `btn-twitter`\n - `btn-vimeo`\n - `btn-vk`\n - `btn-yahoo`\n\nExamples\n--------\n\n```html\n<a class="btn btn-block btn-social btn-twitter">\n  <span class="fa fa-twitter"></span>\n  Sign in with Twitter\n</a>\n\n<a class="btn btn-social-icon btn-twitter">\n  <span class="fa fa-twitter"></span>\n</a>\n```\n\nNotes\n-----\nFor now I won\'t accept any request for new buttons as I\'m planning to\nsplit the main CSS to have separate files for all of the requested ones.\n'
