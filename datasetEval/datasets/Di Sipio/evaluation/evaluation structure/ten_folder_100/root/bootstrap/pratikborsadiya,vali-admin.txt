b"# Vali Admin\n\n![vali-admin](http://pratikborsadiya.in/blog/vali-admin/vali-admin-banner.gif)\n\n[![npm version](https://badge.fury.io/js/vali-admin.svg)](https://badge.fury.io/js/vali-admin)\n\nVali is a free, modular and easy to customize admin theme built using [Bootstrap 4](https://getbootstrap.com), [SASS](http://sass-lang.com) and [Pug.js](https://pugjs.org).\n\n## Getting started\n\nRun a `npm install` command in project root directory to install and build dependencies. If you don't want to edit theme you can use the compiled files inside `docs` folder.\n\nUse `npm run dev` command to watch and compile source files.\n\nUse `npm run build` command to compile all source files.\n\nUse `npm run start` command to start a development server using NodeJs.\n\n> **Note:**\n> * The NodeJs server mentioned in `npm run start` command is for development purpose only. DONOT use it as a production server.\n\n## Customization\nFor more information about customizing theme colors please follow the official [documentation](http://pratikborsadiya.in/blog/vali-admin/).\n\n## RTL Support\nPlease follow the official [documentation](http://pratikborsadiya.in/blog/vali-admin/) to enable RTL support.\n\n## Contributing\nPlease take a look at [contributing guidelines](CONTRIBUTING.md) if you are considering contributing to the repository.\n\n## Contributors\n\n* **[Pratik Borsadiya](http://pratikborsadiya.in)** - *Project Author*\n* **List of [contributors](https://github.com/pratikborsadiya/vali-admin/graphs/contributors)** who participated in this project.\n\n## License\n\nThis project is licensed under the [MIT](LICENSE) License\n"
