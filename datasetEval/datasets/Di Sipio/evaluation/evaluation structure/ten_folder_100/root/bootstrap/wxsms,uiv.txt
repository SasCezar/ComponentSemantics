components **~20KB** Gziped.\n  * Dependencies only **Vue & Bootstrap CSS**.\n  * No extra CSS.\n  * Individually import supported.\n* IE 9+ & modern browsers supported.\n* **SSR** (server-side rendering) supported.\n* All env supported:\n  * UMD build `uiv.min.js` can be used in all environments (including browser)\n  * ES Module build `uiv.esm.js` is for modern bundlers like [webpack 2](https://webpack.js.org) or [rollup](https://rollupjs.org)\n  * CommonJS build `uiv.common.js` is for older bundlers like [browserify](http://browserify.org) or [webpack 1](https://webpack.github.io).\n\n## Documentation\n\nTo check out live examples and docs, visit [https://uiv.wxsm.space](https://uiv.wxsm.space).\n\n## Install\n\nIf you are using module bundlers such as Webpack, you can directly include package into your project via:\n\n**NPM**:\n\n```bash\n$ npm install uiv --save\n```\n\nor **Yarn**:\n\n```bash\n$ yarn add uiv\n```\n\nThen register uiv components and directives all at once in your app\'s entry:\n\n```javascript\n// main.js\nimport \'bootstrap/dist/css/bootstrap.min.css\'\n\nimport Vue from \'vue\'\nimport * as uiv from \'uiv\'\n\nVue.use(uiv)\n```\n\nThat\'s it. Happy coding!\n\n### No Conflict\n\nAll components & directives will be installed with no prefix by default, you can add any prefix to them to avoid conflicts with other libs if needed.\n\nFor example:\n\n```javascript\nVue.use(uiv, {prefix: \'uiv\'})\n```\n\nResults in:\n\n* Components such as `<alert>` becomes `<uiv-alert>`\n* Directives such as `v-tooltip` becomes `v-uiv-tooltip`\n* Global methods such as `$alert` becomes `$uiv_alert`\n\n## Import Individually\n\nIf you don\'t want all of the components for some reason (e.g. to save the bundle size), you can also import them individually.\n\n### Example\n\n```javascript\nimport { Alert } from \'uiv\'\n\nnew Vue({\n  components: {\n    Alert\n  }\n})\n```\n\n## Browsers\n\nYou can load & install uiv package directly in browsers. For example:\n\n```html\n<!-- Remember to import Vue and Bootstrap CSS file before this! -->\n<script src="//unpkg.com/uiv"></script>\n```\n\nThis will simply load the latest version of `uiv.min.js` into your page. For detail usages (e.g. load specify version, IMPORTANT in production mode) and different CDN providers, you can visit:\n\n* [https://unpkg.com](https://unpkg.com)\n* [https://www.jsdelivr.com](https://www.jsdelivr.com/)\n* [https://cdnjs.com/libraries/uiv](https://cdnjs.com/libraries/uiv)\n\n### Complete Usage Example\n\n```html\n<!-- index.html -->\n<html>\n<head>\n  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">\n  <script type="text/javascript" src="//vuejs.org/js/vue.min.js"></script>\n  <script type="text/javascript" src="//unpkg.com/uiv/dist/uiv.min.js"></script>\n</head>\n<body>\n<div id="app">\n  <tabs>\n    <tab>Tab content 1.</tab>\n    <tab>Tab content 2.</tab>\n  </tabs>\n</div>\n<script>\n  // No need to install uiv, we already do this for you after script loaded.\n  new Vue().$mount(\'#app\')\n</script>\n</body>\n</html>\n```\n\nThis will create a working **Tabs** component on your page.\n\n## Browser Compatibility\n\nAll browsers supported by [Vue 2](https://github.com/vuejs/vue) and [Bootstrap 3 CSS](https://github.com/twbs/bootstrap) are suppose to be also supported by this lib  (IE8 and below are not supported).\n\n## Changelog\n\nDetailed changes for each release are documented in the [release notes](https://github.com/wxsms/uiv/releases).\n\n## Contribute\n\n[![JavaScript Style Guide](https://cdn.rawgit.com/standard/standard/master/badge.svg)](https://github.com/standard/standard)\n\nWelcome and thanks to use and contribute to this project.
