b"# Postwill\n\n[![Gem Version](https://badge.fury.io/rb/postwill.svg)](https://badge.fury.io/rb/postwill)\n[![CircleCI](https://circleci.com/gh/postwill/postwill/tree/master.svg?style=svg)](https://circleci.com/gh/postwill/postwill/tree/master)\n[![Maintainability](https://api.codeclimate.com/v1/badges/16f924433bfc8b8ebd6b/maintainability)](https://codeclimate.com/github/postwill/postwill/maintainability)\n\nPosting to the most popular social media\n\n## Installation\n\nAdd this line to your application's Gemfile:\n\n```ruby\ngem 'postwill'\n```\n\nAnd then execute:\n\n    $ bundle\n\nOr install it yourself as:\n\n    $ gem install postwill\n\n## Usage\n\nList of available providers is [here](https://github.com/postwill/postwill/blob/master/PROVIDERS.md).\n\n### Single provider\n\n```ruby\npostwill = Postwill::Client.new(\n                   twitter: {\n                     access_token: 'access_token',\n                     access_token_secret: 'access_token_secret'\n                   }\n                 )\n\npostwill.to(:twitter, text: 'text', image: 'image')\n```\n\nSuccess:\n```ruby\n{\n  twitter: {\n    status: :ok,\n    response: {} # Hash of response data depends on provider\n  }\n}\n```\n\nFailure:\n```ruby\n{\n  twitter: {\n    status: :error,\n    response: '' # String error message\n  }\n}\n```\n\nApplication credentials\n\n`config/initializers/postwill.rb`\n```ruby\nPostwill::Settings.configure do |config|\n  config.providers.twitter = {\n    consumer_key: 'your_consumer_key',\n    consumer_secret: 'your_consumer_secret'\n  }\nend\n```\n\n### Multiple providers\n\n```ruby\npostwill = Postwill::Client.new(\n                   twitter: {\n                     access_token: 'access_token',\n                     access_token_secret: 'access_token_secret'\n                   },\n                   facebook: {\n                     access_token: 'access_token'\n                   }\n                 )\n\npostwill.to(%i[twitter facebook], text: 'text', image: 'image')\n```\n\nResult:\n```ruby\n{\n  twitter: {\n    status: :ok,\n    response: {} # Response data\n  },\n  facebook: {\n    status: :ok,\n    response: {} # Response data\n  }\n}\n```\n\nApplication credentials(for providers which need them)\n\n`config/initializers/postwill.rb`\n```ruby\nPostwill::Settings.configure do |config|\n  config.providers.twitter = {\n    consumer_key: 'your_consumer_key',\n    consumer_secret: 'your_consumer_secret'\n  }\nend\n```\n\n## Contributing\n\nBug reports and pull requests are welcome on GitHub at https://github.com/postwill/postwill.\n\n## License\n\nThe gem is available as open source under the terms of the [MIT License](http://opensource.org/licenses/MIT).\n"
