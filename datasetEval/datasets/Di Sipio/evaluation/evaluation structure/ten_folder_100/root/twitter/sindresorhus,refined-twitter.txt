b'# <img src="source/icon.png" width="45" align="left">Refined Twitter\n\n> Browser extension that simplifies the Twitter interface and adds useful features\n\n[![XO code style](https://img.shields.io/badge/code_style-XO-5ed9c7.svg)](https://github.com/xojs/xo) [![Gitter](https://badges.gitter.im/join_chat.svg)](https://gitter.im/refined-twitter/Lobby)\n\nWe use Twitter a lot and notice many dumb annoyances we\'d like to fix. So here be dragons.\n\n**Read the [blog post](https://blog.sindresorhus.com/refined-twitter-74038424fe2a) and discuss it on [Product Hunt](https://www.producthunt.com/posts/refined-twitter)**\n\n\n## Install\n\n- [**Chrome** extension](https://chrome.google.com/webstore/detail/refined-twitter/nlfgmdembofgodcemomfeimamihoknip)\n- **Opera** extension: Use [this](https://addons.opera.com/en/extensions/details/download-chrome-extension-9/) to enable installing Chrome extensions and then install [Refined Twitter](https://chrome.google.com/webstore/detail/refined-twitter/nlfgmdembofgodcemomfeimamihoknip)\n\n*Note: You **must** be logged in for this extension to work.*\n\n\n## Highlights\n\n*You can configure many of these features in the extension settings.*\n\n- Simplified and improved UI.\n- Hides promoted tweets.\n- Auto-loads new tweets in the stream if you\'re scrolled to the top. No more clicking `See 3 new Tweets`!\n- Fixes the file extension when saving images in tweets. No more `foo.jpg_large`!\n- Uses the system font.\n- [Embeds the photo from Instagram links directly in the tweet.](https://user-images.githubusercontent.com/170270/34315380-12d52994-e77f-11e7-8e23-27b76aee4df2.png)\n- Improves performance. <sup>[1](https://github.com/sindresorhus/refined-twitter/pull/14)</sup> <sup>[2](https://github.com/sindresorhus/refined-twitter/commit/23897e251d2bc8d59526129ce54c7a5bf1ef884c)</sup>\n- Hides "Liked" tweets in the stream.\n- [Hides "And others follow" tweets in the stream.](https://user-images.githubusercontent.com/5341072/39945031-cc81125a-5560-11e8-8334-ea310a9dfdad.png)\n- [Syntax highlighting in code blocks.](https://github.com/sindresorhus/refined-twitter/issues/37)\n- [Adds Markdown-like styling of `text wrapped in backticks`.](https://user-images.githubusercontent.com/12901172/38168571-d9bd82ea-351d-11e8-9858-0d7c8993cdd3.png)\n- Uses the original image in tweet image galleries instead of a downsized version.\n- [Removes the annoying suggestions in the search popover.](https://user-images.githubusercontent.com/170270/33800304-70198358-dd3d-11e7-9870-477a44f74f4d.png)\n- Hides "Notifications" activity for new followers and being added to a list.\n- Preserves unsent text in the Messages modal when it closes.\n- Highlight your mentions in the stream.\n- [Adds a `Likes` button to the main navbar.](https://user-images.githubusercontent.com/14620121/35988497-ace9f93e-0ce5-11e8-8675-17e6ee38cd99.png)\n- Keyboard shortcut to toggle Night Mode (<kbd>Alt</kbd><kbd>m</kbd>).\n- Uses your personal color theme on all profiles.\n- Hides the header image on profile pages.\n- [Shows alternative image text below images when available.](https://user-images.githubusercontent.com/170270/40556400-b46c292c-6076-11e8-8241-f5c4e1a7a161.png)\n\nTip: Twitter has a native [dark mode](https://github.com/sindresorhus/refined-twitter/issues/10) and you can toggle it using <kbd>Alt</kbd><kbd>m</kbd>. And press <kbd>Shift</kbd> <kbd>?</kbd> to see all keyboard shortcuts.\n\n<img src="media/screenshot.gif" width="1272">\n\n\n## Customization\n\nSee the extension settings for what can be configured.\n\n<img src="media/screenshot-settings.png" height="500">\n\nWe\'re happy to receive suggestions and contributions, but be aware this is an opinionated project. There\'s a high bar for adding options.\n\nThis doesn\'t necessarily limit you from manually disabling functionality that is not useful for you. Options include:\n\n1. *(CSS Only)* Use a Chrome extension that allows injecting custom styles into sites, based on a URL pattern. [Stylus](https://add0n.com/stylus.html) is one such tool. [Example](https://github.com/sindresorhus/refined-github/issues/136#issuecomment-204072018)\n\n2. Clone the repository, make the adjustments you need, and [load the unpacked extension in Chrome](https://developer.chrome.com/extensions/getstarted#unpacked), rather than installing from the Chrome Store.\n\n\n## Contribute\n\nSuggestions and pull requests are highly encouraged!\n\nIn order to make modifications to the extension you\'d need to run it locally.\n\nPlease follow the below steps:\n\n```sh\ngit clone https://github.com/sindresorhus/refined-twitter\ncd refined-twitter\nnpm install    # Install dev dependencies\nnpm run build  # Build the extension code so it\'s ready for the browser\nnpm run watch  # Listen for file changes and automatically rebuild\n```\n\nOnce built, load it in the browser of your choice:\n\n<table>\n\t<tr>\n\t\t<th>Chrome</th>\n\t\t<th>Firefox</th>\n\t</tr>\n\t<tr>\n\t\t<td width="50%">\n\t\t\t<ol>\n\t\t\t\t<li>Open <code>chrome://extensions</code>\n\t\t\t\t<li>Check the <strong>Developer mode</strong> checkbox\n\t\t\t\t<li>Click on the <strong>Load unpacked extension</strong> button\n\t\t\t\t<li>Select the folder <code>refined-twitter/distribution</code>\n\t\t\t</ol>\n\t\t</td>\n\t\t<td width="50%">\n\t\t\t<ol>\n\t\t\t\t<li>Open <code>about:debugging#addons</code>\n\t\t\t\t<li>Click on the <strong>Load Temporary Add-on</strong> button\n\t\t\t\t<li>Select the file <code>refined-twitter/extension/manifest.json</code>\n\t\t\t</ol>\n\t\t</td>\n\t</tr>\n</table>\n\n\n## FAQ\n\n#### Don\'t you have another extension with the same name?\n\nThis is reusing the name from the [original Refined Twitter](https://github.com/sindresorhus/refined-twitter-old) extension, which tried to use the mobile Twitter version on the desktop.
