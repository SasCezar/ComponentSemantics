Runs on your computer. Requires Java 8.\nRipMe is a cross-platform tool. It has been tested and confirmed working on Windows, Linux and MacOS. \n\n![Screenshot](https://i.imgur.com/UCQNjeg.png)\n\n## [Downloads](https://github.com/ripmeapp/ripme/releases)\n\nDownload `ripme.jar` from the [latest release](https://github.com/ripmeapp/ripme/releases).\n\n**Note: If you're currently using version 1.2.x, 1.3.x or 1.7.49, you will not automatically get updates to the newest versions. We recommend downloading the latest version from the link above.**\n\nFor information about running the `.jar` file, see [the How To Run wiki](https://github.com/ripmeapp/ripme/wiki/How-To-Run-RipMe).\n\n## [Changelog](https://github.com/ripmeapp/ripme/blob/master/ripme.json) (ripme.json)\n\n# Features\n\n* Quickly downloads all images in an online album (see supported sites below)\n* Easily re-rip albums to fetch new content\n* Built in updater\n* Skips already downloaded images by default\n* Can auto skip e-hentai and nhentai albums containing certain tags [See here for how to enable](https://github.com/RipMeApp/ripme/wiki/Config-options#nhentaiblacklisttags)\n* Download a range of urls [See here for how](https://github.com/RipMeApp/ripme/wiki/How-To-Run-RipMe#downloading-a-url-range)\n\n## [List of Supported Sites](https://github.com/ripmeapp/ripme/wiki/Supported-Sites)\n\n* imgur\n* twitter\n* tumblr\n* instagram\n* flickr\n* photobucket\n* reddit\n* gonewild\n* motherless\n* imagefap\n* imagearn\n* seenive\n* vinebox\n* 8muses\n* deviantart\n* xhamster\n* (more)\n\n## Not Supported?\n\nRequest support for more sites by adding a comment to [this Github issue](https://github.com/RipMeApp/ripme/issues/38).\n\nIf you're a developer, you can add your own Ripper by following the wiki guide\n[How To Create A Ripper for HTML Websites](https://github.com/ripmeapp/ripme/wiki/How-To-Create-A-Ripper-for-HTML-websites).\n\n# Compiling & Building\n\nThe project uses [Maven](http://maven.apache.org/).\nTo build the .jar file using Maven, navigate to the root project directory and run:\n\n```bash\nmvn clean compile assembly:single\n```\n\nThis will include all dependencies in the JAR.\n\n# Running Tests\n\nAfter building you can run tests by running the following:\n\n```bash\nmvn test\n```\n\nPlease note that some tests may fail as sites change and our rippers become out of date.\nStart by building and testing a released version of RipMe\nand then ensure that any changes you make do not cause more tests to break.\n"
