[Read the blog post here](https://blog.twitter.com/developer/en_us/topics/tools/2018/discontinuing-support-for-twitter-kit-sdk.html).**\n\n# Twitter Kit for Android\n\nTwitter Kit is a multi-module gradle project containing several Twitter SDKs including TweetComposer, TwitterCore, and TweetUi. Twitter Kit is designed to make interacting with Twitter seamless and efficient.\n\n## Twitter Kit Features\n\n* Display Tweets and timelines\n* Compose Tweets\n* Monetize with MoPub integration\n* Log in with Twitter\n* Access the Twitter API\n\n## Getting Started\n\n* Generate your Twitter API keys through the [Twitter developer apps dashboard](https://apps.twitter.com/).\n* Install Twitter Kit using instructions below.\n* For extensive documentation, please see the [wiki](https://github.com/twitter/twitter-kit-android/wiki).\n\n### Install using Bintray JCenter\n\nAdd twitter dependency to your build.gradle:\n```groovy\n\nrepositories {\n  jcenter()\n}\n\ndependencies {\n  compile('com.twitter.sdk.android:twitter:3.3.0@aar') {\n    transitive = true\n  }\n}\n\n```\n\n### Building from source\n\nRename samples/app/twitter.properties.sample to samples/app/twitter.properties and populate the consumer key and secret.\n\nTo build the entire project run\n\n```\n./gradlew assemble\n```\n\nRun all automated tests on device to verify.\n\n```\n./gradlew test connectedCheck\n```\n\nTo run the sample app\n\n```\n./gradlew :samples:app:installDebug\n```\n\n\n## Contributing\n\nThe master branch of this repository contains the latest stable release of Twitter Kit.
