b'# Regular Expressions to Match Social Media Profiles\nThis repository lists regular expressions to match and extract URLs of social media profiles. For simplicity URLs have to be filtered beforehand, i.e. the regular expression has to be executed on a single URL. To do this, you could use a regular expression matching valid URLs or use a specific library, for example JSOUP for Java. Furthermore, only exact profile urls are matched (for example no profile urls inside a url referencing a post).\n\nPlease note: I\'ve also created a [Python library called socials](https://github.com/lorey/socials) that uses these expressions to *automate url detection* and extraction.\n\n## Twitter\n    http(s)?:\\/\\/(.*\\.)?twitter\\.com\\/[A-z0-9_]+\\/?\nAllowed for usernames are alphanumeric characters and underscores. \n\n### Verification\nSend request to page and check for username in answer (rate limit?)\n\n## Github\n    http(s)?:\\/\\/(www\\.)?github\\.com\\/[A-z0-9_-]+\\/?\nExclude subdomains as these redirect to github pages sometimes.\n\n    http(s)?:\\/\\/([A-z0-9-_]+)\\.github\\.(com|io)\\/?\nRegex for pages like someuser.github.io.\n\n### Verification\nUse https://api.github.com/users/{user_login} (60 requests/hour unauthenticated)\n\n## Linkedin\n    http(s)?:\\/\\/([\\w]+\\.)?linkedin\\.com\\/in\\/[A-z0-9_-]+\\/?\nRegEx for public URLs.\n\n    http(s)?:\\/\\/([\\w]+\\.)?linkedin\\.com\\/pub\\/[A-z0-9_-]+(\\/[A-z0-9]+){3}\\/?\nMatches public profiles that need three keys(?) after the actual name.\n\n    http(s)?:\\/\\/([\\w]+\\.)?linkedin\\.com\\/recruiter\\/[A-z0-9_-]+\\/?\nRegEx for Linkedin Recruiter URLs.\n\n### Verification\nCheck page for profile specific html (rate limit?)\n\n## Facebook\n    http(s)?:\\/\\/(www\\.)?(facebook|fb)\\.com\\/[A-z0-9_\\-\\.]+\\/?\nMatches facebook.com and fb.com (shortlink).\n\n### Verification\nSince Facebook redirects these URLs to all kinds of objects (user, pages, events, and so on), you have to verifiy that it\'s actually a user. See https://developers.facebook.com/docs/graph-api/reference/profile\n\n#### API\nhttp://graph.facebook.com/v2.3/{{username}} gives the following result, if the user exists:\n\n    {\n        "error": {\n        "message": "(#803) Cannot query users by their username ({{specified username}})",\n        "type": "OAuthException",\n        "code": 803\n        }\n    }\n\n#### GET request\nA GET request on an existing (but hidden) user with a randomly added point seems to redirect to the real username.\n## Instagram\n    https?:\\/\\/(www\\.)?instagram\\.com\\/([A-Za-z0-9_](?:(?:[A-Za-z0-9_]|(?:\\.(?!\\.))){0,28}(?:[A-Za-z0-9_]))?)\n\nThe rules:\n * Matches with one . in them disco.dude but not two .. disco..dude\n * Ending period not matched discodude.\n * Match underscores _ _disco__dude_\n * Max characters of 30 1234567890123456789012345678901234567890\n\n## Google Plus\n    https?:\\/\\/plus\\.google\\.com\\/\\+[^/]+|\\d{21}\n\nMatches username or profile numbers up to 21 digits.\n\n## Skype\n    (?:(?:callto|skype):)(?:[a-z][a-z0-9\\\\.,\\\\-_]{5,31})(?:\\\\?(?:add|call|chat|sendfile|userinfo))?\n\nMatches Skype\'s URLs to add contact, call, chat.\n## Telegram\n    https?:\\/\\/(t(elegram)?\\.me|telegram\\.org)\\/([a-z0-9\\_]{5,32})\\/?\nMatches for t.me, telegram.me and telegram.org\n\n## TODO\n* Verification checks (ideas first and maybe scripts at a later point)\n\nI plan on adding the following social media profiles at the moment.\n* xing\n* pinterest\n* vimeo\n* wordpress\n* medium\n* youtube\n* stackoverflow\n* bitbucket\n \nFeel free to add any social media site you would like to find here!\n'
