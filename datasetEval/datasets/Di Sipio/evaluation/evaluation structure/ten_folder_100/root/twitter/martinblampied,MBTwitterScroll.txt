b'# MBTwitterScroll\n==========\nRecreate Twitter\'s profile page scrolling animation for UITableView and UIScrollViews. Librarys uses Core graphics framework and is based of [thinkandbuild.it\'s swift tutorial](http://www.thinkandbuild.it/implementing-the-twitter-ios-app-ui/) converted into Obj-C.\n\n![alt tag](http://i.imgur.com/iz7BIxt.gif?1)\n\n[![Twitter: @martinblampied](https://img.shields.io/badge/contact-@martinblampied-blue.svg?style=flat)](https://twitter.com/martinblampied)\n[![Version](https://img.shields.io/cocoapods/v/MBTwitterScroll.svg?style=flat)](http://cocoadocs.org/docsets/MBTwitterScroll)\n[![License](https://img.shields.io/cocoapods/l/MBTwitterScroll.svg?style=flat)](http://cocoadocs.org/docsets/MBTwitterScroll)\n[![Platform](https://img.shields.io/cocoapods/p/MBTwitterScroll.svg?style=flat)](http://cocoadocs.org/docsets/MBTwitterScroll)\n\n# Installation\n## From CocoaPods\nMBTwitterScroll is available through [CocoaPods](http://cocoapods.org). To install\nit, simply add the following line to your Podfile:\n\n    pod "MBTwitterScroll"\n\n## Manually\nCopy the source files MBTwitterScroll folder into your project.\n\n\n# Usage\n\nImport the project\n\n```objective-c\n   #import "MBTwitterScroll.h"\n```\n\n## UITableView\n\nTo show animation on UITableVIew use the following code:\n\n```objective-c\n    MBTwitterScroll *myTableView = [[MBTwitterScroll alloc]\n                                    initTableViewWithBackgound:[UIImage imageNamed:@"your image"]\n                                    avatarImage:[UIImage imageNamed:@"your avatar"]\n                                    titleString:@"Main title"\n                                    subtitleString:@"Sub title"\n                                    buttonTitle:@"Follow"];  // Set nil for no button\n    myTableView.delegate = self;\n    [self.view addSubview:myTableView];\n```\n\n## UIScrollView\n\nTo show animation on UIScrollView use the following code:\n\n```objective-c\n   MBTwitterScroll *myScrollView = [[MBTwitterScroll alloc]\n                                      initScrollViewWithBackgound:nil\n                                      avatarImage:[UIImage imageNamed:@"avatar.png"]\n                                      titleString:@"Main title"\n                                      subtitleString:@"Sub title"\n                                      buttonTitle:@"Follow" // // Set nil for no button\n                                      contentHeight:2000];\n    myScrollView.delegate = self;\n    [self.view addSubview:myScrollView];\n```\n\n# License\nMBTwitterScroll is free for all use available under the MIT license. See the LICENSE file for more information.\n\n# Feedback or feature request?\nAny problems or features contact me at [@martinblampied](https://twitter.com/martinblampied)\n'
