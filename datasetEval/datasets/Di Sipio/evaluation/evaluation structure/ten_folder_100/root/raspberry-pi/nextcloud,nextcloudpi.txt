--name nextcloudpi ownyourbits/nextcloudpi $DOMAIN\n```\n\n\n## How to build\n\nInstall git, docker, qemu-user-static, chroot and all the usual building tools.\n\n```\ngit clone order to generate the Docker images\n\n```\ngit clone https://github.com/nextcloud/nextcloudpi.git\ncd nextcloudpi\n./build-docker.sh x86\n./build-docker.sh armhf\n./build-docker.sh arm64\n```\n\nNextCloudPi can be installed in any architecture running the latest Debian\n\n```\n# curl -sSL https://raw.githubusercontent.com/nextcloud/nextcloudpi/master/install.sh | bash\n```\n\n##
