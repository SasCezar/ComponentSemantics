b"\xef\xbb\xbf# WOA Deployer for Raspberry Pi 3\n\nThe application to install Full Windows 10 into your Raspberry Pi!\n\n![image](Docs/Screenshot.png)\n\n# **Super easy to use. No-hassle.**\n\nPlease keep reading carefully. All you need is here.\n\n# Requirements\n- Raspberry Pi 3 Model B (or B+)\n- MicroSD card. Recommended with A1 rating.\n- A Windows 10 ARM64 Image (.wim). Please, check [this link](https://github.com/WOA-Project/guides/blob/master/GettingWOA.md) to get it.\n\n### Additional requirements\n- To run WOA Deployer you need a recent version of Windows 10 (please, use the latest to ensure it'll run correctly, don't open issues otherwise)\n- .NET Framework 4.6.1 (should come included in recent versions of Windows)\n\n# Download it\n\nDownload the **[latest version](https://github.com/WOA-Project/WOA-Deployer-Rpi/releases/latest)** \n\n# Executing the tool\n1. Extract the .zip to a folder in your PC\n2. Navigate to the GUI folder\n3. Find the .exe file\n4. Run it\n\n# Show the love \xf0\x9f\xa7\xa1\n\nDo you like my tool?
