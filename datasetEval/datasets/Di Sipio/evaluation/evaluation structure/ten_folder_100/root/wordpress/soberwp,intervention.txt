b"# Intervention\n\nWordPress plugin containing modules to cleanup and customize wp-admin. Let's get rid of that admin panel we love so much.\n\n## Installation\n\n#### Composer:\n\nRecommended method/s;\n\n[Roots Bedrock](https://roots.io/bedrock/) and [WP-CLI](http://wp-cli.org/)\n```shell\n$ composer require soberwp/intervention\n$ wp plugin activate intervention\n```\n\n[Roots Sage](https://roots.io/sage/)\n```shell\n$ composer require soberwp/intervention:1.2.0-p\n```\n\n#### Manual:\n\n* Download the [zip file](https://github.com/soberwp/intervention/archive/master.zip)\n* Unzip to your sites plugin folder\n* Activate via WordPress\n\n#### WP-CLI:\n\n```shell\n$ wp plugin install https://github.com/soberwp/intervention/archive/master.zip --activate\n```\n\n#### Requirements:\n\n* [PHP](http://php.net/manual/en/install.php) >= 5.6.x\n\n## Usage\n\n#### Import the namespaced function:\n\nPlace at the top of your themes functions.php file.\n```php\n\nuse function \\Sober\\Intervention\\intervention;\n\nif (function_exists('\\Sober\\Intervention\\intervention')) {\n    // now you can use the function to call the required modules and their params\n    intervention('remove-menu-items', 'plugins', 'all');\n}\n```\n\n## Modules\n\nUse function `intervention()` to use plugin modules.\n\n[Example functions.php file](.github/functions.php.md)\n\n```php\nintervention('remove-menu-items', ['themes', 'plugins'], ['editor', 'author']);\n```\n\n**Click on a module below to view its usage documentation:**\n\n*
