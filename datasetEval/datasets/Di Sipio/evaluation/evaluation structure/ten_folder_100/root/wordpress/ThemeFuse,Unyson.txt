b'# [Unyson](https://wordpress.org/plugins/unyson/) Framework [ ![Download](https://rawgit.com/ThemeFuse/Unyson/master/.github/img/download-stable-version.svg) ](https://github.com/ThemeFuse/Unyson/releases/latest)\n\n[Unyson](http://unyson.io/) is a framework for [WordPress](http://wordpress.org/) that facilitates the development of a theme.\n\nThis framework was created from the ground up by the team behind [ThemeFuse](http://themefuse.com/) from the desire to empower developers to build outstanding WordPress themes fast and easy.\n\n## Table of contents\n\n* [Installation](#installation).\n* [Documentation](#documentation).\n* [Extensions](#extensions).\n* [Contributing](#contributing).\n* [Bug reports](#bug-reports).\n* [Copyright and license](#copyright-and-license).\n\n## Installation\n\n1. Open the WordPress dashboard **Plugins > Add New** menu.\n2. Type **Unyson** in the search field.\n3. Press the **Install** button and **activate** the plugin.\n4. Configure the plugin by going to the **Unyson menu**.\n\n## Documentation\n\nUnyson\'s documentation is available on http://manual.unyson.io/.\n\nYou can help us improve the documentation by Check the [Issues page](https://github.com/ThemeFuse/Unyson/issues).\n\nIf your issue has already been reported, great! It will be reviewed in an upcoming release.\n\n#### Submitting a Bug Report\n\nYou can report the issue via [Issues page](https://github.com/ThemeFuse/Unyson/issues).\nA good bug report includes full details to easily understand the issue you are having and the exact steps to reproduce if it is a bug.\n\n## Copyright and license\n\nCode and documentation copyright 2014 ThemeFuse LTD.
