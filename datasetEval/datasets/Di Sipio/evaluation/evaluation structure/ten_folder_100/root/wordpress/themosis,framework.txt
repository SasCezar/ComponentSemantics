b'Themosis framework\n==================\n\n[![Build Status](https://travis-ci.com/themosis/framework.svg?branch=develop)](https://travis-ci.com/themosis/framework)\n[![Join the chat at https://gitter.im/themosis/framework](https://badges.gitter.im/themosis/framework.svg)](https://gitter.im/themosis/framework?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)\n\nThe Themosis framework is a WordPress focused framework. It helps you develop enterprise level WordPress applications and websites using an elegant and simple code syntax.\n\nGetting started\n---------------\n\n### Prerequisites\n\nIn order to start a new installation of the Themosis framework, you need to have all [WordPress requirements](https://wordpress.org/about/requirements/) and [Composer](https://getcomposer.org/) installed on your computer.\n\n> We recommend to install Composer globally on your computer.\n\n### Installing\n\nOnce you have Composer setup, run the following command in your Terminal or Console in order to install a fresh copy of the Themosis framework:\n\n```bash\ncomposer create-project themosis/themosis myproject\n```\n\nThe above command will install a new instance of the Themosis framework under the `myproject` directory.\n\n### Learn more\n\nIn order to learn more about how to use the framework and its features, please follow the official documentation and read the "Getting started" guide:\n\n- [Official documentation](https://framework.themosis.com)\n\nLocal development\n-----------------\n\nThe framework is by default bundled with the [Homestead](https://laravel.com/docs/homestead) virtual machine for local development. Read the Themosis framework [installation guide](https://framework.themosis.com/docs/2.0/installation/) for details on how to configure your local Homestead machine for your project.\n\nContributing\n------------\nAny help is appreciated. The project is open-source and we encourage you to participate. You can contribute to the project in multiple ways by:\n\n- Reporting a bug issue\n- Suggesting features\n- Sending a pull request with code fix or feature\n- Following the project on [GitHub](https://github.com/themosis/framework)\n- Following us on [Twitter](https://twitter.com/Themosis)\n- Sharing the project around your community\n\nFor details about contributing to the framework, please check the [contribution guide](http://framework.themosis.com/docs/2.0/contributing).\n\nVersioning\n----------\n\nWe use SemVer for versioning. For available versions, please check this repository tags if you need a specific version for your project.\n\nWhen starting from scratch, Composer will always install the most recent stable version.\n\nAuthors\n-------\n\n- **Julien Lamb\xc3\xa9** - *Project Lead* - [jlambe](https://github.com/jlambe)\n\nSee also the list of [contributors](https://github.com/themosis/framework/graphs/contributors) who participated in this project.\n\nLicense\n-------\n\nThis project is licensed under the GPL-2.0+ License - see the [LICENSE.md](https://github.com/themosis/framework/LICENSE.md) file for details.\n'
