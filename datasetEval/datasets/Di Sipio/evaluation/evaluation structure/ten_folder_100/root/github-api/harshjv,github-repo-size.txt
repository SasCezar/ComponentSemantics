b"# \xf0\x9f\x9a\x80 Chrome extension to display repository size on GitHub [![JavaScript Style Guide](https://img.shields.io/badge/code%20style-standard-brightgreen.svg)](http://standardjs.com/) [![Chrome Web Store](https://img.shields.io/chrome-web-store/v/apnjnioapinblneaedefcnopcjepgkci.svg)](https://chrome.google.com/webstore/detail/github-repository-size/apnjnioapinblneaedefcnopcjepgkci) [![Chrome Web Store](https://img.shields.io/chrome-web-store/d/apnjnioapinblneaedefcnopcjepgkci.svg)](https://chrome.google.com/webstore/detail/github-repository-size/apnjnioapinblneaedefcnopcjepgkci)\n\n\nAutomatically adds repository size to GitHub's repository summary.\n\n[![Featured on Product Hunt](./product-hunt.png)](https://www.producthunt.com/tech/github-repository-size) [![Install from chrome web store](https://developer.chrome.com/webstore/images/ChromeWebStore_Badge_v2_340x96.png)](https://chrome.google.com/webstore/detail/github-repository-size/apnjnioapinblneaedefcnopcjepgkci)\n\n\n## Screenshot\n\n![Screenshot of repository size on GitHub](https://raw.githubusercontent.com/harshjv/github-repo-size/master/screenshot.png)\n\n\n## Private Repository\n\nTo enable viewing size of private repositories;\n\n1. Install extension from chrome webstore, if you haven't.\n2. Go to https://github.com/settings/tokens to generate your personal access token.\n  - Check `repo` scope to enable this extension on private repo.\n3. Click on the Github Repo Size extension (this extension)'s icon aside the address bar.\n4. Paste your access token there in the prompt box.\n\n### Temporarily override then token\n\nYou can set `x-github-token` in `localStorage` to your access token, and the extension will use this value even if you've previously set token.\n\n    localStorage.setItem('x-github-token', <YOUR-PERSONAL-ACCESS-TOKEN>)\n\nand then remove it to use previously set token;\n\n    localStorage.removeItem('x-github-token')\n\n\n## Development\n\n1. Clone this repo\n2. Go to chrome extensions [chrome://extensions](chrome://extensions)\n3. Enable developer mode\n4. Click on load unpacked extension and select this cloned repo\n\n\n## License\n\nMIT\n"
