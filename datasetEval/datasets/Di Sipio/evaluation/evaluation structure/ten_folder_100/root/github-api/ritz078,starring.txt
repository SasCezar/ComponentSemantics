b'<p align="center">\n  <img src="./logo.png" align="center" alt="" width="300"/>\n</p>\n\n> Star the packages on GitHub being used in the current project or installed globally.\n\n<p align="center">\n  <img src="./demo.gif" align="center" alt="" width="800"/>\n</p>\n\n## Features\n- [x] Star selected dependencies : `starring`\n- [x] Star all dependencies of the current project: `starring --all`\n- [x] Star a particular package: `starring colors`\n- [x] Star selected global dependencies : `starring --global`\n- [x] Star all global dependencies: `starring --global --all`\n\n## Authorization\nYou will need to provide your GitHub usename and password for the first time so that it can create a token. The scope of the token is `\'public_repo\'`. After that it won\'t ask for any credentials.\n\n## Install\n\n```\n$ npm install --global starring\n```\n\n## Usage Instructions\n```\nUsage\n    $ starring [input]\n\n  Options\n    --all  Star all the packages listed in the package.json from current directory. [Default: false]\n    --global Star\n\n  Examples\n    $ starring\n    presents UI to select the packages you want to star.\n    $ starring <package-name>\n    stars package-name npm package on GitHub.\n```\n\n\n## License\n\nMIT \xc2\xa9 [Ritesh Kumar](https://github.com/ritz078/starring)\n\n<a target=\'_blank\' rel=\'nofollow\' href=\'https://app.codesponsor.io/link/8CBegPnJTnjtddvd2E18Su4F/ritz078/starring\'>  <img alt=\'Sponsor\' width=\'888\' height=\'68\' src=\'https://app.codesponsor.io/embed/8CBegPnJTnjtddvd2E18Su4F/ritz078/starring.svg\' /></a>\n'
