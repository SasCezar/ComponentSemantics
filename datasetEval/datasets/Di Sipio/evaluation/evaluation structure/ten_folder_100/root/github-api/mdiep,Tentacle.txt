b'# Tentacle [![MIT license](https://img.shields.io/badge/license-MIT-lightgrey.svg)](https://raw.githubusercontent.com/mdiep/Tentacle/master/LICENSE.md)\nA Swift framework for the GitHub API\n\n```swift\nlet client  = Client(.dotCom, token: "\xe2\x80\xa6")\nlet repo    = Repository(owner: "ReactiveCocoa", name: "ReactiveCocoa")\nlet request = repo.release(forTag: "tag-name")\nclient\n    .execute(request)\n    .startWithResult { result in\n        switch result {\n        case let .success(response, release):\n            print("Downloaded release: \\(release)")\n        case let .failure(error):\n            print("An error occurred: \\(error)")\n        }\n    }\n```\n\nTentacle is built with [ReactiveSwift](https://github.com/ReactiveCocoa/ReactiveSwift).\n\n## License\nTentacle is available under the [MIT License](LICENSE.md)\n'
