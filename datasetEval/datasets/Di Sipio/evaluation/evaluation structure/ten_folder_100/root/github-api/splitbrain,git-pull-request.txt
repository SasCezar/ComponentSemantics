b'git pull-request\n================\n\nAutomatically check out github pull requests into their own branch.\n\nInstallation\n------------\n\nCopy the script to somewhere in your ``PATH`` and make it executable.\n\nSet github.token to your github OAUTH token\n    i.e. git config --global github.token OAUTH-TOKEN\n\nIf you need Python 3 support be sure to use the ``python3`` branch.\n\nUsage\n-----\n\n    git pull-request <OPTIONS> [<pull request number>]\n\nOptions\n-------\n\n    -h, --help\n        Display this message and exit\n\n    -r <repo>, --repo <repo>\n        Use this github repo instead of the \'remote origin\' or \'github.repo\'\n        git config settings.
