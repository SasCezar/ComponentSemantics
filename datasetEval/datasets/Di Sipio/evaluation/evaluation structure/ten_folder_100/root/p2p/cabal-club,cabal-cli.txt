b"# cabal-cli\n\n> Terminal client for cabal, the p2p chat platform.\n\nSee [cabal-core](https://github.com/cabal-club/cabal-core) for the underlying\ndatabase & api.\n\n![](cli-2019-04.png)\n\nchat with us:  \n`npx cabal cabal://cabal-club.github.io`\n\n## Installation\n\n```\n$ npm install --global cabal\n$ cabal --new \n```\n\n## Usage\n#### Start a new instance:\n```\ncabal --new\n```\n\n#### Connect to an existing instance:\n```\ncabal <key>\n```\ne.g.\n```\ncabal cabal://0201400f1aa2e3076a3f17f4521b2cc41e258c446cdaa44742afe6e1b9fd5f82\n```\n\n#### Scan a QR code to join a cabal:\nCabal can use a webcam connected to your computer to read a cabal key from a QR code.\nFor this to work, you'll need to install an additional system dependency:\n- Linux: `sudo apt-get install fswebcam`\n- MacOS: `brew install imagesnap`\n```\n# Hold up your QR code in front of the webcam and then run:\ncabal --qr\n```\n\n#### Headless mode\n\nThis will run cabal without a UI. You can use this to seed a cabal (e.g. on a VPS) and make its data more available:\n```\ncabal <key> --seed\n```\n\n## Commands\n```py\n/add, /cabal\n  add a cabal\n/new\n  create a new cabal\n/nick, /n\n  change your display name\n/emote, /me\n  write an old-school text emote\n/names\n  display the names of the currently online peers\n/channels\n  display the cabal's channels\n/panes\n  set pane to navigate up and down in panes: channels, cabals\n/join, /j\n  join a new channel\n/leave, /l\n  leave a channel\n/clear\n  clear the current backscroll\n/help\n  display this help message\n/qr\n  generate a qr code with the current cabal's address\n/quit, /exit\n  exit the cabal process\n/topic, /motd\n  set the topic/description/message of the day for a channel\n/whoami, /key\n  display your local user key\n/whois\n  display the public keys associated with the passed in nick\n\nalt-n\n  move between channels/cabals panes\nctrl+{n,p}\n  move up/down channels/cabals\n```\n\n## Hotkeys\n`ctrl+u`  \n&nbsp;&nbsp;&nbsp;&nbsp;clear input line  \n`ctrl+w`  \n&nbsp;&nbsp;&nbsp;&nbsp;delete last word in input  \n`up-arrow`  \n&nbsp;&nbsp;&nbsp;&nbsp;cycle through command history  \n`down-arrow`  \n&nbsp;&nbsp;&nbsp;&nbsp;cycle through command history  \n`home`  \n&nbsp;&nbsp;&nbsp;&nbsp;go to start of input line  \n`end`  \n&nbsp;&nbsp;&nbsp;&nbsp;go to end of input line  \n`ctrl+n`  \n&nbsp;&nbsp;&nbsp;&nbsp;go to next channel  \n`ctrl+p`  \n&nbsp;&nbsp;&nbsp;&nbsp;go to previous channel  \n`pageup`  \n&nbsp;&nbsp;&nbsp;&nbsp;scroll up through backlog  \n`pagedown`  \n&nbsp;&nbsp;&nbsp;&nbsp;scroll down through backlog  \n`alt-[1,9]`  \n&nbsp;&nbsp;&nbsp;&nbsp;select channels  1-9  \n`alt-n`  \n&nbsp;&nbsp;&nbsp;&nbsp;tab between the cabals & channels panes \n"
