It was the first project to implement the Zerocoin protocol and has now transitioned to the [Sigma protocol](https://zcoin.io/what-is-sigma-and-why-is-it-replacing-zerocoin-in-zcoin/) which has no trusted setup and small proof sizes. Zcoin also utilises [Dandelion++](https://arxiv.org/abs/1805.11060) to obscure the originating IP of transactions without relying on any external services such as Tor/i2P.\n\nZcoin developed and utilizes [Merkle Tree Proofs (MTP)](https://arxiv.org/pdf/1606.03588.pdf) as its Proof-of-Work algorithm which aims to be memory hard with fast verification.\n\nHow Zcoin\xe2\x80\x99s Privacy Technology Compares to the Competition\n--------------\n![A comparison chart of Zcoin\xe2\x80\x99s solutions with other leading privacy technologies can be found below](https://zcoin.io/wp-content/uploads/2019/04/zcoin_table_coloured5-01.png) \nread more https://zcoin.io/zcoins-privacy-technology-compares-competition/\n\nRunning with Docker\n===================\n\nIf you are already familiar with Docker, then running Zcoin with Docker might be the the easier method for you. To run Zcoin using this method, first install [Docker](https://store.docker.com/search?type=edition&offering=community). After this you may\ncontinue with the following instructions.\n\nPlease note that we currently don't support the GUI when running with Docker. Therefore, you can only use RPC (via HTTP or the `zcoin-cli` utility) to interact with Zcoin via this method.\n\nPull our latest official Docker image:\n\n```sh\ndocker pull zcoinofficial/zcoind\n```\n\nStart Zcoin daemon:\n\n```sh\ndocker run --detach --name zcoind zcoinofficial/zcoind\n```\n\nView current block count (this might take a while since the daemon needs to find other nodes and download blocks first):\n\n```sh\ndocker exec zcoind zcoin-cli getblockcount\n```\n\nView connected nodes:\n\n```sh\ndocker exec zcoind zcoin-cli getpeerinfo\n```\n\nStop daemon:\n\n```sh\ndocker stop zcoind\n```\n\nBackup wallet:\n\n```sh\ndocker cp zcoind:/home/zcoind/.zcoin/wallet.dat .\n```\n\nStart daemon again:\n\n```sh\ndocker start zcoind\n```\n\nLinux Build Instructions and Notes\n==================================\n\nDependencies\n----------------------\n1.
