Search in other Rats clients.\n* Web version (web interface) for servers\n* Top list (mostly common and popular torrents)\n* Feed list (Rats clients activity feed)\n* Translations: English, Russian, Ukrainian, Chinese\n* Drag and drop torrents (expand local search database with specific torrents)\n* Descriptions association from trackers\n\n## Architecture\n![Basic Architecture](docs/img/ratsarch.png)\n\n## Contributing\nWe are welcome any contributing: bug fixes, improvements, code refactoring and other stuffs.\n\n[Translation guide](docs/TRANSLATION.md)\n\n[Own strategy / Other tracker support](docs/TRACKERS.md)\n\n## Usage Manuals\n* [English](docs/USAGE.md)\n* [Russian](docs/USAGE.RU.md)\n\n## Server version install\nClone repository, then make sure that you have Node.js and NPM (with Node.js) installed in your system, then:\n\n```bash\nnpm install\n```\n\nThen compile web version:\n\n```bash\nnpm run buildweb\n```\n\nFinally start server application:\n\n```bash\nnpm run server\n```\n\nNow you can get access to web interface on 8095 port: http://localhost:8095\n\n[More about configuration](docs/SERVER.md)\n\n[More about server compatibility and known issues](docs/SERVER_COMPATIBILITY.md)\n\n## Donation\n\n[**Support Rats Search development on Patreon**](https://www.patreon.com/bePatron?c=1722368)\n\n[**Support Rats Search via PayPal**](https://www.paypal.com/cgi-bin/webscr?cmd=_donations&business=vasiliy.kosyanchuk@gmail.com&lc=US&item_name=Rats+Search+Development+Support&no_note=0&cn=&currency_code=USD&bn=PP-DonationsBF:btn_donateCC_LG.gif:NonHosted)\n\n**Support Rats Search via Yandex.Money**: 410012059502693\n\n## Contacts / Support\n\n- Twitter: [@RatsSearch](https://twitter.com/RatsSearch)\n- [Discord (Support)](https://discord.gg/t9GQtxA)\n\n## Screenshots\n\n![Main Window](docs/img/screen_1.png)\n\n## License\n[MIT](https://github.com/DEgiTx/rats-search/blob/master/LICENSE)\n'
