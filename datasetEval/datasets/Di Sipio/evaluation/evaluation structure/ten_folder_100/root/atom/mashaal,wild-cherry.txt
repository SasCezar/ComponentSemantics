b'# Wild Cherry\n\n> A fairy-tale inspired theme, with tasteful use of emojis - [Zsh](http://www.zsh.org/), [iTerm](http://www.iterm2.com/), [Sublime](http://www.sublimetext.com), [Atom](https://atom.io/),  [Slack](http://slack.com/), [Mou](http://25.io/mou/), [PuTTY](http://www.putty.org), [Notepad++](https://notepad-plus-plus.org), [Vivaldi](https://vivaldi.com) & [DuckDuckGo](https://duckduckgo.com)\n\n![Wild Cherry](https://raw.githubusercontent.com/mashaal/wild-cherry/master/images/wild-cherry.jpg)\n\n> :skull: Git Princess is close to death.\n\n> :tulip: Commit to her and give her a flower.\n\n> :japanese_ogre: Japanese Ogre appears on error.\n\n![Wild Cherry](https://raw.githubusercontent.com/mashaal/wild-cherry/master/images/wild-cherry.gif)\n\n### Git Emojis Explained\n\n> :skull: Dirty\n\n> :ribbon: New file\n\n> :boom: Removed file\n\n> :mushroom: Changed file\n\n> :nail_care: New files added/ ready to commit\n\n> :tulip: Clean\n\n> Plus many more!\n\n## Zsh\n\n* This is a customized version of [agnoster](https://gist.github.com/3712874) and [remy](https://remysharp.com/2013/07/25/my-terminal-setup).\n* In order for this theme to render correctly, you will need a [Powerline-patched font](https://github.com/powerline/fonts) or the [Hack font](https://github.com/source-foundry/Hack)\n\n**Activating...**\n\n1. Move [`zsh/wild-cherry.zsh-theme`](zsh/wild-cherry.zsh-theme) file to [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh/)\'s theme folder: `oh-my-zsh/themes/wild-cherry`.\n2. Go to your `~/.zshrc` file and set `ZSH_THEME="wild-cherry"`. Finally, reload your terminal.\n\n[Zsh theme documentation](https://github.com/robbyrussell/oh-my-zsh/wiki/themes)\n\n## iTerm\n\n**Activating...**\n\n1. *iTerm2 \xe2\x86\x92 `Preferences` \xe2\x86\x92 `Profiles` \xe2\x86\x92 `Colors` Tab*\n2. Click *`Load Presets...`*\n3. Click *`Import...`*\n4. Select the [`iterm/wild-cherry.itermcolors`](iterm/wild-cherry.itermcolors) file\n5. Select the *`wild cherry`* from *`Load Presets...`*\n\n**Note: wild-cherry.itermcolors reflects only the theme colors in iTerm. To get the formatting and emojis, you will need to configure your Zsh theme via the instructions above.**\n\n\n## Atom\n\n![Atom Preview](https://raw.githubusercontent.com/mashaal/wild-cherry/master/images/atom.png)\n\n**[View on Atom Themes](https://atom.io/themes/wild-cherry)**\n\n* Optimised for Atom!\n* Now with Linter Emojis!\n* Recommended to use [Isotope UI Theme](https://atom.io/themes/isotope-ui), with [File Icons](https://atom.io/packages/file-icons) package\n* Thanks to [ramonmcros](https://github.com/ramonmcros) for adding JSON, Markdown, and CriticMarkdown syntax!\n\n#### Atom Linter Emojis\n\n![Atom Preview](https://raw.githubusercontent.com/mashaal/wild-cherry/master/images/error.png)\n> :japanese_ogre: Japanese Ogre does not like trailing semicolons! ([Atom Linter](https://atom.io/packages/linter) required)\n\n![Atom Preview](https://raw.githubusercontent.com/mashaal/wild-cherry/master/images/fire.png)\n> :fire: Dodgy code may catch fire! ([Atom Linter](https://atom.io/packages/linter) required)\n\n\n## Sublime\n\n![Sublime Preview](https://raw.githubusercontent.com/mashaal/wild-cherry/master/images/sublime.png)\n\n* This is a customized version of the [Cobalt2](https://github.com/wesbos/cobalt2) UI.\n* Massive thanks to [ctf0](https://github.com/ctf0) for his help with the UI! :tophat:\n\n**[View on Package Control](https://packagecontrol.io/packages/Wild%20Cherry)**\n\n1. Open package control `tools` \xe2\x86\x92 `Command Palette` and type `Install Package`\n2. Search for `Wild Cherry` and hit enter\n3. Lastly, open `Preferences` \xe2\x86\x92 `Settings - User`. Add the following two lines:\n\n```\n"theme": "wild-cherry.sublime-theme",\n"color_scheme": "Packages/Wild Cherry/wild-cherry.tmTheme"\n```\n`color_scheme` defines how the code looks and `theme` defines how the sidebar, tabs, search, command palette work.\n\n## Slack\n\n![Slack Preview](https://raw.githubusercontent.com/mashaal/wild-cherry/master/images/slack.png)\n\n**Activating...**\n\n`#2B1F32,#FFB86C,#E15D97,#FFFFFF,#4A1386,#FFFFFF,#35BA66,#0AACC5`\n\n\n## Mou\n\n![Mou Preview](https://raw.githubusercontent.com/mashaal/wild-cherry/master/images/mou.png)\n\n**Activating...**\n\n1. In your Cloud9 IDE, click Cloud9 -> Preferences -> Experimental -> Enable Plugin Manager\n2. Click Tools -> Developer -> Start in Debug mode\n3. In the new window, type the following command into the terminal `cd ~/.c9/plugins`.\n4. Then clone the repository: `git clone https://github.com/iSasFTW/wildcherry_c9.git`.\n5. Switch into the newly created directory: `cd wildcherry_c9/`.\n6. Type `c9 build`.\n7. In the Plugin Manager (if not already open, go to Cloud9 -> Preferences -> Experimental) enable wildcherry_c9 by ticking the box next to it. (If you don\'t see it, you might have to refresh the plugin list using the button in the gear menu)\n8. You should now see "wildcherry" as an option in View -> Themes.\n9. Close the tab with debug mode open, and refresh your original cloud9 tab.\n10. Done! You\'ve now installed the Wild Cherry editor theme!\n## Future? :crystal_ball:\n\nWant to give the princess a makeover? :lipstick: Fork- and submit a pull request! :ribbon:\n'
