This\nplugin is intended to help you with development in\n[Haskell](http://haskell.org).\n\n**NOTE:** You must install dependencies in addition to\ninstalling the package itself. Refer to documentation site https://atom-haskell.github.io/ for setup and usage instructions.\n\n## IRC Chat\n\nYou can join #atom-haskell on FreeNode, and devs can usually be found there.
