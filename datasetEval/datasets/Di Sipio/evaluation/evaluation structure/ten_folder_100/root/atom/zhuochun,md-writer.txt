Update table of content (`markdown-writer:update-toc`).\n  - Following options (global/inline) are supported:\n    - `depthFrom`, `depthTo`: range of headings to be displayed.\n    - `insertAnchor`: insert TOC with anchor link.\n- **Insert link** (`shift-cmd-k`) and save links for next time.\n  - Insert inline link.\n  - Insert reference link with title. _Use `-` in title field to create an empty title reference link._\n  - Remove link (and its reference) after URL is deleted.\n  - Search and insert published posts by title in blog ([setup*][9be76601]).\n- **Insert footnote** (`markdown-writer:insert-footnote`), and edit footnote labels.\n- **Insert image from file or clipboard** (`shift-cmd-i`) with preview.\n  - Save or copy the image to your blog\'s images directory ([customize][e6198daa]).\n- **Insert table** (`markdown-writer:insert-table`) and **jump to next table cell** (`cmd-j cmd-t`).\n- **Format table** (`markdown-writer:format-table`) with customizable table alignments.\n- **Continue lists and table rows** when press `enter` ([customize][adaa9527]).\n- **Correct ordered list numbers** (`markdown-writer:correct-order-list-numbers`).\n- **Toggle headings**: `ctrl-alt-[1-5]` to switch among `H1` to `H5`.\n- **Toggle text styles** ([customize][7ddaeaf4]):\n  - `code` (`cmd-\'`)\n  - **bold** (`cmd-b`)\n  - _italic_ (`cmd-i`)\n  - ~~strike through~~ (`cmd-h`)\n  - `\'\'\'code block\'\'\'` (`shift-cmd-"`)\n  - `<kbd>key</kbd>` (`cmd + k`)\n  - `- unordered list` (`shift-cmd-U`)\n  - `0. ordered list` (`shift-cmd-O`)\n  - `> blockquote` (`shift-cmd->`)\n  - `- [ ] task list` (`markdown-writer:toggle-task`)\n- **Jumping commands**:\n  - Jump to previous heading (`cmd-j cmd-p`)\n  - Jump to next heading (`cmd-j cmd-n`)\n  - Jump to next table cell (`cmd-j cmd-t`)\n  - Jump to reference marker/definition (`cmd-j cmd-d`)\n- **Folding commands**:\n  - Fold all headings (`markdown-writer:fold-headings`)\n  - Fold and focus current heading (`markdown-writer:focus-current-heading`)\n  - Fold all inline links (`markdown-writer:fold-links`)\n- **Open links**:\n  - **Open link under cursor in browser** (`markdown-writer:open-link-in-browser`).\n  - **Open link under cursor in editor** (`markdown-writer:open-link-in-file`).\n- **Markdown cheat sheet** (`markdown-writer:open-cheat-sheet`).\n\n[adaa9527]: https://github.com/zhuochun/md-writer/wiki/Settings#use-different-unordered-list-styles "Customizations"\n[e6198daa]: https://github.com/zhuochun/md-writer/wiki/Settings#change-image-directory "Change Image Directory"\n\n### CriticMarkup\n\nFollowing [CriticMarkup][f99bc01e] commands are supported:\n\n- Addition `{++ ++}` (`markdown-writer:toggle-addition-text`)\n- Deletion `{-- --}` (`markdown-writer:toggle-deletion-text`)\n- Substitution `{~~ ~> ~~}` (`markdown-writer:toggle-substitution-text`)\n- Comment `{>> <<}` (`markdown-writer:toggle-comment-text`)\n- Highlight `{== ==}{>> <<}` (`markdown-writer:toggle-highlight-text`)\n\n[f99bc01e]: http://criticmarkup.com/users-guide.php "CriticMarkup"\n\n### AsciiDoc\n\n[AsciiDoc][0e2299b8] is supported with [language-asciidoc][2f0cb1f9] installed.\n\n[0e2299b8]: https://asciidoctor.org/docs/asciidoc-syntax-quick-reference/ "AsciiDoc Quick Reference"\n[2f0cb1f9]: https://atom.io/packages/language-asciidoc "AsciiDoc Language Package for Atom"\n\n### Plugins\n\n**Toolbar for Markdown Writer** is available at [tool-bar-markdown-writer][82a2aced].\n\n[82a2aced]: https://atom.io/packages/tool-bar-markdown-writer "Toobar for Markdown Writer"\n\n## Installation\n\n- In Atom, go to Settings (`cmd-,`) -> Install -> Search `Markdown Writer`.\n- Or, run `apm install markdown-writer`.\n\nAfter installation, you can find and trigger any feature through:\n\n- Open Command Palette (`shift-cmd-P`), enter `Markdown Writer` and find all the commands\n- Or, go to menu `Packages -> Markdown Writer`.\n\n## Setup\n\nExecute command `Markdown Writer: Create Default keymaps` to add the recommended keymaps to your configs. You can modify them based on your needs ([wiki][31ebd53f]).\n\n- `Enter`, `Tab`, `Shift-Tab` are registered by default. You can disable them in _Package\'s Settings > Keybindings_.\n\nConfigure your Package\'s Settings. Menu: _File -> Settings (`cmd-,`) -> Packages -> Markdown-Writer -> Settings_.\n\n- To **manage tags/categories in front matter**, follow this [setup][35eb9cc2].\n- To **manage all/advanced configurations (e.g. project specific settings, change italic text styles)**, follow this [setup][7ddaeaf4].\n\n[31ebd53f]: https://github.com/zhuochun/md-writer/wiki/Settings-for-Keymaps "Settings for Keymaps"\n[3ecd2daa]: https://github.com/atom/settings-view/issues/356 "Viewing a package\'s settings should activate it"\n[35eb9cc2]: https://github.com/zhuochun/md-writer/wiki/Settings-for-Front-Matters "Settings for Front Matters"\n[7ddaeaf4]: https://github.com/zhuochun/md-writer/wiki/Settings "Settings"\n\n## Contributing\n\nYour contributions are really appreciated.
