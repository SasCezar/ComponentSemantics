|\n\n# Brief Intro of Convertors\n\n## Open Neural Network Exchange\n\n**General framework for converting between all kinds of neural networks**\n\n[ONNX](https://github.com/onnx) is an effort to unify converters for neural networks in order to bring some sanity to the NN world. Released by Facebook and Microsoft.\nMore info [here](http://onnx.ai).\n\n## MMdnn\n\n[MMdnn](https://github.com/Microsoft/MMdnn) is a set of tools to help users inter-operate among different deep learning frameworks. E.g. model conversion and visualization. Convert models between CaffeEmit, CNTK, CoreML, Keras, MXNet, ONNX, PyTorch and TensorFlow.\n\n<img src="https://github.com/Microsoft/MMdnn/blob/master/docs/supported.jpg" width="400" height="400">\n\n## MXNet convertor\n\n**Convert to MXNet model**.\n\n### [mdering/CoreMLZoo: A few models converted from caffe to CoreMLs format](https://github.com/mdering/CoreMLZoo)\nA few deep learning models converted from various formats to CoreMLs format.\nModels currently available:\n\n- SqueezeNet\n- VGG19\nPlease feel free to create a pull request with additional models.\n\n\n### [mxnet/tools/caffe_converter](https://github.com/dmlc/mxnet/tree/master/tools/caffe_converter)\n\nKey topics covered include the following:\n* Converting Caffe trained models to MXNet\n* Calling Caffe operators in MXNet\nMore concretly, please refer to [mxnet/tools/caffe_converter](https://github.com/dmlc/mxnet/tree/master/tools/caffe_converter) and [How to | Convert from Caffe to MXNet](https://github.com/dmlc/mxnet/blob/master/docs/how_to/caffe.md).\n\n### [nicklhy/ResNet_caffe2mxnet](https://github.com/nicklhy/ResNet_caffe2mxnet)\n\nThis is a tool to convert the deep-residual-networks from caffe model to mxnet model. The weights are directly copied from caffe network blobs.\n\n## Caffe convertor\n\n**Convert to Caffe model**.\n\n### [pytorch2caffe](https://github.com/longcw/pytorch2caffe)\n\nConvert PyTorch model to Caffemodel.\n\n### [cypw/MXNet2Caffe](https://github.com/cypw/MXNet2Caffe)\n\nConvert MXNet model to Caffe model.\n\n### [wranglerwong/Mxnet2Caffe: Convert MXNet model to Caffe model](https://github.com/wranglerwong/Mxnet2Caffe)\n\nConvert MXNet model to Caffe model.\n\n### [kuangliu/mocha](https://github.com/kuangliu/mocha)\n\nConvert torch model to/from caffe model easily.\n\n### [uhfband/keras2caffe: Keras to Caffe model converter tool](https://github.com/uhfband/keras2caffe)\n\n\nThis tool tested with Caffe 1.0, Keras 2.1.2 and TensorFlow 1.4.0\n\nWorking conversion examples:\n- Inception V3\n- Inception V4 (https://github.com/kentsommer/keras-inceptionV4)\n- Xception V1\n- SqueezeNet (https://github.com/rcmalli/keras-squeezenet)\n- VGG16\n\n\nProblem layers:\n- ZeroPadding2D\n- MaxPooling2D and AveragePooling2D with asymmetric padding\n\n### [facebook/fb-caffe-exts/torch2caffe](https://github.com/facebook/fb-caffe-exts#torch2caffe)\n\nSome handy utility libraries and tools for the Caffe deep learning framework, which has ** A library for converting pre-trained Torch models to the equivalent Caffe models.**\n\n###  [pytorch-caffe-darknet-convert](https://github.com/marvis/pytorch-caffe-darknet-convert)\n\nConvert between pytorch, caffe and darknet models. Caffe darknet models can be load directly by pytorch.\n\n### [Teaonly/trans-torch](https://github.com/Teaonly/trans-torch)\n\nTranslating Torch model to other framework such as Caffe, MxNet ...\n\n### [e-lab/th2caffe](https://github.com/e-lab/th2caffe)\n\nA torch-nn to caffe converter for specific layers.\n\n### [hahnyuan/nn_tools](https://github.com/hahnyuan/nn_tools)\n\na neural network convertor for models among caffe tensorflow keras\n\n## Caffe2 convertor\n\n**Convert to Caffe2 model**.\n\n### [CaffeToCaffe2](https://caffe2.ai/docs/caffe-migration.html#caffe-to-caffe2)\nThis is an official model**.\n\n### [an-kumar/caffe-theano-conversion](https://github.com/an-kumar/caffe-theano-conversion)\n\nThis is part of a project for CS231N at Stanford University, written by Ankit Kumar,
