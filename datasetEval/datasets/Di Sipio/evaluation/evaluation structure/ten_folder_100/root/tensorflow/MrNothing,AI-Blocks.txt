b'# ![AI-Blocks](logo.png)\nA powerful and intuitive WYSIWYG interface that allows anyone to create Machine Learning models!\n\n<img src="https://raw.githubusercontent.com/MrNothing/AI-Blocks/master/ScreenShots/sc5.png" width="800">\n\nThe concept of AI-Blocs is to have a simple scene with draggable objects that have scripts attached to them. The model can be run directly on the editor or be exported to a standalone script that runs on Tensorflow. \n\n<img src="https://raw.githubusercontent.com/MrNothing/AI-Blocks/master/ScreenShots/sc2.png" width="500">\n\nVariables are parsed from python scripts and can be edited from the AI-Blocs properties panel.\n\n<img src="https://raw.githubusercontent.com/MrNothing/AI-Blocks/master/ScreenShots/sc3.png" width="500">\n\n### Install\n\nThe project requires python (https://www.python.org/) and tensorflow (https://www.tensorflow.org/) to run projects. You can still create and edit projects without these dependencies.\n\nTo run AI-Blocs, download the project archive and launch AI-Blocs.exe.\n\nTo run your model simply press the "Play" button and let the magic happen! \n\n<img src="https://raw.githubusercontent.com/MrNothing/AI-Blocks/master/ScreenShots/sc6.png">\n\n### Download\n\nAll releases can be found here: https://github.com/MrNothing/AI-Blocks/releases/\n\n### Build and Run\n\nFirst type:\n```npm install```\n  \nTo run the project from the sources:\n```npm run test```\n  \nTo build the project from the sources:\n```npm run pack```\n\n### Documentation\n\nDocumentation and video tutorials can be found on the website: https://mrnothing.github.io/AI-Blocks/index.html\n\n### License\n\nThe program is distributed under the following license: https://creativecommons.org/licenses/by-nc/3.0/\n\n### Donate \n\nHelp me maintain this project :)\n\n[![Donate](https://img.shields.io/badge/Donate-PayPal-green.svg)](https://www.paypal.com/cgi-bin/webscr?cmd=_donations&business=LGELJCRM28YNW&lc=FR&currency_code=USD&bn=PP%2dDonationsBF%3abtn_donateCC_LG%2egif%3aNonHosted)\n'
