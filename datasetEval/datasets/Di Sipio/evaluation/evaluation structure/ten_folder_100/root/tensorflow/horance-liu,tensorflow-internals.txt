b'# TensorFlow Internals.\n\nIt is open source ebook about TensorFlow kernel and implementation mechanism, including programming model, computation graph, distributed training for machine learning.\n\n## Downloads\n\nYou can obtain full latex source files of TensorFlow Internals from [https://gitlab.com/horance/tensorflow-internals](https://gitlab.com/horance/tensorflow-internals)\n\n## Install TeX distribution\n\nYou should install full texlive packages.\n\n- Ubuntu     \n\n```bash\n$ sudo apt-get install texlive-full\n```\n\n- MacOS\n\nDownload [MacTeX.pkg](http://tug.org/mactex/), and install it.\n\n- Windows\n      \nDownload [TeX Live](http://tug.org/texlive/), and install it (see [insruction](https://liam.page/texlive)).\n\n## Install Missing Fonts\n\nThen you should install some missing chinese fonts. Please download missing fonts from Gitlab.\n\n```\n$ git clone https://gitlab.com/horance/fonts.git\n```\n\nThen install all missing fonts.\n\n- Ubuntu\n\n```bash\n$ sudo cp fonts/* /usr/local/share/fonts\n$ sudo fc-cache\n```\n\n- MacOS \n    \nImport all missing fonts into fontbook, then cache all fonts.\n\n```bash\n$ sudo fc-cache\n```\n\n- Windows\n \nCopy the all missing fonts into C:/WINDOWS/Fonts, then cache all fonts.\n\n```bash\n$ fc-cache\n```\n\n## Build\n\n```bash\n$ make\n```\n\nif you happen to Error begin with `?`, then press `R`(not `r`) and `Enter` to continue.\n\n## Preview PDF\n\n- Mac OSX\n\n```bash\n$ open output/tensorflow-internals.pdf\n```\n\n- Ubuntu\n\n```bash\n$ okular output/tensorflow-internals.pdf\n```\n\nif no okular installed, please install it.\n\n```bash\n$ sudo apt-get install okular\n```\n\n- Windows\n\nopen the file from directory `tensorflow-internals/output`.\n\n## License\n\n[MIT License](http://opensource.org/licenses/mit-license.html) \n\n'
