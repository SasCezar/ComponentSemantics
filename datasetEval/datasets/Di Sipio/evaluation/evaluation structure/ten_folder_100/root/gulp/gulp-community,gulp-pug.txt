b"# gulp-pug [![Build Status][status-img]][status] ![Dependencies][deps] ![Downloads][downloads]\n> Gulp plugin for compiling Pug templates\n\nThis Gulp plugin enables you to compile your Pug templates into HTML or JS, with support for template locals, custom Pug filters, AMD wrapping, and others.  Here is a simple example using `gulp-pug`:\n\n```javascript\nvar pug = require('gulp-pug');\n\ngulp.task('views', function buildHTML() {\n  return gulp.src('views/*.pug')\n  .pipe(pug({\n    // Your options in here.\n  }))\n});\n```\n\n## API\n\n### `pug([opts])`\n\n - `opts` (`Object`): Any options from [Pug's API][api] in addition to `pug`'s own options.\n - `opts.locals` (`Object`): Locals to compile the Pug with. You can also provide locals through the `data` field of the file object, e.g. with [`gulp-data`][gulp-data]. They will be merged with `opts.locals`.\n - `opts.data` (`Object`): Same as `opts.locals`.\n - `opts.client` (`Boolean`): Compile Pug to JavaScript code.\n - `opts.pug`: A custom instance of Pug for `gulp-pug` to use.\n - `opts.verbose`: display name of file from stream that is being compiled.\n \nTo change `opts.filename` use [`gulp-rename`][gulp-rename] before `gulp-pug`.\n\nReturns a stream that compiles Vinyl files as Pug.\n\n## Also See\n\n - [`pug`][pug]\n - [`gulp-data`][gulp-data]: Using locals in your Pug templates easier.\n - [`gulp-rename`][gulp-rename]: Change `opts.filename` passed into Pug.\n - [`gulp-wrap-amd`][gulp-wrap-amd]: Wrap your Pug in an AMD wrapper.\n - [`gulp-frontmatter-wrangler`][gulp-frontmatter-wrangler]: Useful if you need YAML frontmatter at the top of your Pug file.\n\n## Thanks\n\n - Many thanks to [Blaine Bublitz][phated] for the original `gulp-jade` plugin.\n\n## LICENSE\n\n[MIT][license] &copy; Jamen Marzonie\n\n[status]: https://travis-ci.org/gulp-community/gulp-pug\n[status-img]: https://travis-ci.org/gulp-community/gulp-pug.png?branch=master\n[deps]: https://david-dm.org/gulp-community/gulp-pug.svg\n[downloads]: https://img.shields.io/npm/dm/gulp-pug.svg\n[pug]: http://github.com/gulp-community/pug\n[api]: https://pugjs.org/api/reference.html\n[gulp-data]: https://npmjs.com/gulp-data\n[gulp-rename]: https://npmjs.com/gulp-rename\n[gulp-wrap-amd]: https://github.com/phated/gulp-wrap-amd\n[gulp-frontmatter-wrangler]: https://github.com/DougBeney/gulp-frontmatter-wrangler\n[phated]: https://github.com/phated\n[license]: LICENSE\n"
