b'![preview Long Haul](/preview.jpg)\n\nLong Haul is a minimal jekyll theme built with SASS / GULP / BROWSERSYNC / AUTOPREFIXER and focuses on long form blog posts. It is meant to be used as a starting point for a jekyll blog/website.\n\nIf you really enjoy Long Haul and want to give me credit somewhere on the internet send or tweet out your experience with Long Haul and tag me [@brianmaierjr](https://twitter.com/brianmaierjr).\n\n#### [View Demo](http://brianmaierjr.com/long-haul)\n\n## Features\n\n- Minimal, Type Focused Design\n- Built with GULP + SASS + BROWSERSYNC + AUTOPREFIXER\n- SVG Social Icons\n- Responsive Nav Menu\n- XML Feed for RSS Readers\n- Contact Form via Formspree\n- 5 Post Loop with excerpt on Home Page\n- Previous / Next Post Navigation\n- Estimated Reading Time for posts\n- Stylish Drop Cap on posts\n- A Better Type Scale for all devices\n\n## Setup\n\n1. [Install Jekyll](http://jekyllrb.com)\n2. Fork the [Long Haul repo](http://github.com/brianmaierjr/long-haul)\n3. Clone it\n4. [Install Bundler](http://bundler.io/)\n5. Run `bundle install`\n6. Install gulp dependencies by running `npm install`\n7. Run Jekyll and watch files by running `bundle exec gulp`\n8. Customize and watch the magic happen!\n\n## Site Settings\n\nThe main settings can be found inside the `_config.yml` file:\n\n- **title:** title of your site\n- **description:** description of your site\n- **url:** your url\n- **paginate:** the amount of posts displayed on homepage\n- **navigation:** these are the links in the main site navigation\n- **social** diverse social media usernames (optional)\n- **google_analytics** Google Analytics key (optional)\n\n## License\n\nThis is [MIT](LICENSE) with no added caveats, so feel free to use this Jekyll theme on your site without linking back to me or using a disclaimer.\n'
