b'# **An advanced Ionic v1.x template**\n\n![Ionic Framework + Gulp](gh-pages/images/ionic_40.png)\n\n# Introduction\n\nYou need to obfuscate your code and reduce the size of your mobile applications. With this project you can work with Gulp in the best way, allowing improve your development workflow. This project seeks to improve the following tasks:\n\n- Design your application with a maintainable code using **Sass** and writing less code using **Autoprefixer**. Concatenate and Minify styles in a stylesheet.\n- Concatenate and Uglify javascript files using Source maps to debug the code easily.\n- Compress images to reduce the size of your application.\n- Compress html templates, respecting your structure.\n- Clean unnecessary files downloaded with Bower.\n- Use CSS animations with **Animate.css**.\n- Use **SQLite** with a service pattern. **You can use Pre-filled databases**.\n\n# Demo\n\nDo you want to see this starter in action? Run Ionic: \n   - `ionic serve` to test on the browser **(Gulp is running by default)**.\n   - `ionic run android --livereload` to test on the device.\n5. Modify this template and create your hybrid mobile app.\n6. Check the **[John Papa\'s Angular Style Guide](https://github.com/johnpapa/angular-styleguide/blob/master/a1/README.md)**.\n\n# Template Structure\n\nPath         | Explanation\n----------   | -------------\n`./app/img/` | Images in your app.\n`./app/js/`  | Scripts (Controllers, Services, Directives, etc).\n`./app/scss/` | The styles of your app using Sass.\n`./app/templates/` | Views in your app. (Only html files)\n`./app/index.html` | The init page.\n`./www/css/` | Other css styles like **[Animate.css](https://daneden.github.io/animate.css/)**, etc.\n`./www/lib` | Download scripts using bower.\n \t \n# Using bower to download libraries (npm preen included)\n\n* Download the script. Eg: `bower install ionic-datepicker --save`\n* Add the path of the files that you will use in `bower.json` from `www/lib`. Eg:\n```javascript\n"preen": {\n\t//... More libraries\n\t"ionic-datepicker": [\n\t\t"dist/*.js"\n\t\t//Other files and folders will be deleted to reduce the size of your app\n\t]\n}\n```\n* Run gulp in the CLI. Eg: `gulp` or `gulp lib`\n* That\'s all, folks!!\n\n# Animate elements using **[Animate.css](https://daneden.github.io/animate.css/)**\n\n* Do you want to animate Modals? **(pre-populated database)**, you can test in the **browser** using **Google Chrome** or in your **Device**.\n\n![Cordova SQLite](https://github.com/jdnichollsc/Ionic-Starter-Template/blob/gh-pages/images/sqlite.png?raw=true)\n\n* **Debug in the browser:** Test using the **`./app/js/queries.js`** file to create your queries **(Drop tables, create tables, insert data, etc)**.\n* **Debug in the device:** Test using the **`./www/pre.db`** file, you can edit the database using **[DB Browser for SQLite](http://sqlitebrowser.org/)**\n\n## SQLite examples using **Angular Services**\n* Returns the first element in a sequence that satisfies a specified condition, throws an exception if no matching element is found in source:\n```javascript\nvar query = "SELECT * FROM Users WHERE Name LIKE \'%?%\'";\nreturn $q.when($sqliteService.getFirstItem(query, [\'Juan\']));\n```\n* Returns the first element of a sequence that satisfies a specified condition, or a default value if the sequence contains no elements:\n```javascript\nvar query = "SELECT * FROM Users WHERE Name LIKE \'%?%\'";\nreturn $q.when($sqliteService.getFirstOrDefaultItem(query, [\'Juan\']));\n```\n* Returns all the elements of a sequence that satisfies a specified condition:\n```javascript\nvar query = "SELECT * FROM Users WHERE Name LIKE \'%?%\'";\nreturn $q.when($sqliteService.getItems(query, [\'Juan\']));\n```\n* Execute a SQL query:\n```javascript\nvar query = "DELETE FROM Users WHERE Name LIKE \'%?%\'";\nreturn $q.when($sqliteService.executeSql(query, [\'Juan\']));\n```\n\n### **Note**: If you don\'t want to use SQLite, you must perform the following steps:\n1. Remove **`./www/pre.db`** file.\n2. Remove **`./app/js/queries.js`** file.\n3. Remove **`./app/js/services/sqlite.js`** file.\n4. Uninstall the plugin using the CLI: **`ionic plugin rm cordova-sqlite-ext --save`**\n5. Remove the following line from **`./app/js/app.js`** file:\n```javascript\n$sqliteService.preloadDataBase(true);\n```\n\n# Ionic Tips\n* Ionic View LifeCycle: More **[here](http://www.gajotres.net/understanding-ionic-view-lifecycle/)**\n```javascript\n$scope.$on(\'$ionicView.beforeEnter\', function(){\n  alert("Before to enter to the view");\n});\n$scope.$on(\'$ionicView.afterEnter\', function(){\n  alert("After to enter to the view");\n});\n```\n* Reload the current state:\n```javascript\n$state.go($state.current, {}, {reload: true});\n```\n* Disable the back option before to navigate to other state:\n```javascript\n$ionicHistory.nextViewOptions({\n    disableBack: true,\n    disableAnimate : true,\n    historyRoot  : true\n});\n```\n* Clear the cache:\n```javascript\n$ionicHistory.clearCache();\n```\n* Clear the history:\n```javascript\n$ionicHistory.clearHistory();\n```\n* Change the direction before to navigate to other state:\n```javascript\n$ionicViewSwitcher.nextDirection(\'back\');\n```\n* Navigate to other state:\n```javascript\n$state.go("app.login");\n```\n* Disable the drag to open the side menu:\n```javascript\n$ionicSideMenuDelegate.canDragContent(false);\n```\n* Check the current platform\n```javascript\nvar deviceInformation = ionic.Platform.device();\n\nvar isWebView = ionic.Platform.isWebView();\nvar isIPad = ionic.Platform.isIPad();\nvar isIOS = ionic.Platform.isIOS();\nvar isAndroid = ionic.Platform.isAndroid();\nvar isWindowsPhone = ionic.Platform.isWindowsPhone();\n\nvar currentPlatform = ionic.Platform.platform();\nvar currentPlatformVersion = ionic.Platform.version();\n```\n* Disabling the tap system **(To disable the tap for an element and all of its children elements)**\n```html\n<div data-tap-disabled="true">\n    <div id="google-map"></div>\n</div>\n```\n* Using Ionic gestures (Events) with options from Angular Directives\n```javascript\n$ionicGesture.on(\'hold\', function (e) {\n  //Code...\n}, element, { hold_threshold: 20 });\n```\n* Ionic trigger **[events](https://ionicframework.com/docs/v1/api/utility/ionic.EventController/)**\n```javascript\nionic.trigger("hold", { target: document.getElementsByClassName("item")[0] });\n```\n* Dom if not available.\nionic.DomUtil.getPositionInParent(element);    //Find an element\xe2\x80\x99s scroll offset within its container.\nionic.DomUtil.blurAll();                       //Blurs any currently focused input element.\n...\n```\n* Execute when device is ready\n```javascript\nionic.Platform.ready(function(){\n  //Code...\n});\n```\n\n### **Global configuration**:\n* Enable the native scrolling (Enable or Disable jsScrolling):\n```javascript\n$ionicConfigProvider.scrolling.jsScrolling(false);\n```\n* Set the Maximum number of view elements to cache in the DOM:\n```javascript\n$ionicConfigProvider.views.maxCache(5);\n```\n* Select the position of the tabs\n```javascript\n$ionicConfigProvider.tabs.position(\'top\');\n$ionicConfigProvider.tabs.position(\'bottom\');\n```\n* Center align the title in the navBar:\n```javascript\n$ionicConfigProvider.navBar.alignTitle(\'center\');\n```\n* Disable swipeback on iOS:\n```javascript\n$ionicConfigProvider.views.swipeBackEnabled(false);\n```\n* Set the back button text to empty:\n```javascript\n$ionicConfigProvider.backButton.previousTitleText(false).text(\'\');\n```\n* Change global Ionic gestures
