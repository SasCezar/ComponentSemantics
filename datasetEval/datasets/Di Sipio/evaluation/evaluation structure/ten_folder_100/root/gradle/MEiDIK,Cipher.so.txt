Then your can obtain them from the Java interface generated by Cipher.so.\n\n### Features\n\n* Encrypt secure info in a native library via easy configs\n* Reflection free\n\n---\n## Usages\n\n#### Installation\n##### Step 1. in the root build.gradle:  \nAdd `maven { url \'https://jitpack.io\' }` resposity and `classpath \'com.github.MEiDIK:Cipher.so:dev-SNAPSHOT\'` dependency into the buildscript:\n\n```groovy\nbuildscript {\n    repositories {\n        google()\n        maven { url \'https://jitpack.io\' }\n    }\n    dependencies {\n        classpath \'com.android.tools.build:gradle:3.0.1\'\n        classpath \'com.github.MEiDIK:Cipher.so:dev-SNAPSHOT\'\n    }\n}\n```\n\n##### Step 2. in the app module build.gradle:\nAdd `apply plugin:\'cipher.so\'` **before**(*VERY IMPORTANT*) `apply plugin: \'com.android.application\'`\n\n```groovy\napply plugin: \'cipher.so\'\napply plugin: \'com.android.application\'\n```\n\nThat\'s all, Cipher.so is ready to GO.\n\n#### Configuration\n\nIn your app module build.gradle, add the follow-like configs to save key-values.\n\n```groovy\ncipher.so {\n    keys {\n        hello {\n            value = \'Hello From Cipher.so\xf0\x9f\x98\x8a\'\n        }\n        httpsKey {\n            value = \'htkdjfkj@https2017now\'\n        }\n        \xe6\x95\xb0\xe6\x8d\xae\xe5\xba\x93\xe5\xaf\x86\xe7\xa0\x81 {\n            value = \'\xe4\xbb\x8a\xe5\xa4\xa9\xe5\xa4\xa9\xe6\xb0\x94\xe4\xb8\x8d\xe9\x94\x99\xf0\x9f\x98\x82\xf0\x9f\x98\x82\'\n        }\n        ...\n    }\n    encryptSeed = \'HelloSecretKey\' //Secret key used for encryption\n}\n```\n\nThen Rebuild to generate the Java Interface.\n\n#### 3. Call In Java/Kotlin\n\n```Java\nString hello = CipherClient.hello();\nString httpsKey = CipherClient.httpsKey();\nString dbKey = CipherClient.\xe6\x95\xb0\xe6\x8d\xae\xe5\xba\x93\xe5\xaf\x86\xe7\xa0\x81();\n```\n\n\n> Sample: [HelloCipherSo](https://github.com/MEiDIK/HelloCipherSo)\n\n\n\n\n## Contribute?\n\nI am very glad for your contributes.
