     classpath \'com.smallsoho.mobcase:McImage:1.4.0\'\n    }\n}\n```\n\nThen, apply the plugin in your every module.PS: If you have one more Module, you need apply it in every one.\n\n```groovy\napply plugin: \'McImage\'\n```\n\nLast, put the mctools dir in your project root dir.Download it [here](https://github.com/Mobcase/McImage/releases)\n\n\n```\nmctools\n```\n\n### Config\n\nYou can set the config in build.gradle.If you not set this,all config will use default.\n\n```groovy\nMcImageConfig {\n    isCheckSize true //Whether to detect image size\xef\xbc\x8cdefault true\n    optimizeType "Compress" //Optimize Type\xef\xbc\x8c"ConvertWebp" or "Compress"\xef\xbc\x8cdefault "Compress", "CompressWebp" is a better compression ratio but it don\'t support api < 18\n    maxSize 1*1024*1024 //big image size threshold\xef\xbc\x8cdefault 1MB\n    enableWhenDebug false //switch in debug build\xef\xbc\x8cdefault true\n    isCheckPixels true // Whether to detect image pixels of width and height\xef\xbc\x8cdefault true\n    maxWidth 1000 //default 1000\n    maxHeight 1000 //default 1000\n    whiteList = [ //do not do any optimization for the images who in the list \n              "icon_launcher.png"\n    ]\n    mctoolsDir "$rootDir"\n    isSupportAlphaWebp false  //Whether support convert the Image with Alpha chanel to Webp\xef\xbc\x8cdefault false, the images with alpha chanels will be compressed.if config true, its need api level >=18 or do some compatible measures \n    multiThread true  //Whether open multi-thread processing\xef\xbc\x8cdefault true\n    bigImageWhiteList = [] //do not detect big size or large pixels for the images who in the list\n}\n```\n\n###
