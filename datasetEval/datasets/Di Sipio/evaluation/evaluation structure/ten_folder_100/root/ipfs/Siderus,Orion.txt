b"[![Siderus Orion Logo](logo.png)](https://orion.siderus.io/)\n\n# Siderus Orion: Easy to use IPFS Desktop client\n\nSiderus Orion is an easy to use [inter planetary file system](http://ipfs.io)\n(aka IPFS) Desktop client. It helps sharing content on the public peer to peer\nnetwork without any command line or technical knowledge.\nThe app includes everything is needed to start your own personal IPFS node.\n\n**Note**:
