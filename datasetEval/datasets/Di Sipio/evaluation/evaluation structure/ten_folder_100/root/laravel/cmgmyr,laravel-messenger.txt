no longer actively supported)\nInstallation instructions for Laravel 4 can be [found here](https://github.com/cmgmyr/laravel-messenger/tree/v1).\n\n## Installation (Laravel 5.x)\n```\ncomposer require cmgmyr/messenger\n```\n\nOr place manually in composer.json:\n\n```\n"require": {\n    "cmgmyr/messenger": "~2.0"\n}\n```\n\nRun:\n\n```\ncomposer update\n```\n\nAdd the service provider to `config/app.php` under `providers`:\n\n```php\n\'providers\' => [\n    Cmgmyr\\Messenger\\MessengerServiceProvider::class,\n],\n```\n\n> **Note**: If you are using Laravel 5.5, this step is unnecessary. Laravel Messenger supports [Package Discovery](https://laravel.com/docs/5.5/packages#package-discovery).\n\nPublish config:\n\n```\nphp artisan vendor:publish --provider="Cmgmyr\\Messenger\\MessengerServiceProvider" --tag="config"\n```\n\t\nUpdate config file to reference your User Model:\n\n```\nconfig/messenger.php\n```\n\nCreate a `users` table if you do not have one already. If you need one, the default Laravel migration will be satisfactory.\n\n**(Optional)** Define names of database tables in package config file if you don\'t want to use default ones:\n\n```php\n\'messages_table\' => \'messenger_messages\',\n\'participants_table\' => \'messenger_participants\',\n\'threads_table\' => \'messenger_threads\',\n```\n    \nPublish migrations:\n\n```\nphp artisan vendor:publish --provider="Cmgmyr\\Messenger\\MessengerServiceProvider" --tag="migrations"\n```\n\nMigrate your database:\n\n```\nphp artisan migrate\n```\n\nAdd the trait to your user model:\n\n```php\nuse Cmgmyr\\Messenger\\Traits\\Messagable;\n\nclass User extends Authenticatable {\n    use Messagable;\n}\n```\n\n## Examples\n* [Controller](https://github.com/cmgmyr/laravel-messenger/tree/master/examples/MessagesController.php)\n* [Routes](https://github.com/cmgmyr/laravel-messenger/tree/master/examples/routes.php)\n* [Views](https://github.com/cmgmyr/laravel-messenger/tree/master/examples/views)\n\n## Example Projects\n* [WIP] [Pusher](https://github.com/cmgmyr/laravel-messenger-pusher-demo)\n* [WIP] [Lumen API](https://github.com/cmgmyr/lumen-messenger-api)\n\n\n## Contributing? \nPlease format your code before creating a pull-request.
