Clone this repository: `git clone git@github.com:laravelio/laravel-io.git ~/Sites/laravelio`\n2. Run `composer install && homestead make --no-after`\n4. Run `vagrant up`\n5. SSH into your Vagrant box, go to `/home/vagrant/code` and run `composer setup`\n6. Add `192.168.10.10 laravelio.test` to your computer's `/etc/hosts` file\n7. Setup a working e-mail driver like [Mailtrap](https://mailtrap.io/)\n8. (optional) Set up Github authentication (see below)\n\nYou can now visit the app in your browser by visiting [http://laravelio.test](http://laravelio.test). If you seeded the\ndatabase you can login into a test account with `johndoe` & `password`.\n\n### Github Authentication (optional)\n\nTo get Github authentication to work locally, you'll need to\n[register a new OAuth application on Github](https://github.com/settings/applications/new). Use `http://laravelio.test`\nfor the homepage url and `http://laravelio.test/auth/github` for the callback url. When you've created the app, fill in\nthe ID and secret in your `.env` file in the env variables below. You should now be able to authentication with Github.\n\n```\nGITHUB_ID=\nGITHUB_SECRET=\nGITHUB_URL=http://laravelio.test/auth/github\n```\n\n## Maintainers\n\nThe Laravel.io portal is currently maintained by [Dries Vints](https://github.com/driesvints).
