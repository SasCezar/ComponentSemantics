Installation\nPHP 7.2 and Laravel/Lumen 5.7 or higher are required.\n\n> If your application does not meet these requirements, you can check out the 3.x branch for older releases.\n\n```sh\ncomposer require --dev mpociot/laravel-apidoc-generator\n```\n\n### Laravel\nPublish the config file by running:\n\n```bash\nphp artisan vendor:publish --provider="Mpociot\\ApiDoc\\ApiDocGeneratorServiceProvider" --tag=apidoc-config\n```\n\nThis will create an `apidoc.php` file in your `config` folder.\n\n### Lumen\n- When using Lumen, you will need to run `composer require mpociot/laravel-apidoc-generator` instead.\n- Register the service provider in your `bootstrap/app.php`:\n\n```php\n$app->register(\\Mpociot\\ApiDoc\\ApiDocGeneratorServiceProvider::class);\n```\n\n- Copy the config file from `vendor/mpociot/laravel-apidoc-generator/config/apidoc.php` to your project as `config/apidoc.php`. Then add to your `bootstrap/app.php`:\n\n```php\n$app->configure(\'apidoc\');\n```\n\n## Documentation\nCheck out the documentation at [ReadTheDocs](http://laravel-apidoc-generator.rtfd.io/). Don\'t forget to check out the [migration guide](https://laravel-apidoc-generator.rtfd.io/en/latest/migrating.html) if you\'re coming from v3 to v4.\n\n### License\n\nThe Laravel API Documentation Generator is free software licensed under the MIT license.\n'
