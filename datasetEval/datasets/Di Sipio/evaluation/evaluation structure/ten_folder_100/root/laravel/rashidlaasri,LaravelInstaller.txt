b"# Laravel Web Installer | A Web Installer [Package](https://packagist.org/packages/rachidlaasri/laravel-installer)\r\n\r\n[![Total Downloads](https://poser.pugx.org/rachidlaasri/laravel-installer/d/total.svg)](https://packagist.org/packages/rachidlaasri/laravel-installer)\r\n[![Latest Stable Version](https://poser.pugx.org/rachidlaasri/laravel-installer/v/stable.svg)](https://packagist.org/packages/rachidlaasri/laravel-installer)\r\n[![License](https://poser.pugx.org/rachidlaasri/laravel-installer/license.svg)](https://packagist.org/packages/rachidlaasri/laravel-installer)\r\n\r\n- [About](#about)\r\n- [Requirements](#requirements)\r\n- [Installation](#installation)\r\n- [Routes](#routes)\r\n- [Usage](#usage)\r\n- [Contributing](#contributing)\r\n- [Help](#help)\r\n- [Screenshots](#screenshots)\r\n- [License](#license)\r\n\r\n## About\r\n\r\nDo you want your clients to be able to install a Laravel project just like they do with WordPress or any other CMS?\r\nThis Laravel package allows users who don't use Composer, SSH etc to install your application just by following the setup wizard.\r\nThe current features are :\r\n\r\n- Check For Server Requirements.\r\n- Check For Folders Permissions.\r\n- Ability to set database information.\r\n\t- .env text editor\r\n\t- .env form wizard\r\n- Migrate The Database.\r\n- Seed The Tables.\r\n\r\n## Requirements\r\n\r\n* [Laravel 5.1, 5.2, 5.3, 5.4, or 5.5+](https://laravel.com/docs/installation)\r\n\r\n## Installation\r\n\r\n1. From your projects root folder in terminal run:\r\n\r\n```bash\r\n    composer require rachidlaasri/laravel-installer\r\n```\r\n\r\n2. Register the package\r\n\r\n* Laravel 5.5 and up\r\nUses package auto discovery feature, no need to edit the `config/app.php` file.\r\n\r\n* Laravel 5.4 and below\r\nRegister the package with laravel in `config/app.php` under `providers` with the following:\r\n\r\n```php\r\n\t'providers' => [\r\n\t    RachidLaasri\\LaravelInstaller\\Providers\\LaravelInstallerServiceProvider::class,\r\n\t];\r\n```\r\n\r\n3. Publish the packages views, config file, assets, and language files by running the following from your projects root folder:\r\n\r\n```bash\r\n    php artisan vendor:publish --tag=laravelinstaller\r\n```\r\n\r\n## Routes\r\n\r\n* `/install`\r\n* `/update`\r\n\r\n## Usage\r\n\r\n* **Install Routes Notes**\r\n\t* In order to install your application, go to the `/install` route and follow the instructions.\r\n\t* Once the installation has ran the empty file `installed` will be placed into the `/storage` directory. If this file is present the route `/install` will abort to the 404 page.\r\n\r\n* **Update Route Notes**\r\n\t* In order to update your application, go to the `/update` route and follow the instructions.\r\n\t* The `/update` routes countes how many migration files exist in the `/database/migrations` folder and compares that count against the migrations table. If the files count is greater then the `/update` route will render, otherwise, the page will abort to the 404 page.\r\n\r\n* Additional Files and folders published to your project :\r\n\r\n|File|File Information|\r\n|:------------|:------------|\r\n|`config/installer.php`|In here you can set the requirements along with the folders permissions for your application to run, by default the array cotaines the default requirements for a basic Laravel app.|\r\n|`public/installer/assets`|This folder contains a css folder and inside of it you will find a `main.css` file, this file is responsible for the styling of your installer, you can overide the default styling and add your own.|\r\n|`resources/views/vendor/installer`|This folder contains the HTML code for your installer, it is 100% customizable, give it a look and see how nice/clean it is.|\r\n|`resources/lang/en/installer_messages.php`|This file holds all the messages/text, currently only English is available, if your application is in another language, you can copy/past it in your language folder and modify it the way you want.|\r\n\r\n## Contributing\r\n\r\n* If you have any suggestions please let me know : https://github.com/RachidLaasri/LaravelInstaller/pulls.\r\n* Please help us provide more languages for this awesome package please send a pull request https://github.com/RachidLaasri/LaravelInstaller/pulls.\r\n\r\n## Help\r\n\r\n* Cannot figure it out?
