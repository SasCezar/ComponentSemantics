Clone the source code or create new project.\n\n```shell\ngit clone https://github.com/jcc/blog.git\n```\n\nOR\n\n```shell\ncomposer create-project jcc/blog\n```\n\n### 2. Set the basic config\n\n```shell\ncp .env.example .env\n```\n\nEdit the `.env` file and set the `database` and other config for the system after you copy the `.env`.example file.\n\n### 2. Install the extended package dependency.\n\nInstall the `Laravel` extended repositories: \n\n```shell\ncomposer install -vvv\n```\n\nInstall the `Vuejs` extended repositories: \n\n```shel\nnpm install\n```\n\nCompile the js code: \n\n```shel\nnpm run dev\n\n// OR\n\nnpm run watch\n\n// OR\n\nnpm run production\n```\n\n### 3.
