[Miscellaneous](#miscellaneous)\n\n### Introduction\n\n[Bagisto](https://www.bagisto.com) is a hand tailored E-Commerce framework designed on some of the hottest opensource technologies\nsuch as [Laravel](https://laravel.com) a [PHP](https://secure.php.net/) framework, [Vue.js](https://vuejs.org)\na progressive Javascript framework.\n\n**Bagisto is viable attempt to cut down your time, cost and workforce for building online stores or migrating from physical stores\nto the ever demanding online world. Your business whether small or huge it suits all and very simple to set it up.**\n\n**Read our documentation: [Bagisto Docs](https://devdocs.bagisto.com/)**\n\n**We are also having a forum for any type of your concern, feature request discussions. Please visit: [Bagisto Forums](https://forums.bagisto.com/)**\n\n# Visit our live [Demo](https://demo.bagisto.com)\n\nIt packs in lots of demanding features that allows your business to scale in no time:\n\n* Multiple Channels, Locale, Currencies.\n* Built-in Access Control Layer.\n* Beautiful and Responsive Storefront.\n* Descriptive and Simple Admin Panel.\n* Admin Dashboard.\n* Custom Attributes.\n* Built on Modular Approach.\n* Support for Multiple Store Themes.\n* Multistore Inventory System.\n* Orders Management System.\n* Customer Cart, Wishlist, Product Reviews.\n* Simple and Configurable Products.\n* Price rules (Discount) inbuilt.\n* CMS Pages.\n* Check out [click here](https://bagisto.com/features/).\n\n**For Developers**:\nDev guys can take advantage of two of the hottest frameworks used in this project Laravel and Vue.js, both of these frameworks have been used in Bagisto.\nBagisto is using power of both of these frameworks and making best out of it out of the box.\n\n### Documentation\n\n#### Bagisto Documentation [https://devdocs.bagisto.com](https://devdocs.bagisto.com)\n\n### Requirements\n\n* **OS**: Ubuntu 16.04 LTS or higher / Windows 7 or Higher (WampServer / XAMPP).\n* **SERVER**: Apache 2 or NGINX.\n* **RAM**: 3 GB or higher.\n* **PHP**: 7.1.3 or higher.\n* **Processor**: Clock Cycle 1 Ghz or higher.\n* **For MySQL users**: 5.7.23 or higher.\n* **For MariaDB users**: 10.2.7 or Higher.\n* **Node**: 8.11.3 LTS or higher.\n* **Composer**: 1.6.5 or higher.\n\n### Installation and Configuration\n\n**1. Try our new GUI installer to install Bagisto:**\n\n##### a. Download zip from the link below:\n\n[Download](https://github.com/bagisto/bagisto/archive/v0.1.6.zip)\n\n##### b. Extract the contents of zip and execute the project in browser:\n\n~~~\nhttp(s)://localhost/bagisto/public\n~~~\n\nor\n\n~~~\nhttp(s)://example.com/public\n~~~\n\n**2. Try our old fashioned way to install Bagisto:**\n\n##### Execute these commands below as in their order\n\n~~~\n1. composer create-project bagisto/bagisto\n~~~\n\n**Now configure your database:**\n\nIf the above command was completed successfully, then you\'ll find directory **bagisto** and all of the code will be inside it.\n\nFind file **.env** inside **bagisto** directory and set the environment variables listed below:\n\n* **APP_URL**\n* **DB_CONNECTION**\n* **DB_HOST**\n* **DB_PORT**\n* **DB_DATABASE**\n* **DB_USERNAME**\n* **DB_PASSWORD**\n\nAlthough, mailer environment variables are also required to be set up as **Bagisto** requires emails to send to customers and admins for various functionalities that are built in.\n\n~~~\n2. php artisan migrate\n~~~\n\n~~~\n3. php artisan db:seed\n~~~\n\n~~~\n4. php artisan vendor:publish\n-> Press 0 and then press enter to publish all assets and configurations.\n~~~\n\n~~~\n5. php artisan storage:link\n~~~\n\n~~~\n6. composer dump-autoload\n~~~\n\n\n**To execute Bagisto**:\n\n##### On server:\n\nWarning: Before going full on production mode we recommend you to uninstall developer dependencies.\nIn order to do that, run the command below:\n\n> composer install --no-dev\n\n~~~\nOpen the specified entry point in your hosts file in browser or make entry in hosts file if not done.\n~~~\n\n##### On local:\n\n~~~\nphp artisan serve\n~~~\n\n\n**How to log in as admin:**\n\n> *http(s)://example.com/admin/login*\n\n~~~\nemail:admin@example.com\npassword:admin123\n~~~\n\n**How to log in as customer:**\n\n*You can directly register as customer and then login.*\n\n> *http(s)://example.com/customer/register*\n\n\n### License\nBagisto is a truly opensource E-Commerce framework which will always be free under the [MIT License](https://github.com/bagisto/bagisto/blob/master/LICENSE).\n\n### Security Vulnerabilities\nPlease don\'t disclose security vulnerabilities publicly.
