Allows you to abstract the common patterns of business applications, so that developers can easily implement beautiful and elegant interfaces without much effort. The main places of application are: backoffice applications, admin panels, content management systems, etc.\n\n\n<a href="https://raw.githubusercontent.com/orchidsoftware/platform/master/.github/IMAGES/promo-full.png">\n<img src="https://raw.githubusercontent.com/orchidsoftware/platform/master/.github/IMAGES/promo-slim.png" alt="Laravel Orchid Platform" align="right" />\n</a>\n\n\n###### Some features:\n\n- **Form builder** - no need to describe HTML fields of the same type each time.\n- **Screen** - a comfortable balance between CRUD generation and handwriting.\n- **Fields** - over 40 types.\n- **Permissions** - offers convenient management in development and support.\n- Menus, graphics, settings, notifications, etc.\n \n ## Getting started\n\n* :uk: To check out the [guide](https://orchid.software/en/docs/), visit [orchid.software](https://orchid.software/en/docs/). \n* :ru: \xd0\xa7\xd1\x82\xd0\xbe\xd0\xb1\xd1\x8b \xd0\xbe\xd0\xb7\xd0\xbd\xd0\xb0\xd0\xba\xd0\xbe\xd0\xbc\xd0\xb8\xd1\x82\xd1\x8c\xd1\x81\xd1\x8f \xd1\x81 [\xd1\x80\xd1\x83\xd0\xba\xd0\xbe\xd0\xb2\xd0\xbe\xd0\xb4\xd1\x81\xd1\x82\xd0\xb2\xd0\xbe\xd0\xbc](https://orchid.software/ru/docs/), \xd0\xbf\xd0\xbe\xd1\x81\xd0\xb5\xd1\x82\xd0\xb8\xd1\x82\xd0\xb5 \xd1\x81\xd0\xb0\xd0\xb9\xd1\x82 [orchid.software](https://orchid.software/ru/docs/).\n\n##### Local User Groups\n\n* [Orchid Global Community](https://t.me/orchid_community)\n* [Orchid Russia](https://t.me/orchid_russian_community)\n\n## Change log\n\nSee [CHANGELOG](CHANGELOG.md).\n\n## Donate & Support\n\nSince the existence of a healthy open source ecosystem creates real value for the software industry, believe it is fair for maintainers and authors of such software to be compensated for their work with real money.\n\nIf you would like to support development by making a donation you can do so [here](https://www.paypal.me/tabuna/10usd). &#x1F60A;\n\n\n## License\n\nThe MIT License (MIT).
