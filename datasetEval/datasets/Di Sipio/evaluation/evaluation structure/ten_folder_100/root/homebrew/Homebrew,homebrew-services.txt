b"# Homebrew Services\n\nIntegrates Homebrew formulae with macOS' `launchctl` manager.\n\n## Requirements\n\n[Homebrew](https://github.com/Homebrew/brew) is used for installing the services.\n\nThis does not work with Linuxbrew (so don't file Linux issues, please).\n\n## Install\n\n`brew services` is automatically installed when run.\n\n## Usage\n\n### Start\n\nStart the MySQL service at login with:\n\n```bash\nbrew services start mysql\n```\n\nStart the Dnsmasq service at boot with:\n\n```bash\nsudo brew services start dnsmasq\n```\n\nStart all available services with:\n\n```bash\nbrew services start --all\n```\n\n### Run\n\nRun the MySQL service but don't start it at login (nor boot) with:\n\n```bash\nbrew services run mysql\n```\n\n### Stop\n\nStop the MySQL service with:\n\n```bash\nbrew services stop mysql\n```\n\n### Restart\n\nRestart the MySQL service with:\n\n```bash\nbrew services restart mysql\n```\n\n### List\n\nList all services managed by `brew services` with:\n\n```bash\nbrew services list\n```\n\n### Cleanup\n\nRemove all unused services with:\n\n```bash\nbrew services cleanup\n```\n\n## Copyright\n\nCopyright (c) Homebrew maintainers. See [LICENSE.txt](https://github.com/Homebrew/homebrew-services/blob/master/LICENSE.txt) for details.\n"
