b'# golinks\n\n[![Build Status](https://cloud.drone.io/api/badges/prologic/golinks/status.svg)](https://cloud.drone.io/prologic/golinks)\n[![CodeCov](https://codecov.io/gh/prologic/golinks/branch/master/graph/badge.svg)](https://codecov.io/gh/prologic/golinks)\n[![Go Report Card](https://goreportcard.com/badge/prologic/golinks)](https://goreportcard.com/report/prologic/golinks)\n[![GoDoc](https://godoc.org/github.com/prologic/golinks?status.svg)](https://godoc.org/github.com/prologic/golinks) \n[![GitHub license](https://img.shields.io/github/license/prologic/golinks.svg)](https://github.com/prologic/golinks)\n\ngolinks is a web app that allows you to create smart bookmarks, commands and aliases by pointing your web browser\'s default search engine at a running instance. Similar to bunny1 or yubnub.\n\n## Installation\n\n### Source\n\n```bash\n$ go get github.com/prologic/golinks\n```\n\n### OS X Homebrew\n\nThere is a formula provided that you can tap and install from\n[prologic/homebrew-golinks](https://github.com/prologic/homebrew-golinks):\n\n```bash\n$ brew tap prologic/golinks\n$ brew install golinks\n```\n\n**NB:** This installs the latest released binary; so if you want a more\nrecent unreleased version from master you\'ll have to clone the repository\nand build yourself.\n\n## Usage\n\nRun golinks:\n\n```bash\n$ golinks -bind 127.0.0.1:8000 -fqdn localhost:8000\n```\n\nSet your browser\'s default golinks engine to http://localhost:8000/?q=%s\n\n- [Instructions for Chrome](https://support.google.com/chrome/answer/95426)\n- [Instructions for Firefox](https://support.mozilla.org/en-US/kb/add-or-remove-search-engine-firefox#w_add-a-search-engine-from-the-address-bar)\n\nThen type `help` to view the main help page, `g foo bar` to perform a [Google](https://google.com) search for "foo bar" or `list` to list all available commands.\n\n\n### Custom bookmarks\n\nTo add a bookmark (or overwrite an existing one), enter `add [name] [url]` as your search query, where `name` is the shortcut for the bookmark and `url` the URL:\n\n```\nadd imdb https://www.imdb.com\n```\n\nNow you can just enter `imdb` in your search bar to go straight to imdb.com.\n\nYou can also add `%s` to your URL, which will be replaced with your search query:\n\n```\nadd ddg https://duckduckgo.com/?q=%s\n```\n\nNow you can use `ddg [query]` to search via DuckDuckGo, e.g. `ddg free stuff` to find yourself some free stuff.\n\nTo remove a search, use `remove [name]`, so `remove ddg` will remove the above search.\n\n### Other commands\n\nUse `list` to see all your bookmarks and commands (golinks comes with several useful built-ins) and `help` to view the online help page.\n\n## Stargazers over time\n\n[![Stargazers over time](https://starcharts.herokuapp.com/prologic/golinks.svg)](https://starcharts.herokuapp.com/prologic/golinks)\n\n## Support\n\nSupport the ongoing development of Bitcask!\n\n**Sponsor**\n\n- Become a [Sponsor](https://www.patreon.com/prologic)\n\n## Contributing\n\ngolinks is considered "production" software and is used daily.
