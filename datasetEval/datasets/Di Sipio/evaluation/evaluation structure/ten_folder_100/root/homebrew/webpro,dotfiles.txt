b'# .files\n\nThese are my dotfiles. Take anything you want, but at your own risk.\n\nIt targets macOS systems, but it should work on \\*nix as well (with `apt-get`).\n\n## Package overview\n\n- [Homebrew](https://brew.sh) (packages: [Brewfile](./install/Brewfile))\n- [homebrew-cask](https://caskroom.github.io) (packages: [Caskfile](./install/Caskfile))\n- [Node.js + npm LTS](https://nodejs.org/en/download/) (packages: [npmfile](./install/npmfile))\n- Latest Ruby (packages: [Gemfile](./install/Gemfile))\n- Latest Git, Bash 4, Python 3, GNU coreutils, curl\n- [Hammerspoon](https://www.hammerspoon.org) (config: [keybindings & window management](./config/hammerspoon))\n- [Mackup](https://github.com/lra/mackup) (sync application settings)\n- `$EDITOR` (and Git editor) is [GNU nano](https://www.nano-editor.org)\n\n## Install\n\nOn a sparkling fresh installation of macOS:\n\n    sudo softwareupdate -i -a\n    xcode-select --install\n\nThe Xcode Command Line Tools includes `git` and `make` (not available on stock macOS).\nThen, install this repo with `curl` available:\n\n    bash -c "`curl -fsSL https://raw.githubusercontent.com/webpro/dotfiles/master/remote-install.sh`"\n\nThis will clone (using `git`), or download (using `curl` or `wget`), this repo to `~/.dotfiles`. Alternatively, clone manually into the desired location:\n\n    git clone https://github.com/webpro/dotfiles.git ~/.dotfiles\n\nUse the [Makefile](./Makefile) to install everything [listed above](#package-overview), and symlink [runcom](./runcom) and [config](./config) (using [stow](https://www.gnu.org/software/stow/)):\n\n    cd ~/.dotfiles\n    make\n\n## Post-install\n\n- `dotfiles dock` (set [Dock items](./macos/dock.sh))\n- `dotfiles macos` (set [macOS defaults](./macos/defaults.sh))\n- Mackup\n  - Log in to Dropbox (and wait until synced)\n  - `ln -s ~/.config/mackup/.mackup.cfg ~` (until [#632](https://github.com/lra/mackup/pull/632) is fixed)\n  - `mackup restore`\n\n## The `dotfiles` command\n\n    $ dotfiles help\n    Usage: dotfiles <command>\n\n    Commands:\n       clean            Clean up caches (brew, npm, gem, rvm)\n       dock             Apply macOS Dock settings\n       edit             Open dotfiles in IDE (code) and Git GUI (stree)\n       help             This help message\n       macos            Apply macOS system defaults\n       test             Run tests\n       update           Update packages and pkg managers (OS, brew, npm, gem)\n\n## Customize/extend\n\nYou can put your custom settings, such as Git credentials in the `system/.custom` file which will be sourced from `.bash_profile` automatically. This file is in `.gitignore`.\n\nAlternatively, you can have an additional, personal dotfiles repo at `~/.extra`. The runcom `.bash_profile` sources all `~/.extra/runcom/*.sh` files.\n\n## Additional resources\n\n- [Awesome Dotfiles](https://github.com/webpro/awesome-dotfiles)\n- [Homebrew](https://brew.sh)\n- [Homebrew Cask](http://caskroom.io)\n- [Bash prompt](https://wiki.archlinux.org/index.php/Color_Bash_Prompt)\n- [Solarized Color Theme for GNU ls](https://github.com/seebi/dircolors-solarized)\n\n## Credits\n\nMany thanks to the [dotfiles community](https://dotfiles.github.io).\n'
