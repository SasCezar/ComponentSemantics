b"# homebrew-avr [![Build Status](https://dev.azure.com/osx-cross/homebrew-avr/_apis/build/status/osx-cross.homebrew-avr?branchName=master)](https://dev.azure.com/osx-cross/homebrew-avr/_build/latest?definitionId=1&branchName=master)\n\nThis repository contains the GNU AVR Toolchain as formulae for [Homebrew].\n\nAVR is a popular family of micro-controllers, used for example in the [Arduino] project.\n\n## Current Versions\n\n-   `gcc 9.1.0`\n-   `binutils 2.32`\n-   `avr-libc 2.0.0`\n-   `gdb 8.1.1`\n\nOther GCC versions available:\n\n- `8.3.0`\n- `7.4.0`\n- `6.5.0`\n- `5.5.0`\n- `4.9.4`\n\n## Installing homebrew-avr formulae\n\nFirst, make sure you have xcode command line developer tools installed with\n\n```console\n$ xcode-select --install\n```\n\nThen, just run the following to install the latest version of `avr-gcc`:\n\n```console\n$ brew tap osx-cross/avr\n$ brew install avr-gcc\n```\n\nIf you want to install an older version: \n\n```console\n$ brew install avr-gcc@X\n```\n\nWhere `X` being the version number such as `avr-gcc@6`\n\n**Note**: only the latest version will be available in your `$PATH`. The older ones are `keg-only` and thus won't be availble in `/usr/local/bin`.\n\nYou can run `brew info avr-gcc` for more information on the flags available.\n\n## Docs\n\n`brew info avr-gcc`, `brew help`, `man brew`, or the Homebrew [wiki].\n\n## Thanks\n\nThis repository is based on the works of:\n\n-   [Leka]\n-   [larsimmisch]\n-   [plietar]\n-   [0xPIT]\n\n[Homebrew]: http://brew.sh\n[Arduino]: http://arduino.cc\n[wiki]: http://wiki.github.com/mxcl/homebrew\n[Leka]: https://github.com/Leka/homebrew-avr\n[larsimmisch]: https://github.com/larsimmisch/homebrew-avr\n[plietar]: https://github.com/plietar/homebrew-avr/\n[0xPIT]: https://github.com/0xPIT/homebrew-avr\n"
