b"# KYMSU\nKeep Your macOs Stuff Updated (KYMSU)\n\n> Get your Homebrew, Mac App Store, Atom Package Manager (apm), npm, yarn, composer, rustup ... stuff updated.\n\n## Requirements\n\n- [Homebrew](https://brew.sh)\n\n## Installation\n\n`$ git clone git@github.com:welcoMattic/kymsu.git && cd kymsu && ./install.sh`\n\n## Usage\n\nOnly update all the things\n`$ kymsu`\n\nWith cleanup after updates\n`$ kymsu cleanup`\n\n## Plugins system\n\nAll default plugins are placed in `~/.kymsu/plugins.d`, feel free to delete those you don't use, and/or add new ones.\nDon't forget to share them here with a shiny pull request! \xe2\x9c\xa8\n\n## Auto-update\n\nKYMSU will auto-update itself each time you run `kymsu` command.\n\n## Credits\n\n* [All contributors](https://github.com/welcomattic/kymsu/graphs/contributors)\n* Thanks to [@waxzce](https://github.com/waxzce) for his [article](https://medium.com/@waxzce/keeping-macos-clean-this-is-my-osx-brew-update-cli-command-6c8f12dc1731) about keeping macOs clean!\n\n## License\n\nKYMSU is licensed under the MIT License - see the [LICENSE](LICENSE) file\nfor details.\n"
