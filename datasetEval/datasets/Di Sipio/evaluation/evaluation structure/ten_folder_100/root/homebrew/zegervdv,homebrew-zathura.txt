b'# Zathura\n\nHomebrew formulae to install zathura and plugins on Mac OS X\n\n## Installation\nTap the repository:\n```brew tap zegervdv/zathura```\n\n__Before installing__\n\n\nInstall zathura and required plugins\n```\nbrew install zathura\nbrew install zathura-pdf-poppler\n```\nFollow the instructions to link the plugins into place:\n```\n$ mkdir -p $(brew --prefix zathura)/lib/zathura\n$ ln -s $(brew --prefix zathura-pdf-poppler)/libpdf-poppler.dylib $(brew --prefix zathura)/lib/zathura/libpdf-poppler.dylib\n```\n\n### Copying to clipboard\nAdd the following to your `~/.config/zathura/zathurarc`:\n```\nset selection-clipboard clipboard\n```\nThanks to [geigi](https://github.com/geigi) (see [#5](https://github.com/zegervdv/homebrew-zathura/issues/5))\n\n## Known Issues\nCurrently plugins are installed in the zathura folder, which means that uninstalling\na plugin does not work.\n'
