b'# Nuls\n\n**Welcome to Nuls!**\n\n## Introduction\n\nNuls is a global blockchain open-source project which is a highly customizable modular blockchain infrastructure.  \nIt consists of a microkernel and functional modules.  \nNuls provides smart contract, multi-chain mechanism and cross-chain consensus.  \nIt aims to break the technical barriers of blockchain, to reduce the cost of development, and to promote the usage of blockchain technology in the commercial field.\n\n**Project Features**\n\n- Modular design  \n- Parallel chains  \n- Smart contract  \n- POC consensus mechanisms\n\nTo learn more about us, visit [nuls.io](https://nuls.io/).\n\n## Contributing to Nuls\n \nContributions to Nuls are welcomed!\nWe sincerely\xc2\xa0invite\xc2\xa0developers who experienced in blockchain field\xc2\xa0to join in NULS technology community. \nDetails: s: https://nuls.communimunity/d/9-recruitment-of-community-developers\nTo be a great community, Nuls needs to welcome developers from all walks of life, with different backgrounds, and with a wide range of experience.\n \n## Getting Started\n\n### Requirements\n\n**Operating System**  \nNuls is based on the Java and you can choose your favorite OS.  \n\n**Dependencies SDK**  \nJDK:[JDK 1.8](http://www.oracle.com/technetwork/java/javase/downloads/index.html)  \nMaven:[Maven 3.3+](http://maven.apache.org/download.cgi)  \nIDE: Any one you  like\n> JetBrains [IntelliJ IDEA](https://www.jetbrains.com/idea/) is recommended, but we do not rovoke the IDE war.\n\n\n### Code style guide\nUse Alibaba Java Coding Guidelines [@Github](https://github.com/alibaba/p3c).  \n* [Alibaba-Java-Coding-Guidelines](https://alibaba.github.io/Alibaba-Java-Coding-Guidelines/)  \n> recommend use IDE plugin lint code.
