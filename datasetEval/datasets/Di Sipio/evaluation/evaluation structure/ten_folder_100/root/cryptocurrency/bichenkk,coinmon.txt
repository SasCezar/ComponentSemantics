You can add symbols seperated by comma.\n\n```\n$ coinmon -f btc // search coins included keyword btc\n$ coinmon -f btc,eth // search coins included keyword btc or eth\n```\n\n### Find top coin(s)\n\nYou can use the `-t` (or `--top`) with the index to find the top n cryptocurrencies ranked by their market cap.\n\n```\n$ coinmon -t 50 // find top 50\n$ coinmon -t 1000 // find top 1000\n```\n\n### Add your portfolio\n\nYou can use the `-p` (or `--portfolio`) to retrieve price info and value estimation for your personal crypto portfolio specified in $HOME/.coinmon/portfolio.json in the following format:\n\n```\n$ cd\n$ mkdir .coinmon\n$ cd .coinmon\n$ touch portfolio.json\n```\n\nAnd you can use vim or editors to input your portfolio\n\n```\n{\n  "btc": 10,\n  "eth": 100,\n  ...\n}\n```\n\n```\n$ coinmon -p // Retrieve the portfolio from $HOME/.coinmon/portfolio.json\n$ coinmon -p /anotherDir/myportfolio.json // Retrieve the portfolio from specified directory\n```\n\n### Show specific columns\n\nYou can use the `-s` (or `--specific`) to display specific columns. You can add index seperated by comma.\n\n```\n0 - Rank (by Market Cap)\n1 - Name\n2 - Price\n3 - Change 1H\n4 - Change 24H\n5 - Change 7D\n6 - Market Cap\n7 - Balance (when portfolio is enabled)\n8 - Estimated Value (when portfolio is enabled)\n```\n\n```\n$ coinmon -s 0,1,2 // Display only Rank, Coin and Price\n$ coinmon -s 0,1,2,4 // Display only Rank, Coin, Price and Change 24H\n```\n\n### Rank specific columns\n\nYou can use the `-r` (or `--rank`) with the index to sort specific columns.\n\n```\n0 - Rank (by Market Cap)\n1 - Name\n2 - Price\n3 - Change 1H\n4 - Change 24H\n5 - Change 7D\n6 - Market Cap\n7 - Balance (when portfolio is enabled)\n8 - Estimated Value (when portfolio is enabled)\n```\n\n```\n$ coinmon -t 100 -r 4 // Rank top 100 coins based on Change 24H column\n$ coinmon -p -r 7  // Rank my portfolio based on Estimated Value column\n```\n\n### Show option menu\n\nYou can use the `-h` (or `--help`) to find all valid options of coinmon\n\n```\n$ coinmon -h\n```\n\n## Screenshot\n\n<img src="https://raw.githubusercontent.com/bichenkk/coinmon/master/screenshot.png">\n\n## Development\n\nIt\'s simple to run `coinmon` on your local computer.  \nThe following is step-by-step instruction.\n\n```\n$ git clone https://github.com/bichenkk/coinmon.git\n$ cd coinmon\n$ yarn\n$ npm install -g\n$ npm link\n$ coinmon\n```\n\n## Docker\n\n### Initial usage\n\n```\n$ docker run --rm -ti jaymoulin/coinmon\n```\n\nYou can pass parameters just like `coinmon` binary\n\n```\n$ docker run --rm -ti jaymoulin/coinmon --help\n```\n\n### Local usage\n\n#### Build image\n\n```\n$ docker build -t coinmon .\n```\n\n#### Usage\n\n```\n$ docker run --rm -ti coinmon\n```\n\nYou can pass parameters just like `coinmon` binary\n\n```\n$ docker run --rm -ti coinmon --help\n```\n\n## Contributors\n\n* @maticrivo \n* @pgila \n* @vladei \n* @jaymoulin\n* @anilkilic\n\n## License\n\nMIT\n'
