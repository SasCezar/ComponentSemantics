AES256 encrypted config file.\n+ REST API support for all exchanges.\n+ Websocket support for applicable exchanges.\n+ Ability to turn off/on certain exchanges.\n+ Ability to adjust manual polling timer for exchanges.\n+ Communication packages (Slack, SMS via SMSGlobal, Telegram and SMTP)\n+ HTTP rate limiter package.\n+ Forex currency converter packages (CurrencyConverterAPI, CurrencyLayer, Fixer.io, OpenExchangeRates)\n+ Packages for handling currency pairs, tickers and orderbooks.\n+ Portfolio management tool; fetches balances from supported exchanges and allows for custom address tracking.\n+ Basic event trigger system.\n+ WebGUI.\n\n## Planned Features\n\nPlanned features can be found on our [community Trello page](https://trello.com/b/ZAhMhpOy/gocryptotrader).\n\n## instructions\n\nDownload and install Go from [Go Downloads](https://golang.org/dl/) for your\nplatform.\n\n### Linux/OSX\n\nGoCryptoTrader is built using [Go Modules](https://github.com/golang/go/wiki/Modules) and requires Go 1.11 or above\nUsing Go Modules you now clone this repository **outside** your GOPATH\n\n```bash\ngit clone https://github.com/thrasher-corp/gocryptotrader.git\ncd gocryptotrader\ngo build\nmkdir ~/.gocryptotrader\ncp config_example.json ~/.gocryptotrader/config.json\n```\n\n### Windows\n\n```bash\ngit clone https://github.com/thrasher-corp/gocryptotrader.git\ncd gocryptotrader\ngo build\ncopy config_example.json %APPDATA%\\GoCryptoTrader\\config.json\n```\n\n+ Make any neccessary changes to the `config.json` file.\n+ Run the `gocryptotrader` binary file inside your GOPATH bin folder.\n\n## Donations\n\n<img
