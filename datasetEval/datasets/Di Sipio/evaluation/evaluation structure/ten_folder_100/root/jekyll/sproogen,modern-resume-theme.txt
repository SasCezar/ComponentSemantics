If you plan on hosting it seperately then there might be some extra steps that we wont cover.\n\nBefore starting it might be useful to familiarise yourself with [Jekyll](https://jekyllrb.com/docs/home/), [Markdown](https://www.markdownguide.org/getting-started) and [GitHub pages](https://pages.github.com/).\n\n##### Step 1 - GitHub\nStart by creating an account on [GitHub](https://github.com/join)\n\n##### Step 2 - Create Repository\nCreate a repository on GitHub to hold your files and host your resume. You can find out how to do that [here](https://pages.github.com/)\n\n##### Step 3 - Download Resume Template\nDownload and extract the following zip into the git repository you have just created. [resume-template.zip](https://github.com/sproogen/modern-resume-theme/archive/gh-pages.zip)\n\n##### Step 4 - Push it\nCommit and push the resume template to github\n```\n$ git add --all\n$ git commit -m "Initial resume setup"\n$ git push -u origin master\n```\n##### Set 5 - See it\nYou should now be able to see the demo resume template using this theme at `[your-username].github.io`\n\n## Usage\n\nSo now you will be able to see the demo template at your github url. You can can edit the yml files and replace the demo content with your own. Hopefully it will be fairly simple to work out where all the content goes, but here is a quick overview.\n\n##### _config.yml\nThis will contain all the of the main configuration for your resume such as your name, email, social media links and about me content. It will also allow you to change the titles of some of the content sections.\nA full example of the _config.yml can be found [here](https://github.com/sproogen/modern-resume-theme/blob/master/_config.yml)\n\n##### _data/education.yml\nA list of all your education, each education will follow this format\n```\n- layout: left (options: left, right, top, top-right, top-middle)\n  name: Institution name\n  dates: Date Range (eg. 2016 - 2019)\n  qualification: Qualifications (eg. BA Performing Arts)\n  quote: >\n    Short institution or course description (optional)\n  description: | # this will include new lines to allow paragraphs\n    Description of qualification\n```\n\n##### _data/experience.yml\nA list of all your experience, each experience will follow this format\n```\n- layout: left (options: left, right, top, top-right, top-middle)\n  company: Company name\n  link: Link to company (eg. https://google.com)(optional)\n  job_title: Job title\n  dates: Date Range (eg. November 2016 - present)\n  quote: >\n   Short description of the company (optional)\n  description: | # this will include new lines to allow paragraphs\n    Description of role\n```\n\nIf you wish to specify multiple job titles for a single company, use this format\n```\n- layout: left (options: left, right, top, top-right, top-middle)\n  company: Company name\n  link: Link to company (optional)\n  jobs:\n    - title: Job title 1\n      dates: Date Range (eg. November 2016 - present)\n    - title: Job title 2\n      dates: Date Range (eg. January 2015 - November 2016)\n  quote: >\n   Short description of the company (optional)\n  description: | # this will include new lines to allow paragraphs\n    Description of role\n```\n\n##### _data/projects.yml\nA list of all your projects, each project will follow this format\n```\n- layout: left (options: left, right, top, top-right, top-middle)\n  name: Project name\n  link: Link to project (eg. https://sproogen.github.io/modern-resume-theme)(optional)\n  github: Github page for project (eg. sproogen/modern-resume-theme)(optional)\n  quote: >\n    Short overview of the project (optional)\n  description: | # this will include new lines to allow paragraphs\n    Description about the work on/with the project\n```\n\n##### assets/main.scss\nAdd any css changes or additions you want to make here after the line `@import \'modern-resume-theme\';`\n\n## Running locally\n\nBefore you start make sure you have *Ruby* and the gems for *Jekyll* installed locally. You can find out how to do that [here](https://jekyllrb.com/docs/installation/).\n\n1. Clone your resume repository locally *(if you haven\'t already)*\n2. `cd [your-repository-name]`\n3. `bundle install`\n4. `bundle exec jekyll serve`\n5. Open your browser to `http://localhost:4000`\n\nAny changes you make will automatically build and you will be able to see these by refreshing your browser.\n\n*Note: You will need to re-run `bundle exec jekyll serve` to see changes made in `_config.yml`.*\n\n## Contributing\n\nBug reports and pull requests are welcome on GitHub at https://github.com/sproogen/modern-resume-theme.
