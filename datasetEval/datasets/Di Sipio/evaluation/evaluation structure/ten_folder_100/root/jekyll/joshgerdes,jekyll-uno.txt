b"# jekyll-uno\n\nJekyll-Uno - a minimal, responsive theme for Jekyll based on the [Uno](https://github.com/daleanthony/Uno) theme for Ghost.\n\n> :warning:\n  This theme requires ruby and rubygems installed\n\n### Features\n\n* Clean layout\n* Resposive layout\n* Pagination\n* Syntax highlighting\n* Social links\n* Tags listing page\n* Categories listing page\n* Google Analytics integration\n* Disqus integration\n\n---\n\n### Install and Test\n\n1. Download or clone repo `git clone git@github.com:joshgerdes/jekyll-uno.git`\n2. Enter the folder: `cd jekyll-uno/`\n3. If you don't have bundler installed: `gem install bundler`\n3. Install Ruby gems: `bundle install`\n4. Start Jekyll server: `bundle exec jekyll serve --watch`\n\nAccess via: [http://localhost:4000/jekyll-uno/](http://localhost:4000/jekyll-uno/)\n\n---\n\n### Demo and Download\n\n[Demo](http://joshgerdes.com/jekyll-uno/)\n[Download](https://github.com/joshgerdes/jekyll-uno/archive/master.zip)\n\n![jekyll-uno - free Jekyll theme](/screenshot.png)\n\n---\n\n### Copyright and license\n\nIt is under [the MIT license](/LICENSE)."
