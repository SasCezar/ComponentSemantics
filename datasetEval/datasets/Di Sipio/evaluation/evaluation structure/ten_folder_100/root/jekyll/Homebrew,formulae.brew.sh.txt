b'# Homebrew Formulae\n\n[Homebrew Formulae](https://formulae.brew.sh) is an online package browser for [Homebrew](https://brew.sh).\n\nIt displays all packages in [Homebrew/homebrew-core](https://github.com/Homebrew/homebrew-core) and [Homebrew/homebrew-cask](https://github.com/Homebrew/homebrew-cask). A GitHub Action in [homebrew-core](https://github.com/Homebrew/homebrew-core/blob/master/.github/main.workflow) or [homebrew-cask](https://github.com/Homebrew/homebrew-cask/blob/master/.github/main.workflow) is run on each commit, which deploys the site to GitHub Pages.\n\n## JSON API\n\nIt also provides a JSON API for all packages (or individual packages) in [Homebrew/homebrew-core](https://github.com/Homebrew/homebrew-core) and their related analytics. This JSON data is used for the creation of the HTML resources on this site.\n\nCurrently available:\n- List formulae metadata for all homebrew-core formulae\n- Get formula metadata for a homebrew-core formula\n- List analytics events\n- List analytics events for all homebrew-core formulae\n\nRead more in the [JSON API documentation](https://formulae.brew.sh/docs/api/).\n\n## Usage\nOpen https://formulae.brew.sh/ in your web browser.\n\nTo instead run Homebrew Formulae locally, run:\n```bash\ngit clone https://github.com/Homebrew/formulae.brew.sh\ncd formulae.brew.sh\nbundle install\nbundle exec jekyll serve\n```\n\n## License\nCode is under the [BSD 2-clause "Simplified" License](LICENSE.txt).\n'
