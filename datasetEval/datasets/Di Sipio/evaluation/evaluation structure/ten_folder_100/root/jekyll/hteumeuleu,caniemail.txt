b"# Can I email\xe2\x80\xa6\n\n## Table of Contents\n+ [About](#about)\n+ [Links](#links)\n+ [Getting Started](#getting-started)\n+ [Usage](#usage)\n+ [Contributing](CONTRIBUTING.md)\n\n## About\n\ncaniemail.com provides email clients support tables for HTML and CSS features.\n\n## Links\n\n- https://www.caniemail.com/\n\n## Getting started\n\nCaniemail.com runs on [Jekyll](https://jekyllrb.com/) as a [GitHub Pages](https://pages.github.com/) site with a home made theme.\n\nThe site uses [Sass](https://sass-lang.com/) for CSS compilation.\n\n1. Clone the repository. \n\n```\ngit clone https://github.com/hteumeuleu/caniemail.git\n```\n\nSee [Cloning a repository](https://help.github.com/en/articles/cloning-a-repository) on GitHub documentation.\n\nIf you're not familiar with Git or GitHub, I strongly encourage you to try [GitHub's desktop app](https://desktop.github.com/) on macOS or Windows.\n\n2. Install Jekyll.\n\n```\ngem install bundler jekyll\n```\n\nSee [Jekyll Installation Guide](https://jekyllrb.com/docs/installation/).\n\n3. Run Jekyll.\n\n```\nbundle exec jekyll serve\n```\n\n4. Go to [http://localhost:4000](http://localhost:4000).\n\n## Usage\n\n### Folders structure\n\nHere is a quick overview of the project's folders structure:\n\n* Data:\n\n    - `_data`: Data files used throughout the site. This is mainly for settings and labels.\n    - `_features`: Data for every HTML and CSS features support.\n    - `_posts`: Data for the latest news.\n    - `tests`: HTML tests files to test HTML and CSS features.\n\n* Layout:\n\n    - `_includes`: Files included in other theme files.\n    - `_layouts`: The main layouts of the site.\n    - `_sass`: Sass files. All files will be compiled into one in the `assets/css/` folder.\n    - `assets`: CSS, images, and JavaScript files.\n"
