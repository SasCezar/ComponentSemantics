b'# memo\n\nMemo Life For You\n\n![Memo Life For You](https://raw.githubusercontent.com/mattn/memo/master/screenshot.gif)\n\n## Usage\n\n```\nNAME:\n   memo - Memo Life For You\n\nUSAGE:\n   memo [global options] command [command options] [arguments...]\n\nVERSION:\n   0.0.4\n\nCOMMANDS:\n     new, n     create memo\n     list, l    list memo\n     edit, e    edit memo\n     delete, d  delete memo\n     grep, g    grep memo\n     config, c  configure\n     serve, s   start http server\n     help, h    Shows a list of commands or help for one command\n\nGLOBAL OPTIONS:\n   --help, -h     show help\n   --version, -v  print the version\n```\n\n## Installation\n\n```\n$ go get github.com/mattn/memo\n```\n\nLet\'s start create memo file.\n\n```\n$ memo new\nTitle:\n```\n\nInput title for the memo, then you see the text editor launched. After saving markdown, list entries with `memo list`.\n\n```\n$ memo list\n2017-02-07-memo-command.md   : Installed memo command\n```\n\nAnd grep\n\n```\n$ memo grep command\n2017-02-07-memo-command.md:1:# Installed memo command\n```\n\n## Configuration\n\nrun `memo config`.\n\n```toml\nmemodir = "/path/to/you/memo/dir" # specify memo directory\nmemotemplate = "path/to/tmpl.txt" # optional memo template file. default \'~/.config/memo/template.txt\'\neditor = "vim"                    # your favorite text editor\ncolumn = 30                       # column size for list command\nselectcmd = "peco"                # selector command for edit command\ngrepcmd = "grep -nH"              # grep command executable\nassetsdir = "/path/to/assets"     # assets directory for serve command\npluginsdir = "path/to/plugins"    # plugins directory for plugin commands. default \'~/.config/memo/plugins\'.\n```\n\nmemodir, memotemplate and assetsdir can be used `~/` prefix or `$HOME` or OS specific environment variables. editor, selectcmd and grepcmd can be used placeholder below.\n\n|placeholder|replace to     |\n|-----------|---------------|\n|${FILES}   |target files   |\n|${DIR}     |same as memodir|\n|${PATTERN} |grep pattern   |\n\n## Memo Template\n\nYou can use memo template using Go\'s text/template format. A template receives the following attributes.\n\n- Title\n- Date (format: %Y-%m-%d %H:%M)\n- Categories (always empty)\n- Tags (always empty)\n\nThe following is a template example to apply YAML Frontmatter.\n\n```\n---\ntitle: {{.Title}}\ndate: {{.Date}}\n---\n\n{{.Title}}\n===========\n```\n\nYou can also use glidenote/memolist.vim\'s template format like following.\n\n```\ntitle: {{_title_}}\n==========\ndate: {{_date_}}\ntags: [{{_tags_}}]\ncategories: [{{_categories_}}]\n----------\n```\n\n## Supported GrepCmd\n\n\n|Command                                            |Configuration                       |\n|---------------------------------------------------|------------------------------------|\n|GNU Grep                                           |grepcmd = "grep -nH" #default       |\n|[ag](https://github.com/ggreer/the_silver_searcher)|grepcmd = "ag ${PATTERN} ${DIR}"    |\n|[jvgrep](https://github.com/mattn/jvgrep)          |grepcmd = "jvgrep ${PATTERN} ${DIR}"|\n\n## Supported SelectCmd\n\n|Command                               |Configuration    |\n|--------------------------------------|-----------------|\n|[gof](https://github.com/mattn/gof)   |selectcmd = "gof"|\n|[cho](https://github.com/mattn/cho)   |selectcmd = "cho"|\n|[fzf](https://github.com/junegunn/fzf)|selectcmd = "fzf"|\n\n## Extend With Plugin Commands\n\nYou can extend memo with custom commands. \nPlace an executable file in your `pluginsdir`, memo can use it as a subcommand.\nFor example, If you place `foo` file in your `pluginsdir`, you can run it by `memo foo`.\n\nBelow is spec of plugins:\n\n* MUST handle `-usage` option to show briefly, at least.\n* MUST NOT handle `--xxx` option.\n* MUST NOT use multi-byte strings in the usage.\n\n## License\n\nMIT\n\n## Author\n\nYasuhiro Matsumoto (a.k.a. mattn)\n'
