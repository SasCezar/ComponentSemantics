b'# Burger\nBurger is a minimal hamburger menu with fullscreen navigation. It is created by mblode.\n\nThis project officially requires zero external bower dependencies. Woo-hoo!\n\n## Burger in action\n![Burger Gif](http://i.imgur.com/spSuHHZ.gif)\n\n## [Demo on Codepen](http://codepen.io/mblode/pen/qEGWwB)\n\n## Screenshots\n![Burger: Closed](http://i.imgur.com/eElVkQI.png)\n\n![Burger: Opened](http://i.imgur.com/tI0ZeNw.png)\n\n## Quick Start\nSeveral quick start options are available:\n\n- Install with [Bower](http://bower.io): `bower install burger` (recommended).\n- [Download the latest release](https://github.com/mblode/burger/archive/master.zip).\n- Clone the repo: `git clone https://github.com/mblode/burger.git`.\n\nif you have cloned the repo or downloaded from .zip, there are a few steps you must take within the terminal.\n\n1. Change directory: `cd burger`.\n2. Install node modules: `npm install`.\n3. Install scss-lint Ruby gem: `gem install scss-lint`.\n4. To run gulp server: `gulp`.\n5. To run build process: `gulp build`.\n\n## Running Github Pages\nThe gh-pages branch is built using Jekyll and must therefore be install with `gem install jekyll`.\n\n1. Checkout in to gh-pages: `git checkout gh-pages`.\n2. Install burger dependency: `bower install`.\n3. Run jekyll: `jekyll serve`.\n4. Open in browser: `localhost:4000/burger/`.\n\n\n## What\'s Included\nThese are the files that are generated from `bower install burger`\n\n```\n.\n\xe2\x94\x9c\xe2\x94\x80\xe2\x94\x80 README.md\n\xe2\x94\x9c\xe2\x94\x80\xe2\x94\x80 bower.json\n\xe2\x94\x94\xe2\x94\x80\xe2\x94\x80 dist\n    \xe2\x94\x9c\xe2\x94\x80\xe2\x94\x80 css\n    \xe2\x94\x82\xc2\xa0\xc2\xa0 \xe2\x94\x9c\xe2\x94\x80\xe2\x94\x80 burger..min.css\n    \xe2\x94\x82\xc2\xa0\xc2\xa0 \xe2\x94\x94\xe2\x94\x80\xe2\x94\x80 burger.min.css.map\n    \xe2\x94\x9c\xe2\x94\x80\xe2\x94\x80 index.html\n    \xe2\x94\x9c\xe2\x94\x80\xe2\x94\x80 scripts\n    \xe2\x94\x82\xc2\xa0\xc2\xa0 \xe2\x94\x9c\xe2\x94\x80\xe2\x94\x80 burger.js\n    \xe2\x94\x82\xc2\xa0\xc2\xa0 \xe2\x94\x9c\xe2\x94\x80\xe2\x94\x80 burger.min.js.map\n    \xe2\x94\x82\xc2\xa0\xc2\xa0 \xe2\x94\x94\xe2\x94\x80\xe2\x94\x80 burger.min.js\n    \xe2\x94\x94\xe2\x94\x80\xe2\x94\x80 sass\n        \xe2\x94\x94\xe2\x94\x80\xe2\x94\x80 burger.scss\n```\n\n## Documentation\n### HTML Markup\n```\n  <!-- Navigation -->\n  <div class="b-nav">\n    <li><a class="b-link b-link--active" href="#">Home</a></li>\n    <li><a class="b-link" href="#">About</a></li>\n    <li><a class="b-link" href="#">Portfolio</a></li>\n    <li><a class="b-link" href="#">Contact</a></li>\n  </div>\n\n  <!-- Burger-Icon -->\n  <div class="b-container">\n    <div class="b-menu">\n      <div class="b-bun b-bun--top"></div>\n      <div class="b-bun b-bun--mid"></div>\n      <div class="b-bun b-bun--bottom"></div>\n    </div>\n\n    <!-- Burger-Brand -->\n    <a href="#" class="b-brand">Burger</a>\n  </div>\n```\n\n## Browser Compatibility\n* Safari 6.1+\n* IE 10+\n* Firefox 29+\n* Chrome 26+\n* Opera 17+\n\n## Contributing to Burger\n\nPull requests are the way to go.\n\n\n## Creators\n\n**Matthew Blode**\n- <https://twitter.com/mblode>\n- <https://github.com/mblode>\n- <http://codepen.io/mblode>\n\n## License\nMIT \xc2\xa9 [Matthew Blode](http://mblode.github.io)\n'
