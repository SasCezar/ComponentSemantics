b'dbyll\n=====\n\nOpen source stylish, minimalistic theme for jekyll.  \nDemo: http://dbtek.github.io/dbyll/\n\n## New\n- Dbyll for Ghost is now available. Get it [here](https://github.com/dbtek/dbyll-ghost) now.\n- :new: Check out [Paper](https://github.com/dbtek/paper), new Material Design Jekyll theme.\n\n### Features\n- Responsive layout.\n- Supports tags and categories.\n- Social profile and bio of author.\n- Bootstrap based.\n- Glyphicon and Font-Awesome Icons.\n- Pagination.\n- Syntax highlighting with pygments.\n- Disqus comments.\n- :new: Custom sidebar background image.\n\n\n### Download\n* [Download dbyll](https://github.com/dbtek/dbyll/archive/master.zip)\n\n### Install\n- You need to have [ruby](https://www.ruby-lang.org/en/documentation/installation/) installed in your system.\n- Install bundler which helps in specifying and installing dependencies of any Ruby project. ```gem install bundler```\n- Go to the root of the repo and run this - ```bundle install```\n- Then - ```bundle exec jekyll serve```\n\n### Screenshots\n![dbyll-screenshot](assets/media/dbyll-ss.png)\n\n### License\n- [MIT](http://opensource.org/licenses/MIT)\n\n'
