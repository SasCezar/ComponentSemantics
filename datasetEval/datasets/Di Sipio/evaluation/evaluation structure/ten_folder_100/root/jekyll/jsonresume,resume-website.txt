b'# JSON Resume Homepage\n\n[![Join the chat at https://gitter.im/jsonresume/public](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/jsonresume/public?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)\n\nVisit the website at https://jsonresume.org/\n\n## Development\n\nThere are two ways to run this website, locally or inside a VM.\n\n### Locally\n\nDependencies:\n\n- Git\n- Debian-based OS (e.g. Ubuntu)\n\n#### 1. Clone repository\n\n```\ngit clone http://github.com/jsonresume/homepage\n```\n\n#### 2. Install Jekyll\n\nStart by installing this stuff:\n\n```\nsudo apt-get install ruby ruby-dev make\n```\n\n`ruby` and `gem` should now have been installed on your computer.\n\nNow go ahead and install Jekyll:\n\n```\nsudo gem install jekyll --no-rdoc --no-ri\n```\n\n#### 3. Run Jekyll\n\nStep into the `homepage/` folder and run:\n\n```\njekyll serve --watch\n```\n\nThis will run the Jekyll server and watch for any file changes.\n\n### VM\n\nDependencies:\n\n- Vagrant\n- VirtualBox\n\nSimply run:\n\n```\nvagrant up\nvagrant ssh\njekyll serve\n```\n\nThen open [http://localhost:4000/](http://localhost:4000/).\n\nThe initial provision will take some time.\n\n**Note:** The `--watch` flag for Jekyll has issues inside the VM, see [Stack Overflow](https://stackoverflow.com/questions/19822319/jekyll-regeneration-doesnt-work-inside-vagrant) and [jekyll/jekyll#1763](https://github.com/jekyll/jekyll/issues/1763).\n\n## Resources\n\nInstalling Jekyll on Ubuntu:\nhttp://michaelchelen.net/81fa/install-jekyll-2-ubuntu-14-04/\n\n## License\n\nAvailable under [the MIT license](http://mths.be/mit).\n'
