b"# Octicons\n\n[![octicons](https://user-images.githubusercontent.com/54012/37737576-5d1e8c7a-2d11-11e8-8fd9-13956a241549.png)][figma-file]\n\n[![Build Status](https://travis-ci.org/primer/octicons.svg?branch=master)](https://travis-ci.org/primer/octicons)\n\nOcticons are a set of SVG icons built by GitHub for GitHub. This repository is a mono repository that contains [distribution libraries for other platforms/languages](#libraries). The vectors for all the icons are stored in a [Figma file][figma-file].\n\n## Adding/Updating an icon\n\nRead through our [contributing guide](./.github/CONTRIBUTING.md#adding-or-updating-icons) if you're planning on adding or updating one of the icons.\n\n## Libraries\n\nThis repository is a repository containing other packages. Each of them is in the `lib/` folder and gives access to Octicons on a different platform / language.\n\n### JavaScript\n\nThe octicons node.js library is the main JavaScript library.
