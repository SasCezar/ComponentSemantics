b'# jekyll-import\n\n[![Build Status](https://travis-ci.org/jekyll/jekyll-import.svg?branch=master)](https://travis-ci.org/jekyll/jekyll-import)\n\nThe new __Jekyll__ command for importing from various blogs to Jekyll format.\n\n**Note: _migrators_ are now called _importers_ and are only available if one installs the `jekyll-import` _gem_.**\n\n## How `jekyll-import` works:\n\n### Jekyll v2.x and higher\n\n1. Install the _rubygem_ with `gem install jekyll-import`.\n2. Run `jekyll import IMPORTER [options]`\n\n### Jekyll v1.x\n\nLaunch IRB:\n\n```ruby\n# 1. Require jekyll-import\nirb> require \'jekyll-import\'\n# 2. Choose the importer you\'d like to use.\nirb> importer_class = "Behance" # an example, there are many others!\n# 3. Run it!\nirb> JekyllImport::Importers.const_get(importer_class).run(options_hash)\n```\n\n## Documentation\n\njekyll-import has its own documentation site, found at https://import.jekyllrb.com.\nDedicated [documentation for each migrator](https://import.jekyllrb.com/docs/home/) is available there.\n'
