b'# end2end Clean Jekyll theme [![Build Status](https://travis-ci.org/nandomoreirame/end2end.svg?branch=master)](https://travis-ci.org/nandomoreirame/end2end)\n\n\n\n* [x] Clean layout\n* [x] Resposive layout\n* [x] Preprocessor SASS\n* [x] CSS minified\n* [x] Pagination\n* [x] Syntax highlight\n* [x] Author config\n* [x] Comments with Disqus\n* [ ] Search posts\n* [ ] Share posts\n\n---\n\n### Start in 4 steps\n\n1. Download or clone repo `git clone git@github.com:nandomoreirame/end2end.git`\n2. Enter the folder: `cd end2end/`\n3. Install Ruby gems: `bundle install`\n4. Start Jekyll server: `bundle exec jekyll serve`\n\nAccess, [localhost:4000/end2end](http://localhost:4000/end2end)\n\n### Deploy in Github pages in 2 steps\n\n1. Change the variables `GITHUB_REPONAME` and `GITHUB_REPO_BRANCH` in `Rakefile`\n2. Run `rake` or `rake publish` for build and publish on Github\n\n---\n\n### Using Rake tasks\n\n* Create a new page: `rake page name="contact.md"`\n* Create a new post: `rake post title="TITLE OF THE POST"`\n\n---\n\n### Demo and Download\n\n[Demo](https://nandomoreirame.github.io/end2end/)\n[Download](https://github.com/nandomoreirame/end2end/archive/master.zip)\n\n![end2end - free Jekyll theme](/screenshot.png)\n\n---\n\n### Copyright and license\n\nIt is under [the MIT license](/LICENSE).\n\n> :warning:\n  Please remove metas `<meta name="robots" content="noindex">` and `<meta name="googlebot" content="noindex">` in `source/_layouts/default.html`\n\nEnjoy :yum:\n'
