b"# Termux packages\n\n[![Powered by JFrog Bintray](./.github/static/powered-by-bintray.png)](https://bintray.com)\n\n[![build status](https://api.cirrus-ci.com/github/termux/termux-packages.svg?branch=master)](https://cirrus-ci.com/termux/termux-packages)\n[![Join the chat at https://gitter.im/termux/termux](https://badges.gitter.im/termux/termux.svg)](https://gitter.im/termux/termux)\n\nThis project contains scripts and patches to build packages for the [Termux]\nAndroid application.\n\nThere available packages only from main set. We have some additional\nrepositories:\n\n- https://github.com/termux/game-packages\n\n  Game packages, e.g. `angband` or `moon-buggy`.\n\n- https://github.com/termux/science-packages\n\n  Science-related packages like `gap` and `gnucap`.\n\n- https://github.com/termux/termux-root-packages\n\n  All packages which usable only on rooted devices. Some stuff available\n  here requiring custom kernel (like `aircrack-ng` or `lxc`).\n\n- https://github.com/termux/unstable-packages\n\n  Staging repository. Packages that are not stable available only here. New\n  packages most likely will be placed here too.\n\n- https://github.com/termux/x11-packages\n\n  Packages requiring X11 Windowing System.\n\nTermux package management quick how-to available on https://wiki.termux.com/wiki/Package_Management.\nTo learn about using our build environment, read the [Developer's Wiki].\n\n## Project structure\n\nThere 2 main branches available:\n\n- [master] - packages for Android 7.0 or higher.\n\n  Packages are built automatically by [CI] and published on [Bintray].\n\n- [android-5] - packages for Android versions 5.x - 6.x.\n\n  Packages are built by @fornwall and published on https://termux.net.\n\nDirectories:\n\n- [disabled-packages](disabled-packages/):\n\n  Packages that cannot be published due to serious issues.\n\n- [ndk-patches](ndk-patches/):\n\n  Our changes to Android NDK headers.\n\n- [packages](packages/):\n\n  Main set of packages.\n\n- [sample](sample/):\n\n  Sample structure for creating new packages.\n\n- [scripts](scripts/):\n\n  Set of utilities and build system scripts.\n\n## Contributing\n\n### Bug reports\n\nPlease, use templates for submitting bug reports. The *bug report* issue template\ncan be initialized by clicking on https://github.com/termux/termux-packages/issues/new?template=bug_report.md.\n\nGeneral requirements for bug reports are:\n\n- All packages are up-to-date.\n\n- Problem is not related to third-party software.\n\n- Output of `termux-info` attached.\n\n- Be ready to provide more info if requested.\n\n### New packages\n\nUse the *package request* template: https://github.com/termux/termux-packages/issues/new?template=package_request.md.\n\nGeneral requirements for new packages are:\n\n- Packages should be open source and have widely recognised OSS licenses like\n  GNU GPL.\n\n- Packages should not be installable via language-specific package managers such\n  as `gem`, `pip` or `cpan`.\n\n- Packages should not be outdated dead projects.\n\n- Be ready that your package request will not be processed immediately.\n\n## Pull Requests\n\nAll pull requests are welcome.\n\nWe use [CI] for processing all pushes including pull requests. All build logs\nand artifacts are public, so you can verify whether your changes work properly.\n\nPeople who are new for packaging can begin with sending PRs for updating\npackages. Check the outdated packages on https://repology.org/projects/?inrepo=termux&outdated=1.\n\nGet started with information available on [Developer's Wiki].\n\n## Contacts\n\n- General Mailing List: https://groups.io/g/termux\n\n- Developer Mailing List: https://groups.io/g/termux-dev\n\n- Developer Chat: https://gitter.im/termux/dev or #termux/development on IRC/freenode.\n\nIf you are interested in our weekly development sessions, please check the\nhttps://wiki.termux.com/wiki/Dev:Development_Sessions. Also, you may want to\ncheck the https://wiki.termux.com/wiki/Development.\n\n[Bintray]: <https://bintray.com/termux/termux-packages-24>\n[CI]: <https://cirrus-ci.com/termux/termux-packages>\n[Developer's Wiki]: <https://github.com/termux/termux-packages/wiki>\n[Termux]: <https://github.com/termux/termux-app>\n[android-5]: <https://github.com/termux/termux-packages/tree/android-5>\n[master]: <https://github.com/termux/termux-packages/tree/master>\n"
