your web page, simply paste this code into the `head` of your document,\r\n```html\r\n<link rel="stylesheet" href="https://unpkg.com/waffle-grid/dist/waffle-grid.min.css">\r\n<!-- or -->\r\n<link rel="stylesheet" href="https://unpkg.com/waffle-grid/dist/waffle-grid.css">\r\n```\r\nor you can install the waffle grid system using a package manager like [npm](https://npmjs.com/package/waffle-grid)\r\n```bash\r\n$ npm install waffle-grid\r\n# or\r\n$ bower install waffle-grid\r\n```\r\nIf you want you can still download the waffle grid [here](https://github.com/lucasgruwez/waffle-grid/releases).\r\n\r\n## Docs\r\n\r\n```html\r\n<div class="grid">\r\n  <div class="row">\r\n    <!-- For column width, use col-n-of-p. By default, p can be 4, 8, or 12 -->\r\n    <div class="col col-1-of-4"></div>\r\n    <div class="col col-1-of-4"></div>\r\n    <div class="col col-1-of-4"></div>\r\n    <div class="col col-1-of-4"></div>\r\n  </div>\r\n  <div class="row">\r\n    <!-- The center class centers the column. -->\r\n    <div class="col col-7-of-12 center"></div>\r\n  </div>\r\n  <div class="row">\r\n    <!-- use col-m- and col-s- as well as m-hide, s-hide, m-only and s-only to make your grid responsive -->\r\n    <div class="col col-2-of-8 col-m-1-of-4 col-s-1-of-2"></div>\r\n    <div class="col col-3-of-8 col-m-2-of-4 col-s-1-of-2"></div>\r\n    <div class="col col-1-of-8 m-hide"></div>\r\n    <div class="col col-2-of-8 col-m-1-of-4 s-hide"></div>\r\n  </div>\r\n  <div class="row">\r\n    <div class="col col-1-of-4"></div>\r\n    <div class="col col-1-of-2"></div>\r\n    <div class="col col-1-of-4"></div>\r\n  </div>\r\n  <div class="row">\r\n    <!-- Pushes and pulls are used in the same way as col-n-of-p. They can also be responsive, like pull-m-1-of-4 -->\r\n    <div class="col col-1-of-8"></div>\r\n    <div class="col col-2-of-8 push-2-of-12"></div>\r\n    <div class="col col-3-of-8 pull-1-of-12"></div>\r\n  </div>\r\n</div>\r\n```\r\n\r\nThe full docs can be found [here](https://github.com/lucasgruwez/waffle-grid/wiki)\r\n\r\n## Like this project?\r\n\r\nPlease feel free to give the waffle-grid repo a star, and if you want to help support my work, please consider [donating](https://paypal.me/lucasgruwez).\r\n\r\n## Built With\r\n\r\n* [Gulp](http://gulpjs.com/) - Build system\r\n* [Sass](https://sass-lang.com/) - Language\r\n* [Jekyll](https://jekyllrb.com/) - Used for the website\r\n\r\n## Contributing\r\n\r\nPlease read [CONTRIBUTING.md](./.github/CONTRIBUTING.md) for details on our code of conduct, and the process for submitting pull requests to us.\r\n\r\n#### Made a website using the Waffle grid?\r\n\r\nAdd the built with waffle-grid badge to your `README.md`\r\n\r\n[![Built with Waffle Grid](https://lucasgruwez.github.io/waffle-grid/img/built-with-waffle.svg)](https://lucasgruwez.github.io/waffle-grid/img/built-with-waffle.svg)\r\n\r\n\r\nFeel free to send me an email at [gruwez.lucas@gmail.com](mailto:gruwez.lucas@gmail.com), and I might add your site to an examples section I\'m currently working on.\r\n\r\n## Versioning\r\n\r\nWe use [SemVer](http://semver.org/) for versioning.
