Started\n\nCurrently there are only three ways of downloading Shards:\n\n* [Official Website](https://designrevision.com/download/shards)\n* [Release Package](https://github.com/DesignRevision/shards-ui/releases)\n* [`npm`](https://www.npmjs.com/package/shards-ui)\n\nThe distributed Shards assets are also available via CDN through [unpkg](https://unpkg.com/shards-ui@latest/) and [jsDelivr](https://cdn.jsdelivr.net/npm/shards-ui@latest/).\n\nTo install with `npm`:\n\n```bash\nnpm install --save shards-ui bootstrap\n```\n\n<br />\n\n### Dependencies\n\nThe only hard dependency is Bootstrap 4.3.1 which needs to be included in your HTML document before Shards. Datepickers and slider controls also require the Shards JavaScript dependency.\n\n<br />\n\n### Quick Start\n\nIf you\'re using a bundler like `webpack`, you should skip to [Usage with Bundlers](#usage-with-bundlers).\n\nIn order to take advantage of both Bootstrap and Shards\' features you may want to use the following starter template that includes all dependencies.\n\n```html\n<!DOCTYPE html>\n<html lang="en">\n  <head>\n    <meta charset="utf-8">\n    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">\n\n    <!-- CSS Dependencies -->\n    <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css" integrity="sha384-MCw98/SFnGE8fJT3GXwEOngsV7Zt27NXFoaoApmYm81iuXoPkFOJwJ8ERdknLPMO" crossorigin="anonymous">\n    <link rel="stylesheet" href="https://unpkg.com/shards-ui@latest/dist/css/shards.min.css">\n  </head>\n  <body>\n    <h1>Hello, world!</h1>\n\n    <!-- Optional JavaScript -->\n    <!-- JavaScript Dependencies: jQuery, Popper.js, Bootstrap JS, Shards JS -->\n    <script src="https://code.jquery.com/jquery-3.3.1.slim.min.js" integrity="sha384-q8i/X+965DzO0rT7abK41JStQIAqVgRVzpbzo5smXKp4YfRvH+8abtTE1Pi6jizo" crossorigin="anonymous"></script>\n    <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.3/umd/popper.min.js" integrity="sha384-ZMP7rVo3mIykV+2+9J3UJ46jBk0WLaUAdn689aCwoqbBJiSnjAK/l8WvCWPIPm49" crossorigin="anonymous"></script>\n    <script src="https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/js/bootstrap.min.js" integrity="sha384-ChfqqxuZUCnJSK3+MXmPNIyE6ZbWh2IMqE241rYiqJxyMiZ6OW/JmZQ5stwEULTy" crossorigin="anonymous"></script>\n    <script src="https://unpkg.com/shards-ui@latest/dist/js/shards.min.js"></script>\n  </body>\n</html>\n```\n\n<br />\n\n### Usage with Bundlers\n\nIf you\'re using a bundler like [`webpack`](https://webpack.js.org) (with [`style-loader`](https://github.com/webpack-contrib/style-loader)/[`css-loader`](https://github.com/webpack-contrib/css-loader)) or [`parcel`](https://parceljs.org), you can [install `shards` with `npm`](#getting-started) and import it:\n\n```javascript\n// app.js\nimport \'bootstrap/dist/css/bootstrap.min.css\';\nimport \'shards-ui/dist/css/shards.min.css\';\n// Optional JavaScript\n// JavaScript Dependencies: jQuery, Popper.js, Bootstrap JS, Shards JS\n// Install them with `npm` as well\nimport \'jquery/dist/jquery.slim.min\';\nimport \'popper.js/dist/umd/popper.min\';\nimport \'bootstrap/dist/js/bootstrap.min\';\nimport \'shards-ui/dist/js/shards.min\';\n```\n\n<br/>\n\n### Built using\n\n*
