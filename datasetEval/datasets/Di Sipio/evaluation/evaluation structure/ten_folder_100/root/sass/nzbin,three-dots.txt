b'# Three Dots\n\n[![npm](https://img.shields.io/npm/v/three-dots.svg)](https://www.npmjs.com/package/three-dots)\n[![license](https://img.shields.io/github/license/mashape/apistatus.svg)](https://github.com/nzbin/three-dots)\n[![Financial Contributors on Open Collective](https://opencollective.com/three-dots/all/badge.svg?label=financial+contributors)](https://opencollective.com/three-dots) \n\nThe project is a set of CSS loading animations created with three dots which made by just single element. I think the project can not only enhance your CSS skills but also improve your imagination.\n\n> The LESS files are contributed by other people, it\'s not the newest.\n\n## Installation\n\n```sh\n$ npm install three-dots --save\n```\n\n## Usage\n\n1. Include the CSS in your file:\n\n```html\n<link href="/path/to/three-dots.css" rel="stylesheet">\n```\n\n2. Add just one `div` tag:\n\n```html\n<div class="dot-elastic"></div>\n```\n\nHere\xe2\x80\x99s the list of three-dots classes you can choose from:\n\n```\ndot-elastic\ndot-pulse\ndot-flashing\ndot-collision\ndot-revolution\ndot-carousel\ndot-typing\ndot-windmill\ndot-bricks\ndot-floating\ndot-fire\ndot-spin\ndot-falling\ndot-stretching\n```\n\n## Customization\n\n1. Checkout this repo and\n\n```bash\n$ npm install\n```\n\n2. Customize the dots size ( color, spacing, etc. ) in the `_variables.scss` file.\n\n```scss\n$dotWidth: 10px;\n$dotHeight: 10px;\n$dotRadius: $dotWidth/2;\n\n$dotColor: #9880ff;\n$dotBgColor: $dotColor;\n$dotBeforeColor: $dotColor;\n$dotAfterColor: $dotColor;\n\n$dotSpacing: $dotWidth + $dotWidth/2;\n```\n\n3. Compile Sass files\n\n```bash\n$ npm run build\n```\n\n## Contributors\n\n### Code Contributors\n\nThis project exists thanks to all the people who contribute.
