Goto [https://middlemanapp.com/](https://middlemanapp.com/) )\n\n### Alternative install\n- `$ npm install family.scss`\n- `$ bower install family.scss`\n- `gem install family-rails` ([maintained by pzi](https://github.com/pzi/family-rails))\n\nFamily.scss on [npm](https://www.npmjs.com/package/family.scss)\n\n## Usage\nFirst of all, you need to import Family.scss into your project. If you\'re using [eyeglass](http://eyeglass.rocks/) you can import it as such:\n```scss\n@import "family";\n```\nOtherwise import the [Family.scss source file](https://github.com/LukyVj/family.scss/blob/master/source/src/_family.scss).\n\nThen you can use the mixins right away on your stylesheets.\n\nInput :\n```scss\nul li {\n  background: blue;\n\n  @include first(3) {\n    background: blue;\n  }\n}\n```\n\nOutput :\n```sass\nul li {\n  background: blue;\n}\nul li:nth-child(-n + 3) {\n  background: blue;\n}\n\n```\n\n### Why only Sass ?\nIt\'s
