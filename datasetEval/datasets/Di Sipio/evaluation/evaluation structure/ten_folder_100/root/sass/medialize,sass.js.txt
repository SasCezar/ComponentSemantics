b'# Sass.js\n\nSass parser in JavaScript. Have a look at the [Interactive Playground](http://sass.js.org/) to play around with compiling SCSS to CSS in your browser.\n\nThis is a convenience API for [emscripted](https://github.com/kripken/emscripten) [libsass](https://github.com/sass/libsass) (at [v3.6.2](https://github.com/sass/libsass/releases/tag/3.6.2)). If you\'re looking to run Sass in node, you\'re probably looking for [node-sass](https://github.com/sass/node-sass). Sass.js and node-sass should generate the same results.\n\nA fair warning: minified the worker weighs 4.5MB, gzipped it\'s still 827KB. If you\'re on NodeJS, please use the (considerably faster) [node-sass](https://github.com/andrew/node-sass) instead.\n\nYou may also be interested in giving [Dart Sass](https://github.com/sass/dart-sass) a shot.\n\n\n## Documentation\n\n* [Getting started](./docs/getting-started.md)\n* [API documentation](./docs/api.md)\n* [Building Sass.js](./docs/build.md)\n* [Changelog](./CHANGELOG.md)\n\n\n## Who uses Sass.js\n\nSass.js is used in the following tools:\n\n* [grunt-contrib-sassjs](https://github.com/amiramw/grunt-contrib-sassjs)\n* [mobilexag/plugin-sass](https://github.com/mobilexag/plugin-sass) is a SystemJS plugin\n* [kevcjones/plugin-scss](https://github.com/kevcjones/plugin-scss) is a SystemJS plugin\n* [theefer/plugin-sass](https://github.com/theefer/plugin-sass) is a SystemJS plugin\n* [stealjs/steal-sass](https://github.com/stealjs/steal-sass) is a StealJS plugin\n* [nickheiner/sassjs-loader](https://github.com/NickHeiner/sassjs-loader) is a Webpack plugin\n* [lein-sass](https://github.com/yogthos/lein-sass) is a minimum dependency Clojure library designed to compile SASS files using Sass.js running on Nashorn\n* [qwebs](https://www.npmjs.com/package/qwebs) is a "back-end web server"\n* [Crunch 2](http://getcrunch.co/)\n\n\n## Authors\n\n* [Christian Kruse](https://github.com/ckruse) -
