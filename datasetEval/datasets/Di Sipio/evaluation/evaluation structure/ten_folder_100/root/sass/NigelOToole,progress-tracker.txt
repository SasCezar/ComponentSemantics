b'# SASS Progress Tracker\n### A HTML component to illustrate the steps in a multi step process e.g. a multi step form, a timeline or a quiz.\n\n### [View demo](http://nigelotoole.github.io/progress-tracker/)\n\n\n\n## Installation\n\n```javascript\n$ npm install progress-tracker --save\n```\n\n### Import\n\nAfter installation you can import it into your Sass files with the statement below.\n\n```scss\n@import "node_modules/progress-tracker/src/styles/progress-tracker.scss";\n```\n\nThe JS that is part of this site is just for demonstration purposes, add your own JS as needed to toggle the classes for the step states.\n\n### Markup\n\nFollow the HTML code example below for basic usage; each demo sets the first two steps as complete, the third step as active and the last two steps as inactive.\n\nFor additional styles add modifier classes and additional markup as needed in the examples below. You can add multiple modifier classes to achieve additional styles that those shown below.\n\n```html\n<ul class="progress-tracker">\n  <li class="progress-step is-complete">\n    <div class="progress-marker"></div>\n  </li>\n  <li class="progress-step is-complete">\n    <div class="progress-marker"></div>\n  </li>\n  <li class="progress-step is-active">\n    <div class="progress-marker"></div>\n  </li>\n  <li class="progress-step">\n    <div class="progress-marker"></div>\n  </li>\n  <li class="progress-step">\n    <div class="progress-marker"></div>\n  </li>\n</ul>\n```\n\n\n### Demo site\n\nClone or download from Github.\n\n```javascript\n    $ npm install\n    $ gulp serve\n```\n\n\n### License\nMIT \xc2\xa9 Nigel O Toole\n'
