b'# Gutenberg: A Meaningful Web Typography Starter Kit\n\n[Gutenberg](http://matejlatin.github.io/Gutenberg/) is a flexible and simple-to-use web typography starter kit for web designers and developers. It\xe2\x80\x99s a small step towards a better typography on the web. Beautiful typographic styles can be made by setting base type size, line-height (leading) and measure (max-width).\n\nGutenberg sets the baseline grid to establish a proper vertical rhythm and makes sure all elements fit into it. It sets up the macro typography so you can focus on the micro-typographic details.\n\n> _The backbone of your typography._\n\n> \xe2\x80\x94The Smashing Magazine\n\n[View an Example](http://matejlatin.github.io/Gutenberg/example2) | [Read the Docs](https://github.com/matejlatin/Gutenberg/wiki)\n\n## Installation\n\nGutenberg is built with Sass and [Grunt](http://gruntjs.com/). Fork or download the repository to get started.\n\n```npm install gutenberg-web-type```\n\n\n## Why Gutenberg?\n\nJohannes Gutenberg invented a printing press with movable type more than 500 years ago. His invention led to improved readability of books and enabled a distribution on a large scale. His invention improved a medium and took it to another level. This web typography starter kit aims to do exactly the same for another medium \xe2\x80\x94 the web.\n\n## Contributors\n\n* [@matejlatin](https://github.com/matejlatin)\n* [@marcobiedermann](https://github.com/marcobiedermann)\n* [@GeorgeStrakhov](https://github.com/GeorgeStrakhov)\n\nGutenberg is an open source project licensed under [Creative Commons 3.0](https://creativecommons.org/licenses/by-sa/3.0/).
