b'## \xf0\x9f\x9b\x8d\xef\xb8\x8f Simple ecommerce cart application [![CircleCI](https://circleci.com/gh/jeffersonRibeiro/react-shopping-cart.svg?style=svg)](https://circleci.com/gh/jeffersonRibeiro/react-shopping-cart)\n[![All Contributors](https://img.shields.io/badge/all_contributors-5-orange.svg?style=flat-square)](#contributors)\n\n<p align="center">\n\n  <img src="./doc/react-shopping-cart-min.gif">\n</p>\n\n## Basic Overview - [Live Demo](https://react-shopping-cart-67954.firebaseapp.com/)\n\nThis simple shopping cart prototype shows how React components and Redux can be used to build a\nfriendly user experience with instant visual updates and scaleable code in ecommerce applications.\n\n#### Features\n\n- Add and remove products from the floating cart\n- Sort products by highest to lowest and lowest to highest price\n- Filter products by available sizes\n- Products persist in floating cart after page reloads\n- Unit tests, integration tests and e2e testing\n- Responsive design\n\n## Getting started\n\nTry playing with the code on CodeSandbox :)\n\n[![Edit app](https://codesandbox.io/static/img/play-codesandbox.svg)](https://codesandbox.io/s/74rykw70qq)\n\n## Build/Run\n\n#### Requirements\n\n- Node.js\n- NPM\n\n```javascript\n\n/* First, Install the needed packages */\nnpm install\n\n/* Then start both Node and React */\nnpm start\n\n/* To run the tests */\nnpm run test\n\n/* Running e2e tests */\nnpm run wdio\n\n\n```\n\n## About tests\n\n- Unit tests\n  - All components have at least a basic smoke test\n- Integration tests\n  - Fetch product and add to cart properly\n- e2e\n  - Webdriverio - Add and remove product from cart\n\n### Copyright and license\n\nThe MIT License (MIT).
