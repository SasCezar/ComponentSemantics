Arvados supports a variety of cloud, cluster and HPC environments.\n\nArvados consists of:\n\n* *Keep*: a petabyte-scale content-addressed distributed storage system for managing and\n  storing collections of files, accessible via HTTP and FUSE mount.\n\n* *Crunch*: a Docker-based cluster and HPC workflow engine designed providing\n  strong versioning, reproducibilty, and provenance of computations.\n\n* Related services and components including a web workbench for managing files\n  and compute jobs, REST APIs, SDKs, and other tools.\n\n## Quick start\n\nVeritas Genetics maintains a public installation of Arvados for evaluation and trial use, the [Arvados Playground](https://playground.arvados.org). A Google account is required to log in.\n\nTo try out Arvados on your local workstation, you can use Arvbox, which\nprovides Arvados components pre-installed in a Docker container (requires\nDocker 1.9+).  After cloning the Arvados git repository:\n\n```\n$ cd arvados/tools/arvbox/bin\n$ ./arvbox start localdemo\n```\n\nIn this mode you will only be able to connect to Arvbox from the same host.
