To use it, install it globally using:\n\n```sh\nnpm i -g @pnp/office365-cli\n```\n\nor using yarn:\n\n```sh\nyarn global add @pnp/office365-cli\n```\n\nThe beta version of the Office 365 CLI can be installed by using the `@next` tag:\n\n```sh\nnpm i -g @pnp/office365-cli@next\n```\n\n## Getting started\n\nStart the Office 365 CLI by typing in the command line:\n\n```sh\n$ office365\n\no365$ _\n```\n\nRunning the `office365` command will start the immersive CLI with its own command prompt.\n\nStart managing the settings of your Office 365 tenant by logging in to it, using the `login` command, for example:\n\n```sh\no365$ login\n```\n\n> Office 365 CLI will automatically detect the URL of your tenant based on the account that you use to sign in.\n\nTo list all available commands, type in the Office 365 CLI prompt `help`:\n\n```sh\no365$ help\n```\n\nTo exit the CLI, type `exit`:\n\n```sh\no365$ exit\n```\n\nSee the [User Guide](docs/manual/docs/user-guide/installing-cli.md) to learn more about the Office 365 CLI and its capabilities.\n\n## Sharing is Caring\n\nWe\'d love your help!
