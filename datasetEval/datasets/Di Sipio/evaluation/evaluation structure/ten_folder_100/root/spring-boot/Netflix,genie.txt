b"# Genie\n\n[![Download](https://api.bintray.com/packages/netflixoss/maven/genie/images/download.svg)](https://bintray.com/netflixoss/maven/genie/_latestVersion)\n[![License](https://img.shields.io/github/license/Netflix/genie.svg)](http://www.apache.org/licenses/LICENSE-2.0)\n[![Issues](https://img.shields.io/github/issues/Netflix/genie.svg)](https://github.com/Netflix/genie/issues)\n[![NetflixOSS Lifecycle](https://img.shields.io/osslifecycle/Netflix/genie.svg)]()\n\n## Introduction\n\nGenie is a federated job orchestration engine developed by Netflix. Genie provides REST-ful APIs to run a variety of big\ndata jobs like Hadoop, Pig, Hive, Spark, Presto, Sqoop and more. It also provides APIs for managing the metadata of many \ndistributed processing clusters and the commands and applications which run on them.\n\n## Documentation\n\nSee the official [website](https://netflix.github.io/genie) to find documentation about Genie and specific \ndocumentation for various releases.\n\n## Demo\n\nGenie has demo instructions available for all 3.x.x releases. Please see the release you're interested in demoing \non the [releases](https://netflix.github.io/genie/releases/) page. Click on the release and then demo docs.\n\n## Builds\n\nGenie builds are run on Travis CI [here](https://travis-ci.org/Netflix/genie).\n\n| Branch |
