b"# midi2pico\nA Midi to PICO-8 converter.\n\nRequires [MIDI.lua](http://www.pjb.com.au/comp/lua/MIDI.html) [(direct link)](http://www.pjb.com.au/comp/lua/MIDI.lua)\n```\n# luarocks install midi\n```\nLua 5.1 users will also need a bit32 library:\n```\n# luarocks install bit32\n```\n\n## Usage:\n```\nlua midi2pico.lua somesong.mid songdata.p8\n```\nVarious options are available, run the program with no arguments to get help.\n\n## Tips:\n* Mute problematic channels with `--mute`, timidity's `--mute` argument can aid in finding problematic channels.\n* Halve the time division to possibly allow more mid-note effects.\n* Drums are problematic. Use `--dshift` to shift drum pitch, `--drumvol=n` to change drum volume, or `--mute=10` to remove drums all together.\n"
