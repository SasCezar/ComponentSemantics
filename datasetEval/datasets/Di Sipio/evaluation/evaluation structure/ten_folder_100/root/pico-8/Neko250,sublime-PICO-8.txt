b'# Sublime PICO-8\n\nPICO-8 plugin for the [Sublime Text 3](https://www.sublimetext.com/) editor.\n\n![sublime-PICO-8](https://raw.githubusercontent.com/Neko250/sublime-PICO-8/master/img/screenshot.png)\n\nThis package includes:\n\n- `.p8` language definition (by [Overkill](http://www.lexaloffle.com/bbs/?uid=11331)).\n- Syntax highlighting (by [Overkill](http://www.lexaloffle.com/bbs/?uid=11331)).\n- PICO-8 `.ttf` font (by [RhythmLynx](http://www.lexaloffle.com/bbs/?uid=11704)).\n- Cartridge runner to launch your cart right from Sublime Text.\n- Code completion based on the official [PICO-8 API](http://neko250.github.io/pico8-api/).\n- Code snippets.\n\n----\n\n## Installation\n\n### Using [Package Control](https://packagecontrol.io/)\n\n1. Open the command palette with `ctrl+shift+p` (`cmd+shift+p` in OSX).\n1. Run `Package Control: Install Package` command.\n1. Search for `PICO-8` and install it.\n1. Restart Sublime Text.\n\n### Using Git\n\n#### Linux\n\n```bash\ncd ~/.config/sublime-text-3/Packages\nrm -rf PICO-8\ngit clone https://github.com/Neko250/sublime-PICO-8 PICO-8\n```\n\n#### OSX\n\n```bash\ncd ~/Library/Application\\ Support/Sublime\\ Text\\ 3/Packages\nrm -rf PICO-8\ngit clone https://github.com/Neko250/sublime-PICO-8 PICO-8\n```\n\n#### Windows\n\n```bash\ncd "%APPDATA%\\Sublime Text 3\\Packages"\nrd /s /q PICO-8\ngit clone https://github.com/Neko250/sublime-PICO-8 PICO-8\n```\n\n### Manual Installation\n\n1. Download the files using the GitHub `.zip` download option.\n1. Unzip the files and rename the folder to `PICO-8`.\n1. Find your `Packages` directory:\n\t- Linux: `~/.config/sublime-text-3/Packages`\n\t- Mac OS: `~/Library/Application Support/Sublime Text 3/Packages`\n\t- Windows: `%APPDATA%\\Sublime Text 3\\Packages`\n1. Copy the folder into your Sublime Text `Packages` directory.\n\n----\n\n## Post Installation\n\n### Font Setup\n\nIndependently of the installation method you chose, after installing you\'ll need to setup the PICO-8 font.\n\n1. Open the command palette with `ctrl+shift+p` (`cmd+shift+p` in OSX).\n1. Run `PICO-8: Download Font` command.\n1. Find `PICO-8.ttf` in your downloads folder and install it.\n1. Restart Sublime Text.\n\nTest different font sizes until you find one that goes well with your screen resolution !\n\n### Cartridge Runner Setup\n\nTo setup the Cartridge Runner, open the Command Palette in Sublime Text (ctrl+shift+p / cmd+shift+p) and run "PICO-8: Setup PICO-8 Path".\nThen enter the path to PICO-8 in the prompt input.\n\n1. Open the command palette with `ctrl+shift+p` (`cmd+shift+p` in OSX).\n1. Run `PICO-8: Setup PICO-8 Path` command.\n1. Type the path to your PICO-8 executable. Defaults to:\n\t- Linux: `/there/is/no/default/in/linux/shrug/pico8`\n\t- OSX: `/Applications/PICO-8.app/Contents/MacOS/pico8`\n\t- Windows: `C:\\\\Program Files (x86)\\\\PICO-8\\\\pico8.exe`\n\n__IMPORTANT__: Remember using the escape character for the backslash (`\\\\`) instead of a single one (`\\`) in Windows !!!\n\n----\n\n## Version History\n\n- __v2017.3.14__:\n\t- `added`: automatically lowercase the file upon saving.\n\t- `added`: `extcmd` function (syntax and autocomplete).\n\t- `added`: raspberry pi / pocketchip gpio snippet.\n\t- `added`: `forpairs` block snippet.\n\t- `added`: `repeat-until` block snippet.\n\t- `added`: `goto` block snippet.\n\t- `added`: semantic comments inside block snippets.\n\t- `changed`: updated api autocompletions to match PICO-8 0.1.10.\n\t- `changed`: `forin` block snippet renamed to `forall`.\n\t- `fixed`: hex number highlighting when not preceded by space.\n\t- `fixed`: comment start definition (`Toggle Comment` from the Command Palette).\n\n- __v2016.7.1__:\n\t- `added`: `.no-sublime-package`.\n\t- `changed`: package is no longer compressed.\n\t- `fixed`: cartridge runner is now visible to sublime.\n\n- __v2016.6.30__:\n\t- `added`: command palette download font shortcut: `PICO-8: Download Font`.\n\t- `added`: command palette preferences: `Preferences: PICO-8 -
