Make sure you have python3 installed on your OS\n1. Download [picotool](https://github.com/dansanderson/picotool)\n2. Generate a lua-minified version of our p8 file:\n  ```console\n  /path/to/picotool/p8tool luamin ./sk8border.p8\n  ```\n3. Now we\'ll have a new file called `sk8border_fmt.p8`. Don\'t save this, but we\'ll load it in PICO-8!\n4. Inside of PICO-8 run:\n  ```console\n  load sk8border_fmt.p8\n  export index.js\n  load sk8border.p8\n  ```\n5. Delete the minified file: `rm sk8border_fmt.p8`\n\n### Minified p8.png export\n\nFollow the same steps as the regular minified build, but:\n\n- Before step 4, copy the code comment from the top of the Lua section in sk8border.p8 (`-- lay that wall to waste!`) and paste it into sk8border_fmt.p8 at the same spot. We want this for the png export.\n- Instead of `export index.js` do `save sk8border.p8.png`.\n\n### Finish up\n\nCommit changes:\n\n```console\ngit add index.js sk8border.p8.png && git commit -m "Built."\n```\n\nPush:\n\n```console\ngit push origin gh-pages\n```\n\nBack to master:\n\n```console\ngit checkout master\n```\n\n## html\n\nIf you need to update the html content in the `web_template` directory, you can build the html pages by:\n\n0. Install Node.js\n1. Run `npm install` in the project directory\n2. `npm run build`\n\n## traduction\n\nIf you need to update any game text translations, they\'re kept in the [`game_translations/`](/game_translations) directory. In order for your changes to show up in the game, you\'ll need to run [`./update_translation`](/update_translation), which will update `sk8border.p8` automatically, using a compiled string format to save tokens. This requires [Lua](https://www.lua.org/start.html) to be installed on your system (this version doesn\'t ship with PICO-8 so you would need to install it separately).\n\nIf you want to add a new language, just copy an existing translation file and modify it. When you\'re done, update the boilerplate in [`./update_translation`](/update_translation) to make sure it gets included in the updated p8 file.\n'
