b'# Notes\n\n[![Join the chat at https://gitter.im/nuttyartist/notes](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/nuttyartist/notes?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)\n[![Build Status](https://travis-ci.org/nuttyartist/notes.svg?branch=dev)](https://travis-ci.org/nuttyartist/notes)\n[![Build status](https://ci.appveyor.com/api/projects/status/rgque4o6x2y0i92i?svg=true)](https://ci.appveyor.com/project/nuttyartist/notes)\n\nNotes is an open source, cross-platform note taking app which has the potential to be something much bigger, and much better at empowering the people using it.\nThis is where you come into play. Be part of making Notes better.\n\n## [Team Board](http://www.get-notes.com/team-board)\n\n## Screenshot\n![notes_screenshot](https://user-images.githubusercontent.com/16375940/29837038-bc4e58a4-8cff-11e7-9fb1-692e9948c33c.png)\n\n\n## How to clone\nUse this command to clone the repository:\n\n```shell\n$> git clone --recursive  https://github.com/nuttyartist/notes.git\n```\n\n## Dependencies\nMake sure the Qt (>= 5.2.1) development libraries are installed:\n\n- Debian/Ubuntu : qt5-default build-essential qtbase5-private-dev sqlite3\n\n## Compiling\n\n```shell\n$> mkdir build\n$> cd build\n$> qmake ../src\n$> make -j4\n```\n\n## Notes\n\nThe notes database and settings file are stored:\n\n**Windows** : ```C:\\Users\\user\\AppData\\Roaming\\Awesomeness```  \n**Linux** &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;: ```/home/user/.config/Awesomeness/``` **or** ```/home/snap/notes/x1/.config/Awesomeness``` **(using snap)**  \n**Mac** &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;: ```/home/.config/Awesomeness/```  \n\n## Information for Contributors\n\nCheck out our [wiki](https://github.com/nuttyartist/notes/wiki), we discuss there our philosophy, our current state and the future version of Notes.\n\n## Information for End Users\n\nYou can find more info about Notes and how to download it here: <a href="http://get-notes.com" target="_blank">get-notes.com</a>\n'
