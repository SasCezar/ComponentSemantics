b'# Sci-Hub EVA <img src="images/SciHubEVA-icon.png" align="right" alt="logo" width="100" height = "100" style = "border: none; float: right;">\n![Release](https://img.shields.io/github/release/leovan/SciHubEVA.svg)\n![License](https://img.shields.io/github/license/leovan/SciHubEVA.svg)\n![Issues](https://img.shields.io/github/issues/leovan/SciHubEVA.svg)\n![Downloads](https://img.shields.io/github/downloads/leovan/SciHubEVA/total.svg)\n\n---\n\n## Introduction\n\n**Sci-Hub EVA** is a cross-platform [Sci-Hub](https://en.wikipedia.org/wiki/Sci-Hub) GUI application written in Python and Qt (PySide2).\n\n## Usage\n\n![Application_MACOS_EN](docs/scihub-eva-application-macos-en.png)\n\nClick `OPEN` button to choose where to save the downloaded files. Click `SHOW` button will open the directory where you set.\n\nFill the query and click `RAMPAGE` button, then it will search the query and download the file. Currently, you can fill the query with `URL`, `DOI`, `PMID` or search string.\n\nRange pattern in query is supported, e.g. `00.000/{1-99}` will download `00.000/1`, `00.000/2`, ... `00.000/99`. Zero padded format range pattern is allowed, e.g. `00.000/{01-99}` will download `00.000/01`, `00.000/02`, ... `00.000/99`.\n\nAlso you can download with a query list file, in which each line represents a query. Click `LOAD` button to load the query list file.\n\nRight clicking the log area will popup menu, you can open the log file or log directory.\n\nClick `GEAR` icon button, it will open `Preferences` dialog, and you can change options in it.\n\n![Preferences_MACOS_EN](docs/scihub-eva-preferences-macos-en.png)\n\nYou can change filename prefix format with supported keywords. Light and dark theme are supported, also you can choose `System` on macOS to fit system theme automatically. Setting overwrite existing file to `No` will add a timestamp suffix in filename to avoid overwriting previous downloaded files.\n\nDue to the unstable Sci-Hub host accessibility, it may fail to download PDFs sometimes, you can change and add other Sci-Hub URLs, or set a proxy server.\n\nSometimes, you need enter the captcha to continue the download.\n\n![Captcha_MACOS_EN](docs/scihub-eva-captcha-macos-en.png)\n\n## Internationalization Support\n\n- English\n- Simplified Chinese (\xe7\xae\x80\xe4\xbd\x93\xe4\xb8\xad\xe6\x96\x87)\n- Traditional Chinese - Hongkong (\xe7\xb9\x81\xe9\xab\x94\xe4\xb8\xad\xe6\x96\x87 - \xe9\xa6\x99\xe6\xb8\xaf)\n- Traditional Chinese - Taiwan (\xe6\xad\xa3\xe9\xab\x94\xe4\xb8\xad\xe6\x96\x87 - \xe8\x87\xba\xe7\x81\xa3)\n\n## Platform Support\n\n### macOS\n\n<table border="0">\n  <tr align="center">\n    <td><img src="docs/scihub-eva-application-macos-en-light-theme.png" /></td>\n    <td><img src="docs/scihub-eva-application-macos-en-dark-theme.png" /></td>\n  </tr>\n  <tr align="center">\n    <td>Light Theme</td>\n    <td>Dark Theme</td>\n  </tr>\n  </tr>\n</table>\n\n### Windows\n\n<table border="0">\n  <tr align="center">\n    <td><img src="docs/scihub-eva-application-windows-en-light-theme.png" /></td>\n    <td><img src="docs/scihub-eva-application-windows-en-dark-theme.png" /></td>\n  </tr>\n  <tr align="center">\n    <td>Light Theme</td>\n    <td>Dark Theme</td>\n  </tr>\n</table>\n\n## Installing\n\n### macOS\n\n- Install with [DMG file](https://github.com/leovan/SciHubEVA/releases).\n- Install from brew cask.  \n  `brew cask install scihubeva`\n\n### Windows\n\n- Install with [EXE setup file](https://github.com/leovan/SciHubEVA/releases).\n\n## Building\n\nSee [`building/README.md`](building/README.md)\n\n## Licenses\n\n[SciHubEVA](https://github.com/leovan/SciHubEVA): The MIT License (MIT)\n\n[PySide2](https://doc.qt.io/qtforpython): GNU Lesser General Public License (LGPL)\n\n[Material design icons](https://github.com/templarian/MaterialDesign/): [SIL Open Font License 1.1](http://scripts.sil.org/cms/scripts/page.php?item_id=OFL_web)\n\nIcons: GNU General Public License 3.0 (GPL-3.0), borrowed from [Numix Circle](https://github.com/numixproject/numix-icon-theme-circle).\n'
