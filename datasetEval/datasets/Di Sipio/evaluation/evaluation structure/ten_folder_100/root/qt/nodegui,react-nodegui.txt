b'# React NodeGUI\n\n[![Join the NodeGUI community on Spectrum](https://withspectrum.github.io/badge/badge.svg)](https://spectrum.chat/nodegui)\n[![All Contributors](https://img.shields.io/badge/all_contributors-7-orange.svg?style=flat-square)](#contributors)\n\nBuild **performant**, **native** and **cross-platform** desktop applications with React.\xf0\x9f\x9a\x80\n\nReact NodeGUI is powered by **React** \xe2\x9a\x9b\xef\xb8\x8f and **Qt5** \xf0\x9f\x92\x9a which makes it CPU and memory efficient as compared to other chromium based solutions like electron. React NodeGUI is essentially a React renderer for [NodeGUI](https://github.com/nodegui/nodegui).\n\nVisit: https://react.nodegui.org for docs.\n\n<img alt="logo" src="https://github.com/nodegui/nodegui/raw/master/extras/logo/nodegui.png" height="200" />\n\n> This project is in active development. It should be okay for smaller projects but anything complex - electron is the way to go for now. \xf0\x9f\x9a\xa7\n\n## How does it look?\n\n<div style="display:inline; margin: 0 auto;">\n<img alt="demo_linux" src="https://github.com/nodegui/examples/raw/master/react-nodegui/calculator/calculator_linux.png" height="280" />\n<img alt="demo_win" src="https://github.com/nodegui/examples/raw/master/react-nodegui/calculator/calculator_win.jpg" height="280" />\n<img alt="demo_mac" src="https://github.com/nodegui/examples/raw/master/react-nodegui/calculator/calculator_mac.png" height="280" />\n</div>\n\n<div style="display:inline; margin: 0 auto;"><img alt="kitchen" src="https://github.com/nodegui/nodegui/raw/master/extras/assets/kitchen.png" height="280" /><img alt="demo_mac" src="https://github.com/nodegui/examples/raw/master/react-nodegui/weather-app-widget/weather_widget_mac.png" height="280" /><img alt="demo_win" src="https://github.com/nodegui/examples/raw/master/react-nodegui/image-view/image_view_win.jpg" height="280" />\n</div>\n\n**More screenshots?**\n\n[See examples](https://github.com/nodegui/react-nodegui/tree/master/examples/)\n\n## Features\n\n- \xf0\x9f\xa7\xac Cross platform. Should work on major Linux flavours, Windows and MacOS\n- \xf0\x9f\x93\x89 Low CPU and memory footprint. Current CPU stays at 0% on idle and memory usage is under 20mb for a hello world program.\n- \xf0\x9f\x92\x85 Styling with CSS (includes actual cascading). Also has full support for Flexbox layout (thanks to Yoga).\n- \xe2\x9c\x85 Complete Nodejs api support (Currently runs on Node v12.x - and is easily upgradable). Hence has access to all nodejs compatible npm modules.\n- \xf0\x9f\x8e\xaa Native widget event listener support. supports all event available from Qt / NodeJs.\n- \xf0\x9f\x92\xb8 Can be used for Commercial applications.\n- \xf0\x9f\x95\xb5\xef\xb8\x8f\xe2\x80\x8d\xe2\x99\x82\xef\xb8\x8f Good Devtools support (supports react-devtools, node debugger).\n- \xf0\x9f\x93\x9a Good documentation and website.\n- \xf0\x9f\xa7\x99\xe2\x80\x8d\xe2\x99\x82\xef\xb8\x8f Good documentation for contributors.\n- \xf0\x9f\xa6\xb9\xf0\x9f\x8f\xbb\xe2\x80\x8d\xe2\x99\x80\xef\xb8\x8f Good support for dark mode (Thanks to QT).\n- \xf0\x9f\x8f\x85First class Typescript support. (Works on regular JS projects too \xf0\x9f\x98\x89).\n\n## Getting Started\n\n- Check out [react-nodegui-starter](https://github.com/nodegui/react-nodegui-starter) to get up and running with your own React NodeGUI app!\n- Read through the [docs](https://react.nodegui.org)\n\n**Community Guides**\n\n- https://blog.logrocket.com/electron-alternatives-exploring-nodegui-and-react-nodegui/ - Electron alternatives: Exploring NodeGUI and React NodeGUI by [Siegfried Grimbeek](https://blog.logrocket.com/author/siegfriedgrimbeek/).\n\n**Talks/Podcasts**\n\n- [NodeGui and React NodeGui at KarmaJS Nov 2019 meetup: https://www.youtube.com/watch?v=8jH5gaEEDv4](https://www.youtube.com/watch?v=8jH5gaEEDv4)\n\n- <audio data-theme="night" data-src="https://changelog.com/jsparty/96/embed" src="https://cdn.changelog.com/uploads/jsparty/96/js-party-96.mp3" preload="none" class="changelog-episode" controls></audio><p><a href="https://changelog.com/jsparty/96">JS Party 96: Performant Node desktop apps with NodeGUI</a> \xe2\x80\x93 Listen on <a href="https://changelog.com/">Changelog.com</a></p>\n\n\n## Docs for contributing\n\nLooking to contribute?
