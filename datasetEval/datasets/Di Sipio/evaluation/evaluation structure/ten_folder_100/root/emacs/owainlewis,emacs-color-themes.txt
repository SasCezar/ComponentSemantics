b'# Emacs themes\n\nA collection of color themes for Emacs24 +\n\nThe themes are named after important/influential programmers.\n\nFixes, pull requests and feedback very welcome : )\n\n## Contributors\n\nhttps://github.com/owainlewis/emacs-color-themes/graphs/contributors\n\n## Use\n\nThe easiest way to install is to use MELPA\n\n```\nM-x package-install <RET> sublime-themes\n```\n\n## Manual install\n\nRunning install.sh will put the themes in  ~/.emacs.d/themes\n\nTo load a theme add the following to your init.el\n\n```elisp\n(add-to-list \'custom-theme-load-path "~/.emacs.d/themes")\n(load-theme \'hickey t)\n```\n\nYou can browse the themes with\n\n```\nM-x load-theme <RET> name-of-theme\n```\n\nOne line install script\n\n```\ngit clone https://github.com/owainlewis/emacs-color-themes.git && cd emacs-color-themes && ./install.sh\n```\n\n## As an Elisp Package\n\nIf the package directory is in the load path, one can simply\n\n```\n(require \'emacs-color-themes)\n```\n\nand use `load-theme` function to load any theme that is provided.\nThis way, one can add the package to their Emacs configuration as is,\nwithout needing to add any directories, symlinks or copies.\n\n## Brin\n\nA color theme based on the Space Grey ST2 theme named after Sergey Brin.\n\n> We believed we could build a better search.
