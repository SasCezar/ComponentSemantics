I\'m using use-package and el-get and evil\n\n(el-get-bundle slack)\n(use-package slack\n  :commands (slack-start)\n  :init\n  (setq slack-buffer-emojify t) ;; if you want to enable emoji, default nil\n  (setq slack-prefer-current-team t)\n  :config\n  (slack-register-team\n   :name "emacs-slack"\n   :default t\n   :client-id "aaaaaaaaaaa.00000000000"\n   :client-secret "bbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbb"\n   :token "xoxs-sssssssssss-88888888888-hhhhhhhhhhh-jjjjjjjjjj"\n   :subscribed-channels \'(test-rename rrrrr)\n   :full-and-display-names t)\n\n  (slack-register-team\n   :name "test"\n   :client-id "3333333333.77777777777"\n   :client-secret "cccccccccccccccccccccccccccccccc"\n   :token "xoxs-yyyyyyyyyy-zzzzzzzzzzz-hhhhhhhhhhh-llllllllll"\n   :subscribed-channels \'(hoge fuga))\n\n  (evil-define-key \'normal slack-info-mode-map\n    ",u" \'slack-room-update-messages)\n  (evil-define-key \'normal slack-mode-map\n    ",c" \'slack-buffer-kill\n    ",ra" \'slack-message-add-reaction\n    ",rr" \'slack-message-remove-reaction\n    ",rs" \'slack-message-show-reaction-users\n    ",pl" \'slack-room-pins-list\n    ",pa" \'slack-message-pins-add\n    ",pr" \'slack-message-pins-remove\n    ",mm" \'slack-message-write-another-buffer\n    ",me" \'slack-message-edit\n    ",md" \'slack-message-delete\n    ",u" \'slack-room-update-messages\n    ",2" \'slack-message-embed-mention\n    ",3" \'slack-message-embed-channel\n    "\\C-n" \'slack-buffer-goto-next-message\n    "\\C-p" \'slack-buffer-goto-prev-message)\n   (evil-define-key \'normal slack-edit-message-mode-map\n    ",k" \'slack-message-cancel-edit\n    ",s" \'slack-message-send-from-buffer\n    ",2" \'slack-message-embed-mention\n    ",3" \'slack-message-embed-channel))\n\n(use-package alert\n  :commands (alert)\n  :init\n  (setq alert-default-style \'notifier))\n\n```\n## How to get client ID\n\n 1. Log into the Slack team you\'re interested in with a web browser\n 2. Open DevTools\n 3. Open Network tab\n 4. Take the _x_id from Query String Parameters\n\n## How to get token (the easy way)\n\n### 1. From Request Payload\n\n 1. Log into the Slack team you\'re interested in with a web browser\n 2. Open DevTools\n 3. Open Network tab\n 4. Search for (Ctrl-F) "xoxs-" and copy token from Request Payload\n\n### 2. With a legacy token\n\n1. Go to "https://api.slack.com/custom-integrations/legacy-tokens".\n2. Create token for the teams you want to have integrate with.\n\n## How to get token (the harder yet officially sanctioned way)\n\n### 1. Get client-id and client-secret\n\n1. Go to "https://api.slack.com/apps".\n2. Click "Create New App".\n3. Fill "App Name" and "Development Slack Team" and click "Create App".\n4. "Client ID" and "Client Secret" are listed in the "App Credentials" section.\n\n### 2. Configure Redirect URL\n\n1. Click "Permissions" in the "Add features and functionality" of "Building Apps for Slack" section.\n2. Fill in the "Redirect URLs" section (slack-emacs defaults to: "http://localhost:8080").\n3. click "Save URLs".\n\n### 3. Get token\n\n1. Call `slack-register-team` with above "Client ID" and "Client Secret" (ignore the "Token" prompt [i.e., just hit enter.]).\n2. Emacs\' prompt will display: "Enter the code your browser displayed: ".\n3. Code appears in the browser\'s address bar like "http://localhost:8080?code=181818181818.1819919191&state=", enter this code in the previous Emacs\' prompt.\n4. Once you activate `emacs-slack`, token appears in the URL like: "https://api.slack.com/apps/{APPID}/oauth".\n5. Save your token and pass it to the `slack-register-team` function along with the "Client ID" and "Client Secret".\n\n#### Note\n\nemacs-slack uses websocket to communicate with Slack, and need request to [rtm.start method](https://api.slack.com/methods/rtm.start) (you can also test your settings in that page\'s "Tester" tab to make sure things have been configured correctly).\nthis request needs "client" scope when authorize and Slack does not yet have "client" scope in new OAuth scope.\nmake sure `slack-oauth2-auth` requesting with "client" scope and prevent token migration.\n![token_migration](https://github.com/yuya373/emacs-slack/wiki/images/token_migration.png)\n\n\n## How to use\n\nI recommend to chat with slackbot for tutorial using `slack-im-select`.\n\nSome terminology in the `slack-` functions:\n- `im`: An IM (instant message) is a direct message between you and exactly one other Slack user.\n- `channel`: A channel is a Slack channel which you are a member of\n- `group`. Any chat (direct message or channel) which isn\'t an IM is a group.\n\n- `slack-register-team`\n  - set team configuration and create team.\n  - :name, :client-id, and :client-secret are needed for arguments\n- `slack-change-current-team`\n  - change `slack-current-team` var\n- `slack-start`\n  - do authorize and initialize\n- `slack-ws-close`\n  - turn off websocket connection\n- `slack-group-select`\n  - select group from list\n- `slack-im-select`\n  - select direct message from list\n- `slack-channel-select`\n  - select channel from list\n- `slack-group-list-update`\n  - update group list\n- `slack-im-list-update`\n  - update direct message list\n- `slack-channel-list-update`\n  - update channel list\n- `slack-message-embed-mention`\n  - use to mention to user\n- `slack-message-embed-channel`\n  - use to mention to channel\n- `slack-file-upload`\n  - uploads a file\n  - the command allows to choose many channels via select loop. In order to finish the loop input an empty string. For helm that\'s <kbd>C+RET</kbd> or <kbd>M+TET</kbd>. In case of Ivy it\'s <kbd>C+M+j</kbd>.\n\n## Notification\n\nSee [alert](https://github.com/jwiegley/alert).\n'
