b'<h1>Soft Proposal Networks<br>for Weakly Supervised Object Localization</h1>\n\n[![](https://img.shields.io/badge/torch-V1.0-green.svg)](https://github.com/ZhouYanzhao/SPN/tree/torch) [![](https://img.shields.io/badge/pytorch-V1.0-blue.svg)](https://github.com/yeezhu/SPN.pytorch) [![](https://img.shields.io/badge/caffe-coming-red.svg)](#)\n\n[[Project]](http://yzhou.work/SPN) [[Paper]](https://arxiv.org/pdf/1709.01829) [[Supp]](http://yzhu.work/pdffiles/SPN_Supp.pdf) [[More Resources]](http://yzhu.work/spn)\n\n![Illustration](illustration.png)\n\n## Torch Implementation\nThe [torch branch](https://github.com/ZhouYanzhao/SPN/tree/torch) contains:\n\n* the official **torch** implementation of SPN.\n* the PASCAL-VOC point-based localization demo.\n\nPlease follow the instruction below to install it and run the experiment demo.\n\n### Prerequisites\n* Linux (tested on ubuntu 14.04LTS)\n* NVIDIA GPU + CUDA CuDNN\n* [Torch7](http://torch.ch/docs/getting-started.html)\n* [Jupyter Notebook](http://jupyter.org/install.html)\n\n### Getting started\nYou can setup everything via a single command `wget -O - https://git.io/v5wTS | bash` **or** do it manually in case something goes wrong:\n\n1. install the dependencies (required by the demo code):\n    * [loadcaffe](https://github.com/szagoruyko/loadcaffe)    \n    * [torchnet](https://github.com/torchnet/torchnet): `luarocks install torchnet`\n    * [optnet](https://github.com/fmassa/optimize-net): `luarocks install optnet`\n\n2. clone the torch branch: \n\n    ```bash\n    # git version must be greater than 1.9.10\n    git clone https://github.com/ZhouYanzhao/SPN.git -b torch --single-branch SPN.torch\n    cd SPN.torch\n    export DIR=$(pwd)\n    ```\n\n3. install SPN: \n\n    ```bash\n    cd $DIR/install\n    # install the GPU implementation of SPN.\n    bash install.sh\n    ```\n\n4. download the PASCAL-VOC2007 dataset:\n\n    ```bash\n    cd $DIR/demo/datasets\n    # trainval\n    wget http://host.robots.ox.ac.uk/pascal/VOC/voc2007/VOCtrainval_06-Nov-2007.tar\n    tar xvf  VOCtrainval_06-Nov-2007.tar\n    # test\n    wget http://host.robots.ox.ac.uk/pascal/VOC/voc2007/VOCtest_06-Nov-2007.tar\n    tar xvf VOCtest_06-Nov-2007.tar\n    ```\n\n5. download pre-trained VGGNet model for fine-tuning:\n\n    ```bash\n    cd $DIR/demo/models/convert\n    wget http://www.robots.ox.ac.uk/~vgg/software/very_deep/caffe/VGG_ILSVRC_16_layers.caffemodel\n    # convert caffemodel to t7\n    th convertVGG.lua\n    ```\n\n6. run the demo experiment:\n\n    ```bash\n    cd $DIR/demo\n    bash ./scripts/Train_PASCAL.sh\n    ```\n\n7. visualize locating samples via [demo/notebooks/vis.ipynb](https://github.com/ZhouYanzhao/SPN/blob/torch/demo/notebooks/vis.ipynb)\n![Pointing localization](pointing.png)\n\n### Troubleshooting\nIf you run into `error: identifier "THCudaBlas_Sgemv" is undefined` during installation, update Torch7 to the latest version via `cd <TORCH_DIR> && bash ./update.sh`\n\n## PyTorch Implementation\nCheck [here](https://github.com/yeezhu/SPN.pytorch).\n\n## Caffe Implementation\ncoming\n\n## Citation \nIf you use the code in your research, please cite:\n```bibtex\n@INPROCEEDINGS{Zhu2017SPN,\n    author = {Zhu, Yi and Zhou, Yanzhao and Ye, Qixiang and Qiu, Qiang and Jiao, Jianbin},\n    title = {Soft Proposal Networks for Weakly Supervised Object Localization},\n    booktitle = {ICCV},\n    year = {2017}\n}\n```\n'
