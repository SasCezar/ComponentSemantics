b"jQuery.IME\n==========\n\njQuery.IME is a jQuery based input method editor library supporting more than\n135 input methods across more than 62 languages.\n\nThese input methods are well tested. Initially the input methods were\ncontributed by the Wikimedia community. By now many input methods have also\nbeen contributed by Red Hat.\n\nThis project is a Wikimedia foundation initiative to provide language technology\ntools to a wider audience outside the Wikimedia universe.\n\nThis input tool is widely deployed in all Wikimedia projects (eg: wikipedia).\n\n\nQuick start\n-----------\n\n```bash\ngit clone https://github.com/wikimedia/jquery.ime.git\n```\n\nTo add input method support to all editable elements:\n\n```javascript\n$( 'textarea, [contenteditable], input[type=text], input[type=search], input:not([type])' ).ime();\n```\n\njquery.ime provides a jquery plugin function `$.fn.ime()` to add input method\nsupport for any editable elements in a page.\n\nLanguage selection API\n----------------------\n\nInstead of using the default language selector, you can roll your own:\n\n```javascript\n$myDiv= $( '.foo' );\n$myDiv.ime( { showSelector: false } );\nime = $myDiv.data( 'ime' );\n$elements.on( 'imeLanguageChange', function () { ... } );\n$elements.on( 'imeMethodChange', function () { ... } );\n\ncurrentLanguageCode = ime.getLanguage();\ncurrentLanguageName = ime.getAutonym( currentLanguageCode );\nallLanguageCodes = ime.getLanguageCodes();\ninputMethods = ime.getInputMethods( currentLanguageCode );\nime.setIM( inputMethods[ 0 ].id );\n```\n\nSee examples/ced/ced.html for a more complete example.\n\nRunning\n-------\n\n-If you have python installed, run\n-```bash\n python -m SimpleHTTPServer\n-```\n\nAlternatively, many programming languages provide one liner commands to start a simple http static servers. You can use [any one of them](https://gist.github.com/willurd/5720255).\n\n\nand, you can access examples from\n```bash\nhttp://localhost:8000/examples/index.html\n```\nAlternatively you can use tools like webfsd. If you know how to use advanced\nweb servers like Apache or Nginx, you already know how to put the code in a\nwebserver.\n\nPlugin Options\n--------------\n\n| Option  | Description |\n|---------|---------------------|\n| imePath | Relative/Absolute path for the rules folder of jquery.ime. Default value: '../' |\n| languages| Languages to be used- by default all languages. Default value:  []. Eg: ['hi', 'ml']. It can also be a function returning an array.|\n| helpHandler| Called for each ime option in the menu, default value: null|\n| showSelector | Whether the input method selector to be used or not. Default value is true |\n\nBrowser extensions\n------------------\nThis input tool is also available as Firefox and Chrome extensions.\n* [Firefox extension](https://addons.mozilla.org/en-US/firefox/addon/wikimedia-input-tools/)\n* [Chrome extension](https://chrome.google.com/webstore/detail/wikimedia-input-tools/fjnfifedbeeeibikgpggddmfbaeccaoh)\n\nFor Developers\n--------------\n\nRead [rules/README.md](https://github.com/wikimedia/jquery.ime/tree/master/rules)\nto learn how to write an input method for a language.\n\nRead [test/README.md](https://github.com/wikimedia/jquery.ime/tree/master/test)\nto learn how to write and run tests for an input method.\n\nRead the technical specification of the project from\n[wiki](https://github.com/wikimedia/jquery.ime/wiki/Technical-Specification).\n\nDo watch the code walkthrough by Chris Forno: http://www.youtube.com/watch?v=LPec-KIBVeI\n\nLicense\n-------\nThis project is dual licensed with GPLv2+ and MIT license.
