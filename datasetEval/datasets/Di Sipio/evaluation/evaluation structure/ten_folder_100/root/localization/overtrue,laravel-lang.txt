"overtrue/laravel-lang:~3.0"\n```\n\n#### Laravel 5.*\n\nAfter completion of the above, Replace the `config/app.php` content\n\n```php\nIlluminate\\Translation\\TranslationServiceProvider::class,\n```\nwith:\n\n```php\nOvertrue\\LaravelLang\\TranslationServiceProvider::class,\n```\n\n#### Lumen\n\nAdd the following line to `bootstrap/app.php`:\n\n```php\n$app->register(Overtrue\\LaravelLang\\TranslationServiceProvider::class);\n```\n\n# Configuration\n\n### Laravel\nyou can change the locale at `config/app.php`:\n\n```php\n\'locale\' => \'zh-CN\',\n```\n\n### Lumen\n\nset locale in `.env` file:\n\n```\nAPP_LOCALE=zh-CN\n```\n\n# Usage\n\nThere is no difference with the usual usage.\n\nIf you need to add additional language content, Please create a file in the `resources/lang/{LANGUAGE}`  directory.\n\n### Add custom language items\n\nHere, for example in Chinese:\n\n`resources/lang/zh-CN/demo.php`:\n\n```php\n<?php\n\nreturn [\n    \'user_not_exists\'    => \'\xe7\x94\xa8\xe6\x88\xb7\xe4\xb8\x8d\xe5\xad\x98\xe5\x9c\xa8\',\n    \'email_has_registed\' => \'\xe9\x82\xae\xe7\xae\xb1 :email \xe5\xb7\xb2\xe7\xbb\x8f\xe6\xb3\xa8\xe5\x86\x8c\xe8\xbf\x87\xef\xbc\x81\',\n];\n```\nUsed in the template:\n\n```php\necho trans(\'demo.user_not_exists\'); // \xe7\x94\xa8\xe6\x88\xb7\xe4\xb8\x8d\xe5\xad\x98\xe5\x9c\xa8\necho trans(\'demo.email_has_registed\', [\'email\' => \'anzhengchao@gmail.com\']);\n// \xe9\x82\xae\xe7\xae\xb1 anzhengchao@gmail.com \xe5\xb7\xb2\xe7\xbb\x8f\xe6\xb3\xa8\xe5\x86\x8c\xe8\xbf\x87\xef\xbc\x81\n```\n\n### Replace the default language items partially\n\nWe assume that want to replace the `password.reset` message:\n\n`resources/lang/zh-CN/passwords.php`:\n\n```php\n<?php\n\nreturn [\n    \'reset\' => \'\xe6\x82\xa8\xe7\x9a\x84\xe5\xaf\x86\xe7\xa0\x81\xe5\xb7\xb2\xe7\xbb\x8f\xe9\x87\x8d\xe7\xbd\xae\xe6\x88\x90\xe5\x8a\x9f\xe4\xba\x86\xef\xbc\x8c\xe4\xbd\xa0\xe5\x8f\xaf\xe4\xbb\xa5\xe4\xbd\xbf\xe7\x94\xa8\xe6\x96\xb0\xe7\x9a\x84\xe5\xaf\x86\xe7\xa0\x81\xe7\x99\xbb\xe5\xbd\x95\xe4\xba\x86!\',\n];\n```\n\nYou need only add the partials item what you want.\n\n### publish the language files to your project `resources/lang/` directory:\n\n```shell\n$ php artisan lang:publish [LOCALES]
