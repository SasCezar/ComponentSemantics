Translate is a fully featured localization / internationalization (i18n) library for Flutter.\n\nIt lets you define translations for your content in different languages and switch between them easily.\n\n## Example\n<img src="https://raw.githubusercontent.com/bratan/flutter_translate/master/resources/gifs/flutter_translate_screen.gif" width="300"/>\n\n## Features\n\n* Very easy to use\n* ```Pluralization``` support\n* ```Static keys``` support with [flutter_translate_gen](https://github.com/bratan/flutter_translate/wiki/3.-Generating-statically-typed-localization-keys)\n* Supports both ``languageCode (en)`` and ``languageCode_countryCode (en_US)`` locale formats \n* Automatically ```save & restore``` the selected locale [with a simple implementation](https://github.com/bratan/flutter_translate/wiki/2.-Automatically-saving-&-restoring-the-selected-locale)\n* Full support for ```right-to-left``` locales\n* ``Fallback`` locale support in case the system locale is unsupported\n* Supports both ``inline or nested`` JSON\n\n## Documentation\n\nComplete documentation is available [on the wiki page](https://github.com/bratan/flutter_translate/wiki).\n\nTo get started please see [Installation, Configuration &
