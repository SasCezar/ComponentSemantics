b'# Laravel Multilingual Routes\n\n[![Latest Version on Packagist](https://img.shields.io/packagist/v/chinleung/laravel-multilingual-routes.svg?style=flat-square)](https://packagist.org/packages/chinleung/laravel-multilingual-routes)\n[![Build Status](https://img.shields.io/travis/chinleung/laravel-multilingual-routes/master.svg?style=flat-square)](https://travis-ci.org/chinleung/laravel-multilingual-routes)\n[![Quality Score](https://img.shields.io/scrutinizer/g/chinleung/laravel-multilingual-routes.svg?style=flat-square)](https://scrutinizer-ci.com/g/chinleung/laravel-multilingual-routes)\n[![Total Downloads](https://img.shields.io/packagist/dt/chinleung/laravel-multilingual-routes.svg?style=flat-square)](https://packagist.org/packages/chinleung/laravel-multilingual-routes)\n\nA package to register multilingual routes for your application.\n\n## Installation\n\nYou can install the package via composer:\n\n```bash\ncomposer require chinleung/laravel-multilingual-routes\n```\n\nTo detect and change the locale of the application based on the request automatically, you can add the middleware to your `app/Http/Kernel`:\n\n``` php\nprotected $middlewareGroups = [\n    \'web\' => [\n        \\ChinLeung\\LaravelMultilingualRoutes\\DetectRequestLocale::class,\n        // ...\n    ]\n];\n```\n\n## Configuration\n\nBy default, the application locales are going to be `en` and the `app.fallback_locale` is not prefixed. If you want to prefix the fallback locale, please run the following command to publish the configuration file:\n\n``` bash\nphp artisan vendor:publish --provider="ChinLeung\\LaravelMultilingualRoutes\\LaravelMultilingualRoutesServiceProvider" --tag="config"\n```\n\nIf your application supports different locales, you can either set a `app.locales` configuration or follow the configuration instructions from [chinleung/laravel-locales](https://github.com/chinleung/laravel-locales#configuration).\n\n## Example\n\nInstead of doing this:\n\n``` php\nRoute::get(\'/\', \'ShowHomeController\')->name(\'en.home\');\nRoute::get(\'/fr\', \'ShowHomeController\')->name(\'fr.home\');\n```\n\nYou can accomplish the same result with:\n\n``` php\nRoute::multilingual(\'/\', \'ShowHomeController\')->name(\'home\');\n```\n\n## Usage\n\n### Quick Usage\n\nOnce you have configured the locales, you can start adding routes like the following example in your `routes/web.php`:\n\n``` php\nRoute::multilingual(\'test\', \'TestController\');\n```\n\nThis will generate the following:\n\n| Method   | URI     | Name    | Action                              |\n|----------|---------|---------|-------------------------------------|\n| GET\\|HEAD | test | en.test | App\\Http\\Controllers\\TestController |\n| GET\\|HEAD | fr/teste   | fr.test | App\\Http\\Controllers\\TestController |\n\nNote the `URI` column is generated from a translation file located at `resources/lang/{locale}/routes.php` which contains the key of the route like the following:\n\n``` php\n<?php\n\n// resources/lang/fr/routes.php\n\nreturn [\n  \'test\' => \'teste\'\n];\n```\n\nTo retrieve a route, you can use the `localized_route(string $name, array $parameters, string $locale = null, bool $absolute = true)` instead of the `route` helper:\n\n``` php\nlocalized_route(\'test\'); // Returns the url of the current application locale\nlocalized_route(\'test\', [], \'fr\'); // Returns https://app.test/fr/teste\nlocalized_route(\'test\', [], \'en\'); // Returns https://app.test/test\n```\n\n### Renaming the routes\n\n``` php\nRoute::multilingual(\'test\', \'TestController\')->name(\'foo\');\n```\n\n| Method   | URI     | Name   | Action                              |\n|----------|---------|--------|-------------------------------------|\n| GET\\|HEAD | test | en.foo | App\\Http\\Controllers\\TestController |\n| GET\\|HEAD | fr/teste   | fr.foo | App\\Http\\Controllers\\TestController |\n\n### Renaming a route based on the locale\n\n``` php\nRoute::multilingual(\'test\', \'TestController\')->names([\n  \'en\' => \'foo\',\n  \'fr\' => \'bar\',\n]);\n```\n\n| Method   | URI     | Name   | Action                              |\n|----------|---------|--------|-------------------------------------|\n| GET\\|HEAD | test | en.foo | App\\Http\\Controllers\\TestController |\n| GET\\|HEAD | fr/teste   | fr.bar | App\\Http\\Controllers\\TestController |\n\n### Skipping a locale\n\n``` php\nRoute::multilingual(\'test\', \'TestController\')->except([\'fr\']);\n```\n\n| Method   | URI     | Name    | Action                              |\n|----------|---------|---------|-------------------------------------|\n| GET\\|HEAD | test    | en.test | App\\Http\\Controllers\\TestController |\n\n### Restricting to a list of locales\n\n``` php\nRoute::multilingual(\'test\', \'TestController\')->only([\'fr\']);\n```\n\n\n| Method   | URI     | Name    | Action                              |\n|----------|---------|---------|-------------------------------------|\n| GET\\|HEAD | fr/teste | fr.test | App\\Http\\Controllers\\TestController |\n\n### Changing the method of the request\n\n``` php\nRoute::multilingual(\'test\', \'TestController\')->method(\'post\');\n```\n\n| Method | URI     | Name    | Action                              |\n|--------|---------|---------|-------------------------------------|\n| POST   | test | en.test | App\\Http\\Controllers\\TestController |\n| POST   | fr/teste   | fr.test | App\\Http\\Controllers\\TestController |\n\n### Registering a view route\n\n\n``` php\n// Loads test.blade.php\nRoute::multilingual(\'test\');\n```\n\n| Method   | URI     | Name    | Action                              |\n|----------|---------|---------|-------------------------------------|\n| GET\\|HEAD | test | en.test | Illuminate\\Routing\\ViewController |\n| GET\\|HEAD | fr/teste   | fr.test | Illuminate\\Routing\\ViewController |\n\n\n### Registering a view route with a different key for the route and view\n\n``` php\n// Loads welcome.blade.php instead of test.blade.php\nRoute::multilingual(\'test\')->view(\'welcome\');\n```\n\n| Method   | URI     | Name    | Action Illuminate\\Routing\\ViewController |\n\n### Testing\n\n``` bash\ncomposer test\n```\n\n### Changelog\n\nPlease see [CHANGELOG](CHANGELOG.md) for more information what has changed recently.\n\n##
