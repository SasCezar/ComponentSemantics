b'# Fluent [![Build Status](https://travis-ci.org/projectfluent/fluent-rs.svg?branch=master)](https://travis-ci.org/projectfluent/fluent-rs) [![Coverage Status](https://coveralls.io/repos/github/projectfluent/fluent-rs/badge.svg?branch=master)](https://coveralls.io/github/projectfluent/fluent-rs?branch=master)\n\n`fluent-rs` is a collection of Rust crates implementing [Project Fluent](https://projectfluent.org).\n\nThe crates perform the following functions:\n\n## fluent [![crates.io](http://meritbadge.herokuapp.com/fluent)](https://crates.io/crates/fluent)\n\nUmbrella crate combining crates that are ready to be used in production.\n\n## fluent-syntax [![crates.io](http://meritbadge.herokuapp.com/fluent_syntax)](https://crates.io/crates/fluent_syntax)\n\nLow level Fluent Syntax AST and parser API.\n\n## fluent-bundle [![crates.io](http://meritbadge.herokuapp.com/fluent_bundle)](https://crates.io/crates/fluent_bundle)\n\nImplementation of the low-level Fluent Localization System providing localization capabilities for any Rust project.\n\n## fluent-fallback [![crates.io](http://meritbadge.herokuapp.com/fluent_fallback)](https://crates.io/crates/fluent_fallback)\n\nImplementation of the high-level Fluent Localization System providing localization capabilities for any Rust project.\n\n## fluent-resmgr [![crates.io](http://meritbadge.herokuapp.com/fluent_resmgr)](https://crates.io/crates/fluent_resmgr)\n\nResource Manager for localization resources.\n\n## fluent-cli [![crates.io](http://meritbadge.herokuapp.com/fluent_cli)](https://crates.io/crates/fluent_cli)\n\nCollection of command line tools for Fluent.\n'
