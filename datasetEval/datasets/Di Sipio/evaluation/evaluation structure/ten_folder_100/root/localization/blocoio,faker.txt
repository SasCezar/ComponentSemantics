b'# Faker\n\nGenerates fake data for testing or populating a development database.\nRun your tests with realistic data like names, emails, dates, countries...\n\nA Java port of the [Faker ruby gem](https://github.com/stympy/faker/).\nThe goal was to reuse their locale data files.\n\n[![Build Status](https://travis-ci.org/blocoio/faker.svg?branch=master)](https://travis-ci.org/blocoio/faker)\n[![Release](https://img.shields.io/github/release/blocoio/faker.svg?label=maven)](https://jitpack.io/#blocoio/faker)\n\n## Installing\n\n    repositories {\n        // ...\n        maven { url \'https://jitpack.io\' }\n    }\n\n    dependencies {\n        compile \'com.github.blocoio:faker:1.2.8\'\n    }\n\nYou can use ```testCompile``` or ```androidTestCompile```, if you only want to use Faker for testing.\n\n## Usage\n\n    Faker faker = new Faker();\n    faker.name.firstName();    // Returns "Aaron"\n    faker.company.name();      // Returns "Hirthe-Ritchie"\n\n    Faker faker = new Faker("nl");\n    faker.name.firstName();    // Returns "Thijs"\n\nFor full list of available options, check the [original source](https://github.com/stympy/faker/blob/master/README.md).\nWe have tried to keep the operations as close as possible.\n\n### Available components\n\n - Address\n - App\n - Avatar\n - Book\n - Bool\n - Business\n - Color\n - Commerce\n - Company\n - Date\n - Food\n - Internet\n - Lorem\n - Name\n - Number\n - PhoneNumber\n - Placeholdit\n - SlackEmoji\n - Team\n - Time\n - University\n\n### Warnings\n\n - Currently we don\'t support locale specific operations\n like ```Address.postcode_by_state``` or ```Business.swedish_organisation_number```.\n\n## TODO\n\n - Implement remaining components of the [Faker ruby gem](https://github.com/stympy/faker/).\n\n## Proguard\n\nTo use Faker with proguard, here are the rules you need to add:\n\n    # SnakeYAML\n    -keep class org.yaml.snakeyaml.** { public protected private *; }\n    -keep class org.yaml.snakeyaml.** { public protected private *; }\n    -dontwarn org.yaml.snakeyaml.**\n    \n    # Joda Time\n    -dontwarn org.joda.convert.**\n    -dontwarn org.joda.time.**\n    -keep class org.joda.time.** { *; }\n    -keep interface org.joda.time.** { *; }\n\n\n## Contact\n\nFeedback and contributions are welcome.\nFeel free to send an [email](mailto:hello@bloco.io) or submit a pull request.\n\n## License\n\nThis code is free to use under the terms of the [MIT license](http://opensource.org/licenses/MIT).\n'
