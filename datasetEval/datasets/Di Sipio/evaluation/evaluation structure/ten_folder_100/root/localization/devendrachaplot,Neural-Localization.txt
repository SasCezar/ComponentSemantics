b'# Active Neural Localization\nThis is a PyTorch implementation of the ICLR-18 paper:\n\n[Active Neural Localization](https://arxiv.org/abs/1801.08214)<br />\nDevendra Singh Chaplot, Emilio Parisotto, Ruslan Salakhutdinov<br />\nCarnegie Mellon University\n\nProject Website: https://devendrachaplot.github.io/projects/Neural-Localization\n\n### This repository contains:\n- Code for the Maze2D Environment which generates random 2D mazes for active localization.\n- Code for training an Active Neural Localization agent in Maze2D Environment using A3C.\n\n## Dependencies\n- [PyTorch](http://pytorch.org) (v0.3)\n\n## Usage\n\n### Training\nFor training an Active Neural Localization A3C agent with 16 threads on 7x7 mazes with maximum episode length 30:\n```\npython a3c_main.py --num-processes 16 --map-size 7 --max-episode-length 30 --dump-location ./saved/ --test-data ./test_data/m7_n1000.npy\n```\nThe code will save the best model at `./saved/model_best` and the training log at `./saved/train.log`. The code uses `./test_data/m7_n1000.npy` as the test data and makes sure that any maze in the test data is not used while training.\n\n### Evaluation\nAfter training, the model can be evaluated using:\n```\npython a3c_main.py --num-processes 0 --evaluate 1 --map-size 7 --max-episode-length 30 --load ./saved/model_best --test-data ./test_data/m7_n1000.npy\n```\n\n### Pre-trained models\nThe `pretrained_models` directory contains pre-trained models for map-size 7 (max-episode-length 15 and 30), map-size 15 (max-episode-length 20 and 40) and map-size 21 (max-episode-length 30 and 60). The test data used for training these models is provided in the `test_data` directory.\n\nFor evaluating a pre-trained model on maze size 15x15 with maximum episode length 40:\n```\npython a3c_main.py --num-processes 0 --evaluate 1 --map-size 15 --max-episode-length 40 --load ./pretrained_models/m15_l40 --test-data ./test_data/m15_n1000.npy\n```\n\n### Generating test data\nThe repository contains test data of map-sizes 7, 15 and 21 with 1000 mazes each in the `test_data` directory. \n\nFor generating more test data:\n```\npython generate_test_data.py --map-size 7 --num-mazes 100 --test-data-location ./test_data/ --test-data-filename my_new_test_data.npy\n```\nThis will generate a test data file at `test_data/my_new_test_data.npy` containing 100 7x7 mazes.\n\n### All arguments\nAll arguments for a3c_main.py:\n```\n  -h, --help            show this help message and exit\n  -l L, --max-episode-length L\n                        maximum length of an episode (default: 30)\n  -m MAP_SIZE, --map-size MAP_SIZE\n                        m: Size of the maze m x m (default: 7), must be an odd\n                        natural number\n  --lr LR               learning rate (default: 0.001)\n  --num-iters NS        number of training iterations per training thread\n                        (default: 10000000)\n  --gamma G             discount factor for rewards (default: 0.99)\n  --tau T               parameter for GAE (default: 1.00)\n  --seed S              random seed (default: 1)\n  -n N, --num-processes N\n                        how many training processes to use (default: 8)\n  --num-steps NS        number of forward steps in A3C (default: 20)\n  --hist-size HIST_SIZE\n                        action history size (default: 5)\n  --load LOAD           model path to load, 0 to not reload (default: 0)\n  -e EVALUATE, --evaluate EVALUATE\n                        0:Train, 1:Evaluate on test data (default: 0)\n  -d DUMP_LOCATION, --dump-location DUMP_LOCATION\n                        path to dump models and log (default: ./saved/)\n  -td TEST_DATA, --test-data TEST_DATA\n                        Test data filepath (default: ./test_data/m7_n1000.npy)\n```\n\n\n\n## Cite as\n>Chaplot, Devendra Singh, Parisotto, Emilio and Salakhutdinov, Ruslan.\nActive Neural Localization. \nIn *International Conference on Learning Representations*, 2018. \n([PDF](http://arxiv.org/abs/1801.08214))\n\n### Bibtex:\n```\n@inproceedings{chaplot2018active,\n  title={Active Neural Localization},\n  author={Chaplot, Devendra Singh and
