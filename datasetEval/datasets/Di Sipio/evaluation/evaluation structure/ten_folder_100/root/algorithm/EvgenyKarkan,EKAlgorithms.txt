b'[![Build Status](https://travis-ci.org/EvgenyKarkan/EKAlgorithms.svg)](https://travis-ci.org/EvgenyKarkan/EKAlgorithms)\n\nEKAlgorithms\n============\n\nEKAlgorithms is a set of computer exercises implemented in Objective-C. \nData structures, well known algorithms, CS curiosities, you name it!\n\nDon\'t forget to *watch* the repository; Its content will be expanded and updated frequently.\n\nArrays and Lists\n----------------------\n \n1. Index of maximum element in array.\n2. Indexes of maximum and minimum elements simultaneously.\n3. Find longest string in array of strings. \n4. Find shortest string in array of strings.\n5. Array reverse.\n6. Intersection of two arrays.\n7. Union of two arrays (with remove duplicates).\n8. Union of two arrays (with remove duplicates) for some key.\n9. Find duplicates.\n10. Array with N unique/not unique random objects.\n11. Check if array is sorted.\n12. Array shuffle (Fisher-Yates).\n13. Sum of array elements.\n14. N of occurences of each element in array.\n\nSearch Algorithms\n-----------------\n1. Linear search.\n2. Binary search.\n\nSorting Algorithms\n------------------\n1. Bubble sort.\n2. Shell sort.\n3. Merge sort.  \n4. Quick sort.  \n5. Insertion sort.\n6. Selection sort.\n7. Radix Sort.\n8. Partial selection sort.\n9. Heap sort.\n\nSelection Algorithms\n--------------------\n1. Quickselect.\n\nStrings\n-------\n1. Palindrome or not.\n2. String reverse.\n3. Words count.\n4. Permutations of string. \n5. Occurrences of each character (a - z).\n6. Count "needles" in a "haystack".\n7. Random string.\n8. Concatenation of two strings.\n9. Find 1st occurrence of "needle" in a "haystack".\n10. Last occurrence of "needle" in a "haystack".\n11. Longest common subsequence.\n12. Levenshtein distance.\n13. KMP (Knuth\xe2\x80\x93Morris\xe2\x80\x93Pratt).\n14. Boyer\xe2\x80\x93Moore string search algorithm.\n\nNumeric Algorithms\n------------------\n1. Sieve of Eratosthenes.\n2. Great common divisor (GCD).\n3. Least common multiple (LCM).\n4. Factorial.\n5. Fibonacci numbers (5 algos).\n6. Sum of digits.\n7. Binary to decimal conversion.\n8. Decimal to binary conversion.\n9. Fast exponentiation.\n10. Number reverse.\n11. Even/odd check.\n12. Leap year check.\n13. Armstrong number check.\n14. Prime number check.\n15. Find Nth prime.\n16. Swap the value of two NSInteger pointers.\n17. Square root using Newton-Raphson method.\n18. Convert integer to another numeral system (2, 8, 12, 16).  \n19. Fast inverse square root.\n\nData Structures\n---------------\n1. Stack (LIFO).\n2. Queue (FIFO).\n3. Deque.\n4. Linked list.\n5. Graph: \n    - DFS (depth-first search);\n    - BFS (breadth-first search);\n    - MST (minimum spanning tree - Prim\'s algorithm);\n    - MST (minimum spanning tree - Kruskal\'s algorithm);\n    - Shortest path (Dijkstra\'s algorithm);\n    - Topsort.\n6. Tree.  \n7. Binary tree:  \n    - Pre-order traversal;\n    - In-order traversal;\n    - Post-order traversal.\n8. Binary search tree (BST).  \n9. AVL tree.\n10. Binary heap.\n\nProblems\n--------\n1. Josephus Problem.\n2. Modulo bias.\n\nGeometry\n--------\n1. Array of sorted locations according to a distance to a given location.  \n2. Cartesian quadrant selection algorithms\n\nRecursion  \n---------\n1. Tower of Hanoi.\n\nContributions\n-------------   \nPull requests are welcome!
