Language doesn\'t matter. Just make sure you\'re implementing an algorithm.\n * PRs are welcome. To begin developing, follow the structure:\n\n   > algorithm_name/language-name/file_name.extension\n   e.g\n   > bubble_sort/python/bubble_sort.py\n\n * If there is an implementation of the same algorithm in your language, add your username in front of the file name.\n * Please include a description for the algorithm that you are implementing. It doesn\'t matter if it\'s copied from somewhere as long as it helps people that are learning new algorithm.\n * Graphical examples would be very helpful too.\n * Don\'t forget to include tests.\n * Don\'t remove previous implementations of algorithms. Just add a new file with your own implementation.\n * Beautify and cleanup your code for easier reading.\n\n## Resources\n\n Curated list of resources dealing with algorithms.\n\n * **Books**\n   *
