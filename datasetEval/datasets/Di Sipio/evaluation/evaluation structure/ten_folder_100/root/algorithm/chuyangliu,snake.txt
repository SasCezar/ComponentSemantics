b"# Snake\n\n[![][badge-travis]][build-travis] [![][badge-appveyor]][build-appveyor] ![][badge-python]\n\nThis project focuses on the artificial intelligence of the [Snake][wiki-snake] game. The snake's goal is to eat the food continuously and fill the map with its bodies ASAP. The [old version][snake-proj-old] of this project is written in C++. Now it has been rewritten using Python for a user-friendly GUI and the simplicity in the implementations of algorithms.\n\n***[Algorithms >][doc-algorithms]***\n\n## Experiments\n\nWe use two metrics to evaluate the performance of an AI:\n\n1. **Average Length:** Average length the snake has grown to (*max:* 64).\n2. **Average Steps:** Average steps the snake has moved.\n\nTest results (averaged over 1000 episodes):\n\n| Solver | Demo (optimal) | Average Length | Average Steps |\n| :----: | :------------: | :------------: | :-----------: |\n|[Hamilton][doc-hamilton]|![][demo-hamilton]|63.93|717.83|\n|[Greedy][doc-greedy]|![][demo-greedy]|60.15|904.56|\n|[DQN][doc-dqn]<br>(experimental)|![][demo-dqn]|24.44|131.69|\n\n## Installation\n\nRequirements: Python 3.5+ (64-bit) with [Tkinter][doc-tkinter] installed.\n\n```\n$ pip3 install -r requirements.txt\n\n# Use -h for more details\n$ python3 run.py [-h]\n```\n\nRun unit tests:\n\n```\n$ python3 -m pytest -v\n```\n\n## License\n\nSee the [LICENSE](./LICENSE) file for license rights and limitations.\n\n\n[snake-proj-old]:
