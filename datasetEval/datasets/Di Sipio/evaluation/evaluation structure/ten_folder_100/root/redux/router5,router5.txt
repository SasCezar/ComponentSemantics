with prettier](https://img.shields.io/badge/styled_with-prettier-ff69b4.svg)](https://github.com/prettier/prettier)\n\n> Official website: [router5.js.org](https://router5.js.org)\n\nrouter5 is a **framework and view library agnostic router**.\n\n* **view / state separation**: router5 processes routing **instructions** and outputs **state** updates.\n* **universal**: works client-side and server-side\n* **simple**: define your routes, start to listen to route changes\n* **flexible**: you have control over transitions and what happens on transitions\n\n```javascript\nimport createRouter from 'router5'\nimport browserPlugin from 'router5-plugin-browser'\n\nconst routes = [\n    { name: 'home', path: '/' },\n    { name: 'profile', path: '/profile' }\n]\n\nconst router = createRouter(routes)\n\nrouter.usePlugin(browserPlugin())\n\nrouter.start()\n```\n\n**With React \\(new context API\\)**\n\n```javascript\nimport React from 'react'\nimport ReactDOM from 'react-dom'\nimport { RouteProvider, Route } from 'react-router5'\n\nfunction App({ route }) {\n    if (!route) {\n        return null\n    }\n\n    if (route.name === 'home') {\n        return <h1>Home</h1>\n    }\n\n    if (route.name === 'profile') {\n        return <h1>Profile</h1>\n    }\n}\n\nReactDOM.render(\n    <RouteProvider router={router}>\n        <Route>{({ route }) => <App route={route} />}</Route>\n    </RouteProvider>,\n    document.getElementById('root')\n)\n```\n\n**With observables**\n\nYour router instance is compatible with most observable libraries.\n\n```javascript\nimport { from } from 'rxjs/observable/from'\n\nfrom(router).map(({ route }) => {\n    /* happy routing */\n})\n```\n\n### Examples\n\n* With React:
