Currently supports only OS X.\n\n\n### Tests\nThere are 2 linters for `js` & `scss` and unit tests with `jest`.\n\n    // Run only unit tests\n    npm run jest\n\n    // Run linter & unit tests with coverage\n    npm run test\n\n### FAQ\n\n#### My notifications aren't showing?\n\nSome organisations require applications to request access before allowing access to any data (including notifications) about their repositories.\n\nTo check if Gitify is approved by your organisation you can go to https://github.com/settings/applications, then click on **Gitify** and scroll to _Organization access_.\n\n#### Something looks wrong - How can I debug?\n\nSince version `1.1.0` you can now debug Gitify by pressing `alt+cmd+I`. This will open the devtools and then you can see any logs, network requests etc.\n\n\n### Contributing\n\n1. Fork it!\n2. Create your feature branch: `git checkout -b my-new-feature`\n3.
