Please consider using [Formik](https://github.com/jaredpalmer/formik) instead.\n\n## [Read the Documentation](https://davidkpiano.github.io/react-redux-form/docs.html)\n\nReact Redux Form is a collection of reducer creators and action creators that make implementing even the most complex and custom forms with React and Redux simple and performant.\n\n`npm install react-redux-form@latest --save`\n\n## Installation\n\n```bash\n# Dependencies (you probably already have these)\nnpm install react react-dom redux react-redux --save\n\n# version 1.x.x\nnpm install react-redux-form@latest --save\n```\n\n## Zero-Boilerplate `<LocalForm>`\n\nIf you want to get up and running with forms quickly without having to set up Redux, [just use Local Forms](http://davidkpiano.github.io/react-redux-form/docs/guides/local.html):\n\n```js\nimport React from \'react\';\nimport { LocalForm, Control } from \'react-redux-form\';\n\nexport default class MyApp extends React.Component {\n  handleChange(values) { ... }\n  handleUpdate(form) { ... }\n  handleSubmit(values) { ... }\n  render() {\n    return (\n      <LocalForm\n        onUpdate={(form) => this.handleUpdate(form)}\n        onChange={(values) => this.handleChange(values)}\n        onSubmit={(values) => this.handleSubmit(values)}\n      >\n        <Control.text model=".username" />\n        <Control.text model=".password" />\n      </LocalForm>\n    );\n  }\n}\n```\n\nThat\'s all you need. Seriously. [Read the Documentation](http://davidkpiano.github.io/react-redux-form/docs/guides/local.html) for more information.\n\n**NOTE:** Please use `<LocalForm>` with `react-redux` version 4.x.x or 5.x.x. \n\n## Quick Start\nFor more fine-grained control (such as using custom reducers, storing form state globally, dispatching actions, etc.), you\'ll want to set up a Redux store for your forms.\n\nBe sure to read the [step-by-step quick start guide](http://davidkpiano.github.io/react-redux-form/docs/guides/quickstart.html) in the documentation.\n\n```jsx\nimport React from \'react\';\nimport { createStore } from \'redux\';\nimport { Provider } from \'react-redux\';\nimport { combineForms } from \'react-redux-form\';\n\nimport MyForm from \'./components/my-form-component\';\n\nconst initialUser = { name: \'\' };\n\nconst store = createStore(combineForms({\n  user: initialUser,\n}));\n\nclass App extends React.Component {\n  render() {\n    return (\n      <Provider store={ store }>\n        <MyForm />\n      </Provider>\n    );\n  }\n}\n```\n\n```jsx\n// ./components/my-form-component.js\'\nimport React from \'react\';\nimport { connect } from \'react-redux\';\nimport { Control, Form } from \'react-redux-form\';\n\nclass MyForm extends React.Component {\n  handleSubmit(val) {\n    // Do anything you want with the form value\n    console.log(val);\n  }\n\n  render() {\n    return (\n      <Form model="user" onSubmit={(val) => this.handleSubmit(val)}>\n        <label>Your name?</label>\n        <Control.text model=".name" />\n        <button>Submit!</button>\n      </Form>\n    );\n  }\n}\n\n// No need to connect()!\nexport default MyForm;\n```\n\n## Posting Issues\nWhen posting an issue, please include a detailed description along with a relevant code sample.
