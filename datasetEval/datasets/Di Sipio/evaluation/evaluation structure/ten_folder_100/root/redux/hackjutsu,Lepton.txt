(macOS)\n```bash\nbrew cask install lepton\n```\n- Install via SnapCraft (Linux)\n\n[![Get it from the Snap Store](https://snapcraft.io/static/images/badges/en/snap-store-white.svg)](https://snapcraft.io/lepton)\n\n## Development\n\n\n### Install dependencies\n\n```bash\n$ git clone git@github.com:hackjutsu/Lepton.git\n$ cd Lepton && npm i\n```\n\n### Client ID/Secret\n[Register your application](https://github.com/settings/applications/new), and put your client id and client secret in `./configs/account.js`.\n```js\nmodule.exports = {\n  client_id: <your_client_id>,\n  client_secret: <your_client_secret>\n}\n```\n\n### Run\n```bash\n$ npm run build && npm run start\n```\n\n## Build Installer App\n>Read [electron-builder docs](https://github.com/electron-userland/electron-builder#readme) and check out the [code signing wiki](https://github.com/electron-userland/electron-builder#code-signing) before building the installer app.\n\nBuild apps for macOS.\n```bash\n$ npm run dist -- -m\n```\nBuild apps for Windows.\n```bash\n$ npm run dist -- -w\n```\nBuild apps for Linux. \n\n>Need a running [Docker](https://www.docker.com/) daemon to build a `snap` package.\n```bash\n$ npm run dist -- -l\n```\nBuild apps for macOS, Windows and Linux.\n```bash\n$ npm run dist -- -wml\n```\nBuild apps for the current OS with the current arch.\n```bash\n$ npm run dist\n```\n\n## FAQ\n[--> Wiki
