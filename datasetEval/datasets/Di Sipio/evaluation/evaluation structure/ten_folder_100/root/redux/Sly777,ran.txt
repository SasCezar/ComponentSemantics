command.\n\n```bash\ngit clone --depth=1 https://github.com/Sly777/ran.git RAN\ncd RAN\nyarn && yarn setup\n```\n\n- If you are not using Yarn, just run ```npm install && npm run setup``` instead of ```yarn && yarn setup```\n- After everything is finished, run ```yarn dev``` (or ```npm run dev```)\n\nAnd that\'s all!\n\n## Start coding on browser! (via [Glitch](https://glitch.com))\n\nJust click the button and then start to work with RAN!\n\n[![Remix on Glitch](https://cdn.glitch.com/2703baf2-b643-4da7-ab91-7ee2a2d00b5b%2Fremix-button.svg)](https://glitch.com/edit/#!/import/github/Sly777/ran)\n\n#### Clean Setup (without example pages & components)\n\n```bash\ngit clone --depth=1 https://github.com/Sly777/ran.git RAN\ncd RAN\nyarn && yarn setup:clean\n```\n\n#### Beta Version (Unstable)\n\nAlso, there is a ```beta``` version for new features & fixes that we are testing before release. To access beta;\n\n```bash\ngit clone --depth=1 -b beta https://github.com/Sly777/ran.git RAN_beta\ncd RAN_beta\nyarn && yarn setup\n```\n\nIt can be unstable, so that\'s why please use stable version if you are working on the project that is in production.\n\n## Example\n\n[Click here](https://ran.now.sh/) to see example project to understand how RAN! works on production.
