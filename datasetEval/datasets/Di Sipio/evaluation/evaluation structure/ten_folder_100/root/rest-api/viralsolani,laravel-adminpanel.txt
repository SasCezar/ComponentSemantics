[Official Documentation](https://laravel.com/docs/5.6/installation#installation)\n\n\nClone the repository\n\n    git clone https://github.com/viralsolani/laravel-adminpanel.git\n\nSwitch to the repo folder\n\n    cd laravel-adminpanel\n\nIf you have linux system, you can execute the command below only in your project root\n\n    1) sudo chmod -R 777 install.sh\n    2) ./install.sh\n\nIf you have windows system, you can run Artisan Command for database setup, connection and configuration.\n\n    php artisan install:app\n\nGenerate a new application key\n\n    php artisan key:generate\n\nInstall Passport\n\n    php artisan passport:install\n\nRun the database migrations (**Set the database connection in .env before migrating**)\n\n    php artisan migrate\n\nRun the database seeders\n\n    php artisan db:seed\n\nInstall the javascript dependencies using npm\n\n    npm install\n\nCompile the dependencies\n\n    npm run development\n\nFor generating the files of unisharp file manager\n\n    php artisan vendor:publish --tag=lfm_public\n\nFor linking storage folder in public\n\n    php artisan storage:link\n\nStart the local development server\n\n    php artisan serve\n\n\nYou can now access the server at http://localhost:8000\n\n**Command list**\n\n    git clone https://github.com/viralsolani/laravel-adminpanel.git\n    cd laravel-adminpanel\n    cp .env.example .env\n    composer install\n    npm install\n    npm run development\n    php artisan storage:link\n    php artisan key:generate\n    php artisan passport:install\n    php artisan vendor:publish --tag=lfm_public\n    php artisan migrate\n    php artisan passport:install\n\n## Please note\n\n- To run test cases, add SQLite support to your php\n\n## Other Important Commands\n- To fix php coding standard issues run -
