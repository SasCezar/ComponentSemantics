b'MongoDB README\r\n\r\nWelcome to MongoDB!\r\n\r\nCOMPONENTS\r\n\r\n  mongod - The database server.\r\n  mongos - Sharding router.\r\n  mongo  - The database shell (uses interactive javascript).\r\n\r\nUTILITIES\r\n\r\n  mongodump         - Create a binary dump of the contents of a database.\r\n  mongorestore      - Restore data from the output created by mongodump.\r\n  mongoexport       - Export the contents of a collection to JSON or CSV.\r\n  mongoimport       - Import data from JSON, CSV or TSV.\r\n  mongofiles        - Put, get and delete files from GridFS.\r\n  mongostat         - Show the status of a running mongod/mongos.\r\n  bsondump          - Convert BSON files into human-readable formats.\r\n  mongoreplay       - Traffic capture and replay tool.\r\n  mongotop          - Track time spent reading and writing data.\r\n  install_compass   - Installs MongoDB Compass for your platform.\r\n\r\nBUILDING\r\n\r\n  See docs/building.md.\r\n\r\nRUNNING\r\n\r\n  For command line options invoke:\r\n\r\n    $ ./mongod --help\r\n\r\n  To run a single server database:\r\n\r\n    $ sudo mkdir -p /data/db\r\n    $ ./mongod\r\n    $\r\n    $ # The mongo javascript shell connects to localhost and test database by default:\r\n    $ ./mongo\r\n    > help\r\n\r\nINSTALLING COMPASS\r\n\r\n  You can install compass using the install_compass script packaged with MongoDB:\r\n\r\n    $ ./install_compass\r\n\r\n  This will download the appropriate MongoDB Compass package for your platform\r\n  and install it.\r\n\r\nDRIVERS\r\n\r\n  Client drivers for most programming languages are available at\r\n  https://docs.mongodb.com/manual/applications/drivers/. Use the shell\r\n  ("mongo") for administrative tasks.\r\n\r\nBUG REPORTS\r\n\r\n  See https://github.com/mongodb/mongo/wiki/Submit-Bug-Reports.\r\n\r\nPACKAGING\r\n\r\n  Packages are created dynamically by the package.py script located in the\r\n  buildscripts directory.
