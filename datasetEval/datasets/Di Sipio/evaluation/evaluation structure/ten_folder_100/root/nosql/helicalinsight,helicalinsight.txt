b'## Helical Insight\n\nCommunity driven innovation via Helical Insight CE.\n\n## What is it\n\nHelical Insight is world\'s first Open Source Business Intelligence framework which can help you derive insights out of your one or multiple datasources. Helical Insight is having all the features which you generally expect out of any BI tool (be it open source or proprietary). Being a framework, Helical Insight is highly extensible via APIs and SDK, thus features can be extended whenever required without compromising on your business requirement.\n\nHelical Insight also comes with a unique Workflow rule engine, allowing you to call any functionality of Helical Insight or external functionality and thus empowering you to implement any sort of custom business process.\n\nUse HTML skillset and Java skillset to add functionalities at the frontend and backend respectively.\n\n**Click [here](https://helicalinsight.github.io/helicalinsight/#/quickstart) for quick installation guide.**\n\n## Features\n\n* New generation UI with one click access\n* Backend EFW method of reports, dashboards and other data analysis creation\n* User Role Management\n* Exporting to Multiple Formats\n* Email scheduling\n* Data Security\n* XML driven Workflow\n* API Driven Framework\n* Community Support\n* Community Upgrades\n* Direct links to tutorials\n* Mobile & Cloud compatible\n* Cache for faster Performance\n* Compatible with All Modern Browsers\n\n## Supported Databases\n\nWe support all the JDBC4 complaint databases, NoSQL, Big Data, RDBMS, Cloud db, Columnar database etc\n\n#### RDBMS\n\n* MySQL\n* PostgreSQL\n* SQL Server\n* Oracle\n* Firebird\n* Informix\n* Ingres\n* MariaDB\n* Presto\n* Progress\n* SQlite\n\n#### NoSQL & Big Data\n\n* Cassandra\n* Druid\n* HBase\n* MongoDb\n* Hive\n* NuoDB\n* Neo4j\n\n#### Cloud\n\n* Microsoft Azure SQL\n* Amazon RedShift Database\n* Google Cloud Sql\n\n#### Flat Files\n\n* CSV\n* TSV\n* JSON\n\n## How to build ?\n\nPrerequisite:\n\nTo build Helical Insight Community Edition project you need\n\n* `Maven 3` or higher version installed.\n* `JDK 7` or higher version installed.\n*  Apache tomcat 7 or 8 should be installed.\n*  Use Mysql 5.7 or higher version.\n\n `NOTE: In case database is not accessed remotely then grant all priviledges to user.`\n\n*  Database with name `hice` to be created in Mysql.
