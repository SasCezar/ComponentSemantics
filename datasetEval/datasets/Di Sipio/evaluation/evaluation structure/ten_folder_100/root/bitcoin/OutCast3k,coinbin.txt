b'coinbin\n=======\n\nA Open Source Browser Based Bitcoin Wallet. Version 1.4 beta by OutCast3k\n\nLive version available at http://coinb.in/ or http://4zpinp6gdkjfplhk.onion\n\nGithub URL: https://github.com/OutCast3k/coinbin/\n\nCoinb.in supports a number of key features such as: \n\n- Offline Compressed & uncompressed Address creation.\n- Offline Multisignature Address creation.\n- "In browser" Key (re)generation. \n- Send and receive payments.\n- Ability to decode transactions, redeem scripts and more offline.\n- Build custom transactions offline.\n- Sign transactions offline.\n- Signatures are deterministic as per RFC 6979 (https://tools.ietf.org/html/rfc6979#section-3.2)\n- Broadcast transactions.\n- nLockTime support.\n- Add custom data to transactions with the use of OP_RETURN.\n- Support current Dark Wallet Stealth Address structure (as of version Alpha 7) for outputs.\n- Brain wallet support.\n- Compatible with bitcoin-qt\n- An offical .onion address for tor users.\n- Offline qrcode creator and scanning tool\n- HD (bip32) support\n- Supports altcoins such as litecoin\n- Replace by fee (RBF) Support\n- Segwit Support\n- Bech32 address support\n- Fee calculator - https://coinb.in/#fees\n\nDonate to 3K1oFZMks41C7qDYBsr72SYjapLqDuSYuN to see more development!\n'
