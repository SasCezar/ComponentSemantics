b'# trezor-crypto\n\n[![Build Status](https://travis-ci.org/trezor/trezor-crypto.svg?branch=master)](https://travis-ci.org/trezor/trezor-crypto) [![gitter](https://badges.gitter.im/trezor/community.svg)](https://gitter.im/trezor/community)\n\nHeavily optimized cryptography algorithms for embedded devices.\n\nThese include:\n- AES/Rijndael encryption/decryption\n- Big Number (256 bit) Arithmetics\n- BIP32 Hierarchical Deterministic Wallets\n- BIP39 Mnemonic code\n- ECDSA signing/verifying (supports secp256k1 and nist256p1 curves,\n  uses RFC6979 for deterministic signatures)\n- ECDSA public key derivation\n- Base32 (RFC4648 and custom alphabets)\n- Base58 address representation\n- Ed25519 signing/verifying (also SHA3 and Keccak variants)\n- ECDH using secp256k1, nist256p1 and Curve25519\n- HMAC-SHA256 and HMAC-SHA512\n- PBKDF2\n- RIPEMD-160\n- SHA1\n- SHA2-256/SHA2-512\n- SHA3/Keccak\n- BLAKE2s/BLAKE2b\n- Chacha20-Poly1305\n- unit tests (using Check - check.sf.net; in test_check.c)\n- tests against OpenSSL (in test_openssl.c)\n- integrated Wycheproof tests\n\nDistibuted under MIT License.\n\n## Some parts of the library come from external sources:\n\n- AES: https://github.com/BrianGladman/aes\n- Base58: https://github.com/luke-jr/libbase58\n- BLAKE2s/BLAKE2b: https://github.com/BLAKE2/BLAKE2\n- RIPEMD-160: https://github.com/ARMmbed/mbedtls\n- SHA1/SHA2: http://www.aarongifford.com/computers/sha.html\n- SHA3: https://github.com/rhash/RHash\n- Curve25519: https://github.com/agl/curve25519-donna\n- Ed25519: https://github.com/floodyberry/ed25519-donna\n- Chacha20: https://github.com/wg/c20p1305\n- Poly1305: https://github.com/floodyberry/poly1305-donna\n'
