b"Gearboy\n=======\n<b>Copyright &copy; 2012 by Ignacio Sanchez</b>\n\n----------\n[![Build Status](https://travis-ci.org/drhelius/Gearboy.svg?branch=master)](https://travis-ci.org/drhelius/Gearboy)\n\nGearboy is a Nintendo Game Boy / GameBoy Color emulator written in C++ that runs on iOS, Raspberry Pi, Mac, Windows, Linux and RetroArch.\n\nFollow me on Twitter for updates: http://twitter.com/drhelius\n\n----------\n\nDownloads\n--------\n- **iOS**: Build Gearboy with Xcode and transfer it to your device. You can open rom files from other apps like Safari or Dropbox, or use [iTunes file sharing](http://support.apple.com/kb/ht4094).\n- **Mac OS X**: <code>brew install gearboy</code>\n- **Windows**: [Gearboy-2.7.0-Windows.zip](https://github.com/drhelius/Gearboy/releases/download/gearboy-2.7.0/Gearboy-2.7.0-Windows.zip) (NOTE: You may need to install the [Microsoft Visual C++ Redistributable](https://go.microsoft.com/fwlink/?LinkId=746572))\n- **Linux**: [Gearboy-2.7.0-Linux.tar.xz](https://github.com/drhelius/Gearboy/releases/download/gearboy-2.7.0/Gearboy-2.7.0-Linux.tar.xz)\n- **RetroArch**: [Libretro core documentation](https://docs.libretro.com/library/gearboy/).\n- **Raspberry Pi**: Build Gearboy from sources. Optimized projects are provided for Raspberry Pi 1, 2 and 3.\n- **Ubuntu Touch** version by Ryan Pattison: [here](https://uappexplorer.com/app/gearboy.rpattison)\n\nFeatures\n--------\n- Highly accurate CPU emulation, passes cpu_instrs.gb from blargg's tests.\n- Accurate instruction and memory timing, passes instr_timing.gb and mem_timing.gb from blargg's tests.\n- Memory Bank Controllers (MBC1, MBC2, MBC3 with RTC, MBC5), ROM + RAM and multicart cartridges.\n- Accurate LCD controller emulation. Background, window and sprites, with correct timings and priorities including mid-scanline timing.\n- Mix frames: Mimics the LCD ghosting effect seen in the original Game Boy.\n- Sound emulation using SDL Audio and [Gb_Snd_Emu library](http://blargg.8bitalley.com/libs/audio.html#Gb_Snd_Emu).\n- Game Boy Color support.\n- Integrated disassembler. It can dump the full disassembled memory to a text file or access it in real time.\n- Saves battery powered RAM cartridges to file.\n- Save states.\n- Compressed rom support (ZIP deflate).\n- Game Genie and GameShark cheat support.\n- Multi platform. Runs on Windows, Linux, Mac OS X, Raspberry Pi, iOS and as a libretro core (RetroArch).\n\nBuild Instructions\n----------------------\n\n### iOS\n- Install Xcode for Mac OS X. You need iOS SDK 8 or later.\n- Build the project <code>platforms/ios/Gearboy.xcodeproj</code>\n- Run it on real hardware using your iOS developer certificate. Make sure it builds on Release for better performance.\n\n### Raspberry Pi 2 & 3 - Raspbian\n- Install and configure [SDL 2](http://www.libsdl.org/download-2.0.php) for development:\n``` shell\nsudo apt-get update\nsudo apt-get upgrade\nsudo apt-get install build-essential libfreeimage-dev libopenal-dev libpango1.0-dev libsndfile-dev libudev-dev libasound2-dev libjpeg-dev libtiff5-dev libwebp-dev automake\ncd ~\nwget https://www.libsdl.org/release/SDL2-2.0.9.tar.gz\ntar zxvf SDL2-2.0.9.tar.gz\ncd SDL2-2.0.9 && mkdir build && cd build\n../configure --disable-pulseaudio --disable-esd --disable-video-mir --disable-video-wayland --disable-video-x11 --disable-video-opengl --host=armv7l-raspberry-linux-gnueabihf\nmake -j 4\nsudo make install\n```\n- Install libconfig library dependencies for development: <code>sudo apt-get install libconfig++-dev</code>\n- Use <code>make -j 4</code> in the <code>platforms/raspberrypi3/x64/</code> folder to build the project.\n- Use <code>export SDL_AUDIODRIVER=ALSA</code> before running the emulator for the best performance.\n- Gearboy generates a <code>gearboy.cfg</code> configuration file where you can customize keyboard and gamepads. Key codes are from [SDL](https://wiki.libsdl.org/SDL_Keycode).\n\n### Windows\n- You need Visual Studio 2017 or later.\n- Install the [Qt 5 Open Source SDK for Windows](https://www.qt.io/download/).\n- Install the [QtVisualStudioTools Extension](https://marketplace.visualstudio.com/items?itemName=TheQtCompany.QtVisualStudioTools-19123) and point it to the Qt SDK.\n- Open the Gearboy Visual Studio solution <code>platforms/windows/Gearboy/Gearboy.sln</code> and build.\n- You may want to use the <code>platforms/windows/Gearboy/Gearboy.pro</code> project file with Qt Creator instead.\n\n### Mac OS X\n- You need Qt Creator, included in the Qt 5 SDK.\n- Install Xcode and run <code>xcode-select --install</code> in the terminal for the compiler to be available on the command line.\n- Install the [Qt 5 SDK for Mac OS](https://www.qt.io/download/).\n- Download [SDL 2](http://www.libsdl.org/download-2.0.php) source code. Then run this commands:\n``` shell\n./configure\nmake\nsudo make install\n```\n- Open the <code>platforms/macosx/Gearboy/Gearboy.pro</code> project file with Qt Creator and build.\n\n### Linux\n- Ubuntu / Debian:\n``` shell\nsudo apt-get install build-essential qt5-default qttools5-dev-tools freeglut3-dev libsdl2-dev libglew-dev\ncd platforms/linux/Gearboy\nqmake Gearboy.pro && make\n```\n- Fedora:\n``` shell\nsudo dnf install @development-tools gcc-c++ qt5-devel freeglut-devel SDL2-devel glew-devel\ncd platforms/linux/Gearboy\nqmake-qt5 Gearboy.pro && make\n```\n\nAccuracy
