b"# MERN CRUD\n\nA simple records system using MongoDB, Express.js, React.js, and Node.js with real-time Create, Read, Update, and Delete operations using Socket.io. REST API was implemented on the back-end. Semantic UI React was used for the UI.\n\n![Node Version](https://img.shields.io/badge/node-v6.11.0-yellowgreen.svg)\n![NPM Version](https://img.shields.io/badge/npm-v3.10.10-blue.svg)\n![MongoDB Version](https://img.shields.io/badge/mongodb-v3.4.2-blue.svg)\n![Mongoose Version](https://img.shields.io/badge/mongoose-v4.10.8-blue.svg)\n![Language: American English](https://img.shields.io/badge/language-american%20english-red.svg)\n\n\nDemo: [https://mern-crud.herokuapp.com/](https://mern-crud.herokuapp.com/)\n\n![MERN CRUD Screenshot](screenshot.png)\n\n## Show Some :heart: :wave:\n[![GitHub Stars](https://img.shields.io/github/stars/cefjoeii/mern-crud.svg?style=social&label=Star)](https://github.com/cefjoeii/mern-crud)\n[![GitHub Forks](https://img.shields.io/github/forks/cefjoeii/mern-crud.svg?style=social&label=Fork)](https://github.com/cefjoeii/mern-crud/fork)\n[![GitHub Watchers](https://img.shields.io/github/watchers/cefjoeii/mern-crud.svg?style=social&label=Watch)](https://github.com/cefjoeii/mern-crud)\n\n[![Follow on GitHub](https://img.shields.io/github/followers/cefjoeii.svg?style=social&label=Follow)](https://github.com/cefjoeii)\n[![Follow on Facebook](https://img.shields.io/badge/Follow%20%40cefjoeii%20on-Facebook-%233C5A99.svg)](https://facebook.com/cefjoeii)\n[![Follow on Instagram](https://img.shields.io/badge/Follow%20%40cefjoeii%20on-Instagram-C13584.svg)](https://instagram.com/cefjoeii)\n[![Follow on Twitter](https://img.shields.io/twitter/follow/cefjoeii.svg?style=social)](https://twitter.com/cefjoeii)\n\n## Instructions\n\nFork, then download or clone the repo.\n```bash\ngit clone https://github.com/<your-user-name>/mern-crud.git\n```\n\n\n*Make sure MongoDB service is running.*\n\nThe *config* folder contains a file named *db.js*. Before running locally, change the value of db as seen in the code below.\n```js\nmodule.exports = {\n  db: 'mongodb://localhost/mern-crud'\n};\n```\n\nFor the **back-end**, install the dependencies once via the terminal.\n```bash\nnpm install\n```\n\nRun the *main server*. It listens on port 3000.\n```bash\nCORS=1 node server\n```\nView it on the browser.\n\n<br>\n\nIf you want to configure the **front-end**, go to *react-src* folder via the terminal.\n\n```bash\ncd react-src\n```\n\nInstall the dependencies required by React once.\n```bash\nnpm install\n```\n\nRun the *development server* for React. It listens on port 4200.\n```bash\nREACT_APP_API_URL=http://localhost:3000 npm start\n```\n\nTo make a production build, simply run on *react-src* folder via the terminal.\n```bash\nnpm run build\n```\n\nIt re-creates a folder named *public* on the root directory. This is where the production-ready front-end of the web application resides.\n\n## Docker\n```bash\ndocker-compose up\n```\n\n\n## Contribute\nFeel free to help out as I may have other work/life commitments.
