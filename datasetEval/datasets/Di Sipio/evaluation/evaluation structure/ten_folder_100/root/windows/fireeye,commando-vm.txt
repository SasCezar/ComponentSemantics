Create and configure a new Windows Virtual Machine\n  * Ensure VM is updated completely. You may have to check for updates, reboot, and check again until no more remain \n* Take a snapshot of your machine!\n* Download and copy `install.ps1` on your newly configured machine. \n* Open PowerShell as an Administrator\n* Enable script execution by running the following command:\n  * `Set-ExecutionPolicy Unrestricted`\n* Finally, execute the installer script as follows:\n  * `.\\install.ps1`\n  * You can also pass your password as an argument: `.\\install.ps1 -password <password>`\n  \nThe script will set up the Boxstarter environment and proceed to download and install the Commando VM environment. You will be prompted for the administrator password in order to automate host restarts during installation. If you do not have a password set, hitting enter when prompted will also work.\n\nCustom install\n--------------\n1.\tDownload the zip from https://github.com/fireeye/commando-vm into your Downloads folder.\n2.\tDecompress the zip and edit the `${Env:UserProfile}\\Downloads\\commando-vm-master\\commando-vm-master\\profile.json` file by removing tools or adding tools in the \xe2\x80\x9cpackages\xe2\x80\x9d section.
