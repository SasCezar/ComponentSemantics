b"# Media Player Classic - Home Cinema\n\n[![GitHub Tag](https://img.shields.io/github/tag/mpc-hc/mpc-hc.svg?label=version)](https://github.com/mpc-hc/mpc-hc)\n[![Coverity Scan Build Status](https://img.shields.io/coverity/scan/259.svg)](https://scan.coverity.com/projects/259)\n\n\nMedia Player Classic - Home Cinema (MPC-HC) is a free and open-source (OSS) video\nand audio player for Windows. MPC-HC is based on the original Guliverkli project\nand contains many additional features and bug fixes.\n\nWe are in dire need of new developers of any kind. If you can code a little, or you can create\nlogos for the player and images for the file associations, or you can create/maintain the main\nwebsite, please join our IRC channel and talk to us. Any help is always appreciated.\n\nFind some easy ticket on our Trac, fix the problem, send a [pull request](https://github.com/mpc-hc/mpc-hc/pulls),\nor post the patch on [Trac](https://trac.mpc-hc.org/) or come on IRC and we'll review it for you.\nSee [CONTRIBUTING.md](/CONTRIBUTING.md) for more info.\n\n*We care about keeping an open project open and free for contributions wherever they may come from.*\n\n\n## Main Features\n\n* Option to remove [tearing](https://en.wikipedia.org/wiki/Screen_tearing)\n* Support for modern Windows versions, including a 64-bit build\n* Support for [EVR/EVR CP](https://en.wikipedia.org/wiki/Media_Foundation#Enhanced_Video_Renderer) (Enhanced Video Renderer)\n* Supports most types of [subtitles](https://en.wikipedia.org/wiki/Subtitle_%28captioning%29#Subtitle_formats)\n  including [BluRay](https://en.wikipedia.org/wiki/Blu-ray_Disc) .sup files\n* Playback and recording of television programs, if a supported\n  [TV tuner card](https://en.wikipedia.org/wiki/TV_tuner_card) is installed\n* [OSD info (On Screen Display)](https://en.wikipedia.org/wiki/On-screen_display)\n* Multi-Monitor support\n* Various [pixel shaders](https://en.wikipedia.org/wiki/Shader#Pixel_shaders)\n* [Color management](https://en.wikipedia.org/wiki/Color_management)\n* 42 translations available\n\n\n## System Requirements\n\n* An SSE2-capable CPU\n* Windows Vista, 7, 8, 8.1, 10 32-bit/64-bit\n\n\n## Links\n\n* [Website](https://mpc-hc.org/)\n* [Source
