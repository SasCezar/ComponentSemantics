[![Download](http://i.imgur.com/odToka3.png)](https://ci.appveyor.com/project/tidusjar/requestplex/branch/develop/artifacts)       | \n# Features\nHere are some of the features Ombi V3 has:\n* Now working without crashes on Linux.\n* Lets users request Movies, Music, and TV Shows (whether it being the entire series, an entire season, or even single episodes.)\n* Easily manage your requests\n* Allows you to set specific users to automatically have requests approved and added to the relevant service (Sonarr/Radarr/Lidarr/Couchpotato etc)\n* User management system (supports plex.tv, Emby and local accounts)\n* A landing page that will give you the availability of your Plex/Emby server and also add custom notification text to inform your users of downtime.\n* Allows your users to get custom notifications!\n* Secure authentication using best practises\n* Will show if the request is already on plex or even if it\'s already monitored.\n* Automatically updates the status of requests when they are available on Plex/Emby\n* Slick, responsive and mobile friendly UI\n* Ombi will automatically update itself :) (YMMV)\n* Very fast!\n\n### Integration \nWe integrate with the following applications:\n* Plex Media Server\n* Emby\n* Sonarr\n* Radarr\n* Lidarr\n* DogNzb\n* Couch Potato\n\n\n### Notifications\nSupported notifications:\n* SMTP Notifications (Email)\n* Discord\n* Gotify\n* Slack\n* Pushbullet\n* Pushover\n* Mattermost\n* Telegram\n\n### The difference between Version 3 and 2\n\nOver the last year, we focused on the main functions on Ombi, a complete rewrite
