b'[![Build Status](https://travis-ci.com/monero-rs/monero-rs.svg?branch=master)](https://travis-ci.com/monero-rs/monero-rs) [![Crates.io](https://img.shields.io/crates/v/monero.svg)](https://crates.io/crates/monero) [![Documentation](https://docs.rs/monero/badge.svg)](https://docs.rs/monero) [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)\n\nRust Monero Library\n===\n\nLibrary with support for de/serialization, parsing and executing on data structures and network messages related to Monero currency.\n\nSupports (or should support)\n\n * De/serialization of Monero blocks and transactions\n * Address and subaddress creation, de/serialization and validation\n * Private keys and one-time keys creation, de/serialization and validation\n\n## Documentation\n\nCurrently the documentation is very sparse. Patches to add usage examples and to expand on existing docs would be extremely appreciated.\n\nContributing\n===\n\nContributions are welcome.\n\n## Building\n\nThe library can be built and tested using cargo:\n\n```\ngit clone git@github.com:monero-rs/monero-rs.git\ncd monero-rs\ncargo build\n```\n\nYou can run tests with:\n\n```\ncargo test\n```\n\n## Building for WASM\n\nThis library can be built to target wasm platform with:\n\n```\ncargo build --target wasm32-unknown-unknown --no-default-features\n```\n\nAbout\n===\n\nThis is a research project sponsored by TrueLevel, developed by h4sh3d.\n'
