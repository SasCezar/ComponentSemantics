b"# xmr-stak-cpu-ios\niOS port of the XMR-STAK-CPU mining software. The port allows to mine Monero cryptocurrency on iPhones and iPads.\n\nWorkability of this solution was tested on iOS11 and Xcode9.\n\nSome details about the port are given here: https://kazakov.life/2017/11/01/cryptocurrency-mining-on-ios-devices/\n\n# How to build\n0) Prerequisites: Xcode and CocoaPods are installed.\n1) Get the source code and its dependencies:\n```shell\ngit clone --recursive https://github.com/mikekazakov/xmr-stak-cpu-ios\ncd xmr-stak-cpu-ios\npod install\n```\n2) Open xmr-stak-cpu-ios.xcworkspace and set the proper signing certificate.\n3) Adjust config.txt settings according to your pool and wallet information.\n4) Build the app and run it on the device.  \n**Ensure you're using the Release configuration, not the Debug!**  \nTo check it, open in Xcode: Product->Scheme->Edit Scheme...->Run->Info->Build Configuration.\n"
