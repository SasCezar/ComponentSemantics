b"A number of bash scripts simplifying the installation of ccminer and it's dependencies on Ubuntu 16.04. with NVIDIA GPU-powered hardware\n\nThis has been tested on Microsoft Azure VMs of the NV-Class, but should work on all systems with \n- Ubuntu 16.04\n- NVIDIA GPUs\n\nYou need root permissions and the machine will have to be restarted twice\n\n### To Install\n\n1. SSH into your machine\n2. become root (sudo su)\n3. clone this repository\n4. execute the bash scripts in ./steps in the correct order"
