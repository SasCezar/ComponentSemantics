To run in\nforeground:\n\n    ./bin/wownerod\n\nTo list all available options, run `./bin/wownerod --help`.  Options can be\nspecified either on the command line or in a configuration file passed by the\n`--config-file` argument.  To specify an option in the configuration file, add\na line with the syntax `argumentname=value`, where `argumentname` is the name\nof the argument without the leading dashes, for example `log-level=1`.\n\nTo run in background:\n\n    ./bin/wownerod --log-file wownerod.log --detach\n\nTo run as a systemd service, copy\n[wownerod.service](utils/systemd/wownerod.service) to `/etc/systemd/system/` and\n[wownerod.conf](utils/conf/wownerod.conf) to `/etc/`. The [example\nservice](utils/systemd/wownerod.service) assumes that the user `wownero` exists\nand its home is the data directory specified in the [example\nconfig](utils/conf/wownerod.conf).\n\nOnce node is synced to network, run the CLI wallet by entering:\n\n    ./bin/wownero-wallet-cli\n\nType `help` in CLI wallet to see standard commands (for advanced options, type `help_advanced`).\n\n## Tor Anonymity Network\n\n* Install [Tor Browser](https://www.torproject.org/download/)\n* Open `torrc` file in a text editor ([installation directory]/Browser/TorBrowser/Data/Tor/torrc) and add hidden service information as follows:\n\n```\nHiddenServiceDir [installation directory]/Browser/TorBrowser/Data/Tor/wow_node\nHiddenServiceVersion 3\nHiddenServicePort 44568 127.0.0.1:44568\n```\n* Save `torrc` file and restart Tor Browser (keep open)\n* Change directory to the `wow_node` folder, open `hostname` file, and copy your node\'s ".onion" address\n* Start wownerod with the following parameters:\n\n```\n./wownerod --tx-proxy tor,127.0.0.1:9150,10 --add-peer hdps3qwnusz64r7odvynmae6myc2uyvrsc2emap6636qeuzll72eouid.onion:44568 --anonymous-inbound YOUR_NODE_ADDRESS.onion:44568,127.0.0.1:44568,25\n```\n\n### Access remote Tor node from CLI wallet\n\n```\n./wownero-wallet-cli --proxy 127.0.0.1:9150 --daemon-address wow7dhbgiljnkspkzpjyy66auegbrye2ptfv4gucgbhireg5rrjza5ad.onion:34568\n```\n\nUse port `9050` instead of `9150` if you installed Tor as a standalone daemon.
