b'[![Travis CI Build Status](https://travis-ci.org/coinmetrics-io/haskell-tools.svg?branch=master)](https://travis-ci.org/coinmetrics-io/haskell-tools) [![Docker Repository on Quay](https://quay.io/repository/coinmetrics/haskell-tools/status "Docker Repository on Quay")](https://quay.io/repository/coinmetrics/haskell-tools)\n\n# Haskell-based CoinMetrics.io tools\n\nThese tools are used by CoinMetrics.io team for exporting data from blockchains into analytical databases and monitoring full nodes synchronization state.\n\n## Utilities\n\n* [coinmetrics-export](coinmetrics-export/README.md) - utility for exporting data from blockchains in formats suitable for inserting into analytics databases (SQL, Avro).\n* [coinmetrics-monitor](coinmetrics-monitor/README.md) - utility for monitoring blockchain nodes and providing Prometheus-compatible metrics.\n\n## Status\n\nThe project is being used in production at Coin Metrics, but a lot of things are in flux or fragile. Command line interface is more or less stable but may change. Please use with caution.\n\nSupported blockchains:\n\n* Binance Chain\n* [Bitcoin](https://bitcoin.org/)\n* [Cardano](https://www.cardanohub.org/)\n* [Cosmos](https://cosmos.network/)\n* [EOS](https://eos.io/)\n* [Ethereum](https://www.ethereum.org/)\n* [Grin](https://grin-tech.org/)\n* [IOTA](https://iota.org/)\n* [Monero](https://getmonero.org/)\n* [NEM](https://nem.io/)\n* [NEO](https://neo.org/)\n* [Ripple](https://ripple.com/)\n* [Stellar](https://www.stellar.org/)\n* Generic [Tendermint](https://tendermint.com/)\n* [Tezos](https://tezos.com/)\n* [Tron](https://tron.network/) (WIP)\n* [Waves](https://wavesplatform.com/)\n\n## Binaries (experimental)\n\nThere\'re no stable releases yet. All binaries are "bleeding edge" ones built on Travis CI.\n\nOne easy way to run the tools is to use docker.\n\nPull the latest version:\n```bash\ndocker pull quay.io/coinmetrics/haskell-tools\n```\n\nRun e.g. `coinmetrics-export` tool:\n```bash\ndocker run -it --rm --net host quay.io/coinmetrics/haskell-tools coinmetrics-export <arguments>\n```\n\nAlternatively you can download executables:\n\n* [generic Linux](https://bintray.com/coinmetrics/haskell-tools)\n\nOr packages:\n\n* [.deb packages](https://bintray.com/coinmetrics/haskell-tools-deb)\n* [.rpm packages](https://bintray.com/coinmetrics/haskell-tools-rpm)\n\nOr setup package repository to receive updates:\n\nFor .deb:\n```bash\necho \'deb https://dl.bintray.com/coinmetrics/haskell-tools-deb unstable main\' > /etc/apt/sources.list.d/coinmetrics-haskell-tools.list\ncurl \'https://bintray.com/user/downloadSubjectPublicKey?username=bintray\' | apt-key add -\napt update\napt install coinmetrics-export\n```\n\nFor .rpm:\n```bash\n(cd /etc/yum.repos.d/ && curl -JOL https://bintray.com/coinmetrics/haskell-tools-rpm/rpm)\nyum install coinmetrics-export\n```\n\nPackages have the advantage of providing bash completion support out of the box.\n\n## Building from source\n\nGet [stack](https://docs.haskellstack.org/en/stable/install_and_upgrade/).\n\nRun `stack build --install-ghc --copy-bins --local-bin-path <path for executables>`. Executables will be built and placed by specified path.\n\nThe code is only tested on Linux (but maybe works on other OSes too).\n'
