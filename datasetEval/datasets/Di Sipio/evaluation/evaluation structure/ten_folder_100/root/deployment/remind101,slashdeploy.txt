b'# SlashDeploy [![Build Status](https://travis-ci.org/remind101/slashdeploy.svg?branch=master)](https://travis-ci.org/remind101/slashdeploy) [![Code Climate](https://codeclimate.com/github/remind101/slashdeploy/badges/gpa.svg)](https://codeclimate.com/github/remind101/slashdeploy)\n\n[SlashDeploy](https://slashdeploy.io) is a web app for triggering [GitHub Deployments](https://developer.github.com/v3/repos/deployments/) via a `/deploy` command in Slack.\n\n## Installation\n\nSlashDeploy is already hosted at https://slashdeploy.io. All you have to do is add it to your Slack team:\n\n<a href="https://slashdeploy.io/slack/install"><img alt="Add to Slack" height="40" width="139" src="https://platform.slack-edge.com/img/add_to_slack@2x.png"></a>\n\n## Usage\n\nDeploy a repository to the default environment (production):\n\n```console\n/deploy ejholmes/acme-inc\n```\n\nDeploy a repository to a specific environment:\n\n```console\n/deploy ejholmes/acme-inc to staging\n```\n\nDeploy a branch:\n\n```console\n/deploy ejholmes/acme-inc@topic-branch to staging\n```\n\nAnd more at <https://slashdeploy.io/docs>.\n\n## Features\n\n* Create GitHub Deployments directly from Slack.\n* Receive Slack DM\'s whenever GitHub Deployments change status.\n* Trigger GitHub Deployments when a set of commit statuses pass (Continuous Delivery).\n* Environment locking.\n\n## Contributing\n\nContributions are highly welcome! If you\'d like to contribute, please read [CONTRIBUTING.md](./CONTRIBUTING.md)\n'
