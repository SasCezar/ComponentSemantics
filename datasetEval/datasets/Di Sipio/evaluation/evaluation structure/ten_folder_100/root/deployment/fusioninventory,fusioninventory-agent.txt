b'# FusionInventory Agent\n\n[![Travis Build Status](https://travis-ci.org/fusioninventory/fusioninventory-agent.svg?branch=develop)](https://travis-ci.org/fusioninventory/fusioninventory-agent)\n[![Appveyor Build status](https://ci.appveyor.com/api/projects/status/f2oh6p3qnr2bck1b?svg=true)](https://ci.appveyor.com/project/fusioninventory/fusioninventory-agent)\n[![CircleCI Build status](https://circleci.com/gh/fusioninventory/fusioninventory-agent.svg?style=svg)](https://circleci.com/gh/fusioninventory/fusioninventory-agent)\n\n## Summary\n\nThe FusionInventory agent is a generic management agent. It can perform a\ncertain number of tasks, according to its own execution plan, or on behalf of a\nGLPI server with fusioninventory plugin, acting as a control point.\n\n## Description\n\nSee [FusionInventory solution overview](http://fusioninventory.org/overview/)\n\n## Dependencies\n\n### Core\n\nMinimum perl version: 5.8\n\nMandatory Perl modules:\n\n* File::Which\n* LWP::UserAgent\n* Net::IP\n* Text::Template\n* UNIVERSAL::require\n* XML::TreePP\n\nOptional Perl modules:\n\n* Compress::Zlib, for message compression\n* HTTP::Daemon, for web interface\n* IO::Socket::SSL, for HTTPS support\n* LWP::Protocol::https, for HTTPS support\n* Proc::Daemon, for daemon mode (Unix only)\n* Proc::PID::File, for daemon mode (Unix only)\n\n### Inventory task\n\nOptional Perl modules:\n\n* Net::CUPS, for printers detection\n* Parse::EDID, for EDID data parsing\n* DateTime, for reliable timezone name extraction\n\nOptional programs:\n\n* dmidecode, for DMI data retrieval\n* lspci, for PCI bus scanning\n* hdparm, for additional disk drive info retrieval\n* monitor-get-edid-using-vbe, monitor-get-edid or get-edid, for EDID data access\n* ssh-keyscan, for host SSH public key retrieval\n\n### Network discovery tasks\n\nMandatory Perl modules:\n\n* Thread::Queue\n\nOptional Perl modules:\n\n* Net::NBName, for NetBios method support\n* Net::SNMP, for SNMP method Archive::Extract\n* Digest::SHA\n* File::Copy::Recursive\n* JSON::PP\n* URI::Escape\n\nMandatory Perl modules for P2P Support:\n* Net::Ping\n* Parallel::ForkManager\n\n## Related contribs\n\nSee
