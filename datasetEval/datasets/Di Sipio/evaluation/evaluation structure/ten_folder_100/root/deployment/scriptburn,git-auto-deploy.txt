b'# git-auto-deploy\n\n\n## Deploy your GitHub, GitLab or Bitbucket projects automatically on Git push events or web hooks\nInstallation\n--------\n\n\n\n*  With Composer `composer create-project --no-interaction scriptburn/git-auto-deploy my-app`\n* Or simply download and unzip from github\n* Suggest you to create a subdomain like **deploy.example.com** \n* open app\\settings.php and update **db, git_path, composer_path** according to your server\n![Settings](https://github.com/scriptburn/git-auto-deploy/raw/master/img/settings.png "Settings")\n\n* Make sure to add webserver user\'s public ssh keys in **GitHub, BitBucket, GitLab**\n* Login to your ssh as the same user as your webserver process is running and issue these commands And accept \'Yes\' after you have added your public ssh keys to **GitHub, BitBucket, GitLab**\n`$ ssh git@github.com`\n`$ ssh git@bitbucket.com`\n`$ ssh git@gitlab.com`\n\n\n\n* $ pbcopy < ~/.ssh/id_rsa.pub\n* Put  **http://< your-domain >/webhook** as WebHook url in appropriate repository\n* Default login and password is **admin** and **admin**\n* Now Create a new project and appropriate details and you are ready to go \n* After you push the changes to your remote repo deployment process in your server will start and if you entered your email in project form you will get detailed deployment log or you can check last deployment log any time by visting project list page  \n\nPre/Post Hook script\n--------------------\nYou can check `sample-post-hook.php` for sample code used in a post hook script\n\n### Login Page\n![Login page](https://github.com/scriptburn/git-auto-deploy/raw/master/img/login-form.png "Login page")\n\n### Projects list page:\n* Search By project name,status and repository type (GitHub ,BitBucket, GitLab)\n![Projects list page](https://github.com/scriptburn/git-auto-deploy/raw/master/img/project-list.png "Projects list page")\n\n\n\n### Projects Form:\n* Options to add custom pre and post deployment script.\n* Option to send detailed Deployment status report to provided email\n* Mark a project active or inactive\n* Run composer update after successfull deployment\n ![Projects Form](https://github.com/scriptburn/git-auto-deploy/raw/master/img/project-form.png "Projects Form")\n\n### Deployment log\n* View Detailed log of last failed or sucessfull deployment \n![Deployment log](https://github.com/scriptburn/git-auto-deploy/raw/master/img/deployment-log.png "Deployment log")\n\n### Update Profile\n* Change your email and password\n![Update Profile](https://github.com/scriptburn/git-auto-deploy/raw/master/img/edit-profile.png "Update Profile")\n\n### Email setting form\n* You can choose how deployment email will be sent , either by native php command or by any external smtp server\n![Email setting form](https://github.com/scriptburn/git-auto-deploy/raw/master/img/setting-form.png "Email setting form")'
