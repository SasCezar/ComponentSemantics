b'# \xf0\x9f\x9a\x80 Laravel Deployer\n\nLaravel Deployer is a lightweight wrapper of [Deployer.org](https://github.com/deployphp/deployer) giving Artisan the power of zero-downtime deployment.\n\n![Console showing php artisan deploy](https://user-images.githubusercontent.com/3642397/38672390-50ad0194-3e4e-11e8-93c2-d28de8659117.png)\n\n<p align="center">\n  <a href="docs/README.md"><img src="https://user-images.githubusercontent.com/3642397/38672391-50caf9e2-3e4e-11e8-862f-465d55e7e8d9.png" alt="Documentation button" height="50"></a>   \n  <a href="https://www.youtube.com/playlist?list=PLP7iaQb3O2XsexM_5HMrcKNCu0IOcxIDh"><img src="https://user-images.githubusercontent.com/3642397/39360000-a2f5d668-4a1c-11e8-8869-fa7fa027fe96.png" alt="Video tutorials button" height="50"></a>\n</p>\n\n## \xe2\x9c\xa8 Features\n* **Simple** setup process\n* **Zero downtime** deployments\n* Ready-to-use tasks for Laravel\n* Choose your **deployment strategy**\n* Agentless, it\'s just SSH\n* Something went wrong? **Rollback** to the previous release\n\n## 1\xef\xb8\x8f\xe2\x83\xa3 Installation\n\n```bash\ncomposer require lorisleiva/laravel-deployer\n```\n\n## 2\xef\xb8\x8f\xe2\x83\xa3 Configuration\nIn order to generate your deployment configuration file, simply run:\n\n```bash\nphp artisan deploy:init\n```\n\n* It will ask you a few questions to help you get started and generate a `config/deploy.php` file.\n* Read more about the available options, tasks, strategies; about how to customize your hosts, your deployment flow and much more in the [documentation](docs/README.md).\n\n## 3\xef\xb8\x8f\xe2\x83\xa3 Deployment\nWhen you\xe2\x80\x99re ready to deploy, run:\n\n```bash\nphp artisan deploy\n```\n\n* If anything goes wrong during the deployment flow, the release will be discarded just like nothing happened.\n* Because we are using zero-downtime deployments, make sure your server\xe2\x80\x99s root path points to the `{{deploy_path}}/current` symlink.\n* If your project has already been deployed, before using Laravel Deployer, you might be interested in this [first deployment strategy](docs/strategy-first-deploy.md).\n\n## \xf0\x9f\x93\x9c Available commands\n\n```bash\ndeploy                # Deploy your application\ndeploy:configs        # Print host configuration\ndeploy:current        # Show current paths\ndeploy:dump <task>    # Display the task-tree for a given task\ndeploy:hosts          # Print all hosts\ndeploy:init           # Generate a deploy.php configuration file\ndeploy:list           # Lists available tasks\ndeploy:rollback       # Rollback to previous release\ndeploy:run <task>     # Execute a given task on your hosts\nlogs                  # Dump the remote logs of your application\nssh                   # Connect to host through ssh\n```\n'
