(development)\n\n1. Install VueFront [CMS Connect App](https://vuefront.com/cms/) on your site and copy the CMS Connect URL.\n2. Install VueFront app. (requires node.js >= 8, git, and yarn)\n\n```bash\n# Create VueFront app. replace <project-name> with vuefront\nyarn create vuefront-app <project-name>\n# OR npx create-vuefront-app <project-name>\n\nyarn dev\n```\n\n## Switch to production\n1. build your App\n```bash\n# build the app\nyarn build\n```\n\n2. Copy the contents of your app from `/dist`  to your root folder of your CMS where it is hosted.\n\n3. Configure your hosting to load `index.html` first. This can be a bit tricky. \n\nFor OpenCart CMS you can use this:\n\n- Apache\n\n```apache\n# for VueFront to work you need to load index.html before any other index file\nDirectoryIndex index.html index.php\n\n```\n\n- Nginx\n```nginx\n# for VueFront to work you need to load index.html before any other index file\nindex index.html index.php;\n\n# when visiting any other url, it should forward to the root index.html file\nlocation / {\n    try_files $uri $uri/ /index.html;\n}\n```\n'
