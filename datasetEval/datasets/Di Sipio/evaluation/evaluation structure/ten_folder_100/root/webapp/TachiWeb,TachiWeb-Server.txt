</p>\n</details>\n\n<br>\n\n**Java 8 or newer is required to run the application.**\n\n## About\nTachiWeb-Server provides a consistent and flexible API for programs using the Tachiyomi backend.\nTachiWeb-Server allows multiple programs to use the same Tachiyomi library at the same time and provides a foundation for alternative frontends to build on.\n\nCurrently, this project comes bundled with a [React frontend](https://github.com/Ligerx/TachiWeb-React) that is enabled by default and a bootstrapper to integrate the application with your desktop.\n\nTachiWeb is not officially supported by Tachiyomi or it\'s contributors.\n\n## Motivation\nThere is currently no Manga reader that can be easily synced between the desktop and mobile.\nTachiWeb-Server is fully compatible with Tachiyomi\'s data structures and backups, allowing easy migration between the two.\n**Seamless sync between the two programs is a goal and planned feature.**\n\n## Server setup\nTachiWeb can also be installed on servers. Refer to [INSTALL.md](https://github.com/TachiWeb/TachiWeb-Server/blob/master/INSTALL.md) for the server installation procedure.\n\n# State\nTachiWeb-Server usable, but still **alpha** software.\nThe API is not stable and can change at any time.\nDetails on the state of the project can be found here: [https://github.com/TachiWeb/TachiWeb-Server/projects](https://github.com/TachiWeb/TachiWeb-Server/projects).\n\n# License\n```\nCopyright 2019 Andy Bao and contributors\n\nLicensed under the Apache License, Version 2.0 (the "License");\nyou may not use this file except in compliance with the License.\nYou may obtain a copy of the License at\n\nhttp://www.apache.org/licenses/LICENSE-2.0\n\nUnless required by applicable law or agreed to in writing, software\ndistributed under the License is distributed on an "AS IS" BASIS,\nWITHOUT WARRANTIES OR CONDITIONS OF ANY KIND, either express or implied.\nSee the License for the specific language governing permissions and\nlimitations under the License.\n```\n'
