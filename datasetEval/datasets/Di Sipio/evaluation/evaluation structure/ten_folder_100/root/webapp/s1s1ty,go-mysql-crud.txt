b'# go-mysql-crud\nSample crud operation using Golang and MySql\n\n\n## API ENDPOINTS\n\n### All Posts\n- Path : `/posts`\n- Method: `GET`\n- Response: `200`\n\n### Create Post\n- Path : `/posts`\n- Method: `POST`\n- Fields: `title, content`\n- Response: `201`\n\n### Details a Post\n- Path : `/posts/{id}`\n- Method: `GET`\n- Response: `200`\n\n### Update Post\n- Path : `/posts/{id}`\n- Method: `PUT`\n- Fields: `title, content`\n- Response: `200`\n\n### Delete Post\n- Path : `/posts/{id}`\n- Method: `DELETE`\n- Response: `204`\n\n## Required Packages\n- Dependency management\n    * [dep](https://github.com/golang/dep)\n- Database\n    * [MySql](https://github.com/go-sql-driver/mysql)\n- Routing\n    * [chi](https://github.com/go-chi/chi)\n\n## Quick Run Project\nFirst clone the repo then go to go-mysql-crud folder. After that build your image and run by docker. Make sure you have docker in your machine. \n\n```\ngit clone https://github.com/s1s1ty/go-mysql-crud.git\n\ncd go-mysql-crud\n\nchmod +x run.sh\n./run.sh\n\ndocker-compose up\n```\n\n'
