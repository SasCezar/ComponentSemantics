[MolochDAO](https://github.com/MolochVentures/moloch) compatible contracts.\n    * Vote proposals.\n    * Ragequit shares.\n    * Fork contract.\n  - [X] Support [ERC20](https://github.com/ethereum/eips/issues/20) and [ERC721](https://github.com/ethereum/eips/issues/721) tokens.    \n  - [X] Built for EVM-compatible blockchains using [Web3](https://github.com/ethereum/web3.js/).\n  - [X] Mobile & desktop responsive UX.\n  - [X] Templates and customizable settings.\n  \n\n**Release:**\n\n* Currently [`version 0.7.0`](https://github.com/DemocracyEarth/sovereign/releases)\n* This [User Agreement](https://github.com/DemocracyEarth/sovereign/blob/master/UserAgreement.md) governs access to and use of the Democracy Earth platform.\n\n## Setup\n\n1. **Install Framework**\n\n    On Linux & MacOS, load a terminal and type:\n\n    ```sh\n    $ curl https://install.meteor.com/ | sh\n    ```\n\n    This will setup [Meteor](https://github.com/meteor/meteor) (including [Node](https://github.com/nodejs/node) and [Mongo](https://github.com/mongodb/mongo) if necessary).\n\n    > _Note:_ Windows users must [download installer](https://www.meteor.com/install).\n\n2. **Clone Repository**\n\n    ```sh\n    $ git clone https://github.com/DemocracyEarth/sovereign.git\n    $ cd sovereign\n    ```\n\n3. **Install Dependencies**\n\n    if you have npm installed, type:\n\n    ```sh\n    $ npm install\n    ```\n\n    if you only have meteor, type:\n\n    ```sh\n    $ meteor npm install\n    ```\n\n4. **Configure**\n\n    * On `/config/development/settings.json` you can configure Sovereign for your organization\'s governance needs.\n        * Make sure you configure the keys on `public.web3` to connect to an Ethereum node ([Infura](https://infura.io) API keys are supported).\n        * For production, you can use `/config/production/settings.json` file and run `$ meteor --settings=config/production/settings.json` in the console.\n    * Seed DAO settings can be found on `/private/dao.json` which consists of an array of DAOs to be read from the blockchain and persisted in the server DB. \n        * [MolochDAO](https://molochdao.com) settings included as an example by default.\n    * Make sure `ROOT_URL` on `settings.json` is properly set to the domain that will be serving the application. \n        * We recommend the use of `https` **always** when deployed on a server.\n    * On `/public/templates/` different template style folders are available. \n        * The file `templateName.json` can be configured and a css design can be found on `/public/templates/name/css` folder.\n\n\n5. **Run Dapp**\n\n    While in repository directory type:\n\n    ```sh\n    $ meteor npm run start:dev\n    ```\n\n    Load browser and go to [http://localhost:3000/](http://localhost:3000/).\n\n    > _Note:_ For production deploys you should use:\n    > ``` $ meteor npm run start ```\n    > check production/settings.json file to config your keys.\n\n\n## Live\n\n* Democracy Earth: [democracy.earth](https://democracy.earth).\n* Moloch:
