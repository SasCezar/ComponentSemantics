double check for typos and cite any relevant sources in the comments.\n\nIf updating code examples:\n\n- make sure to update both the code in the markdown files as well as the code in the [code](code/) folder.\n\nIf wanting to add a new translation, follow these instructions:\n\n1. Set up [development environment](#development)\n\n2. Add language to `LANGS.md`\n\n3. Copy the the `en` directory and rename it with the 2 letter language code of the language you\'re translating to (e.g. `zh`)\n\n4. Translate content\n\n5. Set `"root"` to `"./"` in `book.json`\n\n## Thanks\n\nThanks to [qbig](https://github.com/qbig) and [gzuhlwang](https://github.com/gzuhlwang) for the Chinese translation.\n\nAnd thanks to all the [contributors](https://github.com/miguelmota/ethereum-development-with-go-book/graphs/contributors) who have contributed to this guide book.\n\n## License\n\n[CC0-1.0](./LICENSE.md)\n'
