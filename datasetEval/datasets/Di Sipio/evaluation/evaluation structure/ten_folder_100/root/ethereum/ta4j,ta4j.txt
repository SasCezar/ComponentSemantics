b'# ta4j [![Build Status](https://travis-ci.org/ta4j/ta4j.svg?branch=master)](https://travis-ci.org/ta4j/ta4j) [![Chat on Riot.im](https://img.shields.io/badge/chat-riot.im-green.svg)](https://riot.im/app/#/room/#ta4j:matrix.org)\r\n\r\n***Technical Analysis For Java***\r\n\r\n![Ta4 main chart](https://raw.githubusercontent.com/wiki/mdeverdelhan/ta4j/img/ta4j_main_chart.png)\r\n\r\nTa4j is an open source Java library for [technical analysis](http://en.wikipedia.org/wiki/Technical_analysis). It provides the basic components for creation, evaluation and execution of trading strategies.\r\n\r\n---\r\n\r\n### Features\r\n\r\n * [x] 100% Pure Java - works on any Java Platform version 8 or later\r\n * [x] More than 130 technical indicators (Aroon, ATR, moving averages, parabolic SAR, RSI, etc.)\r\n * [x] A powerful engine for building custom trading strategies\r\n * [x] Utilities to run and compare strategies\r\n * [x] Minimal 3rd party dependencies\r\n * [x] Simple integration\r\n * [x] One more thing: it\'s MIT licensed\r\n\r\n### Maven configuration\r\n\r\nTa4j is available on [Maven Central](http://search.maven.org/#search). You just have to add the following dependency in your `pom.xml` file.\r\n\r\n```xml\r\n<dependency>\r\n  <groupId>org.ta4j</groupId>\r\n  <artifactId>ta4j-core</artifactId>\r\n  <version>0.13</version>\r\n</dependency>\r\n```\r\n\r\nFor ***snapshots***, add the following repository to your `pom.xml` file.\r\n```xml\r\n<repository>\r\n    <id>sonatype snapshots</id>\r\n    <url>https://oss.sonatype.org/content/repositories/snapshots</url>\r\n</repository>\r\n```\r\nThe current ***snapshot version*** is `0.13-SNAPSHOT` from the [develop](https://github.com/ta4j/ta4j/tree/develop) branch.\r\n```xml\r\n<dependency>\r\n  <groupId>org.ta4j</groupId>\r\n  <artifactId>ta4j-core</artifactId>\r\n  <version>0.14-SNAPSHOT</version>\r\n</dependency>\r\n```\r\n\r\nYou can also download ***example code*** from the maven central repository by adding the following dependency to your pom.xml:\r\n```xml\r\n<dependency>\r\n  <groupId>org.ta4j</groupId>\r\n  <artifactId>ta4j-examples</artifactId>\r\n  <version>0.13</version>\r\n</dependency>\r\n```\r\n### Getting Help\r\nThe [wiki](https://ta4j.github.io/ta4j-wiki/) is the best place to start learning about ta4j.
