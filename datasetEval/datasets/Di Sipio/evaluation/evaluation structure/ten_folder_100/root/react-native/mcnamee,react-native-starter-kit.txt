with a [React web app](https://reactjs.org/) and [React Native app](https://facebook.github.io/react-native/) sharing a single code base. It shares the \'business logic\' (_i.e. models, containers_) across the platforms, whilst allowing flexibility in View components to ensure your project looks and feels native in each platform.\n\nThe project is _super_ helpful to kick-start your next project, as it provides a lot of the common tools you may reach for, all ready to go. Specifically:\n\n- A shared React and React Native structure\n- __Flux architecture__\n    - [Redux](https://redux.js.org/docs/introduction/)\n    - Redux Wrapper: [Rematch](https://github.com/rematch/rematch)\n- __Routing and navigation__\n    - [React Native Router Flux](https://github.com/aksonov/react-native-router-flux) for native mobile\n    - [React Router](https://github.com/ReactTraining/react-router) for web\n- __Data Caching / Offline__\n    - [Redux Persist](https://github.com/rt2zz/redux-persist)\n- __UI Toolkit/s__\n    - [Native Base](https://nativebase.io/) for native mobile\n    - [Bootstrap](https://getbootstrap.com/) for web\n- __Simpler mobile app development__ through\n    - [Expo](https://expo.io/)\n- __User authentication__ example through\n    - [Firebase](https://firebase.google.com/)\n- __API/Data example__\n    - Shows how to read/write data from/to an external API (in our case, [Firebase](https://firebase.google.com/))\n- __Code Linting__ with\n    - [Airbnb\'s JS Linting](https://github.com/airbnb/javascript) guidelines\n\n---\n\n## \xf0\x9f\x93\x96 Docs\n\n- [Setup your own Firebase](/docs/firebase.md)\n- [Understanding the file structure](/docs/file-structure.md)\n- [FAQs & Opinions](/docs/faqs.md)\n- [Testing, Deploying & Publishing](/docs/publishing.md)\n- [Tests & testing](/docs/testing.md) (coming soon...)\n- [Contributing to this project](/docs/contributing.md)\n\n---\n\n## \xf0\x9f\x9a\x80 Getting Started\n\n#### 0. Prerequisites\nMake sure you have Expo CLI installed\n```bash\nnpm install -g expo-cli\n```\n\n#### 1. Clone and Install\n\n_*It\'s recommended that you install [React Native Debugger](https://github.com/jhen0409/react-native-debugger/releases) and open before `yarn start`._\n\n```bash\n# Clone the repo\ngit clone https://github.com/mcnamee/react-native-starter-kit.git\n\n# Install dependencies\nyarn install\n```\n\n#### 2.1. Run the _React Native_ App\n\n```bash\n# Start the React Native packager\nyarn start\n```\n\nInstructions are shown in the terminal. You can select to open it in:\n\n- An emulator (either iOS or Android)\n- Your mobile device with the [Expo app](https://expo.io/). It will reload if you save edits to your files and you will see build errors and logs in the terminal.\n\n#### 2.2. Run the _Web_ App\n\n```bash\n# Starts are local live-reload server at:\n# http://localhost:3001\nyarn run web\n```\n\nVia webpack, starts a localhost server on port 3001 [http://localhost:3001](http://localhost:3001).\n\n- Save code and it auto refreshes\n- Install [Redux DevTools](https://chrome.google.com/webstore/detail/redux-devtools/lmhkpmbekcpmknklioeibfkpmmfibljd?hl=en) into Chrome to see the state of Redux\n\n---\n\n## \xf0\x9f\x91\x8a Further Help?\n\nThis repo is a great place to start, but if you\'d prefer to sit back and have your new project built for you, [get in touch with me directly](https://mcnam.ee) and I can organise a quote.\n\n---\n\n## \xf0\x9f\x92\xa1 Inspiration\n\n- [This blog post](http://jkaufman.io/react-web-native-codesharing/) helped put the ideas into action\n\n_If there\'s any other ideas presented in this repo, that you think worth mentioning - feel free open a pull request :)_\n'
