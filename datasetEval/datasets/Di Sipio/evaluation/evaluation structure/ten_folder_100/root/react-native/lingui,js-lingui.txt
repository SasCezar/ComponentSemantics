b'<div align="center">\n<h1>Lingui<sub>js</sub></h1>\n\n\xf0\x9f\x8c\x8d\xf0\x9f\x93\x96 A readable, automated, and optimized (5 kb) internationalization for JavaScript\n\n<hr />\n\n[![CircleCI][Badge-CI]][CI]\n[![AppVeyor][Badge-AppVeyor]][AppVeyor]\n[![Code Coverage][Badge-Coverage]][Coverage]\n[![PRs Welcome][Badge-PRWelcome]][PRWelcome]\n[![Backers on Open Collective][Badge-ocbackers]][ocbackers-local]\n[![Sponsors on Open Collective][Badge-ocsponsors]][ocsponsors-local]\n[![Join the community on Spectrum](https://withspectrum.github.io/badge/badge.svg)][Spectrum]\n\n[**Documentation**][Documentation] \xc2\xb7 [**Quickstart**](#quickstart) \xc2\xb7 [**Example**](#example) \xc2\xb7 [**Support**](#support) \xc2\xb7 [**Contribute**](#contribute) \xc2\xb7 [**Licence**](#licence)\n</div>\n\n> Internationalization is the design and development of a product, application or document content that enables easy localization for target audiences that vary in culture, region, or language.\n>\n> --- [ W3C Web Internationalization FAQ](https://www.w3.org/International/questions/qa-i18n)\n\n\nLingui is an easy yet powerful internationalization framework for global projects.\n\n- **Clean and readable** - Keep your code clean and readable, while the library uses\n  battle-tested and powerful **ICU MessageFormat** under the hood.\n\n- **Universal** - Use it everywhere. `@lingui/core` provides the essential intl\n  functionality which works in any JavaScript project while `@lingui/react` offers\n  components to leverage React rendering.\n\n- **Full rich-text support** - Use React components inside localized messages\n  without any limitation. Writing rich-text messages is as easy as writing JSX.\n\n- **Powerful tooling** - Manage the whole intl workflow using Lingui [CLI][RefCLI]. It\n  extracts messages from source code, validates messages coming from translators and\n  checks that all messages are translated before shipping to production.\n\n- **Unopinionated** - Integrate Lingui into your existing workflow. It supports\n  message keys as well as auto generated messages. Translations are stored either in\n  JSON or standard PO file, which is supported in almost all translation tools.\n\n- **Lightweight and optimized** - Core library is only [1.9 kB gzipped][BundleCore],\n  React components are additional [3.1 kBs gzipped][BundleReact]. That\'s less than Redux\n  for a full-featured intl library.\n\n- **Active community** - Join us on [Spectrum][Spectrum] to discuss the latest development.\n  At the moment, Lingui is the most active intl project on GitHub.\n\n- **Compatible with react-intl** - Low-level React API is very similar to react-intl\n  and the message format is the same. It\'s easy to migrate an existing project.\n\n## Quickstart\n\n### Install\n\n- [Create React App][TutorialSetupCRA]\n- [React projects][TutorialSetupReact]\n\n### Tutorials\n\n- [React][TutorialReact]\n- [React Native][TutorialReactNative]\n- [Plain JavaScript][TutorialJavaScript]\n- [Working with Command Line Tool][TutorialCLI]\n\nIf you\'re a react-intl user, checkout\n[comparison of react-intl and Lingui](https://lingui.js.org/misc/react-intl.html).\n\n## Example\n\nShort example how i18n looks with JSX:\n\n```js\nimport { Trans } from "@lingui/macro"\n\nfunction App() {\n  return (\n   <Trans id="msg.docs" /* id is optional */>\n     Read the <a href="https://lingui.js.org">documentation</a>\n     for more info.\n   </Trans>\n  )\n}\n```\n\nMessage from this component will be extracted in following format:\n\n```po\nmsgid "msg.docs"\nmsgstr "Read the <0>documentation</0> for more info."\n```\n\nFor more example see the [React tutorial][TutorialReact].\n\n## Support\n\nIf you are having issues, please let us know.\n\n- Join us at [Gitter](https://gitter.im/lingui/js-lingui) to get almost instant feedback.\n- Ask question on [StackOverflow](https://stackoverflow.com/questions/ask?tags=jsLingui)\n  and mark it with [Lingui](https://stackoverflow.com/questions/tagged/jslingui) tag.\n- If something doesn\'t work as documented, documentation is missing or if you just want\n  to suggest a new feature, [create an issue][Issues].\n\n## Contribute\n\nContribution to open-source project is everything from spreading a word, writing\ndocumentation to implementing features and fixing bugs.\n\n- Do you use **Lingui** in production site?
