b'# macos-terminal-themes\n\n# Color schemes for default macOS Terminal.app\n\nThis is a set of color schemes for default macOS Terminal.app (ported from\n[iTerm 2 color schemes, collected by @mbadolato][1]). Screenshots are below and\nin the `screenshots/` directory of this repo.\n\n[1]: https://github.com/mbadolato/iTerm2-Color-Schemes\n\n_Note_: Some default macOS themes are not included here (default themes like\n"Pro", "Basic", "Grass", etc.), because you already have them :)\n\n\n## Pre-requireties\n\nSince terminal themes are just color schemes, you need to enable color formatting for your shell first - see [this comment][2] for more details\n\n[2]: https://github.com/lysyi3m/osx-terminal-themes/issues/1#issuecomment-148635036\n\n\n## Installation Instructions\n\n- Clone or download this repo\n\n- Go to `schemes/` folder\n\n- Double click on selected `*.terminal` file. It will open a new Terminal window with that\n  color scheme.\n\n- Set the scheme as the default one with `Shell -> Use Settings as Default`\n\n\n## Tools\n\n### Convert iTerm 2 Color Scheme\n\nThis repo contains a tool to convert any iTerm 2 color scheme into macOS\nTerminal scheme. To run just execute script:\n\n```\n./tools/iterm2terminal.swift /path/to/my/awesome-scheme.itermcolors\n```\n\n\n### Generate Theme Preview\n\nIf you have an awesome schemes and going to publish them, please generate\npreview image using the following steps.\n\n1. Load the theme in a terminal.\n2. Run\n   ```\n   bash tools/preview.sh\n   ```\n3. Set the font size to 14.\n4. Take a screenshot and put it in the `screenshots/` folder.\n5. Update the `README.md` to include your new screenshot.\n\n_Note_: preview will be generated using [ANSI Escape Sequences][3]\n\n[3]: http://tldp.org/HOWTO/Bash-Prompt-HOWTO/x329.html\n\n\n##
