b'Gogh\n====\n\n![elementary](https://raw.githubusercontent.com/Mayccoll/Gogh/master/images/gogh/Gogh-icons.png)\n\n## :small_orange_diamond: [Visit the Website](http://mayccoll.github.io/Gogh) :small_orange_diamond:\n\n## Color Scheme for Gnome Terminal, Pantheon Terminal, Tilix, and XFCE4 Terminal\n\nColor Schemes For Ubuntu, Linux Mint, Elementary OS and all distributions that use Gnome Terminal, Pantheon Terminal, Tilix, or XFCE4 Terminal; initially inspired by Elementary OS Luna. Also works on iTerm for macOS. You can check out the themes [here](https://mayccoll.github.io/Gogh/).\n\n![elementary](https://raw.githubusercontent.com/Mayccoll/Gogh/master/images/demos/themes.gif)\n\n----\n\n<br/>\n<br/>\n\n## Pre-Install\n\n```bash\n  $ sudo apt-get install dconf-cli uuid-runtime\n```\n\n## [Install](https://github.com/Mayccoll/Gogh/blob/master/content/install.md)\n\n**1. Just copy and paste this one-line command:**\n\n```bash\n  $  bash -c  "$(wget -qO- https://git.io/vQgMr)"\n```\n\nOr, if you are a Mac user:\n\n```bash\n  $  bash -c  "$(curl -sLo- https://git.io/vQgMr)"\n```\n\n**2. Select the color theme installed from the terminal**\n\n**DEMO:**\n\n![gogh-demo-profile](https://raw.githubusercontent.com/Mayccoll/Gogh/master/images/demos/gogh-demo-profile.gif)\n\n<br/>\n<br/>\n\n\n## [Themes](https://mayccoll.github.io/Gogh/)\n\nWe have lots of themes. Check them out [here](https://mayccoll.github.io/Gogh/)!\n\n\n<br/>\n<br/>\n\n\n\n## [How to](https://github.com/Mayccoll/Gogh/blob/master/content/howto.md)\n\nIf you want to create your own color
