b'![Extraterm logo](docs/extraterm_main_logo_512x367.png)\n\n*The swiss army chainsaw of terminal emulators*\n\nMain website: [extraterm.org](http://extraterm.org)\n\n[![Build Status](https://dev.azure.com/simon0627/extraterm/_apis/build/status/sedwards2009.extraterm?branchName=master)](https://dev.azure.com/simon0627/extraterm/_build/latest?definitionId=1&branchName=master)\n\n# About\n\nExtraterm is an open source project to build a terminal emulator and expand it with new features to support modern workflows and do more than just act like a glorified teletype.\n\nMore background information is in the [Why document](http://extraterm.org/why.html).\n\n* *Capture and reuse command output as new command input*\n* *Manipulate and copy text without leaving the keyboard*\n* *Slice and dice command output like a text editor*\n* *Display images directly*\n* *See the [Features list](http://extraterm.org/features.html) for more*\n\n![Directly edit and execute command output](docs/edit_direct.gif)\n\nSee the [Features and Screenshots](http://extraterm.org/features.html)\n\nThe primary developer is [Simon Edwards](mailto:simon@simonzone.com). Extraterm logos were designed and provided by [Gabriel Harel](https://github.com/g-harel).\n\n# Features\n\n* Supports Linux, macOS and Cygwin on Windows\n* xterm compatible\n* Multiple tabs and panes, split and tiled any way you like\n* Keyboard based selection\n* Show images in the terminal\n* Shell integration; can isolate and \'frame\' command output. (bash, zsh and fish supported)\n* Previous command output can be used as input for new commands\n* Command output is editable in place. Multi-cursor editing\n* Keyboard friendly "command palette" for searching menus\n* Text zoom\n* Unicode support\n* Color emoji\n* [MIT license](LICENSE.txt)\n* [...much more to come](https://github.com/sedwards2009/extraterm/issues/30)\n\nSee the more detailed [features list](http://extraterm.org/features.html).\n\n# News\n\n[News is here](http://extraterm.org/news.html)\n\n# Download\n\n[Download and try it out on Linux, Windows+Cygwin and macOS](https://github.com/sedwards2009/extraterm/releases)\n\n:warning: Extraterm is in development and roughly \'beta\' in quality. It is generally stable, but don\'t be too surprised if you encounter a few bugs. Use at your own risk. :warning:\n\nWith that out of the way, the primary author does use it as their main terminal emulator on a daily basis. The basics are in place and it will run most common terminal applications such as vi, emacs and joe without problems and even more demanding applications such as midnight commander.\n\nNightly builds of the `master` branch can be [downloaded here](https://storage.googleapis.com/extraterm_builds/index.html).\n\n# Further Information\n\n* :loudspeaker: :bird: Follow development on [Twitter @ ExtratermDev](https://twitter.com/ExtratermDev)\n* [User Guide](http://extraterm.org/guide.html)\n* [Downloads and Releases](https://github.com/sedwards2009/extraterm/releases) (Linux, Windows and OS X supported)\n* [FAQ](http://extraterm.org/faq.html)\n* [Development and Contributing](http://extraterm.org/development.html)\n* [Current Development Roadmap](https://github.com/sedwards2009/extraterm/issues/30)\n* :octocat: Github: https://github.com/sedwards2009/extraterm\n'
