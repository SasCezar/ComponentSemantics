It combines everything you may need for convenient work, without unnecessary complications, like a real spaceship.\n\n<p align="center">\n  <img alt="Spaceship with Hyper and One Dark" src="https://user-images.githubusercontent.com/10276208/36086434-5de52ace-0ff2-11e8-8299-c67f9ab4e9bd.gif" width="980px">\n</p>\n\n<sub>Vist <a href="./docs/Troubleshooting.md#why-doesnt-my-prompt-look-like-the-preview">Troubleshooting</a> for similar setup and find more examples with different color schemes in <a href="https://github.com/denysdovhan/spaceship-prompt/wiki/Screenshots">Screenshots </a>wiki-page.</sub>\n\n## Features\n\n- Clever hostname and username displaying.\n- Prompt character turns red if the last command exits with non-zero code.\n- Current Git branch and rich repo status:\n  - `?` \xe2\x80\x94 untracked changes;\n  - `+` \xe2\x80\x94 uncommitted changes in the index;\n  - `!` \xe2\x80\x94 unstaged changes;\n  - `\xc2\xbb` \xe2\x80\x94 renamed files;\n  - `\xe2\x9c\x98` \xe2\x80\x94 deleted files;\n  - `$` \xe2\x80\x94 stashed changes;\n  - `=` \xe2\x80\x94 unmerged changes;\n  - `\xe2\x87\xa1` \xe2\x80\x94 ahead of remote branch;\n  - `\xe2\x87\xa3` \xe2\x80\x94 behind of remote branch;\n  - `\xe2\x87\x95` \xe2\x80\x94 diverged changes.\n- Current Mercurial bookmark/branch and rich repo status:\n  - `?` \xe2\x80\x94 untracked changes;\n  - `+` \xe2\x80\x94 uncommitted changes in the index;\n  - `!` \xe2\x80\x94 unstaged changes;\n  - `\xe2\x9c\x98` \xe2\x80\x94 deleted files;\n- Indicator for jobs in the background (`\xe2\x9c\xa6`).\n- Current Node.js version, through nvm/nodenv/n (`\xe2\xac\xa2`).\n- Current Ruby version, through rvm/rbenv/chruby/asdf (`\xf0\x9f\x92\x8e`).\n- Current Elm version (`\xf0\x9f\x8c\xb3`)\n- Current Elixir version, through kiex/exenv/elixir (`\xf0\x9f\x92\xa7`).\n- Current Swift version, through swiftenv (`\xf0\x9f\x90\xa6`).\n- Current Xcode version, through xenv (`\xf0\x9f\x9b\xa0`).\n- Current Go version (`\xf0\x9f\x90\xb9`).\n- Current PHP version (`\xf0\x9f\x90\x98`).\n- Current Rust version (`\xf0\x9d\x97\xa5`).\n- Current version of Haskell GHC Compiler, defined in stack.yaml file (`\xce\xbb`).\n- Current Julia version (`\xe0\xae\x83`).\n- Current Docker version and connected machine (`\xf0\x9f\x90\xb3`).\n- Current Amazon Web Services (AWS) profile (`\xe2\x98\x81\xef\xb8\x8f`) ([Using named profiles](http://docs.aws.amazon.com/cli/latest/userguide/cli-multiple-profiles.html)).\n- Current Python virtualenv.\n- Current Conda virtualenv (`\xf0\x9f\x85\x92`).\n- Current Python pyenv (`\xf0\x9f\x90\x8d`).\n- Current .NET SDK version, through dotnet-cli (`.NET`).\n- Current Ember.js version, through ember-cli (`\xf0\x9f\x90\xb9`).\n- Current Kubectl context (`\xe2\x98\xb8\xef\xb8\x8f`).\n- Current Terraform workspace (`\xf0\x9f\x9b\xa0`).\n- Package version, if there\'s is a package in current directory (`\xf0\x9f\x93\xa6`).\n- Current battery level and status:\n  - `\xe2\x87\xa1` - charging;\n  - `\xe2\x87\xa3` - discharging;\n  - `\xe2\x80\xa2` - fully charged.\n- Current Vi-mode mode ([with handy aliases for temporarily enabling](./docs/Options.md#vi-mode-vi_mode)).\n- Optional exit-code of last command ([how to enable](./docs/Options.md#exit-code-exit_code)).\n- Optional time stamps 12/24hr in format ([how to enable](./docs/Options.md#time-time)).\n- Execution time of the last command if it exceeds the set threshold.\n\nWant more features? Please, [open an issue](https://github.com/denysdovhan/spaceship-prompt/issues/new/choose) or send pull request.\n\n**\xf0\x9f\x92\xa1 Tip:** Follow our Twitter to keep yourself updated about new features, improvements, and bugfixes.\n\n## Requirements\n\nTo work correctly, you will first need:\n\n- [`zsh`](http://www.zsh.org/) (v5.2 or recent) must be installed.\n- [Powerline Font](https://github.com/powerline/fonts) must be installed and used in your terminal (for example, switch font to [Fira Code](https://github.com/tonsky/FiraCode)).\n\n## Installing\n\nNow that the requirements are satisfied, you can install Spaceship ZSH via any of the following tools.\n\n**\xf0\x9f\x92\xa1 Tip:** If you like this project ~~and want to get some stickers and postcards~~, consider becoming a patron:\n\n<a href="https://patreon.com/denysdovhan">\n  <img alt="Become a patron" src="https://c5.patreon.com/external/logo/become_a_patron_button@2x.png" width="150px">\n</a>\n\n### [npm]\n\n```\nnpm install -g spaceship-prompt\n```\n\nDone. This command should link `spaceship.zsh` as `prompt_spaceship_setup` to your `$fpath` and set `prompt spaceship` in `.zshrc`. Just reload your terminal.\n\n**\xf0\x9f\x92\xa1 Tip:** Update Spaceship to new versions as you would any other package.\n\n### [oh-my-zsh]\n\nClone this repo:\n\n```zsh\ngit clone https://github.com/denysdovhan/spaceship-prompt.git "$ZSH_CUSTOM/themes/spaceship-prompt"\n```\n\nSymlink `spaceship.zsh-theme` to your oh-my-zsh custom themes directory:\n\n```zsh\nln -s "$ZSH_CUSTOM/themes/spaceship-prompt/spaceship.zsh-theme" "$ZSH_CUSTOM/themes/spaceship.zsh-theme"\n```\n\nSet `ZSH_THEME="spaceship"` in your `.zshrc`.\n\n### [prezto]\n\n- Follow [prezto-contrib#usage](https://github.com/belak/prezto-contrib#usage) to clone `prezto-contrib` to the proper location.\n- Enable the `contrib-prompt` module (before the `prompt` module).\n- Set `zstyle \':prezto:module:prompt\' theme \'spaceship\'` in your `.zpreztorc`.\n\n### [antigen]\n\nAdd the following snippet in your `~/.zshrc`:\n\n```\nantigen theme denysdovhan/spaceship-prompt\n```\n\n### [antibody]\n\nUpdate your `.zshrc` file with the following line:\n\n```\nantibody bundle denysdovhan/spaceship-prompt\n```\n\n### [zgen]\n\nAdd the following line to your `~/.zshrc` where you\'re adding your other Zsh plugins:\n\n```\nzgen load denysdovhan/spaceship-prompt spaceship\n```\n\n### [zplug]\n\nUse this command in your `.zshrc` to load Spaceship as prompt theme:\n\n```\nzplug "denysdovhan/spaceship-prompt", use:spaceship.zsh, from:github, as:theme\n```\n\n### Linux package manager\n\n#### Arch Linux\n\nInstall the latest master from the AUR package [`spaceship-prompt-git`](https://aur.archlinux.org/packages/spaceship-prompt-git/):\n\n```\ngit clone https://aur.archlinux.org/spaceship-prompt-git.git\ncd spaceship-prompt-git\nmakepkg -si\n```\n\n### Manual\n\nIf you have problems with approaches above, follow these instructions:\n\n- Clone this repo `git clone https://github.com/denysdovhan/spaceship-prompt.git`\n- Symlink `spaceship.zsh` to somewhere in [`$fpath`](http://www.refining-linux.org/archives/46/ZSH-Gem-12-Autoloading-functions/) as `prompt_spaceship_setup`.\n- Initialize prompt system and choose `spaceship`.\n\n#### Example\n\nRun `echo $fpath` to see possible location and link `spaceship.zsh` there, like:\n\n```zsh\n$ ln -sf "$PWD/spaceship.zsh" "/usr/local/share/zsh/site-functions/prompt_spaceship_setup"\n```\n\nFor a user-specific installation, simply add a directory to `$fpath` for that user in `.zshrc`:\n\n```zsh\nfpath=( "$HOME/.zfunctions" $fpath )\n```\n\nThen install the theme like this:\n\n```zsh\n$ ln -sf "$PWD/spaceship.zsh" "$HOME/.zfunctions/prompt_spaceship_setup"\n```\n\nFor initializing prompt system add this to your `.zshrc`:\n\n```zsh\n# .zshrc\nautoload -U promptinit; promptinit\nprompt spaceship\n```\n\n## Customization\n\nSpaceship works well out of the box, but you can customize almost everything if you want.\n\n- [**Options**](./docs/Options.md) \xe2\x80\x94 Tweak section\'s behavior with tons of options.\n- [**API**](./docs/API.md) \xe2\x80\x94 Define a custom section that will do exactly what you want.\n\nYou have ability to customize or disable specific elements of Spaceship. All options must be overridden in your `.zshrc` file **after** the theme.\n\n**\xf0\x9f\x92\xa1 Tip:** Take a look at popular option presets or share your own configuration on [Presets](https://github.com/denysdovhan/spaceship-prompt/wiki/Presets) wiki page.\n\n## Troubleshooting\n\nHaving trouble?
