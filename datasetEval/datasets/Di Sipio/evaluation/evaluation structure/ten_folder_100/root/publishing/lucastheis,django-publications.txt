b"django-publications\n===================\n\nA Django app for managing scientific publications.\n\n[![Build Status](https://travis-ci.org/lucastheis/django-publications.svg?branch=develop)](https://travis-ci.org/lucastheis/django-publications)\n[![Coverage Status](https://coveralls.io/repos/github/lucastheis/django-publications/badge.svg)](https://coveralls.io/github/lucastheis/django-publications)\n\nScreenshots\n-----------\n\n[![frontend][3]][1]\n[![backend][4]][2]\n\n[1]: https://raw.githubusercontent.com/lucastheis/django-publications/media/frontend.png\n[2]: https://raw.githubusercontent.com/lucastheis/django-publications/media/backend.png\n[3]: https://raw.githubusercontent.com/lucastheis/django-publications/media/frontend_small.png\n[4]: https://raw.githubusercontent.com/lucastheis/django-publications/media/backend_small.png\n\nFeatures\n--------\n\n* automatically creates lists for individual authors and keywords\n* BibTex import/export\n* RIS export (EndNote, Reference Manager)\n* unAPI support (Zotero)\n* customizable publication categories/BibTex entry types\n* PDF upload\n* RSS feeds\n* support for images\n\nRequirements\n------------\n\n* Python >= 3.5.0\n* Django >= 2.1.0\n* Pillow >= 2.3.0\n\nInstallation\n------------\n\n1) Run `pip install django-publications`.\n\n2) Add `'publications'` to `INSTALLED_APPS` in your project's `settings.py`.\n\n3) Add the following to your project's `urls.py`:\n\n\t```python\n\turl(r'^publications/', include('publications.urls')),\n\t```\n\n4) Run the following:\n\n\t```\n\t./manage.py makemigrations publications\n\t./manage.py migrate\n\t```\n"
