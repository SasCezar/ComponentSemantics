b'# Wisdom [![License][LicenseIMGURL]][LicenseURL] [![NPM version][NPMIMGURL]][NPMURL] [![Dependency Status][DependencyStatusIMGURL]][DependencyStatusURL] [![Build Status][BuildStatusIMGURL]][BuildStatusURL]\n\nTool for publishing releases to github and npm according to [Semantic Versionin](http://semver.org "Semantic Versioning").\n\nOne command do next things:\n- set env variable `$wisdom_version` and `$WISDOM_VERSION` with future version\n- run command from `scripts.wisdom` of `package.json` (if exist);\n- run command from `scripts.wisdom:type` of `package.json` (if exist);\n- [changelog](http://github.com/coderaiser/changelog-io "ChangeLog");\n- [version](http://github.com/coderaiser/version-io "Version") in `package.json`;\n- tag;\n- [release on github](https://github.com/coderaiser/node-grizzly "Grizzly");\n- push to github;\n- publish to npm;\n- run command from `scripts.wisdom:done` of `package.json` (if exist);\n\nBefore executing `wisdom`, `wisdom:type` and `wisdom:done` scripts will be expanded via [redrun](https://github.com/coderaiser/redrun) which will speed things up.\n\nHere is list of commands that should be executed to get same result:\n```sh\nchangelog {{ version }}\nversion {{ version }}\ngit add --all\ngit commit -m "feature(package) v{{ version }}"\ngit push origin {{ branch }}\ngit tag v{{ version }}\ngit push origin v{{ version }}\ngrizzly -tn "token from url" \\\n-r grizzly -o {{ owner }} -t {{ version }} \\\n-n "{{ repo }} {{ version }}" -b "changelog"\nnpm publish\n```\n\n## Install\n\n`npm i wisdom -g`\n\n## How to use?\n\n```\n$ wisdom\nUsage: wisdom [patch|minor|major]\nOptions:\n-h, --help     : display this help and exit\n-v, --version  : output version information and exit\n```\n\n## Configuration\n\nWhen you need configure `wisdom` you could declare them in `package.json`:\n\n```js\n{\n    "changelog": true,  /* default */\n    "tag": true,        /* default */\n    "release": true,    /* default */\n    "private": false,   /* default */\n    "branch": "master"  /* default */\n    "scripts": {\n        "wisdom": "echo \'do something before publish\'",\n        "wisdom:type": "echo \'do something before publish and add --patch, --minor or --major argument\'",\n        "wisdom:done": "echo \'do something after publish\'"\n    }\n}\n```\n\n##
