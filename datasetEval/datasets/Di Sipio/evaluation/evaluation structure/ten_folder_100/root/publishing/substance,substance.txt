b'# Substance [![Build Status](https://travis-ci.org/substance/substance.svg?branch=devel)](https://travis-ci.org/substance/substance)\n\nSubstance is a JavaScript library for web-based content editing. It provides building blocks for realizing custom text editors and web-based publishing system.\n\nCheck the [project website](http://substance.io).\n\n## Development\n\nInstall the dev dependencies.\n\n```\nnpm install\n```\n\nRun the dev server.\n\n```\nnpm start\n```\n\nNavigate to `http://localhost:4001/test` for the running the browser test suite. The test suite is rebuilt as you make changes to the source files.\n\nTo run the test-suite headless.\n\n```\n$ npm test\n```\n'
