b"# Apple News API client\n\nAPI wrapper module for Apple News API https://developer.apple.com/library/ios/documentation/General/Conceptual/News_API_Ref/index.html\n\nSupports creating, reading, updating, deleting, and searching articles.\nAlso supports reading and listing sections, as well as reading channels!\n\n**For Node versions < 4, use `apple-news@^1`. For Node versions >= 4 use `apple-news@^2`**\n\n## Install\n\n```\nnpm install apple-news --save\n```\n\n## Usage\n\n```js\nvar client = createClient({\n  apiId: '<API-ID>',\n  apiSecret: '<API-SECRET>'\n});\n```\n\n## Methods\n\n```\nclient.readChannel ({ channelId }, cb)\nclient.listSections ({ channelId }, cb)\nclient.readSection ({ sectionId }, cb)\nclient.createArticle ({ channelId, article, bundleFiles, isPreview = true }, cb)\nclient.readArticle ({ articleId }, cb)\nclient.updateArticle ({ articleId, revision, article, isPreview = true }, cb)\nclient.deleteArticle ({ articleId }, cb)\n```\n\n## License\n\nMIT\n"
