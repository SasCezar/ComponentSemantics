b'# [Firefox addon](https://addons.mozilla.org/en-US/firefox/addon/2read/)\n\n# [Chrome extension](https://chrome.google.com/webstore/detail/2read/nooeemkngkckhnlfbmikbkiiabbbnlap)\n\n### Features\n\n- Web article cleaning for readability\n- Upload to [IPFS](https://ipfs.io/)\n- Seeding the content to multiple gateways\n- Automatic "pinning"\n- Automatic bookmarking\n\n### Note\n\nContent shared with this addon is just cached on IPFS servers. If you want to pin the content permanently, you need to have IPFS node running on your computer. You can easily install your local IPFS node using one of these desktop clients:\n\n* [ipfs-desktop](https://github.com/ipfs-shipyard/ipfs-desktop/releases)\n* [Orion](https://orion.siderus.io/)\n'
