b"easybook\n========\n\n*\xc2\xabbook publishing as easy as it should be\xc2\xbb*\n\n[![SensioLabsInsight](https://insight.sensiolabs.com/projects/34c47e7f-a523-4702-8310-ebec02a6a241/mini.png)](https://insight.sensiolabs.com/projects/34c47e7f-a523-4702-8310-ebec02a6a241) [![Travis CI status](https://secure.travis-ci.org/javiereguiluz/easybook.png?branch=master)](http://travis-ci.org/javiereguiluz/easybook) [![Scrutinizer Quality Score](https://scrutinizer-ci.com/g/javiereguiluz/easybook/badges/quality-score.png?s=90c6ed79f22c90ee2c4761937b58ebe9c6b68889)](https://scrutinizer-ci.com/g/javiereguiluz/easybook/) [![Coverage Status](https://coveralls.io/repos/javiereguiluz/easybook/badge.svg?branch=master)](https://coveralls.io/r/javiereguiluz/easybook?branch=master)\n\n**[easybook](http://easybook-project.org)** lets you easily publish books in\nvarious electronic formats (ePub, MOBI, PDF and HTML). It was originally\ndesigned to publish programming books, but you can use **easybook** to\npublish any kind of book, manual or documentation website.\n\n![easybook worflow diagram](doc/easybook-doc-en/Contents/images/what_is_easybook.png)\n\nInstallation\n------------\n\nMake sure to have installed [Composer](https://getcomposer.org/) globally in\nyour system and execute the following command:\n\n```bash\n$ composer create-project easybook/easybook easybook\n```\n\nOnce installed, use **easybook** with the `book` command:\n\n```\n$ cd <easybook-installation-dir>\n$ ./book\n```\n\nIf the last command doesn't work, try `php book` or check `book` script\npermissions.\n\nDocumentation\n-------------\n\n**easybook** is fully documented at http://easybook-project.org/documentation\n\nLicense\n-------\n\n**easybook** is licensed under the [MIT license](LICENSE.md).\n\nTests\n-----\n\nExecute the following command to test **easybook** (it requires PHPUnit):\n\n```\n$ cd <easybook-installation-dir>\n$ phpunit\n```\n\nRequirements\n------------\n\nIn order to generate PDF files, PrinceXML library must be installed.\nIf you haven't installed it yet, you can download a fully-functional demo at:\n\n    http://www.princexml.com/download\n\nIn order to generate MOBI files, KindleGen library must be installed.\nIf you haven't installed it yet, you can download it freely at Amazon:\n\n    http://amzn.to/kindlegen\n"
