b'<img height="32" width="32" hspace="1" vspace="1" align="left" src="https://unpkg.com/simple-icons@latest/icons/github.svg" />\n\n# Github Create Release Action\n\n[![Actions Status](https://wdp9fww0r9.execute-api.us-west-2.amazonaws.com/production/badge/frankjuniorr/github-create-release-action)](https://wdp9fww0r9.execute-api.us-west-2.amazonaws.com/production/results/frankjuniorr/github-create-release-action)\n[![GitHub release](https://img.shields.io/github/release/frankjuniorr/github-create-release-action.svg)](https://GitHub.com/frankjuniorr/github-create-release-action/releases/)\n\nGitHub Action to create GitHub Release automatically.\n\n<p align="center">\n  <img src="images/action_screenshot.png" alt="ubuntu_install" height="300px">\n</p>\n\n# Example Usage\n\n```hcl\nworkflow "Deploy Release" {\n  on = "push"\n  resolves = [" Github Create Release"]\n}\n\naction "Filters for Master branch" {\n  uses = "actions/bin/filter@master"\n  args = "branch master"\n}\n\naction " Github Create Release" {\n  uses = "frankjuniorr/github-create-release-action@master\n  needs = ["Filters for Master branch"]\n  secrets = ["GITHUB_TOKEN"]\n}\n\n```\n\nIn this case above, the Action only triggered if branch was ``master``\n\n# Environment Variables\n\nOptional, but if you pass the env var ``VERSION``, the action will create a git tag with this value.
