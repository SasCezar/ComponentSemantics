b'AAR Publish Plugin\n==================\n\n[![Gradle Plugin Portal](https://img.shields.io/maven-metadata/v/https/plugins.gradle.org/m2/com/github/technoir42/aar-publish/com.github.technoir42.aar-publish.gradle.plugin/maven-metadata.xml.svg?colorB=007ec6&label=gradlePluginPortal)](https://plugins.gradle.org/plugin/com.github.technoir42.aar-publish)\n\nGradle plugin for publishing Android (AAR) libraries using [Maven Publish](https://docs.gradle.org/current/userguide/publishing_maven.html) plugin.\n\n## Requirements\n\n* Gradle 5.3+\n* Android Gradle Plugin 3.3+\n\n## Usage\n\n```groovy\nplugins {\n    id "com.android.library"\n    id "com.github.technoir42.aar-publish" version "1.0.2"\n    id "maven-publish"\n}\n\npublishing {\n    publications {\n        maven(MavenPublication) {\n            from components.android\n        }\n    }\n}\n```\n\nTo learn how to configure publications and repositories refer to the documentation for [Maven Publish](https://docs.gradle.org/current/userguide/publishing_maven.html) plugin.\n\n## Variants\n\nThe plugin registers the following `SoftwareComponent`s:\n\n* `components.android` - for the default variant set using `android.defaultPublishConfig` which is `release` by default.\n* `components.android${capitalizedVariantName}` - for all variants, e.g. `androidMyFlavorRelease` if variant name is `myFlavorRelease`.\n\nChanging the default variant:\n\n```groovy\nandroid {\n    defaultPublishConfig "myFlavorRelease"\n\n    productFlavors {\n        myFlavor {\n        }\n    }\n}\n```\n\nPublishing a specific variant:\n\n```groovy\npublishing {\n    publications {\n        maven(MavenPublication) {\n            afterEvaluate {\n                from components.androidMyFlavorRelease\n            }\n        }\n    }\n}\n```\n\n## Plugin configuration\n\nSources and Javadoc are published by default but you can disable publishing either of them using the following DSL:\n\n```groovy\naarPublishing {\n    publishJavadoc = false\n    publishSources = false\n}\n```\n\n## License\n\n```\nCopyright 2019 Sergey Chelombitko\n\nLicensed under the Apache License, Version 2.0 (the "License");\nyou may not use this file except in compliance with the License.\nYou may obtain a copy of the License at\n\n   http://www.apache.org/licenses/LICENSE-2.0\n\nUnless required by applicable law or agreed to in writing, software\ndistributed under the License is distributed on an "AS IS" BASIS,\nWITHOUT WARRANTIES OR CONDITIONS OF ANY KIND, either express or implied.\nSee the License for the specific language governing permissions and\nlimitations under the License.\n```\n'
