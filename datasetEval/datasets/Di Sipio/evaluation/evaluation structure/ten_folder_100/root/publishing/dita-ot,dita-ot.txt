b'# DITA Open Toolkit [![Build]](http://travis-ci.org/dita-ot/dita-ot) [![Slack]](http://slack.dita-ot.org/)\n\n_DITA Open Toolkit_, or _DITA-OT_ for short, is an open-source publishing engine for XML content authored in the _Darwin Information Typing Architecture_.\n\nVisit the project website at [dita-ot.org][site] for documentation, information about releases, and [download packages][dist].  \n\nFor information on additional DITA and DITA-OT resources, see [SUPPORT]. To report a bug or suggest a feature, [create an issue][issue]. For more information on how you can help contribute to the project, see [CONTRIBUTING].\n\n- [Prerequisites: Java 8](#prerequisites-java-8)\n- [Installing](#installing)\n- [Building output](#building-output)\n- [For developers](#for-developers)\n- [License](#license)\n\n## Prerequisites: Java 8\n\n- To _build_ DITA-OT, you\xe2\x80\x99ll need Java Development Kit (JDK), version 8 or newer\n- To _run_ DITA-OT, the Java Runtime Environment (JRE) is sufficient\n\nYou can download the Oracle JRE or JDK from [oracle.com/technetwork/java][java].\n\n## Installing\n\n1.  Download the distribution package from [dita-ot.org/download][dist].\n2.  Extract the contents of the package to the directory where you want to install DITA-OT.\n\n### Installing on macOS via Homebrew\n\nOn macOS, you can also install DITA-OT using the [Homebrew] package manager:\n\n    brew install dita-ot\n\nHomebrew will automatically download the latest version of the toolkit, install it in a subfolder of the local package Cellar and symlink the `dita` command to `/usr/local/bin/dita`.\n\n## Building output\n\nYou can generate output using the `dita` command-line tool included with DITA Open Toolkit.\n\n1.  On the command line, change to the `bin` folder of the DITA-OT installation directory:\n\n        cd path/to/dita-ot-dir/bin\n\n2.  Run the `dita` command to generate output:\n\n        dita --input=input-file --format=format [options]\n\n    where:\n\n    - _`input-file`_ is the DITA map or DITA file that you want to process\n    - _`format`_ is the output format (or \xe2\x80\x9ctransformation type\xe2\x80\x9d)\n\nSee the [documentation][docs] for arguments and [options].\n\n## For developers\n\n<details>\n<summary>Building the toolkit from source code and compiling the distribution package</summary>\n\n1.  Clone the DITA-OT Git repository:\n\n        git clone git://github.com/dita-ot/dita-ot.git\n\n2.  Change to the DITA-OT directory:\n\n        cd dita-ot\n\n3.  Fetch the submodules:\n\n        git submodule update --init --recursive\n\n4.  In the root directory, run Gradle to compile the Java code and install plugins:\n\n        ./gradlew\n\n### Distribution builds\n\n1.  In the root directory, set up the build environment:\n\n        ./gradlew\n\n2.  Build the distribution packages:\n\n        ./gradlew dist\n\n    Distribution packages are built in the `build/distributions` directory.\n\n    If Gradle throws an error like `java.lang.OutOfMemoryError: Java heap space`, you probably need to increase the maximum Java heap size. One way to do this is to set the `GRADLE_OPTS` environment variable to a value like `-Xmx1024m`.\n\n    For more information on the `-Xmx` option, see the [Java SE Documentation][javadoc].\n\n</details>\n\n## License\n\nDITA Open Toolkit is licensed for use under the [Apache License 2.0][apache].\n\n[build]: https://travis-ci.org/dita-ot/dita-ot.svg?branch=develop\n[slack]: http://slack.dita-ot.org/badge.svg\n[site]: https://www.dita-ot.org/\n[dist]: https://www.dita-ot.org/download\n[support]: https://github.com/dita-ot/.github/blob/master/SUPPORT.md\n[java]: http://www.oracle.com/technetwork/java/javase/downloads\n[homebrew]: https://brew.sh\n[docs]: https://www.dita-ot.org/dev/\n[options]: https://www.dita-ot.org/dev/topics/build-using-dita-command.html\n[javadoc]: http://docs.oracle.com/javase/8/docs/technotes/tools/windows/java.html#BABHDABI\n[apache]: http://www.apache.org/licenses/LICENSE-2.0\n[issue]: https://github.com/dita-ot/dita-ot/issues/new/choose\n[contributing]: https://github.com/dita-ot/.github/blob/master/CONTRIBUTING.md\n'
