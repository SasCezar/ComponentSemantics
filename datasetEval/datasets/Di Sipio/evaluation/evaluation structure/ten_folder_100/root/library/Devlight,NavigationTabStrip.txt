can set such parameters as:\n \n - color:  \n    allows you to set strip color.\n    \n - size:  \n    allows you to set titles size. By default `NTS` use auto titles size.\n    \n - weight:  \n    allows you to set weight(height) of strip.\n    \n - factor:  \n    allows you to set strip resize factor.\n    \n - titles:  \n    allows you to set `NTS` titles. This is your tabs.\n    \n - type:  \n    allows you to set strip type - line or point.\n    \n - gravity:  \n    allows you to set strip gravity - top or bottom.\n    \n - view pager:  \n    allows you to connect `NTS` with `ViewPager`. If you want your can also set `OnPageChangeListener`.    \n    \n - typeface:  \n    allows you to set custom typeface to your titles.\n \n - corners radius:  \n    allows you to set corners radius of strip.\n\n - animation duration:  \n    allows you to set animation duration.\n      \n - inactive color:  \n    allows you to set inactive titles color.\n     \n - active color:  \n    allows you to set active title color.\n     \n - tab strip listener:  \n    allows you to set listener which triggering on start or on end when you set tab index.\n\n<b>Tips</b>\n\nIf your set `ViewPager` you can action down on active tab and do like drag.  \n`NTS` use only upper case titles.\n\n<b>Init</b>\n\nCheck out in code init:\n\n```java\nfinal NavigationTabStrip navigationTabStrip = (NavigationTabStrip) findViewById(R.id.nts);\nnavigationTabStrip.setTitles("Nav", "Tab", "Strip");\nnavigationTabStrip.setTabIndex(0, true);\nnavigationTabStrip.setTitleSize(15);\nnavigationTabStrip.setStripColor(Color.RED);\nnavigationTabStrip.setStripWeight(6);\nnavigationTabStrip.setStripFactor(2);\nnavigationTabStrip.setStripType(NavigationTabStrip.StripType.LINE);\nnavigationTabStrip.setStripGravity(NavigationTabStrip.StripGravity.BOTTOM);\nnavigationTabStrip.setTypeface("fonts/typeface.ttf");\nnavigationTabStrip.setCornersRadius(3);\nnavigationTabStrip.setAnimationDuration(300);\nnavigationTabStrip.setInactiveColor(Color.GRAY);\nnavigationTabStrip.setActiveColor(Color.WHITE);\nnavigationTabStrip.setOnPageChangeListener(...);\nnavigationTabStrip.setOnTabStripSelectedIndexListener(...);\n```\n            \nOther methods check out in sample.\n\nAnd XML init:\n\n```xml\n<com.gigamole.navigationtabstrip.NavigationTabStrip\n    android:layout_width="match_parent"\n app:nts_gravity="top"\n    app:nts_typeface="fonts/typeface.otf"\n    app:nts_corners_radius="1.5dp"\n    app:nts_animation_duration="300"\n    app:nts_active_color="#000"\n
