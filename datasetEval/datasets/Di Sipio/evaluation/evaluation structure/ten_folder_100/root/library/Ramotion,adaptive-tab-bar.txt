The UI elements evolve over time as the user becomes accustomed to them. An example AdaptiveTabBar is provided.\n\n## How to import the control to your project\n\n1. First, drag and drop the AdaptiveController project file into your project or select "add existing file" and choose the AdaptiveController project\n2. Add AdaptiveController to your Target Dependences in the Build Phase of your project.\n3. Add the AdaptiveController framework in Link Binary With Libraries in Build Phases\n4. Import AdaptiveController where needed.\n\n## Using AdaptiveController \n\nFirst, you should set the install date, current date, count of days to transition to small text state, and count of days to transition to big image state. The AdaptiveDateState object automatically determines what UI state to show:\n\n```swift\n    var installDate = NSDate(dateString:"2014-09-18")\n    var adaptiveState = AdaptiveDateState(installDate: installDate, currentDate:NSDate(), countDaysToSmallTextState:countDaysToSmallTextState, countDaysToImageState:countDaysToImageState)\n```\nOr count of launches for different state:\n\n```swift\n    var curentCountLaunches = 6\n    var adaptiveState = AdaptiveLaunchesState(curentCountLaunches:curentCountLaunches, countLaunchesToSmallTextState:5, countLaunchesToImageState:7)\n```\n\nAnd then you should setup an array of appearances. You should setup button images for each state for each apperance object:\n\n```swift\n    var buttonsAppearances = buttonsAppearancesGenerate() //func butonsAppearancesGenerate() -> [AdaptiveButtonApperance]\n```\n\nAnd then you should setup an array of items Appearance object. You should set the button\'s image for each state for each appearance object.\n\nFirst, you should setup a dictionary for the extension of the image for each UI state:\n\t\t\n```swift\n    var imageExtensionsForStates:Dictionary = [ kSmallTitleAdaptiveState:"_smalltitle", kImageAdaptiveState:"_bigimage", kSmallTitleAdaptiveState+selected :"_smalltitle", kImageAdaptiveState+selected:"_bigimage" ]\n```\n Then you should set the Appearance for each UITabBarItem title for each state, fonts for each state, a default image if you describe an extension for each state, or if you do not describe an extension, you should setup each image for each state.\n \n First, init the appearance object and setup the text for each state. If the text is the same for each state, you can set text for the default state:\n \n```swift\n    var watchAppearance = AdaptiveButtonAppearance(); \n    watchAppearance.setButonTitle("watch", state: kDefaultAdaptiveState)\n    watchAppearance.setButonTitle("watch", state: kSmallTitleAdaptiveState)\n    watchAppearance.setButonTitle("", state: kImageAdaptiveState)\n```\n\nThen you should setup your fonts for each state:\n\n```swift\n    watchAppearance.setButonTitleFontForState(yourDefaultFont, state: kDefaultAdaptiveState)\n    watchAppearance.setButonTitleFontForState(yourDefaultSmallTitleModeFont, state: kSmallTitleAdaptiveState)\n```\n\nThen you should set the default image and extensions for each state a function which automaticaly sets images for each state: \n\n```swift\n    watchAppearance.setImageNamesForStatesImageExtesions("watch", imageExtensionsForState:imageExtensionsForStates)\n```\n\nThen you should setup the appearance insets for each state: \n\n```swift\n    watchAppearance.setImageInsets(defaultInsets, state: kDefaultAdaptiveState);\n    watchAppearance.setImageInsets(defaultSmallTitleModeImageInsets, state: kSmallTitleAdaptiveState)\n    watchAppearance.setTitleOffset(defaultOffset, state: kDefaultAdaptiveState)\n    watchAppearance.setImageInsets(defaultImageModeInsets, state: kImageAdaptiveState);\n```\n\nThen you should setup \xd1\x81ustom UITabBarItems which conform to the AdaptiveButtonsProtocol:\n\n```swift \n    var arrayButtons = tabBar.items as [AdaptiveTabBarItem]\n```\n\nIn your custom tab item, you should implement these methods from the protocol:\n\t\t\n```swift\n    @objc protocol AdaptiveApperanceProtocol {\n        optional func setFontToAdaptiveButton(font: UIFont)\n        optional func setTitleToAdaptiveButton(text: NSString)\n        optional func setImageToAdaptiveButton(image: UIImage?)\n        optional func setHighlightedToAdaptiveButton(image: UIImage?)\n        optional func setBackgroundImageToAdaptiveButton(image: UIImage?)\n        optional func setSelectedImageToAdaptiveButton(image: UIImage?)\n        optional func setImageInsetsToAdaptiveButton(insets: UIEdgeInsets)\n        optional func setTitleOffsetToAdaptiveButton(offset: UIOffset)\n        optional func setTitleColorToAdaptiveButton(titleColor: UIColor)\n    }\n```\n\nAn example implemetation of a protocol method from our custom tab bar item:\n\t\n```swift\t\n    func setTitleToAdaptiveButton(text: NSString) {\n    \tself.title = text\n\t}\n```\n\n\nFinally, you should init Adaptive State Manager with the objects you\'ve setup:\n\n```swift\n\tAdaptiveButtonsStateManager(state: adaptiveState, buttonsAray:arrayButtons, buttonsAppearance:butonsAppearances)\n```\n\n\n##Using Custom States For AdaptiveController \n\nLike UITabBar for different custom applications states \n\nWe have a default AdaptiveState and with the help of inheritance, we can add\nnew custom states to our state class. The new class in our situation is \nnamed AdaptiveDateState\n\nIn AdaptiveDateState, we can add new custom states like this:\n\n```swift\n\tlet kSmallTitleAdaptiveState = "kImageAdaptiveState"\n\tlet kImageAdaptiveState = "kNormalImageAdaptiveState"\n```\n\nAnd add methods which decide how the state relates to input parameters:\n\n```swift\n\tvar adaptiveState = AdaptiveDateState(installDate: installDate,currentDate:NSDate(),countDaysToSmallTextState:countDaysToSmallTextState,countDaysToImageState:countDaysToImageState)\n```\n\nIn AdaptiveDateState, we add custom init methods which decided what states have the Adaptive state object according to the input parameters:\n\n```swift \n    private func stateRemainDays(remainDays:Int, countDaysToSmallTextState:Int, countDaysToImageState:Int)->String {\n        var mode:String = kDefaultAdaptiveState\n        print(" DAYS \\(remainDays) ")\n        if remainDays > countDaysToSmallTextState && remainDays < countDaysToImageState{\n            mode = kSmallTitleAdaptiveState\n        } else if remainDays > countDaysToImageState {\n            mode = kImageAdaptiveState\n        }\n        print(mode)\n        return mode\n    }\n```\n\nYou can customize different UI components if they adopt to the AdaptiveApperanceProtocol.\n\n## \xf0\x9f\x93\x84 License\n\nAdaptive Tab Bar is released under the MIT
