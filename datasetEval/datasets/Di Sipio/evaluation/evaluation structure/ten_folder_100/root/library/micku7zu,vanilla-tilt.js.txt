b'# vanilla-tilt.js\n\n[![npm version](https://badge.fury.io/js/vanilla-tilt.svg)](https://badge.fury.io/js/vanilla-tilt)\n\nA smooth 3D tilt javascript library forked from [Tilt.js (jQuery version)](https://github.com/gijsroge/tilt.js).\n\n[View landing page (demos)](https://micku7zu.github.io/vanilla-tilt.js/)\n\n### Usage\n\n```html\n<body>\n<div class="your-element" data-tilt></div>\n\n<!-- at the end of the body -->\n<script type="text/javascript" src="vanilla-tilt.js"></script>\n</body>\n```\n\nIf you want to use this library in IE, you need to include a CustomEvent polyfill: https://github.com/micku7zu/vanilla-tilt.js/issues/49#issuecomment-482711876 or maybe consider the [jQuery version](https://github.com/gijsroge/tilt.js).\n### Options\n```js\n{\n    reverse:                false,  // reverse the tilt direction\n    max:                    35,     // max tilt rotation (degrees)\n    startX:                 0,      // the starting tilt on the X axis, in degrees.\n    startY:                 0,      // the starting tilt on the Y axis, in degrees.\n    perspective:            1000,   // Transform perspective, the lower the more extreme the tilt gets.\n    scale:                  1,      // 2 = 200%, 1.5 = 150%, etc..\n    speed:                  300,    // Speed of the enter/exit transition\n    transition:             true,   // Set a transition on enter/exit.\n    axis:                   null,   // What axis should be disabled. Can be X or Y.\n    reset:                  true,   // If the tilt effect has to be reset on exit.\n    easing:                 "cubic-bezier(.03,.98,.52,.99)",    // Easing on enter/exit.\n    glare:                  false,  // if it should have a "glare" effect\n    "max-glare":            1,      // the maximum "glare" opacity (1 = 100%, 0.5 = 50%)\n    "glare-prerender":      false,  // false = VanillaTilt creates the glare elements for you, otherwise\n                                    // you need to add .js-tilt-glare>.js-tilt-glare-inner by yourself\n    "mouse-event-element":  null,   // css-selector or link to HTML-element what will be listen mouse events\n    "full-page-listening":  false,  // If true, parallax effect will listen to mouse move events on the whole document, not only the selected element\n    gyroscope:              true,   // Boolean to enable/disable device orientation detection,\n    gyroscopeMinAngleX:     -45,    // This is the bottom limit of the device angle on X axis, meaning that a device rotated at this angle would tilt the element as if the mouse was on the left border of the element;\n    gyroscopeMaxAngleX:     45,     // This is the top limit of the device angle on X axis, meaning that a device rotated at this angle would tilt the element as if the mouse was on the right border of the element;\n    gyroscopeMinAngleY:     -45,    // This is the bottom limit of the device angle on Y axis, meaning that a device rotated at this angle would tilt the element as if the mouse was on the top border of the element;\n    gyroscopeMaxAngleY:     45,     // This is the top limit of the device angle on Y axis, meaning that a device rotated at this angle would tilt the element as if the mouse was on the bottom border of the element;\n    gyroscopeSamples:       10      // How many gyroscope moves to decide the starting position.\n}\n```\n\n### Events\n```js\nconst element = document.querySelector(".js-tilt");\nVanillaTilt.init(element);\nelement.addEventListener("tiltChange", callback);\n```\n\n### Methods\n```js\nconst element = document.querySelector(".js-tilt");\nVanillaTilt.init(element);\n\n// Destroy instance\nelement.vanillaTilt.destroy();\n\n// Get values of instance\nelement.vanillaTilt.getValues();\n\n// Reset instance\nelement.vanillaTilt.reset();\n\n// It also supports NodeList\nconst elements = document.querySelectorAll(".js-tilt");\nVanillaTilt.init(elements);\n```\n\n### Install\nYou can copy and
