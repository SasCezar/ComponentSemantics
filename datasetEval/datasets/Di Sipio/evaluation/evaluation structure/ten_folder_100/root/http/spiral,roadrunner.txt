src="https://img.shields.io/badge/discord-chat-magenta.svg"></a>\n</p>\n\nRoadRunner is an open-source (MIT licensed) high-performance PHP application server, load balancer, and process manager.\nIt supports running as a service with the ability to extend its functionality on a per-project basis. \n\nRoadRunner includes PSR-7/PSR-17 compatible HTTP and HTTP/2 server and can be used to replace classic Nginx+FPM setup with much greater performance and flexibility.\n\n<p align="center">\n\t<a href="https://roadrunner.dev/"><b>Official Website</b></a> | \n\t<a href="https://roadrunner.dev/docs"><b>Documentation</b></a>\n</p>\n\nFeatures:\n--------\n- production-ready\n- PSR-7 HTTP server (file uploads, error handling, static files, hot reload, middlewares, event listeners)\n- HTTPS and HTTP/2 support (including HTTP/2 Push, H2C)\n- fully customizable server\n- flexible environment configuration\n- no external PHP dependencies (64bit version required), drop-in (based on [Goridge](https://github.com/spiral/goridge))\n- load balancer, process manager and task pipeline\n- frontend agnostic ([Queue](https://github.com/spiral/jobs), PSR-7, [GRPC](https://github.com/spiral/php-grpc), etc)\n- integrated metrics (Prometheus)\n- works over TCP, UNIX sockets and standard pipes\n- automatic worker replacement and safe PHP process destruction\n- worker create/allocate/destroy timeouts\n- max jobs per worker\n- worker lifecycle management (controller) \n    - maxMemory (graceful stop)\n    - TTL (graceful stop)\n    - idleTTL (graceful stop)\n    - execTTL (brute, max_execution_time)   \n- payload context and body\n- protocol, worker and job level error management (including PHP errors)\n- very fast (~250k rpc calls per second on Ryzen 1700X using 16 threads)\n- integrations with Symfony, Laravel, Slim, CakePHP, Zend Expressive, Spiral\n- works on Windows\n\nInstallation:\n--------\nTo install:\n\n```\n$ composer require spiral/roadrunner\n$ ./vendor/bin/rr get-binary\n```\n\nExample:\n--------\n\n```php\n<?php\n// worker.php\nini_set(\'display_errors\', \'stderr\');\ninclude "vendor/autoload.php";\n\n$relay = new Spiral\\Goridge\\StreamRelay(STDIN, STDOUT);\n$psr7 = new Spiral\\RoadRunner\\PSR7Client(new Spiral\\RoadRunner\\Worker($relay));\n\nwhile ($req = $psr7->acceptRequest()) {\n    try {\n        $resp = new \\Zend\\Diactoros\\Response();\n        $resp->getBody()->write("hello world");\n\n        $psr7->respond($resp);\n    } catch (\\Throwable $e) {\n        $psr7->getWorker()->error((string)$e);\n    }\n}\n```\n\nConfiguration can be located in `.rr.yaml` file ([full sample](https://github.com/spiral/roadrunner/blob/master/.rr.yaml)):\n\n```yaml\nhttp:\n  address:         0.0.0.0:8080\n  workers.command: "php worker.php"\n```\n\n> Read more in [Wiki](https://github.com/spiral/roadrunner/wiki/PHP-Workers).\n\nRun:\n----\nTo run application
