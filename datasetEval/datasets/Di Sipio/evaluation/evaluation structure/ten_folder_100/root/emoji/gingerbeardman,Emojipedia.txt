b'# Emojipedia\n\nMacOS X Dictionary containing Emoji and their meanings\n\n## Installing the dictionary\n\nPick one method of installation: using Dictionaries.app, Terminal, or Finder.\n\n### Using Dictionaries.app\n\n1. Download [Dictionaries.app](https://dictionaries.io) and run it\n2. Click "Add Dictionary"\n3. Select Emojipedia\n4. Click "Add"\n\n### Using Terminal\n\n1. Install using homebrew caskroom `brew cask install emojipedia`\n2. Launch Dictionary.app\n3. In Dictionary > Preferences, scroll to the bottom of the list and check the *Emojipedia* dictionary to enable it\n4. Optional: drag the *Emojipedia* dictionary entry to change the order of dictionaries\n\n### Using Finder\n\n1. Download [Emojipedia.dictionary.zip](https://github.com/gingerbeardman/Emojipedia/releases/download/20190306/Emojipedia.dictionary.zip) and unzip it\n2. Copy `Emojipedia.dictionary` to `~/Library/Dictionaries`\n3. Launch Dictionary.app\n4. In Dictionary > Preferences, scroll to the bottom of the list and check the *Emojipedia* dictionary to enable it\n5. Optional: drag the *Emojipedia* dictionary entry to change the order of dictionaries\n\n---\n\n## Using the dictionary\n\nTry looking up the meaning of these emoji as a test: \xf0\x9f\x98\x8d \xf0\x9f\x94\xb0 \xf0\x9f\x92\xae \xf0\x9f\x92\xa9\n\n* Right-click on the emoji and choose the _Look Up_ menu item\n  (on a track pad you can also use a Three Finger Tap or Force Click)\n* Press Cmd+Ctrl+D whilst pointing the mouse cursor at the emoji to display a popup\n* Launch Dictionary and type the emoji into the search field\n\n## Screenshots\n\nContextual menu Look Up\n![Dictionary lookup](https://github.com/gingerbeardman/Emojipedia/blob/master/screenshot_emoji-dictionary_lookup.png)\n\nDicationary Popup\n![Dictionary popup](https://github.com/gingerbeardman/Emojipedia/blob/master/screenshot_emoji-dictionary_popup.png)\n\nDictionary.app\n![Dictionary.app](https://github.com/gingerbeardman/Emojipedia/blob/master/screenshot_emoji-dictionary_app.png)\n\n---\n\n## Status\n\nNumber of Emoji included: 1719  \nEmoji Data: v12  \n\nFeatures:\n\n* Emoji display\n* Descriptive name\n* Hyperlinked keywords\n* Link to read more at [Emojipedia](http://emojipedia.org)\n\n## Todo\n\n* Add long descriptions and other data provided by [Emojipedia](http://emojipedia.org)\n* Add translations\n* Add language toggle\n* Transition to use [Emoji Data](http://www.unicode.org/Public/emoji/12.0/) instead of copy/pasted [HTML table](http://unicode.org/emoji/charts/emoji-list.html)\n\n---\n\n## Editing the dictionary\n\nThe following steps are only required if you wish to modify the dictonary.\n\nFeel free to send Pull Requests for items on the todo list!\n\n### Requirements\n\n* php\n* [Additional Tools for Xcode](https://developer.apple.com/download/more/)(download the one fit your Xcode\'s version) (contains the Dictionary Development Kit)\n\n### Preparing\n\n1.
