b'# alfred-emoj [![Build Status](https://travis-ci.org/sindresorhus/alfred-emoj.svg?branch=master)](https://travis-ci.org/sindresorhus/alfred-emoj)\n\n> [Alfred 3](https://www.alfredapp.com) workflow to find relevant emoji from text\n\n<img src="screenshot-combined.png" width="895" height="554">\n\nUses the API from this great article on [Emoji & Deep Learning](http://getdango.com/emoji-and-deep-learning.html).<br>\nCheck out the [Dango app](http://getdango.com) if you want something like this on your phone.\n\n\n## Install\n\n```\n$ npm install --global alfred-emoj\n```\n\n*Requires [Node.js](https://nodejs.org) 8+ and the Alfred [Powerpack](https://www.alfredapp.com/powerpack/).*\n\n\n## Usage\n\nIn Alfred, type `emoj`, <kbd>Enter</kbd>, and some text, to find relevant emoji.\n\nSelect an emoji and press <kbd>Enter</kbd> to copy it to the clipboard.\n\nWhen at the top, press <kbd>\xe2\x96\xb2</kbd> and <kbd>Enter</kbd>, to copy all the emoji.\n\n\n## Links\n\n- [Product Hunt submission](https://www.producthunt.com/posts/alfred-emoj)\n\n\n## Related\n\n- [emoj](https://github.com/sindresorhus/emoj) - Find relevant emoji from text on the command-line\n- [alfred-npms](https://github.com/sindresorhus/alfred-npms) - Search for npm packages with npms.io\n- [alfred-dark-mode](https://github.com/sindresorhus/alfred-dark-mode) - Toggle the system dark mode\n- [alfred-xcode](https://github.com/sindresorhus/alfred-xcode) - Open Xcode projects and workspaces\n- [alfred-simple](https://github.com/sindresorhus/alfred-simple) - Simple theme\n- [alfy](https://github.com/sindresorhus/alfy) - Create Alfred workflows with ease\n\n\n## License\n\nMIT \xc2\xa9 [Sindre Sorhus](https://sindresorhus.com)\n'
