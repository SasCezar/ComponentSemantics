b'# Ecoji \xf0\x9f\x8f\xa3\xf0\x9f\x94\x89\xf0\x9f\xa6\x90\xf0\x9f\x94\xbc\n\nEcoji encodes data as 1024 [emojis][emoji], its base1024 with an emoji character set.  As a bonus, includes code to decode emojis to original data.  \n\nMany have asked how Ecoji compares to base64.  The short answer is that Ecoji is more bytes, but less visible characters. With Ecoji each visible char represents 10 bits, but each character is multi-byte.  With base64 each char represents 6 bits and is one byte.  The following table shows encoding sha256 in different ways.\n\nEncoding | Bytes | Characters \n---------|-------|-----------\nnone     | 32    | N/A\nhex      | 64    | 64\nbase64   | 44    | 44\necoji    | 112   | 28\n\n## Installing\n\nEcoji is published to [snapcraft.io](https://snapcraft.io/ecoji) and can be installed with :\n\n```bash\nsudo snap install ecoji\n```\n\n## Examples of
