emojis.\nMinimum SDK Level: 9 (2.3)\n\n\n![image](https://github.com/hani-momanii/SuperNova-Emoji/blob/master/vid.gif)\n\n## Contact\n\n\n[![alt text][1.1]][1]\n[![alt text][2.1]][2]\n[![alt icon with padding)\n\n## Java Usage\n\nTo use default colors : \nEmojIconActions(Context ctx,View rootView,EmojiconEditText emojiconEditText,ImageView emojiButton)\n```\nEmojIconActions  emojIcon=new EmojIconActions(this,rootView,emojiconEditText,emojiButton);\nemojIcon.ShowEmojIcon();\n```\n\n![image](https://github.com/hani-momanii/SuperNova-Emoji/blob/master/ios.png)\n\n\nTo use custom color :\nEmojIconActions(Context ctx,View rootView,EmojiconEditText emojiconEditText,ImageView emojiButton,String iconPressedColor,String tabsColor,String backgroundColor)\n```\nEmojIconActions  emojIcon=new EmojIconActions(this,rootView,emojiconEditText,emojiButton,"#495C66","#DCE1E2","#E6EBEF");\nemojIcon.ShowEmojIcon();\n```\n![image](https://github.com/hani-momanii/SuperNova-Emoji/blob/master/color.png)\n\n\n\nTo Listen to keyboard status \n```\nemojIcon.setKeyboardListener(new EmojIconActions.KeyboardListener() {\n@Override\npublic void onKeyboardOpen() {\n    Log.e("Keyboard","open");\n  }\n\n@Override\npublic void onKeyboardClose() {\n  Log.e("Keyboard","close");\n}\n});\n```\nTo use the device default emoji\n```\nemojIcon.setUseSystemEmoji(true);\nemojiconEditText.setUseSystemEmoji(true);\n```\n![image](https://github.com/hani-momanii/SuperNova-Emoji/blob/master/def.png)\n\n\n\n## XML Usage\n\n```\n<hani.momanii.supernova_emoji_library.Helper.EmojiconEditText\n        android:id="@+id/emojicon_edit_text"\n        android:layout_width="match_parent"\n        android:layout_height="wrap_content"\n        emojicon:emojiconSize="28sp" />\n        \n        \n<hani.momanii.supernova_emoji_library.Helper.EmojiconTextView\n        android:id="@+id/emojicon_text_view"\n        android:layout_width="wrap_content"\n        android:layout_height="wrap_content" \n        emojicon:emojiconSize="28sp"/>\n\n```\n\n\n\n## Usage\n\n* `EmojiconTextView`: a `TextView` which can render emojis.\n* `EmojiconEditText`: a `EditText` which can render emojis.\n* `EmojiconMultiAutoCompleteTextView`: a `MultiAutoCompleteTextView` which can render emojis.\n\n## Building in IntelliJ\n\nVia Gradle:\n\n```\n\nrepositories {\n    maven { url \'https://jitpack.io\' }\n}\ncompile \'com.github.hani-momanii:SuperNova-Emoji:1.1\'\n```\n\n##
