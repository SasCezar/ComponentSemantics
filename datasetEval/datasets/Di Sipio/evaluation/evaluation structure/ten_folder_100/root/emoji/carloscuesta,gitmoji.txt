As there are a lot of different emojis I found the need of creating a guide that can help to use emojis easier.\n\n## Using [gitmoji-cli](https://github.com/carloscuesta/gitmoji-cli)\n\nTo use gitmojis from your command line install [gitmoji-cli](https://github.com/carloscuesta/gitmoji-cli). A gitmoji interactive client for using emojis on commit messages.\n\n```bash\nnpm i -g gitmoji-cli\n```\n\n## Contributing to gitmoji\n\nContributing to gitmoji is a piece of :cake:, read the [contributing guidelines](https://github.com/carloscuesta/gitmoji/blob/master/.github/CONTRIBUTING.md). You can discuss emojis using the [issues section](https://github.com/carloscuesta/gitmoji/issues/new). To add a new emoji to the list create an issue and send a pull request, see [how to send a pull request and add a gitmoji](https://github.com/carloscuesta/gitmoji/blob/master/.github/CONTRIBUTING.md#how-to-add-a-gitmoji).\n\n## License\n\nThe code is available under the [MIT](https://github.com/carloscuesta/gitmoji/blob/master/LICENSE) license.\n'
