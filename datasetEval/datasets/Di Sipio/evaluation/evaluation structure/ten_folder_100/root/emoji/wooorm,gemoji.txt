b"# gemoji\n\n[![Build][build-badge]][build]\n[![Coverage][coverage-badge]][coverage]\n[![Downloads][downloads-badge]][downloads]\n[![Size][size-badge]][size]\n\nGemoji (**G**itHub **Emoji**) maps Gemoji short-codes and unicode emoji to\nobjects containing info on their respective category, description, names,\nand tags.\n\n## Installation\n\n[npm][]:\n\n```bash\nnpm install gemoji\n```\n\n## Usage\n\n```javascript\nvar gemoji = require('gemoji')\n\nconsole.log(gemoji.name.cat)\n```\n\nYields:\n\n```js\n{ category: 'nature',\n  description: 'cat face',\n  names: [ 'cat' ],\n  tags: [ 'pet' ],\n  name: 'cat',\n  emoji: '\xf0\x9f\x90\xb1' }\n```\n\n### By unicode emoji\n\n```javascript\nconsole.log(gemoji.unicode['\xf0\x9f\x90\xb6'])\n```\n\nYields:\n\n```js\n{ category: 'nature',\n  description: 'dog face',\n  names: [ 'dog' ],\n  tags: [ 'pet' ],\n  name: 'dog',\n  emoji: '\xf0\x9f\x90\xb6' }\n```\n\n...and...\n\n```javascript\nconsole.log(gemoji.unicode['\\uD83D\\uDCA9'])\n```\n\nYields:\n\n```js\n{ category: 'people',\n  description: 'pile of poo',\n  names: [ 'hankey', 'poop', 'shit' ],\n  tags: [ 'crap' ],\n  name: 'hankey',\n  emoji: '\xf0\x9f\x92\xa9' }\n```\n\n## Supported Gemoji\n\nSee [support.md][support].\n\n## Data\n\nThe emoji list is crawled from [github/gemoji][gh] and later processed\nfor relevant information.
