b'# Quill Emoji Selector\r\nModule extension for [Quill.js](https://github.com/quilljs/quill) that handles emojis in the toolbar. Through this extension, you can add emojis through the toolbar at the top, or by typing the emoji code.\r\n\r\n![Screenshot](/demo/screenshot.png)\r\n\r\nTo add an emoji via emoji code, type ``:`` followed by the first few letters, and an autocomplete menu will appear. You can then select or ``tab`` to the preferred emoji.\r\n\r\n\r\n\r\n\r\n#### This module is still in active development\r\n\r\n## Installation\r\n\r\n```sh\r\nyarn add quill-emoji\r\n```\r\n\r\n## Usage\r\n### Webpack/ES6\r\n\r\n```javascript\r\nconst toolbarOptions = {\r\n  container: [\r\n    [\'bold\', \'italic\', \'underline\', \'strike\'],\r\n    [\'emoji\'],   \r\n  ],\r\n  handlers: {\'emoji\': function() {}}\r\n}\r\nconst quill = new Quill(editor, {\r\n  // ...\r\n  modules: {\r\n    // ...\r\n    toolbar: toolbarOptions,\r\n    "emoji-toolbar": true,\r\n    "emoji-textarea": true,\r\n    "emoji-shortname": true,\r\n  }\r\n});\r\n```\r\n### Examples\r\n- [Classic HTML/JS](demo/index.html)\r\n- [AngularJS](demo/angular.html) using [ng-quill](https://github.com/KillerCodeMonkey/ng-quill)\r\n\r\n### Options\r\nSee [emoji-list.js](src/emoji-list.js) for emoji list example\r\n\r\n#### Example options\r\n```javascript\r\n// Custom emoji-list\r\nconst emojiList = [ /* emojiList */ ];\r\n\r\n// MDI emojicon instead of default icon\r\nconst emojiIcon = \'<svg class="i" viewBox="0 0 24 24"><use href="#emoticon-happy"></use></svg>\';\r\n\r\nconst quill = new Quill(editor, {\r\n  // ...\r\n  modules: {\r\n    // ...\r\n    "emoji-shortname": {\r\n      emojiList: emojiList,\r\n      fuse: {\r\n        shouldSort: true,\r\n        threshold: 0.1,\r\n        location: 0,\r\n        distance: 100,\r\n        maxPatternLength: 32,\r\n        minMatchCharLength: 1,\r\n        keys: [\r\n          "shortname"\r\n        ]\r\n      },\r\n      onOpen: function() { /* ... */ },\r\n      onClose: function(emojiListItem) { /* ... */ }\r\n    },\r\n    "emoji-toolbar": {\r\n      buttonIcon: emojiIcon\r\n    },\r\n    "emoji-textarea": {\r\n      buttonIcon: emojiIcon\r\n    }\r\n            \r\n  }\r\n});\r\n```\r\n\r\n### Custom Emoji Blot\r\nIf you need to display the emojis in a different way, you can customize the [emoji blot](src/format-emoji-blot.js) by creating a new blot or extending the default emoji blot.\r\n\r\n#### New emoji blot\r\n```javascript\r\nimport Quill from \'quill\';\r\n\r\nconst Embed = Quill.import(\'blots/embed\');\r\n\r\nclass EmojiBlot extends Embed {\r\n    // Customized version of src/format-emoji-blot.js\r\n    // ...\r\n}\r\n\r\nEmojiBlot.blotName = \'emoji\';\r\nEmojiBlot.tagName = \'span\';\r\n\r\nQuill.register({\r\n    \'formats/emoji\': EmojiBlot\r\n}, true);\r\n```\r\n\r\n## Contributing\r\n\r\nPlease check out our [contributing guidelines](CONTRIBUTING.md).\r\n)'
