b'# &lt;image-crop-element&gt;\n\nThis loads an image and creates a crop area that user can modify. An event is fired with x, y, width, and height as the crop area changes.\n\n## Installation\n\n```\n$ npm install --save @github/image-crop-element\n```\n\n## Usage\n\n### Plain\n\n```html\n<image-crop src="/avatar.jpg"></image-crop>\n```\n\n### With loading state\n\n```html\n<image-crop src="/avatar.jpg">\n  <img src="spinner.gif" alt="" data-loading-slot>\n</image-crop>\n```\n\n### With autoupdate inputs\n\n```html\n<image-crop src="/avatar.jpg">\n  <input type="hidden" data-image-crop-input="x" name="x">\n  <input type="hidden" data-image-crop-input="y" name="y">\n  <input type="hidden" data-image-crop-input="width" name="width">\n  <input type="hidden" data-image-crop-input="height" name="height">\n</image-crop>\n```\n\n### Listen to the change event\n\n```javascript\ndocument.addEventListener(\'image-crop-change\', function (event){\n  console.log(\n    \'Crop area changed.\',\n    event.detail.x,\n    event.detail.y,\n    event.detail.width,\n    event.detail.height\n  )\n})\n```\n\n## Browser support\n\nBrowsers without native [custom element support][support] require a [polyfill][]. Legacy browsers require various other polyfills. See [`examples/index.html`][example] for details.\n\n[example]: https://github.com/github/image-crop-element/blob/57080ad88d26e05b42fa10a95470da8035f53967/examples/polyfill.html#L17-L22\n\n- Chrome\n- Firefox\n- Safari\n- Microsoft Edge\n\n[support]: https://caniuse.com/#feat=custom-elementsv1\n[polyfill]: https://github.com/webcomponents/custom-elements\n'
