b'# &lt;clipboard-copy&gt; element\n\nCopy element text content or input values to the clipboard.\n\n## Installation\n\n```\n$ npm install --save @github/clipboard-copy-element\n```\n\n## Usage\n\n```js\nimport \'@github/clipboard-copy-element\'\n```\n\n```html\n<clipboard-copy for="blob-path" class="btn btn-sm BtnGroup-item">\n  Copy path\n</clipboard-copy>\n<div id="blob-path">src/index.js</div>\n```\n\n## Data sources\n\n### Attribute\n\n```html\n<clipboard-copy value="src/index.js">Copy</clipboard-copy>\n```\n\n### Element content\n\n```html\n<clipboard-copy for="blob-path">Copy</clipboard-copy>\n<div id="blob-path">src/index.js</div>\n```\n\n### Form input\n\n```html\n<clipboard-copy for="blob-path">Copy</clipboard-copy>\n<input id="blob-path" value="src/index.js">\n```\n\n### Hyperlink href\n\n```html\n<clipboard-copy for="blob-path">Copy full URL</clipboard-copy>\n<a id="blob-path" href="/path/to#my-blob">Link text will not be copied</a>\n```\n\n## Events\n\nAfter copying to the clipboard, a `clipboard-copy` event is dispatched from\nthe `<clipboard-copy>` element:\n\n```js\ndocument.addEventListener(\'clipboard-copy\', function(event) {\n  const button = event.target\n  button.classList.add(\'highlight\')\n})\n```\n\n## Browser support\n\nBrowsers without native [custom element support][support] require a [polyfill][].\n\n- Chrome\n- Firefox\n- Safari\n- Microsoft Edge\n\n[support]: https://caniuse.com/#feat=custom-elementsv1\n[polyfill]: https://github.com/webcomponents/custom-elements\n\n## Development\n\n```\nnpm install\nnpm test\n```\n\n## License\n\nDistributed under the MIT license.
