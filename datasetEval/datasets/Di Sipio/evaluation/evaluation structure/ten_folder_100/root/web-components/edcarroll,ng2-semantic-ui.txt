It provides Angular component versions of the Semantic UI modules, so that you don\'t need to add jQuery to your app.\n\nNote that only Semantic UI elements that use jQuery are recreated here - those written purely in CSS aren\'t included as they can be used in Angular apps already.\n\n## Installation & Usage\n\nSee the [Documentation](https://edcarroll.github.io/ng2-semantic-ui) for installation instructions and extensive examples.\n\n## Dependencies\n\n* [Angular](https://angular.io) (^6.0.0)\n* [Semantic UI CSS](http://semantic-ui.com/) (^2.3.1) (jQuery is **not** required)\n\n## Component
