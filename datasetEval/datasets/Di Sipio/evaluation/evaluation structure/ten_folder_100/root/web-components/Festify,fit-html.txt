Yet, the total size of the framework is below 3KB, including dependencies.\n\n## Small Example\n\nYou need the following:\n```js\nimport { connect, withStore } from \'fit-html\';\nimport { html } from \'lit-html/lib/lit-extended\';\nimport { createStore } from \'redux\';\n```\n\nSet up redux store:\n```js\nconst todos = (state = [], action) => {\n  switch (action.type) {\n    case \'ADD_TODO\':\n      return state.concat([action.text]);\n    default:\n      return state;\n  }\n};\n\nconst store = createStore(todos, [\'Use Redux\']);\n```\n\nDefine actions and view:\n```js\nfunction addTodo() {\n  return {\n    type: \'ADD_TODO\',\n    text: `Hello ${Math.random()}`\n  };\n}\n\nconst render = ({ addTodo, todos }) => html`\n  <ul>\n    ${todos.map(text => html`<li>${text}</li>`)}\n  </ul>\n\n  <button on-click="${addTodo}">\n    Add\n  </button>\n`;\n\nconst TodosApp = connect(\n  state => ({ todos: state }),\n  { addTodo }\n)(render);\n\n// Define the custom element.\n//\n// The withStore mixin is only required for the root element of your\n// app. All other \xf0\x9f\x92\xaa-elements will get the redux store from that element.\ncustomElements.define(\'todo-app\', withStore(store)(TodosApp));\n```\n\n`index.html`:\n```html\n<html>\n  <head>\n    <title>My cool \xf0\x9f\x92\xaa-html app</title>\n  </head>\n  <body>\n    <todo-app></todo-app>\n  </body>\n</html>\n```\n\nPlease see https://github.com/Festify/fit-html-demo for more and larger examples.\n\n## Compatibility\n\n\xf0\x9f\x92\xaa-html is written for use with evergreen browsers.
