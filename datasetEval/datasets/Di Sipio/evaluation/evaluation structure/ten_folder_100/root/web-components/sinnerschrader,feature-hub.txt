b'# Feature Hub\n\nThe **Feature Hub** is an opinionated JavaScript implementation of the micro\nfrontends approach to creating scalable web applications with multiple teams and\ndifferent technologies.\n\n## Documentation\n\nFor more information, please refer to the [documentation][documentation].\n\n## Contributing\n\nThe main purpose of this monorepo is to further develop the Feature Hub. It is\ndeveloped in the open on GitHub, and we are grateful to the community for\ncontributing bugfixes and improvements.\n\nTo get started, install the dependencies and run the tests:\n\n```sh\nyarn && yarn test\n```\n\n### Code of Conduct\n\nPlease note that this project is released with a [Contributor Code of\nConduct][code-of-conduct]. By participating in this project you agree to abide\nby its terms.\n\n### Development Scripts\n\n- `yarn watch:test` \xe2\x80\x94 Watches all tests.\n  - `yarn watch:test:unit` \xe2\x80\x94 Watches only unit tests.\n  - `yarn watch:test:integration` \xe2\x80\x94 Watches only integration tests.\n- `yarn watch:compile` \xe2\x80\x94 Watches all sources.\n- `yarn watch:demo <demo-name>` \xe2\x80\x94 Watches the given\n  [demo][demos-package-getting-started].\n- `yarn watch:website` \xe2\x80\x94 Watches the website.\n- `yarn test` \xe2\x80\x94 Executes all tests.\n- `yarn compile` \xe2\x80\x94 Compiles all sources.\n- `yarn lint` \xe2\x80\x94 Lints all sources.\n- `yarn verify` \xe2\x80\x94 Verifies non-functional requirements (used on CI).\n- `yarn format` \xe2\x80\x94 Formats all files.\n- `yarn sort-package-jsons` \xe2\x80\x94 Sorts all `package.json` files.\n\n### Publishing a New Release\n\nInstead of letting the CI automatically publish on every master merge, the\nFeature Hub package releases are triggered manually. To create a new semantic\nnpm release for all Feature Hub packages, core team members must trigger a\ncustom [Travis CI][travis-ci] build on the `master` branch. You should leave the\ncustom config and commit message fields empty.\n\n---\n\nCopyright (c) 2018-2019 SinnerSchrader Deutschland GmbH.
