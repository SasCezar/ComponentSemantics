Create a dart file under `lib/route/` (or just duplicate a file, e.g. `cp widgets_icon_ex.dart new_example.dart`);\n2. In the new file, create a class that extends MyRoute;\n3. Add const constructor, the convention is to use the file path as constructor\'s default parameter;\n4. (Optional) override getters: `title`, `description`, `links`;\n5. Override `buildMyRouteContent()`, try to make the code simple, as it\'ll be shown on phone screens;\n6. Open `lib/my_app_meta.dart`, import the new file at the beginning of file;\n7.
