b'Google Santa Tracker for Android \xf0\x9f\x8e\x85\xf0\x9f\xa4\xb6\n================================\n\n## About\n\n[Google Santa Tracker app for Android][play-store] is an educational and entertaining tradition\nthat brings joy to millions of children (and children at heart) across the world over the December\nholiday period.\n\nThe app is a companion to the [Google Santa Tracker][santa-web] website\n([repository here](https://github.com/google/santa-tracker-web)), showcasing unique platform\ncapabilities like Android Wear watchfaces, device notifications and more.\n\n![Village Screenshot](docs/village.png)\n\n## Features\n\n* A beautiful designed village\n* Exciting games like Penguin Swim and Rocket Sleigh\n* Use of Dynamic Feature Modules (each game is a separate module, fetched dynamically on first run)\n* Interactive Android Wear watchfaces (with sound!)\n* Videos, animations and more.\n\n## Building the app\n\nFirst up, Santa Tracker is powered by [Firebase][firebase], so you\'ll need to enable it\non your Google account over at the [Firebase console][fire-console]. Once you\'re in the\nconsole, follow these steps:\n\n * Create a new project\n * Add Firebase to your Android app\n * Package name: `com.google.android.apps.santatracker.debug`\n * Debug signing certificate can be blank, or follow the instructions in the tooltip to find yours.\n * Save the `google-services.json` file to the `santa-tracker/` directory\n\nNow you should be able to plug your phone in (or fire up an emulator) and run:\n\n    ./gradlew santa-tracker:installDebug\n\nAlternatively, import the source code into Android Studio (File, Import Project).\n\nNote: You\'ll need Android SDK version 28.
