Security Rules\n\nIn the Firebase console, go to Database, select "Realtime Database" -> Rules, and paste the rules below:\n\n```\n{\n  "rules": {\n    ".read": false,\n    ".write": false,\n    "users": {\n      "$uid": {\n        ".read": "$uid === auth.uid || root.child(\'users/\'+auth.uid).child(\'roles\').hasChildren([\'ADMIN\'])",\n        ".write": "$uid === auth.uid || root.child(\'users/\'+auth.uid).child(\'roles\').hasChildren([\'ADMIN\'])"\n      },\n      ".read": "root.child(\'users/\'+auth.uid).child(\'roles\').hasChildren([\'ADMIN\'])",\n      ".write": "root.child(\'users/\'+auth.uid).child(\'roles\').hasChildren([\'ADMIN\'])"\n    },\n    "messages": {\n      ".indexOn": ["createdAt"],\n      "$uid": {\n        ".write": "data.exists() ? data.child(\'userId\').val() === auth.uid : newData.child(\'userId\').val() === auth.uid"\n      },\n      ".read": "auth != null",\n      ".write": "auth != null",\n    },\n  }\n}\n```\n\n## Setup
