b'# Auth with React Router V4 and Firebase V3\nThis is an example repo for authenticating with Firebase and React Router.\n\nFor more info, visit [Protected routes and authentication with React Router v4](https://tylermcginnis.com/react-router-protected-routes-authentication/)\n\n*Using React 15.4.0, React Router 4, and Firebase 3.6.1*\n\n#### Features:\n* Protected Routes with React Router\n* Register new users with Firebase\n* Add new users to ```/users``` in your Firebase database\n* Login/Logout Functionality\n* Simple Boostrap UI\n\n#### Instructions:\n* Swap out the firebase config in ```config/constants``` with your own\n* ```npm install```\n* ```npm start```\n* Visit ```localhost:3000```\n\n#### Try it out in a [Docker](https://www.docker.com/) container:\n* Run a container running the prod version: `docker run -p 8080:80 -d allthethings/react-router-firebase-auth`\n* **Or** build a dev version, locally: `docker build -t react-router-firebase-auth .`\n* Then run the image (listens for changes to src): `docker run -v "$(pwd)/src:/code/src" -p 3000:3000 -d --name react-router-firebase-auth react-router-firebase-auth`\n'
