CDN\n\n```html\n<!DOCTYPE html>\n<html>\n<head>\n  <link href="https://cdn.jsdelivr.net/npm/vuesax/dist/vuesax.css" rel="stylesheet">\n  <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no, minimal-ui">\n</head>\n<body>\n  <div id="app">\n    <vs-button vs-type="filled">Hello World</vs-button>\n  </div>\n\n  <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>\n  <script src="https://cdn.jsdelivr.net/npm/vuesax/dist/vuesax.umd.js"></script>\n  <script>\n    new Vue({\n      el: \'#app\'\n    })\n  </script>\n</body>\n</html>\n```\n\n## CDN Links\n\n- https://cdn.jsdelivr.net/npm/vuesax/dist/vuesax.umd.js\n- https://cdn.jsdelivr.net/npm/vuesax/dist/vuesax.css\n\n# Install inside a NPM project\n``` bash\n# npm\nnpm install vuesax\n```\n\n``` bash\n# yarn\nyarn add vuesax\n```\n\n## Use\n\n### All components\n\n```javascript\nimport Vue from \'vue\'\nimport Vuesax from \'vuesax\'\n\nimport \'vuesax/dist/vuesax.css\'\nVue.use(Vuesax)\n```\n\n### Or use individual components:\n\n```javascript\nimport Vue from \'vue\'\nimport { vsButton, vsSelect, vsPopup } from \'vuesax\'\nimport \'vuesax/dist/vuesax.css\'\n\nVue.use(vsButton)\nVue.use(vsSelect)\nVue.use(vsPopup)\n```\n\n## Contribution\nPlease make sure to read the [Contributing
