It helps with the following:\n* Organizing your source text for translation\n* Composing grammatically correct translatable UI\n* Eliminating verbose boilerplate for generating UI\n\n## Examples\n[See our demo here](demo-app/src/example/Example.react.js)\n\n## Requirements\n* [Node.js](https://nodejs.org/)\n* [Yarn](https://yarnpkg.com/)\n* [Babel](https://babeljs.io/)\n\n## Building the fbt library\n```\ngit clone git@github.com:facebookincubator/fbt.git\ncd fbt\nyarn install\n```\n\n**NOTE:** if you make changes to the fbt runtime that you\'d like to\ntest in the demo-app, be sure to run this command to rebuild the JS code:\n\n```\nyarn build-runtime\n```\n\n## Using fbt\nSee how to use the source directly with Babel and Webpack in [our demo-app](demo-app#babelwebpackreact-oss-fbt-demo).\n\n## How FBT works\nFBT works by transforming your `<fbt>` and `fbt(...)` constructs via\nBabel plugins.  These plugins serve to extract strings from source and\nlookup translated payloads generated at build-time.  FBT creates tables\nof all possible variations for the given fbt phrase and accesses this\nat runtime.\n\n## Full documentation\nhttps://facebookincubator.github.io/fbt\n\n## Join the fbt community\n* [Website](https://facebookincubator.github.io/fbt)\n* [Facebook group](https://www.facebook.com/groups/498204277369868)\n* [Discord #fbt channel in reactiflux](https://discord.gg/cQvXZr5)\n* [Twitter](https://twitter.com/fbt_js)\n\nSee the [CONTRIBUTING](CONTRIBUTING.md) file for how to help out.\n\n## Change log\nSee [CHANGELOG](CHANGELOG.md).\n\n## List of npm modules published from this repo:\n\n- [fbt](https://www.npmjs.com/package/fbt) (client-side)\n  - See [package.json](packages/fbt/package.json) config\n- [babel-plugin-fbt](https://www.npmjs.com/package/babel-plugin-fbt) (server-side)\n  - See [package.json](packages/babel-plugin-fbt/package.json) config\n- [babel-plugin-fbt-runtime](https://www.npmjs.com/package/babel-plugin-fbt-runtime) (server-side)\n  - See [package.json](packages/babel-plugin-fbt-runtime/package.json) config\n- [fb-babel-plugin-utils](https://www.npmjs.com/package/fb-babel-plugin-utils) (server-side, used by `babel-plugin-fbt-runtime`)\n  - See [package.json](packages/fb-babel-plugin-utils/package.json) config\n\n## How to release a new npm version\n\n```\n# Go to the module\'s folder (where package.json is located)\ncd fbt-runtime/\n# or\n# cd packages/babel-plugin-fbt/\n# cd packages/babel-plugin-runtime/\n\n# publishes to npm and push new git tag to Github\nyarn publish_to_npm_latest\n```\n\n## License\nFBT is MIT licensed, as found in the [LICENSE](LICENSE) file.\n'
