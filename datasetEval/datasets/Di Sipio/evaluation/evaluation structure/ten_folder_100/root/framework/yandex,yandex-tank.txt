b"# Yandex Tank [![Build Status](https://travis-ci.org/yandex/yandex-tank.svg?branch=master)](https://travis-ci.org/yandex/yandex-tank) [![Gitter](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/yandex/yandex-tank?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)\n\n#### Yandextank 1.9.0 now available! Read [release notes](https://github.com/yandex/yandex-tank/wiki/Release-1.9.0).\n\nYandex.Tank is an extensible open source load testing tool for advanced linux users which is especially good as a part of an automated load testing suite\n\n![Quantiles chart example](/logos/screen.png)\n\n## Main features\n* different load generators supported:\n  * Evgeniy Mamchits' [phantom](https://github.com/yandex-load/phantom) is a very fast (100 000+ RPS) shooter written in C++ (default)\n  * [JMeter](http://jmeter.apache.org/) is an extendable and widely known one\n  * BFG is a Python-based generator that allows you to write your load scenarios in Python\n  * experimental Golang generator: [pandora](https://github.com/yandex/pandora)\n* performance analytics backend service: [Overload](http://overload.yandex.net/). Store and analyze your test results online\n* several ammo formats supported like plain url list or access.log\n* test autostop plugin: stop your test when the results have became obvious and save time\n* customizable and extendable monitoring that works over SSH\n\n## Documentation\n- [Installation](http://yandextank.readthedocs.org/en/latest/install.html)\n\n- Rest of [documentation](https://yandextank.readthedocs.org/en/latest/)\n\n- [Stackoverflow](https://stackoverflow.com/) \xe2\x80\x93 use `load-testing` + `yandex` tags\n\n## Get help\nChat with authors and other performance specialists: [![Gitter](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/yandex/yandex-tank?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)\n\n## See also\n- [Overload\xf0\x9d\x9b\x83](https://overload.yandex.net/) \xe2\x80\x93 performance analytics server\n\n- Evgeniy Mamchits' [phantom](https://github.com/yandex-load/phantom) \xe2\x80\x93 phantom scalable IO engine\n\n- [Vagrant environment](https://github.com/c3037/yandex-tank) with Yandex.Tank by Dmitry Porozhnyakov\n\n- [BlazeMeter Sense](https://sense.blazemeter.com) \xe2\x80\x93 performance testing analytics by [BlazeMeter](https://www.blazemeter.com/).
