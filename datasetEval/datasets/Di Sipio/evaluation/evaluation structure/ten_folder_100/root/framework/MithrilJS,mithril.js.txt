It\'s small (<!-- size -->9.79 KB<!-- /size --> gzipped), fast and provides routing and XHR utilities out of the box.\n\nMithril is used by companies like Vimeo and Nike, and open source platforms like Lichess \xf0\x9f\x91\x8d.\n\nMithril supports IE11, Firefox ESR, and the last two versions of Firefox, Edge, Safari, and Chrome. No polyfills required. \xf0\x9f\x91\x8c\n\n## Installation\n\n### CDN\n\n```html\n<!-- Development: whichever you prefer -->\n<script src="https://unpkg.com/mithril/mithril.js"></script>\n<script src="https://cdn.jsdelivr.net/npm/mithril/mithril.js"></script>\n\n<!-- Production: whichever you prefer -->\n<script src="https://unpkg.com/mithril/mithril.min.js"></script>\n<script src="https://cdn.jsdelivr.net/npm/mithril/mithril.min.js"></script>\n```\n\n### npm\n\n```bash\nnpm install mithril --save\n```\n\nThe ["Getting started" guide](https://mithril.js.org/#getting-started) is a good place to start learning how to use mithril.\n\nTypeScript type definitions are available from DefinitelyTyped.
