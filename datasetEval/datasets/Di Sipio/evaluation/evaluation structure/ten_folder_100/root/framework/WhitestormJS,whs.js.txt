This one creates a _scene_, _camera_ and _renderer_ - we add the following modules to the App.\n\n```js\nconst app = new WHS.App([\n  new WHS.ElementModule(), // Apply to DOM.\n  new WHS.SceneModule(), // Create a new THREE.Scene and set it to app.\n\n  new WHS.DefineModule(\'camera\', new WHS.PerspectiveCamera({ // Apply a camera.\n    position: new THREE.Vector3(0, 0, 50)\n  })),\n\n  new WHS.RenderingModule({bgColor: 0x162129}), // Apply THREE.WebGLRenderer\n  new WHS.ResizeModule() // Make it resizable.\n]);\n\napp.start(); // Run app.\n```\n\n<a
