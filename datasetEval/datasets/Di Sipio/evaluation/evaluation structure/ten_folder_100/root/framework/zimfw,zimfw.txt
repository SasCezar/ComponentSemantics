b'Zsh IMproved FrameWork\n======================\n\n<div align="center">\n  <a href="https://github.com/zimfw/zimfw">\n    <img width=650px src="https://zimfw-static.s3.wasabisys.com/zimfw_banner.png">\n  </a>\n</div>\n\nWhat is Zim?\n------------\nZim is a Zsh configuration framework with [blazing speed][speed] and modular extensions.\n\nZim is very easy to customize, and comes with a rich set of modules and features without compromising on speed or functionality!\n\nWhat does Zim offer?\n-----------------\nIf you\'re here, it means you want to see the cool shit Zim can do. Check out the [available modules][modules]!\n\nBelow is a brief showcase of Zim\'s features.\n\n### Speed\nFor a speed comparison between Zim and other frameworks, see [this wiki entry][speed].\n\n### Themes\n\nTo preview some of the available themes, check the [themes wiki page][themes].\n\n### Fish-shell history navigation\n![history-substring-search][fish_shell]\n\n### Syntax highlighting\n![syntax-highlighting][syntax_highlighting]\n\n### And much more!\nZim has many modules! Enable as many or as few as you\'d like.\n\nInstallation\n------------\nInstalling Zim is easy. If you have a different shell framework installed (like oh-my-zsh or prezto),\n*uninstall those first to prevent conflicts*. It can be installed manually by following the instructions below:\n\n1. Start a Zsh shell:\n\n       zsh\n\n2. Clone the repository:\n\n       git clone --recursive https://github.com/zimfw/zimfw.git ${ZDOTDIR:-${HOME}}/.zim\n\n3. Paste this into your terminal to prepend the initialization templates to your configs:\n\n       for template_file in ${ZDOTDIR:-${HOME}}/.zim/templates/*; do\n         user_file="${ZDOTDIR:-${HOME}}/.${template_file:t}"\n         cat ${template_file} ${user_file}(.N) > ${user_file}.tmp && mv ${user_file}{.tmp,}\n       done\n\n4. Set Zsh as the default shell:\n\n       chsh -s =zsh\n\n5. Open a new terminal and finish optimization (this is only needed once, hereafter it will happen upon desktop/tty login):\n\n       source ${ZDOTDIR:-${HOME}}/.zlogin\n\n6. You\'re done! Enjoy your Zsh IMproved! Take some time to read about the [available modules][modules] and tweak your `.zshrc` file.\n\nUpdating\n--------\n\nTo update Zim, run:\n\n    zmanage update\n\nFor more information about the `zmanage` tool, run `zmanage help`.\n\nUninstalling\n------------\n\nThe best way to remove Zim is to manually delete `~/.zim`, `~/.zimrc`, and\nremove the initialization lines from your `~/.zshrc` and `~/.zlogin`.\n\nHowever, there are some **experimental** convenience functions to remove Zim:\n\n**NOTE: This functionality is experimental!**\n\nTo remove Zim, run:\n\n    zmanage remove\n\n**NOTE: This functionality is experimental!**\n\n\n[fish_shell]: https://zimfw-static.s3.wasabisys.com/zimfw_history-substring-search.gif\n[syntax_highlighting]: https://zimfw-static.s3.wasabisys.com/zimfw_syntax-highlighting.gif\n[speed]: https://github.com/zimfw/zimfw/wiki/Speed\n[modules]: https://github.com/zimfw/zimfw/wiki/Modules\n[themes]: https://github.com/zimfw/zimfw/wiki/Themes\n'
