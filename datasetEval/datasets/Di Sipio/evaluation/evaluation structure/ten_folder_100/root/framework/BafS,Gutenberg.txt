How to use\n\nSimply include the right stylesheet(s) in your html and load it only for a printer.\nGutenberg.css is the base stylesheet but there are themes available in the `themes` folder.\n\nExample with Gutenberg and "old style" theme :\n\n```HTML\n<link rel="stylesheet" href="dist/gutenberg.css" media="print">\n<link rel="stylesheet" href="dist/themes/oldstyle.css" media="print"> <!-- optional -->\n```\n\n![comparison](https://i.imgur.com/tL5cHhn.png)\n\n> Comparison between standard print (left) and Gutenberg (middle, Modern style and right, Old style)\n\n## npm\n\nGutenberg is available on npm\n\n```\nnpm install gutenberg-css\n```\n\n(or `yarn add gutenberg-css` for yarn users)\n\n## CDN\n\nYou can also use the unpkg service as a *CDN*.\n\n```HTML\n<link rel="stylesheet" href="https://unpkg.com/gutenberg-css@0.4" media="print">\n<link rel="stylesheet" href="https://unpkg.com/gutenberg-css@0.4/dist/themes/oldstyle.min.css" media="print">\n```\n\n\n# What does the framework do ?\n\n### Hide elements\n\nTo hide elements to be printed you can simply add the class `no-print`.\n\n### Force break page\n\nGutenberg provides two ways to break a page, the class `page-break-before` will to break before and `page-break-after` to break after.\n\nExample:\n\n```HTML\n<!-- The title will be on a new page -->\n<h1 class="page-break-before">My title</h1>\n\n<p class="page-break-after">I will break after this paragraph</p>\n<!-- Break here, the next paragraph will be on a new page -->\n<p>I am on a new page</p>\n```\n\n### Not reformat links or acronym\n\nIf you do not want to reformat the links, acronym or abbreviation to show the full url or title, you\ncan use the class `no-reformat`.\n\n### Force to print background\n\nTo force backgrounds to be printed (can be useful when you "print" a pdf), add this CSS (compatible with Safari and Chrome) :\n\n```CSS\n-webkit-print-color-adjust: exact;\n        print-color-adjust: exact;\n```\n\n## Dev\n\n - `npm i` to install the dependencies\n - `npm run watch` to "watch" the scss folder and compile to css\n - `npm run build` to compile gutenberg to css\n'
