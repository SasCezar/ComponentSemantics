b'# Trip.js [![Build Status](https://travis-ci.org/EragonJ/Trip.js.png?branch=master)](https://travis-ci.org/EragonJ/Trip.js) [![Join the chat at https://gitter.im/EragonJ/Trip.js](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/EragonJ/Trip.js?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge) [![CDNJS](https://img.shields.io/cdnjs/v/Trip.js.svg)](https://cdnjs.com/libraries/Trip.js)\n\n![Trip.js](https://raw.github.com/EragonJ/Trip.js/master/public/img/logo-tiny.png "Trip.js")\n\nTrip.js is a plugin that can help you customize a tutorial trip easily with more flexibilities. Right now Trip.js support lots of useful stuffs like **keyboard binding**, **animations**, **changing themes** ... etc. If you want to highly customize your own tutorial trip, then Trip.js is definitely worth to give it a try !\n\n## Browser support\n\nTrip.js has been tested and work well on `IE 9+`, `Chrome 32+`, `Firefox 32+` and `Safari 9+`\n\n## Version\n\n3.3.5\n\n## How to\n\n### Use this library\n\nPlease check [website](http://eragonj.github.io/Trip.js), [setup](http://eragonj.github.io/Trip.js/documentations/setup/index.html), [configuration](http://eragonj.github.io/Trip.js/documentations/configuration/index.html) and [API](http://eragonj.github.io/Trip.js/documentations/api/index.html) :P\n\n### Run tests\n\n```bash\ngrunt test # run all QUnit tests\n```\n\n## Development\n\n### Environment\n\n1. use `node 5.1.1`\n2. use `npm 2.15.6`\n3. `npm i # to instasll dependencies`\n\n### Build\n\n```bash\ngrunt build # build js and CSS files\n```\n\n```bash\ngrunt build-js # build only js files\n```\n\n```bash\ngrunt build-css # build only CSS files\n```\n\n### Bump version\n\n```bash\ngrunt bumpversion --oldv=VERSION_FROM --newv=VERSION_TO # below is an example\ngrunt bumpversion --oldv=3.1.7 --newv=3.1.8 # change version number in src\n```\n\n## Special thanks\n\n* Everyone who is engaged in this project.\n* [Animate.css](http://daneden.github.io/animate.css/)\n\n## Support Trip.js !\n\nIf you do love Trip.js, feel free to buy me some beers ! :beers:\n\nMy bitcoin wallet : **3Ar6wuyrta9AciZAYMFi3QCByXF6S5HneT**\n\n## Author\n\nChia-Lung, Chen (EragonJ)\n\n## License\n\nMIT\n'
