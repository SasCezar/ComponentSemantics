b'# jQuery Print Plugin\n\njQuery.print is a plugin for printing specific parts of a page\n\n## Usage\n\nInclude it in your HTML after importing jQuery, like:\n\n```html\n\t<script type="text/JavaScript" src="path/to/jquery.print.js"></script>\n```\n\t\nUse it like:\n\n```js\n\t$("#myElementId").print(/*options*/);\n```\n\t\nor\n\n```js\n\t$.print("#myElementId" /*, options*/);\n```\n\nYou can submit the options object like:\n\n```js\n\t$("#myElementId").print({\n        \tglobalStyles: true,\n        \tmediaPrint: false,\n        \tstylesheet: null,\n        \tnoPrintSelector: ".no-print",\n        \tiframe: true,\n        \tappend: null,\n        \tprepend: null,\n        \tmanuallyCopyFormValues: true,\n        \tdeferred: $.Deferred(),\n        \ttimeout: 750,\n        \ttitle: null,\n        \tdoctype: \'<!doctype html>\'\n\t});\n```\n\nCurrently this plugin supports the following options:\n\n#### globalStyles\n\n - Default: `true`  \n - Acceptable-Values: Boolean  \n - Function: Whether or not the styles from the parent document should be included\n\n#### mediaPrint\n\n - Default: `false`  \n - Acceptable-Values: Boolean  \n - Function: Whether or not link tags with media=\'print\' should be included; Over-riden by the `globalStyles` option\n\n#### stylesheet\n\n - Default: `null`\n - Acceptable-Values: URL-string\n - Function: URL of an external stylesheet to be included\n\n#### noPrintSelector\n\n - Default: `".no-print"`\n - Acceptable-Values: Any valid `jQuery-selector`\n - Function: A selector for the items that are to be excluded from printing\n\n#### iframe\n\n - Default: `true`, creates a hidden iframe if no-vaild iframe selector is passed\n - Acceptable-Values: Any valid `jQuery-selector` or Boolean\n - Function: Whether to print from an iframe instead of a pop-up window; can take the `jQuery-selector` of an existing iframe as value\n\n#### append/prepend\n\n - Default: `null`\n - Acceptable-Values: Any valid `jQuery-selector` or HTML-text\n - Function: Adds custom HTML before (prepend) or after (append) the selected content\n\n#### manuallyCopyFormValues\n\n - Default: `true`\n - Acceptable-Values: Boolean\n - Function: Should it copy user-updated form input values onto the printed markup (this is done by manually iterating over each form element)\n\n#### deferred\n\n - Default: `$.Deferred()`\n - Acceptable-Values: Any valid `jQuery.Deferred` object\n - Function: A jQuery.Deferred object that is resolved once the print function is called. Can be used [to setup callbacks - see wiki](https://github.com/DoersGuild/jQuery.print/wiki/Using-the-deferred-option-to-set-up-a-callback-after-printing)\n\n#### timeout\n\n - Default: `750`\n - Acceptable-Values: Time in Milliseconds for `setTimeout`\n - Function: To change the amount of max time to wait for the content, etc to load before printing the element from the new window/iframe created, as a fallback if the [`load` event](https://developer.mozilla.org/en-US/docs/Web/Events/load) for the new window/iframe has not fired yet\n \n#### title\n\n - Default: `null`, uses the host page title\n - Acceptable-Values: Any single-line string\n - Function: To change the printed title\n\n#### doctype\n\n - Default: `\'<!doctype html>\'`\n - Acceptable-Values: Any valid doctype string\n - Function: To prepend a doctype to the printed document frame\n \n## Tested with\n\n### jQuery\n* [jQuery](http://jquery.com/) v. 1.7.2\n* [jQuery](http://jquery.com/) v. 1.9.1\n* [jQuery](http://jquery.com/) v. 2.2.0\n* [jQuery](http://jquery.com/) v. 3.1.1\n\n### Browsers\n* Google Chrome - v 20, 26, 48, 55\n* Internet Explorer - v 10, 11\n* Firefox - v 35\n\n## License\n[CC-BY](http://creativecommons.org/licenses/by/3.0/).\n\n## Demo\n[jQuery.print/demo](http://doersguild.github.io/jQuery.print/demo/)\n\n---------------------------------------\nLike our [work](http://doersguild.com)?
