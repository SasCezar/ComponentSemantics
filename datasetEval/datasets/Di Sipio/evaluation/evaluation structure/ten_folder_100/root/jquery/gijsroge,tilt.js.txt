b'[![Build Status](https://travis-ci.org/gijsroge/tilt.js.svg?branch=master)](https://travis-ci.org/gijsroge/tilt.js)\n\n# Tilt.js\nA tiny requestAnimationFrame powered 60+fps lightweight parallax tilt effect for jQuery. \n\nWeights just \xe2\x9a\x96**1.71kb Gzipped**\n![Tilt.js demo gif](http://gijsroge.github.io/tilt.js/tilt.js.gif)\n\n#### Take a look at the **[landing page](http://gijsroge.github.io/tilt.js/)** for demos.\n\n### Usage\n\n```html\n<!DOCTYPE html>\n<body>\n    <div data-tilt></div> <!-- Tilt element -->\n    <script src="jquery.js" ></script> <!-- Load jQuery first -->\n    <script src="tilt.js"></script> <!-- Load Tilt.js library -->\n</body>\n```\n\n### Options\n```js\nmaxTilt:        20,\nperspective:    1000,   // Transform perspective, the lower the more extreme the tilt gets.\neasing:         "cubic-bezier(.03,.98,.52,.99)",    // Easing on enter/exit.\nscale:          1,      // 2 = 200%, 1.5 = 150%, etc..\nspeed:          300,    // Speed of the enter/exit transition.\ntransition:     true,   // Set a transition on enter/exit.\ndisableAxis:    null,   // What axis should be disabled. Can be X or Y.\nreset:          true,   // If the tilt effect has to be reset on exit.\nglare:          false,  // Enables glare effect\nmaxGlare:       1       // From 0 - 1.\n```\n\n### Events\n```js\nconst tilt = $(\'.js-tilt\').tilt();\ntilt.on(\'change\', callback);  // parameters: event, transforms\ntilt.on(\'tilt.mouseLeave\', callback); // parameters: event\ntilt.on(\'tilt.mouseEnter\', callback); // parameters: event\n```\n\n### Methods\n```js\nconst tilt = $(\'.js-tilt\').tilt();\n\n// Destroy instance\ntilt.tilt.destroy.call(tilt);\n\n// Get values of instance\ntilt.tilt.getValues.call(tilt); // returns [{},{},etc..]\n\n// Reset instance\ntilt.tilt.reset.call(tilt);\n```\n\n### Install\n- **yarn:** `yarn add tilt.js`\n- **npm:** `npm install --save tilt.js`\n\n### CDN\n- https://cdnjs.cloudflare.com/ajax/libs/tilt.js/1.2.1/tilt.jquery.min.js\n- https://unpkg.com/tilt.js@1.2.1/dest/tilt.jquery.min.js\n\n### Alternatives\n- **Vanilla JS:** https://github.com/micku7zu/vanilla-tilt.js\n- **React:** https://github.com/jonathandion/react-tilt\n- **Polymer:** https://github.com/YingshanDeng/polymer-tilt\n'
