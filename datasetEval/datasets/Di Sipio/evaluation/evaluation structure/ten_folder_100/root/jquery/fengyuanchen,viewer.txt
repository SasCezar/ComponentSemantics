b'# Viewer\n\n[![Build Status](https://travis-ci.org/fengyuanchen/viewer.svg)](https://travis-ci.org/fengyuanchen/viewer) [![Downloads](https://img.shields.io/npm/dm/imageviewer.svg)](https://www.npmjs.com/package/imageviewer) [![Version](https://img.shields.io/npm/v/imageviewer.svg)](https://www.npmjs.com/package/imageviewer)\n\n> A simple jQuery image viewing plugin. As of v1.0.0, the core code of Viewer is replaced with [Viewer.js](https://github.com/fengyuanchen/viewerjs).\n\n- [Demo](https://fengyuanchen.github.io/viewer)\n- [Viewer.js](https://github.com/fengyuanchen/viewerjs) - JavaScript image viewer (**recommended**)\n- [jquery-viewer](https://github.com/fengyuanchen/jquery-viewer) - A jQuery plugin wrapper for Viewer.js (**recommended** for jQuery users to use this instead of Viewer)\n\n## Main\n\n```text\ndist/\n\xe2\x94\x9c\xe2\x94\x80\xe2\x94\x80 viewer.css\n\xe2\x94\x9c\xe2\x94\x80\xe2\x94\x80 Installation\n\n```shell\nnpm install imageviewer jquery\n```\n\nInclude files:\n\n```html\n<script src="/path/to/jquery.js"></script><!-- jQuery is required -->\n<script src="/path/to/viewer.js"></script>\n<link  href="/path/to/viewer.css" rel="stylesheet">\n```\n\n### Usage\n\nInitialize with `$.fn.viewer` method.\n\n```html\n<!-- a block container is required -->\n<div>\n  <img id="image" src="picture.jpg" alt="Picture">\n</div>\n\n<div>\n  <ul id="images">\n    <li><img src="picture-1.jpg" alt="Picture 1"></li>\n    <li><img src="picture-2.jpg" alt="Picture 2"></li>\n    <li><img src="picture-3.jpg" alt="Picture 3"></li>\n  </ul>\n</div>\n```\n\n```js\nvar $image = $(\'#image\');\n\n$image.viewer({\n  inline: true,\n  viewed: function() {\n    $image.viewer(\'zoomTo\', 1);\n  }\n});\n\n// Get the Viewer.js instance after initialized\nvar viewer = $image.data(\'viewer\');\n\n// View a list of images\n$(\'#images\').viewer();\n```\n\n## Options\n\nSee the available [options](https://github.com/fengyuanchen/viewerjs#options) of Viewer.js.\n\n```js\n$().viewer(options);\n```\n\n## Methods\n\nSee the available [methods](https://github.com/fengyuanchen/viewerjs#methods) of Viewer.js.\n\n```js\n$().viewer(\'method\', argument1, , argument2, ..., argumentN);\n```\n\n## Events\n\nSee the available [events](https://github.com/fengyuanchen/viewerjs#events) of Viewer.js.\n\n```js\n$().on(\'event\', handler);\n```\n\n## No conflict\n\nIf you have to use other plugin with the same namespace, just call the `$.fn.viewer.noConflict` method to revert to it.\n\n```html\n<script src="other-plugin.js"></script>\n<script src="viewer.js"></script>\n<script>\n  $.fn.viewer.noConflict();\n  // Code that uses other plugin\'s "$().viewer" can follow here.\n</script>\n```\n\n## Browser support\n\nIt is the same as the [browser support of Viewer.js](https://github.com/fengyuanchen/viewerjs#browser-support).
