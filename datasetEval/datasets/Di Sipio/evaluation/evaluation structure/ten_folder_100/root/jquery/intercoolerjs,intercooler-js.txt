b'<center>\n  <img src="http://i.imgur.com/CQh4tnm.png">\n</center>\n\n## Introduction\n\nIntercooler is a small (6.74KB gzipped) [jQuery](https://jquery.com/) or [zepto.js](http://intercoolerjs.org/docs.html#zepto) based library that allows you to add AJAX to \nyour application using HTML attributes.\n\nHere is an example:\n\n```html\n    <!-- When this button is clicked an AJAX POST request is sent to /example and the \n         response content is swapped in to the body of the button -->\n    <button ic-post-to="/example">\n        Click Me!\n    </button>\n```\n\nDespite this simplicity, intercooler supports quite a bit of functionality:\n\n* It allows you to [specify the UI event](http://intercoolerjs.org/docs.html#triggers) that triggers the AJAX request\n* It makes [progress indicators](http://intercoolerjs.org/docs.html#progress) very simple to add\n* It supports many [custom HTTP response headers](http://intercoolerjs.org/docs.html#responses) for things like client-side redirection\n* It has a [REST-ful dependency mechanism](http://intercoolerjs.org/docs.html#dependencies)\n* It has simple [AJAX history & back-button support](http://intercoolerjs.org/docs.html#history)\n* It provides a [rich event model](http://intercoolerjs.org/docs.html#events)\n* [And much more...](http://intercoolerjs.org/docs.html)\n\nThese features allow you to build modern web applications with little fuss, using a simple and intuitive REST-ful architecture that ensures good performance, excellent user experience and a minimum of complexity.\n\nIt also is very easy to incrementally retrofit intercooler into existing web applications to add AJAX functionality where\nit is most valuable.\n\nFull documentation is available on the [main intercooler website](http://intercoolerjs.org/)\n\n## Installing\n\nIntercooler can be downloaded or hot-linked [from the downloads page](http://intercoolerjs.org/download.html).\n\nOr installed via bower:\n\n```javascript\n     "dependencies": {\n        "intercooler-js" : "1.2.0"\n      }\n```\n\nIntercooler depends on [jQuery](https://jquery.com/) v1.10.0 or higher.\n\n## License\n\nIntercooler is licenced under the [MIT License](https://raw.githubusercontent.com/LeadDyno/intercooler-js/master/LICENSE)\n\n##
