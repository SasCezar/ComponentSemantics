b'# VvvebJs\n\n\n## Drag and drop website builder javascript library.\nBuilt with jQuery and Bootstrap 4.\n\n[Two panel Live Demo](http://www.vvveb.com/vvvebjs/editor.html)\n\n[One panel Live Demo](http://www.vvveb.com/vvvebjs/editor.html#no-right-panel)\n\n[![Slack](https://img.shields.io/badge/slack-join-blueviolet.svg)](https://join.slack.com/t/vvvebjs/shared_invite/enQtNTY5MTg4NTIyMzg5LWNiN2JhNWRjNmQ4NzYxMzhiZjA1NjIyYzU0NjAzZGVkZjE1OGY4ZGFmYjU3NjJhNzUxMzc4Yzc4NmMzODQ0YTE) [![npm](https://img.shields.io/npm/v/vvvebjs.svg)](https://www.npmjs.com/package/vvvebjs)\n\nUsing [Startbootstrap landing page](https://startbootstrap.com/template-overviews/landing-page/) for demo page and Bootstrap 4 components.\n\n<img src="http://www.vvveb.com/img/browser.png?v=1">\n\n### Features\n\n* Components and blocks/snippets drag and drop.\n* Undo/Redo operations.\n* One or two panels interface.\n* File manager and component hierarchy navigation.\n* Add new page.\n* Live code editor.\n* Image upload with example php script included.\n* Page download or export html or save page on server with example php script included.\n* Components/Blocks list search.\n* Bootstrap 4 components.\n* Youtube, Google maps, Charts.js etc widgets.\n\nBy default the editor comes with Bootstrap 4 and Widgets components and can be extended with any kind of components and inputs.\n\n## Usage\n\n```html\n<!-- jquery-->\n<script src="js/jquery.min.js"></script>\n<script src="js/jquery.hotkeys.js"></script>\n\n<!-- bootstrap-->\n<script src="js/popper.min.js"></script>\n<script src="js/bootstrap.min.js"></script>\n\n<!-- builder code-->\n<script src="libs/builder/builder.js"></script>\t\n<!-- undo manager-->\n<script src="libs/builder/undo.js"></script>\t\n<!-- inputs-->\n<script src="libs/builder/inputs.js"></script>\t\n<!-- components-->\n<script src="libs/builder/components-bootstrap4.js"></script>\t\n<script src="libs/builder/components-widgets.js"></script>\t\n\n\n<script>\n$(document).ready(function() \n{\n\tVvveb.Builder.init(\'demo/index.html\', function() {\n\t\t//load code after page is loaded here\n\t\tVvveb.Gui.init();\n\t});\n});\n</script>\n```\nFor editor html and components/input javascript templates check editor.html\n\nFor css changes edit scss/editor.scss and scss/_builder.scss\n\n## Documentation\n\nFor documentation check the
