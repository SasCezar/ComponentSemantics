b'jCarousel - Riding carousels with jQuery\n========================================\n\njCarousel is a jQuery plugin for controlling a list of items in horizontal or\nvertical order. It provides a full-featured and flexible toolset for navigating\nany HTML based content in a carousel-like fashion.\n\n**Note**: Version 0.3.x and 0.2.x are **not** compatible. Old versions can be\ndownloaded from the [release page](https://github.com/jsor/jcarousel/releases).\n\nDocumentation\n-------------\n\nThe documentation can be found [here](http://sorgalla.com/jcarousel/docs/).\n\n### Reference\n\n  * [Installation](http://sorgalla.com/jcarousel/docs/reference/installation.html)\n  * [Configuration](http://sorgalla.com/jcarousel/docs/reference/configuration.html)\n  * [API](http://sorgalla.com/jcarousel/docs/reference/api.html)\n  * [Usage](http://sorgalla.com/jcarousel/docs/reference/usage.html)\n  * [Events](http://sorgalla.com/jcarousel/docs/reference/events.html)\n\n### Plugins\n\nBy default, the jCarousel core only contains the minimum functionality needed to\ncreate carousel widgets. Specialized features are provided through plugins.\n\njCarousel comes with the following plugins:\n\n  * [Control Plugin](http://sorgalla.com/jcarousel/docs/plugins/control/)\n  * [Pagination Plugin](http://sorgalla.com/jcarousel/docs/plugins/pagination/)\n  * [Autoscroll Plugin](http://sorgalla.com/jcarousel/docs/plugins/autoscroll/)\n  * [ScrollIntoView Plugin](http://sorgalla.com/jcarousel/docs/plugins/scrollintoview/)\n\n#### Third-Party Plugins\n\n* [jcarouselSwipe](https://github.com/snake-345/jcarouselSwipe) - Adds support for user-friendly swipe gestures.\n* [jcarouselLazyLoading](https://github.com/snake-345/jcarouselLazyLoading) - Adds lazy loading to jcarousel.\n* [jcarouselFade](https://github.com/snake-345/jcarouselFade) - New method for support fade animation to jcarousel.\n* [jcarousel-status](https://github.com/fzoccara/jcarousel-status) - Display the status of your carousel.\n* [jcarousel-smoothscroll](https://github.com/aduth/jcarousel-smoothscroll) - Animates a jCarousel at a constant pace, regardless of varying item widths.\n\nDownload\n--------\n\nAll ready-to-use files are located in the [`dist/`](dist/) directory.\n\nThe [`jquery.jcarousel.js`](dist/jquery.jcarousel.js?raw=1) and\n[`jquery.jcarousel.min.js`](dist/jquery.jcarousel.min.js?raw=1) files contain\nthe core and all plugins concenated together.\n\nAll files are also available from [CDNJS](http://cdnjs.com/libraries/jcarousel).\n\nExamples\n--------\n\nYou can find some example implementations in the [`examples/`](examples/)\ndirectory. You can also browse the [functional tests](test/functional/) to get\nan impression of how to setup jCarousel.\n\nCredits\n-------\n\njCarousel is written on top of [jQuery](http://jquery.com) and was originally\ninspired by the [Carousel Component](http://billwscott.com/carousel/) by\n[Bill Scott](http://looksgoodworkswell.com).\n\nLicense\n-------\n\nCopyright (c) 2006-2018 Jan Sorgalla.\nReleased under the [MIT](LICENSE) license.\n'
