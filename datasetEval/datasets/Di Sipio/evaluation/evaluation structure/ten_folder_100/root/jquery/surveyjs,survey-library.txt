started\n\nInstall the library using npm.\n\nAngular2 version:\n\n```\nnpm install survey-angular\n```\n\njQuery version:\n\n```\nnpm install survey-jquery\n```\n\nKnockout version:\n\n```\nnpm install survey-knockout\n```\n\nReact version:\n\n```\nnpm install survey-react\n```\n\nVue version:\n\n```\nnpm install survey-vue\n```\n\nOr use Azure CDN:\n\n* survey.js from sources\n\nTo build library yourself:\n\n1.  **Clone the repo from GitHub**\n\n    ```\n    git clone https://github.com/surveyjs/survey-library.git\n    cd survey-library\n    ```\n\n2.  **Acquire build dependencies.** Make sure you have [Node.js](http://nodejs.org/) installed on your workstation. You need a version of Node.js greater than 6.0.0 and npm greater than 2.7.0. This is only needed to _build_ surveyjs from sources.\n\n    ```\n    npm install -g karma-cli\n    npm install\n    ```\n\n3.  **Build the library**\n\n    ```\n    npm run build_prod\n    ```\n\n    After that you should have the libraries (angular, jquery, knockout, react and vue) at \'packages\' directory.\n\n4.  **Run samples**\n\n    ```\n    npm start\n    ```\n\n    This command will run local http server at the http://localhost:7777\n    You can open http://localhost:7777/examples/knockout to view KnockoutJS samples, http://localhost:7777/examples/react to view ReactJS samples and so on\n\n5.  **Run unit tests**\n    ```\n    karma start\n    ```\n    This command will run unit tests using [Karma](https://karma-runner.github.io/0.13/index.html)\n\n### WordPress integration\n\n[SurveyJS WordPress plugin](https://wordpress.org/plugins/surveyjs/)\n\n## License\n\n[MIT license](https://github.com/surveyjs/survey-library/blob/master/LICENSE)\n'
