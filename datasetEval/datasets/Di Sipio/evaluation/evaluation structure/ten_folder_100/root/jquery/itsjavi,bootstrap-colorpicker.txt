Colorpicker\n\n<!--version-->\n\n[Bootstrap Colorpicker](https://github.com/farbelous/bootstrap-colorpicker/) is a modular color picker plugin for Bootstrap 4.\n\n[![Build Status](https://img.shields.io/travis/farbelous/bootstrap-colorpicker/master.svg?style=flat-square)](https://travis-ci.org/farbelous/bootstrap-colorpicker)\n[![npm](https://img.shields.io/npm/v/bootstrap-colorpicker.svg?style=flat-square)](https://www.npmjs.com/package/bootstrap-colorpicker)\n\n## Install\nYou can get the latest version in many different ways:\n\n- Downloading [a ZIP file from the releases](https://github.com/farbelous/bootstrap-colorpicker/releases)\n- Cloning using Git: `git clone https://github.com/farbelous/bootstrap-colorpicker.git`\n- Installing via NPM: `npm install bootstrap-colorpicker`\n- Installing via Yarn: `yarn add bootstrap-colorpicker`\n- Installing via Composer: `composer require itsjavi/bootstrap-colorpicker`\n\nNote that the `dist` files are only distributed via the NPM and Yarn installations.\n\nFor the rest methods, you will need to generate the files initializing the project with `yarn install`\nand then building the code using `npm run build`.\n\n## Versions\n\n<table class="table table-bordered table-striped">\n  <thead>\n    <tr>\n Bootstrap\ndependencies.\n\n\n## Basic example\n\n```html\n<!DOCTYPE html>\n<html lang="en">\n<head>\n  <meta charset="utf-8">\n  <link href="//cdn.rawgit.com/twbs/bootstrap/v4.1.3/dist/css/bootstrap.min.css" rel="stylesheet">\n  <link href="dist/css/bootstrap-colorpicker.css" rel="stylesheet">\n</head>\n<body>\n  <div class="jumbotron">\n      <h1>Bootstrap Colorpicker Demo</h1>\n      <input id="demo" type="text" class="form-control" value="rgb(255, 128, 0)" />\n  </div>\n  <script src="//code.jquery.com/jquery-3.3.1.js"></script>\n  <script src="//cdn.rawgit.com/twbs/bootstrap/v4.1.3/dist/js/bootstrap.bundle.min.js"></script>\n  <script src="dist/js/bootstrap-colorpicker.js"></script>\n  <script>\n    $(function () {\n      // Basic instantiation:\n      $(\'#demo\').colorpicker();\n      \n      // Example using an event, to change the color of the .jumbotron background:\n      $(\'#demo\').on(\'colorpickerChange\', function(event) {\n        $(\'.jumbotron\').css(\'background-color\', event.color.toString());\n      });\n    });\n  </script>\n</body>\n```\n\n## Contributions\n* [Issues](https://github.com/farbelous/bootstrap-colorpicker/issues)\n* [Pull
