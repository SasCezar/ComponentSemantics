b'# jQuery DrawSVG\n\nThis plugin uses the jQuery built-in animation engine to transition the `stroke` on every `<path>` inside the selected `<svg>` element, using `stroke-dasharray` and `stroke-dashoffset` properties.\n\n* Weighs less than 2KB minified and 800 bytes gzipped.\n* Easy to use.\n* Easing and stagger support.\n* Free!\n\n## Usage\n\nInclude jQuery DrawSVG after jQuery\n\n```html\n<script src="http://code.jquery.com/jquery-latest.min.js"></script>\n<script src="jquery.drawsvg.js"></script>\n```\n\nInitialize the plugin on the <svg> element you want to animate and store in a variable\n\n```js\nvar mySVG = $(\'#my_svg_element\').drawsvg();\n```\n\nRun the animation\n\n```js\nmySVG.drawsvg(\'animate\');\n```\n\nLook at the demos for more advanced usages.\n\n## Options\n\n| Option     | Type     | Default         | Description                                                                                                                                                  |\n| ---------- | -------- | --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |\n| `duration` | Integer  | `1000`          | The time to complete the animation of each path.                                                                                                             |\n| `stagger`  | Integer  | `200`           | Delay to start animating each individual path.                                                                                                               |\n| `easing`   | String   | `swing`         | Which easing function each path will use to transition. <br> Use [jQuery Easing Plugin](http://gsgd.co.uk/sandbox/jquery/easing/) for different easing functions. |\n| `reverse`  | Boolean  | `false`         | Direction that the line will be drawn.                                                                                                                       |\n| `callback` | Function | `function() {}` | A function to call once the animation has been completed. |\n\n## Demos\n\n[Simple usage](http://codepen.io/lcdsantos/pen/zvGXbr/)\n\n[Draw on scroll](http://codepen.io/lcdsantos/pen/zvGQYB/)\n\n[Callback example](http://codepen.io/lcdsantos/pen/vNNXrm/)\n\n[Animate mask path](http://codepen.io/lcdsantos/pen/xwwEJw/)\n\n'
