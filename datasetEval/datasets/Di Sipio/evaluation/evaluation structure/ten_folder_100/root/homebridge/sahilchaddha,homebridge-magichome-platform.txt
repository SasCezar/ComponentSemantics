b'# homebridge-magichome-platform\n\n[![NPM](https://nodei.co/npm/homebridge-magichome-platform.png?downloads=true&downloadRank=true&stars=true)](https://nodei.co/npm/homebridge-magichome-platform/)\n\n[![npm](https://img.shields.io/npm/dm/homebridge-magichome-platform.svg)](https://www.npmjs.com/package/homebridge-magichome-platform)\n[![npm](https://img.shields.io/npm/v/homebridge-magichome-platform.svg)](https://www.npmjs.com/package/homebridge-magichome-platform)\n[![CircleCI](https://circleci.com/gh/sahilchaddha/homebridge-magichome-platform.svg?style=svg)](https://circleci.com/gh/sahilchaddha/homebridge-magichome-platform)\n\n\n**A Homebridge plugin for devices running fade/strobe effects on the Magic Home Wi-Fi system.**\n\n## Description\n\nThis plugin will create LightBulbs in Homekit capable of turning on/off, change color, change hue, change saturation.\n\nThis plugin can also create preset patterns Switches (color cycle, fade, strobe).\n\nIts a great utility tool to set house mood to party/soothing with custom music.\n\nCan cycle through colors, sync all lights to strobe/fade.\n\n## Installation\n\n```shell\n    $ npm install -g --unsafe-perm homebridge\n    $ npm install -g --unsafe-perm homebridge-magichome-platform\n```\n\nEdit config.json. Refer to `config-sample.json`.\n\n## Available Presets Scenes\n\n```\n\tseven_color_cross_fade\n\tred_gradual_change\n\tgreen_gradual_change\n\tblue_gradual_change\n\tyellow_gradual_change\n\tcyan_gradual_change\n\tpurple_gradual_change\n\twhite_gradual_change\n\tred_green_cross_fade\n\tred_blue_cross_fade\n\tgreen_blue_cross_fade\n\tseven_color_strobe_flash\n\tred_strobe_flash\n\tgreen_strobe_flash\n\tblue_stobe_flash\n\tyellow_strobe_flash\n\tcyan_strobe_flash\n\tpurple_strobe_flash\n\twhite_strobe_flash\n\tseven_color_jumping\n```\n\n### Demo\n\n![Demo](https://raw.githubusercontent.com/sahilchaddha/homebridge-magichome-platform/master/demo.gif)\n\n## Compatible Devices\n\nAny devices created by Zengge and running on the Magic Home Wi-Fi (or other apps by the same developer such as LED Magic Color) app should work with this plugin. Some examples of compatible devices are:\n\n- [5 Channel Controller for RGB LED Strip](http://amzn.to/2eAljEV) `RGBWW`\n- [Magic UFO RGBW LED Strip controller](http://amzn.to/2eyoRdE)\n- [SuperLegends Wi-Fi smart bulb](http://amzn.to/2eCxq6a) `RGBW`\n- [Victorstar Wi-Fi Smart Light Bulb](http://amzn.to/2eCCM13)\n- [Flux Wi-Fi Light Bulb](http://amzn.to/2eCx3IC)\n- [Fen-Yi Light Bulb](http://amzn.to/2ehjP3s)\n- [Waterproof RGB LED Strips WIFI Controller](http://amzn.to/2eoDQZx) `RGBW`\n- [Eastlion RGB Wi-Fi Strip Controller](http://amzn.to/2eCF8wV)\n\n\n## Sample Config : \n\n```json\n{\n    "platforms": [\n        {\n            "platform": "MagicHome-Platform",\n            "debug": true,\n            "lights": [\n                {\n                    "name": "Kitchen LED Strip",\n                    "ip": "192.168.1.111",\n                    "setup": "RGBW",\n                    "purewhite": false,\n                    "timeout": 10000\n                },\n                {\n                    "name": "Living Room LED Strip",\n                    "ip": "DC4F22C5XXXX",\n                    "setup": "RGBWW",\n                    "purewhite": true\n                }\n            ],\n            "presetSwitches": [\n                {\n                    "name": "Kitchen Color Strobe Flash (Party)",\n                    "ips": {\n                        "192.168.1.111": "255,255,255"\n                    },\n                    "preset": "seven_color_strobe_flash",\n                    "speed": 60\n                },\n                {\n                    "name": "All Lights Cross Fade (Soothing)",\n                    "ips": {\n                        "192.168.1.111": "0,150,255",\n                        "DC4F22C5XXXX": "102, 255, 102"\n\n                    },\n                    "preset": "seven_color_cross_fade",\n                    "speed": 40,\n                    "shouldTurnOff": true\n                },\n                {\n                    "name": "Jungle Mood (Soothing)",\n                    "ips": {\n                        "192.168.1.111": "0,150,255",\n                        "DC4F22C5XXXX": "102, 255, 102"\n                    },\n                    "preset": "green_gradual_change",\n                    "speed": 40,\n                }\n            ]\n        }\n    ]\n}\n```\n\nPreset Switch Configuration\n\n`ips` must be a key-value object where `key` is MagicHome LED IP Address e.g. `192.168.1.11` or `DC4F22C5XXXX` MAC Address & `value` is default rgb color of the light. e.g. `"255,255,255" (White)`.\nTurning off Preset Pattern Switch , all lights will be reset to this color.\n\nYou can use MAC Address instead of IP Address as well. Please note format of MAC Address. It should be in capital letters and `:` should not be present. e.g. `DC4F22C5XXXX`\n\nDo note : While using MACS : This plugin auto discover connected lights on the network and map IP\'s to their corresponding MAC. Initially it can take time to discover all devices. All devices should be discovered and mapped in 60-120s. Once mapped IP & MACs are cached, and gets rediscovered every 30s to map new IP to the light. This feature is useful for people unable to assign Static IP to their lights.\n\nDefault Discovery Interval => 5 Minutes\nDefault Device Status Update Interval => 1 Minute\n\nSetting Device `timeout` to 0, will disable polling device for status update.\n\n## Lint\n\n```shell\n    $ npm run lint\n```\n\n## Need Help ?\n\nGet Slack Invite => `https://slackin-znyruquwmv.now.sh/`\n\nSlack Channel => `https://homebridgeteam.slack.com/messages/homebridge-magichome`\n\nSlack User => `@sahilchaddha`\n\n### Author\n\nSahil Chaddha\n\nmail@sahilchaddha.com\n'
