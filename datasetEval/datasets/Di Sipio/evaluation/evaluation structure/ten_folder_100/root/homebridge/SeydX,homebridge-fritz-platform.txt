b'<p align="center">\n    <img src="https://i.imgur.com/YAEtHb5.png">\n</p>\n\n\n\n# Fritz!Platform\n\nThis dynamic platform plugin allows almost full control of **AVM** hardware like:\n\n- Fritz!Box, \n- Fritz!Fon,\n- Fritz!WLAN Repeater,\n- Fritz!DECT Repeater,\n- Fritz!DECT Outlets,\n- Fritz!DECT Thermostats,\n- Comet!DECT Thermostats,\n- Fritz!Powerline Outlets (510, 540),\n- HAN-FUN sensors (e.g. Deutsche Telekom)\n\nAny system capable of running [Homebridge](https://github.com/nfarina/homebridge/) can be used to run **homebridge-fritz-platform**. The only need is network access to the device or program in question.\n\nAmong other things, **homebridge-fritz-platform** allows the switching of all WLAN frequencies, guest WLAN, answering machine, wake-up call, occupancy sensors, callmonitor as motion sensors with [FakeGato](https://github.com/simont77/fakegato-history) and much more!\n\n\n\n##
