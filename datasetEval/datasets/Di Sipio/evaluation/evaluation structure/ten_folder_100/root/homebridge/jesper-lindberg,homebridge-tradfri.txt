b'# homebridge-tradfri\n[![NPM Version](https://img.shields.io/npm/v/homebridge-tradfri.svg)](https://www.npmjs.com/package/homebridge-tradfri)\n\nIKEA Tr\xc3\xa5dfri Gateway plugin for [Homebridge](https://github.com/nfarina/homebridge).\n\n## Features\n- Automatic discovery of the Tr\xc3\xa5dfri gateway\n- Turn on/off lights\n- Adjust the brightness\n\n## To be implemented\n- Adjust color temperature (figuring out how to do this the best way)\n- Make the plugin work with third party HomeKit apps\n- Better error handling and logging\n\n## Installation\n\n1. Install Homebridge using: `npm install -g homebridge`\n2. Install this plugin using: `npm install -g homebridge-tradfri`\n3. Update your configuration file. See the sample below.\n\n## Sample Configuration\n\nThe PSK (Pre-Shared Key) can be found printed on the bottom of the IKEA Tr\xc3\xa5dfri Gateway.\nThis plugin makes use of coap-client which is part of libcoap. \n\nThe plugin comes with prebuilt x86-64 binaries for macOS and Linux. There is no need to compile ```coap-client``` if this is what you\'re using. If not, you\'ll need to compile ```coap-client``` yourself and use the config property ```coapClientPath``` to point to the location of the ```coap-client``` binary.\n```js\n"platforms": [{\n    "platform" : "Tradfri",\n    "name" : "Tradfri",\n    "psk" : "23asqw9123j33",\n    "coapClientPath": "/usr/local/bin/coap-client" // OPTIONAL: See above\n}],\n```\n\n## Compiling libcoap\nInstalling [libcoap](https://github.com/obgm/libcoap) as per the following instructions for Debian/Ubuntu:\n```shell\n$ apt-get install libtool git build-essential autoconf automake\n$ git clone --recursive https://github.com/obgm/libcoap.git\n$ cd libcoap\n$ git checkout dtls\n$ git submodule update --init --recursive\n$ ./autogen.sh\n$ ./configure --disable-documentation --disable-shared\n$ make\n$ sudo make install\n```\n\nYou\'ll find the ```coap-client``` binary in ```./examples```\n\n## Credits\nBig thanks to everyone discussing Tr\xc3\xa5dfri in this issue: https://github.com/bwssytems/ha-bridge/issues/570\n'
