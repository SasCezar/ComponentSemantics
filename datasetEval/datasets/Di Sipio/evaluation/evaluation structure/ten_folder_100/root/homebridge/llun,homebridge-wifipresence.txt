b'# HomeBridge Wifi Presence\n\nDetect presence in the room via wifi. This plugin is using MAC address to detect is anyone in the room or not.\n\n## Setup\n\nInstall plugin `npm install -g homebridge-wifipresence` and add accessories to homebridge config.\n\n```json\n{\n  "bridge": {\n    "name": "Homebridge",\n    "username": "CD:22:3D:E3:CE:30",\n    "port": 51826,\n    "pin": "031-45-154"\n  },\n\n  "description": "This is an example configuration for the WifiPresence homebridge plugin",\n\n  "accessories": [\n    {\n      "accessory": "WifiPresence",\n      "name": "Main Wifi",\n      "room": "Living room",\n      "clients": ["MAC ADDRESS1"],\n      "presenceFile": "/var/lib/misc/presence.wifi"\n    }\n  ],\n\n  "platforms": [\n\n  ]\n}\n```\n\n`MAC ADDRESS1` is device wifi MAC for telling homebridge when this MAC is present, make the room occipied. It can have more than 1 MAC address.\n\n`presenceFile` is path to list of MAC address, currently default is `/var/lib/misc/presence.wifi` same as in [presence.sh](presence.sh) script.\n\nRun `presence.sh` file in router/access point to gathering devices MAC address from wifi interfaces.\n\n## License\n\nISC\n'
