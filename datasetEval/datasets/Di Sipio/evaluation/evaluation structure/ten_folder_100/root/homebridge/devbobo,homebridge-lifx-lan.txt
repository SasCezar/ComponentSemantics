b'# homebridge-lifx-lan\n\n[![npm package](https://nodei.co/npm/homebridge-lifx-lan.png?downloads=true&downloadRank=true&stars=true)](https://nodei.co/npm/homebridge-lifx-lan/)\n\n[![donate](https://img.shields.io/badge/%24-Buy%20me%20a%20coffee-ff69b4.svg)](https://www.buymeacoffee.com/devbobo)\n[![Slack Channel](https://img.shields.io/badge/slack-homebridge--lifx-e01563.svg)](https://homebridgeteam.slack.com/messages/C1NE2GM0S/)\n\nLiFx LAN platform plugin for [Homebridge](https://github.com/nfarina/homebridge).\n\nThis platform uses only the LiFx LAN protocol.\n\nCurrently supports:\n- On/Off\n- Brightness\n- Kelvin\n- Hue (Color models only)\n- Saturation (Color models only)\n\n# Installation\n\n1. Install homebridge using: npm install -g homebridge\n2. Install this plugin using: npm install -g homebridge-lifx-lan\n3. Update your configuration file. See the sample below.\n\n# Updating\n\n- npm update -g homebridge-lifx-lan\n\n# Configuration\n\nConfiguration sample:\n\n ```javascript\n"platforms": [\n    {\n        "platform": "LifxLan",\n        "name": "LiFx",\n        "ignoredDevices" : ["abcd1234561", "efabcd6721"]\n    }\n]\n\n```\n\n\n# Credits\n\n- Marius Rumpf for his awesome [node-lifx](https://github.com/MariusRumpf/node-lifx) library\n'
