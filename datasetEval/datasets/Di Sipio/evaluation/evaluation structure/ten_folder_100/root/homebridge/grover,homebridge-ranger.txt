b'# homebridge-ranger\n\nA HomeKit range extender for Bluetooth Low Energy (BLE) accessories.\n\nBridges existing HomeKit BLE accessories to IP via [Homebridge](https://github.com/nfarina/homebridge) and makes them accessible even if they\'re out of range of your HomeKit Hub.\n\n## Beware\n\n**This plugin is fairly new. It has already been used in several situations, but there\'s likely still some rough edges. Contributions are welcome to improve it. Want to help? Take a look at the [CONTRIBUTING](docs/CONTRIBUTING.md) page.**\n\nStarting with version 0.3.0 of the plugin you\'ll need **homebridge@0.4.35** - earlier versions will not\nprovide event notifications and rules based on accessories provided by this plugin will not work.\n\n## Documentation\n\nPlease read the documentation in its entirety as this plugin is a bit more complex than your usual Homebridge plugin.\n\n- [Why](docs/why.md)\n- [A few words of caution](docs/warnings.md)\n- [Requirements](docs/requirements.md)\n- [Installation instructions](docs/install.md)\n  - [Raspberry Pi](docs/install/raspberrypi.md)\n  - [macOS](docs/install/macos.md)\n  - [Verify your installation of homebridge-ranger](docs/install/verify.md)\n- [Pairing accessories](docs/pairing/pairing.md)\n  - [Attribute database](docs/pairing/attribute-database.md)\n  - [Removing a pairing](docs/pairing/remove-pairing.md)\n  - [Supported accessories](docs/pairing/accessories.md)\n- [Contributing](docs/CONTRIBUTING.md)\n  - [Logging](docs/diagnostics/logging.md)\n- [Credits](docs/credits.md)\n\n## Some asks for friendly gestures\n\nIf you use this and like it - please leave a note by staring this package here or on GitHub.\n\nIf you use it and have a problem, file an issue at [GitHub](https://github.com/grover/homebridge-ranger/issues) - I\'ll try to help.\n\nIf you tried this, but don\'t like it: tell me about it in an issue too.
