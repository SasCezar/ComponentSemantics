Install HomeBridge, please follow it\'s [README](https://github.com/nfarina/homebridge/blob/master/README.md).   \nIf you are using Raspberry Pi, please read [Running-HomeBridge-on-a-Raspberry-Pi](https://github.com/nfarina/homebridge/wiki/Running-HomeBridge-on-a-Raspberry-Pi).   \n2. Install packages.   \n```\nnpm install -g homebridge-mi-gateway-fm\n```\n\n## Configuration\n```\n"accessories": [{\n    "accessory": "MiGatewayFM",\n    "name": "MiGatewayFM",\n    "ip": "192.168.88.xx",\n    "token": "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"\n}]\n```\nIf you want play your own channel, you can set the following in the config.   \n```\n"accessories": [{\n    "accessory": "MiGatewayFM",\n    "name": "MiGatewayFM",\n    "ip": "192.168.88.xx",\n    "token": "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",\n    "url": "http://live.xmcdn.com/live/1005/64.m3u8"\n}]\n```\n\n## Get token\n### Get token by miio2.db\nsetup MiJia(MiHome) app in your android device or android virtual machine.   \nopen MiJia(MiHome) app and login your account.   \nrefresh device list and make sure device display in the device list.   \nget miio2.db(path: /data/data/com.xiaomi.smarthome/databases/miio2.db) file from your android device or android virtual machine.   \nopen website [[Get MiIo Tokens By DataBase File](http://miio2.yinhh.com/)], upload miio2.db file and submit.    \n### Get token by network\nOpen command prompt or terminal. Run following command:   \n```\nmiio --discover\n```\nWait until you get output similar to this:   \n```\nDevice ID: xxxxxxxx   \nModel info: Unknown   \nAddress: 192.168.88.xx   \nToken: xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx via auto-token   \nSupport: Unknown   \n```\n"xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx" is token.   \nIf token is "???", then reset device and connect device created Wi-Fi hotspot.   \nRun following command:   \n```\nmiio --discover --sync\n```\nWait until you get output.   \nFor more information about token, please refer to [OpenMiHome](https://github.com/OpenMiHome/mihome-binary-protocol) and [miio](https://github.com/aholstenson/miio).   \n\n## Version Logs\n### 0.2.6 (2018-02-10)\n1.update \'README.md\'.   \n### 0.2.5 (2018-02-10)\n1.optimized code.   \n### 0.2.4 (2018-02-10)\n1.update \'package.json\'.   \n### 0.2.3 (2017-09-11)\n1.optimized code.   \n### 0.2.2 (2017-09-05)\n1.optimized code.   \n### 0.2.1 (2017-08-27)\n1.optimized code.   \n### 0.2.0 (2017-08-26)\n1.fixed bug that homebridge not works when device is not responding.   \n2.add support for aqara ac partner FM.   \n### 0.1.1 (2017-08-26)\n1.optimized code.   \n### 0.1.0 (2017-08-24)\n1.add support for mi ac partner FM.   \n### 0.0.3 (2017-08-22)\n1.optimized code.   \n### 0.0.2 (2017-08-22)\n1.optimized code.   \n### 0.0.1 (2017-08-20)\n1.Switch on/off XiaoMi Gateway FM.   \n'
