b'# react-fontawesome\n\n[![styled with prettier](https://img.shields.io/badge/styled_with-prettier-ff69b4.svg)](https://github.com/prettier/prettier)\n[![Travis CI build status](https://img.shields.io/travis/danawoodman/react-fontawesome.svg)](https://travis-ci.org/danawoodman/react-fontawesome)\n[![Dependency Status](https://img.shields.io/david/danawoodman/react-fontawesome.svg)](https://david-dm.org/danawoodman/react-fontawesome)\n[![View on npm](https://img.shields.io/npm/dm/react-fontawesome.svg)](https://www.npmjs.com/package/react-fontawesome)\n[![View on npm](https://img.shields.io/npm/v/react-fontawesome.svg)](https://www.npmjs.com/package/react-fontawesome)\n\n> A React component for the font-awesome icon library.\n\n**If you are using Font Awesome v5+, please use [the official React component!](https://github.com/FortAwesome/react-fontawesome). `react-fontawesome` does not support FontAwesome v5!**\n\n## Install\n\n```bash\nnpm install --save react-fontawesome\n```\n\n_**Note:** This component does not include any of the Font Awesome CSS or fonts, so you\'ll need to make sure to include those on your end somehow, either by adding them to your build process or linking to [CDN versions](https://www.bootstrapcdn.com/fontawesome/)._\n\n\n### Using CDN version\n\nThe fastest way to get started is to import FontAwesome with a link tag in your page\'s `<head>`:\n\n```html\n<link href="https://maxcdn.bootstrapcdn.com/font-awesome/4.7.0/css/font-awesome.min.css" rel="stylesheet" />\n```\n\nYou can change the version number to use whatever version of FontAwesome you\'d like.\n\n### Using with Webpack\n\nIf you\'re using WebPack, you will need to install FontAwesome, import the FontAwesome Sass/Less, and configure the font path. Please see this [blog post](https://medium.com/@chanonroy/webpack-2-and-font-awesome-icon-importing-59df3364f35c) on how to get Webpack and FontAwesome working together.\n\n## Usage\n\n```js\nvar React = require(\'react\')\nvar FA = require(\'react-fontawesome\')\n\nReact.render(<FA name="rocket" />, document.body)\n```\n\n## Features\n\n* No dependencies (other than React)\n* Simple API that mirrors Font Awesome\'s class names.\n* Supports all Font Awesome modifiers (see [API](#API) below).\n* Make use of Css Modules\n* Add your own `className`s, styles and other props (all additional props are passed directly to the component).\n\n## Examples\n\n**Regular usage**\n\n```js\nvar React = require(\'react\')\nvar FontAwesome = require(\'react-fontawesome\')\n\nvar MyComponent = React.createClass({\n  render: function() {\n    return (\n      <FontAwesome\n        className="super-crazy-colors"\n        name="rocket"\n        size="2x"\n        spin\n        style={{ textShadow: \'0 1px 0 rgba(0, 0, 0, 0.1)\' }}\n      />\n    )\n  },\n})\n```\n\n**With the use of CSS Modules**\n\n```js\nimport React from \'react\'\nimport FontAwesome from \'react-fontawesome\'\nimport faStyles from \'font-awesome/css/font-awesome.css\'\n\nvar MyComponent = React.createClass({\n  render: function() {\n    return (\n      <FontAwesome\n        className="super-crazy-colors"\n        name="rocket"\n        cssModule={faStyles}\n        size="2x"\n        spin\n        style={{ textShadow: \'0 1px 0 rgba(0, 0, 0, 0.1)\' }}\n      />\n    )\n  },\n})\n```\n\nThe above will create output like this:\n\n```\n<span class="font-awesome__fa___2otTb font-awesome__fa-rocket___lfSov font-awesome__super-crazy-colors___3k583"></span>\n```\n\n## API\n\nSee [the API docs](api.md) for full documentation.\n\n## Contributing\n\nPull requests welcome!\n\nRun the test suite with `npm test` and format your code with `npm run format`. Make sure tests are passing and that you write tests for new features and document changes to the API with updates to the JSDocs.\n\nBefore you submit your pull request, run `npm run dist` to build the project and commit the changes.\n\n## License\n\n[MIT](license) &copy; [Dana Woodman][author]\n\n[author]: https://github.com/danawoodman\n'
