b'# DECOVAR\n\nA variable decorative sans by David Berlow.\n\n![Examples of Decovar](https://raw.githubusercontent.com/TypeNetwork/fb-Decovar/master/documentation/decovar-samples.png)\n\nThe [documentation](documentation/) contains a description of the project and typographic contents of the repo.\n\nThe [fonts](fonts/) folder contains the DecovarAlpha-VF.ttf file, the variable font.\n\nThe [sources](sources/) folder contains the fonts with axis data in their names, used to create the Variations Font.\n\n![Screenshot of Decovar in RoboFont](https://raw.githubusercontent.com/TypeNetwork/fb-Decovar/master/documentation/decovar-screenshot.png)\n\n## Axes\n\nEach of Decovar\xe2\x80\x99s axes has a default value of 0 and a maximum value of 1000, and are defined in the [designspace file](https://github.com/TypeNetwork/fb-Decovar/blob/master/sources/2-build/DecovarAlpha.designspace). Currently, Decovar\xe2\x80\x99s axes (and their four letter tags) are as follows:\n\n* BLDA: Inline\n* BLDB: Worm\n* WMX2: Weight\n* SKLA: Inline Skeleton\n* SKLB: Worm Skeleton\n* SKLD: Stripes\n* TRMA: Rounded\n* TRMB: Flared\n* TRMC: Rounded Slab\n* TRMD: Sheared\n* TRME: Bifurcated\n* TRMF: Open Inline Terminal\n* TRMG: Slab\n* TRMK: Inline Terminal\n* TRML: Worm Terminal\n\nYou can implement them in CSS like this:\n\n```\nfont-variation-settings: "SKLA" 1000, "TRMG" 750;\n```\n\n## Generating\n\nThe Decovar variable font was created with RoboFont and is generated using [fontmake](https://github.com/googlei18n/fontmake). A build script `build.sh` is included that will build the variable font using fontmake, copy it to the `/fonts` folder, and offer a subsetted version.\n\n```\n$ cd sources/2-build\n$ bash build.sh\n```\n'
