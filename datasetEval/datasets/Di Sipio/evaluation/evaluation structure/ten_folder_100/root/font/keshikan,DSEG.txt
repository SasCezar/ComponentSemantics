b'DSEG Font Family (v0.45)\n====\n\n## Overview\n\nDSEG is a free font which imitate LCD Display (7SEG, 14SEG, Weather icons etc.).\nDSEG have special features:\n\n - Includes the roman-alphabet and symbol glyphs.\n - Many types(over 50) are available.\n - Licensed under [SIL OPEN FONT LICENSE Version 1.1](http://scripts.sil.org/OFL). You can use DSEG for non-commercial and commercial purposes.\n\n![DSEG Image](http://www.keshikan.net/img/DSEG_sample_github.png)\n\n## Sample\n![DSEG Sample Image](http://www.keshikan.net/img/DSEG_weather_sample.png)\n\n## Usage\n\n - Colon and Space have same width.\n\n   ![DSEG usage 1](http://www.keshikan.net/img/dseg_usage1.png)\n\n - Period has zero width.\n\n   ![DSEG usage 2](http://www.keshikan.net/img/dseg_usage2.png)\n\n - All-off (Exclamation)\n\n   ![DSEG usage 3](http://www.keshikan.net/img/dseg_usage3.png)\n\n - All-on ("8" or Tilda)\n\n   ![DSEG usage 4](http://www.keshikan.net/img/dseg_usage4.png)\n\n## Others\n\nFor more information, visit DSEG support page.\n\n - English:[https://www.keshikan.net/fonts-e.html](https://www.keshikan.net/fonts-e.html)\n - Japanese:[https://www.keshikan.net/fonts.html](https://www.keshikan.net/fonts.html)\n\n## How to Install\n\n### Download release binary\n\n[release binary](https://github.com/keshikan/DSEG/releases)\n\n### Build from source(*.sfd)\n\nInstall [FontForge](https://fontforge.github.io/en-US/) and [Google woff2](https://github.com/google/woff2), and type below.\n\n     $ make\n\n### Install in Ubuntu(18.04 or later)\n    $ sudo apt-get install fonts-dseg\n    \n### Install from npm\n    $ npm i dseg\n\n## Changelog\n\n - [v0.45(2018-01-09)](https://github.com/keshikan/DSEG/releases/download/v0.45/fonts-DSEG_v045.zip)\n    - Added makefile and build script.
