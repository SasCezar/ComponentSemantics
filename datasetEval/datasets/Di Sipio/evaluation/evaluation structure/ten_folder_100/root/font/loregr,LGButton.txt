b'\n![logo](media/header_btn.png)\n\n[![build](https://travis-ci.org/loregr/LGButton.svg?branch=master)](https://travis-ci.org/loregr/LGButton) ![platform](https://img.shields.io/badge/platform-ios-blue.svg) [![license](https://img.shields.io/badge/license-MIT-lightgrey.svg)](./LICENSE.md) [![Version](https://img.shields.io/cocoapods/v/LGButton.svg?style=flat)](http://cocoadocs.org/docsets/LGButton)\n\nA fully customisable subclass of the native `UIControl` which allows you to create beautiful buttons without writing any line of code.\n\n\n## Preview\n\nYou\'ll be able to create buttons like these by using only the interface builder:\n\n![preview](media/preview_btn.png)\n\nMoreover you can use them for asynchronous operations.\n\n![async](media/async_demo.gif)\n\n\n\n## Install\n\nTo integrate _LGButton_ into your Xcode project using CocoaPods, specify it in your `Podfile`:\n\n```ruby\nuse_frameworks!\n\npod \'LGButton\'\n```\n\n## Icons and images\n\nYou can add images/icons to your button, icons are taken from different ready to use font icons collections, images can be imported directly from your project assets folder.\n\nYou can customise icons/images on the left and/or on the right of the button title.\n\n### Icons\n\nIcons are managed by [SwiftIconFont](https://github.com/0x73/SwiftIconFont) integration, so you can easily add and customise button icons by choosing from the most popular font icons collections. In particular:\n\n\n| Collection         | Font name | Cheat Sheet                               |\n|--------------|--------|-------------------------------------------|\n| Font Awesome | fa    | [List](https://fontawesome.com/v4.7.0/icons/) |\n| Ion Icons    | io    | [List](http://ionicons.com)               |\n| Octicons     | oc    | [List](https://octicons.github.com)       |\n| Open Iconic  | ic    | [List](https://useiconic.com/open/)       |\n| Material Icon   | ma    | [List](https://design.google.com/icons/)       |\n| Themify   | ti    | [List](https://themify.me/themify-icons)       |\n| Map Icons   | mi    | [List](http://map-icons.com)       |\n\nThe main properties that you need to set in order to use font icons are:\n- **Icon Font Name**: you can pick one value from the table above, different values will be ignored <i>(please consider that not all the icons may be available since this library may not be updated right after a font library update).</i>\n\n- **Icon String**: the icon name taken from the cheatsheet without the collection prefix.\n\n_So, if you want to use `fa-align-left` from `Font Awesome`, **Icon Font Name** will be `fa` and **Icon String** will be `align-left`._\n\n### Images\nSet the **Image source** property in order to use an image from your assets folder.\n\n_Note: if **Image source** is set, every other property about the font icon on the same side will be ignored._\n\n## Usage\n\nDrag and drop an `UIView` object into your view controller and set its class and module to `LGButton`.\n\n![class](media/class_name.png)\n\nCustomise your button by setting the properties from the Interface Builder.\n\n![properties_1](media/properties_1.jpg) ![properties_2](media/properties_2.jpg)\n\n### Supported Attributes\n\n| Attribute        | Description      | Default value  |\n| ------------- |-------------| -----|\n| Bg Color      | Main  | nil |\n| Gradient End Color      | The second color of the gradient background     | nil |\n| Gradient End Color      | The second color of the gradient background     | nil |\n| Gradient | The corner radius     | 0.0 |\n| Fully Rounded Corners      | Apply a corner radius equals to height/2  | false |\n| Border Color      | The border color     | white |\n| Border Width      | The border width     | 0.0 |\n| Title Color      | The title color     | white |\n| Title String      | The title string      | _EMPTY&#95;STRING_ |\n| Title Font Name      | The title font name    | nil |\n| Title Font Size      | The title font size    | 14.0 |\n| Vertical orientation      | If `true` the left icon will be placed on the top and the right icon will be placed on the bottom    | false |\n| Shadow Offset      | The shadow offset   | (0,0)|\n| Shadow Radius      | The shadow radius   | 0.0|\n| Shadow Opacity      | The shadow opacity   | 1.0|\n| Shadow Color      | The shadow color   | black|\n| Loading spinner Color      | The loading spinner color   | white|\n| Loading String      | The loading message text   | _EMPTY&#95;STRING_ |\n| Loading Font Name      | The loading message
