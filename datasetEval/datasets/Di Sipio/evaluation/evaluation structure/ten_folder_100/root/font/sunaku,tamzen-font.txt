b'Tamzen font\n===========\n\nThis is [my personal fork][1] of the wonderful [Tamsyn font by Scott Fial][2].\nIt is programatically forked from Tamsyn version 1.11 by the `Rakefile` script\n(see the "Building" section below) which backports glyphs from older versions\nwhile deleting deliberately empty glyphs (which are marked as unimplemented)\nto allow secondary/fallback fonts to provide real glyphs at those codepoints.\n\nThe "TamzenForPowerline" fonts provide additional [Powerline symbols][5],\nwhich are programatically injected with [ZyX_I\'s bitmap-font-patcher][6].\nI later hand-tweaked those symbols to perfection in the gbdfed(1) editor:\n\n* For all icons, I expanded them to occupy the maximum available space.\n* For the fork icon (\xee\x82\xa0 U+E0A0), I made the branch larger than the trunk.\n* For the newline icon (\xee\x82\xa1 U+E0A1), I made the "N" at the bottom larger.\n* For the padlock icon (\xee\x82\xa2 U+E0A2), I replaced the keyhole with // lines.\n\nScreenshots\n-----------\n\n![man manual page](https://sunaku.github.io/gruvbox-terminal-color-scheme-man.png)\n![tmux, vim, etc.](https://sunaku.github.io/tmux-layout-dwindle.png)\n\n```sh\n$ cat sample.txt\nABCDEFGHIJKLMNOPQRSTUVWXYZ 12345\nabcdefghijklmnopqrstuvwxyz 67890\n{}[]()<>$*-+=/#_%^@\\&|~?\'"`!,.;:\nIllegal1i = oO0    \xee\x82\xa0 \xee\x82\xa1 \xee\x82\xa2 \xee\x82\xb0 \xee\x82\xb1 \xee\x82\xb2 \xee\x82\xb3\nThe quick brown fox, (..) Hello,\njumps over lazy dog. /__\\ World!\n```\n\n### a package manager:\n\n[![Packaging status](https://repology.org/badge/vertical-allrepos/fonts:tamzen.svg)](https://repology.org/project/fonts:tamzen/versions)\n\nManually, on any Linux:\n\n* [Download a release]( https://github.com/sunaku/tamzen-font/releases )\n  or clone this Git repository into `~/.fonts/tamzen-font` and then run:\n\n      xset +fp ~/.fonts/tamzen-font/bdf\n      xset fp rehash\n\n* You should now be able to see the "Tamzen" font family in `xfontsel`.\n\nIn Microsoft Windows:\n\n* Copy the files in `fnt/` and `ttf/` folders into your Windows Fonts folder.\n\nOn other platforms:\n\n* Try using the font files in these subfolders:\n  * `ttf/`   - TrueType Font\n  * `pt3/`   - PS Type3 Bitmap\n  * `dfont/` - Apple bitmap only sfnt (dfont)\n\n### Linux VT integration\n\n1. Switch to a Linux VT (virtual terminal) by pressing Control-Alt-F1.\n\n2. Run `showconsolefont` to see how the character map currently looks.\n\n3. Run `setfont ~/.fonts/tamzen-font/psf/TamzenForPowerline10x20.psf`.\n\n4. See how the character map now displays glyphs from the Tamzen font.\n\n5. Try running `setfont` with other Tamzen fonts in the `psf/` folder.\n\nSee <https://wiki.archlinux.org/index.php/Fonts#Console_fonts> for details.\n\n### XTerm integration\n\nThe following Xresources let you dynamically switch between the various Tamzen\nfonts by holding down Control and right-clicking in [the XTerm terminal][3].\n\n    XTerm*font  : -*-tamzen-medium-*-*-*-16-*-*-*-*-*-*-*\n    XTerm*font1 : -*-tamzen-medium-*-*-*-9-*-*-*-*-*-*-*\n    XTerm*font2 : -*-tamzen-medium-*-*-*-12-*-*-*-*-*-*-*\n    XTerm*font3 : -*-tamzen-medium-*-*-*-13-*-*-*-*-*-*-*\n    XTerm*font4 : -*-tamzen-medium-*-*-*-15-*-*-*-*-*-*-*\n    XTerm*font5 : -*-tamzen-medium-*-*-*-16-*-*-*-*-*-*-*\n    XTerm*font6 : -*-tamzen-medium-*-*-*-20-*-*-*-*-*-*-*\n\n    !
