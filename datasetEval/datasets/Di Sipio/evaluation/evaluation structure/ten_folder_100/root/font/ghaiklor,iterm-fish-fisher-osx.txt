b'# iTerm 2 + Fish Shell + Fisher + Themes\\Plugins\r\n\r\nThis guide is targeted for those, who wants to have a nice UI in the terminal with the rich features.\r\n\r\nPlease, raise your issue if something strange happened.\r\nI\'d like to improve this script, so it could work without any errors during the installation.\r\n\r\n## The Problem\r\n\r\nEvery time, I\'ve re-installed my operating system, I spend around ~30 minutes to set up my terminal environment again.\r\nI bored of it, so I decided to make a list of all actions I doing, when setting it up, alongside with automatic script to do so, and share it with you all.\r\n\r\n## Screenshots & GIFs\r\n\r\nThey are available [here](./SCREENSHOTS.md).\r\n\r\n## Key Features\r\n\r\nThe script installs things related to terminal environment only:\r\n\r\n- Installs Command Line Tools, Homebrew, iTerm2, Fish, Fisher.\r\n- Installs Material Design color preset for iTerm2 and patched Meslo Nerd Font.\r\n- Theme `bobthefish`, which is based on popular `agnoster`.\r\n- Completions for `brew`, `git`, `./node_modules/.bin` and so on, so on...\r\n\r\nInstalls a lot of useful plugins for Fish Shell:\r\n\r\n- `bass` plugin that makes easy to use utilities written for Bash Shell in Fish Shell.\r\n- `battery` function shows current level of the battery.\r\n- Plugin `done` that notifies you when the process is finished. i.e. you can run `npm install` command and switch back into your browser. When `npm install` is done, you will get OSX notification.\r\n- `errno-grep` function allows to search POSIX error codes and their messages.\r\n- `fzy` plugin adds a hotkey <key>Ctrl</key>+<key>R</key> that allows to show and search in your command history.\r\n- `license` function for generating GitHub license in your current folder.\r\n- `pj` function allows to easily jump between your favorite directories. It installs with configured `~/Library/Projects` folder, so you can jump to any of your projects by calling `pj <PROJECT_FOLDER_NAME>`.\r\n- `shark` function build sparklines right in your terminal.\r\n- `upto` function gets you to a parent folder. I.e. you\'re inside `a/b/c/d/e/`, calling `upto b` will navigate you into `a/b`.\r\n- `z` function shows the most recent and frequently visited folders, so you can go there without typing `cd` everytime.\r\n\r\n## How To Setup\r\n\r\nYou can achieve the same setup as mine, by manually setting up the environment ([following the guide](#manual-installation) below) or automatically [by executing the installer](#automatic-installation) `install.sh`.\r\n\r\n## Automatic Installation\r\n\r\n__Highly recommended__ to run the script below under Bash session in default Terminal.app.\r\nI can not guarantee proper installation outside of Terminal.app + Bash Shell.\r\n\r\n```shell\r\n$ bash <(curl -s https://raw.githubusercontent.com/ghaiklor/iterm-fish-fisher-osx/master/install.sh)\r\n```\r\n\r\n## Manual Installation\r\n\r\n### iTerm2\r\n\r\n#### Install iTerm 2\r\n\r\n- [Download](https://www.iterm2.com/downloads.html) and install iTerm2 (it has better color fidelity than the built in Terminal).\r\n\r\n```shell\r\n$ brew cask install iterm2\r\n```\r\n\r\n#### Install Color Scheme\r\n\r\nGet the iTerm color settings:\r\n\r\n- [Material Design Theme](https://raw.githubusercontent.com/MartinSeeler/iterm2-material-design/master/material-design-colors.itermcolors)\r\n- [More themes @ iterm2colorschemes](http://iterm2colorschemes.com)\r\n\r\nJust save it somewhere and open the file(s).\r\nThe color settings will be imported into iTerm2.\r\nApply them in iTerm through iTerm -> Preferences -> Profiles -> Colors -> Load Presets.\r\nYou can create a different profile, other than Default if you wish to do so.\r\n\r\n#### Install Patched Font\r\n\r\n- [Meslo LG M DZ Regular Nerd Font Complete Mono](https://github.com/ryanoasis/nerd-fonts/blob/25eec835188d2316ef3fe59820950d9f90c5bcf4/patched-fonts/Meslo/M-DZ/Regular/complete/Meslo%20LG%20M%20DZ%20Regular%20Nerd%20Font%20Complete%20Mono.ttf?raw=true)\r\n- [Others @ powerline fonts](https://github.com/ryanoasis/nerd-fonts)\r\n\r\nOpen the downloaded font and press "Install Font".\r\n\r\nSet this font in iTerm2 (iTerm -> Preferences -> Profiles -> Text).\r\n\r\n- Regular Font -> "Change Font"\r\n- Non-ASCII Font -> "Change Font"\r\n\r\nRestart iTerm2 for all changes to take effect.\r\n\r\n### Fish Shell\r\n\r\n#### Install Fish Shell\r\n\r\nDownload and install [Fish Shell](https://fishshell.com).\r\n\r\n```shell\r\n$ brew install fish\r\n$ echo "/usr/local/bin/fish" | sudo tee -a /etc/shells\r\n$ chsh -s /usr/local/bin/fish\r\n```\r\n\r\n#### Install Fisher\r\n\r\n[Fisher](https://github.com/jorgebucaran/fisher) is a plugin manager for Fish Shell.\r\n\r\n```shell\r\n$ curl https://git.io/fisher --create-dirs -sLo ~/.config/fish/functions/fisher.fish\r\n```\r\n\r\n#### Install Themes and Plugins\r\n\r\n```shell\r\nfisher add igalic/anicode\r\nfisher add edc/bass\r\nfisher add oh-my-fish/plugin-battery\r\nfisher add oh-my-fish/theme-bobthefish\r\nfisher add laughedelic/brew-completions\r\n\r\nbrew install terminal-notifier\r\nfisher add franciscolourenco/done\r\n\r\nfisher add Shadowigor/plugin-errno-grep\r\n\r\nbrew install fzy\r\nfisher add gyakovlev/fish-fzy\r\n\r\nbrew install grc\r\nfisher add oh-my-fish/plugin-grc\r\n\r\nbrew install jq\r\nfisher add oh-my-fish/plugin-license\r\n\r\nfisher add oh-my-fish/plugin-node-binpath\r\n\r\nfisher add oh-my-fish/plugin-pj\r\nset -U PROJECT_PATHS ~/Library/Projects\r\n\r\nfisher add fisherman/shark\r\nfisher add Markcial/upto\r\nfisher add jethrokuan/z\r\n```\r\n\r\n- [Anicode](https://github.com/fisherman/anicode)\r\n\r\nFind arbitrary unicode characters matching a search pattern.\r\nThe last result match will be copied to your clipboard.\r\n\r\n- [Bass](https://github.com/edc/bass)\r\n\r\nBass makes it easy to use utilities written for Bash in fish shell.\r\n\r\n- [Battery](https://github.com/oh-my-fish/plugin-battery)\r\n\r\nOS X and Linux compatible battery utility.\r\n\r\n- [Bob The Fish](https://github.com/oh-my-fish/theme-bobthefish)\r\n\r\nA Powerline-style, Git-aware fish theme optimized for awesome.\r\n\r\n- [Brew Completions](https://github.com/laughedelic/brew-completions)\r\n\r\nFish shell completions for Homebrew.\r\n\r\n- [done](https://github.com/fisherman/done)\r\n\r\nA fish plugin to automatically receive notifications when long processes finish.\r\n\r\n- [errno-grep](https://github.com/Shadowigor/plugin-errno-grep)\r\n\r\nSearch for error codes, labels or messages via `errno-grep`.\r\n\r\n- [fzy](https://github.com/fisherman/fzy)\r\n\r\n`fzy` picks up history item and adds it to your shell.\r\nYou need to execute manually.\r\n\r\nRun `fkill` and type process you want to kill.\r\n`fkill` kills immediately.\r\nPress enter and process will be killed.\r\n\r\n- [GRC](https://github.com/oh-my-fish/plugin-grc)\r\n\r\nGeneric Colouriser is yet another colouriser for beautifying your logfiles or output of commands.\r\n\r\n- [License](https://github.com/oh-my-fish/plugin-license)\r\n\r\nFish Shell plugin for generating GitHub licenses.\r\n\r\n- [Node BinPath](https://github.com/oh-my-fish/plugin-node-binpath)\r\n\r\nAutomatically add `node_modules/.bin` to `PATH` when present.\r\n\r\n- [PJ](https://github.com/oh-my-fish/plugin-pj)\r\n\r\n`pj` allows you to easily jump between your favourite directories in a predictable manner.\r\nYou tell pj where to look for your projects, and it will allow you to jump to them easily with tab completion.\r\nIt even provides a convenient ability to open an editor in that directory from anywhere!\r\n\r\n- [Shark](https://github.com/fisherman/shark)\r\n\r\nShark is a sparkline generator for fish.\r\n\r\n- [upto](https://github.com/fisherman/upto)\r\n\r\nGets you to a parent folder, heavily inspired by the plugin upto made by driv.\r\n\r\n- [z](https://github.com/fisherman/z)\r\n\r\n`z` tracks the directories you visit.\r\nWith a combination of frequency and recency, it enables you to jump to the directory in mind.\r\n\r\n## License\r\n\r\n[MIT](./LICENSE)\r\n'
