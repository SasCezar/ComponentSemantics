release](https://github.com/rastikerdar/vazir-font/releases/latest) file.\n\n#### Bower\n```\nbower install vazir-font --save\n```\n\n#### npm\n```\nnpm install vazir-font\n```\nFonts will be copied to `node_modules/vazir-font/dist` directory\n\n#### CDN\nLink fonts from the [RawGit](https://rawgit.com) CDN:\n\n```html\n<link href="https://cdn.rawgit.com/rastikerdar/vazir-font/v[X.Y.Z]/dist/font-face.css" rel="stylesheet" type="text/css" />\n```\n\nReplace [X.Y.Z] with the latest version (e.g. 21.1.2) and integrate the font into your CSS:\n\n```\nfont-family: \'Vazir\', sans-serif;\n```\n\n#### Homebrew\nYou can install the font on macOS machines by tapping the caskroom/fonts repo:\n\n```shell\nbrew tap caskroom/fonts\nbrew install font-vazir\n```\n\n#### Arch Linux\nArch user\'s could use [vazir-fonts](https://aur.archlinux.org/packages/vazir-fonts/) package from [AUR](https://aur.archlinux.org/) repository to install vazir font.\nUse your favourite [AUR helper](https://wiki.archlinux.org/index.php/AUR_helpers) like pacaur or yaourt for installing package:\n\n```shell\npacaur -S vazir-fonts\n```\n#### Ubuntu/Debian\nAn unofficial PPA is available for installing Vazir and other free Persian fonts. It only works on Ubuntu 18.04 and higher. Open a Terminal and enter these commands:\n```shell\nsudo add-apt-repository ppa:blacksuited/fonts\nsudo apt update\nsudo apt install fonts-vazir\n```\n\n## License\n2015 Saber Rastikerdar ([@rastikerdar](https://github.com/rastikerdar)).
