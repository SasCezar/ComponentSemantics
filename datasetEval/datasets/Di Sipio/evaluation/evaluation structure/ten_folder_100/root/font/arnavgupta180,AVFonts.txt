b'# AVFonts\n\n[![CI Status](http://img.shields.io/travis/Arnav/AVFonts.svg?style=flat)](https://travis-ci.org/Arnav/AVFonts)\n[![Version](https://img.shields.io/cocoapods/v/AVFonts.svg?style=flat)](http://cocoapods.org/pods/AVFonts)\n[![License](https://img.shields.io/cocoapods/l/AVFonts.svg?style=flat)](http://cocoapods.org/pods/AVFonts)\n[![Platform](https://img.shields.io/cocoapods/p/AVFonts.svg?style=flat)](http://cocoapods.org/pods/AVFonts)\n\n\n### What\'s that for\n\nAVFonts allows you to do anything you want to do with the fonts anywhere and everywhere in the app.\n\n### Why should you use this?\n\nAVFonts can easily swap the system font or specific font you are using with the new font throughout the app. You can increment or decrement font size if needed, too. Now there\'s no need to select custom fonts in the Interface Builder. Save time by simply configuring its size and style, and then change "system font" to your "new font" everywhere with a single line of code. It\'s that easy.\n\n![](AVFonts.gif?raw=true "AVFonts screenshot")                            ![](AVFonts2.gif?raw=true "AVFonts  italic screenshot")\n\n### Usage\n\n**1.** import `AVFonts` to `AppDelegate`.\n\n**2.** In `didFinishLaunchingwithOptions` change the font :\n\n**a)** This will swap/change font from system or **currentFontName** to **newFontName** everywhere in the app.\n```bash\nAVFonts.changeFont(toFont: "Avenir-Heavy")\nAVFonts.swapFont(currentFont: "Avenir-Light", toFont: "Avenir-Heavy")\n```\n**b)** Font swap/change for **types provided** i.e. [ .button, .label, .textField, .textView] throughout the app.\n\n```bash\nAVFonts.changeFont(toFont: "Avenir-Heavy", [.button, .label])\nAVFonts.swapFont(currentFont: "Avenir-Light", toFont: "Avenir-Heavy", [.button, .label])\n```\n**c)** **increment/decrement** your orignal font size.\n\n```bash\nAVFonts.changeFont(toFont: "Avenir-Heavy", [.button,.label,.textfield], increment: 2)\nAVFonts.changeFont(toFont: "Avenir-Heavy", [.button,.label,.textfield], increment: -2)\n\nAVFonts.swapFont(currentFont: "Avenir-Light", toFont: "Avenir-Heavy", [.button, .label, .textfield], increment: 2)\nAVFonts.swapFont(currentFont: "Avenir-Light", toFont: "Avenir-Heavy", [.button, .label, .textfield, .textview], increment: -2)\n\n```\n**3.** Call **AVFonts.applyAVFonts()** to apply all the changes.\n\n###  Example\n\nTo run the example project, clone the repo, and run `pod install` from the Example directory first.\n\n## Requirements\n\n- iOS 8.0+ / macOS 10.10+ / tvOS 9.0+ / watchOS 2.0+\n- Xcode 10.2\n- Swift 5.0\n\n\n## Installation\n\n### CocoaPods\n\n[CocoaPods](http://cocoapods.org) is a dependency manager for Cocoa projects. You can install it with the following command:\n\n```bash\n$ gem install cocoapods\n```\n\n> CocoaPods 1.1.0+ is required to build  AVFonts.\n\nTo integrate AVFonts into your Xcode project using CocoaPods, specify it in your `Podfile`:\n\n```ruby\nsource \'https://github.com/CocoaPods/Specs.git\'\nplatform :ios, \'10.0\'\nuse_frameworks!\n\ntarget \'<Your Target Name>\' do\npod "AVFonts"\nend\n```\n\nThen run the following command:\n\n```bash\n$ pod install\n```\n\n\n## Author\n\nArnav Gupta, arnavgupta180@gmail.com\n\n## License\n\nAVFonts is available under the MIT license.
