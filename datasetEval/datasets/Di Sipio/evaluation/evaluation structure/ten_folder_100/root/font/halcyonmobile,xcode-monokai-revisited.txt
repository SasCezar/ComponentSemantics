b"# Monokai Revisited Xcode Theme\n\nAnother take on the Monokai theme for Xcode, with a couple of subtle changes like string, project function names and selection coloring.\nPreview screenshots include the beautiful [Input](http://input.fontbureau.com/) font by Font Bureau.\n\n![Monokai Xcode Theme](http://f.cl.ly/items/2r122h2U0A3Q3L0a1a2l/monokai_revisited_preview.png)\n\n## Differences\n\n![Theme Differences](http://f.cl.ly/items/310Z1O0K0s2m3w2k3y0e/monokai_revisited_compared.png)\n\n## Installation\n\n### Using the provided shell script\n\n1. Clone the repository\n\n```bash\ngit clone git@github.com:halcyonmobile/xcode-monokai-revisited.git\n```\n\n2. Run the install script inside the cloned repository.\n\n```bash\nsh install.sh\n```\n\n### Manual\n\n1. Copy **Monokai Revisited.xccolortheme** to the Xcode's ``FontAndColorThemes`` folder ``(~/Library/Developer/Xcode/UserData/FontAndColorThemes/)``. *Note that the directory might not exist.*\n2. Restart Xcode.\n3. Go to `Xcode` > `Preferences` > `Fonts & Colors`.\n4. Select **Monokai Revisited**.\n"
