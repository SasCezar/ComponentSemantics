By default, Bridge will use the Namespace name as the file name. In this case: **demo.js**. There are many options to control the output of your JavaScript files, and the [Attribute Reference](https://github.com/bridgedotnet/Bridge/wiki/attribute-reference) is important [documentation](https://github.com/bridgedotnet/Bridge/wiki) to review.\n\n```javascript\nBridge.define("Demo.Program", {\n    main: function Main() {\n        System.Console.WriteLine("Hello World!");\n    }\n});\n```\n\n## Installation\n\nA full list of installation options available at [bridge.net/download/](http://bridge.net/download/), including full support for Visual Studio and Visual Studio Community on Windows, and Visual Studio Mac.\n\n### Bridge for Visual Studio\n\nIf you\'re using Visual Studio for Windows, the easiest way to get started is by adding the Bridge.NET for Visual Studio [extension](https://visualstudiogallery.msdn.microsoft.com/dca5c80f-a0df-4944-8343-9c905db84757).\n\nFrom within Visual Studio, go to the `Tools > Extensions and Updates...`.\n\n![Bridge for Visual Studio](https://user-images.githubusercontent.com/62210/29292228-932ebb7e-8103-11e7-952a-3088274acf10.png)\n\nFrom the options on the left side, be sure to select **Online**, then search for **Bridge**. Clicking **Download** will install Bridge for Visual Studio. After installation is complete, Visual Studio may require a restart. \n\n![Visual Studio Extensions and Updates](https://user-images.githubusercontent.com/62210/29292229-93406b44-8103-11e7-90a0-30232486a5a7.png)\n\nOnce installation is complete you will have a new **Bridge.NET** project type. When creating new Bridge enabled projects, select this project type. \n### NuGet\n\nAnother option is installation of Bridge into a new **C# Class Library** project using [NuGet](https://www.nuget.org/packages/bridge). Within the NuGet Package Manager, search for **Bridge** and click to install. \n\nBridge can also be installed using the NuGet Command Line tool by running the following command:\n\n```\nInstall-Package Bridge\n```\n\nMore information regarding Nuget package installation for Bridge is available in the [Documentation](https://github.com/bridgedotnet/Bridge/wiki/nuget-installation).\n\n## Contributing\n\nInterested in contributing to Bridge?
