it?\r\n\r\n1. Install the latest [.NET Core SDK](https://dot.net).\r\n2. Run `dotnet new --install Boxed.Templates` to install the project template.\r\n3. Run `dotnet new api --help` to see how to select the feature of the project.\r\n5.
