b'# ncTelegram\n\nA ncurses Telegram client developed in Python with the urwid library.\n\n<p align="center">\n  <img src="http://pix.toile-libre.org/upload/original/1457204711.png" alt="screenshot of ncTelegram"/>\n</p>\n\n## Dependencies\n\n* [Telegram-cli](https://github.com/vysheng/tg)\n* [urwid](http://urwid.org)\n* [pytg](https://github.com/luckydonald/pytg)\n* python-psutil\n* libnotify (optional)\n* caca-utils (optional)\n\n## Installation\n\n### Arch Linux:\n\nYou can use the pkgbuild from [AUR](https://aur.archlinux.org/packages/nctelegram-git/):\n\n```\n$ yaourt -S nctelegram-git\n\nOR\n\n$ pacaur -y nctelegram-git\n```\n\n### Fedora:\n\nInstall dependencies:\n\n```\n$ sudo dnf install telegram-cli python3-urwid python3-psutil python3-pip\n```\n\nInstall ncTelegram through pip:\n```\n$ sudo pip3 install --upgrade https://github.com/Nanoseb/ncTelegram/archive/master.tar.gz\n```\n\nIf you want notifications and inline images:\n\n```\n$ sudo dnf install libnotify caca-utils\n```\n\n### Debian/Ubuntu:\n\nTo install telegram-cli just follow its [readme](https://github.com/vysheng/tg).\n\nInstall urwid:\n\n```\n$ sudo apt-get install python3-urwid python3-psutil python3-pip\n```\n\nInstall ncTelegram through pip:\n\n```\n$ sudo pip3 install --upgrade https://github.com/Nanoseb/ncTelegram/archive/master.tar.gz\n```\n\nIf you want notifications and inline images:\n\n```\n$ sudo apt-get install libnotify-bin caca-utils\n```\n\n### Crux:\n\nInstall repository:\n\n```\n$ wget -O /etc/ports/wildefyr.httpup "https://crux.nu/portdb/?a=getup&q=wildefyr"\n$ ports -u\n```\n\nInstall ncTelegram:\n\n```\n$ prt-get depinst nctelegram\n```\n\nIf you want notifications and inline images:\n\n```\n$ prt-get depinst libnotify libcaca\n```\n\n## Usage\n\nBefore the first run of `nctelegram`, `telegram-cli` must be launched first in\norder to register your account on your machine for the first time. (phone number\nand verification code).\n\nAn example of the configuration file can be found in `/etc/ncTelegram.conf`,\ncopy it to your personal folder before modifying it:\n\n```\n$ cp /etc/ncTelegram.conf ~/.ncTelegram.conf\n```\n\nFinally, make sure that the path of `telegram-cli` and the public key are set\ncorrectly in `~/.ncTelegram.conf`.\n\n#### Tips\n\n- Press `esc` to leave the text area.\n\n- Press `q` to quit the ncTelegram.\n\n- Press `esc` twice to draw a separator at the end of the message list.\n\n- Use `ctrl+o` to open the last file or url sent to the current channel.\n\n- Use `ctrl+r` to insert a linebreak.\n\n- To send files you can simply drag and drop them into your terminal or send as\na message their path between single quotes (\').\n\n- You can navigate the channel list with `ctrl+p` and `ctrl+n`.\n\n- Select `Download message buffer` to navigate quickly between channels.\n\n- Vim-like key bindings are also available, use `hjkl` to navigate between\nlists, use `i` to select the text area and `esc` to leave it.\n\n- You can also use `ctrl+w` to remove words left of the cursor and `ctrl+u` to remove the whole line.\n\n- Use `shift + Up` or `shift + Down` to navigate in your own message history.\n\n#### Troubleshooting\n\nIf `nctelegram` crashes, please [report the\nissue](https://github.com/Nanoseb/ncTelegram/issues/new) with a backtrace if\npossible.\n\nBefore restarting `nctelegram` again, check if `telegram-cli` is\nstill running; if so kill it before restarting `nctelegram`.\n'
