bits)](https://github.com/Drakkar-Software/OctoBot/wiki/Installation)\n- Open the OctoBot-Launcher [release page](https://github.com/Drakkar-Software/OctoBot-Launcher/releases)\n- Download launcher (*laucher_windows.exe* or *launcher_linux*)\n- Start the launcher\n- Click on "Update OctoBot"\n\n#### [With Docker](https://github.com/Drakkar-Software/OctoBot/wiki/With-Docker)\n```\ndocker run -it -d --name MyAmazingOctobot -v $(pwd)/user:/bot/octobot/user -v $(pwd)/logs:/bot/octobot/logs -v $(pwd)/tentacles:/bot/octobot/tentacles drakkarsoftware/octobot:beta\n```\n\n#### [With python sources](https://github.com/Drakkar-Software/OctoBot/wiki/With-Python-only) (unix)\n- Install python3.7 (https://www.python.org/downloads/)\n```\ngit clone git@github.com:Drakkar-Software/OctoBot.git && cd OctoBot\npip3 install -r pre_requirements.txt\npip3 install -r requirements.txt\npython3 start.py -p install all\n```\n\nMore details in [wiki
