\xe2\xad\x90\xef\xb8\x8f Introduction \xe2\xad\x90\xef\xb8\x8f\n\nBroid Integrations is an open source project released by [Broid](http://www.broid.ai?utm_source=github&utm_medium=readme&utm_campaign=introduction) that allows you to easily converse on all the major communication platforms (messaging & voice) without integrating each API.\n\nIt provides a suite of libraries to convert all messaging platforms events to [Activity Streams 2](https://www.w3.org/TR/activitystreams-core?utm_source=github&utm_medium=readme&utm_campaign=introduction&link=as2) schemas.\n\n**TL;DR**\n\n* All events are Observables ([RxJS](https://github.com/ReactiveX/rxjs))\n* All functions return a Promise ([Bluebird](http://bluebirdjs.com/docs/getting-started.html))\n* Highly modular to keep your focus on high level features\n<br><br>\n\n## \xf0\x9f\x92\xaa Motivation \xf0\x9f\x92\xaa\n\nWith billions of daily users Messaging\xf0\x9f\x92\xac is now the prefered channel of communication but there is a clear lack of open standard for the various APIs. \n<br><br>\nEach one come with a specfic schema, docs, etc. We want to empower the community to built great things with messaging so we open-sourced our [Premium API](https://www.broid.ai?utm_source=github&utm_medium=readme&utm_campaign=motivation) so that you can avoid the tedious work of integrating new channels and focus on your core product \xf0\x9f\x92\xa1.\n<br><br>\n## \xf0\x9f\x94\x8e Quick Example \xf0\x9f\x94\x8e\n\nA quick example showing off listening for new messages on Facebook Messenger and Discord with Observables.\n\n```shell\nnpm i --save @broid/messenger @broid/discord rxjs ramda\n```\n\n```javascript\nconst Rx = require("rxjs/Rx");\nconst BroidDiscord = require("@broid/discord");\nconst BroidMessenger = require("@broid/messenger");\n\nconst clients = {\n\tdiscord: new BroidDiscord({token: \'DISCORD_TOKEN\'}),\n\tmessenger: new BroidMessenger({token: \'FACEBOOK_TOKEN\', tokenSecret: \'FACEBOOK_SECRET\'})\n};\n\nRx.Observable.merge(...Object.values(clients).map(client => client.connect()))\n\t.subscribe({\n\t\tnext: data => console.log(JSON.stringify(data, null, 2)),\n\t\terror: err => console.error(`Something went wrong: ${err.message}`),\n\t});\n\nRx.Observable.merge(...Object.values(clients).map(client => client.listen()))\n\t.subscribe({\n\t\tnext: message => console.log(JSON.stringify(message, null, 2)),\n\t\terror: err => console.error(`Something went wrong: ${err.message}`),\n\t});\n```\n\n## \xf0\x9f\x94\x8c Get Started \xf0\x9f\x94\x8c\n\nHere are 2
