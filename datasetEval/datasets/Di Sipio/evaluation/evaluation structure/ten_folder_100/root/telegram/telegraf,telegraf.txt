Introduction\n\nBots are special [Telegram](https://telegram.org) accounts designed to handle messages automatically. \nUsers can interact with bots by sending them command messages in private or group chats. \nThese accounts serve as an interface for code running somewhere on your server.\n\n### Features\n\n- Full [Telegram Bot API 4.4](https://core.telegram.org/bots/api) support\n- [Telegram Payment Platform](https://telegram.org/blog/payments)\n- [HTML5 Games](https://core.telegram.org/bots/api#games)\n- [Inline mode](https://core.telegram.org/bots/api#inline-mode)\n- Incredibly fast\n- `TypeScript` typings\n\n### Installation\n\n```\n$ npm install telegraf\n```\nor using `yarn`:\n```\n$ yarn add telegraf\n```\n\n### Documentation\n\n[Telegraf developer
