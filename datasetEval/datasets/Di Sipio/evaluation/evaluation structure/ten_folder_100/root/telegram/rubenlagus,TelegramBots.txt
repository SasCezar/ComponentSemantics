b'# Telegram Bot Java Library\n[![Telegram](http://trellobot.doomdns.org/telegrambadge.svg)](https://telegram.me/JavaBotsApi)\n\n\n[![Build Status](https://travis-ci.org/rubenlagus/TelegramBots.svg?branch=master)](https://travis-ci.org/rubenlagus/TelegramBots)\n[![Jitpack](https://jitpack.io/v/rubenlagus/TelegramBots.svg)](https://jitpack.io/#rubenlagus/TelegramBots)\n[![Maven Central](https://maven-badges.herokuapp.com/maven-central/org.telegram/telegrambots/badge.svg)](http://mvnrepository.com/artifact/org.telegram/telegrambots)\n[![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat)](https://github.com/rubenlagus/TelegramBots/blob/master/LICENSE)\n\nA simple to use library to create Telegram Bots in Java\n\n## Contributions\nFeel free to fork this project, work on it and then make a pull request against **DEV** branch. Most of the times I will accept them if they add something valuable to the code.\n\nPlease, **DO NOT PUSH ANY TOKEN OR API KEY**, I will never accept a pull request with that content.\n\n## Webhooks vs GetUpdates\nBoth ways are supported, but I recommend long polling method.\n\n## Usage\n\nJust import add the library to your project with one of these options:\n\n  1. Using Maven Central Repository:\n\n```xml\n    <dependency>\n        <groupId>org.telegram</groupId>\n        <artifactId>telegrambots</artifactId>\n        <version>4.4.0.2</version>\n    </dependency>\n```\n\n```gradle\n    compile "org.telegram:telegrambots:4.4.0.2"\n```\n\n  2. Using Jitpack from [here](https://jitpack.io/#rubenlagus/TelegramBots/4.4.0.2)\n  3. Download the jar(including all dependencies) from [here](https://mvnrepository.com/artifact/org.telegram/telegrambots/4.4.0.2)\n\nIn order to use Long Polling mode, just create your own bot extending `org.telegram.telegrambots.bots.TelegramLongPollingBot`.\n\nIf you like to use Webhook, extend `org.telegram.telegrambots.bots.TelegramWebhookBot`\n\n\nOnce done, you just need to create a `org.telegram.telegrambots.meta.TelegramBotsApi`and register your bots:\n\n```java\n\n    // Example taken from https://github.com/rubenlagus/TelegramBotsExample\n    public class Main {\n        public static void main(String[] args) {\n            ApiContextInitializer.init();\n            TelegramBotsApi telegramBotsApi = new TelegramBotsApi();\n            try {\n                telegramBotsApi.registerBot(new ChannelHandlers());\n                telegramBotsApi.registerBot(new DirectionsHandlers());\n                telegramBotsApi.registerBot(new RaeHandlers());\n                telegramBotsApi.registerBot(new WeatherHandlers());\n                telegramBotsApi.registerBot(new TransifexHandlers());\n                telegramBotsApi.registerBot(new FilesHandlers());\n            } catch (TelegramApiException e) {\n                e.printStackTrace();\n            }\n        }\n    }\n\n```\n\nFor detailed explanation, visite our [How To](https://github.com/rubenlagus/TelegramBots/wiki/Getting-Started) (thanks Clevero)\n\n\n## Example bots\nOpen them and send them */help* command to get some information about their capabilities:\n\nhttps://telegram.me/weatherbot (**Use custom keyboards**)\n\nhttps://telegram.me/directionsbot (**Basic messages**)\n\nhttps://telegram.me/filesbot (**Send files by file_id**)\n\nhttps://telegram.me/TGlanguagesbot (**Send files uploding them**)\n\nhttps://telegram.me/RaeBot (**Inline support**)\n\nhttps://telegram.me/SnowcrashBot (**Webhook support**)\n\nYou can see code for those bots at [TelegramBotsExample](https://github.com/rubenlagus/TelegramBotsExample) project.\n\n## Telegram Bot API\nThis library use [Telegram bot API](https://core.telegram.org/bots), you can find more information following the link.\n\n## Questions or Suggestions\nFeel free to create issues
