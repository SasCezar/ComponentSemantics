developers.\nIt was built on top of [Apple/SwiftNIO](https://github.com/apple/swift-nio) which help to demonstrate excellent performance.\n> SwiftNIO is a cross-platform asynchronous event-driven network application framework for rapid development of maintainable high performance protocol servers and clients.\n\nAlso Telegrammer use some submodules of [Vapor 3 server side swift framework](https://github.com/vapor/vapor)\n\nJoin to our [Telegram developers chat](https://t.me/joinchat/AzGW3kkUjLoK2dr3CZFrFQ)\n\nJoin to our [Telegrammer channel](https://discord.gg/yjspy8b) on [Vapor Discord server](https://discord.gg/3jG8QFV) \n\nIt\'s easy, from scratch\n-------------\n_main.swift_\n```swift\nimport Telegrammer\n\nlet bot = try! Bot(token: "BOT_TOKEN_HERE")\n\nlet commandHandler = CommandHandler(commands: ["/hello"], callback: { (update, _) in\n    guard let message = update.message, let user = message.from else { return }\n    try! message.reply(text: "Hello \\(user.firstName)", from: bot)\n})\n\nlet dispatcher = Dispatcher(bot: bot)\ndispatcher.add(handler: commandHandler)\n\n_ = try! Updater(bot: bot, dispatcher: dispatcher).startLongpolling().wait()\n```\n\nDocumentation\n-------------\n\n- Read [An official introduction for developers](https://core.telegram.org/bots) \n- Check out [bot FAQ](https://core.telegram.org/bots/faq)\n- Official [Telegram Bot API](https://core.telegram.org/bots/api)\n\nHOWTO Guides\n-------------\n\n- [Create Telegram Bot](https://core.telegram.org/bots#3-how-do-i-create-a-bot)\n- [Setup Telegram Bot webhooks](https://core.telegram.org/bots/webhooks)\n\nUsage without Vapor\n---------------\n\n1. Create package with Swift Package Manager (SPM)\n```\n$ mkdir MyBot\n$ cd MyBot\n$ swift package init --type executable\n```\n2. Define Dependencies in Package.swift file\n```swift\nlet package = Package(\n    name: "MyBot",\n    dependencies: [\n        .package(url: "https://github.com/givip/Telegrammer.git", from: "0.5.0")\n        ],\n    targets: [\n        .target( name: "MyBot", dependencies: ["Telegrammer"])\n        ]\n)\n```\n3. Resolve dependencies\n```\n$ swift package resolve\n```\n4. Generate XCode project (for macOS only)\n```\n$ swift package generate-xcodeproj\n```\n5. Open in XCode (for macOS only)\n```\n$ open MyBot.xcodeproj\n```\nYou project is ready to create new Telegram Bot.\n\nIf you need more help through this steps, you can read [How to create a Telegram bot with Telegrammer on Ubuntu / macOS](https://github.com/givip/Telegrammer/wiki/Creating-Telegram-bot-in-Swift)\n\nUsage with Vapor\n---------------\n\nYou may also use previous way to create project with Vapor, only include Vapor as dependency in Package.swift\n\n1. Install [Vapor CLI](https://docs.vapor.codes/3.0/install/macos/)\n```\n$ brew install vapor/tap/vapor\n```\n2. Create Vapor project with [Telegrammer template](https://github.com/givip/telegram-bot-template/)\n```\n$ vapor new MyBot --template=https://github.com/givip/telegram-bot-template\n```\n3. Generate XCode project\n```\n$ vapor xcode\n```\n\nDemo bots\n---------\n\n#### All sample bots\n1. Responds with "hello" message on command "/greet"\n\n[SchedulerBot sources](https://github.com/givip/Telegrammer/tree/master/Sources/SchedulerBot)\nDemonstrate Jobs Queue scheduling mechanism. \nCommand "/start X" starts repeatable job, wich will send you a message each X seconds.\nCommand "/once X" will send you message once after timeout of X seconds.\nCommand "/stop" stops JobsQueue only for you. Other users continues to receive scheduled messages.\n\n[SpellCheckerBot sources](https://github.com/givip/Telegrammer/tree/master/Sources/SpellCheckerBot)\nDemonstrate how works InlineMenus and Callback handlers.\nCommand "/start" will start bot.\nSend any english text to bot and it will be checked for mistakes.
