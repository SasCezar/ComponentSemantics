Here are its salient points:\n* Has a pluggable query layer, and supports two distributed SQL APIs:\n    * **[Yugabyte SQL (YSQL)](https://docs.yugabyte.com/latest/api/ysql/)** - PostgreSQL-compatible fully relational API\n    * **[Yugabyte Cloud QL (YCQL)](https://docs.yugabyte.com/latest/api/ycql/)** - Semi-relational SQL-like API with documents/indexing support and Apache Cassandra QL roots \n* Automated sharding, Raft consensus replication and distributed transactions, based on the Google Spanner architecture\n* Offers horizontal write scalability, strong write consistency and tunable read consistency (strong reads by default with follower & observer reads as options)\n* Extremely resilient with native failover and repair - can tolerate disk, node, zone and region failures automatically\n* Supports geo-distributed deployments (multi-zone, multi-region, multi-cloud)\n* Built-in enterprise features such as distributed backups, in-flight/at-rest encryption and read replicas (for observer reads)\n* Can be deployed in public clouds and natively inside Kubernetes\n* Best fit for powering massively-scalable, globally-distributed, cloud-native applications that require absolute data correctness and high tolerance to failures\n* 100% open source under the [Apache 2.0 license](https://github.com/yugabyte/yugabyte-db/blob/master/LICENSE.md)\n\nRead more about YugabyteDB in our [Docs](https://docs.yugabyte.com/latest/introduction/).\n\n# Get Started\n\n* [Install YugabyteDB](https://docs.yugabyte.com/latest/quick-start/install/)\n* [Create a local cluster](https://docs.yugabyte.com/latest/quick-start/create-local-cluster/)\n* [Connect and try out SQL commands](https://docs.yugabyte.com/latest/quick-start/explore-ysql/)\n* [Build an app](https://docs.yugabyte.com/latest/quick-start/build-apps/) using a PostgreSQL-compatible driver or ORM.\n* Try a real-world app:\n    * [Microservices-oriented e-commerce app](https://github.com/yugabyte/yugastore-java)\n    * [Streaming IoT app with Kafka and Spark Streaming](https://docs.yugabyte.com/latest/develop/realworld-apps/iot-spark-kafka-ksql/)\n\nCannot find what you are looking for?
