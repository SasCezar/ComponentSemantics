b'# ![pipe](./pipe.png) Tekton Pipelines\n\n[![Go Report Card](https://goreportcard.com/badge/tektoncd/pipeline)](https://goreportcard.com/report/tektoncd/pipeline)\n\nThe Tekton Pipelines project provides k8s-style resources for declaring\nCI/CD-style pipelines.\n\nTekton Pipelines are **Cloud Native**:\n\n- Run on Kubernetes\n- Have Kubernetes clusters as a first class type\n- Use containers as their building blocks\n\nTekton Pipelines are **Decoupled**:\n\n- One Pipeline can be used to deploy to any k8s cluster\n- The Tasks which make up a Pipeline can easily be run in isolation\n- Resources such as git repos can easily be swapped between runs\n\nTekton Pipelines are **Typed**:\n\n- The concept of typed resources means that for a resource such as an `Image`,\n  implementations can easily be swapped out (e.g. building with\n  [kaniko](https://github.com/GoogleContainerTools/kaniko) v.s.\n  [buildkit](https://github.com/moby/buildkit))\n\n## Want to start using Pipelines\n\n-
