See\nthe [Linkerd Getting Started Guide][getting-started] for how.\n\nFor more comprehensive documentation, start with the [Linkerd\ndocs][linkerd-docs]. (The doc source code is available in the\n[website][linkerd-website] repo.)\n\n## Working in this repo ##\n\n[`BUILD.md`](BUILD.md) includes general information on how to work in this repo.\n\nWe :heart: pull requests!
