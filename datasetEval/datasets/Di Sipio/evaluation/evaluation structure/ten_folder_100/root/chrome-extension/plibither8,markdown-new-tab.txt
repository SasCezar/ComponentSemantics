Refer to [this brilliant guide](https://github.github.com/gfm/) to get familiar with the markdown syntax.\n\n### Features\n\n* Take down _**`styled`**_ notes \xf0\x9f\x97\x92\xef\xb8\x8f, create checklists \xe2\x98\x91\xef\xb8\x8f, links \xf0\x9f\x94\x97, tables and reminders \xe2\x8f\xb0, add images \xf0\x9f\x96\xbc\xef\xb8\x8f (and all other frills associated with \xef\xbd\x8d\xef\xbd\x81\xef\xbd\x92\xef\xbd\x8b\xef\xbd\x84\xef\xbd\x8f\xef\xbd\x97\xef\xbd\x8e [M\xe2\x86\x93])\n\n* \xe2\x8f\xb0 \xf0\x9f\x92\xbe Automatically saves (and deletes) revision history for you to look back and reminisce\n\n* \xe2\x8c\xa8\xef\xb8\x8f Use [keyboard shortcuts](#Usage) to toggle between edit and save the notes\n\n* \xf0\x9f\x92\xaa \xf0\x9f\x92\xaa ACTIVATE POWERMODE! (enable in settings)\n\n* Sync notes, revision history and settings between all Chrome browsers you are logged into\n\n### Upcoming\n\n* Change background and foreground colours\n\n* Split editing to show live preview\n\n## Usage\n\n* You can edit and save the notes either by pressing the buttons on the top right, or by using the shortcuts <kbd>Ctrl</kbd> + <kbd>X</kbd> (or <kbd>Cmd</kbd> + <kbd>X</kbd> on Mac) to edit the text and <kbd>Ctrl</kbd> + <kbd>S</kbd> (or <kbd>Cmd</kbd> + <kbd>S</kbd> on Mac) to save the text.\n\n* To save and edit the notes by using <kbd>Ctrl</kbd> + <kbd>\xe2\x86\xb5</kbd> (or <kbd>Cmd</kbd> + <kbd>\xe2\x86\xb5</kbd> on Mac), go to settings and enable it.\n\n* Revision history can be accessed by clicking "Last Edited: ____" on the bottom right corner.\n\n## Development\n\n1. Clone this repo:\n\n```sh\n$ git clone https://github.com/plibither8/markdown-new-tab\n```\n\n2. Open Chrome and go to `chrome://extensions`\n3. Enable \'Developer Mode\' by checking the tickbox (on the top of the page).\n4. Click the \'Load Unpacked Extension\' button and select the `dist/` folder of the cloned repository.\n5. The extension should be loaded now and the \'New Tab\' page should be Markdown New Tab. \xf0\x9f\x8e\x89\n\n> The code makes use of `localStorage()` to save the raw text, revision history, last edited time and date, settings and last cursor position.\n\n### Testing in Firefox\n\nIn Firefox the extension can be installed temporarily until you restart the browser.
