b'# Vue.js Chrome Extension Template ([wcer](https://github.com/YuraDev/wcer))\n> Template for quick creation of Chrome extension on Vuejs c hot reloading when developing.\n\n![Vue.js Chrome Extension Template images](/docs/images/mini.jpg)\n\n## Installation:\nThis boilerplate was built as a template for [vue-cli](https://github.com/vuejs/vue-cli) and includes options to customize your final scaffolded app. \n``` bash\n# install vue-cli\n$ npm install -g vue-cli\n# create a new project using the template\n$ vue init YuraDev/vue-chrome-extension-template my-project\n# install dependencies and go!\n$ cd my-project\n$ npm install # or yarn\n$ npm run dev # or yarn dev\n```\n\n## Structure\n* [backend](https://developer.chrome.com/extensions/background_pages): Background work of your scripts\n* [content](https://developer.chrome.com/extensions/content_scripts) Run in the context of web pages \n* [devtools](https://developer.chrome.com/extensions/devtools) - It can add new UI panels and sidebars, interact with the inspected page, get information about network requests, and more.\n* [options](https://developer.chrome.com/extensions/options) - To allow users to customize the behavior of your extension, you may wish to provide an options page.\n* popup - The page (window) that will be displayed when the icon is clicked\n* tab - Your application will work in a separate tab\n* ext - Shared scripts\n* [manifest.js](https://developer.chrome.com/extensions/manifest) - Descriptions of the application, its rights and possibilities'
