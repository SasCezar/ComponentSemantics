b'## <img src="icons/enhanced-github128.png" align="center"> Enhanced GitHub\n\n> Chrome extension that adds useful features on GitHub website\n\n![Chrome Web Store](https://img.shields.io/chrome-web-store/d/anlikcnbgdeidpacdbdljnabclhahhmd.svg)\n![Chrome Web Store](https://img.shields.io/chrome-web-store/rating/anlikcnbgdeidpacdbdljnabclhahhmd?color=orange&style=flat-square)\n![Chrome Web Store](https://img.shields.io/chrome-web-store/v/anlikcnbgdeidpacdbdljnabclhahhmd.svg)\n\n## Link\n\n\xf0\x9f\x9a\x80 Install extension from [Chrome Web Store](https://chrome.google.com/webstore/detail/enhanced-github/anlikcnbgdeidpacdbdljnabclhahhmd)\n\n## Features\n\n- Displays repo size.\n- Displays each file size for every active branch (not applicable for folder / symlink).\n- Show download link for each individual file (not applicable for folder / symlink).\n- Copy file\'s contents directly to Clipboard (just won\'t work for markdown files).\n- Download file while viewing it\'s contents.\n\n**_Note:_** For private repos([Issue #6](https://github.com/softvar/enhanced-github/issues/6)), GitHub Access Token is required. Follow the steps [mentioned below](#github-api-rate-limiting) to add your GitHub Access Token.\n\n[Featured on ProductHunt](https://www.producthunt.com/tech/github-plus)\n\n<img src="screenshots/product_hunt.png" height="300" />\n\n## Author\n\n| Author | [Varun Malhotra](http://varunmalhotra.xyz) |\n| ------ | ------------------------------------------ |\n\n\n## GitHub API Rate Limiting\n\nSince this extension fetches data using GitHub public v3 API for showing file _size_ and _download_url_, it consumes free quota which is very less [GitHub API Rate Limiting](https://developer.github.com/v3/rate_limit/).\n\nTo tackle this, create a new GitHub Access Token.\n\n1. If logged-in, visit [https://github.com/settings/tokens](https://github.com/settings/tokens)\n2. Generate a new token, select `repo` scope and create a one.\n3. Copy and store the generated token which looks something like: `17c1a8d5b399d66b6212382d98d4c67a94d58955` (a fake one :P).\n4. Click on extension icon and then on `Settings-Options` on top right.\n   <img src="screenshots/extension-popup-screenshot.png" width="380" />\n\n   OR\n\n   Right click on enhanced-github extension and and click on `Options` in the dropdown menu.\n\n   <img src="screenshots/extension-options-page.png" width="380" />\n\n5. Enter the valid GitHub Access Token\n6. Click on `SAVE` and Enjoy the benefits.\n\nThis Chrome extension will automatically pick this valid access token and Bingo!\n\n## Screenshots\n\n#### Screenshot of each file size and it\'s download link\n\n<img src="screenshots/screenshot-home.png">\n\n#### Screenshot of Copy fle contents and download link\n\n<img src="screenshots/screenshot-file.png">\n\n## Libraries Used\n\n- Thanks to [@zenorocha](https://github.com/zenorocha/) for [Clipboard.js](https://github.com/zenorocha/clipboard.js) - Modern copy to clipboard.
