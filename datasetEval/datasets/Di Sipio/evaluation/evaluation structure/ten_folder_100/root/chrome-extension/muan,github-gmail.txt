b'# Helper for GitHub notifications in Gmail\n\nA web extension for Chrome and Firefox which adds keyboard shortcuts for opening and muting GitHub notification emails.\n\n## Features\n\n### Mail View Shortcut\n\nOpen your GitHub notifications in mail view:\n\n- Chrome: <kbd>shift</kbd> + <kbd>g</kbd>, <kbd>shift</kbd> + <kbd>b</kbd> in the background\n- Firefox: <kbd>ctrl</kbd> + <kbd>g</kbd>, <kbd>ctrl</kbd> + <kbd>b</kbd> in the background\n\n![Mail view button](https://user-images.githubusercontent.com/1153134/42123231-69153916-7c1c-11e8-8bf5-1d8fa2510b63.png)\n\n### Mute Thread\n\nMute thread in mail view. It will open a background window to load the mute thread URL, and close itself when done. This only works if you have an active GitHub session.\n\n- Chrome: <kbd>shift</kbd> + <kbd>h</kbd>\n- Firefox: <kbd>ctrl</kbd> + <kbd>h</kbd>\n\n![Mute thread button](https://user-images.githubusercontent.com/1153134/42123234-7c6d271c-7c1c-11e8-9b13-3cd0cbea4eab.png)\n\n### List View Shortcut\n\n<kbd>ctrl</kbd> + <kbd>return</kbd> to trigger one the action button when an email is selected using gmail key navigation (when the blue bar appears):\n\n![action button in list view](https://user-images.githubusercontent.com/1153134/42123260-fa87c648-7c1c-11e8-8d64-9ddd8899e594.png)\n\n## Installation\n\n- [Download in Chrome Web Store](https://chrome.google.com/webstore/detail/github-notification-helpe/gmhijkhbpihfmkmhmcfebmlkaekgmaje)<br>\n- [Download in Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/github-for-gmail/)\n\n## Shortcuts\n\n- Chrome: `Select extension` -> `Details` -> `Extension options`\n- [Setting extension shortcuts in Firefox](https://support.mozilla.org/en-US/kb/manage-extension-shortcuts-firefox)\n\n## Development\n\nLoad the extension manually and modify the code accordingly in these places:\n\nChrome: `chrome/src/inject/inject.js`.<br>\nFirefox: `firefox/src/inject/inject.js`.<br>\n'
