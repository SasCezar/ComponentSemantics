b'  <img src="https://user-images.githubusercontent.com/799578/50462941-8075fe80-09c3-11e9-89e7-af0cb7991406.png" width="80">\n\nCODELF(\xe5\x8f\x98\xe9\x87\x8f\xe5\x91\xbd\xe5\x90\x8d\xe7\xa5\x9e\xe5\x99\xa8)\n=================\n  A search tool helps dev to solve the naming things problem.\n\n  Search over projects from Github, Bitbucket, Google Code, Codeplex, Sourceforge, Fedora Project, GitLab to find real-world usage variable names.\n  \n  Also a daily Algorithm Copybook.\n  \n  Also a GitHub stars, repositories tagger and organizer tool.\n\n  >There are only two hard things in Computer Science: cache invalidation and naming things.-- Phil Karlton\n  >\n  >![twohardtings](https://user-images.githubusercontent.com/799578/50462942-8075fe80-09c3-11e9-9c7f-b38d495b925d.jpg)\n  \n  ![image](https://user-images.githubusercontent.com/799578/51435509-a2595d00-1cb3-11e9-8f4e-85ecbc3a2325.png)\n\nWIKI\n=================\n[\xe7\xae\x80\xe4\xbd\x93\xe4\xb8\xad\xe6\x96\x87](https://github.com/unbug/codelf/wiki)\n  \nPlugins\n=================\n### Codelf for VS Code\n\n##### Install\n  Codelf is on VS Code Extension Marketplace. [Download Codelf for Sublime](https://github.com/unbug/codelf/archive/st-0.0.5.zip) or [Select a "st-" version to download](https://github.com/unbug/codelf/tags)\n  2. Unzip and rename the folder to `Codelf`\n  3. Copy the folder into `Packages` directory, which you can find using the menu item `Sublime Text -> Preferences -> Browse Packages...`\n  4. Restart Sublime Text\n\n##### Usage\n  1. Select text, right-click and click "Codelf". \n  \n  ![ST Right Click](https://cloud.githubusercontent.com/assets/799578/12536608/655d4f72-c2e5-11e5-8836-7ce733f47eed.png)\n  2.
