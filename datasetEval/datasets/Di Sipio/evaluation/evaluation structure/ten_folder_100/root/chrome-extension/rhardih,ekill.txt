b'# ekill\n\nIt\'s like [**xkill**](https://en.wikipedia.org/wiki/Xkill), but for annoying web pages instead.\n\nChrome and Firefox plugin for quickly getting rid of elements on a web page.\n\n## Installation\n\n- [Chrome web store](https://chrome.google.com/webstore/detail/ekill/lcgdpfaiipaelnpepigdafiogebaeedg?hl=en)\n- [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/ekill/)\n\n![Example](https://media.githubusercontent.com/media/rhardih/ekill/master/example.gif)\n\n## Keyboard shortcut\n\nBy default **ekill** is toggled with *ctrl+shift+k*, but this can be modified at will.\n\nGo to [chrome://extensions/shortcuts](chrome://extensions/shortcuts), find the item labeled "ekill" and set it to whatever is most convenient.\n\n\n## Options\n\n### Grudge (Experimental)\n\nTurning this feature on, will let ekill hold a grudge against offending\nelements.\n\nBy keeping a record of killed of elements on a per page basis, ekill will try\nit\'s best to remove these elements on subsequent visits to the same page.\n\nA rudimentary ui for toggling *Grudge*, as well as  listing and editing the hit\nlist, is included in the options page.\n\n## License\n\nMIT: http://rhardih.mit-license.org\n\n## Changelog\n\n**1.8**\n\n- Changes the default hot-key from Ctrl+k to Ctrl+Shift+k.\n\n**1.7**\n\n- Adds a kill count badge to the extension icon.\n\n**1.6**\n\n- Adds the Grudge feature\n- Adds options page to control Grudge\n- Adds changelog notifications on version upgrades\n\n**1.5**\n\n- Adds light icons for Firefox dark theme.\n\n**1.4**\n\n- Adds support for Firefox\n\n**1.1 - 1.3**\n\n- Introduces ability to toggle on/off, as well as dismiss with Esc key\n- Permissions narrowed to *activeTab* only\n- Targets anything with *role=button*\n\n**1.0**\n\n- Initial version\n'
