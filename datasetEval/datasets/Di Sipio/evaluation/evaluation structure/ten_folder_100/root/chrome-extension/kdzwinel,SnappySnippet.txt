b"SnappySnippet\n======================\n\nChrome/Chromium extension that allows easy CSS+HTML extraction of specific DOM element. Created snippet can be then exported to CodePen, jsFiddle or JS Bin with one click.\n\nOther features:\n- cleans up HTML (removing unnecessary attributes, fixing indentation)\n- optimizes CSS to make it readable\n- fully configurable (all filters can be turned off)\n- nice UI thanks to Bootstrap & Flat-UI projects\n- works with :before and :after pseudo elements\n\n![Screenshot](https://github.com/kdzwinel/SnappySNippet/blob/master/gfx/screenshots/2.png?raw=true)\n\nInstallation\n------------\n\nYou may install this extension from its google chrome webstore page\n\nhttps://chrome.google.com/webstore/detail/snappysnippet/blfngdefapoapkcdibbdkigpeaffgcil\n\nor download it and manually load as an 'Unpacked extension' via chrome extensions page.\n\n\nUsage\n-----\n\nOpen Developer Tools, you should see the new SnappySnippet tab. Inspect an element on the page and switch to the new tab, you will see options for settings and exporting the code for the element you selected.\n\nFor more details, see [this demonstration video](https://www.youtube.com/watch?v=if5pXoIJ4EU).\n\n\nBugs and Features\n-----------------\n\nIf you found a bug or have a feature request, please create an issue here on GitHub.\n\nhttps://github.com/kdzwinel/SnappySnippet/issues\n\nChangelog\n---------\n\n### 0.5 ###\n\n+ [#13](https://github.com/kdzwinel/SnappySnippet/issues/13) IFrames support (by @crdev)\n+
