Provides a common way for programs to create, send, read messages.\n\nThis is a main development repository. It provides a friendly environment for productive development and testing of all Enqueue related features&packages.\n\nFeatures:\n\n* [Feature rich](docs/quick_tour.md).\n\n* Adopts [queue interoperable](https://github.com/queue-interop/queue-interop) interfaces (inspired by [Java JMS](https://docs.oracle.com/javaee/7/api/javax/jms/package-summary.html)).\n* Battle-tested.
