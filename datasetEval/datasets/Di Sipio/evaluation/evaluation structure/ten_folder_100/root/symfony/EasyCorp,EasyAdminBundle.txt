b'EasyAdmin\n=========\n\nEasyAdmin creates administration backends for your Symfony applications with\nunprecedented simplicity.\n\n<img src="https://raw.githubusercontent.com/EasyCorp/EasyAdminBundle/master/doc/images/easyadmin-promo.png" alt="Symfony Backends created with EasyAdmin" align="right" />\n\n* [Installation][1]\n* [Creating Your First Backend][2]\n* [Documentation][3]\n\n**Features**\n\n  * **CRUD** operations on Doctrine entities (create, edit, list, delete).\n  * Full-text **search**, **pagination** and column **sorting**.\n  * Supports Symfony 4.1 or higher\n  * Translated into tens of languages.\n  * **Fast**, **simple** and **smart** where appropriate.\n\n**Requirements**\n\n  * Symfony 4.1 or higher applications.\n  * Doctrine ORM entities (Doctrine ODM not supported).\n  * Entities with composite keys or using inheritance are not supported.\n\nVersions\n--------\n\n* [1.x version](https://github.com/EasyCorp/EasyAdminBundle/tree/1.x): no longer\n  maintained. Recommended only for legacy projects using a Symfony version prior to 4.1;\n* [2.x version](https://github.com/EasyCorp/EasyAdminBundle/tree/2.x): the current\n  stable version. Recommended for all projects using Symfony 4.1 or newer;\n* [3.x version](https://github.com/EasyCorp/EasyAdminBundle/tree/master): it\'s\n  still in early development. Don\'t use it in any project yet.\n\nDemo Application\n----------------\n\n[easy-admin-demo](https://github.com/javiereguiluz/easy-admin-demo) is a complete\nSymfony application created to showcase EasyAdmin features.\n\nLicense\n-------\n\nThis software is published under the [MIT License](LICENSE.md)\n\n[1]: https://symfony.com/doc/current/bundles/EasyAdminBundle/book/installation.html\n[2]: https://symfony.com/doc/current/bundles/EasyAdminBundle/book/your-first-backend.html\n[3]: https://symfony.com/doc/current/bundles/EasyAdminBundle\n'
