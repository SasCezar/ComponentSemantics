e-commerce, blogs, portfolio\'s, you name it\r\n- Integrable - Allow you to integrate any kind of data source with it\r\n- Reliable - You do not have to worry about high traffic and scalability\r\n- Extensible - Allow you to extend your project with as many custom features as necessary\r\n\r\n### Why should you use Falcon\r\n\r\n- You can work separately and independently on the front or back-end - save crucial development time\r\n- Get started within a couple minutes using our CLI tool `create-falcon-app`\r\n- Client rendering (SPA) to increase pagespeed and reduce server load\r\n- Server Side Rendering (SSR) built-in - no longer do you have to worry about SPA SEO complications\r\n- Service worker to provide application caching and PWA features such as offline capabilities and add to homescreen\r\n- Falcon\'s architecture allows it to be very modular, lightweight and scalable\r\n\r\nThese are just a couple of advantages and use-cases, feel free to ask anything in the official slack channel [slack.deity.io](https://slack.deity.io)\r\n\r\n### Start developing right away\r\n\r\nUsing the app generator:\r\n\r\n```bash\r\nnpx create-falcon-app my-app\r\n```\r\n\r\n![create-falcon-app](https://user-images.githubusercontent.com/40865297/48708977-64cd4a80-ec04-11e8-80cf-757db1610bf1.gif)\r\n\r\n### Demo\r\n\r\nWe\'ve created a Demo project which is using Algolia for search, WordPress for blog content and Magento 2 for the rest.\r\n\r\nTry it out yourself here [https://demo.deity.io](https://demo.deity.io)\r\n\r\n![demo gif](https://user-images.githubusercontent.com/40865297/48703128-1a8f9d80-ebf3-11e8-8b61-afa490d26c24.gif)\r\n\r\n## Community\r\n\r\nAny contributions, small or big, are very welcome!
