([\xe4\xb8\xad\xe6\x96\x87](/README_zh.md))\n\nxmake is a cross-platform build utility based on lua. \n\nThe project focuses on making development and building easier and provides many features (e.g package, install, plugin, macro, action, option, task ...), \nso that any developer can quickly pick it up and enjoy a productivity boost when developing and building projects.\n\n<img src="https://xmake.io/assets/img/index/package_manage.png" width="650px" />\n\nIf you want to know more, please refer to:\n\n* [Documents](https://xmake.io/#/home)\n* [HomePage](https://xmake.io)\n* [Github](https://github.com/xmake-io/xmake)\n* [Gitee](https://gitee.com/tboox/xmake)\n\n## Installation\n\n#### via curl\n\n```bash\nbash <(curl -fsSL https://raw.githubusercontent.com/xmake-io/xmake/master/scripts/get.sh)\n```\n\n#### via wget\n\n```bash\nbash <(wget https://raw.githubusercontent.com/xmake-io/xmake/master/scripts/get.sh -O -)\n```\n\n#### via powershell\n\n```bash\nInvoke-Expression (Invoke-Webrequest \'https://raw.githubusercontent.com/xmake-io/xmake/master/scripts/get.ps1\' -UseBasicParsing).Content\n```\n\n## Simple description\n\n<img src="https://xmake.io/assets/img/index/showcode1.png" width="340px" />\n\n## Package dependences\n\n<img src="https://xmake.io/assets/img/index/add_require.png" width="600px" />\n\nAn official xmake package repository: [xmake-repo](https://github.com/xmake-io/xmake-repo)\n\n## Build project\n\n```bash\n$ xmake\n```\n\n## Run target\n\n```bash\n$ xmake run console\n```\n\n## Debug target\n\n```bash\n$ xmake run -d console\n```\n\n## Configure platform\n\n```bash\n$ xmake f -p [windows|linux|macosx|android|iphoneos ..] -a [x86|arm64 ..] -m [debug|release]\n$ xmake\n```\n\n## Menu configuration\n\n```bash\n$ xmake f --menu\n```\n\n<img src="https://xmake.io/assets/img/index/menuconf.png" width="650px" />\n\n## Package management\n\n### Download and build\n\n<img src="https://xmake.io/assets/img/index/package_manage.png" width="650px" />\n\n### Processing architecture\n\n<img src="https://xmake.io/assets/img/index/package_arch.png" width="650px" />\n\n## Supported platforms\n\n* Windows (x86, x64)\n* macOS (i386, x86_64)\n* Linux (i386, x86_64, cross-toolchains ...)\n* Android (armv5te, armv6, armv7-a, armv8-a, arm64-v8a)\n* iOS (armv7, armv7s, arm64, i386, x86_64)\n* WatchOS (armv7k, i386)\n* MinGW (i386, x86_64)\n\n## Supported Languages\n\n* C\n* C++\n* Objective-C and Objective-C++\n* Swift\n* Assembly\n* Golang\n* Rust\n* Dlang\n* Cuda\n\n## Supported Projects\n\n* Static Library\n* Shared Library\n* Console\n* Cuda Program\n* Qt Application\n* WDK Driver (umdf/kmdf/wdm)\n* WinSDK Application\n* MFC Application\n\n## Builtin Plugins\n\n#### Generate IDE project file plugin\xef\xbc\x88makefile, vs2002 - vs2019 .. \xef\xbc\x89\n\n```bash\n$ xmake project -k vs2017 -m "debug,release"\n$ xmake project -k cmakelists\n$ xmake project -k compile_commands\n```\n\n#### Macros script plugin\n\n```bash\n$ xmake m -b                        # start to record\n$ xmake f -p iphoneos -m debug\n$ xmake\n$ xmake f -p android --ndk=~/files/android-ndk-r16b\n$ xmake\n$ xmake m -e                        # stop to record\n$ xmake m .                         # playback commands\n```\n\n#### Run the custom lua script plugin\n\n```bash\n$ xmake l ./test.lua\n$ xmake l -c "print(\'hello xmake!\')"\n$ xmake l lib.detect.find_tool gcc\n```\n\n#### Generate doxygen document plugin\n\n```bash\n$ xmake doxygen [srcdir]\n```\n\n## More Plugins\n\nPlease download and install from the plugins repository [xmake-plugins](https://github.com/xmake-io/xmake-plugins).\n\n## IDE/Editor Integration\n\n* "mode.release")\n\ntarget("console")\n    set_kind("binary")\n    add_files("src/*.c") \n    if is_mode("debug") then\n        add_defines("DEBUG")\n    end\n```\n\nDownload and use packages in [xmake-repo](https://github.com/xmake-io/xmake-repo):\n\n```lua\nadd_requires("libuv master", "ffmpeg", "zlib 1.20.*")\nadd_requires("tbox >1.6.1", {optional = true, debug = true})\ntarget("test")\n    set_kind("shared")\n    add_files("src/*.c")\n    add_packages("libuv", "ffmpeg", "tbox", "zlib")\n```\n\nDownload and use packages in third-party package manager:\n\n```lua\nadd_requires("brew::pcre2/libpcre2-8", {alias = "pcre2"})\nadd_requires("conan::OpenSSL/1.0.2n@conan/stable", {alias = "openssl"}) \ntarget("test")\n    set_kind("shared")\n    add_files("src/*.c")\n    add_packages("pcre2", "openssl")\n```\n\nFind and use local packages:\n\n```lua\ntarget("test")\n    set_kind("shared")\n    add_files("src/*.c")\n    on_load(function (target)\n        target:add(find_packages("zlib", "openssl", "brew::pcre2/libpcre2-8", "conan::OpenSSL/1.0.2n@conan/stable"))\n    end)\n```\n\n## Project Examples\n\nSome projects using xmake:\n\n*
