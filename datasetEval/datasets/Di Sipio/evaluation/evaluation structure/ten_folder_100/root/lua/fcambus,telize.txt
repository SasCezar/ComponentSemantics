It outputs\nJSON-encoded IP geolocation data, and supports both JSON and JSONP.\n\nGeolocation operations are performed using libmaxminddb which caches the\ndatabase in RAM. Therefore, Telize has very minimal overhead and should\nbe blazing fast.\n\n## Requirements\n\n### Dependencies\n\nTelize requires [Kore][1] 3.2.0+ and [libmaxminddb][2].\n\n### GeoIP2 databases\n\nTelize requires the free [GeoLite2 databases][3] from MaxMind.\n\n\tmkdir -p /var/db/GeoIP\n\tcd /var/db/GeoIP\n\twget https://geolite.maxmind.com/download/geoip/database/GeoLite2-City.tar.gz\n\twget https://geolite.maxmind.com/download/geoip/database/GeoLite2-ASN.tar.gz\n\ttar xfz GeoLite2-City.tar.gz\n\ttar xfz GeoLite2-ASN.tar.gz\n\tmv */*mmdb .\n\n## Building\n\nUsing the kodev tool:\n\n\tkodev build\n\n## Configuration\n\n## Running\n\nUsing the kodev tool:\n\n\tkodev run\n\n## Access and Error logs\n\nThe default Telize configuration does not have logging enabled, it must be\nconfigured manually.\n\nIf your Telize instance produces lots of logs, [Logswan][4] might be of\ninterest.\n\n## Telize and proxies\n\n## CORS Support (Cross-origin resource sharing)\n\nTelize has CORS enabled by default with the following policy:\n\n\tAccess-Control-Allow-Origin: *\n\n## Usage\n\nFor complete API documentation and usage examples, please check the\nproject site.\n\n### Get IP address in Plain text format\n\n- Example: http://127.0.0.1/ip\n\n### Get IP address in JSON format\n\n- Example (JSON): http://127.0.0.1/jsonip\n- Example (JSONP): http://127.0.0.1/jsonip?callback=getip\n\n### Get IP address location in JSON format\n\nCalling the API endpoint without any parameter will return the visitor\nIP address:\n\n- Example (JSON): http://127.0.0.1/location\n- Example (JSONP): http://127.0.0.1/location?callback=getgeoip\n\nAppending an IP address as parameter will return location information for\nthe given address:\n\n- Example (JSON): http://127.0.0.1/location/46.19.37.108\n- Example (JSONP): http://127.0.0.1/location/46.19.37.108?callback=getgeoip\n\n## License\n\nTelize is released under the BSD 2-Clause license.
