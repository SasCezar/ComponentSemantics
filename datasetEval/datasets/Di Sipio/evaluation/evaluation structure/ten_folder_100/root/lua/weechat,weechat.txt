All plugins (including https://weechat.org/files/doc/stable/weechat_user.en.html#irc_plugin[IRC]) are independent and can be unloaded.\n* *Multi-platform*: WeeChat runs on GNU/Linux, *BSD, GNU/Hurd, Haiku, macOS and Windows (Bash/Ubuntu and Cygwin).\n* *Multi-protocols*: WeeChat is designed to support multiple protocols by plugins, like IRC.\n* *Standards-compliant*: the IRC plugin is compliant with RFCs https://tools.ietf.org/html/rfc1459[1459], https://tools.ietf.org/html/rfc2810[2810], https://tools.ietf.org/html/rfc2811[2811], https://tools.ietf.org/html/rfc2812[2812] and https://tools.ietf.org/html/rfc2813[2813].\n* *Small, fast and very light*: the core is and should stay as light and fast as possible.\n* *Customizable and extensible*: there are a lot of options to customize WeeChat, and it is extensible with C plugins and https://weechat.org/scripts/[scripts] (https://weechat.org/scripts/stable/language/perl/[Perl], https://weechat.org/scripts/stable/language/python/[Python], https://weechat.org/scripts/stable/language/ruby[Ruby], https://weechat.org/scripts/stable/language/lua/[Lua], https://weechat.org/scripts/stable/language/tcl/[Tcl], https://weechat.org/scripts/stable/language/guile/[Scheme], https://weechat.org/scripts/stable/language/javascript/[JavaScript] and https://weechat.org/scripts/stable/language/php/[PHP]).\n* *Fully documented*: there is comprehensive https://weechat.org/doc/[documentation], which is https://weechat.org/files/doc/stable/weechat_dev.en.html#translations[translated] into several languages.\n* *Developed from scratch*: WeeChat was built from scratch and is not based on any other client.\n* *Free software*: WeeChat is released under https://www.gnu.org/licenses/gpl-3.0.html[GPLv3].\n\npass:[<p align="center">] https://weechat.org/about/screenshots/[more screenshots].\n\n== Install\n\n=== Dependencies\n\nFollowing packages are *required*:\n\n* CMake\n* libncurses\n* libcurl\n* zlib\n* libgcrypt\n\nFollowing packages are optional:\n\n* for i18n: gettext\n* for SSL: gnutls, ca-certificates\n* for spell checking: aspell or enchant\n* for scripting: python, perl, ruby, lua, tcl, guile, libv8 (javascript), php\n* for building doc and man page: asciidoctor\n* for building tests: C++ compiler, CppUTest\n\nFor a complete list of dependencies and versions recommended, please look at\nhttps://weechat.org/files/doc/devel/weechat_user.en.html#dependencies[user\'s guide].\n\n=== Compile\n\nWeeChat can be built with https://cmake.org/[CMake] (recommended) or autotools.\n\n[NOTE]\nOnly CMake is officially supported to build WeeChat. You should only use\nautotools if you are not able to use CMake. +\nBuilding with autotools requires more dependencies and is slower than with CMake.\n\n* Installation in system directories (requires _root_ privileges):\n\n----\n$ mkdir build\n$ cd build\n$ cmake ..\n$ make\n$ sudo make install\n----\n\n* Installation in custom directory (for example your home):\n\n----\n$ mkdir build\n$ cd build\n$ cmake .. -DCMAKE_INSTALL_PREFIX=/path/to/directory\n$ make\n$ make install\n----\n\nFor more information or installation with autotools, please look at\nhttps://weechat.org/files/doc/devel/weechat_user.en.html#compile_with_autotools[user\'s guide].\n\n=== Run tests\n\nFollowing packages are *required* to compile tests:\n\n* libcpputest-dev\n* C++ compiler\n\nTests must be enabled when compiling WeeChat:\n\n----\n$ cmake .. -DENABLE_TESTS=ON\n----\n\nThey can be launched after compilation from the build directory:\n\n----\n$ ctest -V\n----\n\n== Copyright\n\nCopyright (C) 2003-2019 S\xc3\xa9bastien Helleu <flashcode@flashtux.org>\n\nThis file is part of WeeChat, the extensible chat client.\n\nWeeChat is free software; you can redistribute it and/or modify\nit under the terms of the GNU General Public License as published by\nthe Free Software Foundation; either version 3 of the License, or\n(at your option) any later version.\n\nWeeChat is distributed in the hope that it will be useful,\nbut WITHOUT ANY WARRANTY; without even the implied warranty of\nMERCHANTABILITY or FITNESS FOR A PARTICULAR PURPOSE.
