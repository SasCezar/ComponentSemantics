b'<img src="http://www.cocos2d-x.org/attachments/801/cocos2dx_portrait.png" width=200>\n\n\ncocos2d-x\n=========\n\n|Win32|Others|\n| ----|------|\n[![Build status](https://ci.appveyor.com/api/projects/status/nlgirox464j6ldg5/branch/v3?svg=true)](https://ci.appveyor.com/project/minggo/cocos2d-x/branch/v3)|[![Build Status](https://travis-ci.org/cocos2d/cocos2d-x.svg?branch=v3)](https://travis-ci.org/cocos2d/cocos2d-x)\n\n\n[cocos2d-x][1] is a multi-platform framework for building 2d games, interactive books, demos and other graphical applications.\nIt is based on cocos2d-iphone, but instead of using Objective-C, it uses C++.\nIt works on iOS, Android, OS X, Windows, Linux and Web platforms.\n\n**Cocos2d-x Framework Architecture**:\n\n![](docs/framework_architecture.jpg "")\n\ncocos2d-x is:\n\n  * Fast\n  * Free\n  * Easy to use\n  * Community supported\n\nGit user attention\n-----------------------\n\n1. Clone the repo from GitHub.\n\n         $ git clone https://github.com/cocos2d/cocos2d-x.git\n\n2. After cloning the repo, please execute `download-deps.py` to download and install dependencies.\n\n         $ cd cocos2d-x\n         cocos2d-x $ python download-deps.py\n\n3. After running `download-deps.py`.\n\n         cocos2d-x $ git submodule update --init\n\nDownload stable versions\n-----------------------\n\n* [Cocos2d-x stable versions](http://www.cocos2d-x.org/download)\n\nHow to start a new game\n-----------------------\n\n1. Download the code from [cocos2d download site](http://www.cocos2d-x.org/download) or clone this repo (instructions above)\n2. Run `setup.py`\n3. Run the `cocos` script\n\nExample:\n\n    $ cd cocos2d-x\n    $ ./setup.py\n    $ source FILE_TO_SAVE_SYSTEM_VARIABLE\n    $ cocos new MyGame -p com.your_company.mygame -l cpp -d NEW_PROJECTS_DIR\n    $ cd NEW_PROJECTS_DIR/MyGame\n\nYou can also create a JS project or Lua project with `-l js` or `-l lua`.\n\n### Build and run a new project for Android ###\n\nCocos2d-x supports Android Studio. Simple open the `proj.android` directory from within the Android Studio environment. More information can be found in our [documentation](https://docs.cocos2d-x.org/cocos2d-x/v3/en/installation/Android-Studio.html).\n   \n### Build and run a new project for iOS ###\n\n    $ cocos run -p ios\n\n### Build and run a new project for OSX ###\n\n    $ cocos run -p mac\n\n### Build and run a new project for Linux ###\n\nIf you never run cocos2d-x on Linux, you need to install all dependencies by the\nscript in **cocos2d/build/install-deps-linux.sh**\n\n    $ cd cocos2d-x/build\n    $ ./install-deps-linux.sh\n\nThen\n\n    $ cd NEW_PROJECTS_DIR/MyGame\n    $ cocos run -p linux\n\nRun\n\n    $ bin/MyGame\n\n### Build and run new project for win32 ###\n\n    $ cocos run -p win32\n\n### Build and run new project for web ###\n\nOnly JS project can be published to web platforms, so you will need to create a JS project first:\n\n    $ cocos new -l js WebGame\n\nThen you can run your game in a web browser:\n\n    $ cocos run -p web\n\nOr you can publish your game to `publish/html5/` folder:\n\n    $ cocos run -p web -m release [--advanced]\n\nUsing CMake\n--------------------------------\n\nCocos2d-x supports CMake, a cross-platform build system. Example usage:\n\n    $ cd cocos2d-x\n    $ mkdir cmake-build && cd cmake-build\n    $ cmake ..\n\n* [Detail CMake Guide](cmake/README.md)\n\nDocumentations and samples\n-------------\n* [All Docs in   * Effects: Lens, Ripple, Waves, Liquid, etc.\n   * Actions (behaviours):\n     * Transformation Actions: Move, Rotate, Scale, Fade, Tint, etc.\n     * Composable actions: Sequence, Spawn, Repeat, Reverse\n     * Ease Actions: Exp, Sin, Cubic, Elastic, etc.\n     * Misc actions: CallFunc, OrbitCamera, Follow, Tween\n   * Basic menus and buttons\n   * Integrated with physics engines: [Box2d][5] and [Chipmunk][6]\n   * Particle system\n   * Skeleton Animations: [Spine][7] and Armature support\n   * Fonts:\n     * Fast font rendering using Fixed and Variable width fonts\n     * Support for .ttf fonts\n   * Tile Map support: Orthogonal, Isometric and Hexagonal\n   * Parallax scrolling\n   * Motion Streak\n   * Render To Texture\n   * Touch/Accelerometer on mobile devices\n   * Touch/Mouse/Keyboard on desktop\n   * Sound Engine support (CocosDenshion library) based on OpenAL\n   * Integrated Slow motion/Fast forward\n   * Fast and compressed textures: PVR compressed and uncompressed textures, ETC1 compressed textures, and more\n   * Resolution Independent\n   * Language: C++, with Lua and JavaScript bindings\n   * Open Source Commercial Friendly(MIT): Compatible with open and closed source projects\n   * OpenGL ES 2.0 (mobile) / OpenGL 2.1 (desktop) based\n\nBuild Requirements\n------------------\n\n* Mac OS X 10.7+, Xcode 8+\n* or Ubuntu 14.04+, CMake 3.1+\n* or
