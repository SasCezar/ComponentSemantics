b"# Temporary Containers Firefox Add-on [![Build Status](https://travis-ci.org/stoically/temporary-containers.svg?branch=master)](https://travis-ci.org/stoically/temporary-containers)\n\nDetailed information about the Add-on [can be found on AMO](https://addons.mozilla.org/firefox/addon/temporary-containers/). There's also [this long-form article](https://medium.com/@stoically/enhance-your-privacy-in-firefox-with-temporary-containers-33925cd6cd21) and the [project wiki](https://github.com/stoically/temporary-containers/wiki).\n\n## Development\n\n### Requirements\n\n- Clone the repository\n- `npm install`\n- `npm run watch`\n  - When doing UI dev and Parcel HMR is needed then set `security.csp.enable` to `false` in `about:config` and change back to `true` when you're done\n\n### Run in Firefox\n\n- `npx web-ext run -s dist`\n  - starts the default system Firefox with a temporary profile, loads the Add-on and watches for changes\n  - append `-p profilename` to start Firefox with a specific profile\n\nor\n\n- Open `about:debugging` and `Load Temporary Add-on` which is located in the `dist` directory\n\nCheck `about:debugging` and click `Inspect` to the right of Temporary Containers to see the console.\n\n### Run the tests\n\n- Once: `npm test`\n  - Shows a coverage summary and generates a detailed report in the `coverage` directory\n- Watcher: `npm run test:watch`\n\n### Release\n\n#### AMO and GitHub\n\n- Bump manifest version\n- Commit, tag and push\n- Upload zip web-ext-artifact to AMO\n- Download published AMO xpi\n- Create and publish GitHub release with AMO xpi\n\n#### Pre-Release on GitHub\n\n- Bump manifest version\n- Commit and push\n- git tag v1.0beta1\n- git push origin v1.0beta1\n- git log \\$(git tag --sort=-version:refname | sed -n 2p)..HEAD --pretty=format:%s\n- Add release notes and publish\n\n## Libraries\n\n[Vue.js](https://vuejs.org) and [SemanticUI](https://semantic-ui.com/) are used for the preferences & popup UI.\n\n## License\n\nMIT\n"
