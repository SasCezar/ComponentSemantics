b'## dockselpy\n\nDockerfile example on how to *"assemble"* together Selenium (with support for Chrome, Firefox and PhantomJS), Python and Xfvb.\n\n### Information\n\nRecent struggle with finding a docker image for Selenium that supports headless versions for both Firefox and Chrome, \nled to the process of building my own version.\n\nThe image is build with the following dependencies:\n- latest Chrome and chromedriver\n- latest Firefox and geckodriver\n- latest stable PhantomJS webkit (v2.1.1)\n- Selenium\n- Python 3\n- Xvfb and the python wrapper - pyvirtualdisplay\n\n\n### Running:\n\n- docker\n    ```\n    docker build -t selenium_docker .\n    docker run --privileged -p 4000:4000 -d -it selenium_docker \n    ```\n\n- docker-compose\n\n    ```\n    docker-compose stop && docker-compose build && docker-compose up -d\n    ```\n    \n    \n### Example\n\n```python\nfrom pyvirtualdisplay import Display\nfrom selenium import webdriver\n\ndisplay = Display(visible=0, size=(800, 600))\ndisplay.start()\n\nbrowser = webdriver.Firefox()\nbrowser.get(\'https://www.google.com/\')\nprint(browser.title)\n\nbrowser.quit()\ndisplay.stop()\n\n```\n\nDetailed examples on how to use Firefox with custom profile, Google Chrome with desired options or PhantomJS can be found in the source.\n'
