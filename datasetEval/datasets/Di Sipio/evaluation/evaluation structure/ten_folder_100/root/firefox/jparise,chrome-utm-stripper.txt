b'# Tracking Token Stripper\n\nThis is a Chrome and Firefox browser extension that strips Google Analytics\n(i.e. [Urchin Tracking Monitor][utm]) tokens, and Google and Facebook click\ntracking identifiers, from URL query strings. This is done *before* the web\nrequest is made and results in both more private browsing as well as more\naesthetically pleasing URLs.\n\nInstall from the [Chrome Web Store][store] or [Firefox Add-ons][addons].\n\nThe following query string parameters are stripped:\n\n - fbclid\n - gclid\n - _hsenc\n - _hsmi\n - ICID\n - mc_cid\n - mc_eid\n - mkt_tok\n - utm_source\n - utm_medium\n - utm_term\n - utm_campaign\n - utm_content\n - utm_cid\n - utm_reader\n - utm_referrer\n - utm_name\n - utm_social\n - utm_social-type\n\nThis extension requires these [permissions][]:\n\n - `webRequest`, to use the [`chrome.webRequest` API][webRequest]\n - `webRequestBlocking`, to use `chrome.webRequest` in a blocking fashion\n - `http://*/*?*`, to filter http URLs\n - `https://*/*?*`, to filter https URLs\n\n[![Urchin Logo](icon-128.png "Urchin Logo")](http://www.openclipart.org/detail/69997)\n\n[Urchin Logo](http://www.openclipart.org/detail/69997) by Jordan Irwin / Deluge.\n\n[utm]: https://en.wikipedia.org/wiki/UTM_parameters\n[store]: https://chrome.google.com/webstore/detail/kcpnkledgcbobhkgimpbmejgockkplob\n[addons]: https://addons.mozilla.org/addon/utm-tracking-token-stripper/\n[permissions]: https://developer.chrome.com/extensions/declare_permissions\n[webRequest]: https://developer.chrome.com/extensions/webRequest\n'
