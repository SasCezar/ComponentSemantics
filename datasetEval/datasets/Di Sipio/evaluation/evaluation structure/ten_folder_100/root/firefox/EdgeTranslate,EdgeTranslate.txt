the same package) from [GitHub Releases](https://github.com/EdgeTranslate/EdgeTranslate/releases).\n\nIt is always recommended to download the latest version.\n\n### Chrome\n\n* Extract the `.zip` package to somewhere on your disk.\n\n* Navigate to: `chrome://extensions`.\n\n* Enable `Developer mode`. It should be in the upper right corner of the page.\n\n* Click `Load unpacked` in the upper left corner.\n\n* Navigate to the directory of the extracted package.\n\n### QQ Browser\n\n* Extract the `.zip` package to somewhere on your disk.\n\n* Navigate to: `qqbrowser://extensions/manage`.\n\n* Enable `Developer mode`. It should be in the upper right corner of the page.\n\n* Click `Load unpacked` in the upper left corner.\n\n* Navigate to the directory of the extracted package.\n\n### Firefox\n\n* Installation will start automatically after the `.xapi` package is downloaded through Firefox browser.\n\n### 360 Browser\n\n* Right click on the package file and choose "Use 360 Browser to open", then it will be installed automatically.\n\n## Build It By Yourself\n\nTo build the extension, you need to have [Node.js](https://nodejs.org/) installed.\n\nClone the repository:\n\n```shell\ngit clone https://github.com/EdgeTranslate/EdgeTranslate.git\n```\n\nInstall dependencies:\n\n```shell\nnpm install\n```\n\nBuild Chrome version:\n\n```shell\nnpm run build:chrome\n```\n\nBuild Firefox version:\n\n```shell\nnpm run build:firefox\n```\n\nAfter building finished, you will get the unpacked extension under `./build/chrome/` and `./build/firefox/`.\n\n## Load Unpacked Extension In Your Browser\n\n### Chrome\n\n* Navigate to: `chrome://extensions`.\n\n* Enable `Developer mode`. It should be in the upper right corner of the page.\n\n* Click `Load unpacked` in the upper left corner.\n\n* Navigate to the repository you just cloned, select `build/chrome`.\n\n* Now you can try this extension in Chrome.\n\n### Firefox\n\n* Navigate to: `about:debugging`.\n\n* Check the `Enable add-on debugging` box if it\'s not checked.\n\n* Click `Load Temporary Add-on`.\n\n* Navigate to the repository you just cloned, open `build/firefox`, select any file in this directory.\n\n* Now you can try this extension in Firefox.\n\n## More Information\n\n[Wiki](./docs/wiki/en/Introduction.md)\n\n## Contact Us\n\nE-mails: [nickyc975](mailto:chenjinlong2016@outlook.com), [Mark Fenng](mailto:f18846188605@gmail.com)\n\n## License\n\n[MIT](./LICENSE.MIT) AND [NPL](./LICENSE.NPL)'
