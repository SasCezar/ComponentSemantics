b"# News Feed Eradicator\n\nA browser extension that deletes your Facebook news feed\nand replaces it with a nice quote.\n\n[Install Chrome Extension](https://chrome.google.com/webstore/detail/news-feed-eradicator-for/fjcldmjmjhkklehbacihaiopjklihlgg?hl=en)\n\n[Install Firefox Add-on](https://addons.mozilla.org/en-US/firefox/addon/news-feed-eradicator/)\n\n![Screenshot](https://raw.githubusercontent.com/jordwest/news-feed-eradicator/master/assets/screenshot.jpg)\n\n## Development\n\nThis plugin is built as a WebExtension - a standard for browser plugins currently supported in both Chrome and Firefox.\n\nTo build for either browser, clone the repository and then run:\n\n    make dev\n\nIf everything is successful, check the `build` folder for the extension contents. You can load the `build` directory into either Chrome or Firefox as an _unpacked_ or _temporary_ extension. See the instructions for [Chrome](https://developer.chrome.com/extensions/getstarted#unpacked) or [Firefox](https://developer.mozilla.org/en-US/Add-ons/WebExtensions/Temporary_Installation_in_Firefox).\n\nRunning `make dev` will watch for changes and recompile, however each time you make changes you'll need to tell the browser to reload the temporary extension.\n\nTo build a distributable `.zip` for production, just run:\n\n    make\n\nThe extension package can be found in the `dist` folder.\n"
