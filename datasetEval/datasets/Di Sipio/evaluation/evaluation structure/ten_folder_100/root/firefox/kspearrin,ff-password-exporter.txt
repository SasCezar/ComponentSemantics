b'# FF Password Exporter\n\nIt can be difficult to export your passwords from Firefox. Since version 57 of Firefox (Quantum) existing password export addons  no longer work. Mozilla provides no other official alternatives. FF Password Exporter makes it quick and easy to export all of your passwords from Firefox. You can use FF Password Exporter on Windows, macOS, and Linux distributions.\n\n![FF Password Exporter](http://imgur.com/xjPNUOJ.png "FF Password Exporter")\n\n## Download/Install\n\n- Windows 7, 8, 10 and later\n  - [Portable (.exe, x86 and x64)](https://github.com/kspearrin/ff-password-exporter/releases/download/v1.1.1/FF-Password-Exporter-Portable-1.1.1.exe)\n  - [Installer (.exe, x86 and x64)](https://github.com/kspearrin/ff-password-exporter/releases/download/v1.1.1/FF-Password-Exporter-Installer-1.1.1.exe)\n- macOS (OS X) Mavericks and later \n  - [Portable (.app, x64)](https://github.com/kspearrin/ff-password-exporter/releases/download/v1.1.1/ff-password-exporter-1.1.1-mac.zip)\n  - [Installer (.dmg, x64)](https://github.com/kspearrin/ff-password-exporter/releases/download/v1.1.1/FF-Password-Exporter-1.1.1.dmg)\n- Linux\n  - [Portable (.AppImage, x64)](https://github.com/kspearrin/ff-password-exporter/releases/download/v1.1.1/FF-Password-Exporter-1.1.1-x86_64.AppImage)\n\n## How to Use\n\n1. Download and install/run FF Password Exporter. Use the links above.\n2. Choose the Firefox user\'s profile directory you want to export passwords from.\n3. If you have set a master password to protect your Firefox passwords, enter it.\n4. Click the export button.\n5. Choose your file format (CSV or JSON), and save the file to your device.\n\n## Supported Firefox Versions\n\n- Firefox 58+ with `key4.db` profiles\n\n## Contributing\n\nFF Password Exporter is written in Electron with JavaScript. Contributions/pull requests are welcome!\n\n**Requirements**\n\n- [Node.js](https://nodejs.org/)\n\n**Run the app**\n\n```bash\nnpm install\nnpm run electron\n```\n'
