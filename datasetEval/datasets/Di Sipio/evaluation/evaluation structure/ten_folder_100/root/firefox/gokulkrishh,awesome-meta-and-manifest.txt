Adding ```192px``` size icon will scale perfectly for most of the devices.\n- ```gcm_user_visible_only``` key removed in [Chrome 45](https://www.chromestatus.com/feature/5778950739460096) favor of the specified solution: [```userVisibleOnly```](https://developer.mozilla.org/en-US/docs/Web/API/PushManager/subscribe). \n\n#### `iOS`\n\n- In safari mobile browser, add to home screen will add black background for icon if its in PNG format. Make it as JPG to work.\n- Safari doesn\'t support manifest.json for add to home screen yet.\n\n#### `UC Browser`\n\n- Meta tag ```browsermode``` is not working.\n- Link tag ```icon``` for home screen is not working.\n\n### Tools\n\n- [Manifest Validator](https://manifest-validator.appspot.com)\n- [Add To Home Icon Generator](https://romannurik.github.io/AndroidAssetStudio/index.html)\n- [Icon Size Generator](http://realfavicongenerator.net/)\n\n### References\n\n- [Apple Developer Site](https://developer.apple.com/library/ios/documentation/AppleApplications/Reference/SafariWebContent/ConfiguringWebApplications/ConfiguringWebApplications.html)\n- [Web App Manifest](https://developers.google.com/web/fundamentals/web-app-manifest/)\n- [UC Browser](http://www.uc.cn/download/UCBrowser_U3_API.doc)\n- [Microsoft Internet Explorer](https://github.com/joshbuchea/HEAD#microsoft-internet-explorer)\n- [PWACompat](https://github.com/GoogleChromeLabs/pwacompat) - to add support for splash screen in any device.\n\n#### Contributions\n\n- If you wish to contribute to this repository, fork it and send a PR \xf0\x9f\x98\xac.\n- And, if you like the repo, \xf0\x9f\x8c\x9f it.\n\n##### MIT Licensed\n'
