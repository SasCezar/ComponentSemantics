b'# Mint\n[![Build Status](https://travis-ci.org/mint-lang/mint.svg?branch=master)](https://travis-ci.org/mint-lang/mint)\n[![Gitter chat](https://badges.gitter.im/gitterHQ/gitter.png)](https://gitter.im/mint-lang/Lobby)\n[![Join the community on Spectrum](https://withspectrum.github.io/badge/badge.svg)](https://spectrum.chat/mint-lang)\n[![Backers on Open Collective](https://opencollective.com/mint/backers/badge.svg)](#backers)\n[![Sponsors on Open Collective](https://opencollective.com/mint/sponsors/badge.svg)](#sponsors)\n[![FOSSA Status](https://app.fossa.io/api/projects/git%2Bgithub.com%2Fmint-lang%2Fmint.svg?type=shield)](https://app.fossa.io/projects/git%2Bgithub.com%2Fmint-lang%2Fmint?ref=badge_shield)\n\n------\n\nA refreshing programming language for the front-end web, aiming to solve the most common issues of **Single Page Applications (SPAs)** at a language level:\n* Reusable components\n* Styling\n* Routing\n* Global and local state handling\n* Synchronous and asynchronous computations that might fail\n\nWhile focusing on:\n* Developer happiness\n* Fast compilation\n* Readability\n\nProject Status\n--------------\nThe project is in development, we are still tweaking the language and standard library. \n\nThere are two bigger applications which can be used as examples / learning material:\n\n* the Mint website (~3100 LOC) [SOURCE](https://github.com/mint-lang/mint-website)\n* the Mint implementation of [realworld.io](https://github.com/gothinkster/realworld) (~3300 LOC) - [DEMO](https://mint-realworld.netlify.com) [SOURCE](https://github.com/mint-lang/mint-realworld)  \n\nInstalling\n----------\n[Follow these instructions](https://www.mint-lang.com/install)\n\nDocumentation\n----------\n* [Official docs](https://guide.mint-lang.com)\n* [Roadmap](https://www.mint-lang.com/roadmap)\n\nCommunity\n---------\nQuestions or suggestions?
