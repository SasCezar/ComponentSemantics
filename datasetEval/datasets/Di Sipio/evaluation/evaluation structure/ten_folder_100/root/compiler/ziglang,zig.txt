b'![ZIG](https://ziglang.org/zig-logo.svg)\n\nA general-purpose programming language for maintaining **robust**, **optimal**,\nand **reusable** code.\n\n## Resources\n\n * [Introduction](https://ziglang.org/#Introduction)\n * [Download & Documentation](https://ziglang.org/download)\n * [Community](https://github.com/ziglang/zig/wiki/Community)\n * [Contributing](https://github.com/ziglang/zig/blob/master/CONTRIBUTING.md)\n * [Frequently Asked Questions](https://github.com/ziglang/zig/wiki/FAQ)\n * [Community Projects](https://github.com/ziglang/zig/wiki/Community-Projects)\n\n## Building from Source\n\n[![Build Status](https://dev.azure.com/ziglang/zig/_apis/build/status/ziglang.zig?branchName=master)](https://dev.azure.com/ziglang/zig/_build/latest?definitionId=1&branchName=master)\n\nNote that you can\n[download a binary of master branch](https://ziglang.org/download/#release-master).\n\n### Stage 1: Build Zig from C++ Source Code\n\n#### Dependencies\n\n##### POSIX\n\n * cmake >= 2.8.5\n * gcc >= 5.0.0 or clang >= 3.6.0\n * LLVM, Clang, LLD development libraries == 9.x, compiled with the same gcc or clang version above\n   - Use the system package manager, or [build from source](https://github.com/ziglang/zig/wiki/How-to-build-LLVM,-libclang,-and-liblld-from-source#posix).\n\n##### Windows\n\n * cmake >= 3.15.3\n * Microsoft Visual Studio. Supported versions:\n   - 2015 (version 14)\n   - 2017 (version 15.8)\n   - 2019 (version 16)\n * LLVM, Clang, LLD development libraries == 9.x\n   - Use the [pre-built binaries](https://github.com/ziglang/zig/wiki/Building-Zig-on-Windows) or [build from source](https://github.com/ziglang/zig/wiki/How-to-build-LLVM,-libclang,-and-liblld-from-source#windows).\n\n#### Instructions\n\n##### POSIX\n\n```\nmkdir build\ncd build\ncmake ..\nmake install\n```\n\n##### MacOS\n\n```\nbrew install cmake llvm@9\nbrew outdated llvm@9 || brew upgrade llvm@9\nmkdir build\ncd build\ncmake .. -DCMAKE_PREFIX_PATH=$(brew --prefix llvm)\nmake install\n```\n\n##### Windows\n\nSee https://github.com/ziglang/zig/wiki/Building-Zig-on-Windows\n\n### Stage 2: Build Self-Hosted Zig from Zig Source Code\n\n*Note: Stage 2 compiler is not complete. Beta users of Zig should use the\nStage 1 compiler for now.*\n\nDependencies are the same as Stage 1, except now you can use stage 1 to compile\nZig code.\n\n```\nbin/zig build --prefix $(pwd)/stage2\n```\n\nThis produces `./stage2/bin/zig` which can be used for testing and development.\nOnce it is feature complete, it will be used to build stage 3 - the final compiler\nbinary.\n\n### Stage 3: Rebuild Self-Hosted Zig Using the Self-Hosted Compiler\n\n*Note: Stage 2 compiler is not yet able to build Stage 3. Building Stage 3 is\nnot yet supported.*\n\nOnce the self-hosted compiler can build itself, this will be the actual\ncompiler binary that we will install to the system. Until then, users should\nuse stage 1.\n\n#### Debug / Development Build\n\n```\n./stage2/bin/zig build --prefix $(pwd)/stage3\n```\n\n#### Release / Install Build\n\n```\n./stage2/bin/zig build install -Drelease\n```\n'
