b'Hy\n==\n\n[![Version](https://img.shields.io/pypi/v/hy.svg)](https://pypi.python.org/pypi/hy)\n\n<a href="https://xkcd.com/224/"><img title="We lost the documentation on quantum mechanics. You\'ll have to decode the regexes yourself." alt="XKCD #224" src="https://raw.github.com/hylang/shyte/18f6925e08684b0e1f52b2cc2c803989cd62cd91/imgs/xkcd.png"></a>\n\nLisp and Python should love each other. Let\'s make it happen.\n\nHy is a Lisp dialect that\'s embedded in Python. Since Hy transforms its Lisp\ncode into Python abstract syntax tree (AST) objects, you have the whole\nbeautiful world of Python at your fingertips, in Lisp form.\n\nTo install the latest stable release of Hy, just use the command `pip3 install\n--user hy`. Then you can start an interactive read-eval-print loop (REPL) with\nthe command `hy`, or run a Hy program with `hy myprogram.hy`.\n\n* [Why Hy?](http://docs.hylang.org/en/master/whyhy.html)\n* [Tutorial](http://docs.hylang.org/en/master/tutorial.html)\n\nProject\n-------\n\n* Code: https://github.com/hylang/hy\n* Documentation:\n  * stable, for use with the latest stable release: http://hylang.org/\n  * master, for use with the latest revision on GitHub: http://docs.hylang.org/en/master\n* Bug reports: We have no bugs!
