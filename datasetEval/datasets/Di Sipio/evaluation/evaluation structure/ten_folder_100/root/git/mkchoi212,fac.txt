It has a prompt input at the bottom of the screen where the user inputs various commands.\r\n\r\nThe commands have been preset to the following specifications\r\n\r\n| Keybinding                      | Description          |\r\n| ------------------------------- | -------------------- |\r\n| <kbd>w</kbd>                    | show more lines up   |\r\n| <kbd>s</kbd>                    | show more lines down |\r\n| <kbd>a</kbd>                    | use local version    |\r\n| <kbd>d</kbd>                    | use incoming version |\r\n| <kbd>e</kbd>                    | manually edit code   |\r\n| <kbd>j</kbd>                    | scroll down          |\r\n| <kbd>k</kbd>                    | scroll up            |\r\n| <kbd>v</kbd>                    | [v]iew orientation   |\r\n| <kbd>n</kbd>                    | [n]ext conflict      |\r\n| <kbd>p</kbd>                    | [p]revious conflict  |\r\n| <kbd>h</kbd>, <kbd>?</kbd>      | [h]elp               |\r\n| <kbd>q</kbd>, <kbd>Ctrl+c</kbd> | [q]uit               |\r\n\r\n```\r\n[w,a,s,d,?] >> [INPUT HERE]\r\n```\r\n\r\n### \xe2\x9a\x99\xef\xb8\x8f Customization\r\n\r\nThe above key-bindings and various behaviors can be altered by creating a `.fac.yml` file in your home directory.\r\nPlease refer to the [configuration README.md](./binding/README.md) for more information.\r\n\r\n## \xe2\x9c\x8b Contributing\r\n\r\nThis is an open source project so feel free to contribute by\r\n\r\n- Opening an [issue](https://github.com/mkchoi212/fac/issues/new)\r\n- Sending me feedback via [email](mailto://mkchoi212@icloud.com)\r\n- Or [tweet](https://twitter.com/Bananamlkshake2) at me!\r\n\r\n## \xf0\x9f\x91\xae License\r\n\r\nSee [License](./LICENSE)\r\n'
