b'# GitHub Extension for Visual Studio\n\n## About\n\nThe GitHub Extension for Visual Studio provides GitHub integration in Visual Studio 2015 and newer.\nMost of the extension UI lives in the Team Explorer pane, which is available from the View menu.\n\nOfficial builds of this extension are available at the [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=GitHub.GitHubExtensionforVisualStudio).\n\n[![Build Status](https://github-editor-tools.visualstudio.com/VisualStudio/_apis/build/status/github.VisualStudio?branchName=master)](https://github-editor-tools.visualstudio.com/VisualStudio/_build/latest?definitionId=10&branchName=master)\n\n[![Follow GitHub for Visual Studio](https://img.shields.io/twitter/follow/GitHubVS.svg?style=social "Follow GitHubVS")](https://twitter.com/githubvs?ref_src=twsrc%5Etfw) [![Join the chat at https://gitter.im/github/VisualStudio](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/github/VisualStudio?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)\n\n## Documentation\nVisit the [documentation](https://github.com/github/VisualStudio/tree/master/docs) for details on how to use the features in the GitHub Extension for Visual Studio.\n\n## Build requirements\n\n* Visual Studio 2019\n  * `.NET desktop development` workload\n  * `.NET Core cross platform development` workload\n  * `Visual Studio extension development` workload\n\nThe built VSIX will work with Visual Studio 2015 or newer\n\n## Build\n\nClone the repository and its submodules.\n\nTo be able to use the GitHub API, you\'ll need to:\n\n- [Register a new developer application](https://github.com/settings/developers) in your profile\n- Create an environment variable `GitHubVS_ClientID` with your `Client ID`\n- Create an environment variable `GitHubVS_ClientSecret` with your `Client Secret`\n\nExecute `build.cmd`\n\n## Visual Studio Build\n\nBuild `GitHubVS.sln` using Visual Studio 2019.\n\n## Logs\nLogs can be viewed at the following location:\n\n`%LOCALAPPDATA%\\GitHubVisualStudio\\extension.log`\n\n## Troubleshooting\n\nIf you have issues building with failures similar to:\n\n> "The type or namespace name does not exist..."\n\nor\n\n> "Unable to find project... Check that the project reference is valid and that the project file exists."*\n\nClose Visual Studio and run the following command to update submodules and clean your environment.\n\n```txt\nclean.cmd\n```\n\n## More information\n- Andreia Gaita\'s [presentation](https://www.youtube.com/watch?v=hz2hCO8e_8w) at Codemania 2016 about this extension.\n\n## Contributing\n\nVisit the [Contributor Guidelines](CONTRIBUTING.md) for details on how to contribute as well as the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md) for details on how to participate.\n\n## Copyright\n\nCopyright 2015 - 2019 GitHub, Inc.\n\nLicensed under the [MIT License](LICENSE.md)\n'
