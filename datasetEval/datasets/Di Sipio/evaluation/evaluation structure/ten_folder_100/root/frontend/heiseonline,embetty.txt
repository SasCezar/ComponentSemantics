b'# embetty \xc2\xb7 [![Build Status](https://travis-ci.org/heiseonline/embetty.svg?branch=master)](https://travis-ci.org/heiseonline/embetty) [![Greenkeeper badge](https://badges.greenkeeper.io/heiseonline/embetty.svg)](https://greenkeeper.io/) [![Dependency Status](https://img.shields.io/david/heiseonline/embetty.svg?style=flat-square)](https://david-dm.org/heiseonline/embetty) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)\n\n<p align="center">\n  <img alt="Embetty" src="assets/embetty.png" width="360">\n</p>\n\nEmbetty displays remote content like tweets or videos without compromising your privacy.\n\nSee it in action on our [demo pages](https://heiseonline.github.io/embetty/).\n\n## Quick Start\n\n1. Setup your [embetty-server](https://github.com/heiseonline/embetty-server).\n3. Include the embetty lib into your HTML document.\n4. Insert an embed by using a custom tag (see embeds section below).\n\nExample:\n\n```html\n<!DOCTYPE html>\n<html lang="en">\n<head>\n  <meta data-embetty-server="/path/to/embetty-server">\n  <script async src="embetty.js"></script>\n</head>\n<body>\n  <embetty-tweet status="1166685910030790662"></embetty-tweet>\n</body>\n</html>\n```\n\n## Documentation\n\n### Configure the Embetty server\n\nEmbetty needs a [server component](https://github.com/heiseonline/embetty-server) that you need to run on your infrastructure. Configure the server URL for embetty using a `<meta data-embetty-server>` tag:\n\n```html\n<head>\n  <meta data-embetty-server="/path/to/embetty-server">\n</head>\n```\n\n### Including embetty.js\n\nThere are three options.\n\n1. Download an Embetty archive from the [releases page](https://github.com/heiseonline/embetty/releases). Make `embetty.js` available on your site.\n2. Use Embetty in your npm project: `yarn add @heise/embetty` or `npm install @heise/embetty --save`. Then import embetty into your main script (i.e. `import \'@heise/embetty\'`).\n3. Clone this repository and build `./dist/embetty.js`:\n   ```sh\n   $ git clone https://github.com/heiseonline/embetty\n   $ cd embetty\n   $ yarn\n   $ yarn build\n   ``` \n\n### Supported embed types\n\nCurrently, tweets and various video platforms are supported.\n\n#### Tweet\n\nUse the `status` attribute to embed a tweet with its tweet ID. Example:\n\n```html\n<embetty-tweet status="950371792874557440"></embetty-tweet>\n```\n\nUse the `include-thread` attribute to include the thread _above_ the tweet (i.e. the conversation so far, for context). Example:\n\n```html\n<embetty-tweet status="950371792874557440" include-thread></embetty-tweet>\n```\n\n#### Video\n\nUse the `type` attribute with a value of `facebook`, `vimeo` or `youtube`. Set the `video-id` attribute to the video ID.\n\n```html\n<embetty-video type="facebook" video-id="10156049485672318"></embetty-video>\n```\n\nFor videos of type `vimeo` or `youtube`, it\'s also possible to set a `start-at` attribute with a value of time in seconds to start the video at a specific timecode.\n\n```html\n<embetty-video type="youtube" start-at="96" video-id="3L4fHrIJ3A4"></embetty-video>\n<embetty-video type="vimeo" start-at="96" video-id="223099532"></embetty-video>\n```\n\n### Events\n\nEmbetty triggers the following events:\n\n| Name | Description |\n|------|-------------|\n| `initialized` | The Embetty embed will enter the viewport *after* this has been triggered. |\n| `activated` | The Embetty **video** embed has been replaced with an iframe containing the original video player. The content of the iframe may still be loading. |\n\nExample code:\n\n```js\ndocument.querySelector(\'embetty-tweet\').addEventistener(\'initialized\', function(e) { ... })\n```\n\n## Testing\n\n1. Clone this repository.\n2. `yarn`\n3. export the `TWITTER_` tokens mentioned in https://github.com/heiseonline/embetty-server\n4. `yarn test`\n\n\n## Caveats\n\nThe Embetty server component **does not proxy video data**. This means that **the tracking protection becomes ineffective after the play button has been clicked**.\n\n## License\n\nEmbetty is [MIT licensed](./LICENSE).\n'
