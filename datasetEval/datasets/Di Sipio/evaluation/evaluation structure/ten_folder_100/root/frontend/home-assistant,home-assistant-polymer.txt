b'# Home Assistant Polymer [![Build Status](https://travis-ci.org/home-assistant/home-assistant-polymer.svg?branch=master)](https://travis-ci.org/home-assistant/home-assistant-polymer)\n\nThis is the repository for the official [Home Assistant](https://home-assistant.io) frontend.\n\n[![Screenshot of the frontend](https://raw.githubusercontent.com/home-assistant/home-assistant-polymer/master/docs/screenshot.png)](https://demo.home-assistant.io/)\n\n- [View demo of Home Assistant](https://demo.home-assistant.io/)\n- [More information about Home Assistant](https://home-assistant.io)\n- [Frontend development instructions](https://developers.home-assistant.io/docs/en/frontend_index.html)\n\n## Development\n\n- Initial setup: `script/setup`\n- Development: [Instructions](https://developers.home-assistant.io/docs/en/frontend_development.html)\n- Production build: `script/build_frontend`\n- Gallery: `cd gallery && script/develop_gallery`\n- Hass.io: [Instructions](https://developers.home-assistant.io/docs/en/hassio_hass.html)\n\n## Frontend development\n\n### Classic environment\nA complete guide can be found at the following [link](https://www.home-assistant.io/developers/frontend/). It describes a short guide for the build of project.\n\n### Docker environment\nIt is possible to compile the project and/or run commands in the development environment having only the [Docker](https://www.docker.com) pre-installed in the system. On the root of project you can do:\n* `sh ./script/docker_run.sh build` Build all the project with one command\n* `sh ./script/docker_run.sh bash` Open an interactive shell (the same environment generated by the *classic environment*) where you can run commands. This bash work on your project directory and any change on your file is automatically present within your build bash.\n\n**Note**: if you have installed `npm` in addition to the `docker`, you can use the commands `npm run docker_build` and `npm run bash` to get a full build or bash as explained above\n\n## License\n\nHome Assistant is open-source and Apache 2 licensed. Feel free to browse the repository, learn and reuse parts in your own projects.\n\nWe use [BrowserStack](https://www.browserstack.com) to test Home Assistant on a large variation of devices.\n'
