be used by using the package manager or downloading the source directly.\nHowever, we highly recommend using the package manager.\n\n### Via Package Manager\n\nTOAST UI products are registered in two package managers, [npm](https://www.npmjs.com/) and [bower](https://bower.io/).\nYou can conveniently install it using the commands provided by each package manager.\nWhen using npm, be sure to use it in the environment [Node.js](https://nodejs.org) is installed.\n\n#### npm\n\n``` sh\n$ npm install --save tui-calendar # Latest version\n$ npm install --save tui-calendar@<version> # Specific version\n```\n\n#### bower\n\n``` sh\n$ bower install tui-calendar # Latest version\n$ bower install tui-calendar#<tag> # Specific version\n```\n\n### Via Contents Delivery Network (CDN)\nTOAST UI products are available over the CDN powered by [TOAST Cloud](https://www.toast.com).\n\nYou can use the CDN as below.\n\nInsert style sheet files\n```html\n<link rel="stylesheet" type="text/css" href="https://uicdn.toast.com/tui-calendar/latest/tui-calendar.css" />\n\n<!-- If you use the default popups, use this. -->\n<link rel="stylesheet" type="text/css" href="https://uicdn.toast.com/tui.date-picker/latest/tui-date-picker.css" />\n<link rel="stylesheet" type="text/css" href="https://uicdn.toast.com/tui.time-picker/latest/tui-time-picker.css" />\n```\n\nInsert JavaScript file\n```html\n<script src="https://uicdn.toast.com/tui.code-snippet/latest/tui-code-snippet.js"></script>\n<script src="https://uicdn.toast.com/tui.dom/v3.0.0/tui-dom.js"></script>\n<script src="https://uicdn.toast.com/tui.time-picker/latest/tui-time-picker.min.js"></script>\n<script src="https://uicdn.toast.com/tui.date-picker/latest/tui-date-picker.min.js"></script>\n<script src="https://uicdn.toast.com/tui-calendar/latest/tui-calendar.js"></script>\n```\n\nIf you want to use a specific version, use the tag name instead of `latest` in the url\'s path.\n\nThe CDN directory has the following structure.\n\n```\ntui-calendar/\n\xe2\x94\x9c\xe2\x94\x80 latest/\n\xe2\x94\x82
