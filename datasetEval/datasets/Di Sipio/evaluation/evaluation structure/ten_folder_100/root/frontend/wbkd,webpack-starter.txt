b'# Webpack Frontend Starterkit\n\n[![Dependabot badge](https://flat.badgen.net/dependabot/wbkd/webpack-starter?icon=dependabot)](https://dependabot.com/)\n\nA lightweight foundation for your next webpack based frontend project.\n\n\n### Installation\n\n```\nnpm install\n```\n\n### Start Dev Server\n\n```\nnpm start\n```\n\n### Build Prod Version\n\n```\nnpm run build\n```\n\n### Features:\n\n* ES6 Support via [babel](https://babeljs.io/) (v7)\n* SASS Support via [sass-loader](https://github.com/jtangelder/sass-loader)\n* Linting via [eslint-loader](https://github.com/MoOx/eslint-loader)\n\nWhen you run `npm run build` we use the [mini-css-extract-plugin](https://github.com/webpack-contrib/mini-css-extract-plugin) to move the css to a separate file. The css file gets included in the head of the `index.html`.\n'
