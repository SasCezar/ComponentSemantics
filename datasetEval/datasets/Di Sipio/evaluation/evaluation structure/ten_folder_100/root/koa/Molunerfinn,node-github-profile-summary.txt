b'# node-github-profile-summary\n\n<p align="center">\n  <img src="https://user-images.githubusercontent.com/12621342/35959365-41f352b0-0ce0-11e8-8a5b-05ad054ea378.png">\n</p>\n\n> The node version of github-profile-summary which is inspired by tipsy\' [github-profile-summary](https://github.com/tipsy/github-profile-summary)\n\nSite: https://gh-profile-summary.teamsz.xyz\n\n## Major skills or tools:\n\n- Vue\n- Chart.js\n- Koa\n- Github graphql api\n\n## Screenshot\n\n### Cool & beautiful profile list\n\n![](https://user-images.githubusercontent.com/12621342/35951697-89e3338e-0cb7-11e8-9986-dc258f257b97.png)\n\n### Share system & Save your profile as Image\n\n![](https://user-images.githubusercontent.com/12621342/35951773-d1c9cf50-0cb7-11e8-80b2-08ae7d876533.png)\n\n## Setup\n\nFirst generate a token at https://github.com/settings/tokens\n\nThen You need to create a `.env` file in the project folder.\n\nThe `.env` file looks like:\n\n```env\nGH_TOKEN=Your token here\nJWT_SECRET=Any string\nKOA_PORT=8888\nHTTPS=false\n```\n\n> KOA_PORT can be any port you like which is available.\n\n**If you set `HTTPS=true`, then the websocket will connect the 443 port, it\'s useful for https production.**\n\n## Local run\n\n### Node.js\n\n> You need to have node environment!\n\n### Install\n\n``` bash\ngit clone https://github.com/Molunerfinn/node-github-profile-summary.git\ncd node-github-profile-summary\nnpm install # or yarn\n```\n\n### Run\n\n```bash\nnpm run build\nnpm start\n```\n\n**Notice:** 8888 is the `KOA_PORT` in `.env` file\n\nAnd then open the link `http://localhost:8888`, Done.\n\n### Docker \n\n> You need to have docker environment!\n\n### Build\n\n```bash\ndocker build -t github-profile .\n```\n\n### Run\n\n```bash\ndocker run -it --rm --name github-profile -p 8888:8888 github-profile\n```\n\n**Notice:** 8888 is the `KOA_PORT` in `.env` file\n\nAnd then open the link `http://localhost:8888`, Done.\n\n## TODOS\n\n- More useful charts\n- ~~Docker support~~\n- ...\n\n\n## License\n\n[MIT](http://opensource.org/licenses/MIT)\n\nCopyright (c) 2018 Molunerfinn\n\n\n'
