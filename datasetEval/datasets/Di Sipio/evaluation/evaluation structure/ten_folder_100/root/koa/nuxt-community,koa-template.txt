b"# Nuxt.js with Koa\n\n> [KoaJS](http://koajs.com/) + [Nuxt.js](https://nuxtjs.org) = :zap:\n\n## Installation\n\nThis is a project template for [vue-cli](https://github.com/vuejs/vue-cli).\n\n```bash\nvue init nuxt-community/koa-template <project-name>\ncd <project-name> # move to your project\nnpm install # or yarn install*[see note below]\n```\n\n> Make sure to use a version of vue-cli >= 2.1 (vue -V).\n\n*Note: Due to a bug in yarn's engine version detection code if you are\nusing a prerelease version of Node (i.e. v7.6.0-rc.1) you will need to either:\n\n  1. Use `npm install`\n  2.
