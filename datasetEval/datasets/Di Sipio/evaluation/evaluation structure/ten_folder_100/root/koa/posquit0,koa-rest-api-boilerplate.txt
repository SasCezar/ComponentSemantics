template for building RESTful API application with Koa.\n\nThis boilerplate include the following features:\n\n- Logging to STDOUT/STDERR stream using [Pino](http://getpino.io/)\n- A super small and optimized [Docker](https://www.docker.com/) image based on [Node.js Alpine image](https://hub.docker.com/_/node/)\n- [Swagger](https://swagger.io/) API documentation based on JSDoc\n- Continuous integration and delivery using [CircleCI](https://circleci.com/)\n- Unit Test and Integration Test along with Test Coverage using [Jest](https://facebook.github.io/jest/) testing framework\n\n---\n\n\n## Getting Started\n\n```zsh\n$ git clone https://github.com/posquit0/koa-rest-api-boilerplate your-project-name\n$ cd your-project-name\n$ rm -rf .git && git init\n```\n\n```zsh\n$ yarn\n$ yarn start\n```\n\n\n## Commands\n\n### Run\n\n```zsh\n# Run normally\n$ yarn start\n# Run the application with nodemon for development\n$ yarn dev\n```\n\n### Test\n\n```zsh\n# Test\n$ yarn test                           # Run all test\n$ yarn test:unit                      # Run only unit test\n$ yarn test:integration               # Run only integration test\n# Test (Watch Mode for development)\n$ yarn test:watch                     # Run all test with watch mode\n$ yarn test:watch:unit                # Run only unit test with watch mode\n$ yarn test:watch:integration         # Run only integration test with watch mode\n# Test Coverage\n$ yarn test:coverage                  # Calculate the coverage of all test\n$ yarn test:coverage:unit             # Calculate the coverage of unit test\n$ yarn test:coverage:integration      # Calculate the coverage of integration test\n# Test consistent coding style (Lint)\n$ yarn lint                           # Lint all sourcecode\n$ yarn lint:app                       # Lint app sourcecode\n$ yarn lint:test                      # Lint test sourcecode\n```\n\n### Archive\n\n```zsh\n$ yarn pack\n```\n\n\n## Test\n\nAll test for this boilerplate uses following tools.\n\n- [Jest](https://github.com/facebook/jest)\n- [supertest](https://github.com/visionmedia/supertest) - Easy HTTP assertions for integration test\n\n\n## Contributing\n\nThis project follows the [**Contributor Covenant**](http://contributor-covenant.org/version/1/4/) Code of Conduct.\n\n#### Bug Reports & Feature
