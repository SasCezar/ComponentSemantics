Daruk\n\n[Chinese Document](./CN.md)\n\n[Daruk](https://darukjs.github.io/daruk.org) is a lightweight web framework base on [Koa](https://github.com/koajs/koa) and written in [Typescript](https://www.typescriptlang.org/). Koa is original http server implementation if you used it. In the daily development,we have to use many open source middleware in Koa to implement scaffold, routing, abstraction and some components such as logging, monitoring, etc.\n\nWe can create your web project by Daruk scaffold for business. Daruk core is lightweight and easy extension and learing, but it can provides more options for developer to rapid development such as constraints directory specification and code style.\n\nDaruk, it comes from [The Legend of Zelda: Breath of the Wild](https://en.wikipedia.org/wiki/The_Legend_of_Zelda:_Breath_of_the_Wild) one of the four heroes, it have symbol of guardian ability. The goal of the framework is also as it.\n\nDaruk is base on Koa2, includes features:\n\n- Scaffolding\n- Directory and file specification\n- Auto loader and decorator combination\n- Complete Typescript development experience\n- Lifecycle and hook.\n- Custom(performance logs & business logs) & tracked\n- Online performance analysis\n- Lightweight link tracking\n- The MVC development of object-oriented\n- Support the development of the module\n\n## Installation scaffolding\n\n```bash\n# install daruk scaffold cli\n$ cnpm i -g daruk-cli\n\n# init project\n# --ignore ignore local template cache\n$ daruk init --ignore daruk-example\n\n# run the project\n$ cd daruk-example\n$ npm run dev\n```\n\n## Quick start\n\nYou can write your code shown below :\n\n```typescript\nimport { Daruk } from \'daruk\';\n\nconst myApp = new Daruk(\'darukProject\', {\n  rootPath: __dirname,\n  debug: process.env.NODE_ENV === \'dev\'\n});\n\nmyApp.listen(process.env.PORT || 3000);\n```\n\n`controllers/index.ts`:\n\n```typescript\nimport { BaseController, config, Context, Daruk, get, middleware, util } from \'daruk\';\n\nexport default class Index extends BaseController {\n  @util(\'getToday\')\n  public getToday: Daruk[\'util\'][\'getToday\'];\n\n  @config(\'author\')\n  public author: Daruk[\'config\'][\'author\'];\n\n  @config(\'version\')\n  public version: Daruk[\'config\'][\'version\'];\n\n  @middleware(\'cors\')\n  @get(\'/\')\n  public async index(ctx: Context, next: Function) {\n    const weather = await ctx.service.weather.getWeather();\n    ctx.body = `Hi, ${this.author}, project version is ${\n      this.version\n    }, Today is ${this.getToday()}, weather is ${weather}`;\n  }\n}\n```\n\nMore information you can check out [example/03-weather](./example/03-weather).\n\n## Docs\n\nMore docs: [Quick Start](http://darukjs.com/)\n\n## Features Todo\n\nBeta [TodoList](./docs/todo-list.md)\n\n## How to Contribute\n\nPlease let us know how can we help.
