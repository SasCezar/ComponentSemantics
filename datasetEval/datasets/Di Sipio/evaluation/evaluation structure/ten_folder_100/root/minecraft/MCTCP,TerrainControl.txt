b'##  TerrainControl\n\n| <h3>Project status</h3> |\n| -------------- |\n| Terrain Control is not really maintained anymore. There are still a few people left that make changes every now and then, but no proper release has been made for a long time. The forums and build server are also gone, so you need to compile the source code yourself. A possible alternative is [OpenTerrainGenerator](https://openterraingenerator.com/), which is a fork of Terrain Control. |\n\n[![CircleCI](https://circleci.com/gh/MCTCP/TerrainControl.svg?style=shield)](https://circleci.com/gh/MCTCP/TerrainControl)\n\nTerrainControl is an advanced world generator for Minecraft. Through various configuration files it gives the user control over almost every aspect of Minecraft terrain generation, with no knowledge of Java required. It can be used on Bukkit or Forge servers and on singleplayer with Forge. Terrain Control is the successor to the old <a href="http://www.minecraftforum.net/topic/313991-phoenixterrainmod/">PhoenixTerrainMod</a>, which was based on the popular <a href="http://www.minecraftforum.net/topic/71565-biomemod/">BiomeTerrainMod</a>. \n\n<ul>\n<li>Control horizontal and vertical noise of terrain generation</li>\n<li>Control terrain height and volatility per biome</li>\n<li>Generating biome maps from images</li>\n<li>Control the size of biomes, islands and continents</li>\n<li>Custom biomes</li>\n<li>Many options for caves, canyons</li>\n<li>Control vanilla structures like mineshafts, strongholds and villages</li>\n<li>Full control of ores, plants, dungeons, ponds and many more</li>\n<li>Custom trees and other objects (BO2 and BO3)</li>\n<li>Block replacement</li>\n<li>Visual settings like custom foliage colors (TerrainControl client required)</li>\n<li>Essentially everything!</li>\n</ul>\nTo learn more, check out <a href="https://github.com/MCTCP/TerrainControl/wiki/Features"><strong>all of TerrainControl\'s features</strong></a>!\n\n## Links\n* Project home at [Minecraft Forums](http://www.minecraftforum.net/forums/mapping-and-modding/minecraft-mods/2554291-terraincontrol-advanced-world-generator) / [SpigotMC](https://www.spigotmc.org/resources/terraincontrol.2214/) / [BukkitDev](http://dev.bukkit.org/bukkit-plugins/terrain-control/)\n* [Wiki](https://github.com/MCTCP/TerrainControl/wiki/)\n* [Releases](https://github.com/MCTCP/TerrainControl/releases)\n* [Installation](https://github.com/MCTCP/TerrainControl/wiki/Installation-instructions) for Spigot and Forge\n* [Compilation](./COMPILING.md) instructions for command-line, Eclipse and IntelliJ\n* [Version history](https://github.com/MCTCP/TerrainControl/wiki/Version-history)\n'
