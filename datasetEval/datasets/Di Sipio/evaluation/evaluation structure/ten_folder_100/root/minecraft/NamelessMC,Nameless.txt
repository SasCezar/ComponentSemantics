Until it\'s ready for production use, any pre-releases can be found here: https://github.com/NamelessMC/Nameless/releases.\n\n## Version 1.0.21\n\nNamelessMC is a free, easy to use & powerful website software for your Minecraft server, which includes a large range of features.\n\nVersion 1.0.21 features:\n\n- Template and theme system\n- Languages\n- Addon system\n- New "core modules"\n- Forum topic labels (not yet permission based, working on that)\n- Subforums\n- Alerts system, including popup notifications upon user tagging or a new staff application\n- Email configuration\n- API\n- Ability to disable UUID linking, allowing support for offline mode servers\n- Custom page system\n- Server statistics integration\n- Staff and member lists\n- Minecraft status\n- reCAPTCHA option\n- Voice server viewer\n- Two factor authentication\n- Minecraft account verification\n- Announcements\n\nIncluded addons:\n- Ban appeal\n- Bug report\n- Donate\n- Member list\n- Voting\n- Infractions\n\nCore modules:\n- Google Analytics\n- Social media\n- Registration\n- Voice server status\n- Staff Applications\n\n## Downloads & Support\n*
