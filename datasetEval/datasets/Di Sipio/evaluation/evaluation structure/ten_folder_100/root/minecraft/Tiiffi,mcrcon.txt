b'### Installing:\r\n\r\nfrom sources:\r\n```sh\r\ngit clone https://github.com/Tiiffi/mcrcon.git\r\ncd mcrcon\r\nmake\r\nsudo make install\r\n```\r\nCheck **INSTALL** for more details.\r\n\r\nYou can also download precompiled binaries*: https://github.com/Tiiffi/mcrcon/releases/latest\r\n\r\n<sub>*At the moment binaries are provided for Linux and Windows.</sub>\r\n\r\n---\r\n\r\n### Usage:\r\nmcrcon [OPTIONS] [COMMANDS]\r\n\r\nSends rcon commands to Minecraft server.\r\n\r\n```\r\nOption:\r\n  -h            Print usage\r\n  -H            Server address\r\n  -P            Port (default is 25575)\r\n  -p            Rcon password\r\n  -t            Interactive terminal mode\r\n  -s            Silent mode (do not print received packets)\r\n  -c            Disable colors\r\n  -r            Output raw packets (debugging and custom handling)\r\n  -v            Output version information\r\n```\r\nCommands with arguments must be enclosed in quotes.\r\n\r\nServer address, port and password can be set using following environment variables:\r\n```\r\nMCRCON_HOST\r\nMCRCON_PORT\r\nMCRCON_PASS\r\n```\r\nNote that command-line options will override environment variables.\r\n\r\nExample:\r\n  ```mcrcon -H my.minecraft.server -p password "say Server is restarting!" save-all stop```\r\n\r\n---\r\n\r\n### Enable rcon on server\r\nRemember to enable rcon by adding following lines to ```server.properties``` file.\r\n```\r\nenable-rcon=true\r\nrcon.port=25575\r\nrcon.password=your_rcon_pasword\r\n```\r\n\r\n---\r\n\r\n#### Contact:\r\n\r\n* WWW:            https://github.com/Tiiffi/mcrcon/\r\n* MAIL:           tiiffi_at_gmail_dot_com\r\n* IRC:            tiiffi @ quakenet\r\n* BUG REPORTS:    https://github.com/Tiiffi/mcrcon/issues/\r\n\r\n---\r\n\r\n<sub>Master:</sub> ![Master build](https://api.travis-ci.org/Tiiffi/mcrcon.svg?branch=master)\r\n<sub>Develop:</sub> ![Develop build](https://api.travis-ci.org/Tiiffi/mcrcon.svg?branch=develop)\r\n'
