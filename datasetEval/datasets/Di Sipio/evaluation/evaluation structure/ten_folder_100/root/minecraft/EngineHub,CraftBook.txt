b'![CraftBook](http://static.sk89q.com/readme/craftbook.png)\n===========\n\nCraftBook is a heavily customizable plugin adding exciting new gameplay elements and advanced redstone functions!\n\n* Modular, and extremely light on your server.\n* Enable features that you need, disable ones that you don\'t. Even tweak features to your liking!\n* Redstone ICs (AND/OR gates, lightning striker gates)\n* Elevators for your server so people can jump between floors quickly (Or smoothly, if you choose so)\n* Working cauldrons so players can cook inside them\n* Your own custom crafting recipes\n* Hidden switches for players\n* Minecart booster blocks and more\n* Togglable areas and bridges\n* Chairs you can sit on\n* Working pipes that can transfer items from place to place!\n* Footprints where your players walk!\n* Mobs and players drop their heads.\n* The ability to bind commands to items, with cool-downs, permissions and timers.\n* And much more!\n\nAn extensive list of features is available on [the docs!](http://enginehub.github.io/CraftBookDocs/)\n\nBranch Structure\n----------------\n\nCurrently the \'master\' branch contains CraftBook 4, the next version of CraftBook. Currently this version only supports Sponge.\n\nFor the prior version of CraftBook for Bukkit and Spigot, the \'3.x\' branch is what you want.\n\nStarting from CraftBook 4, there are \'version\' branches that contain the last supported version for a branch. This is used for backporting reasons. The format is as follows, `version/1.x`.\n\nCompiling\n---------\n\nThe project is written for Java 8 and our build process makes use of [Gradle](http://gradle.org/).\nsimply run:\n\n    gradlew build\n\nDependencies are automatically handled by Gradle.\n\nContributing\n------------\n\nWe happily accept contributions.
