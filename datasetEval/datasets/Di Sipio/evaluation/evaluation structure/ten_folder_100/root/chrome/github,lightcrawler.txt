b'# lightcrawler\nCrawl a website and run it through Google lighthouse\n\n```bash\nnpm install --save-dev lightcrawler\n\nlightcrawler --url https://atom.io/ --config lightcrawler-config.json\n```\n\nwhere `lightcrawler-config.json` looks something like this:\n```json\n{\n  "extends": "lighthouse:default",\n  "settings": {\n    "crawler": {\n      "maxDepth": 2,\n      "maxChromeInstances": 5\n    },\n    "onlyCategories": [\n      "Accessibility",\n      "Performance",\n      "Best Practices"\n    ],\n    "onlyAudits": [\n      "accesskeys",\n      "aria-allowed-attr",\n      "external-anchors-use-rel-noopener",\n      "geolocation-on-start",\n      "no-document-write",\n      "no-mutation-events",\n      "no-old-flexbox",\n      "time-to-interactive",\n      "user-timings",\n      "viewport",\n      "without-javascript"\n    ]\n  }\n}\n```\n\nEnjoy!\n'
