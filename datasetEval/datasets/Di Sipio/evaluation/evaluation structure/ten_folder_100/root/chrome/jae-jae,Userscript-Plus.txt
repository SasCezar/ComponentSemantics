lower right corner, recommend scripts for you, and will automatically disappear after 10 seconds\n-  Filter the top 50 script with the highest score\n-  The script list supports search filtering\n-  The script list supports custom sorting\n-  Supports one-click installation script\n-  Support for viewing script details\n-  Support access to the script home page and author home page\n-  Click the close button, the current site `this session` all the pages will no longer appear prompt box\n-  A prompt box is displayed only when found available script for the current site\n-  i18n international multilingual support.(**[Locale contributions](https://github.com/jae-jae/Userscript-Plus/tree/master/src/common/lang) are welcomed!**)\n-  Vue + iView UI design smooth and beautiful\n\n## Whitelist
