b"# XMage \xe2\x80\x94 Magic, Another Game Engine\n\n[![Join the chat at https://gitter.im/magefree/mage](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/magefree/mage?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge) [![Build Status](https://travis-ci.org/magefree/mage.svg?branch=master)](https://travis-ci.org/magefree/mage)\n\nXMage allows you to play Magic against one or more online players or computer opponents. It includes full rules enforcement for over **19.200** unique cards (over 37.400 counting all cards from different editions, you can find event custom sets like Star Wars). All regular sets have nearly all the cards implemented. A more detailed information which cards are implemented can be found [here](https://github.com/magefree/mage/wiki/Set-implementation-list). \n\nIt's support single matches and tournaments with dozens game modes like duel, multiplayer, standard, modern, commander, pauper, oathbreaker, freeform and much more.\n\nThere are [public servers](http://xmageservers.online/) where you can play XMage against other players. You can also host your own server to play against the AI and/or your friends.\n\nXMage community:\n* [Official XMage forum](http://www.slightlymagic.net/forum/viewforum.php?f=70);\n* [Official XMage support and feature request on github](https://github.com/magefree/mage/issues);\n* [Reddit XMage group](https://www.reddit.com/r/XMage/);\n* [Reddit XMage discord channel](https://discord.gg/Pqf42yn).\n\nServers status:\n* http://xmageservers.online/\n\n## Features\n\n* Multiplatform app: Windows, Linux, MacOS;\n* Deck editor (support multiple deck formats and deck sources);\n* Two player duel or a multiplayer free-for-all game with up to 10 players;\n* Computer AI opponents;\n* Players rating system (Glicko);\n* Supports special formats like Commander (up to 10 players), Oathbreaker, Cube, Tiny Leaders, Super Standard, Historic Standard and more;\n* Single matches or tournaments supported (elimination or swiss type handling), which can be played with up to 16 players:\n    * Booster (also Cube) draft tournaments (4-16)\n    * Sealed (also from Cube) tournaments (2-16)\n\n## Issues / bugs\n\nBefore you create a new issue, take a look at the [List of things already fixed but not yet released](https://github.com/magefree/mage/wiki/Features-and-fixes-not-released-yet#features-and-fixes-not-released-yet) to avoid creating uneccessary new issues.\nAlso there is always a bug thread in the [Official XMage forum](http://www.slightlymagic.net/forum/viewforum.php?f=70) which we check regularly.\n\n## Installation\n\n* Download [latest XMage launcher file](http://xmage.de) and put it to any folder like `D:\\games\\xmage`;\n* You need to have [Java version 8 or later](http://java.com/) to run it;\n* If you can't run it then create `run-LAUNCHER.cmd` text file in launcher folder and put that line to it and save as ANSI format:\n  * `java -Djava.net.preferIPv4Stack=true -jar XMageLauncher-0.3.8.jar` \n\nLook [here](http://www.slightlymagic.net/forum/viewtopic.php?f=70&t=13632) for more detailed instructions.\n[Here](http://github.com/magefree/mage/wiki/Release-changes) you can find a log of the latest changes.\n\n## Developer\n\nIf you are interested in developing XMage, here are some useful resources:\n\n* [Developer Getting
