With it, you can use the best aspects of OpenGL/OpenGLES in an easy and organized way optimized for game programming. It is built around the concept that beginners should be able to start with the basics and then move up into a more complex plane of development with the veterans, all on the same platform.\n\nLGame puts all of its effort into keeping things short and simple. The initial setup of a game consists only of making a single class; then you are done.
