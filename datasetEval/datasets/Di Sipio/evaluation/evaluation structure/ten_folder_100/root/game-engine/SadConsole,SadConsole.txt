b'![SadConsole Logo](images/SadConsoleLogo.gif)\n\n\n[![Chat on discord](https://img.shields.io/discord/501465397518925843.svg)](https://discord.gg/mttxqAs)\n[![Join us on Reddit](https://img.shields.io/badge/reddit-SadConsole-red.svg)](http://reddit.com/r/sadconsole)\n[![NuGet](https://img.shields.io/nuget/v/SadConsole.svg)][nuget]\n\nSadConsole is an MonoGame 3.7 and FNA game library that provides an engine to emulate old-school console and command prompt style graphics. One or more textures are used to represent the standard ascii character set. SadConsole allows you to create console instances which can be managed independently of each other. A console is made up of individual cells which can have a foreground, background, character, and a special effect applied to it. \n\nSadConsole targets .NET Standard 2.0\n\n## Features\n\nHere are some of the features SadConsole supports.\n\n* Show any number of consoles\n* Uses PNG graphic fonts supporting more than 256 characters\n* Multiple fonts in your game\n* Draggable console windows within the game\n* Text GUI controls\n* Full keyboard support\n* Full mouse support\n* Read ansi files from the good old DOS days\n* Animation engine\n* Instruction engine\n\n#### String display and parsing\n![string pic](images/stringparseexample.gif)\n\n#### GUI library\n![GUI library pic](images/ui-example.gif)\n\n#### Scrolling\n![scrolling console](images/scrolling-example2.gif)\n\n## Dependencies\nSadConsole uses NuGet for its .NET dependencies.\n\n## Demo Project\n[DemoProject](./src/DemoProject) shows how to use SadConsole in a multi-platform environment and demonstrates various things you can do with SadConsole.\n\n[nuget]: http://www.nuget.org/packages/SadConsole/'
