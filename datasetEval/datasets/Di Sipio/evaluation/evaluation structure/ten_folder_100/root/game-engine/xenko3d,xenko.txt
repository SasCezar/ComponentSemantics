b'![Xenko](https://xenko.com/images/external/xenko-logo-side.png)\n=======\n\nWelcome to the Xenko source code repository!\n\nXenko is an open-source C# game engine for realistic rendering and VR. \nThe engine is highly modular and aims at giving game makers more flexibility in their development.\nXenko comes with an editor that allows you create and manage the content of your games or applications in a visual and intuitive way.\n\n![Xenko Editor](https://xenko.com/images/external/script-editor.png)\n\nTo learn more about Xenko, visit [xenko.com](https://xenko.com/).\n\n## License\n\nXenko is covered by [MIT](LICENSE.md), unless stated otherwise (i.e. for some files that are copied from other projects).\n\nYou can find the list of third party projects [here](THIRD%20PARTY.md).\n\nContributors need to sign the following [Contribution License Agreement](docs/ContributorLicenseAgreement.md).\n\n## Documentation\n\nFind explanations and information about Xenko:\n* [Xenko Manual](https://doc.xenko.com/latest/manual/index.html)\n* [API Reference](https://doc.xenko.com/latest/api/index.html)\n* [Release Notes](https://doc.xenko.com/latest/ReleaseNotes/index.html)\n\n## Community\n\nAsk for help or report issues:\n* [Chat with the community on Discord](https://discord.gg/f6aerfE) [![Join the chat at https://discord.gg/f6aerfE](https://img.shields.io/discord/500285081265635328.svg?style=flat&logo=discord&label=discord)](https://discord.gg/f6aerfE)\n* [Discuss topics on our forums](http://forums.xenko.com/)\n* [Report engine issues](https://github.com/xenko3d/xenko/issues)\n* [Donate to support the project](https://www.patreon.com/xenko)\n* [List of Projects made by users](https://github.com/xenko3d/xenko-community-projects)\n* [Localization](docs/localization.md)\n\n## Building from source\n\n### Prerequisites\n\n1. [Git](https://git-scm.com/downloads) (recent version that includes LFS, or install [Git LFS](https://git-lfs.github.com/) separately).\n2. [Visual Studio 2019](https://www.visualstudio.com/downloads/) with the following workloads:\n  * `.NET desktop development` with `.NET Framework 4.7.2 targeting pack`\n  * `Desktop development with C++` with `Windows 10 SDK (10.0.17763.0)` or later, and `VC++ 2017 version 15.9 v14.16 latest v141 tools` or later (both should be enabled by default)\n  * `.NET Core cross-platform development`\n  * Optional (to target UWP): `Universal Windows Platform development` with `Windows 10 SDK (10.0.17763.0)`\n  * Optional (to target iOS/Android): `Mobile development with .NET` and `Android SDK setup (API level 27)` individual component, then in Visual Studio go to `Tools > Android > Android SDK Manager` and install `NDK` (version 19+) from `Tools` tab.\n3. [FBX SDK 2019.0 VS2015](https://www.autodesk.com/developer-network/platform-technologies/fbx-sdk-2019-0)\n\n### Build Xenko\n\n1. Clone Xenko: `git clone https://github.com/xenko3d/xenko.git`\n2. Open `<XenkoDir>\\build\\Xenko.sln` with Visual Studio 2019, and build.\n3. Open `<XenkoDir>\\samples\\XenkoSamples.sln` and play with the samples.\n4. Optionally, open and build `Xenko.Android.sln`, `Xenko.iOS.sln`, etc...\n\n### Contribution Guidelines\n\nPlease check our [Contributing Guidelines](docs/CONTRIBUTING.md).\n\n### Build
