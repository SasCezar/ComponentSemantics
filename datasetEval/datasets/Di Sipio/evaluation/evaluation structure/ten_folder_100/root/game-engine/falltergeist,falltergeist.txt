b"Falltergeist [![Build Status](https://travis-ci.org/falltergeist/falltergeist.svg?branch=develop)](https://travis-ci.org/falltergeist/falltergeist) [![Version](https://img.shields.io/github/release/falltergeist/falltergeist.svg)](https://github.com/falltergeist/falltergeist/releases/latest) [![Discord](https://img.shields.io/discord/401990446747877376.svg)](https://discord.gg/jxs6WRq)\n============\n\nOpensource crossplatform Fallout 2\xe2\x84\xa2 engine writen in C++ and SDL.\nhttps://falltergeist.org/\n\n## Dependencies\n\n- [SDL2](http://www.libsdl.org)\n- [SDL2\\_mixer](http://www.libsdl.org/projects/SDL_mixer/)\n- [SDL2\\_image](http://www.libsdl.org/projects/SDL_image/)\n- [GLEW](http://glew.sourceforge.net/)\n- [GLM](http://glm.g-truc.net/)\n- [Zlib](http://www.zlib.net/)\n\n## Compilation\n\n### Linux\n\n```\ncmake . && make\n```\n\n### Windows (Visual Studio)\n\nYou could add dependencies manually but it is easier to use [Conan](https://docs.conan.io/en/latest/howtos/vs2017_cmake.html). \n\nFirst, set up your [Conan profile](https://docs.conan.io/en/latest/reference/profiles.html) to match your compiler or let conan autodetect your settings (`conan profile new vs --detect` and use ` --profile=vs` when running `conan install`). Visual Studio 2019 profile should look like this:\n\n```ini\n[settings]\nos=Windows\nos_build=Windows\narch=x86_64\narch_build=x86_64\ncompiler=Visual Studio\ncompiler.version=16\nbuild_type=Debug\n```\n\nThen open 'Developer PowerShell for Visual Studio' and execute the following commands:\n\n```\nconan install . --build missing\ncmake .\ndevenv /build Debug falltergeist.sln\n```\n(or open and build the solution in Visual Studio instead of running the devenv command)\n\n\n## Running\n\nPut **master.dat** and **critter.dat** files into the falltergeist data directory, that is\n\n* `~/.local/share/falltergeist/`  or `/usr/local/share/falltergeist` (for global installs) on Linux,\n* `~/Library/Application Support/falltergeist` on OS X,\n* `%APPDATA%/falltergeist` on Windows,\n\nor mount CD-ROM with original game, then run `./falltergeist` on \\*nix or `.\\bin\\falltergeist.exe` on Windows.\n\n## Development roadmap\n\nThis is a pretty highlevel roadmap to get to a playable game. The idea behind it is to step by step get closer to the full game.\n\n- Worldmap support\n- Elevators\n- NPCs\n- Combat AI\n- Full scripting support\n- Skills\n"
