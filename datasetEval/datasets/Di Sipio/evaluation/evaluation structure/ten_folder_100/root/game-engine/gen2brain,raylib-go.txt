b'![logo](https://goo.gl/XlIcXz)\n## raylib-go\n[![TravisCI Build Status](https://travis-ci.org/gen2brain/raylib-go.svg?branch=master)](https://travis-ci.org/gen2brain/raylib-go)\n[![AppVeyor Build status](https://ci.appveyor.com/api/projects/status/qv2iggrqtgl7xhr0?svg=true)](https://ci.appveyor.com/project/gen2brain/raylib-go)\n[![GoDoc](https://godoc.org/github.com/gen2brain/raylib-go/raylib?status.svg)](https://godoc.org/github.com/gen2brain/raylib-go/raylib)\n[![Go Report Card](https://goreportcard.com/badge/github.com/gen2brain/raylib-go)](https://goreportcard.com/report/github.com/gen2brain/raylib-go)\n[![Examples](https://img.shields.io/badge/learn%20by-examples-0077b3.svg?style=flat-square)](https://github.com/gen2brain/raylib-go/tree/master/examples)\n\nGolang bindings for [raylib](http://www.raylib.com/), a simple and easy-to-use library to learn videogames programming.\n\n### Requirements\n\n##### Ubuntu\n\n###### X11\n\n    apt-get install libgl1-mesa-dev libxi-dev libxcursor-dev libxrandr-dev libxinerama-dev \n\n###### Wayland \n\n    apt-get install libgl1-mesa-dev libwayland-dev libxkbcommon-dev \n\n##### Fedora\n\n###### X11\n\n    dnf install mesa-libGL-devel libXi-devel libXcursor-devel libXrandr-devel libXinerama-devel\n\n###### Wayland \n\n    dnf install mesa-libGL-devel wayland-devel libxkbcommon-devel\n\n##### macOS\n\nOn macOS you need Xcode or Command Line Tools for Xcode.\n\n##### Windows\n\nOn Windows you need C compiler, like [Mingw-w64](https://mingw-w64.org) or [TDM-GCC](http://tdm-gcc.tdragon.net/).\nYou can also build binary in [MSYS2](https://msys2.github.io/) shell.\n\n##### Android\n\n[Android example](https://github.com/gen2brain/raylib-go/tree/master/examples/others/android/example).\n\n##### Raspberry Pi\n\n[RPi example](https://github.com/gen2brain/raylib-go/tree/master/examples/others/rpi/basic_window).\n\n### Installation\n\n    go get -v -u github.com/gen2brain/raylib-go/raylib\n\n### Build tags\n\n* `noaudio` - disables audio functions\n* `opengl21` - uses OpenGL 2.1 backend (default is 3.3 on desktop)\n* `opengl11` - uses OpenGL 1.1 backend (pseudo OpenGL 1.1 style)\n* `wayland` - builds against Wayland libraries\n\n### Documentation\n\nDocumentation on [GoDoc](https://godoc.org/github.com/gen2brain/raylib-go/raylib). Also check raylib [cheatsheet](http://www.raylib.com/cheatsheet/cheatsheet.html).\n\n### Example\n\n```go\npackage main\n\nimport "github.com/gen2brain/raylib-go/raylib"\n\nfunc main() {\n\trl.InitWindow(800, 450, "raylib [core] example - basic window")\n\n\trl.SetTargetFPS(60)\n\n\tfor !rl.WindowShouldClose() {\n\t\trl.BeginDrawing()\n\n\t\trl.ClearBackground(rl.RayWhite)\n\n\t\trl.DrawText("Congrats! You created your first window!", 190, 200, 20, rl.LightGray)\n\n\t\trl.EndDrawing()\n\t}\n\n\trl.CloseWindow()\n}\n```\n\nCheck more [examples](https://github.com/gen2brain/raylib-go/tree/master/examples) organized by raylib modules.\n\n\n### License\n\nraylib-go is licensed under an unmodified zlib/libpng license.
