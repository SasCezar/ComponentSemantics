b'<h1 align="center">\n\t<img width="320" src="https://cdn.rawgit.com/brenolf/polyjuice/master/logo.svg" alt="Polyjuice">\n</h1>\n\n[![Build Status](https://travis-ci.org/brenolf/polyjuice.svg)](https://travis-ci.org/brenolf/polyjuice)\n[![Coverage Status](https://coveralls.io/repos/brenolf/polyjuice/badge.svg?branch=master&service=github)](https://coveralls.io/github/brenolf/polyjuice?branch=master)\n[![Code Climate](https://codeclimate.com/github/brenolf/polyjuice/badges/gpa.svg)](https://codeclimate.com/github/brenolf/polyjuice)\n[![npm version](https://badge.fury.io/js/polyjuice.svg)](http://badge.fury.io/js/polyjuice)\n> Converts `.jshintrc` and `.jscsrc` files into `.eslintrc` and vice-versa\n\nPolyjuice is a utility that converts [JSHint](http://jshint.com/) and [JSCS](http://jscs.info/) files into [ESLint](http://eslint.org/) files and the other way around.\n\n## Install\n`$ npm install --save-dev polyjuice`\n\n## Usage\n```js\nvar polyjuice = require(\'polyjuice\')\n\nvar to_eslint = {\n  jshint: polyjuice.from.jshint([\'./.jshintrc\']),\n  jscs  : polyjuice.from.jscs([\'./.jscsrc\']),\n  both  : polyjuice.to.eslint([\'./.jshintrc\'], [\'./.jscsrc\'])\n}\n\nvar to_jscs_and_jshint = {\n  jshint: polyjuice.to.jshint([\'./.eslintrc\')],\n  jscs  : polyjuice.to.jscs([\'./.eslintrc\')],\n  both  : polyjuice.from.eslint([\'./.eslintrc\'])\n}\n```\n\n> When using `polyjuice.from.eslint` an object with keys `jscs` and `jshint` is returned.\n\n## CLI\n\nIf installed globally, you can use polyjuice as a command line utility. You can check which rules have been discarded by adding `--discarded` or `-d` to your command.\n\n### To ESLint\n\nYou must provide the files to be assessed by using the options `--jshint` and `--jscs`, for instance:\n\n`$ polyjuice --jshint .jshintrc --jscs .jscsrc > .eslintrc`\n\n> Note you are allowed to pass in any number of files\n\n### To JSHint and JSCS\n\nYou must provide the file to be assessed by using the option `--eslint` and also which output you want by using either `--to-jshint` or `--to-jscs`, for instance:\n\n`$ polyjuice --eslint .eslintrc --to-jshint > .jshintrc`\n\n> Note you are allowed to pass in any number of files\n\n## Contributing\nContributions are always welcome! If you want to have a better understanding about the conversion files, I encourage you to take a look at [the conversion page](/doc/DICTIONARIES.md). You can also check the rules discarded when [parsing JSCS](/doc/JSCS.md), [parsing JSHint](/doc/JSHINT.md) and [parsing ESLint](/doc/ESLINT.md).\n\n## Versions used\nThese are the versions used as source of the translation. Therefore, you may expect all rules up to those versions to have been mapped to the target tool.\n\n* ESLint: `v1.7.2`\n* JSCS: `v3.0.2`\n* JSHint: `v2.9.2`\n\n## License\nMIT\n'
