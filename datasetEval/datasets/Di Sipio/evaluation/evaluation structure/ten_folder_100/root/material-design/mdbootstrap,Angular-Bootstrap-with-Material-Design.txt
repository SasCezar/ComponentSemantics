at the end. It will clone files directly into current folder.\n- Run `npm i`\n- Run `npm start`\n- Voil\xc3\xa0! Open browser and visit http://localhost:4200\n\nNow you can navigate to [our documentation](https://mdbootstrap.com/docs/angular/), pick any component and place within your project.\n\n## Demo application\n\nFeel free to check our live example components: Just type `ng serve mdb-demo` in terminal!\n\nType one of the below commands to remove demo application from this project:\n* npm `run remove-demo-unix` to remove demo application on UNIX based systems,\n* npm `run remove-demo-windows` to remove demo application on Windows systems.\n\n# Available commands\n\n* npm run build:lib - building library,\n* npm run pack - copying assets and packaging /dist directory into .tgz archive\n* npm run version - adjusting src/package.json version from main package.json file,\n* npm run compile - Executing above commands with correct sequence.\n\n# Modules list\n\n* ButtonsModule,\n* CarouselModule,\n* ChartsModule,\n* CollapseModule,\n* InputsModule,\n* ModalModule,\n* NavbarModule,\n* PopoverModule,\n* TooltipModule,\n* WavesModule,\n* MDBBootstrapModule - contains every MDB modules.\n\n# How to install MDB via npm\n\n- create new project `ng new project_name --style=scss`\n- `npm i angular-bootstrap-md --save`\n- to app.module.ts add\n```javascript\nimport { NgModule, NO_ERRORS_SCHEMA } from \'@angular/core\';\nimport { MDBBootstrapModule } from \'angular-bootstrap-md\';\n\n@NgModule({\n    imports: [\n        MDBBootstrapModule.forRoot()\n    ],\n    schemas: [ NO_ERRORS_SCHEMA ]\n});\n```\n- in angular.json change:\n\n`"styleExt": "css"` to `"styleExt": "scss"`\n\nrename /src/styles.css to styles.scss\n\n- if you want to change styles in exisiting project you can use `ng config schematics.@schematics/angular:component.styleext scss`\n\n- add below lines to angular.json:\n```javascript\n"styles": [\n    "node_modules/font-awesome/scss/font-awesome.scss",\n    "node_modules/angular-bootstrap-md/assets/scss/bootstrap/bootstrap.scss",\n    "node_modules/angular-bootstrap-md/assets/scss/mdb.scss",\n    "src/styles.scss"\n],\n"scripts": [\n  "node_modules/chart.js/dist/Chart.js",\n  "node_modules/hammerjs/hammer.min.js"\n],\n```\n- install external libs\n```bash\nnpm install -\xe2\x80\x93save chart.js@2.5.0 font-awesome hammerjs\n```\n\n### Run server\n\n```bash\nng serve --open\n```\n\n# Supported browsers\n\nMDBootstrap supports the **latest, stable releases** of all major browsers and platforms.\n\nAlternative browsers which use the latest version of WebKit, Blink, or Gecko, whether directly or via the platform\xe2\x80\x99s web view API, are not explicitly supported.
