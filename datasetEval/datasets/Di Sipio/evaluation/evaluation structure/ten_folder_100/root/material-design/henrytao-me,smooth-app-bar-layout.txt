Please use support design instead. \n\nThis is a smooth version of `Google Support Design AppBarLayout`. If you are using `AppBarLayout`, you will know it has an issue with fling. Check out these threads to know about that problem:\n[http://stackoverflow.com/questions/30923889/flinging-with-recyclerview-appbarlayout](http://stackoverflow.com/questions/30923889/flinging-with-recyclerview-appbarlayout)\n[https://code.google.com/p/android/issues/detail?id=177729...](https://code.google.com/p/android/issues/detail?id=177729&q=appbarlayout&colspec=ID%20Type%20Status%20Owner%20Summary%20Stars)\n\nIf you find that it still doesn\'t meet your need, don\'t hesitate to send me a request. [Send a request HERE](https://github.com/henrytao-me/smooth-app-bar-layout/issues)\n\n\n## Installation\n\n``` groovy\ncompile "me.henrytao:smooth-app-bar-layout:<latest-support-design-version>.<latest-smooth-app-bar-layout-fix>"\n```\n\nExample: if the latest support design version is `24.1.0` and SmoothAppBarLayout fix is `0`, please use `compile "me.henrytao:smooth-app-bar-layout:24.1.0.0"`. Please check `TAGS` section to see all supported versions.\n\n- `smooth-app-bar-layout` is an UI library and pretty much depended on AppCompat and Support Design. So that, versioning is quite the same with those libraries from Google.\n- `smooth-app-bar-layout` is deployed to `jCenter`. Make sure you have `jcenter()` in your project gradle.\n\n\n## Tested environments\n\n- `"com.android.support:design:23.1.1"`\n- `"com.android.support:appcompat-v7:23.1.1"`\n\n\n## Demo\n\n[![Get it on Google Play](https://raw.githubusercontent.com/henrytao-me/smooth-app-bar-layout/master/screenshots/google-play.png)](https://play.google.com/store/apps/details?id=me.henrytao.smoothappbarlayout)\n\nPlease note that the app on the Play store is not always the latest version.\n\n\n## Concepts\n\n ![concepts](./screenshots/concept.final.jpg)\n\n\n## Features\n\n- Scroll\n- EnterAlways\n- EnterAlwaysCollapsed\n- ExitUntilCollapsed\n- QuickReturn\n- Custom NestedScrollView (NEW)\n- ViewPager Scroll\n- ViewPager ExitUntilCollapsed (NEW)\n- ViewPager QuickReturn (NEW)\n- Support SwipeRefreshLayout\n\n\nCheckout these demo videos:\n\n- Google Design Support AppBarLayout: [https://youtu.be/oaN2UTZIRUk](https://youtu.be/oaN2UTZIRUk)\n- SmoothAppBarLayout: [https://youtu.be/la6ixCL9RKs](https://youtu.be/la6ixCL9RKs)\n\n\n ![screenshots](./screenshots/screenshots.jpg)\n\n\n## Important Notes\n\n- Remember to set `android:id` for `me.henrytao.smoothappbarlayout.SmoothAppBarLayout` in layout file so that it can call `onSaveInstanceState` and `onRestoreInstanceState` correctly. Otherwise, it won\'t work correctly with `onOrientationChanged`.\n- Remember to set `android:minHeight` for `me.henrytao.smoothappbarlayout.SmoothAppBarLayout` in layout file if you use `ViewPager`.\n- `clipToPadding` in `RecyclerView` won\'t work. You have to set HeaderHolder in apdater and it has to be placed at index 0.\n\n \n## Usage\n\nSuper easy! Just need to do 3 steps:\n\n- Change `android.support.design.widget.AppBarLayout` to `me.henrytao.smoothappbarlayout.SmoothAppBarLayout` and set `android:id`\n- Remove `app:layout_behavior="@string/appbar_scrolling_view_behavior"`.\n- Add header to your NestedScrollView or RecyclerView.\n\n#### Original AppBarLayout from Google Support Design\n\n``` xml\n<android.support.design.widget.CoordinatorLayout xmlns:android="http://schemas.android.com/apk/res/android"\n  xmlns:app="http://schemas.android.com/apk/res-auto"\n  android:layout_width="match_parent"\n  android:layout_height="match_parent">\n\n  <android.support.v7.widget.RecyclerView\n    android:id="@android:id/list"\n    android:layout_width="match_parent"\n    android:layout_height="match_parent"\n    app:layout_behavior="@string/appbar_scrolling_view_behavior" />\n\n  <android.support.design.widget.AppBarLayout\n    android:layout_width="match_parent"\n    android:layout_height="@dimen/app_bar_height">\n\n    <android.support.design.widget.CollapsingToolbarLayout\n      android:id="@+id/collapsing_toolbar_layout"\n      android:layout_width="match_parent"\n      android:layout_height="match_parent"\n      app:layout_scrollFlags="scroll|exitUntilCollapsed">\n\n      <android.support.v7.widget.Toolbar\n        android:id="@+id/toolbar"\n        app:layout_collapseMode="pin"\n        app:navigationIcon="@drawable/ic_toolbar_arrow_back"\n        style="@style/AppStyle.MdToolbar" />\n    </android.support.design.widget.CollapsingToolbarLayout>\n  </android.support.design.widget.AppBarLayout>\n</android.support.design.widget.CoordinatorLayout>\n```\n\n#### SmoothAppBarLayout \n\n```
