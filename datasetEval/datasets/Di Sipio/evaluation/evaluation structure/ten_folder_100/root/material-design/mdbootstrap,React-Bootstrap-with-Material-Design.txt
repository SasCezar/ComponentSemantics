at the end. It will clone files directly into current folder.\n\n- Run `npm i`\n- Run `npm start`\n- Voil\xc3\xa0! Open browser and visit http://localhost:3000\n\nNow you can navigate to [our documentation](http://mdbootstrap.com/docs/react/), pick any component and place within your project.\n\n# Available commands\n\n- npm start - runs the app in development mode.\n- npm run remove-demo - remove demo directory from your project and generate a boilerplate for your app\n- npm run build - builds the app for production to the build folder.\n- npm test - runs the test watcher in an interactive mode.\n\n# How to install MDB via npm:\n\n- create new project `create-react-app myApp`\n- `cd myApp`\n- `npm install --save mdbreact`\n- Import style files into the src/index.js before the App.js file:\n\n```javascript\nimport "@fortawesome/fontawesome-free/css/all.min.css";\nimport "bootstrap-css-only/css/bootstrap.min.css";\nimport "mdbreact/dist/css/mdb.css";\n```\n\n### Run server\n\n```bash\nnpm start\n```\n\n# Supported browsers\n\nMDBootstrap supports the **latest, stable releases** of all major browsers and platforms.\n\nAlternative browsers which use the latest version of WebKit, Blink, or Gecko, whether directly or via the platform\xe2\x80\x99s web view API, are not explicitly supported.
