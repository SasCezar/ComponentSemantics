b'[![Android Arsenal](https://img.shields.io/badge/Android%20Arsenal-Carbon-brightgreen.svg?style=flat)](https://android-arsenal.com/details/1/1491)&nbsp;&nbsp;\n[![Maven Central](https://img.shields.io/badge/Maven%20Central-0.16.0.1-brightgreen.svg)](https://oss.sonatype.org/content/groups/public/tk/zielony/carbon/0.16.0.1/)&nbsp;&nbsp;\n[![Dropbox](https://img.shields.io/badge/Dropbox-Sample%20app-brightgreen.svg)](https://www.dropbox.com/s/wllgpan9cl01mh3/samples.apk?raw=1)\n\n[![Twitter](https://img.shields.io/badge/Twitter-GreenMakesApps-blue.svg)](https://twitter.com/GreenMakesApps)\n\nCarbon\n================\nMaterial Design implementation for Android 4.0 and newer. This is not the exact copy of the Lollipop\'s API and features. It\'s a custom implementation of the most useful things as shown in the design specification. Carbon tries to:\n\n - make things easier (specify cornerRadius=\'dp\' instead of creating an xml and/or a ViewOutlineProvider)\n - make it all work and look the same on all APIs (like CheckBox\'s left padding)\n - really backport features (don\'t use gradients for shadows!)\n - fix Android\'s everlasting bugs (FrameLayout ignores child\'s padding when no gravity is set)\n \n### What\'s new\n\n - custom item layouts for BottomNavigationView, TabLayout and NavigationView\n - DayNight themes\n - updated control colors\n - fixed color animations, background tint, menu inflation, shadow transformations\n\n### Features\n\n##### Android 4.x\n\n - generated, animated shadows with elevation system\n - the touch ripple\n - rounded corners with content clipping\n - circular reveal\n - theme xml attribute\n - widget and drawable tinting\n - font resources\n - edge effects\n\n##### All APIs\n\n - simple to use xml attributes for stroke, cut corners, rounded corners and ripples\n - colored shadows\n - brightness/saturation fade\n - SVG support (with transformations, text, gradients, etc.)\n \n###### Useful extensions and bugfixes\n\n - html text\n - percent layouts, anchors, colored insets\n - view stroke\n - visibility animations\n - color state animations\n - ttf/otf fonts\n - text auto size\n - TextMarker\n\n##### Material constants, widgets and components\n\n - colors, dimensions, typography\n - ready-to-use rows and adapters for RecyclerView\n - DropDown, FloatingActionMenu, RangeSeekBar, ExpandableRecyclerView, FlowLayout, TableView, BackdropLayout, Banner, BottomNavigationView, TabLayout\n\n### Samples\n<img src="https://github.com/ZieIony/Carbon/blob/master/images/shadow.png" width="135px" height="240px"/> <img
