b'<img src="../images/materia-logo.svg" alt="materia-logo" align="right" />\n\n# Materia\n\nMateria is a [Material Design](https://material.io) theme for GNOME/GTK based desktop environments.\n\nIt supports GTK 2, GTK 3, GNOME Shell, Budgie, Cinnamon, MATE, Unity, Xfce, LightDM, GDM, Chrome theme, etc.\n\n## Previews\n\n![widget-factory](../images/widget-factory.png?raw=true)\n![widget-factory-dark](../images/widget-factory-dark.png?raw=true)\n\n## Features\n\nSupports **ripple animations** for GTK 3:\n\n![Button](../images/Button.gif?raw=true)\n\n**3 color variants** and **2 size variants** are available:\n\n**Materia** | **standard** | **compact**\n:-: | :-: | :-:\n**standard** | ![Materia][1] | ![Materia-compact][2]\n**dark** | ![Materia-dark][3] | ![Materia-dark-compact][4]\n**light** | ![Materia-light][5] | ![Materia-light-compact][6]\n\n[1]: ../images/Materia.png?raw=true\n[2]: ../images/Materia-compact.png?raw=true\n[3]: ../images/Materia-dark.png?raw=true\n[4]: ../images/Materia-dark-compact.png?raw=true\n[5]: ../images/Materia-light.png?raw=true\n[6]: ../images/Materia-light-compact.png?raw=true\n\n## Supported Desktop Environments\n\n- Budgie `>=10.4`\n- Cinnamon `>=3.x`\n- GNOME Shell `>=3.18`\n- MATE `>=1.14`\n- Unity `>=7.4`\n- Xfce `>=4.13`\n\n## Unsupported\n\n- Downstream customized GNOME sessions\n  > e.g. "Ubuntu" session, "Pop" session. To properly use Materia on GNOME Shell, please install `gnome-session` and then switch to "GNOME" or "GNOME on Xorg" session from your display manager.\n- elementary apps\n  > Because they are based on [their own stylesheet](https://github.com/elementary/stylesheet) that conflicts with GTK standards.\n\n## Requirements\n\n- GTK `>=3.20`\n- `gnome-themes-extra` (or `gnome-themes-standard`)\n- Murrine engine \xe2\x80\x94 The package name depends on the distro.\n  - `gtk-engine-murrine` on Arch Linux\n  - `gtk-murrine-engine` on Fedora\n  - `gtk2-engine-murrine` on openSUSE\n  - `gtk2-engines-murrine` on Debian, Ubuntu, etc.\n- `bc` \xe2\x80\x94 build dependency\n\n## Installation\n\n### Distro Packages\n\nDistro | Package Name | Annotation\n--- | --- | ---\nArch Linux | `materia-gtk-theme` | [Link](https://www.archlinux.org/packages/community/any/materia-gtk-theme/)\nDebian 10 or later | `materia-gtk-theme` | [Link](https://packages.debian.org/materia-gtk-theme)\nFedora | `materia-gtk-theme` | Available from [@LaurentTreguier\'s Copr](https://copr.fedorainfracloud.org/coprs/tcg/themes)\nSolus | `materia-gtk-theme` | [Link](https://dev.getsol.us/source/materia-gtk-theme/)\nUbuntu 18.04 or later | `materia-gtk-theme` | [Link](https://packages.ubuntu.com/materia-gtk-theme)\n\n<!-- For contributors, please add your package alphabetically! -->\n\n**Warning:** **Ubuntu disco (19.04) & eoan (19.10)** packages are very outdated and incompatible with GNOME 3.32 or later! If you are using GNOME on Ubuntu, you should _manually_ install the theme.\n\n> Note: Some distro packages (especially Ubuntu ones) may be outdated and incompatible with your desktop environment. You can check the latest version [here](https://github.com/nana-4/materia-theme/releases).\n\n### Flatpak\n\nAll 6 variants are available via Flathub:\n\n```\nflatpak remote-add --if-not-exists flathub https://dl.flathub.org/repo/flathub.flatpakrepo\nflatpak install flathub org.gtk.Gtk3theme.Materia{,-dark,-light}{,-compact}\n```\n\n### Manual Installation\n\nRun the following commands in the terminal:\n\n```sh\ngit clone --depth 1 https://github.com/nana-4/materia-theme\ncd materia-theme\nsudo ./install.sh\n```\n\n> Tip: `./install.sh` allows the following options:\n\n```\n-d, --dest DIR          Specify destination directory (Default: /usr/share/themes)\n-n, --name NAME         Specify theme name (Default: Materia)\n-c, --color VARIANT...  Specify color variant(s) [standard|dark|light] (Default: All variants)\n-s, --size VARIANT      Specify size variant [standard|compact] (Default: All variants)\n-g, --gdm               Install and apply GDM theme (for advanced users)\n                        See also: src/gnome-shell/README.md\n-h, --help              Show help\n```\n\n> For more information, run: `./install.sh --help`\n\n### Manual Uninstallation\n\nDelete the installed directories:\n\n```sh\nsudo rm -rf /usr/share/themes/Materia{,-dark,-light}{,-compact}\n```\n\n## Recommendations\n\n### Font\n\n- To properly display the theme, use a font family including **Medium** weight (e.g. [Roboto](https://github.com/google/roboto) or [M+](https://mplus-fonts.osdn.jp)).\n- Set the font size to `9.75` (= 13px at 96dpi) or `10.5` (= 14px at 96dpi).\n\n### Chrome Theme\n\nTo use the Chrome theme;\n\n1. Open the `chrome` folder on `/usr/share/themes/Materia<-variant>`.\n2. Drag and drop the `.crx` files into the Chrome/Chromium Extensions page (`chrome://extensions`).\n\n### GDM Theme\n\nYou can change the GDM (lock/login screen) theme by replacing the default GNOME Shell theme.  \nSee [`gnome-shell/README.md`](src/gnome-shell/README.md) for details.\n\n## Customization\n\nMateria can be customized with GUI application, [**oomox theme designer**](https://github.com/themix-project/oomox).\n\nMateria also allows you to change the color scheme relatively easily in other ways. See [`HACKING.md`](HACKING.md#how-to-change-the-color-scheme) for details.\n\n## Contributing\n\nIf you find bugs or have suggestions, please report it to the [issue tracker](https://github.com/nana-4/materia-theme/issues).
