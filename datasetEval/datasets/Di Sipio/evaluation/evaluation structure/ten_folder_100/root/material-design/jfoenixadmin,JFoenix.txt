build JFoenix, execute the following command:\n\n    gradlew build\n\nTo run the main demo, execute the following command:\n\n    gradlew run\n\n**NOTE** : You need to set JAVA_HOME environment variable to point to Java 1.8 directory.\n\n**NOTE** : JFoenix requires **Java 1.8u60** and above.\n\n# Demo\nFor detailed steps on getting the demo running, see [the wiki].\n\n# How can I use JFoenix?\nYou can download the source code of the library and build it as mentioned previously. Building JFoenix will generate jfoenix-0.0.0-SNAPSHOT.jar under the jfoenix/build/libs folder. To use JFoenix, import jfoenix-0.0.0-SNAPSHOT.jar into your project and start using the new material design Java components :).\n\n## Gradle\n### How to Include In Gradle Project\n```groovy\nrepositories {\n    mavenCentral()\n}\n```\nReference the repository from this location using:\n* Java 9+\n    ```groovy\n    dependencies {\n        compile \'com.jfoenix:jfoenix:9.0.8\'\n    }\n    ```\n* Java 8\n    ```groovy\n    dependencies {\n        compile \'com.jfoenix:jfoenix:8.0.8\'\n    }\n    ```\n\n## Maven\n### How to Include In Maven Project\n* Java 9+\n    ```xml\n    <dependency>\n        <groupId>com.jfoenix</groupId>\n        <artifactId>jfoenix</artifactId>\n        <version>9.0.8</version>\n    </dependency>\n    ```\n* Java 8\n    ```xml\n    <dependency>\n        <groupId>com.jfoenix</groupId>\n        <artifactId>jfoenix</artifactId>\n        <version>8.0.8</version>\n    </dependency>\n    ```\n#
