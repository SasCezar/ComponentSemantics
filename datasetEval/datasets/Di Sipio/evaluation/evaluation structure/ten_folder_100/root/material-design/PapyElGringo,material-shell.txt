b"# Material Shell\nA **simple**, **productivity oriented** GNOME Shell replacement that provides an **innovative** and **automated** **mouse and keyboard [workflow](./documentation/material-shell.md#workflow)** which aims to be **faster** and **easier** to use and creates a **great user experience**.\n\nMade by following the **[Material Design guidelines](https://material.io)** - a solid baseline that allows us to provide an **aesthetically pleasing** and **highly accessible** interface.\n\n#### [Read more about the workflow and Material Shell](./documentation/material-shell.md#workflow)\n\n## Demo\n\n![Demo GIF](demo.gif)\n\n### Discord\nGet notified about updates and join me at [https://discord.gg/vBb7D9a](https://discord.gg/vBb7D9a)\n#\n#### STATUS: BETA (expect bugs!)\n#### REQUIRES: gnome-shell >=3.32.0\n\n## Installation\n### From source\n1) Clone the project to the gnome-shell extensions folder:\n```bash\ngit clone https://github.com/PapyElGringo/material-shell.git ~/.local/share/gnome-shell/extensions/material-shell@papyelgringo\n```\n2) Reload GNOME Shell:\n  + On X.org: Hit `Alt+F2` and type the command `r`\n  + On Wayland: Log out and back in\n3) Open `gnome-tweaks` and activate the `Material Shell` extension **OR** enable it using \n```bash\ngnome-shell-extension-tool -e material-shell@papyelgringo\n```\n\n### Arch Linux\n1) You can install [Material Shell from the AUR](https://aur.archlinux.org/packages/gnome-shell-extension-material-shell-git/). \\\nAssuming you're using yay:\n```\nyay -S gnome-shell-extension-material-shell-git\n```\n2) Reload GNOME Shell:\n  + On X.org: Hit `Alt+F2` and type the command `r`\n  + On Wayland: Log out and back in\n3) Open `gnome-tweaks` and activate the `Material Shell` extension **OR** enable it using \n```bash\ngnome-shell-extension-tool -e material-shell@papyelgringo\n```\n\n### Other Distributions\n- Fedora: `sudo dnf install gnome-shell-extension-material-shell`\nAnd more to come...\n\n## Workflow Hotkeys\nSome hotkeys might already be used by GNOME Shell - please check your keybindings first.\n#### Desktop navigation\n* `Super+W` Navigate to the upper workspace/category.\n* `Super+S` Navigate to the lower workspace/category.\n* `Super+A` Focus the window at the left of the current window.\n* `Super+D` Focus the window at the right of the current window.\n\n#### Window manipulation\n* `Super+Q` Kill the current window focused.\n* `Super+[MouseDrag]` Move window around.\n* `Super+Shift+A` Move the current window to the left.\n* `Super+Shift+D` Move the current window to the right.\n* `Super+Shift+W` Move the current window to the upper workspace.\n* `Super+Shift+S` Move the current window to the lower workspace.\n\n#### Extra Hotkeys\n* `Super+Space` Cycle the tiling layout of the current workspace.\n* `Super+Escape` Toggle the UI of Material-shell, like a Zen mode.\n\n## Recommended Additional Configuration\n* GTK and GNOME Shell theme: [Plata Theme](https://gitlab.com/tista500/plata-theme)\n* Icon theme: [Tela Icon Theme](https://github.com/vinceliuice/Tela-icon-theme)\n\nThe project is based on my earlier work on [Material Awesome](https://github.com/PapyElGringo/material-awesome).\n"
