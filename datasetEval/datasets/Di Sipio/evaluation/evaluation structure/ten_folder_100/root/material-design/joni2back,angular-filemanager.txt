Multiple file support\n  - Multilanguage\n  - List and Icon view\n  - Multiple file upload\n  - Pick files callback for third parties apps\n  - Search files\n  - Directory tree navigation\n  - Copy, Move, Rename (Interactive UX)\n  - Delete, Edit, Preview, Download\n  - File permissions (Unix chmod style)\n  - Mobile support\n\n### TODO\n  - Drag and drop\n  - Dropbox and Google Drive connectors\n  - Remove usage of jQuery\n\n### Backend API\n[Read the docs](API.md)\n\n---------\n\n### Using in your existing project\n**1) Install deps using yarn with**\n```yarn install```\n\n**2) Include the dependencies in your project**\n```html\n<!-- third party -->\n  <script src="node_modules/jquery/dist/jquery.min.js"></script>\n  <script src="node_modules/angular/angular.min.js"></script>\n  <script src="node_modules/angular-translate/dist/angular-translate.min.js"></script>\n  <script src="node_modules/ng-file-upload/dist/ng-file-upload.min.js"></script>\n  <script src="node_modules/bootstrap/dist/js/bootstrap.min.js"></script>\n  <link rel="stylesheet" href="node_modules/bootswatch/paper/bootstrap.min.css" />\n\n<!-- angular-filemanager -->\n  <link rel="stylesheet" href="dist/angular-filemanager.min.css">\n  <script src="dist/angular-filemanager.min.js"></script>\n```\n\n**3) Use the angular directive in your HTML**\n```html\n<angular-filemanager></angular-filemanager>\n```\n\n---------\n\n### Extending the configuration file by adding a script\n```html\n<script type="text/javascript">\nangular.module(\'FileManagerApp\').config([\'fileManagerConfigProvider\', function (config) {\n  var defaults = config.$get();\n  config.set({\n    appName: \'angular-filemanager\',\n    pickCallback: function(item) {\n      var msg = \'Picked %s "%s" for external use\'\n        .replace(\'%s\', item.type)\n        .replace(\'%s\', item.fullPath());\n      window.alert(msg);\n    },\n\n    allowedActions: angular.extend(defaults.allowedActions, {\n      pickFiles: true,\n      pickFolders: false,\n    }),\n  });\n}]);\n</script>\n```\n\n### Create a new build with your changes\n```sh\n  gulp build || node node_modules/gulp/bin/gulp.js build\n```\n\nYou can do many things by extending the configuration. Like hide the sidebar or the search button. See [the list of default configurations](/src/js/providers/config.js).\n\n---------\n\n### Contribute\nTo contribute to the project you can simply fork this repo. To build a minified version, you can simply run the Gulp\ntask `gulp build`. The minified/uglified files are created in the `dist` folder.\n\n### Versioning\nFor transparency into our release cycle and in striving to maintain backward compatibility, angular-filemanager is maintained under [the Semantic Versioning guidelines](http://semver.org/).\n\n### Copyright and license\nCode and documentation released under [the MIT license](https://github.com/joni2back/angular-filemanager/blob/master/LICENSE).\n'
