<img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> <a href="https://code.visualstudio.com/shortcuts/keyboard-shortcuts-linux.pdf">Linux</a> `Ctrl+P`\n  - <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> <a href="https://code.visualstudio.com/shortcuts/keyboard-shortcuts-macos.pdf">macOS</a> `\xe2\x8c\x98P`\n  - <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> <a href="https://code.visualstudio.com/shortcuts/keyboard-shortcuts-windows.pdf">Windows</a> `Ctrl+P`\n\nPaste the following command and press `Enter`:\n\n```shell\next install material theme\n```\n\nAnd pick the one by **Mattia Astorino** (me) as author.\n\n#### Packaged VSIX Extension\n\n[Download the latest .vsix release](https://marketplace.visualstudio.com/_apis/public/gallery/publishers/Equinusocio/vsextensions/vsc-material-theme/latest/vspackage) file from the marketplace and install it from the command line\n\n```shell\ncode --install-extension vsc-material-theme-*.*.*.vsix\n```\n\nor from within VS Code by launching *Quick Open* and running the *Install from VSIX...* command.\n\n##### GitHub Repository Clone\n\nChange to your `.vscode/extensions` [VS Code extensions directory](https://code.visualstudio.com/docs/extensions/install-extension#_side-loading).\nDepending on your platform it is located in the following folders:\n\n  - <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> **Linux** `~/.vscode/extensions`\n  - <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> **macOs** `~/.vscode/extensions`\n  - <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> **Windows** `%USERPROFILE%\\.vscode\\extensions`\n\nClone the Material Theme repository as `Equinusocio.vsc-material-theme`:\n\n```shell\ngit clone https://github.com/equinusocio/vsc-material-theme.git Equinusocio.vsc-material-theme\n```\n\n\n## Activate theme\n\nLaunch *Quick Open*:\n\n  - <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> <a href="https://code.visualstudio.com/shortcuts/keyboard-shortcuts-linux.pdf">Linux</a> `Ctrl + Shift + P`\n  - <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> <a href="https://code.visualstudio.com/shortcuts/keyboard-shortcuts-macos.pdf">macOS</a> `\xe2\x8c\x98 + Shift + P`\n  - <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> <a href="https://code.visualstudio.com/shortcuts/keyboard-shortcuts-windows.pdf">Windows</a> `Ctrl + Shift + P`\n\nType `theme`, choose `Preferences: Color Theme`, and select one of the Material Theme variants from the list. After activation, the theme will set the correct icon theme based on your active theme variant.\n\n## Set the accent color\n\nLaunch *Quick Open*:\n\n  - <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> <a href="https://code.visualstudio.com/shortcuts/keyboard-shortcuts-linux.pdf">Linux</a> `Ctrl + Shift + P`\n  - <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> <a href="https://code.visualstudio.com/shortcuts/keyboard-shortcuts-macos.pdf">macOS</a> `\xe2\x8c\x98 + Shift + P`\n  - <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> <a href="https://code.visualstudio.com/shortcuts/keyboard-shortcuts-windows.pdf">Windows</a> `Ctrl + Shift + P`\n\nType `material theme`, choose `Material Theme: Set accent color`, and pick one color from the list.\n\n\n## Override theme colors\nYou can override the Material Theme UI and schemes colors by adding these theme-specific settings to your configuration. For advanced customisation please check the [relative section on the VS Code documentation](https://code.visualstudio.com/docs/getstarted/themes#_customizing-a-color-theme).\n\n### Color Scheme override\n\n**Basic example**\n```js\n"editor.tokenColorCustomizations": {\n    "[Material Theme]": {\n        "comments": "#229977"\n    }\n},\n```\n\n**Advanced example**\n\n```js\n"editor.tokenColorCustomizations": {\n    "[Material Theme VARIANT]": {\n        "textMateRules": [\n            {\n                "scope": [\n                    "punctuation.definition.comment",\n                    "comment.block",\n                    "comment.line",\n                    "comment.block.documentation"\n                ],\n                "settings": {\n                    "foreground": "#FF0000"\n                }\n            }\n        ]\n    },\n},\n\n"workbench.colorCustomizations": {\n\t"[Material Theme VARIANT]": {\n\t\t"sideBar.background": "#ff0000",\n\t}\n},\n```\n\n## Recommended settings for a better experience\n\n```js\n{\n    // Controls the font family.\n    "editor.fontFamily": "Operator Mono",\n    // Controls the line height. Use 0 to compute the lineHeight from the fontSize.\n    "editor.lineHeight": 24,\n    // Enables font ligatures\n    "editor.fontLigatures": true,\n    // Controls if file decorations should use badges.\n    "explorer.decorations.badges": false\n}\n```\n\n## Official Portings\nThanks to the Material Theme community you can use is with other softwares, here the official portings:\n\n- Atom
