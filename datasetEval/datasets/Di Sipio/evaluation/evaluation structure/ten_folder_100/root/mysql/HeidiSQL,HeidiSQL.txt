b"![](https://img.shields.io/github/license/HeidiSQL/HeidiSQL.svg?style=flat)\r\n![](https://img.shields.io/github/release/HeidiSQL/HeidiSQL.svg?style=flat)\r\n![](https://img.shields.io/github/languages/top/HeidiSQL/HeidiSQL.svg?style=flat)\r\n![](https://img.shields.io/github/languages/code-size/HeidiSQL/HeidiSQL.svg?style=flat)\r\n\r\n# HeidiSQL\r\nHeidiSQL is a useful and reliable tool designed for web developers using the popular [MariaDB](http://www.mariadb.org/) or [MySQL](http://www.mysql.com/) server, [Microsoft SQL databases](http://www.microsoft.com/sql/) or [PostgreSQL](http://www.postgresql.org/). It enables you to browse and edit data, create and edit tables, views, procedures, triggers and scheduled events. Also, you can export structure and data, either to SQL file, clipboard or to other servers. Read about [features](https://www.heidisql.com/#featurelist) or see some [screenshots](https://www.heidisql.com/screenshots.php). \r\n\r\n# Need help?\r\nLook at [the online help page](http://www.heidisql.com/help.php) to learn how to use HeidiSQL.\r\n\r\n# Building\r\nDelphi 10.3 is required for building HeidiSQL. Older Delphi versions will most likely fail; newer Delphi versions may work or fail. Unfortunately, Lazarus or one \r\nof the other free compilers cannot currently compile HeidiSQL.\r\n\r\nOnce Delphi is installed, you need to load the SynEdit project from the components folder. Build both run-time and design-time packages. Install the \r\ndesign-time package. Do the same for the VirtualTree component project, and install madExcept.\r\n\r\nAfterwards, load the HeidiSQL project from the packages folder.\r\n\r\n# Translation\r\nIf you'd like to contribute by translating HeidiSQL into your mother tongue, you need to register at\r\n[Transifex](https://www.transifex.com/heidisql/heidisql/), and join an existing language or request a\r\nnew one.\r\n\r\n# Icons8 copyright\r\nIcons added in January 2019 into a TImageCollection component are copyright by [Icons8](https://icons8.com).
