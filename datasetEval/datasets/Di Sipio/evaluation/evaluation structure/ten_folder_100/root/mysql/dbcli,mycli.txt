quit.\n      --help                        Show this message and exit.\n\nFeatures\n--------\n\n`mycli` is written using [prompt_toolkit](https://github.com/jonathanslenders/python-prompt-toolkit/).\n\n* Auto-completion as you type for SQL keywords as well as tables, views and\n  columns in the database.\n* Syntax highlighting using Pygments.\n* Smart-completion (enabled by default) will suggest context-sensitive completion.\n    - `SELECT * FROM <tab>` will only show table names.\n    - `SELECT * FROM users WHERE <tab>` will only show column names.\n* Support for multiline queries.\n* Favorite queries with optional positional parameters. Save a query using\n  `\\fs alias query` and execute it with `\\f alias` whenever you need.\n* Timing of sql statments and table rendering.\n* Config file is automatically created at ``~/.myclirc`` at first launch.\n* Log every query and its results to a file (disabled by default).\n* Pretty prints tabular data (with colors!)\n* Support for SSL connections\n\nContributions:\n--------------\n\nIf you\'re interested in contributing to this project, first of all I would like\nto extend my heartfelt gratitude.
