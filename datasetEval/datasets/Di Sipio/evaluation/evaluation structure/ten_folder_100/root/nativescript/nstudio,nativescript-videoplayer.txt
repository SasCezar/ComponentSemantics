Usage\n\n###\n\n```XML\n<Page xmlns="http://schemas.nativescript.org/tns.xsd"\n      xmlns:VideoPlayer="nativescript-videoplayer">\n        <StackLayout>\n\n            <VideoPlayer:Video id="nativeVideoPlayer"\n            controls="true" loop="true" autoplay="false" height="280"\n            src="~/videos/big_buck_bunny.mp4" />\n\n            <!-- Remote file to test with https://clips.vorwaerts-gmbh.de/big_buck_bunny.mp4 -->\n\n        </StackLayout>\n</Page>\n```\n\n```typescript\nimport { Video } from \'nativescript-videoplayer\';\n\nconst video = topmost().currentPage.getViewById(\'nativeVideoPlayer\') as Video;\n// Setting event listeners on the Video\nvideo.on(Video.pausedEvent, () => {\n  console.log(\'Video has been paused.\');\n});\n\nvideo.on(Video.mutedEvent, () => {\n  console.log(\'Video has been muted.\');\n});\n\n// changing the src\nvideo.src = \'some video file or url\';\n\n// set loop\nvideo.loop = false;\n```\n\n## Angular NativeScript Usage\n\n```TS\n// somewhere at top of your component or bootstrap file\nimport { registerElement } from "nativescript-angular/element-registry";\nimport { Video } from \'nativescript-videoplayer\';\nregisterElement("VideoPlayer", () => Video);\n// documentation: https://docs.nativescript.org/angular/plugins/angular-third-party.html#simple-elements\n```\n\n_With AngularNative you have to explicitly close all components so the correct template code is below._\n\n```XML\n  <VideoPlayer\n      src="https://clips.vorwaerts-gmbh.de/big_buck_bunny.mp4"\n      autoplay="true"\n      height="300"></VideoPlayer>\n```\n\n## Properties\n\n| Property                            | Description                                                                                                                                                                                                                             |\n| ----------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |\n| **src**                             | The src file for the video. Set the video file to play, for best performance use local video files if possible. The file must adhere to the platforms accepted video formats. For reference check the platform specs on playing videos. |\n| **autoplay - (boolean)**            | Set if the video should start playing as soon as possible or to wait for user interaction.                                                                                                                                              |\n| **controls - (boolean)**            | Set to use the native video player\'s media playback controls.                                                                                                                                                                           |\n| **muted - (boolean)**               | Mutes the native video player.                                                                                                                                                                                                          |\n| **loop - (boolean)**                | Sets the native video player to loop once playback has finished.                                                                                                                                                                        |\n| **fill - (boolean)**                | If true, the aspect ratio of the video will not be honored and it will fill the entire space available.                                                                                                                                 |\n| **observeCurrentTime - (boolean)**  | If true, currentTimeUpdated callback is possible.                                                                                                                                                                                       |\n| **headers - (Map<string, string>)** | Set headers to add when loading a video from URL.                                                                                                                                                                                       |\n\n## API\n\n| Method                        | Description                                                         |\n| ----------------------------- | ------------------------------------------------------------------- |\n| **play**                      | Start playing the video.                                            |\n| **pause**                     | Pause the video                                                     |\n| **seekToTime(time: number)**  | Seek the video to a time (milliseconds)                             |\n| **getCurrentTime**            | Returns the current time in the video duration (milliseconds)       |\n| **getDuration**               | Returns the current time in the video duration (milliseconds)       |\n| **destroy**                   | Destroy the video player and free resources                         |\n| **mute(boolean)**             | If true, mutes the video. If false, unmute the video.               |\n| **setVolume(volume: number)** | Set the volume - Must be between 0 and 1.                           |\n| **ANDROID ONLY** - **stop**   | Stop the playback - this resets the player and remove the video src |\n\n## Observable Properties\n\n* **currentTime()** - Current time of video.\n\n## Events\n\n| Event                   | Description                                                         |\n| ----------------------- | ------------------------------------------------------------------- |\n| errorEvent              | This event fires when an error in the source code is thrown.        |\n| playbackReadyEvent      | This event fires when the video is ready.                           |\n| playbackStartEvent      | This event fires when video starts playback.                        |\n| seekToTimeCompleteEvent | This event fires when seeking is complete.                          |\n| currentTimeUpdatedEvent | This event fires when the current time of playing video is changed. |\n| finishedEvent           | This event fires when the video is complete.                        |\n| mutedEvent              | This event fires when video is muted.                               |\n| unmutedEvent            | This event fires when video is unmutedEvent.                        |\n| pausedEvent             | This event fires when video is paused.                              |\n| volumeSetEvent          | This event fires when the volume is set.                            |\n\n## iOS Logging\n\nWhen running the iOS Simulator, after playing a video the iOS system may write\nlog messages to the console every few seconds of the form\n\n```\n[aqme] 254: AQDefaultDevice (173): skipping input stream 0 0 0x0\n```\n\nThey will continue being logged even after the video has been properly destroyed.\nThese messages can be safely ignored. To turn them off completely, run the following\ncommand in your shell before running `tns run ios`:\n\n```\nexport SIMCTL_CHILD_OS_ACTIVITY_MODE="disable"\n```\n'
