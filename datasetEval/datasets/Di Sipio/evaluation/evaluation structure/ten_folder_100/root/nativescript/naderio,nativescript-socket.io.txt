Both Android and iOS implementations are stable and in sync.\n\nPlease check [releases and changelog](https://github.com/naderio/nativescript-socket.io/releases) and [roadmap](https://github.com/naderio/nativescript-socket.io/issues/3) for more information.  \n\n# Dependencies\n\nAndroid: [Socket.IO-client Java](https://github.com/socketio/socket.io-client-java) v1.*\n\niOS: [Socket.IO-Client-Swift](https://github.com/socketio/socket.io-client-swift) v12.*\n\n# Install\n\n```sh\ntns plugin add nativescript-socket.io\n```\n\n# Usage\n\n```javascript\nconst SocketIO = require(\'nativescript-socket.io\');\n\nSocketIO.enableDebug(); // optionnal\n\n// or use your own debug function\n// SocketIO.enableDebug(myCustomDebugFunction);\n\nconst options = {\n  query: {\n    token: \'SOME_TOKEN_HERE\',\n  },\n  android: {\n    // http://socketio.github.io/socket.io-client-java/apidocs/io/socket/client/IO.Options.html\n  }\n  ios: {\n    // https://nuclearace.github.io/Socket.IO-Client-Swift/Enums/SocketIOClientOption.html\n  }\n};\n\nconst socketio = SocketIO.connect(\'http://somewhere/namespace\', options);\n\nsocketio.on(\'connect\', function(){\n  console.log(\'connect\');\n});\n\nsocketio.on(\'hello\', function(){\n  console.log(\'hello\', arguments);\n});\n\nsocketio.on(\'request\', function(info, ack){\n  console.log(\'request\', info);\n  if (info === \'datetime\') {\n    ack(new Date());\n  } elseif (info === \'random\') {\n    ack(Math.random());\n  } else {\n    ack(null);\n  }\n});\n\nsocketio.emit(\'hello\', {\n  username: \'someone\',\n});\n\nsocketio.emit(\'hello-ack\', {\n  username: \'someone\',\n}, function ack(){\n  console.log(\'hello-ack\', arguments);\n})\n\n```\n\n## Usage with TypeScript\n\nimport using either of the following:\n\n```typescript\nimport * as SocketIO from "nativescript-socket.io"`\n\n// OR\n\nconst SocketIO = require("nativescript-socket.io")\n```\n\n# Demo\n\n1. replace all instance of `192.168.1.111` with your IP address or domain.\n\n2. start the socket.io demo server\n\n```sh\ncd ./demo.server\nnpm start\n```\n\n3. build and run the app\n\n```sh\ncd ./demo\ntns run android\n```\n'
