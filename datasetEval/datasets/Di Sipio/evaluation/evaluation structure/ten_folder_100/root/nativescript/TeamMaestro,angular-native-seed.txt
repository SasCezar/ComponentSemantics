b'# Angular NativeScript Seed\n\nBuild truly web and native applications using NativeScript and Angular. This seed allows your project to share code between the environments; having complete control over dependencies.\n\n## Highlights\n- i18n Translations\n- Lazy Loaded Modules\n- Angular CLI // Webpack // Testing\n- Target Phone and Tablet Templates Individually. The following extensions are supported:\n\nExtension                        | Platform\n-------------------------------- | -----------\n`.{html/scss}` \xc2\xa0 \xc2\xa0 \xc2\xa0 \xc2\xa0 \xc2\xa0 \xc2\xa0 \xc2\xa0 \xc2\xa0 \xc2\xa0 | Recommended for Web. Does apply to both platforms when **.tns** equivalent does not exist.\n`.tns.{html/scss}` \xc2\xa0 \xc2\xa0 \xc2\xa0 \xc2\xa0 \xc2\xa0 \xc2\xa0 \xc2\xa0 | Only for mobile\n`.tns.ios.{html/scss}` \xc2\xa0         | Only for iOS\n`.tns.android.{html/scss}`       | Only for Android\n`.tns.ios.phone.{html/scss}`     | Only for iOS Phone \n`.tns.android.phone.{html/scss}` | Only for Android Phone\n- Docker build provided using NGINX to serve web content and load balance reverse proxied backends. (See nginx folder for setup instructions)\n\n## Getting Started\n\nWeb is setup from the root.\n\n```\nnpm i\n```\n\nMobile is setup from the `nativescript` folder.\n\n```\ncd nativescript\nnpm i\n```\n\nEach platform is run respective to it\'s folder.
