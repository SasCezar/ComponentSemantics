Like the IBM\'s [LoopBack], Google\'s [Angular 2], Telerik\'s [NativeScript 2] and [Ionic 2] Frameworks.\n\n[FireLoop] also provides you with exclusive modules that will allow you to automatically build your [FireBase] alike Real-Time APIs, Client SDKs, Statistics, **Back and Front Ends development in [TypeScript]!!!**.\n\n*By the way, it\'s cool and open source!!!*\n\n[Public Cheering in the Background]\n\n## FireLoop Features\n\n- Nice and easy setup.\n- 1 command line tool for all (Server, Web Clients, Mobile Clients, SDK Builder).\n- Back and Front Ends in [TypeScript].\n- [LoopBack] 2 and 3 Integration.\n- [Angular 2] Integration.\n- [NativeScript 2] Integration.\n- [Ionic 2] Integration.\n- [SDK Builder] Integration.\n- [Real-Time] Integration.\n\n## Installing FireLoop CLI\n\n````sh\n$ npm install -g @mean-expert/fireloop\n````\n\n## Create FireLoop Project, Clients and SDK\n````sh\n$ mkdir myproject && cd myproject\n$ fireloop\n````\n\n## Documentation\n\nYou can read the complete documentation within the following [Documentation Link](http://docs.fireloop.io/)\n\n## Contributors\n\nThanks goes to these wonderful people ([emoji key](https://github.com/kentcdodds/all-contributors#emoji-key)):\n\n<!-- ALL-CONTRIBUTORS-LIST:START - Do not remove or modify this section -->\n| [<img src="https://avatars0.githubusercontent.com/u/1533239?v=3" width="100px;"/><br /><sub>Jonathan Casarrubias</sub>](http://mean.expert/)<br />[\xf0\x9f\x92\xbb](https://github.com/mean-expert-official/fireloop.io/commits?author=jonathan-casarrubias) | [<img src="https://avatars2.githubusercontent.com/u/6089253?v=3" width="100px;"/><br /><sub>Brannon N.
