the page._\n\n### NativeScript Plain\n\n<span style="color:red">IMPORTANT: </span>_Make sure you include `xmlns:Card="@nstudio/nativescript-cardview"` on the Page element_\n\n#### XML\n\n```xml\n<Page xmlns:Card="@nstudio/nativescript-cardview">\n   <StackLayout>\n      <Card:CardView class="cardStyle" margin="10" elevation="40" radius="5">\n           <grid-layout rows="200, auto, auto" columns="auto, auto, *">\n               <image src="~/images/batman.jpg" stretch="aspectFill" colSpan="3" row="0" />\n               <label text="Batman wants to be friends?" class="info" textWrap="true" row="1" colSpan="3" />\n               <button text="DECLINE" tap="goAway" row="2" col="0" />\n               <button text="ACCEPT" row="2" col="1" />\n           </grid-layout>\n       </Card:CardView>\n   </StackLayout>\n</Page>\n```\n\n#### CSS\n\n```css\n.cardStyle {\n  background-color: #3489db;\n  color: #fff;\n}\n```\n\n---\n\n### NativeScript Angular\n\n```typescript\nimport { registerElement } from \'nativescript-angular/element-registry\';\nimport { CardView } from \'@nstudio/nativescript-cardview\';\nregisterElement(\'CardView\', () => CardView);\n```\n\n```html\n<CardView class="cardStyle" margin="10" elevation="40" radius="1">\n  <GridLayout rows="10,30,30,250, auto, auto,10" columns="10,40, *, 30,10">\n    <image\n      src="res://profile"\n      stretch="aspectFit"\n      verticalAlignment="stretch"\n      col="1"\n      row="1"\n      rowSpan="2"\n    ></image>\n    <label\n      class="createdBy text-left"\n      horizontalAlignment="left"\n      [text]="item.CreatedBy"\n      row="1"\n      col="2"\n      textWrap="true"\n    ></label>\n    <label\n      class="createdOn text-left"\n      horizontalAlignment="left"\n      [text]="item.UpdatedDate"\n      row="2"\n      col="2"\n    ></label>\n    <image\n      [src]="\'https://img.youtube.com/vi/\'+item.MediaURL+\'/mqdefault.jpg\'"\n      stretch="aspectFit"\n      colSpan="3"\n      col="1"\n      row="3"\n    ></image>\n    <label\n      horizontalAlignment="left"\n      [text]="item.Title"\n      colSpan="3"\n      row="4"\n      textWrap="true"\n      col="1"\n    ></label>\n    <label\n      horizontalAlignment="left"\n      [text]="item.Summary"\n      textWrap="true"\n      col="1"\n      row="5"\n      colSpan="3"\n    ></label>\n  </GridLayout>\n</CardView>\n```\n\n---\n\n### NativeScript Vue\n\n```javascript\nimport Vue from \'nativescript-vue\';\nVue.registerElement(\n  \'CardView\',\n  () => require(\'@nstudio/nativescript-cardview\').CardView\n);\n```\n\n```html\n<card-view margin="10" elevation="40" radius="1">\n  <stack-layout>\n    <label text="Hello World"></label>\n  </stack-layout>\n</card-view>\n```\n\n---\n\n### Attributes\n\n- **radius** _optional_\n\nAn attribute to control the \'border-radius\' of the card.\n\n### Platform specific options\n\n#### Android\n\n- **elevation** _optional_\n\nAn attribute to set the elevation of the card. This will increase the \'drop-shadow\' of the card.\nThere can be some performance impact when using a very high elevation value.\n\n- **ripple** _optional_\n\nSet to \'true\' to show a ripple when the card is tapped. Tap event handlers in the card content will prevent the ripple.\n\n#### iOS\n\n- **shadowOffsetWidth** _optional_\n\nAn attribute to offset the x position of the shadow behind the card.\n\n- **shadowOffsetHeight** _optional_\n\nAn attribute to offset the y position of the shadow behind the card.\n\n- **shadowColor** _optional_\n\nAn attribute to set the color of the shadow behind the card.\n\n- **shadowOpacity** _optional_\n\nAn attribute to set the opacity of the shadow behind the card.\n\n- **shadowRadius** _optional_\n\nAn attribute to set the radius of the shadow (shadow spread) behind the card.\n\nThe default values are set to:\n\n```\nradius = 2;\nshadowOffsetWidth = 0;\nshadowOffsetHeight = 2;\nshadowColor = new Color(\'#000\').ios\nshadowOpacity = 0.4;\nshadowRadius = 1;\n```\n\n### Sample Screenshots\n\n#### Android\n\n| Sample 1                            | Sample 2                            |\n| ----------------------------------- |
