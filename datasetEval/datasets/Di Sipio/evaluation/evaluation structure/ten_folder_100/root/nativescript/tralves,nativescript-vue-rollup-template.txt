Dependencies:\n- Install NativeScript tools (see http://docs.nativescript.org/start/quick-setup)\n- Install rollup globally (`npm install --global rollup`) -OR- add `./node_modules/.bin` to the `PATH`.\n\n2. Create app from this template\n```bash\ntns create hello-ns-vue --template nativescript-vue-rollup-template\n\ncd hello-ns-vue\n```\n\n3. Watch for changes while developing\n\nIn two separate terminals run:\n```\n# terminal 1\nrollup -c -w\n\n# terminal 2\ncd tns\ntns run android\n# or\ntns run ios\n```\n\n4. Code!\nCheck [Groceries Vue](https://github.com/tralves/groceries-ns-vue) to see this template\nin action in a complete app.\n"
