b'# NativeScript Slides for iOS and Android\n\n[![npm](https://img.shields.io/npm/v/nativescript-slides.svg)](https://www.npmjs.com/package/nativescript-slides)\n[![npm](https://img.shields.io/npm/dt/nativescript-slides.svg?label=npm%20downloads)](https://www.npmjs.com/package/nativescript-slides)\n\n### _The plugin formally known as nativescript-intro-slides_\n\n### Intro slides example:\n\n[![Nativescript Slides. Click to Play](https://img.youtube.com/vi/kGby8qtSDjM/0.jpg)](https://www.youtube.com/embed/kGby8qtSDjM)\n\n### Image carousel example:\n\n[![Nativescript Slides. Click to Play](https://img.youtube.com/vi/RsEqGAKm62k/0.jpg)](https://www.youtube.com/embed/RsEqGAKm62k)\n\n_videos by [Brad Martin](https://github.com/bradmartin)_\n\n## Example Usage:\n\n### XML\n\n```xml\n\n\t<Slides:SlideContainer id="slides" pageIndicators="true">\n\t\t\t<Slides:Slide class="slide-1">\n\t\t\t\t<Label text="This is Panel 1"  />\n\t\t\t</Slides:Slide>\n\t\t\t<Slides:Slide class="slide-2">\n\t\t\t\t<Label text="This is Panel 2"  />\n\t\t\t</Slides:Slide>\n\t\t\t<Slides:Slide class="slide-3">\n\t\t\t\t<Label text="This is Panel 3"  />\n\t\t\t</Slides:Slide>\n\t\t\t<Slides:Slide class="slide-4">\n\t\t\t\t<Label text="This is Panel 4"  />\n\t\t\t</Slides:Slide>\n\t\t\t<Slides:Slide class="slide-5">\n\t\t\t\t<Label text="This is Panel 5"  />\n\t\t\t</Slides:Slide>\n\t</Slides:SlideContainer>\n\n```\n\n### CSS\n\n```css\n.slide-1 {\n  background-color: darkslateblue;\n}\n\n.slide-2 {\n  background-color: darkcyan;\n}\n.slide-3 {\n  background-color: darkgreen;\n}\n\n.slide-4 {\n  background-color: darkgoldenrod;\n}\n.slide-5 {\n  background-color: darkslategray;\n}\nlabel {\n  text-align: center;\n  width: 100%;\n  font-size: 35;\n  margin-top: 35;\n}\n```\n\nGreat for Intros/Tutorials to Image Carousels.\n\nTo use the intro slide plugin you need to first import it into your xml layout with `xmlns:Slides="nativescript-slides"`\n\nwhen using the intro slide plugin you need at least two `<Slides:Slide>` views inside of the `<Slides:SlideContainer>`.\n\nadd as many `<Slides:Slide>` as you want.\n\n### Methods for SlideContainer\n\n- **nextSlide()** - navigate to the next slide (right direction)\n- **previousSlide()** - navigate to the previous slide (left direction)\n- **goToSlide(index)** : - goes to the slide at the specified index\n\n### Attributes for SlideContainer\n\n- **loop : boolean** - If true will cause the slide to be an endless loop. The suggested use case would be for a Image Carousel or something of that nature.\n\n- **disablePan : boolean** - If true panning is disabled. So that you can call nextSlide()/previousSlide() functions to change the slide. If slides is used to get details about users like email, phone number, username etc. in this case you don\'t want users to move from one slide to another slide without filling details.\n\n- **pagerOffset : string** - Margin-top for the pager. Number or percentage, default 88%.\n\n- **pageIndicators : boolean** - If true adds indicator dots to the bottom of your slides.\n\n- \\*\\*slideWidth: number - set the width of your slides. (Only currently works on android).\n\n#### Indicators\n\nIf the property `pageIndicators` is `true` you won\'t see the page indicators anymore as of 2.0.0 right away. there are two css classes exposed that you can setup however you like for active and inactive indicators. below is an example for semi translucent dots.\n\n```css\n.slide-indicator-inactive {\n  background-color: #fff;\n  opacity: 0.4;\n  width: 10;\n  height: 10;\n  margin-left: 2.5;\n  margin-right: 2.5;\n  margin-top: 0;\n  border-radius: 5;\n}\n\n.slide-indicator-active {\n  background-color: #fff;\n  opacity: 0.9;\n  width: 10;\n  height: 10;\n  margin-left: 2.5;\n  margin-right: 2.5;\n  margin-top: 0;\n  border-radius: 5;\n}\n```\n\n#### Events\n\n- **start** - Start pan\n- **changed** - Transition complete\n- **cancelled** - User didn\'t complete the transition, or at start\\end with no slides\n- **finished** - Last slide has come into view\n\n#### Angular 2 compatibility\n\nI\'ve started working on a Angular 2 version they can be checked out here:\n[Angular 2 version of slides](https://github.com/JoshDSommer/nativescript-ng2-slides)\n\nIf you want to use this plugin with Angular 2 the `registerElement` from `nativescript-angular` you will want to set the `SlideContainer`\'s property of `angular` to `true`. Then in your angular component in the `ngAfterViewInit`. you will want to have an instance of your slide container to call the function `constructView()`.\n[Follow the example](https://github.com/JoshDSommer/nativescript-slides/issues/37#issuecomment-224820901)\n\n#### Plugin Development Work Flow:\n\n- Clone repository to your machine.\n- Run `npm run setup` to prepare the demo project\n- Build with `npm run build`\n- Run and deploy to your device or emulator with `npm run demo.android` or `npm run demo.ios`\n\n#### Known issues\n\n- There appears to be a bug with the loop resulting in bad transitions going right to left.\n- Currently in Android there is an known issue when a slide component inside of a scroll view.\n\n#### How To: Load slides dynamically\n\nYou want to hook into the loaded event of the view and then create your view elements.\n\n[Demo Code](https://github.com/JoshDSommer/nativescript-slides/blob/master/demo/app/dynamic-page.xml)\n\n```xml\n<Slides:SlideContainer loaded="onSlideContainerLoaded"\n```\n\n```ts\nimport * as slides from \'nativescript-slides/nativescript-slides\';\n\nexport function onSlideContainerLoaded(args) {\n  let slideContainer = <slides.SlideContainer>args.object;\n\n  //Construct the slides\n  slideContainer.addChild(getSlide(\'Page 1\', \'slide-1\'));\n  slideContainer.addChild(getSlide(\'Page 2\', \'slide-2\'));\n  slideContainer.addChild(getSlide(\'Page 3\', \'slide-3\'));\n  slideContainer.addChild(getSlide(\'Page 4\', \'slide-4\'));\n  slideContainer.addChild(getSlide(\'Page 5\', \'slide-5\'));\n}\n\nfunction getSlide(labelText: string, className: string) {\n  let slide = new slides.Slide();\n  slide.className = className;\n  let label = new labelModule.Label();\n  label.text = labelText;\n  slide.addChild(label);\n\n  return slide;\n}\n```\n\n### Thanks to these
