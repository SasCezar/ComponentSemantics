b'# Nativescript-Angular-Drawer-Template\nA starter template to quickly create nativescript angular projects with drawer pages.\n\n## Usage\n`tns create my-project-name --template nativescript-angular-drawer-template`\n\n## Preview\n### Android\n![android-preview](http://ngxp.io/wp-content/product-previews/nativescript-angular-drawer-template/ns-ng-drawer-template-android.gif)\n\n### iOS\n![ios-preview](http://ngxp.io/wp-content/product-previews/nativescript-angular-drawer-template/ns-ng-drawer-template-ios.gif)\n\n## How To Change Menu Items\nYou can change the menu items of drawer from `app/modules/shared/side-drawer-page/side-drawer-page.component.ts` file as shown below:\n\n```\nnavMenu: any[] = [\n    { name: \'Home\', commands: [\'/\'] },\n    { name: \'About\', commands: [\'/about\'] },\n    { name: \'Contact\', commands: [\'/contact\'] }\n];\n```\n\n## How To Create New Page\nYou just need to wrap the content template of the new page inside `<side-drawer-page>` tag as shown below:\n\n```\n<side-drawer-page>\n    <GridLayout>\n        <Label text="Hello World"></Label>\n    </GridLayout>\n</side-drawer-page>\n``` \nYou can refer home, contact or about sample modules for reference.\n\n'
