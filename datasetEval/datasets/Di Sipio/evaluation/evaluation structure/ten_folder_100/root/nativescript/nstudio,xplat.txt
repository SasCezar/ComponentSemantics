Version](https://badge.fury.io/js/%40nstudio%2Fschematics.svg)](https://www.npmjs.com/@nstudio/schematics)\n\n</div>\n<hr>\n\n**xplat** is an added value pack for [Nx](https://nrwl.io/nx) which provides additional app generators and optional supporting architecture for different platform/framework combinations.\n\n## Currently supported platforms\n\n- [Electron](https://electronjs.org/)\n  > Build cross platform desktop apps with JavaScript, HTML, and CSS.\n- [Ionic](https://ionicframework.com/)\n  > Build amazing apps in one codebase, for any platform, with the web.\n- [NativeScript](https://www.nativescript.org/)\n  > Open source framework for building truly native mobile apps with Angular, Vue.js, TypeScript, or JavaScript.\n\n## Documentation\n\n- [Getting Started](https://nstudio.io/xplat/getting-started)\n- [Fundamentals](https://nstudio.io/xplat/fundamentals)\n- [API Documentation](https://nstudio.io/xplat/api)\n- [Design
