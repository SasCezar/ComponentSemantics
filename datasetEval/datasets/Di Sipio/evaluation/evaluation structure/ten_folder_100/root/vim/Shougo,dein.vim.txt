b'# dein.vim\n\n[![Join the chat at https://gitter.im/Shougo/dein.vim](https://badges.gitter.im/Shougo/dein.vim.svg)](https://gitter.im/Shougo/dein.vim?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge) [![Build Status](https://travis-ci.org/Shougo/dein.vim.svg?branch=master)](https://travis-ci.org/Shougo/dein.vim)\n\nDein.vim is a dark powered Vim/Neovim plugin manager.\n\n<!-- vim-markdown-toc GFM -->\n\n- [Requirements](#requirements)\n- [Quick start](#quick-start)\n  - [Unix/Linux or Mac OS X](#unixlinux-or-mac-os-x)\n- [Features](#features)\n- [Future works (TODO)](#future-works-todo)\n  - [Options](#options)\n\n<!-- vim-markdown-toc -->\n\n## Requirements\n\n- Vim 8.0 or above or NeoVim.\n- "xcopy" command in $PATH (Windows)\n- "git" command in $PATH (if you want to install github or vim.org plugins)\n\nNote: If you use Vim 7.4, please use dein.vim ver.1.5 instead.\n\nIf you need vim-plug like install UI, you can use dein-ui.vim.\nhttps://github.com/wsdjeg/dein-ui.vim\n\n## Quick start\n\n**Note**: You must define the installation directory before to use dein.  The\ndirectory that you will want to use depends on your usage.\n\nFor example, `~/.vim/bundles` or `~/.cache/dein` or `~/.local/share/dein`.\ndein.vim does not define a default installation directory.\nYou must **not** set the installation directory as `~/.vim/plugin` or\n`~/.config/nvim/plugin`.\n\n### Unix/Linux or Mac OS X\n\n1. Run below script.\n\n```sh\ncurl https://raw.githubusercontent.com/Shougo/dein.vim/master/bin/installer.sh > installer.sh\n# For example, we just use `~/.cache/dein` as installation directory\nsh ./installer.sh ~/.cache/dein\n```\n\n2. Edit your .vimrc like this.\n\n```vim\nif &compatible\n  set nocompatible\nendif\n" Add the dein installation directory into runtimepath\nset runtimepath+=~/.cache/dein/repos/github.com/Shougo/dein.vim\n\nif dein#load_state(\'~/.cache/dein\')\n  call dein#begin(\'~/.cache/dein\')\n\n  call dein#add(\'~/.cache/dein/repos/github.com/Shougo/dein.vim\')\n  call dein#add(\'Shougo/deoplete.nvim\')\n  if !has(\'nvim\')\n    call dein#add(\'roxma/nvim-yarp\')\n    call dein#add(\'roxma/vim-hug-neovim-rpc\')\n  endif\n\n  call dein#end()\n  call dein#save_state()\nendif\n\nfiletype plugin indent on\nsyntax enable\n```\n\n3. Open vim and install dein\n\n```vim\n:call dein#install()\n```\n\n## Features\n\n- Faster than NeoBundle\n\n- Simple\n\n- No commands, Functions only to simplify the implementation\n\n- Easy to test and maintain\n\n- No Vundle/NeoBundle compatibility\n\n- neovim/Vim8 asynchronous API installation support\n\n- Local plugin support\n\n- Non github plugins support\n\n- Go like clone directory name ex:"github.com/{user}/{repository}"\n\n- Merge the plugins directories automatically to avoid long \'runtimepath\'\n\n## Future works (TODO)\n\n- Other types support (zip, svn, hg, ...)\n\n- Metadata repository support\n\n### Options\n\nSome common options. For a more detailed list, run `:h dein-options`\n\n| Option   | Type                 | Description                                                                           |\n| -------- | -------------------- | ------------------------------------------------------------------------------------- |\n| `name`   | `string`             | The name for a plugin. If it is omitted, the tail of the repository name will be used |\n| `rev`    | `string`             | The revision number or branch/tag name for the repo                                   |\n| `build`  | `string`             | Command to run after the plugin is installed                                          |\n| `on_ft`  | `string` or `list`   | Load a plugin for the current filetype                                                |\n| `on_cmd` | `string` or `list`   | Load the plugin for these commands                                                    |\n| `rtp`    | `string`             | You can use this option when the repository has the Vim plugin in a subdirectory               |\n| `if`     | `string` or `number` | If it is String, dein will eval it.                                                   |\n| `merged` | `number`             | If set to 0, dein doesn\'t merge the plugin directory.                                 |\n'
