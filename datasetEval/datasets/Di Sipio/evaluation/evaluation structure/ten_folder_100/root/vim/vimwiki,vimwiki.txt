b'# VimWiki: A Personal Wiki For Vim\n\n[\xe4\xb8\xad\xe6\x96\x87](README-cn.md)\n\n- [Intro](#intro)\n- [Installation](#installation)\n    - [Prerequisites](#prerequisites)\n    - [VIM Packages](#installation-using-vim-packages-since-vim-741528)\n    - [Pathogen](#installation-using-pathogen)\n    - [Vim-Plug](#installation-using-vim-plug)\n    - [Vundle](#installation-using-vundle)\n- [Basic Markup](#basic-markup)\n    - [Lists](#lists)\n- [Key Bindings](#key-bindings)\n- [Commands](#commands)\n- [Changing Wiki Syntax](#changing-wiki-syntax)\n- [Getting Help](#getting-help)\n- [Helping VimWiki](#helping-vimwiki)\n- [Wiki](https://github.com/vimwiki/vimwiki/wiki)\n- [License](#license)\n\n## Intro\n\nVimWiki is a personal wiki for Vim -- a number of linked text files that have\ntheir own syntax highlighting.\n\nWith VimWiki you can:\n\n * Organize notes and ideas\n * Manage to-do lists\n * Write documentation\n * Maintain a diary\n * Export everything to HTML\n\nTo do a quick start press `<Leader>ww` (this is usually `\\ww`) to go to your\nindex wiki file. By default it is located in `~/vimwiki/index.wiki`.\n\nFeed it with the following example:\n\n```\n\n= My knowledge base =\n    * Tasks -- things to be done _yesterday_!!!\n    * Project Gutenberg -- good books are power.\n    * Scratchpad -- various temporary stuff.\n\n```\n\nPlace your cursor on `Tasks` and press Enter to create a link. Once pressed,\n`Tasks` will become `[[Tasks]]` -- a VimWiki link. Press Enter again to\nopen it. Edit the file, save it, and then press Backspace to jump back to your\nindex.\n\nA VimWiki link can be constructed from more than one word. Just visually\nselect the words to be linked and press Enter. Try it with `Project Gutenberg`.\nThe result should look something like:\n\n```\n\n= My knowledge base =\n    * [[Tasks]] -- things to be done _yesterday_!!!\n    * [[Project Gutenberg]] -- good books are power.\n    * Scratchpad -- various temporary stuff.\n\n```\n\n## Screenshots\n\n![Lists View](doc/lists.png)\n![Entries View](doc/entries.png)\n![Todos View](doc/todos.png)\n![Wiki View](doc/wiki.png)\n\n## Installation\n\n### Prerequisites\n\nMake sure you have these settings in your vimrc file:\n\n```vim\n\nset nocompatible\nfiletype plugin on\nsyntax on\n\n```\n\nWithout them VimWiki will not work properly.\n\n\n#### Installation using [Vim packages](http://vimhelp.appspot.com/repeat.txt.html#packages) (since Vim 7.4.1528)\n\n```sh\n\ngit clone https://github.com/vimwiki/vimwiki.git ~/.vim/pack/plugins/start/vimwiki\n\n```\n\n#### Installation using [Pathogen](http://www.vim.org/scripts/script.php?script_id=2332)\n\n```sh\n\ncd ~/.vim\nmkdir bundle\ncd bundle\ngit clone https://github.com/vimwiki/vimwiki.git\n\n```\n\n#### Installation using [Vim-Plug](https://github.com/junegunn/vim-plug)\n\nAdd the following to the plugin-configuration in your vimrc:\n\n```vim\n\nPlug \'vimwiki/vimwiki\'\n\n```\n\nThen run `:PlugInstall`.\n\n#### Installation using [Vundle](https://github.com/VundleVim/Vundle.vim)\n\nAdd `Plugin \'vimwiki/vimwiki\'` to your vimrc file and run\n\n```sh\n\nvim +PluginInstall +qall\n\n```\n\nOr download the [zip\narchive](https://github.com/vimwiki/vimwiki/archive/master.zip) and extract it\nin `~/.vim/bundle/`\n\nThen launch Vim, run `:Helptags` and then `:help vimwiki` to verify it was\ninstalled.\n\n## Basic Markup\n\n```\n= Header1 =\n== Header2 ==\n=== Header3 ===\n\n\n*bold* -- bold text\n_italic_ -- italic text\n\n[[wiki link]] -- wiki link\n[[wiki link|description]] -- wiki link with description\n```\n\n### Lists:\n\n```\n* bullet list item 1\n    - bullet list item 2\n    - bullet list item 3\n        * bullet list item 4\n        * bullet list item 5\n* bullet list item 6\n* bullet list item 7\n    - bullet list item 8\n    - bullet list item 9\n\n1. numbered list item 1\n2. numbered list item 2\n    a) numbered list item 3\n    b) numbered list item 4\n```\n\nFor other syntax elements, see `:h vimwiki-syntax`\n\n## Key bindings\n\nNormal mode:\n\n * `<Leader>ww` -- Open default wiki index file.\n * `<Leader>wt` -- Open default wiki index file in a new tab.\n * `<Leader>ws` -- Select and open wiki index file.\n * `<Leader>wd` -- Delete wiki file you are in.\n * `<Leader>wr` -- Rename wiki file you are in.\n * `<Enter>` -- Follow/Create wiki link\n * `<Shift-Enter>` -- Split and follow/create wiki link\n * `<Ctrl-Enter>` -- Vertical split and follow/create wiki link\n * `<Backspace>` -- Go back to parent(previous) wiki link\n * `<Tab>` -- Find next wiki link\n * `<Shift-Tab>` -- Find previous wiki link\n\nFor more keys, see `:h vimwiki-mappings`\n\n## Commands\n\n * `:Vimwiki2HTML` -- Convert current wiki link to HTML\n * `:VimwikiAll2HTML` -- Convert all your wiki links to HTML\n * `:help vimwiki-commands` -- list all commands\n * `:help vimwiki` -- General vimwiki help docs\n\n## Changing Wiki Syntax\n\nVimWiki currently ships with 3 syntaxes: VimWiki (default), Markdown\n(markdown), and MediaWiki (media)\n\nIf you would prefer to use either Markdown or MediaWiki syntaxes, set the\nfollowing option in your .vimrc:\n\n```vim\n\nlet g:vimwiki_list = [{\'path\': \'~/vimwiki/\',\n                      \\ \'syntax\': \'markdown\', \'ext\': \'.md\'}]\n\n```\n\n## Getting help\n\n**Have a question?**  \nVisit the IRC channel [`#vimwiki`](https://webchat.freenode.net/?channels=#vimwiki) on Freenode ([webchat](https://webchat.freenode.net/?channels=#vimwiki), also synced to Matrix/Riot: `#vimwiki:matrix.org`) or post to the [mailing list](https://groups.google.com/forum/#!forum/vimwiki).\n\n##
