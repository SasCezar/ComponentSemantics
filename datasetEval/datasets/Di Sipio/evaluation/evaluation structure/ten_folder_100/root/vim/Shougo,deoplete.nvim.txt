b'# deoplete.nvim\n\n> Dark powered asynchronous completion framework for neovim/Vim8\n\n[![Build Status](https://travis-ci.org/Shougo/deoplete.nvim.svg?branch=master)](https://travis-ci.org/Shougo/deoplete.nvim)\n[![Join the chat at https://gitter.im/Shougo/deoplete.nvim](https://badges.gitter.im/Shougo/deoplete.nvim.svg)](https://gitter.im/Shougo/deoplete.nvim?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)\n[![Doc](https://img.shields.io/badge/doc-%3Ah%20deoplete-orange.svg)](doc/deoplete.txt)\n\nDeoplete is the abbreviation of "dark powered neo-completion".  It\nprovides an extensible and asynchronous completion framework for\nneovim/Vim8.\n\ndeoplete will display completions via `complete()` by default.\n\nHere are some [completion sources](https://github.com/Shougo/deoplete.nvim/wiki/Completion-Sources) specifically made for deoplete.nvim.\n\n<!-- vim-markdown-toc GFM -->\n\n- [Install](#install)\n  - [Requirements](#requirements)\n- [Configuration](#configuration)\n- [Screenshots](#screenshots)\n\n<!-- vim-markdown-toc -->\n\n## Install\n\n**Note:** deoplete requires Neovim (0.3.0+ and of course, **latest** is\nrecommended) or Vim8 with Python3.6.1+ and timers enabled.  See\n[requirements](#requirements) if you aren\'t sure whether you have this.\n\nNote: If you really need to use older Neovim, please use deoplete ver.4.1\ninstead.\n\nhttps://github.com/Shougo/deoplete.nvim/releases/tag/4.1\n\nFor vim-plug\n\n```viml\nif has(\'nvim\')\n  Plug \'Shougo/deoplete.nvim\', { \'do\': \':UpdateRemotePlugins\' }\nelse\n  Plug \'Shougo/deoplete.nvim\'\n  Plug \'roxma/nvim-yarp\'\n  Plug \'roxma/vim-hug-neovim-rpc\'\nendif\nlet g:deoplete#enable_at_startup = 1\n```\n\nFor dein.vim\n\n```viml\ncall dein#add(\'Shougo/deoplete.nvim\')\nif !has(\'nvim\')\n  call dein#add(\'roxma/nvim-yarp\')\n  call dein#add(\'roxma/vim-hug-neovim-rpc\')\nendif\nlet g:deoplete#enable_at_startup = 1\n```\n\nFor manual installation(not recommended)\n\n1. Extract the files and put them in your Neovim or .vim directory\n   (usually `$XDG_CONFIG_HOME/nvim/`).\n\n2. Write `call deoplete#enable()` or `let g:deoplete#enable_at_startup = 1` in\n   your `init.vim`\n\n### Requirements\n\ndeoplete requires Neovim or Vim8 with `if_python3`.\n\nIf `:echo has("python3")` returns `1`, then you have python 3 support; otherwise, see below.\n\nYou can enable Python3 interface with pip:\n\n    pip3 install --user pynvim\n\nPlease install nvim-yarp and vim-hug-neovim-rpc for Vim8.\n\n- <https://github.com/roxma/nvim-yarp>\n- <https://github.com/roxma/vim-hug-neovim-rpc>\n\n**Note: Python3 must be enabled before updating remote plugins**\n\nIf Deoplete was installed prior to Python support being added to Neovim,\n`:UpdateRemotePlugins` should be executed manually in order to enable\nauto-completion.\n\n**Note: deoplete needs pynvim ver.0.3.0+.**\n\nYou need update pynvim module.\n\n    pip3 install --user --upgrade pynvim\n\nIf you want to read the Neovim-python/python3 interface install documentation,\nyou should read `:help provider-python` and the Wiki.\n<https://github.com/deoplete-plugins/deoplete-jedi/wiki/Setting-up-Python-for-Neovim>\n\n## Configuration\n\n```vim\n" Use deoplete.\nlet g:deoplete#enable_at_startup = 1\n```\n\nSee `:help deoplete-options` for a complete list of options.\n\n## Screenshots\n\nDeoplete for JavaScript\n<https://www.youtube.com/watch?v=oanoPTpiSF4>\n\n![File Name
