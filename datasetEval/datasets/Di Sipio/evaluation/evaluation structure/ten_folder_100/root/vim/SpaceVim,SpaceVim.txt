It manages collections\nof plugins in layers, which help to collect related packages together to provide IDE-like features.\nSpaceVim is not just a vimrc but an ultimate Vim configuration, It contains many built-in features.\n\n![welcome-page](https://user-images.githubusercontent.com/13142418/68079142-904e4280-fe1f-11e9-993e-b834ea3d39ea.png)\n\nThe last release is v1.3.0, check out [following-HEAD](https://github.com/SpaceVim/SpaceVim/wiki/Following-HEAD) page for what happened since last release.\n\n## Instructions\n\n**See the followings below for more information:**\n\n- [Quick Start Guide](https://spacevim.org/quick-start-guide/)\n- [Documentation](https://spacevim.org/documentation/)\n- [Available Layers](https://spacevim.org/layers/)\n\nIf you have any questions about SpaceVim, please follow the [Getting Help Guide](https://github.com/SpaceVim/SpaceVim/wiki/Getting-help).\n\n<!-- vim-markdown-toc GFM -->\n\n- [New Features](#new-features)\n- [Project layout](#project-layout)\n- [Support SpaceVim](#support-spacevim)\n- [Credits & Thanks](#credits--thanks)\n\n<!-- vim-markdown-toc -->\n\n### New Features\n\nThis is a list of latest features implemented in SpaceVim:\n\n**Use toml as default configuration**\n\nHere is an example for using toml as SpaceVim config:\n\n```toml\n# This is basic configuration example for SpaceVim.\n\n# All SpaceVim options below [options] snippet.\n[options]\n    # Set SpaceVim theme. By default colorscheme layer is not loaded.\n    # If you want to use more colorschemes, please load the colorscheme\n    # layer.\n    colorscheme = "gruvbox"\n    colorscheme_bg = "dark"\n    # Disable guicolors in basic mode, many terminal do not support 24bit\n    # true colors\n    enable_guicolors = false\n    # Disable statusline separator, if you want to use other value, please\n    # install nerd fonts\n    statusline_separator = "nil"\n    statusline_inactive_separator = "bar"\n    buffer_index_type = 4\n    windows_index_type = 3\n    enable_tabline_ft_icon = false\n    enable_statusline_mode = false\n    statusline_unicode_symbols = false\n    # Enable Vim compatible mode, avoid changing origin Vim key bindings\n    vimcompatible = true\n\n# Enable autocomplete layer\n[[layers]]\n    name = \'autocomplete\'\n    auto-completion-return-key-behavior = "complete"\n    auto-completion-tab-key-behavior = "cycle"\n\n[[layers]]\n    name = \'shell\'\n    default_position = \'top\'\n    default_height = 30\n```\n\n**Iedit mode**\n\nSpaceVim uses a powerful iedit mode to quick edit multiple occurrences of a symbol or selection. Two new modes:`iedit-Normal`/`iedit-Insert`.\n\nThe default color for iedit is `red`/`green` which is based on the current colorscheme.\n\n![iedit mode](https://user-images.githubusercontent.com/13142418/44941560-be2a9800-add2-11e8-8fa5-e6118ff9ddcb.gif)\n\n**Highlight cursor symbol**\n\nSpaceVim supports highlighting of the current symbol on demand and adds\na transient state to easily navigate and rename this symbol.\n\n![highlight cursor symbol](https://user-images.githubusercontent.com/13142418/36210381-e6dffde6-1163-11e8-9b35-0bf262e6f22b.gif)\n\n[**Fly Grep in Vim**](https://spacevim.org/grep-on-the-fly-in-spacevim/)\n\nWith this feature, Vim will display the searching result as you type. Of course, it is running\nasynchronously. Before using this feature, you need to install a searching tool. FlyGrep works\nthrough search tools: `ag`, `rg`, `ack`, `pt` and `grep`, Choose one you like.\n\n![searching project](https://user-images.githubusercontent.com/13142418/35278709-7856ed62-0010-11e8-8b1e-e6cc6374b0dc.gif)\n\n[**Mnemonic key bindings navigation**](https://spacevim.org/mnemonic-key-bindings-navigation/)\n\nYou don\'t need to remember any key bindings, as the mapping guide will show up after the <kbd>SPC</kbd> is pressed.\nThe mapping guide is also available for `g`, `z`, and `s`.\n\n![mapping guide](https://user-images.githubusercontent.com/13142418/35568184-9a318082-058d-11e8-9d88-e0eafd1d498d.gif)\n\n[**Help description for key bindings**](https://spacevim.org/help-description-for-key-bindings/)\n\nUse <kbd>SPC h d k</kbd> to get the help description of a key binding, and `gd` to find definition of key bindings.\n\n![describe key bindings](https://user-images.githubusercontent.com/13142418/35568829-e3c8e74c-058f-11e8-8fa8-c0e046d8add3.gif)\n\n[**Asynchronous plugin manager**](https://spacevim.org/Asynchronous-plugin-manager/)\n\nCreate an UI for [dein.vim](https://github.com/Shougo/dein.vim/) - the best asynchronous vim plugin manager\n\n![UI for dein](https://user-images.githubusercontent.com/13142418/34907332-903ae968-f842-11e7-8ac9-07fcc9940a53.gif)\n\nFor more features, please read [SpaceVim\'s Blog](https://spacevim.org/blog/)\n\n###
