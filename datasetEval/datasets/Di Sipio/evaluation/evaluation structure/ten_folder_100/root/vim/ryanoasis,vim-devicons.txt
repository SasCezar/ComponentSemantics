[Screenshots](https://github.com/ryanoasis/vim-devicons/wiki/screenshots) for more.\n\nFeatures\n--------\n\n- Adds filetype glyphs (icons) to various vim plugins.\n- Customizable and extendable glyphs settings.\n- Supports a wide range of file type extensions.\n- Supports popular full filenames, like `.gitignore`, `node_modules`, `.vimrc`, and many more.\n- Supports byte order marker (BOM).\n- Works with patched fonts, especially [Nerd Fonts](https://github.com/ryanoasis/nerd-fonts).\n\n> See [Detailed Features](https://github.com/ryanoasis/vim-devicons/wiki/Detailed-Features) for more.\n\n> See [Configuration](https://github.com/ryanoasis/vim-devicons/wiki/Extra-Configuration) for a list of configuration and customization options.\n\nInstallation\n------------\n\n1. Install a [Nerd Font compatible font](https://github.com/ryanoasis/nerd-fonts#font-installation) or [patch your own](https://github.com/ryanoasis/nerd-fonts#font-patcher). Then set your terminal font (or `guifont` if you are using GUI version of Vim).\n1. Install the Vim plugin with your favorite plugin manager, e.g. [vim-plug](https://github.com/junegunn/vim-plug):\n\n    ```vim\n    Plug \'ryanoasis/vim-devicons\'\n    ```\n\n    > Always load the vim-devicons as the very last one.\n\n1. Configure Vim\n\n    ```vim\n    set encoding=UTF-8\n    ```\n\n\t> No need to set explicitly under Neovim: always uses UTF-8 as the default encoding.\n\n\n> See [Installation](https://github.com/ryanoasis/vim-devicons/wiki/Installation) for detailed setup instructions\n\nUse `:help devicons` for further configuration.\n\nDevelopers\n----------\n\nSee [DEVELOPER](DEVELOPER.md) for how to use the API.\n\nTroubleshooting\n---------------\n\nSee [FAQ](https://github.com/ryanoasis/vim-devicons/wiki/FAQ-&-Troubleshooting).\n\nContributing\n------------\n\n### [Code of Conduct](CODE_OF_CONDUCT.md)\n\nThis project has adopted a Code of Conduct that we expect project participants to adhere to.
