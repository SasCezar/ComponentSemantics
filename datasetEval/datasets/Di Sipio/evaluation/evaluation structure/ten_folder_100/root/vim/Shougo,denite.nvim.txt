b'denite.nvim\n===========\n\n[![Build Status](https://travis-ci.org/Shougo/denite.nvim.svg?branch=master)](https://travis-ci.org/Shougo/denite.nvim)\n\nNote: Denite.nvim does not define any of default mappings.  You need to define\nthem.\n\n\n## About\n\n[![Join the chat at https://gitter.im/Shougo/denite.nvim](https://badges.gitter.im/Shougo/denite.nvim.svg)](https://gitter.im/Shougo/denite.nvim?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)\n\nDenite is a dark powered plugin for Neovim/Vim to unite all interfaces.\nIt can replace many features or plugins with its interface.\nIt is like a fuzzy finder, but is more generic.\nYou can extend the interface and create the sources.\n\nSome things you can do with it include:\n\n* Opening files\n\n* Switching buffers\n\n* Inserting the value of a register\n\n* Changing current directory\n\n* Searching for a string\n\n[Unite.vim](https://github.com/Shougo/unite.vim) was meant to be like [Helm](https://github.com/emacs-helm/helm) for Vim.\nBut the implementation is ugly and it\'s very slow.\n\nDenite resolves Unite\'s problems. Here are some of its benefits:\n\n* Theoretically faster because the main process is executed by Python\n\n* Theoretically more stable because no other processes can be performed when\nit runs.\n\n* The implementation is simpler than unite\n\n* Has greater potential to implement new features\n\n* Python3 is easier to work with than Vimscript\n\n* There are a lot of useful tools to keep code simple (linter, tester, etc...)\nin Python3.\n\n* Unite is officially obsolete, minor bugs (or even major bugs) are\nnot fixed anymore\n\n\n## Requirements\n\nDenite requires Neovim 0.3.0+ or Vim 8.0+ with `if_python3`.\nIf `:echo has("python3")` returns `1`, then you\'re done.\n\nNote: You need to install Python3.6.1+.\n\nFor neovim:\n\nYou must install "pynvim" module with pip\n\n    pip3 install --user pynvim\n\nIf you want to read the pynvim/python3 interface install documentation,\nyou should read `:help provider-python`.\n\nFor Vim8:\n\nPlease install nvim-yarp plugin for Vim8.\nhttps://github.com/roxma/nvim-yarp\n\nPlease install vim-hug-neovim-rpc plugin for Vim8.\nhttps://github.com/roxma/vim-hug-neovim-rpc\n\nYou must install "pynvim" module with pip\n\n    pip3 install --user pynvim\n\n\n### For Windows users\n\n1. Install Vim from [Vim Win32 Installer\n   releases](https://github.com/vim/vim-win32-installer/releases)\n2. Download [Python latest embeddable zip\n   file](https://www.python.org/downloads/windows/) and copy the all files in\n   the zip file to the folder where you installed Vim.\n\n**Note:** You need to do 1. and 2. with the common-arch files (x86 or x64).\n\n## Installation\n\nFor [dein.vim](https://github.com/Shougo/dein.vim)\n\n```\ncall dein#add(\'Shougo/denite.nvim\')\nif !has(\'nvim\')\n  call dein#add(\'roxma/nvim-yarp\')\n  call dein#add(\'roxma/vim-hug-neovim-rpc\')\nendif\n```\n\nFor [vim-plug](https://github.com/junegunn/vim-plug)\n\n```\nif has(\'nvim\')\n  Plug \'Shougo/denite.nvim\', { \'do\': \':UpdateRemotePlugins\' }\nelse\n  Plug \'Shougo/denite.nvim\'\n  Plug \'roxma/nvim-yarp\'\n  Plug \'roxma/vim-hug-neovim-rpc\'\nendif\n```\n\n## Examples\n\n```vim\n" Define mappings\nautocmd FileType denite call s:denite_my_settings()\nfunction! s:denite_my_settings() abort\n  nnoremap <silent><buffer><expr> <CR>\n  \\ denite#do_map(\'do_action\')\n  nnoremap <silent><buffer><expr> d\n  \\ denite#do_map(\'do_action\', \'delete\')\n  nnoremap <silent><buffer><expr> p\n  \\ denite#do_map(\'do_action\', \'preview\')\n  nnoremap <silent><buffer><expr> q\n  \\ denite#do_map(\'quit\')\n  nnoremap <silent><buffer><expr> i\n  \\ denite#do_map(\'open_filter_buffer\')\n  nnoremap <silent><buffer><expr> <Space>\n  \\ denite#do_map(\'toggle_select\').\'j\'\nendfunction\n```\n\n\n## Screenshots\n\n### Old UI\n\n![denite old UI](https://user-images.githubusercontent.com/13142418/65154984-06107180-da5f-11e9-8cbf-e0a544d0dbc5.jpg)\n\n### New UI\n\n![denite new UI](https://user-images.githubusercontent.com/13142418/65154937-f002b100-da5e-11e9-8aef-723233e3704d.jpg)\n\n### Others\n\n![denite new UI2](https://user-images.githubusercontent.com/1239245/58742567-a155ea80-8460-11e9-9925-09082def2c80.gif)\n![denite new UI3](https://user-images.githubusercontent.com/41671631/58790351-cf832800-8622-11e9-912d-813408876b86.gif)\n'
