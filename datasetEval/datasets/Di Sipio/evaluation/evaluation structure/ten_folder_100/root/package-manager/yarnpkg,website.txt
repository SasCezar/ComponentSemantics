b'Yarn Website\n[![Netlify Status](https://api.netlify.com/api/v1/badges/85057564-01fa-49d4-b898-30acb74ae19e/deploy-status)](https://app.netlify.com/sites/yarnpkg/deploys)\n============\n\nThis repo contains the source code for the Yarn website.\n\nIn order to get started:\n\n```sh\n$ git clone git@github.com:yarnpkg/website.git yarn-website\n$ cd yarn-website\n```\n\nYou should ensure you have Yarn and [Bundler](http://bundler.io/) installed:\n\n```sh\n$ gem install bundler\n```\n\nAnd then:\n\n```sh\n$ make\n```\n\nOr:\n\n```sh\n$ make install\n$ make serve\n```\n\nOn Windows, `make` is not available, so you need to execute `bundle` and `jekyll` directly:\n\n```sh\nbundle install\nbundle exec jekyll serve --incremental\n```\n\n---\n\n**Special thanks to [Netlify](https://www.netlify.com/) for powering the website.**\n'
