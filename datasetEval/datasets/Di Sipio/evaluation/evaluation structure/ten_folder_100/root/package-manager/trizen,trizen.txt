b"trizen\n======\n\ntrizen is a lightweight wrapper for AUR, written in Perl.\n\nMain features include:\n\n* Installation of packages from the AUR\n* Search support for AUR packages\n* Reading AUR comments for packages\n* Upgrade support for AUR packages\n* Recursive resolver of AUR dependencies\n* Built-in interaction with 'pacman'\n* Edit support for text files\n* Input/output UTF-8 support\n\n# INSTALLATION\n\n* Tagged version:\n```bash\ngit clone https://aur.archlinux.org/trizen.git\ncd trizen\nmakepkg -si\n```\n\n* `-git` version:\n\n```bash\ngit clone https://aur.archlinux.org/trizen-git.git\ncd trizen-git\nmakepkg -si\n```\n\n# Screenshots\n\n#### Search results\n\n![trizen -Ss youtube](https://user-images.githubusercontent.com/614513/32417050-f5e5a310-c25b-11e7-8598-056431ce9a1d.png)\n\n#### Package info\n\n![trizen -Si sidef](https://user-images.githubusercontent.com/614513/32417040-d137a68a-c25b-11e7-89f3-362b084b8873.png)\n\n#### Options\n\n![trizen -h](https://user-images.githubusercontent.com/614513/34314381-d7e4df12-e77b-11e7-84f3-322444c17824.png)\n\n![trizen -Sh](https://user-images.githubusercontent.com/614513/34314382-d98a810a-e77b-11e7-9df0-64af9f5f3633.png)\n\n# Usage\n\n```\nusage: trizen [options] [pkgname] [pkgname] [...]\n\nMain options:\n\n    -S, --sync         : install packages (see: trizen -Sh)\n    -C, --comments     : display AUR comments for a package\n    -G, --get          : clones a package in the current directory\n    -R, --remove       : remove packages from the system (see: pacman -Rh)\n    -Q, --query        : query the package database (see: pacman -Qh)\n    -F, --files        : query the files database (see: pacman -Fh)\n    -D, --database     : operate on the package database (see: pacman -Dh)\n    -T, --deptest      : check dependencies (see: pacman -Th)\n    -U, --upgrade      : install built packages from '--clone-dir' or `pwd`\n\nOther options:\n\n    -q, --quiet        : do not display any warnings\n    -r, --regular      : use only the regular repositories\n        --stats        : show stats about the installed packages\n        --nocolors     : disable text colors\n        --forcecolors  : force colors when not writing to STDOUT\n        --debug        : activate the debug/verbose mode\n        --help         : print this message and exit\n        --version      : print version and exit\n\nSee also:\n\n    trizen -Sh\n    trizen -Gh\n\n:: Each configuration key is a valid option when preceded with '--'\n```\n\n# Sync options (`-S`)\n\n```\nusage: trizen {-S --sync} [options] [package(s)]\n\nMain options:\n\n  -s, --search        : search for packages\n  -i, --info          : show info for packages\n  -m, --maintainer    : show packages maintained by <username>\n  -p, --pkgbuild      : show PKGBUILD only\n  -l, --local         : build and install packages from `pwd`\n  -u, --sysupgrade    : upgrade installed packages\n  -y, --refresh       : refresh package databases (with: -u)\n  -c, --clean         : clean the cache directory of `trizen` and `pacman`\n  -a, --aur           : only AUR operations (with: -c, -u, -s, -i)\n\nOther options:\n\n      --devel         : update VCS packages during -Su\n      --show-ood      : show out-of-date flagged packages during -Su\n      --noinfo        : do not display package info after cloning\n      --nopull        : do not `git pull` new changes\n      --noedit        : do not prompt to edit files\n      --nobuild       : do not build packages (implies --noedit)\n      --noinstall     : do not install packages after building\n      --needed        : do not reinstall up-to-date packages\n      --asdeps        : install packages as non-explicitly installed\n      --asexplicit    : install packages as explicitly installed\n      --skipinteg     : pass the `--skipinteg` argument to `makepkg`\n      --noconfirm     : do not ask for any confirmation\n      --movepkg       : move built packages into pacman's cache directory\n      --movepkg-dir=s : move built packages in this directory (implies --movepkg)\n      --clone-dir=s   : directory where to clone and build packages\n      --editor=s      : editor command used to edit build files\n      --pager-mode    : display the build files of a package in pager mode\n      --pager=s       : pager command used to display the build files\n      --ignore=s      : space-separated list of packages to ignore during -Su\n\nExamples:\n\n    trizen -S  <package>     # install <package>\n    trizen -Ss <keyword>     # search for <keyword>\n    trizen -Si <package>     # show info about <package>\n```\n\n# Get options (`-G`)\n\n```\nusage: trizen {-G --get} [options] [package(s)]\n\nMain options:\n\n    -d, --with-deps     : clones a package with all needed AUR dependencies\n\nExamples:\n\n    trizen -G  <package>     # clones <package>\n    trizen -Gd <package>     # clones <package> along with its AUR dependencies\n```\n\nA configuration file is automatically generated at: `~/.config/trizen/trizen.conf`\n"
