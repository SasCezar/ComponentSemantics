follow the instructions on the page for your specific operating system.\n\nIf you are running macOS and use [Homebrew](https://brew.sh), you can use our official [Homebrew tap](https://github.com/habitat-sh/homebrew-habitat).\n```\n$ brew tap habitat-sh/habitat\n$ brew install hab\n```\n\nIf you are running Windows and use [Chocolatey](https://chocolatey.org), you can install our [chocolatey package](https://chocolatey.org/packages/habitat)\n```\nC:\\> choco install habitat\n```\n\nIf you do _not_ run Homebrew or Chocolatey, or if you use Linux, you can use the Habitat [install.sh](https://github.com/habitat-sh/habitat/blob/master/components/hab/install.sh) or [install.ps1](https://github.com/habitat-sh/habitat/blob/master/components/hab/install.ps1) script.\n\nBash:\n```\n$ curl https://raw.githubusercontent.com/habitat-sh/habitat/master/components/hab/install.sh | sudo bash\n```\n\nPowershell:\n```\nC:\\> Set-ExecutionPolicy Bypass -Scope Process -Force\nC:\\> iex ((New-Object System.Net.WebClient).DownloadString(\'https://raw.githubusercontent.com/habitat-sh/habitat/master/components/hab/install.ps1\'))\n```\n\n## Contribute\n\nWe are
