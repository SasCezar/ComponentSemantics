b'## [C++ Archive Network](https://cppan.org/)\n\n[![Build Status](https://travis-ci.org/cppan/cppan.svg?branch=v1)](https://travis-ci.org/cppan/cppan)\n[![Build status](https://ci.appveyor.com/api/projects/status/iacnrt6byhy8ox5v/branch/v1?svg=true)](https://ci.appveyor.com/project/egorpugin/cppan/branch/v1)\n\n### Resources\n\n- Homepage: https://cppan.org/\n- Docs: https://github.com/cppan/cppan/blob/v1/doc/\n- Download: https://cppan.org/client/\n- Issue tracking: https://github.com/cppan/cppan/issues\n- Forum: https://groups.google.com/forum/#!forum/cppan\n\n\n### Quick Start\n\n1. Download the latest client application, unpack and put it to PATH.\n1. (!) Run once `cppan` from any directory without any arguments to perform initial configuration.\n1. In your `CMakeLists.txt` write:\n```\nfind_package(CPPAN REQUIRED)\ncppan_add_package(\n    pvt.cppan.demo.sqlite3\n    pvt.cppan.demo.fmt-4\n    pvt.cppan.demo.madler.zlib-*\n    pvt.cppan.demo.boost.asio-1.66\n    ...\n    libs you want to add\n)\ncppan_execute()\n\n# near your target\nadd_executable(myexe ...)\ntarget_link_libraries(myexe\n  pvt.cppan.demo.sqlite3\n  pvt.cppan.demo.madler.zlib\n  pvt.cppan.demo.boost.asio\n  ...\n)\n```\n4. Perform other usual CMake steps.\n\n\n### Dependencies\n\n- `CMake >= 3.2`\n\n### Download & Install\n\n#### Linux\n\n##### Ubuntu 16.04\n\n```\nsudo apt install cmake\nwget https://cppan.org/client/cppan-master-Linux-client.deb \nsudo dpkg -i cppan-master-Linux-client.deb\n```\n\n##### Ubuntu 14.04\n\n```\nwget https://www.cmake.org/files/v3.6/cmake-3.6.1-Linux-x86_64.sh\nsudo sh cmake-3.6.1-Linux-x86_64.sh --skip-license --prefix=/usr\nsudo add-apt-repository ppa:ubuntu-toolchain-r/test \nsudo apt-get update && sudo apt-get install gcc-5\nwget https://cppan.org/client/cppan-master-Linux-client.deb \nsudo dpkg -i cppan-master-Linux-client.deb\n```\n\n#### Windows\n\n[Download client](https://cppan.org/client/cppan-master-Windows-client.zip), unzip it and put under PATH.\n\n#### macOS\n\n```\nwget https://cppan.org/client/cppan-master-macOS-client.zip \nunzip cppan-master-macOS-client.zip \nsudo cp cppan /usr/local/bin/\n```\n\n### Build\n\n```\ngit clone https://github.com/cppan/cppan cppan_client\ncd cppan_client\ncppan\nmkdir build && cd build\ncmake ..\n# for linux\n# cmake .. -DCMAKE_C_COMPILER=gcc-5 -DCMAKE_CXX_COMPILER=g++-5\nmake -j4\n```\n\n### Support CPPAN\n\nMore info about supporting C++ Archive Network can be found [here](https://github.com/cppan/cppan/blob/master/doc/support.md).\n'
