b'Paket.VisualStudio\n==================\n\n[![Join the chat at https://gitter.im/fsprojects/Paket.VisualStudio](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/fsprojects/Paket.VisualStudio?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)\n\nManage your Paket (http://fsprojects.github.io/Paket/) dependencies from Visual Studio!\n\n![](https://raw.githubusercontent.com/fsprojects/Paket.VisualStudio/master/docs/files/img/paket.visualstudio.png)\n\nFeatures\n--------\n\n* Solution Explorer Extensions for Paket files\n* Paket commands in Solution Explorer menu\n* IntelliSense in paket.dependencies and paket.references\n\n\n### Build [![Build status](https://ci.appveyor.com/api/projects/status/do913wfhcmnit5ps/branch/master?svg=true)](https://ci.appveyor.com/project/SteffenForkmann/paket-visualstudio/branch/master)\n\nRequire Visual Studio 2013 and Visual Studio 2013 SDK (or later). \nRun the `build.cmd` script or `Paket.VisualStudio.sln` to build the solution.\n\nInstallation\n------------\n\nPaket.VisualStudio is available in the [Visual Studio Gallery](https://visualstudiogallery.msdn.microsoft.com/ce104917-e8b3-4365-9490-8432c6e75c36).\n\nYou can also download the lastest (unreleased) installer from [AppVeyor](https://ci.appveyor.com/project/SteffenForkmann/paket-visualstudio/history):\nChoose the latest build and navigate to the "artifacts" tab.\n\n### Getting involved\n                                      \nContributions are always welcome.\n\n### License\nThe extension is available under MIT license.
