etc.\n\xe2\x94\x82   \xe2\x94\x9c\xe2\x94\x80\xe2\x94\x80 server/                 # Node.js app\n\xe2\x94\x82   \xe2\x94\x82   \xe2\x94\x9c\xe2\x94\x80\xe2\x94\x80 ssr.js              #   - Server-side rendering, e.g. ReactDOMServer.renderToString(<App />)\n\xe2\x94\x82   \xe2\x94\x82   \xe2\x94\x9c\xe2\x94\x80\xe2\x94\x80 api.js              #   - GraphQL API endpoint\n\xe2\x94\x82   \xe2\x94\x82   \xe2\x94\x94\xe2\x94\x80\xe2\x94\x80 index.js            #   - Node.js app entry point\n\xe2\x94\x82   \xe2\x94\x94\xe2\x94\x80\xe2\x94\x80 index.js                # Client-side app entry point, e.g. ReactDOM.hydrate(<App />, container)\n\xe2\x94\x94\xe2\x94\x80\xe2\x94\x80 package.json                # List of project dependencies and NPM scripts\n```\n\n#### `package.json`\n\n```diff\n{\n  "main": "build/server.js",\n  "engines": {\n    "node": ">=8.10"\n  },\n  "dependencies": {\n+   "express": "^4.6.14",\n    "react": "^16.8.4",\n    "react-dom": "^16.8.4"\n  },\n  "devDependencies": {\n-   "react-scripts": "^1.1.1"\n+   "react-app-tools": "^3.1.0-preview.7"\n  },\n  "scripts": {\n-   "start": "react-scripts start",\n+   "start": "react-app start",\n-   "build": "react-scripts build",\n+   "build": "react-app build",\n-   "test": "react-scripts test"\n+   "test": "react-app test"\n  }\n}\n```\n\n#### `src/index.js` - Client-side rendering\n\n```js\nimport React from \'react\';\nimport ReactDOM from \'react-dom\';\nimport App from \'./components/App\';\n\nReactDOM.hydrate(<App />, document.getElementById(\'root\'));\n```\n\n#### `src/server/index.js` - Server-side rendering and/or API endpoint\n\n```js\nconst path = require(\'path\');\nconst express = require(\'express\');\nconst React = require(\'react\');\nconst ReactDOMServer = require(\'react-dom/server\');\nconst App = require(\'../components/App\');\nconst stats = require(\'./stats.json\');\n\nconst app = express();\n\napp.get(\'*\', (req, res) => {\n  res.send(`\n    <html>\n      <body>\n        <div id="root">${ReactDOMServer.renderToString(<App />)}</div>\n        ${stats.entrypoints.main.assets.map(\n          src => `<script src="${src}"></script>`\n        )}\n      </body>\n    </html>\n  `);\n});\n\nif (process.env.NODE_ENV === \'production\') {\n  app.listen(process.env.PORT || 8080);\n} else {\n  module.exports.default = app;\n}\n```\n\nYou can launch the app in development mode by running:\n\n```sh\n$ npm install\n$ npm start\n```\n\nThen open [http://localhost:3000/](http://localhost:3000/) to see your app.<br>\nWhen you\xe2\x80\x99re ready to deploy to production, create a minified bundle with `npm run build`.\n\n<p align="center"><img src=\'https://camo.githubusercontent.com/506a5a0a33aebed2bf0d24d3999af7f582b31808/687474703a2f2f692e696d6775722e636f6d2f616d794e66434e2e706e67\' width=\'600\' alt=\'npm start\'></p>\n\nFor more information refer to Create React App documentation:\n\n- [Getting Started](https://github.com/facebookincubator/create-react-app#getting-started) \xe2\x80\x93 How to create a new app.\n- [User Guide](https://github.com/facebookincubator/create-react-app/blob/master/packages/react-scripts/template/README.md) \xe2\x80\x93 How to develop apps bootstrapped with Create React App.\n\nJoin our Telegram chat for support and feature requests -
