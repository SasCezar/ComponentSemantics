b'# `babel-plugin-styled-components`\n\nThis plugin is a highly recommended supplement to the base styled-components library, offering some useful features:\n\n- consistently hashed component classNames between environments (a must for server-side rendering)\n- better debugging through automatic annotation of your styled components based on their context in the file system, etc.\n- various types of minification for styles and the tagged template literals styled-components uses\n\n## Quick start\n\nInstall the plugin first:\n\n```\nnpm install --save-dev babel-plugin-styled-components\n```\n\nThen add it to your babel configuration:\n\n```JSON\n{\n  "plugins": ["babel-plugin-styled-components"]\n}\n```\n\n## Documentation\n\n**The documentation for this plugin lives on [the styled-components website](https://www.styled-components.com/docs/tooling#babel-plugin)!**\n\n- [Usage](https://www.styled-components.com/docs/tooling#usage)\n- [Better debugging](https://www.styled-components.com/docs/tooling#better-debugging)\n- [Minification](https://www.styled-components.com/docs/tooling#minification)\n\n## License\n\nLicensed under the MIT License, Copyright \xc2\xa9 2016-present Vladimir Danchenkov and Maximilian Stoiber.\n\nSee [LICENSE.md](./LICENSE.md) for more information.\n'
