you can set such parameters as:\n \n - models:  \n     allows you to set `NTB` models, where you set icon and color. Can be set up only via code.\n    \n - behavior:  \n     allows you to set bottom translation behavior.\n    \n - view pager:  \n     allows you to connect `NTB` with `ViewPager`. If you want your can also set `OnPageChangeListener`.\n         \n - background color:  \n    allows you to set background to `NTB` which automatically set with offset relative to badge gravity and corners radius.\n\n - model selected icon:  \n     allows you to set selected icon when current model is active.\n\n - model title:  \n     allows you to enable title in you model.\n    \n - model badge:  \n     allows you to enable badge in you model.\n     \n - use custom typeface on badge:  \n     allows you to handle set of custom typeface in your badge.\n    \n - title mode:  \n     allows you to handle mode of the model title show. Can show all or only active.\n     \n - title size:  \n     allows you to set titles size. \n     \n - scale mode:  \n     allows you to handle mode of the model icon and title scale.\n     \n - tint mode:  \n     allows you to enable or disable icon tinting.\n      \n - badge size:  \n     allows you to set badges size.\n     \n - badge position:  \n     allows you to set the badge position in you model. Can be: left(25%), center(50%) and right(75%).\n\n - badge gravity:  \n     allows you to set the badge gravity in NTB. Can be top or bottom.\n     \n - badge colors:  \n     allows you to set the badge bg and title colors.\n    \n - typeface:  \n     allows you to set custom typeface to your title.\n \n - corners radius:  \n     allows you to set corners radius of pointer.\n\n - icon size fraction:  \n     allows you to set icon size fraction relative to smaller model side.\n\n - animation duration:  \n     allows you to set animation duration.\n      \n - inactive color:  \n     allows you to set inactive icon color.\n     \n - active color:  \n     allows you to set active icon color.\n     \n - tab bar listener:  \n     allows you to set listener which triggering on start or on end when you set model index.\n        \n - preview colors:  \n     allows you to set preview colors, which generate count of models equals to count of colors.\n\n<b>Tips</b>\n\nCreation of models occurs through `Builder` pattern.  \n`ModelBuilder` requires two fields: icon and color. Title, badge title and selected icon is optional.\n\nYou can set selected icon. Resize and scale of selected icon equals to original icon.  \nOrientation automatically detected according to `View` size.\n\nBy default badge bg color is the active model color and badge title color is the model bg color. To reset colors just set `AUTO_COLOR` value to badge bg and title color.  \nBy default badge sizes and title sizes is auto fit. To reset calculation just set `AUTO_SIZE` value to badge size and title size.  \nBy default icon size fraction is `0.5F` (half of smaller side of `NTB` model). To reset scale fraction of icon to automatic just put in method `AUTO_SCALE` value.\n\nIf your set `ViewPager` and enable swipe you can action down on active pointer and do like drag.\n\n<b>Init</b>\n\nCheck out in code init:\n\n```java\nfinal NavigationTabBar navigationTabBar = (NavigationTabBar) findViewById(R.id.ntb);\nfinal ArrayList<NavigationTabBar.Model> models = new ArrayList<>();\nmodels.add(\n        new NavigationTabBar.Model.Builder(\n                getResources().getDrawable(R.drawable.ic_first),\n                Color.parseColor(colors[0])\n        ).title("Heart")\n                .badgeTitle("NTB")\n                .build()\n);\nmodels.add(\n        new NavigationTabBar.Model.Builder(\n                getResources().getDrawable(R.drawable.ic_second),\n                Color.parseColor(colors[1])\n        ).title("Cup")\n                .badgeTitle("with")\n                .build()\n);\nmodels.add(\n        new NavigationTabBar.Model.Builder(\n                getResources().getDrawable(R.drawable.ic_third),\n                Color.parseColor(colors[2])\n        ).title("Diploma")\n                .badgeTitle("state")\n                .build()\n);\nmodels.add(\n        new NavigationTabBar.Model.Builder(\n                getResources().getDrawable(R.drawable.ic_fourth),\n                Color.parseColor(colors[3])\n        ).title("Flag")\n                .badgeTitle("icon")\n                .build()\n);\nmodels.add(\n        new NavigationTabBar.Model.Builder(\n                getResources().getDrawable(R.drawable.ic_fifth),\n                Color.parseColor(colors[4])\n        ).title("Medal")\n                .badgeTitle("777")\n                .build()\n);\nnavigationTabBar.setModels(models);\nnavigationTabBar.setViewPager(viewPager, 2);\n\nnavigationTabBar.setTitleMode(NavigationTabBar.TitleMode.ACTIVE);\nnavigationTabBar.setBadgeGravity(NavigationTabBar.BadgeGravity.BOTTOM);\nnavigationTabBar.setBadgePosition(NavigationTabBar.BadgePosition.CENTER);\nnavigationTabBar.setTypeface("fonts/custom_font.ttf");\nnavigationTabBar.setIsBadged(true);\nnavigationTabBar.setIsTitled(true);\nnavigationTabBar.setIsTinted(true);\nnavigationTabBar.setIsBadgeUseTypeface(true);\nnavigationTabBar.setBadgeBgColor(Color.RED);\nnavigationTabBar.setBadgeTitleColor(Color.WHITE);\nnavigationTabBar.setIsSwiped(true);\nnavigationTabBar.setBgColor(Color.BLACK);\nnavigationTabBar.setBadgeSize(10);\nnavigationTabBar.setTitleSize(10);\nnavigationTabBar.setIconSizeFraction(0.5);\n```\n\nIf your models is in badge mode you can set title, hide, show, toggle and update badge title like this:\n```java\nmodel.setTitle("Here some title to model");\nmodel.hideBadge();\nmodel.showBadge();\nmodel.toggleBadge();\nmodel.updateBadgeTitle("Here some title like NEW or some integer value");\n```\n          \nTo enable behavior translation inside `CoordinatorLayout` when at bottom of screen:\n```java\nnavigationTabBar.setBehaviorEnabled(true);\n```\n\nTo deselect active index and reset pointer:\n```java\nnavigationTabBar.deselect();\n```\n\nOther methods check out in sample.\n\nAnd XML init:\n\n```xml\n<devlight.io.library.ntb.NavigationTabBar\n   android:id="@+id/ntb"\n
