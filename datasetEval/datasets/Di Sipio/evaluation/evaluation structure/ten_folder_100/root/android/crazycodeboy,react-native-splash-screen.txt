Works on iOS and Android.\n\n## Content\n\n- [Changes](#changes)\n- [Installation](#installation)\n- [Examples](#examples)\n- [Getting started](#getting-started)\n- [API](#api)\n- [Testing](#testing)\n- [Troubleshooting](#troubleshooting)\n- [Contribution](#contribution)\n\n\n## Changes\nFor React Native >= 0.47.0 use [v3.+](https://github.com/crazycodeboy/react-native-splash-screen/releases), for React Native < 0.47.0 use [v2.1.0](https://github.com/crazycodeboy/react-native-splash-screen/releases/tag/v1.0.9)\n\n## Examples  \n* [Examples](https://github.com/crazycodeboy/react-native-splash-screen/tree/master/examples)\n\n![react-native-splash-screen-Android](https://raw.githubusercontent.com/crazycodeboy/react-native-splash-screen/v3.0.0/examples/Screenshots/react-native-splash-screen-Android.gif)\n![react-native-splash-screen-iOS](https://raw.githubusercontent.com/crazycodeboy/react-native-splash-screen/v3.0.0/examples/Screenshots/react-native-splash-screen-iOS.gif)\n\n\n\n## Installation\n\n### First step(Download):\nRun `npm i react-native-splash-screen --save`\n\n### Second step(Plugin Installation):\n\n#### Automatic installation\n\n`react-native link react-native-splash-screen` or `rnpm link react-native-splash-screen`\n\n#### Manual installation  \n\n**Android:**\n\n1. In your `android/settings.gradle` file, make the following additions:\n```java\ninclude \':react-native-splash-screen\'   \nproject(\':react-native-splash-screen\').projectDir = new File(rootProject.projectDir, \'../node_modules/react-native-splash-screen/android\')\n```\n\n2. In your android/app/build.gradle file, add the `:react-native-splash-screen` project as a compile-time dependency:\n\n```java\n...\ndependencies {\n    ...\n    implementation project(\':react-native-splash-screen\')\n}\n```\n\n3. Update the MainApplication.java file to use `react-native-splash-screen` via the following changes:   \n\n```java\n// react-native-splash-screen >= 0.3.1\nimport org.devio.rn.splashscreen.SplashScreenReactPackage;\n// react-native-splash-screen < 0.3.1\nimport com.cboy.rn.splashscreen.SplashScreenReactPackage;\n\npublic class MainApplication extends Application implements ReactApplication {\n\n    private final ReactNativeHost mReactNativeHost = new ReactNativeHost(this) {\n        @Override\n        public boolean getUseDeveloperSupport() {\n            return BuildConfig.DEBUG;\n        }\n\n        @Override\n        protected List<ReactPackage> getPackages() {\n            return Arrays.<ReactPackage>asList(\n                    new MainReactPackage(),\n            new SplashScreenReactPackage()  //here\n            );\n        }\n    };\n\n    @Override\n    public ReactNativeHost getReactNativeHost() {\n        return mReactNativeHost;\n    }\n}\n```\n\n**iOS:**\n\n1. In XCode, in the project navigator, right click `Libraries` \xe2\x9e\x9c `Add Files to [your project\'s name]`\n2. Go to `node_modules` \xe2\x9e\x9c `react-native-splash-screen` and add `SplashScreen.xcodeproj`\n3. In XCode, in the project navigator, select your project. Add `libSplashScreen.a` to your project\'s `Build Phases` \xe2\x9e\x9c `Link Binary With Libraries`\n4. To fix `\'RNSplashScreen.h\' file not found`, you have to select your project \xe2\x86\x92 Build Settings \xe2\x86\x92 Search Paths \xe2\x86\x92 Header Search Paths to add:\n\n   `$(SRCROOT)/../node_modules/react-native-splash-screen/ios`\n\n\n\n### Third step(Plugin Configuration):\n\n**Android:**\n\nUpdate the `MainActivity.java` to use `react-native-splash-screen` via the following changes:\n\n```java\nimport android.os.Bundle; // here\nimport com.facebook.react.ReactActivity;\n// react-native-splash-screen >= 0.3.1\nimport org.devio.rn.splashscreen.SplashScreen; // here\n// react-native-splash-screen < 0.3.1\nimport com.cboy.rn.splashscreen.SplashScreen; // here\n\npublic class MainActivity extends ReactActivity {\n   @Override\n    protected void onCreate(Bundle savedInstanceState) {\n        SplashScreen.show(this);  // here\n        super.onCreate(savedInstanceState);\n    }\n    // ...other code\n}\n```\n\n**iOS:**\n\nUpdate `AppDelegate.m` with the following additions:\n\n\n```obj-c\n#import "AppDelegate.h"\n\n#import <React/RCTBundleURLProvider.h>\n#import <React/RCTRootView.h>\n#import "RNSplashScreen.h"  // here\n\n@implementation AppDelegate\n\n- (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions\n{\n    // ...other code\n\n    [RNSplashScreen show];  // here\n    // or\n    //[RNSplashScreen showSplash:@"LaunchScreen" inRootView:rootView];\n    return YES;\n}\n\n@end\n\n```\n\n## Getting started  \n\nImport `react-native-splash-screen` in your JS file.\n\n`import SplashScreen from \'react-native-splash-screen\'`    \n\n### Android:\n\nCreate a file called `launch_screen.xml` in `app/src/main/res/layout` (create the `layout`-folder if it doesn\'t exist). The contents of the file should be the following:\n\n```xml\n<?xml version="1.0" encoding="utf-8"?>\n<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"\n    android:orientation="vertical" android:layout_width="match_parent"\n    android:layout_height="match_parent">\n    <ImageView android:layout_width="match_parent" android:layout_height="match_parent" android:src="@drawable/launch_screen" android:scaleType="centerCrop" />\n</RelativeLayout>\n```\n\nCustomize your launch screen by creating a `launch_screen.png`-file and placing it in an appropriate `drawable`-folder. Android automatically scales drawable, so you do not necessarily need to provide images for all phone densities.\nYou can create splash screens in the following folders:\n* `drawable-ldpi`\n* `drawable-mdpi`\n* `drawable-hdpi`\n* `drawable-xhdpi`\n* `drawable-xxhdpi`\n* `drawable-xxxhdpi`\n\nAdd a color called `primary_dark` in `app/src/main/res/values/colors.xml`\n\n```\n<?xml version="1.0" encoding="utf-8"?>\n<resources>\n    <color name="primary_dark">#000000</color>\n</resources>\n```\n\n\n**Optional steps\xef\xbc\x9a**\n\nIf you want the splash screen to be transparent, follow these steps.\n\nOpen `android/app/src/main/res/values/styles.xml` and add `<item name="android:windowIsTranslucent">true</item>` to the file. It should look like this:\n\n```xml\n<resources>\n    <!-- Base application theme. -->\n    <style name="AppTheme" parent="Theme.AppCompat.Light.NoActionBar">\n        <!-- Customize your theme here. -->\n        <!--\xe8\xae\xbe\xe7\xbd\xae\xe9\x80\x8f\xe6\x98\x8e\xe8\x83\x8c\xe6\x99\xaf-->\n        <item name="android:windowIsTranslucent">true</item>\n    </style>\n</resources>\n```\n\n**To learn more see [examples](https://github.com/crazycodeboy/react-native-splash-screen/tree/master/examples)**\n\n\nIf you want to customize the color of the status bar when the splash screen is displayed:\n\nCreate `android/app/src/main/res/values/colors.xml` and add\n```xml\n<?xml version="1.0" encoding="utf-8"?>\n<resources>\n    <color name="status_bar_color"><!-- Colour of your status bar here --></color>\n</resources>\n```\n\nCreate a style definition for this in `android/app/src/main/res/values/styles.xml`:\n```xml\n<?xml version="1.0" encoding="utf-8"?>\n<resources>\n    <style name="SplashScreenTheme" parent="SplashScreen_SplashTheme">\n        <item name="colorPrimaryDark">@color/status_bar_color</item>\n    </style>\n</resources>\n```\n\nChange your `show` method to include your custom style:\n```java\nSplashScreen.show(this, R.style.SplashScreenTheme);\n```\n\n### iOS    \n\nCustomize your splash screen via `LaunchImage` or `LaunchScreen.xib`,\n\n**Learn more to see [examples](https://github.com/crazycodeboy/react-native-splash-screen/tree/master/examples)**\n\n## Usage\n\nUse like so:\n\n```javascript\nimport SplashScreen from \'react-native-splash-screen\'\n\nexport default class WelcomePage extends Component {\n\n    componentDidMount() {\n    \t// do stuff while splash screen is shown\n        // After having done stuff (such as async tasks) hide the splash screen\n        SplashScreen.hide();\n    }\n}\n```\n\n## API\n\n\n| Method | Type     | Optional | Description                         |\n|--------|----------|----------|-------------------------------------|\n| show() | function | false    | Open splash screen (Native Method ) |\n| hide() | function | false    | Close splash screen                 |\n\n## Testing\n\n### Jest\n\nFor Jest to work you will need to mock this component. Here is an example:\n\n```\n// __mocks__/react-native-splash-screen.js\nexport default {\n  show: jest.fn().mockImplementation( () => { console.log(\'show splash screen\'); } ),\n  hide: jest.fn().mockImplementation( () => { console.log(\'hide splash screen\'); } ),\n}\n```\n\n## Troubleshooting \n\n### Splash screen always appears stretched/distorted\nAdd the ImageView with a scaleType in the `launch_screen.xml`, e.g.:\n```\n<?xml version="1.0" encoding="utf-8"?>\n<FrameLayout\n  xmlns:android="http://schemas.android.com/apk/res/android"\n  android:layout_width="match_parent"\n  android:layout_height="match_parent"\n  android:orientation="vertical"\n>\n  <ImageView \n    android:src="@drawable/launch_screen"\n    android:layout_width="match_parent"\n    android:layout_height="match_parent"\n    android:scaleType="centerCrop"\n  >\n  </ImageView>\n</FrameLayout>\n```\n\n## Contribution\n\nIssues are welcome.
