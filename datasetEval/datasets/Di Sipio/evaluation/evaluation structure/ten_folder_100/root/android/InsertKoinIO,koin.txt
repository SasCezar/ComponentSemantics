unstable\nkoin_version = \'2.1.0-alpha-4\'\n```\n\n## Gradle \n\n### Jcenter \n\nCheck that you have the `jcenter` repository. \n\n```gradle\n// Add Jcenter to your repositories if needed\nrepositories {\n\tjcenter()    \n}\n```\n\n### Dependencies\n\nPick one of your Koin dependency:\n\n#### Core features\n\n```gradle\n// Koin for Kotlin\nimplementation "org.koin:koin-core:$koin_version"\n// Koin extended & experimental features\nimplementation "org.koin:koin-core-ext:$koin_version"\n// Koin for Unit tests\ntestImplementation "org.koin:koin-test:$koin_version"\n// Koin for Java developers\nimplementation "org.koin:koin-java:$koin_version"\n```\n\n#### Android\n\n```gradle\n// Koin for Android\nimplementation "org.koin:koin-android:$koin_version"\n// Koin Android Scope features\nimplementation "org.koin:koin-android-scope:$koin_version"\n// Koin Android ViewModel features\nimplementation "org.koin:koin-android-viewmodel:$koin_version"\n// Koin Android Experimental features\nimplementation "org.koin:koin-android-ext:$koin_version"\n```\n\n#### AndroidX\n\n```gradle\n\n// Koin AndroidX Scope features\nimplementation "org.koin:koin-androidx-scope:$koin_version"\n// Koin AndroidX ViewModel features\nimplementation "org.koin:koin-androidx-viewmodel:$koin_version"\n// Koin AndroidX Fragment features\nimplementation "org.koin:koin-androidx-fragment:$koin_version"\n// Koin AndroidX Experimental features\nimplementation "org.koin:koin-androidx-ext:$koin_version"\n```\n\n#### Ktor\n\n```gradle\n// Koin for Ktor Kotlin\nimplementation "org.koin:koin-ktor:$koin_version"\n```\n\n# Quickstart\n\n\n## Declare a Koin module\n\nWrite with the Koin DSL what you need to assemble:\n\n```kotlin\n// Given some classes \nclass Controller(val service : BusinessService) \nclass BusinessService() \n\n// just declare it \nval myModule = module { \n  single { Controller(get()) } \n  single { BusinessService() } \n} \n```\n\n## Starting Koin\n\nUse the startKoin() function to start Koin in your application.\n\nIn a Kotlin app:\n\n```kotlin\nfun main(vararg args : String) { \n  // start Koin!\n  startKoin {\n    // your modules\n    modules(myModule)\n  }\n} \n```\n\nIn an Android app:\n\n```kotlin\nclass MyApplication : Application() {\n  override fun onCreate(){\n    super.onCreate()\n    // start Koin!\n    startKoin {\n      // Android context\n      androidContext(this@MyApplication)\n      // modules\n      modules(myModule)\n    }\n  } \n} \n```\n\n# Talking about Koin\n\n### Articles\n\n* [Dagger is dead.
