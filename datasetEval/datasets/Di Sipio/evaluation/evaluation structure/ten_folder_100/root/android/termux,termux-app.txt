b'# Termux application\n\n[![Build status](https://api.cirrus-ci.com/github/termux/termux-app.svg?branch=master)](https://cirrus-ci.com/termux/termux-app)\n[![Join the chat at https://gitter.im/termux/termux](https://badges.gitter.im/termux/termux.svg)](https://gitter.im/termux/termux)\n\n[Termux](https://termux.com) is an Android terminal application and Linux environment.\n\n- [Termux Reddit community](https://reddit.com/r/termux)\n- [Termux Wiki](https://wiki.termux.com/wiki/)\n- [Termux Twitter](http://twitter.com/termux/)\n\nNote that this repository is for the app itself (the user interface and the\nterminal emulation). For the packages installable inside the app, see\n[termux/termux-packages](https://github.com/termux/termux-packages)\n\n## Installation\n\nTermux:Widget application can be obtained from:\n\n- [Google Play](https://play.google.com/store/apps/details?id=com.termux)\n- [F-Droid](https://f-droid.org/en/packages/com.termux/)\n- [Kali Nethunter Store](https://store.nethunter.com/en/packages/com.termux/)\n\nAdditionally we offer development builds for those who want to try out latest\nfeatures ready to be included in future versions. Such build can be obtained\ndirectly from [Cirrus CI artifacts](https://api.cirrus-ci.com/v1/artifact/github/termux/termux-app/debug-build/output/app/build/outputs/apk/debug/app-debug.apk).\n\nSignature keys of all offered builds are different. Before you switch the\ninstallation source, you will have to uninstall the Termux application and\nall currently installed plugins.\n\n## Terminal resources\n\n- [XTerm control sequences](http://invisible-island.net/xterm/ctlseqs/ctlseqs.html)\n- [vt100.net](http://vt100.net/)\n- [Terminal codes (ANSI and terminfo equivalents)](http://wiki.bash-hackers.org/scripting/terminalcodes)\n\n## Terminal emulators\n\n- VTE (libvte): Terminal emulator widget for GTK+, mainly used in gnome-terminal.\n  [Source](https://github.com/GNOME/vte), [Open Issues](https://bugzilla.gnome.org/buglist.cgi?quicksearch=product%3A%22vte%22+),\n  and [All (including closed) issues](https://bugzilla.gnome.org/buglist.cgi?bug_status=RESOLVED&bug_status=VERIFIED&chfield=resolution&chfieldfrom=-2000d&chfieldvalue=FIXED&product=vte&resolution=FIXED).\n\n- iTerm 2: OS X terminal application. [Source](https://github.com/gnachman/iTerm2),\n  [Issues](https://gitlab.com/gnachman/iterm2/issues) and [Documentation](http://www.iterm2.com/documentation.html)\n  (which includes [iTerm2 proprietary escape codes](http://www.iterm2.com/documentation-escape-codes.html)).\n\n- Konsole: KDE terminal application. [Source](https://github.com/chromium/hterm),\n  including [tests](https://github.com/chromium/hterm/blob/master/js/hterm_vt_tests.js),\n  and [Google group](https://groups.google.com/a/chromium.org/forum/#!forum/chromium-hterm).\n\n- xterm: The grandfather of terminal emulators.\n  [Source](http://invisible-island.net/datafiles/release/xterm.tar.gz).\n\n- Connectbot: Android SSH client. [Source](https://github.com/connectbot/connectbot)\n\n- Android Terminal Emulator: Android terminal app which Termux terminal handling\n  is based on. Inactive.
