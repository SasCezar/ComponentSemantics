b'PLV8 - A Procedural Language in Javascript powered by V8\n=================================================\n\nPLV8 is a shared library that provides a PostgreSQL procedural language powered\nby V8 Javascript Engine.  With this program you can write in Javascript your\nfunction that is callable from SQL.\n\n## Installing\n\n    =# CREATE EXTENSION plv8;\n\nThis will install PLV8 into your database if it exists as an extension.\n\n## Testing\n\nTo test, you can execute:\n\n    =# DO $$ plv8.elog(NOTICE, "hello there!"); $$ LANGUAGE plv8;\n\nFor full documentation, see [https://plv8.github.io/](https://plv8.github.io/).\n'
