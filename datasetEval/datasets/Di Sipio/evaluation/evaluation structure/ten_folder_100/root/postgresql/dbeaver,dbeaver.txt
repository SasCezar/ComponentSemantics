EE version also supports non-JDBC datasources (WMI, MongoDB, Cassandra, Redis).\n\n* Has a lot of <a href="https://github.com/dbeaver/dbeaver/wiki">features</a> including metadata editor, SQL editor, rich data editor, ERD, data export/import/migration, SQL execution plans, etc.\n* Based on <a href="http://www.eclipse.org/">Eclipse</a> platform.\n* Uses plugins architecture and provides additional functionality for the following databases: MySQL/MariaDB, PostgreSQL, Greenplum, Oracle, DB2 LUW, Exasol, SQL Server, Sybase/SAP ASE, SQLite, Firebird, H2, HSQLDB, Derby, Teradata, Vertica, Netezza, href="https://github.com/dbeaver/dbeaver/releases">GitHub releases</a>.\n\n## Running\n\nDBeaver requires Java (JRE) 1.8+ to run.\n\n* <b>Windows</b> and <b>MacOS X</b> installers include JRE so just use them and don\'t think about internals.\n* On <b>Linux</b> you may need to install Java manually (usually by running `sudo apt-get install default-jre` or something similar).\n* If you don\'t use installer (on Windows or Mac OS X) you may need to download Java (JDK) from <a href="https://jdk.java.net/java-se-ri/8" target="_blank">OpenJDK web site</a>.\n\n## Documentation\n\n* <a Java (JDK) 8 or Java 9.\n 2. Apache Maven 3+\n 3. Internet access\n\n#### Build\n\n```sh\ngit clone https://github.com/dbeaver/dbeaver.git dbeaver\ncd dbeaver\nmvn package\n```\nBinaries are in `product/standalone/target/products`\n\n## Notes\n\n- If you have any questions, suggestions, ideas, etc - <a href="mailto:dbeaver@jkiss.org">write us</a>.\n- Pull requests are welcome.\n- Visit https://dbeaver.io or https://dbeaver.com for more information.\n- Follow us on Twitter: https://twitter.com/dbeaver_news/\n- Thanks for using DBeaver!
