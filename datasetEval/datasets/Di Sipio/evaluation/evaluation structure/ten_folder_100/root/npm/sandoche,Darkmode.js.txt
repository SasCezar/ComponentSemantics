b'# Darkmode.js ![GitHub](https://img.shields.io/github/license/mashape/apistatus.svg)\n\n\xf0\x9f\x8c\x93 Add a Dark Mode / Night Mode to your website in a few seconds\n\nThis library uses the CSS `mix-blend-mode` to bring Dark Mode to any of your websites.\nJust copy-paste the snippet and you will get a widget to turn on and off the Dark Mode. You can also use it without the widget programmatically. The plugin is lightweight, built-in Vanilla.\nIt also uses local storage by default, so your last setting will be remembered!\n\nI have been inspired by this article: https://dev.wgao19.cc/2019-05-04__sun-moon-blending-mode/\n\n## Features\n- Widget appears automatically\n- Saving users choice\n- Automatically shows Darkmode if the OS prefered theme is dark (if the browsers support `prefers-color-scheme`)\n- Can be used programmatically without widget\n\n## \xe2\x9c\xa8 Demo\nCheck out the demo in these websites: \n- https://darkmodejs.learn.uno\n- you don\'t want to show the widget and enable/disable Darkmode programatically you can use the method `toggle()`. You can also check if the Dark Mode is activated with the method `isActivated()`. See them in action in the following example:\n```javascript\nconst darkmode =  new Darkmode();\ndarkmode.toggle();\nconsole.log(darkmode.isActivated()) // will return true\n```\n\n## Override style\n* A CSS class `darkmode--activated` is added to the body tag when the darkmode is activated. You can take advantage of it to override the style and have a custom style\n* Use the class `darkmode-ignore` where you don\'t want to apply darkmode\n* You can also add this style: `isolation: isolate;` in your CSS, this will also ignore the darkmode.\n* It is also possible to revert the Dark Mode with this style `mix-blend-mode: difference;`\n\n### Examples\n```css\n.darkmode--activated p, .darkmode--activated li {\n  color: #000;\n}\n\n.button {\n  isolation: isolate;\n}\n\n.darkmode--activated .logo {\n  mix-blend-mode: difference;\n}\n```\n```html\n<span class="darkmode-ignore">\xf0\x9f\x98\xac<span>\n```\n\n## Debug\nIf it does not work you may have to add the following code, but this will invalidate the classes to override.\n```css\n.darkmode-layer, .darkmode-toggle {\n  z-index: 500;\n}\n```\n\n## Browser compatibility\nThis library uses the CSS `mix-blend-mode: difference;` to provide the Dark Mode.\nIt may not be compatible with all the browsers.
