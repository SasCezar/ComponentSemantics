b'![Rick Hickey achievements](https://raw.githubusercontent.com/someteam/acha/master/hickey.png)\n\n## Usage\n\nIf you have a docker use the following command to boot acha-acha on localhost:\n\n    docker run --rm -p 8080:8080 someteam/acha\n\nAfter that open http://localhost:8080/ to see acha-acha in your browser\n\n### Custom private key\n\nIf you want to use acha-acha for private repositories or use ssh urls you need to provide the private key with read access:\n\n    docker run --rm -p 8080:8080 -v /path/to/private/id_rsa:/root/.ssh/id_rsa someteam/acha\n\n### Mount a custom storage\n\nIf you want to store acha-acha state outside the container, use the following command to mount external working directory:\n\n    docker run --rm -p 8080:8080 -v /path/to/working_dir:/app/.acha someteam/acha\n\n## Building from source\n\n    lein do clean, cljsbuild once prod, uberjar\n    java -jar target/acha-uber.jar --ip 0.0.0.0 --port 8080 --dir .acha --private-key ~/.ssh/custom_key\n\n## Development mode\n\n    lein cljsbuild auto dev &\n    lein run --reload &\n    open http://localhost:8080/index_dev.html\n\n## License\n\nCopyright \xc2\xa9 Nikita Prokopov, Julie Prokopova, Renat Idrisov, Andrey Vasenin, Dmitry Ivanov\n\nDistributed under the Eclipse Public License either version 1.0 or (at your option) any later version.\n'
