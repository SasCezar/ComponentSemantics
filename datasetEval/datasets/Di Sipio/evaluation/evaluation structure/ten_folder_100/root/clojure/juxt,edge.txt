link:https://github.com/asciidoctor/asciidoctor-browser-extension[Asciidoctor browser extension] to view these as html in your browser.\n\n== Intro\n\nFor newcomers to the Clojure language it can be quite a daunting task to put together a real Clojure application from scratch.\nThat\'s why we\'ve done this initial step for you and creating a project that you can learn from and build upon.\n\nThis isn\'t merely a toy application.\nEdge is JUXT\'s baseline architecture which we use for our own professional Clojure projects.\nThe name is taken from https://en.wikipedia.org/wiki/Datum_reference[Datum edge].\n\n== Features\n\nEdge is a simple foundation integrating the following:\n\n* link:https://clojure.org/guides/deps_and_cli[Dependency management]\n* Development setup\n* link:https://github.com/juxt/aero[Configuration]\n* link:https://github.com/juxt/yada[Web server] & link:https://github.com/juxt/bidi[routing]\n\nWith examples for:\n\n* link:https://github.com/juxt/edge/tree/master/examples/tutorial.vent[A single page application]\n* link:https://github.com/juxt/edge/tree/master/examples/phonebook-api[A REST API]\n* link:https://github.com/juxt/edge/tree/master/examples/phonebook-graphql[A GraphQL API]\n\nFor more information on why you\'d use Edge, see link:https://juxt.pro/edge/docs/why-edge.html[Why
