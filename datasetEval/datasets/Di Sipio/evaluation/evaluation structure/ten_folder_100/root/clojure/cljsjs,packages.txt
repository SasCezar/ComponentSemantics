b'# CLJSJS Packages\n\n<img src="http://cljsjs.github.io/cljsjs.svg" alt="CLJSJS logo" align="right" />\n\nCLJSJS is an effort to package Javascript libraries to be able to use\nthem from within ClojureScript.\n\n## Using these packages\n\nAll packages provided by [cljsjs][clojars-cljsjs] provide `deps.cljs`\nfiles that will be automatically picked up by the ClojureScript\ncompiler as of [0.0-2727][2727].
