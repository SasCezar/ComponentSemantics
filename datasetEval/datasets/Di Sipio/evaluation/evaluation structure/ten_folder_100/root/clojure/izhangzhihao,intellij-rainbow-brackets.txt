Plugin</kbd> > <kbd>Restart IntelliJ IDEA</kbd>\n\nFor Mac - <kbd>IntelliJ IDEA</kbd> > <kbd>Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Browse repositories...</kbd> > <kbd>Search for "Rainbow Brackets"</kbd> > <kbd>Install Plugin</kbd>  > <kbd>Restart IntelliJ IDEA</kbd>\n\n## Customize colors\n\n<kbd>Settings/Preferences</kbd> > <kbd>Editor</kbd> > <kbd>Color Scheme</kbd> > <kbd>Rainbow Brackets</kbd>:\n\n![Customize colors](./screenshots/customize-colors.png)\n\n## Config file path\n\nIf you want to customize advanced configuration, you could edit the config file then restart your IDE. \nConfig file path in `APP_CONFIG/rainbow_brackets.xml`. \n\nIn MAC OS env maybe like `/Users/izhangzhihao/Library/Preferences/IntelliJIdea2018.3/options/rainbow_brackets.xml`.\n\nIn Linux env maybe like `~/.IntelliJIdea/config/options/rainbow_brackets.xml`.\n\nIn Windows env maybe like `C:\\Users\\izhangzhihao\\.IntelliJIdea2018.3\\config\\options\\rainbow_brackets.xml`.\n\n## HTML code in js\n\nTo enable rainbow brackets for HTML inside js code like this:\n\n```javascript\nvar html = \'<div><div><div>Hello</div></div></div>\';\n```\n\nThis plugin will automatically override color scheme property "HTML_CODE" [cause our rainbow color been covered by intellij built-in functionality](https://intellij-support.jetbrains.com/hc/en-us/community/posts/360000117450-My-HighlightVisitor-been-covered-by-intellij-built-in-functionality).\nYou still could set `<option name="rainbowifyHTMLInsideJS" value="false" />` in config file to disable just like [Config brackets colors](#config-brackets-colors).\n\n## Kotlin function literal braces and arrow\n\nTo enable rainbow brackets for multiple level lambda Kotlin code like this:\n\n```kotlin\nevent.throwable?.let { throwable ->\n    IdeErrorsDialog.findPluginId(throwable)?.let { pluginId ->\n        PluginManager.getPlugin(pluginId)?.let { ideaPluginDescriptor ->\n            if (!ideaPluginDescriptor.isBundled) {\n                bean.pluginName = ideaPluginDescriptor.name\n                bean.pluginVersion = ideaPluginDescriptor.version\n            }\n        }\n    }\n}\n```\n\nThis plugin will automatically override color scheme property "KOTLIN_FUNCTION_LITERAL_BRACES_AND_ARROW" cause our rainbow color been covered by kotlin plugin built-in functionality.\nYou still could set `<option name="rainbowifyKotlinFunctionLiteralBracesAndArrow" value="false" />` in config file to disable just like [Config brackets colors](#config-brackets-colors).\n\n## Disable rainbow brackets for specific languages\n\nIf you want to disable rainbow brackets for javascript languages and java languages, you could set `languageBlacklist` property in config file just like: \n\n```xml\n<application>\n  <component name="RainbowSettings">\n    <option name="languageBlacklist">\n      <array>\n        <option value="java" />\n        <option value="javascript" />\n      </array>\n    </option>\n  </component>\n</application>\n```\n\nNOTE: The languages name should be **lowercase**, and do **NOT** use shorthand.\n\n## Contribute\n\nNOTE: To view the PSI tree and explore the internal structure of source code, you need to setup your IDE follow
