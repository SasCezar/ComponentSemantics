b"# \xce\xbb AWSome Lambda Layers\n\n**A curated list of awesome AWS Lambda Layers**\n\n## What are Lambda Layers?\n\n> Lambda Layers are a new type of artifact that can contain arbitrary code and data, and may be referenced by zero, one, or more functions at the same time. Lambda functions in a serverless application typically share common dependencies such as SDKs, frameworks, and now runtimes. With layers, you can centrally manage common components across multiple functions enabling better code reuse.\n\n\xe2\x80\x94
