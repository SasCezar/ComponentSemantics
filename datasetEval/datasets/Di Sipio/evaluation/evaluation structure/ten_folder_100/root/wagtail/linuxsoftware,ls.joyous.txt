b'ls.joyous\n===============\n\n.. image:: https://secure.travis-ci.org/linuxsoftware/ls.joyous.svg?branch=master\n   :target: https://travis-ci.org/linuxsoftware/ls.joyous\n.. image:: https://coveralls.io/repos/github/linuxsoftware/ls.joyous/badge.svg?branch=master\n   :target: https://coveralls.io/github/linuxsoftware/ls.joyous?branch=master\n\nAbout\n------\nJoyous is a reusable calendar application for Wagtail. Features include rrule\nbased recurring events with cancellations and postponements; iCal import and export; Gcal export; event permissioning; timezone handling; and multi-site aware calendars.\n\nDocumentation\n-------------\nThe documentation for using and developing with Joyous can be found at \nhttps://joyous.readthedocs.io.\n\nDemo\n----\nYet another demonstration Wagtail website `code <http://github.com/linuxsoftware/orange-wagtail-site>`_ | `live <http://demo.linuxsoftware.nz>`_\n\nGetting Help\n-------------\nPlease report bugs or ask questions using the `Issue Tracker <http://github.com/linuxsoftware/ls.joyous/issues>`_.\n\nThanks To\n---------\n\nContributors\n~~~~~~~~~~~~\nThanks to everyone who has `contributed <https://github.com/linuxsoftware/ls.joyous/graphs/contributors>`_ to Joyous.\n\nTools\n~~~~~\n`Sauce Labs <https://saucelabs.com>`_ for their cross-browser testing platform.\n\n.. image:: /docs/powered-by-sauce-labs.png\n\n'
