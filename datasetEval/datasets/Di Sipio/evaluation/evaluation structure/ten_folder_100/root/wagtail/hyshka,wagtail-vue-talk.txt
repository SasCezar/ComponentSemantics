b'# wagtail-vue-talk\nHow to vue wagtail in a better light\n\n\n## Development with Docker\n\nThe Django server will run on port `8000`, and the Node.js server compiling the Vue.js app will run on port `8080`.\n\nTo access the Wagtail admin go to http://localhost:8000/admin/, login is `demo` / `demo123`.\n\nThe Vue.js app is expecting the Wagtail API to be avaialable at `localhost:8000/api/v2/`.\n\n```\n# build image and start containers\nmake up\n\n# enter backend (python) container\nmake enter\n\n# start django server (inside container)\ndjango-admin.py runserver 0.0.0.0:8000\n\n# enter frontend (node.js) container\nmake enter_fe\n\n# start node server (inside container)\nnpm run serve\n\n# stop and remove containers\nmake clean\n```\n'
