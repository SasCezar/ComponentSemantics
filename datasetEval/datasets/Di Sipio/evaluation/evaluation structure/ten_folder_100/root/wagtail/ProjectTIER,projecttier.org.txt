b"![Project TIER](project_tier/static/img/logo.svg)\n\n[![Build Status](https://travis-ci.org/ProjectTIER/projecttier.org.svg?branch=master)](https://travis-ci.org/ProjectTIER/projecttier.org)\n[![Coverage Status](https://coveralls.io/repos/github/ProjectTIER/projecttier.org/badge.svg?branch=master)](https://coveralls.io/github/ProjectTIER/projecttier.org?branch=master)\n[![Known Vulnerabilities](https://snyk.io/test/github/projecttier/projecttier.org/badge.svg?targetFile=requirements.txt)](https://snyk.io/test/github/projecttier/projecttier.org?targetFile=requirements.txt)\n[![Requirements Status](https://requires.io/github/ProjectTIER/projecttier.org/requirements.svg?branch=master)](https://requires.io/github/ProjectTIER/projecttier.org/requirements/?branch=master)\n[![License: AGPL v3](https://img.shields.io/badge/License-AGPL%20v3-blue.svg)](https://www.gnu.org/licenses/agpl-3.0)\n\n**Project TIER** (Teaching Integrity in Empirical Research) promotes the integration of principles and practices related to transparency and replicability in the research training of social scientists.\n\nThis repository is Project TIER's website (https://www.projecttier.org/), which is developed in Python using the Django-based [Wagtail](https://wagtail.io/) framework.\n\n## Local development\n\n1. Install [Vagrant](https://www.vagrantup.com/) and [VirtualBox](https://www.virtualbox.org/), if you haven't already.\n2. Clone the project: `git clone https://github.com/ProjectTIER/projecttier.org.git`\n3. Enter the project directory: `cd projecttier.org`\n4. Start the Vagrant VM: `vagrant up`\n5. Shell into the VM: `vagrant ssh`\n6. Run the development server: `djrun`\n7. Visit `0.0.0.0:8000` in your browser.\n\n### Pull production data/media\n\nYour local version of the project will have an empty database and no media uploads.\nYou can copy the production database and files to your local version.\n\n* To pull the database: `fab pull_production_data`\n* To pull file uploads: `fab pull_production_media`\n\nSee `fabfile.py` for more information.\nYou'll need authentication to run these commands.\n\n## License and credits\n\nCopyright \xc2\xa9 2018 Richard Ball.\n\nLicensed under the GNU AGPL 3.0.
