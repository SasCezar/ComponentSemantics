b'# Western Friend website (prototype)\n\nPrototype website for [Western Friend](https://westernfriend.org), part of the Religious Society of Friends.\n\n## Development\n\nThis project is built with [Python](https://www.python.org/), [Django](https://www.djangoproject.com/), and [Wagtail CMS](https://wagtail.io/).\n\nIn order to get started contributing, set up a development environment with the following steps:\n\n1. clone this repository\n   - `git clone git@github.com:WesternFriend/WF-website.git`\n2. change into cloned directory and project folder (`wf_website`)\n   - `cd WF-website/wf_website`\n3. create a Python environment\n   - `python -m venv env`\n4. activate the environment\n   - `source env/bin/activate`\n5. install the requirements\n   - `pip install -r requirements.txt\n6. run migrations\n   - `python manage.py migrate`\n7. create a super user\n   - `python manage.py createsuperuser`\n8. run the project\n\n   - `python manage.py runserver`\n\nOnce the server is running, you can access it from http://localhost:8000\n\nFrom there, you can begin adding the basic content, such as:\n\n- Home page\n- Community Page\n  - Contact(s)\n- Magazine Index\n  - Magazine Issue(s)\n    - Magazine Article(s)\n- Tags Index\n- etc.\n\nIf you have any difficulty or questions, please [open a support ticket](https://github.com/WesternFriend/WF-website/issues).\n\n## Attribution\n\nPortions of code for the Store and Cart apps comes used, with permission, from the book [Django 2 by Example](https://www.packtpub.com/application-development/django-2-example) by Antonio Mel\xc3\xa9.\n'
