b'# Longclaw\n\n\n[![Pypi Version](https://badge.fury.io/py/longclaw.svg)](https://badge.fury.io/py/longclaw)\n[![Codecov Status](https://codecov.io/gh/JamesRamm/longclaw/branch/master/graph/badge.svg)](https://codecov.io/gh/JamesRamm/longclaw)\n[![Build Status](https://travis-ci.org/JamesRamm/longclaw.svg?branch=master)](https://travis-ci.org/JamesRamm/longclaw)\n[![Code Health](https://landscape.io/github/JamesRamm/longclaw/master/landscape.svg?style=flat)](https://landscape.io/github/JamesRamm/longclaw/master)\n\nAn e-commerce extension for [Wagtail CMS](https://github.com/wagtail/wagtail)\n\n\n\nCheckout the [documentation](https://jamesramm.github.io/longclaw/)\n\n![Image of the dashboard](docs/assets/dashboard.png)\n\n## Quickstart\n\nInstall Longclaw:\n\n```bash\n  $ pip install longclaw\n```\n\nSetup a Longclaw project\n\n```bash\n  $ longclaw start my_project\n```\n\n## Features\n\n- Tightly integrated with Wagtail. Create products, manage orders, configure shipping and view statistics all from the Wagtail admin.\n- Multiple payment backends. Longclaw currently supports Stripe, Braintree and PayPal (v.zero) payments.\n- Comprehensive REST API & javascript client easily loaded via a template tag\n- Create your catalogue as Wagtail pages, with complete control over your product fields\n- Easy setup. Just run `longclaw start my_project` to get going\n- Simple to use, simple to change. Write your frontend as you would any other wagtail website. No complicated overriding, forking etc in order to customise behaviour.\n\n\n### Screenshots\n\n![Order Detail](docs/assets/order_detail.png)\n\n\n## Support\n\n\nPlease raise bugs/feature requests using the github issue tracker and ask questions on stackoverflow.\nFor further support contact ramshacklerecording@gmail.com\n\nIf you use Longclaw, please consider supporting the developer:\n<a href="https://www.buymeacoffee.com/pHtXDM748" target="_blank"><img src="https://www.buymeacoffee.com/assets/img/custom_images/orange_img.png" alt="Buy Me A Coffee" style="height: auto !important;width: auto !important;" ></a>\n\n'
