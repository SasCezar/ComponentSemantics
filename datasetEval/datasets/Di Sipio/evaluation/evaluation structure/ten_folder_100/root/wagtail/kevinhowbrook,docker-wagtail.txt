b'# Docker Wagtail starter\n\n\n\nSimple container to run wagtail.\n- Python 3\n- Wagtail:latest\n\n### Usage\n\nClone this repo. If you have a wagtail project already, clone into the repo root eg ```./mysite```\n\nIf you are starting a new site:\n\nStart the containers\n\n``` docker-compose up -d```\n\nSSH to the web container\n\n``` docker exec -it [web container name] /bin/bash ```\n\nCreate app with wagtail@\n\n``` wagtail start mysite```\n\nThen proceed as per the wagtail docs:\n\n##### Create the database\nBy default, this would create an SQLite database file within the project directory.\n\n```python manage.py migrate```\n\n##### Create an admin user #####\n\n```python manage.py createsuperuser```\n\n##### Run the development server\n\n```python manage.py runserver 0.0.0.0:8000```\n\nYour site is now accessible at 0.0.0.0:8000 (or watever ip your docker is set to), with the admin backend available at http://localhost:8000/admin/.\n\n[Read more on the wagtail docs](http://docs.wagtail.io/en/v1.13.1/)\n\n*you may need to run ```   sudo chown -R $USER:$USER . ``` if you create a wagtail project from within docker\n'
