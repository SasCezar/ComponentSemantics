b"Wagtail-Constance\n~~~~~~~~~~~~~~~~~\n\nI know there is `wagtail.contrib.settings <http://docs.wagtail.io/en/latest/reference/contrib/settings.html>`_ available, but I'm a big fan of `django-constance <https://github.com/jazzband/django-constance>`_ with redis backend!\n\n\n\nInstallation\n------------\n\nMake sure you installed `django-constance <https://github.com/jazzband/django-constance>`_ then install wagtail-constance:\n\n.. code-block:: sh\n    \n    pip install wagtail-constance\n\n\nAnd then just add ``wagtailconstance`` into ``INSTALLED_APPS``:\n\n.. code-block:: py\n\n    INSTALLED_APPS = (\n       ...\n       'wagtailconstance',\n       ...\n    )\n\nNote\n----\n\nI'm not added wagtail or constance as requirements in setup.py.\nI'm tested it out only with python 3.5.1\n\n\nCredits\n-------\n\n* Released under `MIT License <http://www.opensource.org/licenses/mit-license.php>`_\n* `wagtail <https://github.com/torchbox/wagtail>`_\n* `django-constance <https://github.com/jazzband/django-constance>`_\n* this `snippet <https://djangosnippets.org/snippets/10462/>`_"
