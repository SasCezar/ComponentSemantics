"encoding/json"\njson.Marshal(&data)\n```\n\nwith \n\n```go\nimport "github.com/json-iterator/go"\n\nvar json = jsoniter.ConfigCompatibleWithStandardLibrary\njson.Marshal(&data)\n```\n\nReplace\n\n```go\nimport "encoding/json"\njson.Unmarshal(input, &data)\n```\n\nwith\n\n```go\nimport
