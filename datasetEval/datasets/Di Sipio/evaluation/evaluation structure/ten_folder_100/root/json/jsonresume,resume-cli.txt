b'# resume-cli\n\n[![Gitter](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/jsonresume/public?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)\n[![Build Status](https://travis-ci.org/jsonresume/resume-cli.svg?branch=master)](https://travis-ci.org/jsonresume/resume-cli) [![Dependency Status](https://david-dm.org/jsonresume/resume-cli.svg)](https://david-dm.org/jsonresume/resume-cli) [![devDependency Status](https://david-dm.org/jsonresume/resume-cli/dev-status.svg)](https://david-dm.org/jsonresume/resume-cli#info=devDependencies)  [![](https://badge.fury.io/js/resume-cli.svg)](https://www.npmjs.org/package/resume-cli)\n\nThis is the command line tool for [JSON Resume](https://jsonresume.org), the open source initiative to create a JSON-based standard for resumes.\n\n[Read more...](https://jsonresume.org/schema/)\n\n\n# Getting Started\n\nInstall the command-line tool:\n\n```\nnpm install -g resume-cli\n```\n\n\n# Usage\n\n## `resume --help`\n\nShow a list of options and commands for the <abbr title="Command Line Interface">CLI</abbr>.\n\n\n## `resume init`\n\nCreates a new `resume.json` file in your current working directory.\n\nComplete the `resume.json` with your text editor. Be sure to follow the schema \n(available at http://jsonresume.org).\n\n\n## `resume test`\n\nValidates your `resume.json` against our schema tests to ensure it complies with \nthe standard. Tries to identify where any errors may be occurring.\n\n\n## `resume export [fileName]`\n\nExports your resume locally in a stylized HTML, Markdown, or PDF format.\n\nA list of available themes can be found here: http://jsonresume.org/themes/\n\nPlease npm install the theme you wish to use locally before attempting to export it.\n\nOptions:\n  - `--format <file type>` Example: `--format pdf`\n  - `--theme <name>` Example: `--theme flat`  \n\n## `resume serve`\n\nStarts a web server that serves your local `resume.json`.  \n\nOptions: \n  - `--port <port>`\n  - `--theme <name>`\n\nIf no theme is specified, it will look for the file `index.js` and call \n`render()`. This is useful when developing themes.\n\n\n\n# License\n\nAvailable under [the MIT license](http://mths.be/mit).\n'
