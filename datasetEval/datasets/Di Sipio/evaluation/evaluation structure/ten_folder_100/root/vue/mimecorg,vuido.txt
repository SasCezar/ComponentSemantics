Application using Vuido can run on Windows, OS X and Linux, using native GUI components, and don\'t require Electron.\n\nUnder the hood, Vuido uses the [libui](https://github.com/andlabs/libui) library which provides native GUI components for each desktop platform, and the [libui-node](https://github.com/parro-it/libui-node) bindings for Node.js.\n\n<p align="center">\n  <img src="https://raw.githubusercontent.com/mimecorg/vuido/master/docs/.gitbook/assets/vuido-demo.png" alt style="max-width:100%;">\n</p>\n\n## Documentation\n\nYou can find the full documentation of Vuido at [vuido.mimec.org](https://vuido.mimec.org/).\n\n## Acknowledgements\n\nVuido is largely based on Vue.js and shares most of its code, except for the platform specific code related to libui.\n\nVuido was inspired by [Proton Native](https://github.com/kusti8/proton-native), an environment for creating native desktop applications using React.\n\n## License\n\nVuido is licensed under the MIT license\n\nCopyright (C) 2018 Micha\xc5\x82 M\xc4\x99ci\xc5\x84ski\n'
