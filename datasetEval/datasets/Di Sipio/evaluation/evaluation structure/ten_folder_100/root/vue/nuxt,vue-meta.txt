It is inspired by and works similar as [`react-helmet`](https://github.com/nfl/react-helmet) for react. However, instead of setting your data as props passed to a proprietary component, you simply export it as part of your component\'s data using the `metaInfo` property.\n\nThese properties, when set on a deeply nested component, will cleverly overwrite their parent components\' `metaInfo`, thereby enabling custom info for each top-level view as well as coupling metadata directly to deeply nested subcomponents for more maintainable code.\n\n## Documentation\n\nPlease find the documention on https://vue-meta.nuxtjs.org\n\n> :globe_with_meridians: Please help us translate the documentation into your language, see [here](#how-to-translate-documentation) for more information\n\n## Examples\n\nLooking for more examples what vue-meta can do for you?
