<atom-spinner\n          :animation-duration="1000"\n          :size="60"\n          :color="\'#ff1d5e\'"\n     />\n  </div>\n</template>\n\n<script>\n  // To use minified css and js files instead of .vue single file components:\n  // import \'epic-spinners/dist/lib/epic-spinners.min.css\'\n  // import {AtomSpinner} from \'epic-spinners/dist/lib/epic-spinners.min.js\'\n  \n  // To get tree shaking from webpack (won\'t import all spinners when you only need one)\n  // import AtomSpinner from \'epic-spinners/src/components/lib/AtomSpinner\'\n  \n  import {AtomSpinner} from \'epic-spinners\'\n  export default {\n    components: {\n      AtomSpinner\n    }\n  }\n</script>\n```\nTo use pure html/css version, visit our [gallery](http://epic-spinners.epicmax.co) and click any spinner to see its html/css source code\n\n## Vue.js components list\nYou can easily configure spinners\' size, color and animation speed\n\n``` vue\n<flower-spinner\n  :animation-duration="2500"\n  :size="70"\n
