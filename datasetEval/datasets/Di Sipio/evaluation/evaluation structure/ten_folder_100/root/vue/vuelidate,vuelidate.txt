Support for nested models\n* Contextified validators\n* Easy to use with custom validators (e.g. Moment.js)\n* Support for function composition\n* Validates different data sources: Vuex getters, computed values, etc.\n\n## Demo & docs\n\n[https://vuelidate.netlify.com/](https://vuelidate.netlify.com/)\n\n## Installation\n\n```bash\nnpm install vuelidate --save\n```\n\nYou can import the library and use as a Vue plugin to enable the functionality globally on all components containing validation configuration.\n\n```javascript\nimport Vue from \'vue\'\nimport Vuelidate from \'vuelidate\'\nVue.use(Vuelidate)\n```\n\nAlternatively it is possible to import a mixin directly to components in which it will be used.\n\n```javascript\nimport { validationMixin } from \'vuelidate\'\n\nvar Component = Vue.extend({\n  mixins: [validationMixin],\n  validations: { ... }\n})\n```\n\nThe browser-ready bundle is also provided in the package.\n\n```html\n<script src="vuelidate/dist/vuelidate.min.js"></script>\n<!-- The builtin validators is added by adding the following line. -->\n<script src="vuelidate/dist/validators.min.js"></script>\n```\n\n```javascript\nVue.use(window.vuelidate.default)\n```\n\n## Basic usage\n\nFor each value you want to validate, you have to create a key inside validations options. You can specify when input becomes dirty by using appropriate event on your input box.\n\n```javascript\nimport { required, minLength, between } from \'vuelidate/lib/validators\'\n\nexport default {\n  data () {\n    return {\n      name: \'\',\n      age: 0\n    }\n  },\n  validations: {\n    name: {\n      required,\n      minLength: minLength(4)\n    },\n    age: {\n      between: between(20, 30)\n    }\n  }\n}\n```\n\nThis will result in a validation object:\n\n```javascript\n$v: {\n  name: {\n    "required": false,\n    "minLength": false,\n    "$invalid": true,\n    "$dirty": false,\n    "$error": false,\n    "$pending": false\n  },\n  age: {\n    "between": false\n    "$invalid": true,\n    "$dirty": false,\n    "$error": false,\n    "$pending": false\n  }\n}\n```\n\nCheckout the docs for more examples: [https://vuelidate.netlify.com/](https://vuelidate.netlify.com/)\n\n## Contributing\n\n``` bash\n# install dependencies\nnpm install\n\n# serve with hot reload at localhost:8080\nnpm run dev\n\n# create UMD bundle.\nnpm run build\n\n# Create docs inside /gh-pages ready to be published\nnpm run docs\n\n# run unit tests\nnpm run unit\n\n# run all tests\nnpm test\n```\n\nFor detailed explanation on how things work, checkout the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).\n\n## Contributors\n\n### Current\n\n- [Damian Dulisz](https://github.com/shentao)\n- [Natalia Tepluhina](https://github.com/NataliaTepluhina)\n- [Dobromir Hristov](https://github.com/dobromir-hristov)\n- [Ben Hong](https://github.com/bencodezen)\n\n### Emeriti\n\nHere we honor past contributors who have been a major part on this project.\n\n- [Monterail](https://github.com/monterail)\n- [Pawe\xc5\x82 Grabarz](https://github.com/Frizi)\n\n## License\n\n[MIT](http://opensource.org/licenses/MIT)\n'
