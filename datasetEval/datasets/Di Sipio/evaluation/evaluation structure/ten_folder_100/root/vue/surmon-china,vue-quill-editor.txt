javascript\r\nimport Vue from \'vue\'\r\nimport VueQuillEditor from \'vue-quill-editor\'\r\n\r\n// require styles\r\nimport \'quill/dist/quill.core.css\'\r\nimport Difference\xef\xbc\x88\xe4\xbd\xbf\xe7\x94\xa8\xe6\x96\xb9\xe6\xb3\x95\xe7\x9a\x84\xe5\x8c\xba\xe5\x88\xab\xef\xbc\x89\r\n\r\n**SSR and the only difference in the use of the SPA:**\r\n- SPA worked by the `component`, find quill instance by `ref attribute`.\r\n- SSR worked by the `directive`, find quill instance by `directive arg`.\r\n- Other configurations, events are the same.\r\n\r\n### SSR\r\n\r\n``` vue\r\n<!-- You can custom the "myQuillEditor" name used to find the quill instance in current component -->\r\n<template>\r\n  <!-- bidirectional data v-quill:myQuillEditor="editorOption">\r\n  </div>\r\n\r\n  <!-- Or manually control the data synchronization\xef\xbc\x88\xe6\x89\x8b\xe5\x8a\xa8\xe6\x8e\xa7\xe5\x88\xb6\xe6\x95\xb0\xe6\x8d\xae\xe6\xb5\x81\xef\xbc\x89  -->\r\n  <div class="quill-editor" \r\n       :content="content"\r\n       @change="onEditorChange($event)"\r\n       v-quill:myQuillEditor="editorOption">\r\n  </div>\r\n</template>\r\n\r\n<script>\r\n  export default {\r\n    data() {\r\n      return {\r\n        content: \'<p>example content</p>\',\r\n        editorOption: { /* quill options */ }\r\n      }\r\n    },\r\n    mounted() {\r\n      console.log(\'this is current quill instance object\', this.myQuillEditor)\r\n    },\r\n    methods: {\r\n      onEditorChange(event) {\r\n        console.log(\'onEditorChange\')\r\n      }\r\n    },\r\n    // Omit the same parts as in the following component sample code\r\n    // ...\r\n  }\r\n</script>\r\n```\r\n\r\n\r\n### SPA\r\n\r\n``` vue\r\n<template>\r\n  <!-- bidirectional data    methods: {\r\n      onEditorBlur(quill) {\r\n        console.log(\'editor blur!\', quill)\r\n      },\r\n      onEditorFocus(quill) {\r\n        console.log(\'editor focus!\', quill)\r\n
