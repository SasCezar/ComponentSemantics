Work with node-canvas.\n- [Sprite-vue](http://vue.spritejs.org). Work with [Vue](https://github.com/vuejs/vue).\n- [Sprite-React](http://react.spritejs.org). Work with [React](https://github.com/vuejs/react).\n- [\xe5\xbe\xae\xe4\xbf\xa1\xe5\xb0\x8f\xe7\xa8\x8b\xe5\xba\x8f](https://github.com/spritejs/sprite-wxapp)\n\n## Quick Start\n\n**SpriteJS** - spritejs.org\n\n```html\n<script src="https://unpkg.com/spritejs/dist/spritejs.min.js"></script>\n<div id="container"></div>\n<script>\n    const imgUrl = \'https://s5.ssl.qhres.com/static/ec9f373a383d7664.svg\'\n    const {Scene, Sprite} = spritejs;\n    const paper = new Scene(\'#container\', { viewport: [400, 400] })\n\n    const sprite = new Sprite(imgUrl)\n    sprite.attr({\n      bgcolor: \'#fff\',\n      pos: [0, 0],\n      size: [400, 400],\n      borderRadius: \'200\'\n    })\n\n    paper.layer().appendChild(sprite)\n</script>\n```\n\n**With Document CSS** - Just set `useDocumentCSS` option to true, spritejs will auto load style rules from the document.\n\n```html\n<script src="https://unpkg.com/spritejs/dist/spritejs.min.js"></script>\n<div id="container"></div>\n<style>\n  sprite.myclass {\n    background-color: red;\n    --sprite-x: 0;\n    --sprite-y: 0;\n    width: 400px;\n    height: 400px;\n    border-radius: 200px;\n  }\n</style>\n<script>\n    const imgUrl = \'https://s5.ssl.qhres.com/static/ec9f373a383d7664.svg\'\n    const {Scene, Sprite} = spritejs;\n    const paper = new Scene(\'#container\', {\n      viewport: [400, 400],\n      useDocumentCSS: true,\n    })\n\n    const sprite = new Sprite(imgUrl)\n    sprite.className = \'myclass\';\n\n    paper.layer().appendChild(sprite)\n</script>\n```\n\n**React JSX** - react.spritejs.org\n\n```jsx\nimport React from \'react\';\nimport ReactDOM from \'react-dom\';\nimport {Scene} from \'sprite-react\';\n\nclass Block extends React.Component {\n  constructor(props) {\n    super(props);\n    this.state = {color: \'green\'};\n  }\n\n  handleClick() {\n    this.setState({\n      color: \'blue\',\n    });\n  }\n\n  render() {\n    return (\n      <sprite pos={[100, 100]} bgcolor={this.state.color} size={[50, 50]} onClick={this.handleClick.bind(this)}></sprite>\n    );\n  }\n}\n\n\nReactDOM.render(\n  <Scene>\n    <layer id="fglayer" handleEvent={true}>\n      <group>\n        <sprite pos={[200, 100]} size={[50, 50]} bgcolor="red" onClick={function () { this.attr(\'bgcolor\', \'blue\') } }></sprite>\n        <Block/>\n      </group>\n    </layer>\n  </Scene>,\n  document.getElementById(\'app\')\n);\n```\n\n**Vue Component** - vue.spritejs.org\n\n```xml\n<template>\n  <scene id="container" :viewport="[300, 300]">\n    <layer>\n      <sprite\n        :textures="imgUrl"\n        bgcolor="#fff"\n        :pos="[0,0]"\n        :size="[400, 400]"\n        borderRadius="200"\n      />\n    </layer>\n  </scene>\n</template>\n\n<script>\nexport default {\n  data() {\n    return {\n      imgUrl: \'https://s5.ssl.qhres.com/static/ec9f373a383d7664.svg\'\n    }\n  }\n}\n</script>\n```\n\n<div style="font-size: 1.5rem">Learn more at <strong style="font-size: 2.5rem"><a
