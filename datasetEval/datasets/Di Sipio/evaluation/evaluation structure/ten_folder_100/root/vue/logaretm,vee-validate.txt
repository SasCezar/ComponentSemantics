The errors will be automatically generated with 40+ locales supported. [Many rules are available out of the box](https://logaretm.github.io/vee-validate/guide/rules.html).\r\n\r\nThis plugin is inspired by [PHP Framework Laravel\'s validation](https://laravel.com/).\r\n\r\n### Features\r\n\r\n- Template based validation that is both familiar and easy to setup.\r\n- \xf0\x9f\x8c\x8d i18n Support and error Messages in 40+ locales.\r\n- \xf0\x9f\x92\xab Async and Custom Rules Support.\r\n- \xf0\x9f\x92\xaa Written in TypeScript.\r\n- No dependencies.\r\n\r\n### Installation\r\n\r\n#### yarn\r\n\r\n```bash\r\nyarn add vee-validate\r\n```\r\n\r\n#### npm\r\n\r\n```bash\r\nnpm i vee-validate --save\r\n```\r\n\r\n#### CDN\r\n\r\nvee-validate is also available on these CDNs:\r\n\r\n- [jsdelivr](https://cdn.jsdelivr.net/npm/vee-validate@latest/dist/vee-validate.js) [![jsDelivr Hits](https://data.jsdelivr.com/v1/package/npm/vee-validate/badge?style=rounded)](https://www.jsdelivr.com/package/npm/vee-validate)\r\n- [unpkg](https://unpkg.com/vee-validate)\r\n\r\n> When using a CDN via script tag, all the exported modules on VeeValidate are available on the VeeValidate Object. ex: VeeValidate.Validator\r\n\r\n### Getting Started\r\n\r\nInstall the rules you will use in your app, we will install the `required` rule for now:\r\n\r\n```js\r\nimport { extend } from \'vee-validate\';\r\nimport { required } from \'vee-validate/dist/rules\';\r\n\r\n// Add the required rule\r\nextend(\'required\', {\r\n  ...required,\r\n  message: \'This field is required\'\r\n});\r\n```\r\n\r\nImport the `ValidationProvider` component and register it:\r\n\r\n#### Global Registration\r\n\r\n```js\r\nimport { ValidationProvider } from \'vee-validate\';\r\n\r\n// Register it globally\r\n// main.js or any entry file.\r\nVue.component(\'ValidationProvider\', ValidationProvider);\r\n```\r\n\r\n#### Local Registration\r\n\r\n```js\r\nimport { ValidationProvider } from \'vee-validate\';\r\n\r\nexport default {\r\n  components: {\r\n    ValidationProvider\r\n  }\r\n};\r\n```\r\n\r\nAll the JavaScript work is done. Next in the template add the inputs you want to validate them:\r\n\r\n```vue\r\n<ValidationProvider name="email" rules="required|email">\r\n  <div slot-scope="{ errors }">\r\n    <input v-model="email">\r\n    <p>{{ errors[0] }}</p>\r\n  </div>\r\n</ValidationProvider>\r\n```\r\n\r\nThe validation provider accepts two props: `rules` which is in its simplest form, a string containing the validation rules separated by a `|` character, and a `name` prop which is the field name that will be used in error messages.\r\n\r\nand That\'s it, your input will be validated automatically, notice that the `ValidationProvider` uses [scoped slots](https://vuejs.org/v2/guide/components-slots.html#Scoped-Slots) to pass down validation state and results.\r\n\r\nThere is more that can be done! You can customize events, validate initial values, manually validate or reset the field and much more. Make sure to [read the docs](https://logaretm.github.io/vee-validate).\r\n\r\n### Documentation\r\n\r\nRead the [documentation and demos](https://logaretm.github.io/vee-validate/).\r\n\r\n### Compatibility\r\n\r\nThis library uses ES6 Promises so be sure to provide a polyfill for it for the browsers that do not support it.\r\n\r\n### Contributing\r\n\r\nYou are welcome to contribute to this project, but before you do, please make sure you read the [contribution guide](CONTRIBUTING.md).\r\n\r\n### Tutorials and Examples\r\n\r\n#### Docs Highlights\r\n\r\n- [Backend and Server Side Validation](https://logaretm.github.io/vee-validate/advanced/server-side-validation.html)\r\n- [Localization (i18n)](https://logaretm.github.io/vee-validate/guide/localization.html)\r\n- [Handling Forms](https://logaretm.github.io/vee-validate/guide/forms.html)\r\n- [UI Libraries Validation Samples](https://logaretm.github.io/vee-validate/guide/3rd-party-libraries.html)\r\n  - [Vuetify](https://logaretm.github.io/vee-validate/guide/3rd-party-libraries.html#vuetify)\r\n  - [Quasar](https://logaretm.github.io/vee-validate/guide/3rd-party-libraries.html#quasar)\r\n  - [BootstrapVue](https://logaretm.github.io/vee-validate/guide/3rd-party-libraries.html#bootstrapvue)\r\n  - [Buefy](https://logaretm.github.io/vee-validate/guide/3rd-party-libraries.html#buefy)\r\n  - [Ant Design](https://logaretm.github.io/vee-validate/guide/3rd-party-libraries.html#ant-design)\r\n\r\n### Credits\r\n\r\n- Inspired by Laravel\'s [validation syntax](https://laravel.com/docs/5.4/validation).\r\n\r\n## Contributors\r\n\r\n### Emeriti\r\n\r\nHere we honor past contributors and sponsors who have been a major part on this project.\r\n\r\n- [Baianat](https://github.com/baianat).\r\n\r\n### Code Contributors\r\n\r\nThis project exists thanks to all the people who contribute.
