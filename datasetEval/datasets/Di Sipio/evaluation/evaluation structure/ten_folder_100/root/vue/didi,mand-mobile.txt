width="200"/>\n\n## Install & Usage\n\n### Template for new project\n\n#### Vue CLI 2\nNew project can be initialized and integrated with mand-mobile by [vue-cli-2](https://github.com/vuejs/vue-cli/tree/v2) with [mand-mobile-template](https://github.com/mand-mobile/mand-mobile-template).\n\n```bash\nvue init mand-mobile/mand-mobile-template my-project\n```\n\n#### Vue CLI 3\nNew project can be initialized and integrated with mand-mobile by [vue-cli](https://github.com/vuejs/vue-cli/tree/master) with [vue-cli-plugin-mand](https://github.com/mand-mobile/vue-cli-plugin-mand).\n\n```bash\nvue create my-project\ncd my-project\nnpm install --save-dev vue-cli-plugin-mand\nvue invoke mand\n```\n\n### Manually\n\n```bash\nnpm install mand-mobile --save\n```\n\n### Import\n\n* Use <a href="https://github.com/ant-design/babel-plugin-import" target="_blank">babel-plugin-import</a>\n  or\n  <a href="https://github.com/Brooooooklyn/ts-import-plugin" target="_blank">ts-import-plugin</a> (Recommended)\n\n```javascript\nimport { Button } from \'mand-mobile\'\n```\n\n* Manually import\n\n```javascript\nimport Button from \'mand-mobile/lib/button\'\n```\n\n* Totally import\n\n```javascript\nimport Vue from \'vue\'\nimport mandMobile from \'mand-mobile\'\nimport \'mand-mobile/lib/mand-mobile.css\'\n\nVue.use(mandMobile)\n```\n\n### Usage\n\nSelect the components you need to build your webapp. Find more details in [Quick Start](https://didi.github.io/mand-mobile/#/en-US/docs/started).\n\n## Development\n\n```bash\ngit clone git@github.com:didi/mand-mobile.git\ncd mand-mobile\nnpm install\nnpm run dev\n```\nOpen your
