installed:\n- [Node.js](https://nodejs.org/en/) (>=8.9)\n- [npm](https://www.npmjs.com/get-npm) version 3+ (or [yarn](https://yarnpkg.com/lang/en/docs/install/#mac-stable) version 1.16+)  and [Git](https://git-scm.com/).\n\nAfter checking the prerequisites, follow these simple steps to install and use Vuestic Admin:\n\n```\n# clone the repo\n$ git clone https://github.com/epicmaxco/vuestic-admin.git myproject\n\n# go into app\'s directory and install dependencies:\n$ cd myproject\n\n```\n\nThen, if you use npm:\n\n```\n$ npm install\n\n# serve with hot reload at localhost:8080 by default.\n$ npm run serve\n\n# build for production \n$ npm run build\n\n# build for production and view the bundle analyzer report.\n$ npm run build --report\n```\n\nIf you use yarn:\n```\n$ yarn install\n\n# serve with hot reload at localhost:8080 by default.\n$ yarn serve\n\n# build for production \n$ yarn build\n\n# build for production and view the bundle analyzer report.\n$ yarn build --report\n```\n\n## Features\n[Responsive layout](https://vuestic.epicmax.co/#/admin/dashboard) |\n[charts
