b'<img src="https://i.imgur.com/L5nX9pT.png" width="100%" />\n\n# ElectroCRUD 2 ![GitHub Release Date](https://img.shields.io/github/release-date/garrylachman/ElectroCRUD.svg?style=for-the-badge) [![Open Collective backers and sponsors](https://img.shields.io/opencollective/all/electrocrud.svg?style=for-the-badge)](https://opencollective.com/electrocrud)\n### CRUD?\nCreate, Read, Update, and Delete (CRUD) are the four basic functions of persistent storage (Database).\n\n### Why?\nAfter a failed search for a basic admin panel that does not involve writing code or installs web-based solutions, I came up with ElectroCRUD idea.\n\n### ElectroCRUD v1\nThe ElectroCRUD v1 was born in 2016. An Electron application that made it possible to create a friendly dashboard with data tables, search, widgets, filter & relations with few clicks without any experience in programming.\n\n### ElectroCRUD v2\nAfter a few years of the project being inactive, I decided to bring it to life again. The codebase was built from scratch using Electron+Angular8. The new version contains most of the features of the first version, and new features are added on a weekly basis.\n\n[![forthebadge](https://forthebadge.com/images/badges/built-with-love.svg)](https://forthebadge.com) \n\n### Functionality\n- [x] MySQL Support\n- [x] PostgreSQL Support\n- [x] SSH Tunnels (Keys & Password)\n- [x] Accounts (represent a database)\n  - [x] Permissions (C,R,U,D)\n  - [x] Terminology\n  - [x] Selection of columns to display or search by\n  - [x] Add new record / Edit record\n  - [x] Edit record\n  - [x] Connect existing view as sub views\n  - [ ] Export data in various formats like: csv, dump\n- [x] Subviews (represent a sub table)\n- [x] Views relations\n  - [x] Drill from view to view (subview)\n  - [x] Column based tables relation (left join) \n- [x] View widgets (display aggragated data in the view)\n  -
