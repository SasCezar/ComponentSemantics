With Bower\n\n```bash\n$ bower install jQuery-QueryBuilder\n```\n\n#### With npm\n\n```bash\n$ npm install jQuery-QueryBuilder\n```\n\n#### Via CDN\n\njQuery-QueryBuilder is available on [jsDelivr](https://www.jsdelivr.com/package/npm/jQuery-QueryBuilder).\n### Dependencies\n * [jQuery 3](https://jquery.com)\n * [Bootstrap 3](https://getbootstrap.com/docs/3.3) (CSS only)\n * [jQuery.extendext](https://github.com/mistic100/jQuery.extendext)\n * [doT.js](https://olado.github.io/doT)\n * [MomentJS](https://momentjs.com) (optional, for Date/Time validation)\n * [SQL Parser](https://github.com/mistic100/sql-parser) (optional, for SQL methods)\n * Other Bootstrap/jQuery plugins used by plugins\n\n($.extendext and doT.js are directly included in the [standalone](https://github.com/mistic100/jQuery-QueryBuilder/blob/master/dist/js/query-builder.standalone.js) file)\n\n### Browser support\n * Internet Explorer >= 9\n * All other recent browsers\n\n\n\n## Build\n\n#### Prerequisites\n\n * NodeJS + NPM: `apt-get install nodejs-legacy npm`\n * Grunt CLI: `npm install -g grunt-cli`\n\n#### Run\n\nInstall Node dependencies `npm install` then run `grunt` in the root directory to generate production files inside `dist`.\n\n#### Options\n\nYou can choose which plugins to include with `--plugins` :\n```bash\n# include "sql-support" and "mongodb-support" plugins\ngrunt --plugins=sql-support,mongodb-support\n\n# disable all plugins\ngrunt --plugins=false\n```\nAll plugins are included by default.\n\nYou can also include language files with `--languages` :\n```bash\n# include French & Italian translation\ngrunt --languages=fr,it\n```\n\n#### Other commands\n\n * `grunt test` to run jshint/jscs/scsslint and the QUnit test suite.\n * `grunt serve` to open the example page with automatic build and livereload.\n * `grunt doc` to generate the documentation.\n\n\n## License\nThis library is available under the MIT license.\n'
