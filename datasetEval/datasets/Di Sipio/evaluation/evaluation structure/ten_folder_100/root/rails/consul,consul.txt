[docs](https://docs.consulproject.org)\n\nPrerequisites: install git, Ruby 2.4.9, `bundler` gem, Node.js and PostgreSQL (>=9.4).\n\n```bash\ngit clone https://github.com/consul/consul.git\ncd consul\nbundle install\ncp config/database.yml.example config/database.yml\ncp config/secrets.yml.example config/secrets.yml\nbin/rake db:create\nbin/rake db:migrate\nbin/rake db:dev_seed\nRAILS_ENV=test rake db:setup\n```\n\nRun the app locally:\n\n```bash\nbin/rails s\n```\n\nPrerequisites for testing: install ChromeDriver >= 2.33\n\nRun the tests with:\n\n```bash\nbin/rspec\n```\n\nYou can use the default admin user from the seeds file:\n\n **user:** admin@consul.dev\n **pass:** 12345678\n\nBut for some actions like voting, you will need a
