b'<div align="center">\n  <img src="example.png">\n</div>\n\n>  Put device frames around your mobile/web/progressive app screenshots.\n\n<div align="center">\n  <strong>This tool is in active development. If something seems off, file an issue!</strong>\n</div>\n\n# Get it\n\n    npm install -g deviceframe\n\n# Use it\n\n<div align="center">\n  <img src="animation.gif">\n</div>\n\n<br>\nPass in filenames, file globs, websites urls, or image urls. In any combination or order.\n\n```\n$ dframe cat.jpg\n$ dframe http://github.com\n$ dframe http://github.com dog.png https://i.imgur.com/aw2bc01.jpg *.bmp\n```\n\ndeviceframe will prompt you for the frames you want to use. You can select multiple frames and search by typing. Once you have selected all the frames you want to use, hit ESC.\n\nThe frames will be downloaded from a CDN and cached locally in case you want to re-use them.\n\nIf you want to use it non-interactively you can pass frames in with the `--frame flag`:\n\n```\n$ dframe cat.jpg --frame "iphone 6"\n$ dframe cat.jpg --frame "iphone 7 silver" --frame "iphone 5 blue"\n```\n\nYou can also use one flag and separate the devices by commas:\n\n```\n$ dframe cat.jpg --frame "iphone 6s, macbook air"\n```\n\n# Mobile Apps\n\n_Coming Soon_\n\n# Hybrid Apps\n\nMake a screenshot of an Ionic app locally:\n\n```\n$ ionic serve\n```\n\nIn another terminal:\n\n```\n$ dframe --delay 5 http://localhost:8100\n```\n\n# Websites\n\nFor website URLs, deviceframe will load the page with the aspect ratio and pixel density of selected device(s).\n\nHere\'s Product Hunt\'s website on a Macbook Air, iPad, and iPhone:\n\n<div align="center">\n  <img src="multi-shot.png">\n</div>\n\n# Devices\n\nCurrently 52 unique devices. Run `dframe --devices` to get a list\n\n* 166 phone + color combinations (Apple, Google, Samsung, HTC, Motorola)\n* 6 tablets (iPad, Surface, Nexus 9)\n* 9 computers (Apple, Dell, Microsoft)\n* 5 monitors\n* 4 watches (Apple Watch 1+2, Moto 360, Sony SmartWatch 3) either open or closed.\n\n# Notes\n\n* deviceframe caches frame images in a cache directory using [env-paths](https://github.com/sindresorhus/env-paths)\n* Pixel ratio information comes from https://mydevice.io/devices/\n* iPhone X renders the image below the notch.\n\n# Attributions\n\nLogo icon created by Vallone Design from the Noun Project\n'
