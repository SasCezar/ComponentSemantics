It\'s recommended to make sure eruda is loaded only when eruda is set to true on url(http://example.com/?eruda=true), for example:\n\n```javascript\n;(function () {\n    var src = \'//cdn.jsdelivr.net/npm/eruda\';\n    if (!/eruda=true/.test(window.location) && localStorage.getItem(\'active-eruda\') != \'true\') return;\n    document.write(\'<scr\' + \'ipt src="\' + src + \'"></scr\' + \'ipt>\');\n    document.write(\'<scr\' + \'ipt>eruda.init();</scr\' + \'ipt>\');\n})();\n```\n\n## Configuration\n\nWhen initialization, a configuration object can be passed in.\n\n* container: Container element. If not set, it will append an element directly\nunder html root element.\n* tool: Choose which default tools you want, by default all will be added.\n\nFor more information, please check the [documentation](doc/API.md).\n\n```javascript\nlet el = document.createElement(\'div\');\ndocument.body.appendChild(el);\n\neruda.init({\n    container: el,\n    tool: [\'console\', \'elements\']\n});\n```\n\n## Plugins\n\n* [eruda-fps](https://github.com/liriliri/eruda-fps): Display page fps info.\n* [eruda-features](https://github.com/liriliri/eruda-features): Browser feature detections.\n* [eruda-timing](https://github.com/liriliri/eruda-timing): Show performance and resource timing.\n* [eruda-memory](https://github.com/liriliri/eruda-memory): Display page memory info.\n* [eruda-code](https://github.com/liriliri/eruda-code): Run JavaScript code.\n* [eruda-benchmark](https://github.com/liriliri/eruda-benchmark): Run JavaScript benchmarks.\n* [eruda-geolocation](https://github.com/liriliri/eruda-geolocation): Test geolocation.\n* [eruda-dom](https://github.com/liriliri/eruda-dom): Navigate dom tree.\n* [eruda-orientation](https://github.com/liriliri/eruda-orientation): Test orientation api.\n* [eruda-touches](https://github.com/liriliri/eruda-touches): Visualize screen touches.\n\nIf you want to create a plugin yourself, follow the guides [here](./doc/PLUGIN.md).\n\n## Related Projects\n\n* [licia](https://github.com/liriliri/licia): Utility library used by eruda.\n* [eruda-webpack-plugin](https://github.com/huruji/eruda-webpack-plugin): Eruda webpack plugin.\n\n## Contribution\n\nRead [Contributing Guide](.github/CONTRIBUTING.md) for development setup instructions.\n'
