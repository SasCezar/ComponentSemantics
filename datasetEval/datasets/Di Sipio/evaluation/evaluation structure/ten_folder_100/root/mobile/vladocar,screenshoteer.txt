b'<h1 align="center"> Screenshoteer </h1>\n\n<p align="center"> Makes web screenshots and mobile emulations from the command line. </p>\n\n<p align="center"><img src="carbon-shot.png" /></p>\n<hr/>\n\n<p>Tool based on <a href="https://github.com/GoogleChrome/puppeteer">puppeteer</a>. </p>\n\n<h4>Installation </h4>\n\n```shell\nnpm i -g screenshoteer\n```\n<p>You can use  screenshoteer like this:</p>\n\n```shell\nscreenshoteer  --url https://www.example.com\n\nor .html localy copy the url path from the browser\n\nscreenshoteer --url file:///Users/../index.html\nscreenshoteer --url file:///C:/Users/../Random-HTML-file.html\n```\n\n<p>And with the help of puppeteer(Headless Chrome) it will generate screenshot of the entire web page.</p>\n\n<p>\nParameters:  \n\n\n-h help  \n--url web page url  \n--emulate - emulate web device example: --emulate "iPhone 6"  \n--fullpage - can be true or false. It will take screenshot of entire web page if is true. True is the default parameter.  \n--pdf - generate additional pdf  \n--w - width of the Web Page in px  \n--h - height of the Web Page in px  \n--waitfor - wait time for the page load in milliseconds  \n--waitforselector - wait for the selector to appear in page\n--el - css selector document.querySelector  \n--auth - basic http authentication  \n--no - exclude "image", "stylesheet", "script", "font"  \n--click - example: ".selector>a" excellent way to close popups or to click some buttons on the page.  \n--file - output file name (optional, otherwise based on page title and timestamp)\n<p>\n\n<h4>Example: </h4>\n\n```shell\nscreenshoteer  --url https://news.ycombinator.com --fullpage false\n\nscreenshoteer  --url https://www.reddit.com/r/nodejs --emulate "iPhone 7"\n\nscreenshoteer  --url https://www.nytimes.com  --emulate "Nexus 4"\n\nscreenshoteer --url https://www.reddit.com/r/javascript/ --w 600 --h 800 --fullpage false\n\nscreenshoteer --url https://www.reddit.com/r/javascript/ --w 600 --h 0 --fullpage false\n\nscreenshoteer --url https://lobste.rs --pdf\n\nscreenshoteer --url https://lobste.rs --w 500\n\nscreenshoteer --url  https://news.ycombinator.com/item?id=18598672 --el ".fatitem"\n\nscreenshoteer --url  https://site.com --auth "username;password"\n\nscreenshoteer --url https://www.nytimes.com --no "image"\n\nscreenshoteer --url https://www.nytimes.com --no "script"\n\nscreenshoteer --url https://www.economist.com/ --click ".ribbon__close-button"\n\nscreenshoteer --url file:///Users/../index.html\n\nscreenshoteer --url https://www.slashdot.org --file /tmp/slashdot.png\n```\n<p> List of of supported mobile devices: https://github.com/GoogleChrome/puppeteer/blob/master/DeviceDescriptors.js\n</p>\n\n<h3>License</h3>\n\nThis project is licensed under the MIT License\n'
