<wxc-button text="Open Popup"\n                @wxcButtonClicked="buttonClicked">\n    </wxc-button>\n    <wxc-popup width="500"\n               pos="left"\n               :show="isShow"\n               @wxcPopupOverlayClicked="overlayClicked">\n    </wxc-popup>\n  </div>\n</template>\n\n<script>\n  import { WxcButton, WxcPopup } from \'weex-ui\';\n  // or\n  // import WxcButton from \'weex-ui/packages/wxc-button\';\n  // import WxcPopup from \'weex-ui/packages/wxc-popup\';\n  module.exports = {\n    components: { WxcButton, WxcPopup },\n    data: () => ({\n      isShow: false\n    }),\n    methods: {\n      buttonClicked () {\n        this.isShow = true;\n      },\n      overlayClicked () {\n        this.isShow = false;\n      }\n    }\n  };\n</script>\n```\n\n### Before use\n\nIn order to not pack all the components, you need to use [`babel-plugin-component`](https://www.npmjs.com/package/babel-plugin-component) to import the specified component.\nAt the same time, if you haven\'t installed `babel-preset-stage-0`, its necessary to install it.\n\n```shell\nnpm i babel-preset-stage-0 babel-plugin-component -D\n```\n\n```json\n{\n    "presets": ["es2015", "stage-0"],\n    "plugins": [\n        [\n            "component",\n            {\n                "libraryName": "weex-ui",\n                "libDir": "packages",\n                "style": false\n            }\n        ]\n    ]\n}\n```\n\n### More\n\n* If babel-loader in `webpack.config.js` has a exclude for node_modules, please turn on for week-ui as `exclude:
