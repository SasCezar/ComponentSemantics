b'HubCommander\n=====================\n<img align="left" src="docs/logos/HC_full_logo.png" alt="HubCommander Logo" width="50%" />\n\n[![NetflixOSS Lifecycle](https://img.shields.io/badge/NetflixOSS-active-brightgreen.svg)]()\n[![Gitter chat](https://badges.gitter.im/gitterHQ/gitter.png)](https://gitter.im/Netflix/hubcommander)\n\nA user-extendable Slack bot for GitHub organization management.\n\nHubCommander provides a chat-ops means for managing GitHub organizations.\nIt creates a simple way to perform privileged GitHub organization management tasks without granting\nadministrative or `owner` privileges to your GitHub organization members.\n\n<br />\n<br />\n\n| Service   | Master | Develop |\n|:-----------:|:--------:|:---------:|\n|Travis CI|[![Build Status](https://travis-ci.org/Netflix/hubcommander.svg?branch=master)](https://travis-ci.org/Netflix/hubcommander)|[![Build Status](https://travis-ci.org/Netflix/hubcommander.svg?branch=develop)](https://travis-ci.org/Netflix/hubcommander)|\n\n\nHow does it work?\n-------------\nHubCommander is based on [slackhq/python-rtmbot](https://github.com/slackhq/python-rtmbot)\n(currently, dependent on release [0.4.0](https://github.com/slackhq/python-rtmbot/releases/tag/0.4.0))\n\nYou simply type `!help`, and the bot will output a list of commands that the bot supports. Typing\nthe name of the command, for example: `!CreateRepo`, will output help text on how to execute the command.\n\nAt a minimum, you will need to have the following:\n* Python 3.5+\n* Slack and Slack credentials\n* A GitHub organization\n* A GitHub bot user with ownership level privileges\n\nA Docker image is also available to help get up and running quickly.\n\nFeatures\n-------------\nOut of the box, HubCommander has the following GitHub features:\n* Repository creation\n* Repository deletion\n* Repository description and website modification\n* Granting outside collaborators specific permissions to repositories\n* Repository default branch modification\n* Repository PR listing\n* Repository deploy Key listing/creation/deletion\n* Repository topics creation/deletion\n* Repository branch protection enabling/disabling\n\nHubCommander also features the ability to:\n* Enable Travis CI on a GitHub repo\n* Safeguard commands with 2FA via Duo\n\nYou can add additional commands by creating plugins. For example, you can create a plugin to invite users\nto your organizations.\n\nHubCommander also supports Slack ephemeral messages and threads.\n\n\nInstallation Documentation\n-----------\nPlease see the documentation [here](docs/installation.md) for details.\n\n\nContributing\n---------------\nIf you are interested in contributing to HubCommander, please review the [contributing documentation](docs/contributing.md).\n '
