Run `yarn` to fetch node packages.\n1. Run `yarn build` to build the core, the UI and the modules.\n1. Run `yarn start` to start the server.\n\n<details><summary><strong>Building Issues</strong></summary>\n<p>\n\nIf you encounter errors when building modules (timeout, random errors, etc), try the following:\n\n1.
