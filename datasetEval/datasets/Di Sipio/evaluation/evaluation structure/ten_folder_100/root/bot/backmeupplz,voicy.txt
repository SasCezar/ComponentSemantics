Please, feel free to fork, add features and create pull requests so that everybody (over 600 000 chats) can experience the features you've built.\n\nYou can also help by translating the bot to other languages or fixing some texts in existing languages by modifying the `locales` folder.\n\n# List of repositories\n\n- [Voicy](https://github.com/backmeupplz/voicy) \xe2\x80\x94 the main [@voicybot](https://t.me/voicybot) code\n- [Voicy payments](https://github.com/backmeupplz/voicy-payments) \xe2\x80\x94 payments service that used stripe to process payments for the Google Speech seconds of recognition; currently retired as the stats server for [voicybot.com](http://voicybot.com)\n- [Voicy landing](https://github.com/backmeupplz/voicy-landing) \xe2\x80\x94 [voicybot.com](http://voicybot.com) landing page\n\n# Installation and local launch\n\n1. Clone this repo: `git clone https://github.com/backmeupplz/voicy`\n2. Launch the [mongo database](https://www.mongodb.com/) locally\n3. Create `.env` file with the environment variables listed below\n4. Install ffmpeg on your machine\n5. Run `yarn install` in the root folder\n6. Run `yarn start`\n\n# Environment variables in `.env` file\n\n- `MONGO_URL` \xe2\x80\x94 Url for the mongo database used\n- `TOKEN` \xe2\x80\x94 Telegram bot token\n- `SALT` \xe2\x80\x94 Random salt to generate various encrypted stuff\n- `ADMIN_ID` \xe2\x80\x94 Chat id of the person who shall receive valuable logs\n- `WIT_LANGUAGES` \xe2\x80\x94 A map of language names to Wit.ai tokens\n\nExtra info is available in `.env.sample` file.\n\n# Continuous integration\n\nAny commit pushed to master gets deployed to [@voicybot](https://t.me/voicybot) via [CI Ninja](https://github.com/backmeupplz/ci-ninja).\n\n# License\n\nMIT \xe2\x80\x94 use for any purpose.
