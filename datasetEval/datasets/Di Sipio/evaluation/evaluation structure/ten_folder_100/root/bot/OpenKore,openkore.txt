[Download OpenKore](https://github.com/OpenKore/openkore/archive/master.zip) & extract it. Alternatively, you could press the **Windows Key + R**, type in ``cmd`` & enter. Run the following command in the cmd to clone.\n***Note: [Git](https://git-scm.com/) required.***\n```\ngit clone https://github.com/OpenKore/openkore.git\n```\n\n2. Configure OpenKore: [documentation](http://openkore.com/index.php/Category:Control).\n3. Run openkore.pl _(You can run start.exe or wxstart.exe if you use Windows)_.\n\n## F.A.Q.
