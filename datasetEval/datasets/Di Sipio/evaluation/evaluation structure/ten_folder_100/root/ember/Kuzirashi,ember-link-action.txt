b"Ember Link Action\n==============================================================================\n[![Build Status](https://travis-ci.org/Kuzirashi/ember-link-action.svg?branch=master)](https://travis-ci.org/Kuzirashi/ember-link-action) [![npm version](https://badge.fury.io/js/ember-link-action.svg)](https://badge.fury.io/js/ember-link-action) [![Ember Observer Score](http://emberobserver.com/badges/ember-link-action.svg)](http://emberobserver.com/addons/ember-link-action) [![Greenkeeper badge](https://badges.greenkeeper.io/Kuzirashi/ember-link-action.svg)](https://greenkeeper.io/)\n\nAn Ember CLI addon. Allows to combine `{{link-to}}` helper with firing an action.\n\nIt is useful when you want to fire an action at the same time when user transitions to other route by clicking `{{link-to}}`. **It is OK for SEO solution.**\n\nUsage\n------------------------------------------------------------------------------\n\nYou can pass closure action as `invokeAction` attribute of `{{link-to}}` component:\n\n``` hbs\n{{#link-to 'other-route' invokeAction=(action 'testAction')}}\n  Link to other route\n{{/link-to}}\n```\n\nTo pass parameters to action you can use:\n\n``` hbs\n{{#link-to 'other-route' invokeAction=(action 'testAction' param1 param2)}}\n  Link to other route\n{{/link-to}}\n```\n\nCompatibility\n------------------------------------------------------------------------------\n\n\n* Ember.js v2.12 or above\n* Ember CLI v2.12 or above\n* Node.js v6 or above\n\n\nAddon works with `>= 3.0`, `>= 2.0` versions of Ember. Tests confirm that it works as expected with release, beta and canary.\n\nFor `1.13.13` compatibility use tested version: `0.0.37`. In other cases - use the latest version.\n\nInstallation\n------------------------------------------------------------------------------\n\n```\nember install ember-link-action\n```\n\n\nContributing\n------------------------------------------------------------------------------\n\nSee the [Contributing](CONTRIBUTING.md) guide for details.\n\n\nLicense\n------------------------------------------------------------------------------\n\nThis project is licensed under the [MIT License](LICENSE.md).\n"
