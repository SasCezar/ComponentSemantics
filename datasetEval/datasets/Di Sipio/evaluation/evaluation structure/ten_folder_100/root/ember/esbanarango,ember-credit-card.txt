b'# Ember Credit card\n\n![Download count all time](https://img.shields.io/npm/dt/ember-credit-card.svg) [![Build Status](https://travis-ci.org/esbanarango/ember-credit-card.svg)](https://travis-ci.org/esbanarango/ember-credit-card) [![npm version](https://badge.fury.io/js/ember-credit-card.svg)](http://badge.fury.io/js/ember-credit-card) [![Ember Observer Score](http://emberobserver.com/badges/ember-credit-card.svg)](http://emberobserver.com/addons/ember-credit-card) [![Greenkeeper badge](https://badges.greenkeeper.io/esbanarango/ember-credit-card.svg)](https://greenkeeper.io/)\n\nThis is an Ember wrapper for the amazing [Card](https://github.com/jessepollak/card) project.\n\n## Installation\n\n`ember install ember-credit-card`\n\n## Usage\n\n![card](http://i.imgur.com/qG3TenO.gif)\n\n```handlebars\n<CreditCard/>\n```\n\n##### Default\n\nBy simply adding `{{credit-card}}` a basic form will be generated:\n\n```handlebars\n<div class=\'card-wrapper\'></div>\n<br/>\n<form id="ember-credit-card-form">\n  {{input type="text" name="number" value=number placeholder=\'Credit Card Number\'}}\n  {{input type="text" name="name" value=name placeholder=\'Name on Credit Card\'}}\n  {{input type="text" name="expiry" value=expiry}}\n  {{input type="text" name="cvc" value=cvc}}\n</form>\n```\n\n##### Specifying settings\n\nYou can also specify all the settings that [Card](https://github.com/jessepollak/card/blob/master/src/coffee/card.coffee#L63) supports.\n\n```handlebars\n<CreditCard\n  @form="#my-form"\n  @placeholders={{cardValues}}\n  @width={{150}}\n  @formatting={{false}}\n/>\n```\n\n##### With template\n\nThe default form and inputs used by default are so simple, so you can pass a template to use your own styled inputs.\n\n> Remeber to either use the default name inputs or specify them on the `{{credit-card}}` arguments\n\n```handlebars\n<CreditCard @form="#my-form">\n  <form id=\'my-form\' class=\'payment-info\'>\n    <div class="ui input">\n      <label>Credit Card Number:</label>\n      {{input type="text" value=creditCard.number name=\'number\' placeholder=\'Credit Card Number\'}}\n    </div>\n\n    <div class="ui input">\n      <label>Name on Credit Card:</label>\n      {{input type="text" value=creditCard.name name=\'name\' placeholder=\'Name on Credit Card\'}}\n    </div>\n\n    <div class="ui input">\n      <label>Expiration:</label>\n      {{input type="text" name=\'expiry\' value=creditCard.expiration placeholder=\'MM/YY\'}}\n    </div>\n\n    <div class="ui input">\n      <label>CVC:</label>\n      {{input type="text" name=\'cvc\' value=creditCard.cvc placeholder=\'CVC\'}}\n    </div>\n\n  </form>\n</CreditCard>\n```\n\n### Linting\n\n- `yarn lint:js`\n- `yarn lint:js --fix`\n\n## Running Tests\n\n- `ember test` \xe2\x80\x93 Runs the test suite on the current Ember version\n- `ember test --server` \xe2\x80\x93 Runs the test suite in "watch mode"\n- `ember try:each` \xe2\x80\x93 Runs the test suite against multiple Ember versions\n\n### Running the dummy application\n\n- `ember serve`\n- Visit the dummy application at [http://localhost:4200](http://localhost:4200).\n\n## License\n\nThis project is licensed under the [MIT
