b"# Ember CP Validations\n\n[![Build Status](https://travis-ci.org/offirgolan/ember-cp-validations.svg?branch=master)](https://travis-ci.org/offirgolan/ember-cp-validations)\n[![npm version](https://badge.fury.io/js/ember-cp-validations.svg)](http://badge.fury.io/js/ember-cp-validations)\n[![Download Total](https://img.shields.io/npm/dt/ember-cp-validations.svg)](http://badge.fury.io/js/ember-cp-validations)\n[![Ember Observer Score](http://emberobserver.com/badges/ember-cp-validations.svg)](http://emberobserver.com/addons/ember-cp-validations)\n\nAn EmberJS validation framework that is completely and utterly computed property based.\n\n## Features\n\n**No observers were used nor harmed while developing and testing this addon.**\n\n- Lazily computed validations\n- Ruby on rails inspired validators\n- Support for Ember Data Models, Objects, Components, Services, etc.\n- Support for nested models and objects\n- Synchronous and asynchronous support\n- Easily integrated with Ember Data's [DS.Errors](http://emberjs.com/api/data/classes/DS.Errors.html)\n- No observers. Seriously... there are none. Like absolutely zero....\n- Custom validators\n- I18n support\n- Debounceable validations\n- Warning validations\n\n[![Introduction to Ember CP Validations](https://cloud.githubusercontent.com/assets/2922250/21854491/ebda55b8-d7e8-11e6-8d13-00dff93be8d8.png)](https://embermap.com/video/ember-cp-validations)\n\nYou can also learn more by watching this Global Ember Meetup talk:\n\n[![Introduction to ember-cp-validations](https://i.vimeocdn.com/video/545445254.png?mw=1920&mh=1080&q=70)](https://vimeo.com/146857699)\n\n## Installation\n\n```shell\nember install ember-cp-validations\n```\n\n## Upgrading to 4.x\n\nIf you are upgrading from 3.x to 4.x, please checkout the [upgrading documentation](UPGRADING.md).\n\n## Helpful Links\n\n- ### [Live Demo](http://offirgolan.github.io/ember-cp-validations)\n\n- ### Documentation\n\n  - [4.x](http://offirgolan.github.io/ember-cp-validations/docs)\n  - [3.x](https://rawgit.com/offirgolan/ember-cp-validations/c4123c983e54f24dd790ffa1bad66cfdf2f47ec6/docs/index.html)\n\n- ### [Changelog](CHANGELOG.md)\n\n## Looking for help?\n\n- If it is a bug [please open an issue on GitHub](http://github.com/offirgolan/ember-cp-validations/issues).\n- Ask a question in the `#e-cp-validations` channel at the [Ember.js Community Slack](https://embercommunity.slack.com)\n"
