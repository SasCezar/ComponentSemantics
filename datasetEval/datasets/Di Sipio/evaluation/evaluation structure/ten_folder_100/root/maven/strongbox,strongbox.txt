With a well-developed architecture, it provides native \nimplementations for various layout formats, such as [Maven][docs-maven], [NPM][docs-npm], [NuGet][docs-nuget], and [Raw][docs-raw].\n\nAll of the implemented layout formats (a.k.a. "[layout providers][docs-providers]") are written natively in Java. \nOur goal is to implement a universal repository manager that can host and serve artifacts in every mainstream format.\n\nStrongbox has a search engine and an [Artifact Query Language][docs-aql].\n\n# What\'s in the works (Q3/2019)\n\n* Web UI\n* NuGet (protocol v2) layout support finalization
