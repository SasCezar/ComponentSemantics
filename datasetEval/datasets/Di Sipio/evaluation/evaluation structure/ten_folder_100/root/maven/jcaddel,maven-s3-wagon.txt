b"Maven S3 Wagon\r\n-------\r\n\r\nThis wagon enables communication between Maven and Amazon S3.\r\n\r\npom's with a reference to this wagon can publish build artifacts (.jar's, .war's, etc) to S3.\r\n\r\nWhen uploading the contents of a directory, API calls to S3 are multi-threaded.\r\n\r\nThis allows directories with a lot of content (eg when invoking mvn site-deploy) to be published very quickly\r\n\r\nCheck [Maven Central](http://search.maven.org/#search|gav|1|g%3A%22org.kuali.maven.wagons%22%20AND%20a%3A%22maven-s3-wagon%22) for the latest version\r\n\r\nDocumentation\r\n-------\r\n\r\n[Usage](https://github.com/jcaddel/maven-s3-wagon/wiki/Usage)\r\n\r\n[Authentication](https://github.com/jcaddel/maven-s3-wagon/wiki/Authentication)\r\n\r\n[Permissions](https://github.com/jcaddel/maven-s3-wagon/wiki/Permissions)\r\n\r\n[Maven generated site](http://site.kuali.org/maven/wagons/maven-s3-wagon/latest/)\r\n\r\n"
