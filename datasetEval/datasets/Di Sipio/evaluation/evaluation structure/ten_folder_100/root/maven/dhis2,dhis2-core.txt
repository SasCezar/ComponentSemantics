b'\r\n# DHIS 2\r\n\r\nDHIS 2 is a flexible information system for data capture, management, validation, analytics and visualization. It allows for data capture through clients ranging from Web browsers, Android devices, Java feature phones and SMS. DHIS 2 features data visualization apps for dashboards, pivot tables, charting and GIS. It provides metadata management and configuration. The data model and services are exposed through a RESTful Web API.\r\n\r\n## Overview\r\n\r\nIssues can be reported and browsed in [JIRA](https://jira.dhis2.org).\r\n\r\nFor documentation please visit the [doc page](https://www.dhis2.org/documentation/).\r\n\r\nYou can download pre-built WAR files from the [continuous integration server](http://ci.dhis2.org/).\r\n\r\nYou can explore various demos of DHIS 2 in the [play environment](https://play.dhis2.org/).\r\n\r\nFor support and discussions visit the [community forum](https://community.dhis2.org/).\r\n\r\nFor general info please visit the [project web page](https://www.dhis2.org/).\r\n\r\nFor software development guides please visit the [project wiki](http://dhis2.github.io/).\r\n\r\nThe software is open source and released under the [BSD license](https://opensource.org/licenses/BSD-2-Clause).\r\n\r\n## Build process\r\n\r\nThis repository contains the source code for the server-side component of DHIS 2, which is developed in [Java](https://www.java.com/en/) and built with [Maven](https://maven.apache.org/). \r\n\r\nTo build it you must first install the root POM file, navigate to the dhis-web directory and then build the web POM file.\r\n\r\nCheck [contributing](https://github.com/dhis2/dhis2-core/blob/master/CONTRIBUTING.md) for the procedure to make it run locally.\r\n\r\n[![Build Status](https://travis-ci.com/dhis2/dhis2-core.svg?branch=master)](https://travis-ci.org/dhis2/dhis2-core)\r\n'
