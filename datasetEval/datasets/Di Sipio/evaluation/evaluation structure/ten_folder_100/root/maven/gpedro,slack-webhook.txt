b'slack-webhook\n=============\n\nSlack WebHook Integration for Java\n\n# Basic Examples\n\n```java\n\n// Send simple message\nSlackApi api = new SlackApi("https://hooks.slack.com/services/id_1/id_2/token");\napi.call(new SlackMessage("my message"));\n\n// Send simple message with custom name\nSlackApi api = new SlackApi("https://hooks.slack.com/services/id_1/id_2/token");\napi.call(new SlackMessage("Mafagafo", "my message"));\n\n// Send simple message in different room\nSlackApi api = new SlackApi("https://hooks.slack.com/services/id_1/id_2/token");\napi.call(new SlackMessage("#general", null, "my message"));\n\n// Send simple message in different room with custom name\nSlackApi api = new SlackApi("https://hooks.slack.com/services/id_1/id_2/token");\napi.call(new SlackMessage("#general", "Mafagafo", "my message"));\n\n// Send simple message in different room with custom name and tag someone in the message so he will get notified\nSlackApi api = new SlackApi("https://hooks.slack.com/services/id_1/id_2/token");\napi.call(new SlackMessage("#general", "Mafagafo", "Hi @gpedro : your API rocks").setLinkNames(true));\n\n\n```\n\n# Installation\nAdd the following dependency in __pom.xml__\n```xml\n<dependency>\n  <groupId>net.gpedro.integrations.slack</groupId>\n  <artifactId>slack-webhook</artifactId>\n  <version>1.4.0</version>\n</dependency>\n```\n\n# Configuration\n\n1. Go to *your_team.slack.com/services/new*\n2. Search for *Incoming WebHook* and click in `Add`\n3. Choose Channel to Post and press `Add Incoming WebHooks Integration`\n4. Into *Setup Instructions*, you\'ve a WebHook URL. He is the argument [you must pass the constructor](/src/main/java/net/gpedro/integrations/slack/SlackApi.java#L18).
