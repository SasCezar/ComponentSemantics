It is required to have Node.js with version 8.5.0 or higher. To see what version of Node.js is installed on your machine type the following command in the terminal:\n\n```\nnode -v\n```\n\n2. If you do not have installed Node.js in your machine then go to [this link](https://nodejs.org/en/download/) in order to install node.\n\n## How to use\n\n1. Clone this repository.\n\n```\ngit clone https://github.com/salomonelli/best-resume-ever.git\n```\n\n2. Go to the cloned directory (e.g. `cd best-resume-ever`).\n\n3. Run `npm install`.\n\n4. Customize your resume in the `resume/` directory: edit your data `data.yml` and replace the default profile-picture `id.jpg` with your picture. Rename your picture as `id.jpg` and copy it in the `resume/` directory.\n\n5. Preview resumes with `npm run dev`. The command will start a server instance and listen on port 8080.  Open (http://localhost:8080/) in your browser. The page will show some resume previews. To see the preview of your resume, with your picture and data, click on one layout that you like and the resume will be opened in the same window.\n\n![Resume previews](/readme-images/resumePreviews.png)\n\n\n6. Export your resume as pdf by running the command `npm run export`. In order to avoid errors due to the concurrency of two  `npm run` commands, stop the execution of the previus `npm run dev` and then type the export command.\n\nAll resumes will be exported to the `pdf/` folder.\n\n<br>\n\n## Creating and Updating Templates\n\nPlease read the <a href="DEVELOPER.md">developer docs</a> on how to create or update templates.\n\n<br>\n\n## Contribute\n\nFeel free to add your own templates, language supports, fix bugs or improve the docs. Any kind of help is appreciated!
