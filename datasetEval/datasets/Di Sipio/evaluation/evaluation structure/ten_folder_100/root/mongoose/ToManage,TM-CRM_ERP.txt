Node.js 6.x\n - install MongoDB 3.2\n\n```shell\ncurl -sL https://deb.nodesource.com/setup_6.x | sudo bash -\napt-get install nodejs\n```\n\n```shell\ngit clone git@github.com:ToManage/framework.git\n```\n\n```shell\nnpm install\n```\n\nUsing demo mongoDB database from dump directory\n\n```shell\ncp config.sample config\n```\n\nA demo database is in dump directory\n\nEdit and replace demo name database in config file\n\n## Start\n\n```shell\nnode debug.js\n```\n\n## Run node.js service with systemd\n\nCreate /etc/systemd/system/nodeserver.service\n\n```shell\n[Unit]\nDescription=ToManage ERP\nDocumentation=https://www.tomanage.fr\nAfter=network.target\nRequires=mongodb.service\n\n[Service]\nEnvironment=NODE_PORT=8000\nType=simple\nUser=root\nGroup=root\n# Output to syslog\nStandardOutput=syslog\nStandardError=syslog\nSyslogIdentifier=nodeserver\nExecStart=/usr/bin/node /path/to/tomanage/debug.js\nWorkingDirectory=/path/to/tomanage\nRestart=always\nRestartSec=10\n\n[Install]\nWantedBy=multi-user.target\n```\nEnable the service\n\n```shell\nsystemctl enable nodeserver.service\nCreated symlink from /etc/systemd/system/multi-user.target.wants/nodeserver.service to /etc/systemd/system/nodeserver.service.\n```\nStart the service\n\n```shell\nsystemctl start nodeserver.service\n```\n\nDemo authentication : admin/admin\n\nGood coding :)\n\n\nFollow us :\n<p><a href="https://twitter.com/ToManage_js" target="_blank">On Twitter</a></p>\n<p><a
