b'# Aminal - A Modern Terminal Emulator\n\n[![Travis Build Status](https://travis-ci.org/liamg/aminal.svg?branch=master)](https://travis-ci.org/liamg/aminal)\n[![GoReportCard](https://goreportcard.com/badge/github.com/liamg/aminal)](https://goreportcard.com/report/github.com/liamg/aminal)\n[![Github Release](https://img.shields.io/github/release/liamg/aminal.svg)](https://github.com/liamg/aminal/releases)\n[![Slack](https://img.shields.io/badge/slack-%23aminal-%23ffcc00.svg)](http://gophers.slack.com/messages/aminal)\n![License](https://img.shields.io/github/license/liamg/aminal.svg)\n\nAminal is a modern terminal emulator for Mac/Linux/Windows implemented in Golang and utilising OpenGL. \n\n![Demo GIF](demo.gif)\n\nThe project is experimental at the moment, so you probably won\'t want to rely on Aminal as your main terminal for a while.\n\nEnsure you have your latest graphics card drivers installed before use.\n\n## Features\n\n- Unicode support\n- OpenGL rendering\n- Customisation options\n- True colour support\n- Support for common ANSI escape sequences a la xterm\n- Scrollback buffer\n- Clipboard access\n- Clickable URLs\n- Multi platform support (Windows, Linux, OSX)\n- Sixel support\n- Hints/overlays\n- Built-in patched fonts for powerline\n- Retina display support\n\n## Installation\n\n### MacOS\n\n```\nbrew tap liamg/aminal\nbrew install aminal\n```\n\n### Windows\n\nYou can run Aminal with a binary from the [releases](https://github.com/liamg/aminal/releases) page.\n\nDev environment setup instructions are available [here](windows.md).\n\n### Prebuilt Binaries\n\nPrebuilt binaries are available for Linux, OSX and Windows on the [releases](https://github.com/liamg/aminal/releases) page. \n\nDownload the binary and `sudo cp aminal-* /usr/local/bin/aminal && chmod +x /usr/local/bin/aminal`.\n\n### Install with Go\n\n```\ngo get -u github.com/liamg/aminal\n```\n\n## Build \n\n### Dependencies\n\n- On macOS, you need Xcode or Command Line Tools for Xcode (`xcode-select --install`) for required headers and libraries.\n- On Ubuntu/Debian-like Linux distributions, you need `libgl1-mesa-dev xorg-dev`.\n- On CentOS/Fedora-like Linux distributions, you need `libX11-devel libXcursor-devel libXrandr-devel libXinerama-devel mesa-libGL-devel libXi-devel`.\n\n### Building Locally\n\nThere are various make targets available, the most obvious being:\n\n```\nmake test\nmake build\nmake install\n```\n\nAs long as you have your `GOBIN` environment variable set up properly (and in `PATH`), you should be able to run `aminal`.\n\n## Keyboard/Mouse Shortcuts\n\n| Operation            | Key(s)               |\n| -------------------- | -------------------- |\n| Select text          | click + drag         |\n| Select word          | double click         |\n| Select line          | triple click         |\n| Copy                 | `ctrl + shift + c` (Mac: `super + c`) |\n| Paste                | `ctrl + shift + v` (Mac: `super + v`) |\n| Search online for selected text | `ctrl + shift + g` (Mac: `super + g`) |\n| Toggle debug display | `ctrl + shift + d` (Mac: `super + d`) |\n| Toggle slomo         | `ctrl + shift + ;` (Mac: `super + ;`) |\n| Report bug in aminal | `ctrl + shift + r` (Mac: `super + r`) |\n\n## Configuration\n\nAminal looks for a config file in the following places, and stops when it finds one:\n\n* `$XDG_CONFIG_HOME/aminal/config.toml`\n* `$HOME/.config/aminal/config.toml`\n* `$HOME/.aminal.toml`\n\nNote that on Windows Aminal uses `%USERPROFILE%` environment variable instead of `$HOME`.\n\nIt will write a config file to whichever of those directories exists (preferring the top of the list) the first time it runs, if one doesn\'t already exist.\n\nYou can ignore the config and use defaults by specifying `--ignore-config` as a CLI flag.\n\n### Config File\n\n```toml\ndebug = false               # Enable debug logging to stdout. Defaults to false.\nslomo = false               # Enable slow motion output mode, useful for debugging shells/terminal GUI apps etc. Defaults to false.\nshell = "/bin/bash"         # The shell to run for the terminal session. Defaults to the users shell.\nsearch_url = "https://www.google.com/search?q=$QUERY" # The search engine to use for the "search selected text" action. Defaults to google. Set this to your own search url using $QUERY as the keywords to replace when searching.\nmax_lines = 1000            # Maximum number of lines in the terminal buffer.\ncopy_and_paste_with_mouse = true # Text selected with the mouse is copied to the clipboard on end selection, and is pasted on right mouse button click.\ndpi-scale = 0.0             # Override DPI scale. Defaults to 0.0 (let Aminal determine the DPI scale itself).\n\n[colours]\n  cursor        = "#e8dfd6" \n  foreground    = "#e8dfd6" \n  background    = "#021b21" \n  black         = "#032c36" \n  red           = "#c2454e" \n  green         = "#7cbf9e"\n  yellow        = "#8a7a63"\n  blue          = "#065f73"\n  magenta       = "#ff5879"\n  cyan          = "#44b5b1"\n  light_grey    = "#f2f1b9"\n  dark_grey     = "#3e4360"\n  light_red     = "#ef5847"\n  light_green   = "#a2db91"\n  light_yellow  = "#beb090"\n  light_blue    = "#61778d"\n  light_magenta = "#ff99a1"\n  light_cyan    = "#9ed9d8"\n  white         = "#f6f6c9"\n  selection     = "#333366" # Mouse selection background colour\n\n[keys]\n  copy      = "ctrl + shift + c"    # Copy highlighted text to system clipboard\n  paste     = "ctrl + shift + v"    # Paste text from system clipboard\n  debug     = "ctrl + shift + d"    # Toggle debug panel overlay\n  google    = "ctrl + shift + g"    # Google selected text\n  report    = "ctrl + shift + r"    # Send bug report\n  slomo     = "ctrl + shift + ;"    # Toggle slow motion output mode (useful for debugging)\n```\n\n### CLI Flags\n\n| Flag              | Description                                                                                                                   |\n| ----------------- | ----------------------------------------------------------------------------------------------------------------------------- |\n| `--debug`         | Enable debug mode, with debug logging and debug info terminal overlay.\n| `--slomo`         | Enable slomo mode, delay the handling of each incoming byte (or escape sequence) from the pty by 100ms. Useful for debugging.\n| `--shell [shell]` | Use the specified shell program instead of the user\'s usual one. \n| `--version`       | Show the version of aminal and exit.\n\n# Contributors\n\n[![](https://sourcerer.io/fame/liamg/liamg/aminal/images/0)](https://sourcerer.io/fame/liamg/liamg/aminal/links/0)[![](https://sourcerer.io/fame/liamg/liamg/aminal/images/1)](https://sourcerer.io/fame/liamg/liamg/aminal/links/1)[![](https://sourcerer.io/fame/liamg/liamg/aminal/images/2)](https://sourcerer.io/fame/liamg/liamg/aminal/links/2)[![](https://sourcerer.io/fame/liamg/liamg/aminal/images/3)](https://sourcerer.io/fame/liamg/liamg/aminal/links/3)[![](https://sourcerer.io/fame/liamg/liamg/aminal/images/4)](https://sourcerer.io/fame/liamg/liamg/aminal/links/4)[![](https://sourcerer.io/fame/liamg/liamg/aminal/images/5)](https://sourcerer.io/fame/liamg/liamg/aminal/links/5)[![](https://sourcerer.io/fame/liamg/liamg/aminal/images/6)](https://sourcerer.io/fame/liamg/liamg/aminal/links/6)[![](https://sourcerer.io/fame/liamg/liamg/aminal/images/7)](https://sourcerer.io/fame/liamg/liamg/aminal/links/7)\n'
