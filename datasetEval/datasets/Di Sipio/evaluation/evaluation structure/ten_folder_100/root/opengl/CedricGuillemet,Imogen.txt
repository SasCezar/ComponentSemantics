b'# Imogen\nGPU/CPU Texture Generator\n\nGPU Texture generator using dear imgui for UI. Not production ready and a bit messy but really fun to code. This is an hybrid project that can run natively or on the web (thanks to emscripten).\nBasically, add GPU and CPU nodes in a graph to manipulate and generate images.\nA web build is available here : http://skaven.fr/imogen/\n![Image of Imogen 0.13 Web Edition](https://i.imgur.com/ahkqR7v.png)\nCurrently nodes can be written in GLSL or C or Python.\n\n\n![Image of Imogen 0.9](https://i.imgur.com/sQfO5Br.png)\n![Image of Imogen 0.9](https://i.imgur.com/jQbx2Yu.png)\n\nUse CMake and VisualStudio to build it. Windows and web builds are available.\n\nWeb Edition limitations:\n- no threaded jobs\n- no C/Python nodes\n- no Python plugins\n- no file load/save\n\nFeatures:\n- Node based texture editing\n- material library browser\n- edit/change node shaders inside the app\n- bake textures to .png, .jpg, .tga, .bmp, .hdr, mp4\n- PBR preview\n- timeline for parameters animation\n\nCurrently implemented nodes\n- circle and square generator\n- sine generator\n- checker transform\n- transform\n- Mul/Add\n- smoothstep\n- pixelize\n- blur\n- normal map from height map\n- sphere/plan previewer\n- Hexagon\n- Mul-Add colors\n- Blend (add, mul, min, max)\n- Invert color\n- Circle Splatter\n- Ramp\n- Tile\n- Polar coordinates\n- ...\n\nCheck the project page for roadmap.\n\n-----------\nThis software uses the following (awesome) projects:\n\nDear ImGui - Omar Cornut https://github.com/ocornut/imgui\n\nstb_image, stb_image_write - Sean T.
