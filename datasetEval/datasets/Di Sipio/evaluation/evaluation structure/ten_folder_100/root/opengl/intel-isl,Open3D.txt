b'<p align="center">\n<img src="docs/_static/open3d_logo_horizontal.png" width="320" />\n</p>\n\n# Open3D: A Modern Library for 3D Data Processing\n\n<h4>\n    <a href="http://www.open3d.org">open3d.org</a> |\n    <a href="http://www.open3d.org/docs">Documentation</a> |\n    <a href="http://www.open3d.org/docs/release/getting_started.html">Quick Start</a> |\n    <a href="http://www.open3d.org/docs/release/compilation.html">Build from Source</a> |\n    <a href="http://www.open3d.org/docs/release/index.html#python-api-index">Python API</a> |\n    <a href="http://www.open3d.org/docs/release/cpp_api/index.html">C++ API</a> |\n    <a href="http://www.open3d.org/docs/release/contribute.html">Contribute</a> |\n    <a href="https://www.youtube.com/watch?v=I3UjXlA4IsU">Demo</a>\n</h4>\n\nOpen3D is an open-source library that supports rapid development of software that deals with 3D data. The Open3D frontend exposes a set of carefully selected data structures and algorithms in both C++ and Python. The backend is highly optimized and is set up for parallelization. We welcome contributions from the open-source community.\n\n## Core features\n\n* 3D data structures\n* 3D data processing algorithms\n* Scene reconstruction\n* Surface alignment\n* 3D visualization\n* Python binding\n\n## Supported OSes and compilers\n\n* Ubuntu 16.04 or newer: GCC 5.x or newer [![Build Status](https://travis-ci.org/intel-isl/Open3D.svg?branch=master)](https://travis-ci.org/intel-isl/Open3D)\n* macOS: XCode 8.0 or newer [![Build Status](https://travis-ci.org/intel-isl/Open3D.svg?branch=master)](https://travis-ci.org/intel-isl/Open3D)\n* Windows: Visual Studio 2017 or newer [![Build status](https://ci.appveyor.com/api/projects/status/3hasjo041lv6srsi/branch/master?svg=true)](https://ci.appveyor.com/project/yxlao/open3d/branch/master)\n\n## Citation\nPlease cite [our work](https://arxiv.org/abs/1801.09847) if you use Open3D.\n\n```\n@article{Zhou2018,\n\tauthor    = {Qian-Yi Zhou and Jaesik Park and Vladlen Koltun},\n\ttitle     = {{Open3D}: {A} Modern Library for {3D} Data Processing},\n\tjournal   = {arXiv:1801.09847},\n\tyear      = {2018},\n}\n```\n'
