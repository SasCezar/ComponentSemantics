b'# Micrometer Application Metrics\n\n[![Build Status](https://circleci.com/gh/micrometer-metrics/micrometer.svg?style=shield)](https://circleci.com/gh/micrometer-metrics/micrometer)\n[![Apache 2.0](https://img.shields.io/github/license/micrometer-metrics/micrometer.svg)](https://www.apache.org/licenses/LICENSE-2.0)\n[![Maven Central](https://img.shields.io/maven-central/v/io.micrometer/micrometer-core.svg)](https://mvnrepository.com/artifact/io.micrometer/micrometer-core)\n[![Javadocs](https://www.javadoc.io/badge/io.micrometer/micrometer-core.svg)](https://www.javadoc.io/doc/io.micrometer/micrometer-core)\n\nAn application metrics facade for the most popular monitoring tools. Instrument your code with dimensional metrics with a\nvendor neutral interface and decide on the monitoring backend at the last minute.\n\nMore info and the user manual are available on [micrometer.io](https://micrometer.io).\n\nMicrometer is the instrumentation library underpinning Spring Boot 2.0\'s metrics collection.\n\n## Long-term support versions\n\nSee [Micrometer\'s support policy](https://micrometer.io/docs/support) for more details about long-term support (LTS) versus non-LTS versions.\n\n| Minor version line | LTS | Final patch |\n| ------------------ | --- | ----------- |\n| `1.0.x`            | Yes | `1.0.11`    |\n| `1.1.x`            | Yes |  |\n| `1.2.x`            | No  | `1.2.2` |\n| `1.3.x`            | Yes |  |\n\n## Join the discussion\n\nJoin the [Micrometer Slack](http://slack.micrometer.io) to share your questions, concerns, and feature requests.\n\n## Snapshot builds\n\nSnapshots are published to `repo.spring.io` for every successful build on the `master` branch and maintenance branches.\n\nTo use:\n\n```groovy\nrepositories {\n    maven { url \'https://repo.spring.io/libs-snapshot\' }\n}\n\ndependencies {\n    compile \'io.micrometer:micrometer-core:latest.integration\'\n}\n```\n\n## Building Locally\n\nRun `./gradlew clean pTML` to publish a Maven-style snapshot to your Maven local repo. To consume:\n\n```groovy\nrepositories {\n    mavenLocal()\n}\n\ndependencies {\n    compile \'io.micrometer:micrometer-core:latest.integration\'\n}\n```\n\nThe build automatically calculates the "next" version for you when publishing snapshots.\n\n## Documentation\n\nThe reference documentation is managed in [a separate GitHub repository](https://github.com/micrometer-metrics/micrometer-docs).\n\n-------------------------------------\n_Licensed under [Apache Software License 2.0](https://www.apache.org/licenses/LICENSE-2.0)_\n\n_Sponsored by [Pivotal](https://pivotal.io)_\n'
