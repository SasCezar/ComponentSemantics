b'[![Gitter chat](https://badges.gitter.im/smashing.svg)](https://gitter.im/Smashing)\n\n# [Smashing](https://github.com/Smashing/smashing/wiki)\n\nSmashing, the spiritual successor to [Dashing](https://github.com/Shopify/dashing), is a Sinatra based framework that lets you build excellent dashboards. It looks especially great on TVs.\n\n## Community\n\nFeel free to submit issues for bugs, new features, and enhancements in [GitHub](https://github.com/Smashing/smashing/issues). For more general questions, or help with widgets, please use the [gitter chatroom](https://gitter.im/Smashing).\n\n## Installation\n\n```bash\n# Install bundler\n$ gem install bundler\n# Install smashing\n$ gem install smashing\n# Create a new project\n$ smashing new my-project\n# Change Directory into the project\n$ cd my-project\n# Install the bundle of project specific gems\n$ bundle\n# Start the example dashboard!\n$ smashing start\n```\n\n[Check out our wiki](https://github.com/Smashing/smashing/wiki).\n\nNote: This is a fork of the Dashing project, which is no longer being maintained. Read about that [here](https://github.com/Shopify/dashing/issues/711).\n\n## License\nDistributed under the [MIT license](MIT-LICENSE).\n'
