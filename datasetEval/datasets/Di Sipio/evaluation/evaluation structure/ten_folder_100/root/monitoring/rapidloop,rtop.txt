b"\n# rtop\n\n[![Join the chat at https://gitter.im/rapidloop/rtop](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/rapidloop/rtop?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)\n\n`rtop` is a remote system monitor. It connects over SSH to a remote system\nand displays vital system metrics (CPU, disk, memory, network). No special\nsoftware is needed on the remote system, other than an SSH server and\nworking credentials.\n\nOnly Linux systems can be monitored, and most modern distros will work.\n\n`rtop` is MIT-licensed and can be used anywhere with attribution.\n\n*`rtop`'s [home page](http://www.rtop-monitor.org/) has more information\nand screenshots!*\n\n## build\n\n`rtop` is written in [go](http://golang.org/), and requires Go version 1.2\nor higher. To build, `go get` it:\n\n    go get github.com/rapidloop/rtop\n\nYou should find the binary `rtop` under `$GOPATH/bin` when the command\ncompletes. There are no runtime dependencies or configuration needed.\n\n## contribute\n\nPull requests welcome. Keep it simple.\n\n## changelog\n* 9-May-2015: first public release\n"
