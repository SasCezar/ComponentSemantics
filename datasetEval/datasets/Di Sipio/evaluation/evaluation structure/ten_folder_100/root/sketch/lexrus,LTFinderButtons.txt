b"## LTFinderButtons\nMy Finder buttons collection for macOS.\n\n![ltfinderbuttons 2x](https://user-images.githubusercontent.com/219689/46614141-a4097580-cb47-11e8-8540-795da4cd07e0.png)\n\n## Buttons\n- [x] iTerm2 (2.9+)\n- [x] Sublime Text 3\n- [x] VSCode\n- [x] Terminal\n- [x] TextMate\n- [x] Atom\n- [x] Brackets\n- [x] CotEditor\n- [x] BBEdit\n\n## Usage\nDownload or `git clone` the buttons in a proper place. Hold on command button and drag each one into your Finder toolbar.\n\n## Contribute a button\n0. Fork this repo.\n1. Duplicate iTerm2FinderButton.app and rename it to MyShinnyFinderButton.app.\n2. Open `MyShinnyFinderButton.app/Contents/Resources/Scripts/main.scpt` with Script Editor then you gonna know what to do.\n3. Open LTFinderButtons.sketch and draw your button.\n4. Export the `iconset` folders from Sketch.\n5. Convert images into icns files one by one (e.g. `iconutil -c icns MyShinnyFinderButton.iconset`)\n6. Replace `MyShinnyFinderButton.app/Contents/Resources/droplet.icns` with your icns.\n7. Send a pull request.\n8. *Please write scripts in JavaScript or Swift, I'll reject AppleScript PRs ^_^.\n\n\n## Enhanced TerminalButton\nThere's a fork which refined the main script of TerminalButton with Apple Script:\nhttps://github.com/ashfinal/LTFinderButtons\nIt fixed some issues of TerminalFinderbutton. But I rejected the [PR](https://github.com/lexrus/LTFinderButtons/pull/17) since Apple Script was not my taste for a long time. Please have a try once you face any problem with TerminalButton.\n\n## Contacts\n[Lex Tang](https://github.com/lexrus/) ([@lexrus on Twitter](https://twitter.com/lexrus/))\n\n## License\nThis project is distributed under the terms and conditions of the MIT license.\n"
