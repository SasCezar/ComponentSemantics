Save your work on Sketch.  \n2. Select the LAYERGROUPs your want to export on the layer list(group it if the layer is not a layergroup), Run `Plugins` -> `Sketch2AE` -> `Generate ExportSlices`.  \n3.  Run `Plugins` -> `Sketch2AE` -> `Export Artboards`, then on the popup panel set the composition length and `FPS`, pick artboards you want to export. Hit `OK` and choose a location to save the generated `.jsx` script file and `slice` folder.  \n4.  In After Effect\xef\xbc\x8crun `File` -> `Script` -> `Run Script File`, choose the `.jsx` file, it will generate compositions automaticly.  \n````\n**NOTE: Keep the ".jsx" file and "slice" folder under a same parent folder if you want to move it.\n````  \nBack to Sketch, you can run Plugins -> Sketch2AE -> Clear ExportSlices to clear exsiting ExportSlices.  \n\n<img src="https://raw.githubusercontent.com/bigxixi/ReadMe-Resources/master/Sketch2AE/step1.gif" width="50%" height="50%">  \n\n<img src="https://raw.githubusercontent.com/bigxixi/ReadMe-Resources/master/Sketch2AE/step2.gif" width="50%" height="50%">  \n\n<img src="https://raw.githubusercontent.com/bigxixi/ReadMe-Resources/master/Sketch2AE/step3.gif" width="50%" height="50%">  \n## Known Issues:\n * Can not export vactor shapes, editable texts, layer styles.All flatten.  \n * By default it generate 1x slices.
