b"# sketch-module-web-view\n\nA Sketch module for creating a complex UI with a webview. The API is mimicking the [BrowserWindow](https://electronjs.org/docs/api/browser-window) API of Electron.\n\n## Installation\n\nTo use this module in your Sketch plugin you need a bundler utility like [skpm](https://github.com/skpm/skpm) and add it as a dependency:\n\n```bash\nnpm install -S sketch-module-web-view\n```\n\nYou can also use the [with-webview](https://github.com/skpm/with-webview) skpm template to have a solid base to start your project with a webview:\n\n```bash\nskpm create my-plugin-name --template=skpm/with-webview\n```\n\n_The version 2.x is only compatible with Sketch >= 51. If you need compatibility with previous versions of Sketch, use the version 1.x_\n\n## Usage\n\n```js\nimport BrowserWindow from 'sketch-module-web-view'\n\nexport default function() {\n  const options = {\n    identifier: 'unique.id',\n  }\n\n  const browserWindow = new BrowserWindow(options)\n\n  browserWindow.loadURL(require('./my-screen.html'))\n}\n```\n\n## Documentation\n\n- [Communicating between the Plugin and the WebView](/docs/communication-plugin-webview.md)\n- [Frameless-window](/docs/frameless-window.md)\n- [Opening links in browser](/docs/opening-links-in-browser.md)\n\n## API References\n\n- [Browser window](/docs/browser-window.md)\n- [Web Contents](/docs/web-contents.md)\n\n## License\n\nMIT\n"
