b'![](android_res_export.png)\n\n**[\xe4\xb8\xad\xe6\x96\x87\xe8\xaf\xb4\xe6\x98\x8e](https://github.com/Ashung/Android_Res_Export/blob/master/README_zh.md)**\n\nExport Android resources in Sketch.\n\n## Features\n\n- Export PNG or WebP assets into multiple sizes.\n- Preview and export nine-patch assets into multiple sizes.\n- Export vector drawable asset, support gradient and mask.\n- Export app launcher icon for Android 8 and older version.\n- View and export shape drawable XML code from selected layer.\n- View and export color resource XML code from selected layers.\n\n## Installation\n\n### Install plugin\n\n- Search "Android Res Export" from [Sketch Runner](http://sketchrunner.com/).\n- Download [master.zip](https://github.com/Ashung/Android_Res_Export/archive/master.zip) and unzip, then double-click "Android_Res_Export.sketchplugin" to install.\n\n## How it Works\n\nDesign at MDPI (1x) size, NOT support for other sizes.\n\nYou need to create new asset from selected layers, by run "Plugins" - "Android Res Export" - "New ...", then run "Export ..." to export them. while you have selected layers, the export feature will only export the layer which is selected, and deselected all to export all assets in current document.\n\nFor export bitmap and nine-patch assets, you can choose what dpi will exported in "Android Res Export" - "Preferences" dialog.\n\n**For export vector drawable, you need to install [Node.js](https://nodejs.org/en/) (> 8.0) and [svg2vectordrawable](https://github.com/Ashung/svg2vectordrawable) (2.6.5+)**.\n\n## License\n\nMIT\n\n## Donate\n\n[Buy me a coffee](https://www.buymeacoffee.com/ashung) or donate [$5.00](https://www.paypal.me/ashung/5) [$10.00](https://www.paypal.me/ashung/10) via PayPal.\n'
