b'# Artboard Tricks plugin for Sketch\n\nContains a set of artboard tricks for Sketch. See below for more.\n\n# Installation\n\n* [Download the latest release](https://github.com/romannurik/Sketch-ArtboardTricks/releases/latest) and double-click the `.sketchplugin` file to install.\n\n* Or you can install the plugin with [Sketch Runner](http://bit.ly/SketchRunnerWebsite).\n\n  <a href="http://bit.ly/SketchRunnerWebsite"><img src="http://bit.ly/RunnerBadgeBlue" width="160"></a>\n\n# The commands\n\n## Rearrange Artboards Into Grid\n\nAutomatically rearranges your artboards into rows containing an arbitrary number of columns.\nThis is useful for organizing your artboards into distinct "flows", where each row corresponds to a flow.\n\n![Demo Screencap](https://raw.githubusercontent.com/romannurik/Sketch-ArtboardTricks/master/art/rearrange.gif)\n\nRows are 400px apart and columns are 100px apart by default (configurable per page).\n\nThe command also re-orders the list of artboards in the sidenav according to its position. \n\nNote that the arrangement is based on the current position of each artboard.\n\n## Prefix Artboard Names With Numbers\n\nPrefixes each artboard in your page with a number such as 102, 300, 517, etc. based on X and Y position.\nThis is designed to be used with the "Rearrange Artboards Into Grid" command. The first digit is the row,\nthe second 2 digits are the column, like so:\n\n```\n100 101 102\n200 201\n300 301 302 303 304\n```\n\n## Select Artboards Containing Selection\n\nSelects the artboard(s) that contain your selection. Useful if you want to select multiple artboards quickly\n(click and drag to select a bunch of layers, then run this script to select the container artboards).\n'
