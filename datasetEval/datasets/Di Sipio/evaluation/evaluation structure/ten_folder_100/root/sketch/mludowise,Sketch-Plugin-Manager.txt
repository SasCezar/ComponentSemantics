b"Sketch Plugin Manager\n=====================\n\nSketch Plugin Manager uses the power of Git to keep all of your installed plugins up to date.\n\n### Supported Operating Systems\nSketch Plugin Manager is currently only supported on **OS X 10.12 Sierra**. A build for 10.11 El Capitan is in the works. To get notified of changes, **watch** this [project on GitHub](https://github.com/mludowise/Sketch-Plugin-Manager).\n\nUsage\n-----\n\nOpen the plugin manager from the plugins menu.\n\n![Menu Items](img/menu.png)\n\n\n### Catalog\n\nSketch Plugin Manager's Plugin Catalog connects to the [Sketch Plugin Directory](https://sketchapp.com/extensions/plugins/) to let you browse and install plugins available on GitHub.\n\n![Catalog Tab](img/catalog-view.png)\n\n\n### Updates\n\nEvery time you start Sketch, Sketch Plugin Manager will check for updates to your installed plugins.\n\n![Update Alert](img/update-alert.png)\n\nYou will be able to review the list of changes before accepting updates to each plugin.\n\n![Updates Tab](img/updates-view.png)\n\n### Installed Plugins\n\nYou can manage all of your installed plugins from one place.\n\n![Updates Tab](img/installed-view.png)\n\nInstallation\n------------\n1. Download and unzip the [attached Zip file](https://github.com/mludowise/Sketch-Plugin-Manager/archive/master.zip).\n2. Double-click on **sketch-plugin-manager.sketchplugin** to install it.\n\nDonation\n--------\nSupport future updates and maintentence.\n\n[![Donate](https://www.paypalobjects.com/en_US/i/btn/btn_donateCC_LG.gif)](https://www.paypal.me/mludowise/5)\n\nTranslation Help\n----------------\nAre you fluent in multiple languages? [Help translate Sketch Plugin Manager](https://github.com/mludowise/Sketch-Plugin-Manager/tree/localize).\n\nPlugin Developers\n-----------------\n\nTo add your plugin to Sketch Plugin Manager, follow the instructions in [DEVELOPERS.md](DEVELOPERS.md).\n"
