b"gmusicapi: an unofficial API for Google Play Music\n==================================================\n\ngmusicapi allows control of\n`Google Music <http://music.google.com>`__ with Python.\n\n.. code-block:: python\n\n    from gmusicapi import Mobileclient\n    \n    api = Mobileclient()\n    # after running api.perform_oauth() once:\n    api.oauth_login('<a previously-registered device id>')\n    # => True\n    \n    library = api.get_all_songs()\n    sweet_track_ids = [track['id'] for track in library\n                       if track['artist'] == 'The Cat Empire']\n    \n    playlist_id = api.create_playlist('Rad muzak')\n    api.add_songs_to_playlist(playlist_id, sweet_track_ids)\n    \n**gmusicapi is not supported nor endorsed by Google.**\n\nThat said, it's actively maintained, and powers a bunch of cool projects:\n\n-  alternate clients, including\n   `one designed for the visually impaired <https://github.com/chrisnorman7/gmp3>`__,\n   `a web-based jukebox which ships with its own server <https://github.com/chrisnorman7/jukebox>`__,\n   `command line <https://github.com/christopher-dg/pmcli>`__\n   `clients <https://github.com/and3rson/clay>`__,\n   `a FUSE filesystem <https://github.com/EnigmaCurry/GMusicFS>`__,\n   and `an Alexa skill <https://github.com/stevenleeg/geemusic>`__\n-  library management tools for\n   `syncing tracks <https://github.com/thebigmunch/gmusicapi-scripts>`__,\n   `syncing playlists <https://github.com/soulfx/gmusic-playlist>`__,\n   and `migrating to a different account <https://github.com/brettcoburn/gmusic-migrate>`__\n-  proxies for media players, such as\n   `gmusicproxy <https://github.com/gmusicproxy/gmusicproxy>`__ and\n   `gmusicprocurator <https://github.com/malept/gmusicprocurator>`__,\n   as well as plugins for \n   `Mopidy <https://github.com/hechtus/mopidy-gmusic>`__,\n   `Squeezebox <https://github.com/squeezebox-googlemusic/squeezebox-googlemusic>`__ and\n   `Tizonia <https://github.com/tizonia/tizonia-openmax-il>`__.\n-  enhancements like `autoplaylists / smart playlists <https://autoplaylists.simon.codes>`__\n\n\nGetting started\n---------------\nStart with `the usage docs <http://unofficial-google-music-api.readthedocs.io/en/latest/usage.html#usage>`__, which will guide you through installation and the available apis.\n\nOnce you're up and running, you can explore the rest of the docs at http://unofficial-google-music-api.readthedocs.io.\n\nIf the documentation doesn't answer your questions, or you just want to get\nin touch, either `drop by #gmusicapi on Freenode\n<http://webchat.freenode.net/?channels=gmusicapi>`__ or shoot me an email.\n\nStatus and
