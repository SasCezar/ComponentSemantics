manager for Cocoa projects.\nYou can install it with the following command:\n\n```bash\n$ gem install cocoapods\n```\n\nVersaPlayer is available through [CocoaPods](https://cocoapods.org). To install\nit, simply add the following line to your Podfile:\n\n```ruby\npod \'VersaPlayer\'\n```\n\n### Carthage\n\n[Carthage](https://github.com/Carthage/Carthage) is a decentralized dependency manager that builds your dependencies and provides you with binary frameworks. To integrate Alamofire into your Xcode project using Carthage, specify it in your `Cartfile`:\n\n```ogdl\ngithub "josejuanqm/VersaPlayer" "3.0.0"\n```\n\n## Usage\n\n### Basic Usage\n\nVersaPlayer aims to be simple to use but also flexible, to start using VersaPlayer first create a view programatically or via storyboard. Then add this few lines of code to start playing your video.\n\n```swift\n    @IBOutlet weak var playerView: VersaPlayerView!\n\n    override func viewDidLoad() {\n        super.viewDidLoad()\n        if let url = URL.init(string: "http://rmcdn.2mdn.net/Demo/html5/output.mp4") {\n            let item = VersaPlayerItem(url: url)\n            playerView.set(item: item)\n        }\n    }\n```\n\n### Adding Controls\n\nTo add controls for your player use the VersaPlayerControls class, which comes packed with outlets to control your player, you can also add as many as you like by making a custom implementation.\n\nVersaPlayerControls class include the following outlets:\n\nOutlet Name             | Type             |  Action\n------------------------- | ------------------------- | -------------------------\nplayPauseButton | VersaStatefullButton | Toggle playback\nfullscreenButton | VersaStatefullButton | Toggle fullscreen mode\npipButton | VersaStatefullButton | Toggle PIP mode in supported devices\nrewindButton | VersaStatefullButton | Rewind playback\nforwardButton | VersaStatefullButton | Fast forward playback\nskipForwardButton | VersaStatefullButton | Skip forward the time specified in second at skipSize (found in VersaPlayerControls)\nskipBackwardButton | VersaStatefullButton | Skip backward the time specified in second at skipSize (found in VersaPlayerControls)\nseekbarSlider | VersaSeekbarSlider | Seek through playback\ncurrentTimeLabel | VersaTimeLabel | Indicate the current time\ntotalTimeLabel | VersaTimeLabel | Indicate the total time\nbufferingView | UIView | Shown when player is buffering\n\n```swift\n    @IBOutlet weak var playerView: VersaPlayerView!\n    @IBOutlet weak var controls: VersaPlayerControls!\n\n    override func viewDidLoad() {\n        super.viewDidLoad()\n        playerView.use(controls: controls)\n        if let url = URL.init(string: "http://rmcdn.2mdn.net/Demo/html5/output.mp4") {\n            let item = VersaPlayerItem(url: url)\n            playerView.set(item: item)\n        }\n    }\n```\n\n### Advanced Usage\n\n#### DRM\n\nVersaPlayer also brings support for encrypted content, to make use of this funcionality you must implement VersaPlayerDecryptionDelegate and assign it to VersaPlayer\'s decryptionDelegate property.\n\nTo read more about this topic go to:\n\nhttps://josejuanqm.github.io/Libraries-Documentation/VersaPlayerCore/Protocols/VersaPlayerDecryptionDelegate.html\n\n#### Tracks\n\nTo make use of different media tracks, such as audio, video, or captioning, use VersaPlayerMediaTracks, found in VersaPlayer class.\n\nto learn more about this properties go to:\n\nhttps://josejuanqm.github.io/Libraries-Documentation/VersaPlayerCore/Classes/VersaPlayerMediaTrack.html\n\n##### Audio Tracks\n\nAudio tracks are specially helpfull when dealing with different languages, for example for a movie playback. \n\nTo select an audio track simply fetch available tracks with VersaPlayer\'s audioTracks property.\n\n```swift\n    @IBOutlet weak var playerView: VersaPlayer!\n\n    ...\n    \n    let tracks: [VersaPlayerMediaTrack] = playerView.player.currentItem?.audioTracks\n    /// the name of the track\n    let name = tracks.first?.name\n    /// the language of the track\n    let name = tracks.first?.language\n    /// selecting the first one\n    tracks.first?.select(for: playerView.player)\n```\n\n##### Video Tracks\n\nVideo tracks are most helpfull when dealing with different renditions or different streams per video quality. \n\nTo select an video track simply follow the same principles as an audio track.\n\n```swift\n    @IBOutlet weak var playerView: VersaPlayer!\n\n    ...\n    \n    let tracks: [VersaPlayerMediaTrack] = playerView.player.currentItem?.videoTracks\n    /// the name of the track\n    let name = tracks.first?.name\n    /// selecting the first one\n    tracks.first?.select(for: playerView.player)\n```\n\n##### Caption Tracks\n\nCaption tracks are almost always helpfull. This can be used from a movie playback all the way to assitive content. \n\nTo select an video track simply follow the same principles as video and audio tracks.\n\n```swift\n    @IBOutlet weak var playerView: VersaPlayer!\n\n    ...\n    \n    let tracks: [VersaPlayerMediaTrack] = playerView.player.currentItem?.captionTracks\n    /// the name of the track\n    let name = tracks.first?.name\n    /// the language of the track\n    let name = tracks.first?.language\n    /// selecting the first one\n    tracks.first?.select(for: playerView.player)\n```\n\n###### Caption Styling\n\nCaption styling are not usually managed by the user, but when necessary, captionStyling property from VersaPlayer comes in handy.\n\nExplore all the available attributes that can be changed here:\n\nhttps://josejuanqm.github.io/Libraries-Documentation/VersaPlayerCore/Classes/VersaPlayerCaptionStyling.html\n\n## Extensions\n\nVersa is aimed to be versatile, and that\'s why it comes with an extensions feature, that lets you customize any aspect of the player by inheriting from VersaPlayerExtension.\n\nThis class comes with a player attribute that points to the player instance from which is being used.\nTo add an extension use the add(extension ext:) method found in https://josejuanqm.github.io/Libraries-Documentation/VersaPlayerCore/Classes/VersaPlayer.html.\n\nHere are some extensions for VersaPlayer that may be useful for you.\n\n1.
