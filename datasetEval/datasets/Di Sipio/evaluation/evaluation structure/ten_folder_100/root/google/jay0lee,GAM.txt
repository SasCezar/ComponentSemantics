b'GAM is a command line tool for Google G Suite Administrators to manage domain and user settings quickly and easily. [![Build Status](https://travis-ci.org/jay0lee/GAM.svg?branch=master)](https://travis-ci.org/jay0lee/GAM)\n# Quick Start\n## Linux / MacOS\nOpen a terminal and run:\n```\nbash <(curl -s -S -L https://git.io/install-gam)\n```\nthis will download GAM, install it and start setup.\n## Windows\nDownload the MSI Installer from the [GitHub Releases] page. Install the MSI and you\'ll be prompted to setup GAM.\n# Documentation\nThe GAM documentation is hosted in the [GitHub Wiki]\n# Mailing List / Discussion group\nThe GAM mailing list / discussion group is hosted on [Google Groups].
