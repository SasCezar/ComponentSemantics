b'google-hosts\n============\n\n[**What**](#what) | [**How**](#how) | [**Must**](#must) | [**Contributing**](#contributing) | [**License**](#license)\n\nWhat\n---\n\nThis project provides some tools to help you find google IP.\n\nHow\n---\n\n> [Requirements][Requirements]\n\n**`getssl.sh` get IP detail**\n\n```\n$ cd google-hosts/scripts\n\n# get detail of 192.168.1.1\n$ ./getssl.sh 192.168.1.1\n```\n\n\n**`find.sh` wrapper of getssl.sh, get IP detail from CIDR**\n\n```\n$ cd google-hosts/scripts\n\n# find IP from 192.168.1.1/24\n$ ./find.sh 192.168.1.1/24\n```\n\n**`filter.sh` filter IP from output directory(generated by find.sh) for some domain**\n\n```\n$ cd google-hosts/scripts\n\n# filter IP for *.google.com\n$ ./filter.sh *.google.com\n\n# filter IP for mail.google.com\n$ ./filter.sh mail.google.com\n```\n\n**`use.sh` use IP for some domain and update hosts.all**\n\n```\n$ cd google-hosts/scripts\n\n# use 192.168.1.1 for *.google.com \n$ ./use.sh *.google.com 192.168.1.1\n\n# use 192.168.1.1 for mail.google.com \n$ ./use.sh mail.google.com 192.168.1.1\n```\n\n**`select.sh` run filter.sh, use.sh, use the best IP for domains in hosts.all**\n\n```\n$ cd google-hosts/scripts\n$ ./select.sh\n```\n\n**`apply.sh` update ../hosts from hosts.all** \n\n```\n$ cd google-hosts/scripts\n$ ./apply.sh\n```\n\n**`auto.sh` find CIDR from a DNS and run find.sh, select.sh, apply.sh**\n\n```\n$ cd google-hosts/scripts\n$ ./auto.sh DNS # DNS is like 8.8.8.8, but you should try DNS in diffrent countries.\n```\n\nExplaination of output \n\n| IP | LOSS | TIME | SSL |\n| --- | --- | --- | --- |\n| IP | packet loss | ping time | ssl domain |\n\nMust\n---\n\n* Use google dns / opendns / other\n* Use international google.
