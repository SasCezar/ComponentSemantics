b"# Vagrant Parallels Provider\n[![Gem Version](https://badge.fury.io/rb/vagrant-parallels.svg)](https://badge.fury.io/rb/vagrant-parallels)\n[![Build Status](https://travis-ci.org/Parallels/vagrant-parallels.svg?branch=master)](https://travis-ci.org/Parallels/vagrant-parallels)\n[![Code Climate](https://codeclimate.com/github/Parallels/vagrant-parallels.svg)](https://codeclimate.com/github/Parallels/vagrant-parallels)\n\n_Vagrant Parallels Provider_ is a plugin for [Vagrant](http://www.vagrantup.com),\nallowing to manage [Parallels Desktop](http://www.parallels.com/products/desktop/)\nvirtual machines on macOS hosts.\n\n### Requirements\n- [Vagrant v1.8](http://www.vagrantup.com) or higher\n(_there are known issues with Vagrant v1.9.5\n[[GH-297](https://github.com/Parallels/vagrant-parallels/issues/297#issuecomment-304458691)]\nand v1.9.6 [[GH-301]](https://github.com/Parallels/vagrant-parallels/issues/301)_)\n- [Parallels Desktop 11 for Mac](http://www.parallels.com/products/desktop/) or higher\n\n*Note:* Only **Pro** and **Business** editions of **Parallels Desktop for Mac**\nare compatible with this Vagrant provider.\nStandard edition doesn't have a full command line functionality and can not be used\nwith Vagrant.\n\n## Features\nThe Parallels provider supports all basic Vagrant features, including Shared Folders,\nPrivate and Public Networking, Forwarded ports and Vagrant Share.\n\nIf you're just getting started with Vagrant, it is highly recommended that you\nread the official [Vagrant documentation](http://docs.vagrantup.com/v2/) first.\n\n## Installation\nMake sure that you have [Parallels Desktop for Mac](http://www.parallels.com/products/desktop/)\nand [Vagrant](http://www.vagrantup.com/downloads.html) properly installed.\nWe recommend that you use the latest versions of these products.\n\nParallels provider is a plugin for Vagrant. Run this command to install it:\n\n```\n$ vagrant plugin install vagrant-parallels\n```\n\n## Provider Documentation\n\nMore information about the Parallels provider is available in\n[Vagrant Parallels Documentation](http://parallels.github.io/vagrant-parallels/docs/)\n\nWe recommend you to start from these pages:\n* [Usage](http://parallels.github.io/vagrant-parallels/docs/usage.html)\n* [Getting Started](http://parallels.github.io/vagrant-parallels/docs/getting-started.html)\n* [Boxes](http://parallels.github.io/vagrant-parallels/docs/boxes/index.html)\n\n## Getting Help\n\nIf you have an issue with the Parallels provider or discover a bug,\nplease report it on the [Issue Tracker](https://github.com/Parallels/vagrant-parallels/issues).\n\n## License and Authors\n\n* Author: Youssef Shahin <yshahin@gmail.com>\n* Author: Mikhail Zholobov <legal90@gmail.com>\n* Copyright 2013-2018, Parallels International GmbH.\n\nVagrant Parallels Provider is open-sourced software licensed under the [MIT license](http://opensource.org/licenses/MIT).\n"
