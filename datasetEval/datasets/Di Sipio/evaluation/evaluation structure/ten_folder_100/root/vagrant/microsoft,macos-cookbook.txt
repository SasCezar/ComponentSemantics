b"# macOS Cookbook\n\n[![Build Status](https://dev.azure.com/office/OE/_apis/build/status/Microsoft.macos-cookbook?branchName=master)](https://dev.azure.com/office/OE/_build/latest?definitionId=5072&branchName=master)\n\nChef resources and recipes for managing and provisioning macOS.\n\n- [Chef Requirements](#chef-requirements)\n- [Supported OS Versions](#supported-os-versions)\n- [Attributes](#attributes)\n- [Recipes](#recipes)\n- [Data Bags](#data-bags)\n- [Resources](#resources)\n\n## Supported Chef Versions\n\n- Chef 15\n\n## Supported OS Versions\n\n- macOS High Sierra 10.13\n- macOS Mojave 10.14\n- macOS Catalina 10.15\n\n## Attributes\n\n### Admin User and Password\n\n```ruby\nnode['macos']['admin_user'] = 'vagrant'\nnode['macos']['admin_password'] = 'vagrant'\n```\n\nEach of these attributes defaults to vagrant since our resources are developed\nwith the Vagrant paradigm. In other words, the user and password declared here\nshould be an admin user with passwordless super-user rights.\n\n## Recipes\n\n### Keep Awake\n\nPrevent macOS from falling asleep, disable the screensaver, reboot upon power failure,\nenable wake on LAN, enable remote login (SSH) and adjust several other settings\nto always keep macOS on and available.\n\n**Usage:** `include_recipe 'macos::keep_awake'`\n\n| Attributes used                         | Default value           |\n|-----------------------------------------|-------------------------|\n| `node['macos']['remote_login_enabled']` | `true`                  |\n| `node['macos']['network_time_server']`  | `'time.windows.com'`    |\n| `node['macos']['time_zone']`            | `'America/Los_Angeles'` |\n\n## Resources\n\n- [`automatic_software_updates`](https://github.com/Microsoft/macos-cookbook/blob/master/documentation/resource_automatic_software_updates.md)\n- [`certificate`](https://github.com/Microsoft/macos-cookbook/blob/master/documentation/resource_certificate.md)\n- [`command_line_tools`](https://github.com/Microsoft/macos-cookbook/blob/master/documentation/resource_command_line_tools.md)\n- [`keychain`](https://github.com/Microsoft/macos-cookbook/blob/master/documentation/resource_keychain.md)\n- [`macos_user`](https://github.com/Microsoft/macos-cookbook/blob/master/documentation/resource_macos_user.md)\n- [`plist`](https://github.com/Microsoft/macos-cookbook/blob/master/documentation/resource_plist.md)\n- [`remote_management`](https://github.com/Microsoft/macos-cookbook/blob/master/documentation/resource_remote_management.md)\n- [`spotlight`](https://github.com/Microsoft/macos-cookbook/blob/master/documentation/resource_spotlight.md)\n- [`xcode`](https://github.com/Microsoft/macos-cookbook/blob/master/documentation/resource_xcode.md)\n"
