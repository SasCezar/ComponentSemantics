b"# Vagrant PHP7 \n\nA simple Vagrant LAMP setup with PHP 7.1 running on Ubuntu 16.04 LTS.\n\n## What's inside?\n\n- Ubuntu 16.04 LTS (Xenial Xerus)\n- Vim, Git, Curl, etc.\n- Apache\n- PHP 7.1 with some extensions\n- MySQL 5.7\n- Node.js 8 with NPM\n- RabbitMQ\n- Redis\n- Composer\n- phpMyAdmin\n\n## Prerequisites\n- [Vagrant](https://www.vagrantup.com/downloads.html)\n- Plugin vagrant-vbguest (``vagrant plugin install vagrant-vbguest``)\n\n## How to use\n\n- Clone this repository into your project\n- Run ``vagrant up``\n- Add the following lines to your hosts file:\n````\n192.168.11.11 app.local\n192.168.11.11 phpmyadmin.local\n````\n- Navigate to ``http://app.local/`` \n- Navigate to ``http://phpmyadmin.local/`` (both username and password are 'root')\n"
