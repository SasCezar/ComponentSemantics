build with packer for use with vagrant.\nThis repository includes the packer templates to build the baseboxes.\nThe build boxes are available through \n\n* [Vagrantcloud](https://app.vagrantup.com/ffuenf)\n* Amazon S3 (eu-central-1)\n\nTools\n=====\n\n* [vagrant](http://vagrantup.com)\n* [packer](http://packer.io)\n* [virtualbox](https://www.virtualbox.org/)\n* [VMware Fusion](http://www.vmware.com/de/products/fusion/)\n* [AWS Command Line Interface](http://aws.amazon.com/cli/)\n* [Thor](http://whatisthor.com/)\n\nUsage\n=====\n\nMake sure you have the above tools installed.\n\nThe following env vars must be present:\n* `PACKER_ATLAS_TOKEN` Atlas API Token\n* `AWS_ACCESS_KEY_ID` Amazon Webservices KEY-ID\n* `AWS_SECRET_ACCESS_KEY` Amazon Webservices ACCESS-KEY\n\nYou have to adjust the Thorfile line 50 to match your own S3 bucket.\n\nrun the following command to build/upload individual boxes:\n```\n$ bundle exec thor packer:build \\\n  --atlas_version=1.0.0 \\                         # version tag\n  --os=debian \\                                   # os distribution (debian/ubuntu)\n  --os_version=9.5.0 \\                            # os version\n  --providers=virtualbox,vmware_desktop,parallels # providers to build\n```\n\nrun the following command to build/upload ALL boxes synchronously:\n```\n$ ./build_boxes.sh\n```\n\nrun the following command to delete temporary artifacts\n```\n$ bundle exec thor packer:clean cache\n```\n\nrun the following command to delete all local box files\n```\n$ bundle exec thor packer:clean boxes\n```\n\nBoxes\n=====\n\n### Ubuntu\n\n#### Ubuntu Bionic Beaver 18.04.1 LTS / CHECKSUMS / manifest\n\n*
