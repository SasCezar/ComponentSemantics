b'# JSONlite\n\n##### A simple, self-contained, serverless, zero-configuration, json document store.\n\nJSONlite sandboxes the current working directory similar to SQLite. The JSONlite data directory is named `jsonlite.data` by default, and each json document is validated and saved pretty printed as a uuid.\n\n## Requirements\n\n1. bash\n2. uuidgen\n3. python -m json.tool\n  - Alternatively, install [yajl](http://lloyd.github.io/yajl/) for `json_reformat` *or* [jq](https://github.com/stedolan/jq) to get a roughly 3x performance improvement setting documents.\n\n  `json_reformat` is the fastest of the three. `jq` comes in at a close second and `python -m json.tool` is significantly the slowest. If possible, avoid using `python -m json.tool`.\n\n## Installation\n\n````\ngit clone https://github.com/nodesocket/jsonlite.git\nln -s "$PWD"/jsonlite/jsonlite.bash /usr/local/bin/jsonlite\n````\n\n## Configuration\n\nYou may optionally set the path to the data directory. It defaults to `$PWD/jsonlite.data` but can manually be set with the `JSONLITE_DATA_DIR` environment variable.\n\n````\n# default\nexport JSONLITE_DATA_DIR="$PWD"/jsonlite.data\n\n# manually set the data directory\nexport JSONLITE_DATA_DIR=/tmp/jsonlite.data\n````\n\n## API/Commands\n\n### set\n\n> set \\<json\\> - Writes a json document and returns the document id\n\n````\n\xe2\x9e\x9c jsonlite set \'{"name":"John Doe","active":true,"permissions":{"read":true,"write":false}}\'\n666B81D6-3F8A-4D57-BA3F-11FA8FC47246\n````\n\n`set` also supports piping into it:\n\n````\necho \'{"key":"value"}\' | jsonlite set\n4472B861-4C10-4C0A-A63B-E5D45AA679C0\n````\n````\ncat foobar.json | jsonlite set\n9DF4DC1F-121E-46DC-B580-E1663B645AED\n````\n\n### get\n\n> get \\<document-id\\> - Retrieves a json document by document id\n\n````\n\xe2\x9e\x9c jsonlite get 666B81D6-3F8A-4D57-BA3F-11FA8FC47246\n{\n    "active": true,\n    "name": "John Doe",\n    "permissions": {\n        "read": true,\n        "write": false\n    }\n}\n````\n\n### count\n\n> count - Total number of json documents in the database\n\n````\n\xe2\x9e\x9c jsonlite count\n293\n````\n\n### delete\n\n> delete \\<document-id\\> - Deletes a json document by document id\n\n````\n\xe2\x9e\x9c jsonlite delete 666B81D6-3F8A-4D57-BA3F-11FA8FC47246\n````\n\n### drop\n\n> drop (--force) - Drops the database\n\n````\n\xe2\x9e\x9c jsonlite drop\nDrop database \'/tmp/jsonlite.data\'? [Y/n] Y\n````\n\n````\n\xe2\x9e\x9c jsonlite drop --force\n````\n\n### help\n\n> help - Displays help\n\n````\n\xe2\x9e\x9c jsonlite help\nUsage: jsonlite command <command-specific-options>\n\n  set <json>             Writes a json document and returns a document id\n  get <document-id>      Retrieves a json document by document id\n  count                  Total number of json documents in the database\n  delete <document-id>   Deletes a json document by document id\n  drop (--force)         Drops the database\n  help                   Displays help\n  version                Displays the current version\n\n````\n\n### version\n\n> version - Displays the current version\n\n````\n\xe2\x9e\x9c jsonlite version\n1.1.1\n````\n\n### default\n\n> default - Displays the current version, configuration, and help\n\n```\n\xe2\x9e\x9c jsonlite\nJSONlite 1.1.1\n  json formatter: json_reformat (fastest)\n  data directory: /tmp/jsonlite.data\n\nUsage: jsonlite command <command-specific-options>\n\n  set <json>             Writes a json document and returns a document id\n  get <document-id>      Retrieves a json document by document id\n  count                  Total number of json documents in the database\n  delete <document-id>   Deletes a json document by document id\n  drop (--force)         Drops the database\n  help                   Displays help\n  version                Displays the current version\n\n```\n\n## Changelog\n\nhttps://github.com/nodesocket/jsonlite/blob/master/CHANGELOG.md\n\n## Support, Bugs, And Feature Requests\n\nCreate issues
