Requires Windows 10 Creators Update.\n\n## Feature\n\n**wslusc**\nA WSL shortcut creator to create a shortcut on your Windows 10 Desktop.\n\n**wslsys**\nA WSL system information printer to print out system informations from Windows 10 or Linux.\n\n**wslfetch**\nA WSL Screenshoot Information Tool to print information in an elegant way.\n\n**wslvar**\nA WSL tool to help you get Windows system environment variables.\n\n**wslview**\nA fake WSL browser that can help you open link in default Windows browser.\n\n**wslupath** \xe2\x9a\xa0 *Deprecated*\nA WSL tool to convert path styles.\n\n## Status\n\n| | Service | Master | Develop |\n| ------ | ------ |:------:|:-------:|\n| License Check | FOSSA | [![FOSSA Status](https://app.fossa.com/api/projects/git%2Bgithub.com%2Fpatrick330602%2Fwslu.svg?type=shield)](https://app.fossa.com/projects/git%2Bgithub.com%2Fpatrick330602%2Fwslu%2develop?ref=badge_shield) | - On older installations please install ubuntu-wsl that depends on wslu:\n\n```\nsudo apt update\nsudo apt install ubuntu-wsl\n```\n\n### For other Debian-based systems(Debian/Kali Linux)\n\nYou can download the `.deb` package from release and install it using `sudo dpkg -i wslu*`.\n\n#### Debian\n\nYou can use the following command for easy installation:\n\n```\n$ curl -s https://packagecloud.io/install/repositories/whitewaterfoundry/wslu/script.deb.sh | sudo bash\n```\n\n### For RPM-based systems(OpenSUSE/SLES/Pengwin Enterprise/Oracle Linux)\n\nYou can use the following command for easy installation:\n\n```\n$ curl -s https://packagecloud.io/install/repositories/whitewaterfoundry/wslu/script.rpm.sh | sudo bash\n```\n\nYou can download the `.rpm` package from release and install it using `sudo rpm -ivh "wslu*"`.\n\n### Arch Linux\n\nwslu is live on [AUR](https://aur.archlinux.org/packages/wslu/). You can download *PKGBUILD* and install manually or install via PKGBUILD helpers like yay.\n\n### Alpine Linux\n\nYou can now install wslu from Alpine Linux testing with following way:\n\n```\n$ echo "@testing http://dl-cdn.alpinelinux.org/alpine/edge/testing/" | sudo tee -a /etc/apk/repositories\n$ sudo apk update\n$ sudo apk add wslu@testing\n```\n\nYou can also download the `.apk` package from release and install it using `sudo apk add --allow-untrusted "wslu*"`.\n\n### Other distributions\n\n> **\xe2\x9a\xa0 Not Recommend**\n> \n> `curl | bash` method is not secure. [Related article](https://sandstorm.io/news/2015-09-24-is-curl-bash-insecure-pgp-verified-install)\n\nRun the following command in your preferred distro: `curl -sL https://wslu.patrickwu.space/install | bash`\n\n## Contributors\n\nThis project exists thanks to all the people who contribute.
