b'# Back In Time\n\nCopyright (C) 2008-2019 Oprea Dan, Bart de Koning, Richard Bailey, Germar Reitze, Taylor Raack\n\n[![Build Status](https://travis-ci.org/bit-team/backintime.svg?branch=master)](https://travis-ci.org/bit-team/backintime)\n[![Coverage Status](https://coveralls.io/repos/github/bit-team/backintime/badge.svg?branch=master)](https://coveralls.io/github/bit-team/backintime?branch=master)\n[![Documentation Status](https://readthedocs.org/projects/backintime-dev/badge/?version=latest)](http://backintime.readthedocs.org/projects/backintime-dev/en/latest/?badge=latest)\n\n## About\n\nBack In Time is a simple backup tool for Linux, inspired by "flyback project".\n\nIt provides a command line client \'backintime\' and a Qt5 GUI \'backintime-qt\'\nboth written in Python3.\n\nYou only need to specify 3 things:\n* where to save snapshots\n* what folders to backup\n* backup frequency (manual, every hour, every day, every month)\n\n## Documentation\n\nThe documentation is currently under development in https://backintime.readthedocs.org/\n\n## Support\n\nPlease ask questions and report bug on\nhttps://github.com/bit-team/backintime/issues\n\n## Download\n\nPlease find the latest versions on\nhttps://github.com/bit-team/backintime/releases/latest\n\n## INSTALL\n\nBack In Time is included in many distributions and can be installed from their\nrepositories.\n\n##### Ubuntu PPA\n\nWe provide a PPA (Private Package Archive) with current stable version\n(ppa:bit-team/stable) and a testing PPA (ppa:bit-team/testing)\n\n    sudo add-apt-repository ppa:bit-team/stable\n    sudo apt-get update\n    sudo apt-get install backintime-qt4\n\nor\n\n    sudo add-apt-repository ppa:bit-team/testing\n    sudo apt-get update\n    sudo apt-get install backintime-qt\n\n##### Debian/Ubuntu make packages\n\n    ./makedeb.sh\n    sudo dpkg -i ../backintime-common-<version>.deb\n    sudo dpkg -i ../backintime-qt-<version>.deb\n\n##### ArchLinux\n\nBack In Time is available through AUR. You need to import a public key once\nbefore installing\n\n    gpg --keyserver pgp.mit.edu --recv-keys 615F366D944B4826\n    # Fingerprint: 3E70 692E E3DB 8BDD A599  1C90 615F 366D 944B 4826\n    wget https://aur.archlinux.org/cgit/aur.git/snapshot/backintime.tar.gz\n    tar xvzf backintime.tar.gz\n    cd backintime\n    makepkg -srci\n\n### From sources\n\n##### Common\n\n* dependencies\n    - python3 (>= 3.3)\n    - rsync\n    - cron-daemon\n    - openssh-client\n    - python3-keyring\n    - python3-dbus\n\n* recomended\n    - sshfs\n    - encfs\n\n* Command\n\n        cd common\n        ./configure\n        make\n        make test\n        sudo make install\n\n\n##### Qt5 GUI\n\n* dependencies\n    - x11-utils\n    - python3-pyqt5\n    - libnotify-bin\n    - policykit-1\n    - python3-dbus.mainloop.pyqt5\n    - backintime-common\n\n* recomended\n    - python3-secretstorage or\n    - python3-keyring-kwallet or\n    - python3-gnomekeyring\n    - kompare or\n    - meld\n\n* Command\n\n        cd qt\n        ./configure\n        make\n        sudo make install\n\n\n## configure options\n\n    first value is default:\n    --no-fuse-group | --fuse-group (only COMMON)\n        Some distributions require user to be in group \'fuse\' to use\n        sshfs and encfs. This toggles the check on or off.\n\n    --python3 | --python (all)\n        Use either \'python3\' or \'python\' to start Python Version 3.x\n\n## NewsFeed\n\nBack In Time has a RSS feed\nhttps://feeds.launchpad.net/backintime/announcements.atom\n\n## Contribute\n\nThere is a dev-docu on https://backintime-dev.readthedocs.org\nIt\'s not complete yet but I\'m working on it. If you\'d like to contribute\nplease add docstrings following the\n[Google style guide](https://sphinxcontrib-napoleon.readthedocs.org/en/latest/example_google.html)\nand add unit-tests for new methods in common. To run unit-test locally you can\nrun `cd common && ./configure && make test`\n\n<sub>December 2016</sub>\n'
