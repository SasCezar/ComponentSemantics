b"# lightGallery\nA modern, electron and nodejs based image viewer for Mac, Windows and Linux.\n![lightgallery](http://sachinchoolur.github.io/electron-gallery/lightgallery.png)\n\nMain features\n---\n\n* Built with Electron.\n  * LightGallery uses HTML, CSS, and JavaScript with Chromium and Node.js to build the app.\n* Cross-platform.\n  * LightGallery works across operating systems. You can use it on OS X, Windows, or Linux. \n* 20+ Animations\n  * LightGallery comes with numerous number of beautiful in-built animations.\n* Animated thumbnails\n  * You also have the option to enable animated thumbnails from the settings.\n* Zoom & Fullscreen\n  * You can double-click on the image to see its actual size. Zoom-in and zoom-out controls can be used for changing the zoom values of the image.\n*  Mouse Drag & keyboard Navigations\n   * LightGallery allows users to navigate between slides via mouse drag and keyboard arrows.\n* Pagers\n* Auto slideshow\n* Support various kind of image formats (jpg, png, gif, webp).\n* Highly customizable\n* And many more.\n\n##### Watch :star: this repository.
