b"# QTodoTxt\n[![Build Status](https://travis-ci.org/QTodoTxt/QTodoTxt.png?branch=master)](https://travis-ci.org/QTodoTxt/QTodoTxt) [![GPLv3+](https://img.shields.io/badge/license-GPLv3%2B-blue.svg)](https://github.com/QTodoTxt/QTodoTxt/blob/master/gpl.txt)[![Translation](https://hosted.weblate.org/widgets/qtodotxt/-/svg-badge.svg)](https://hosted.weblate.org/projects/qtodotxt/)\n[![Scrutinizer Code Quality](https://scrutinizer-ci.com/g/QTodoTxt/QTodoTxt/badges/quality-score.png?b=master)](https://scrutinizer-ci.com/g/QTodoTxt/QTodoTxt/?branch=master)\n\nQTodoTxt is a cross-platform UI client for todo.txt files (see [todo.txt](http://todotxt.com)).\n\n![Main view](https://raw.githubusercontent.com/wiki/QTodoTxt/QTodoTxt/screenshots/QTodoTxt_main_view_orange.png)\n\n[more screenshots..](https://github.com/QTodoTxt/QTodoTxt/wiki/Screenshots)\n\n\n### Latest alpha/beta relase and QTodotTxt2:\n\nDownload code from git or use pip: pip install qtodotxt --upgrade --pre\n\nWe have also started work on QTodoTxt2 https://github.com/QTodoTxt/QTodoTxt2/\nMuch much much cleaner code and a new interface based on qml that offer a lot of new possibilities. Try it and give us feedback\n\nNow also available using pip:  pip install qtodotxt2 --upgrade --pre\n\nAnd please report bugs to github!\n\n###Latest stable release:\n\nV1.9.0 is released but not all binary packages have been created yet and the following links may point to older versions (pip should alway point to latest release). If you want a missing binary release, generate it then contact the authors to upload the file and update the link in this README\n\n- For Ubuntu / Debian: [.deb package](https://github.com/QTodoTxt/QTodoTxt/releases/download/v1.6.1/qtodotxt_1.6.1_all_deb.zip)\n- For Windows: [Windows installer](https://github.com/QTodoTxt/QTodoTxt/releases/download/v1.7.0/qtodotxt_1.7.0.exe)\n- For Mac OS X: [Mac OS X installer](http://dl.bintray.com/mnantern/generic/QTodoTxt_1.4.0.dmg)\n- Everywhere: ```pip install qtodotxt --upgrade``` (BUT to void unwanted pyqt5 compilation, pyqt5 is not specified as a dependency on pip, install pyqt5 manually before!)\n- Everywhere: [tar.gz archive](https://github.com/QTodoTxt/QTodoTxt/archive/v1.9.0.tar.gz)\n\n\n## Running from sources\nTo run the application without installing just:\n\n* Download [latest code archive](https://github.com/QTodoTxt/QTodoTxt/archive/master.zip)\n* Run:\n  * On Linux/Mac: bin/qtodotxt\n  * On Windows: bin/qtodotxt.pyw\n \nBe sure to have all [prerequisites](http://github.com/QTodoTxt/QTodoTxt/wiki) installed !\n\n\n## Documentation\n- [User Documentation](https://github.com/QTodoTxt/QTodoTxt/wiki)\n- [Screenshots](https://github.com/QTodoTxt/QTodoTxt/wiki/Screenshots)\n- [Downloads](https://github.com/QTodoTxt/QTodoTxt/wiki/Downloads)\n- [Changelog](https://github.com/QTodoTxt/QTodoTxt/wiki/Changelog)\n- [Defects and Enhancements](https://github.com/QTodoTxt/QTodoTxt/issues)\n- [Contributor's Guide](https://github.com/QTodoTxt/QTodoTxt/blob/master/CONTRIBUTING.md)\n- [Building QTodoTxt Binaries](https://github.com/QTodoTxt/QTodoTxt/wiki/Building-QTodoTxt-Binaries)\n- [Translation QTodoTxt into other languages](https://github.com/QTodoTxt/QTodoTxt/blob/master/TRANSLATION.md)\n"
