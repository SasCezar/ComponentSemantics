b'# Brightness Controller\n\nThis is version 2 of Brightness Controller. Please note that it is compatible with python2 only. It supports an arbitrary number of displays!\n\nIf you like this, do not forget to give us a Star!  [![GitHub stars](https://img.shields.io/github/stars/lordamit/brightness.svg?style=flat-square)](https://github.com/lordamit/brightness/stargazers) People already did!\n\n![](img/BrightnessController.gif)\n\n## Installation via PPA\n\nThanks to package maintainer @apandada1, we have PPA repository For Ubuntu and likewise users:\n\n```bash\nsudo add-apt-repository ppa:apandada1/brightness-controller\nsudo apt-get update\nsudo apt-get install brightness-controller\n```\n\n## Manual Installation\nFirst, install pyside.\n\n```bash\nsudo apt-get install python-pyside\n```\nSomtimes it is not enough to install and integrate pyside, so you might also try installing pyside using pip.\n\n```bash\npip install pyside\n```\n\nNext, download the latest zip file from [here](https://github.com/lordamit/Brightness/archive/master.zip).\n\nExtract it and open a terminal. Change directory to the `Brightness` folder. Next type this command:\n\n```bash\npython src/init.py\n```\n\nAchean also created a detailed tutorial on how to install it in Debian based on his experience. You can find it [here](https://github.com/LordAmit/Brightness/issues/98#event-1218811468).\n\n## Features\n\nThe following features are implemented:\n\n1. Brightness Control\n1. Saving color profile\n1. Loading color profile\n\n\nWe are working on the following features:\n\n1. Auto-loading of color and brightness settings\n2. Checking for update\n\nIt should be mentioned that through color profile Red, Green and Blue color ratios are changed in the screen.\n\n## Dependencies\nThere are several dependencies:\n\n1. python-pyside\n2. xrandr support in your system\n3. python2\n\n## Bugs\n\nPlease test v2.
