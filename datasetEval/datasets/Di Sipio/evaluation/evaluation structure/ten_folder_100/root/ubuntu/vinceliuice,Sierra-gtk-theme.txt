b'<img src="https://github.com/vinceliuice/Sierra-gtk-theme/blob/imgs/logo.png" alt="Logo" align="right" /> Sierra Gtk Theme\n======\n\nSierra is a Mac OSX like theme for GTK 3, GTK 2 and Gnome-Shell which supports GTK 3 and GTK 2 based desktop environments like Gnome, Pantheon, XFCE, Mate, etc.\n\nThis theme is based on Arc gtk theme of horst3180. Thanks horst3180 sincerely for his great job! \nhorst3180 - Arc gtk theme: https://github.com/horst3180/Arc-theme\n\n## Info\n\n### GTK+ 3.20 or later\n\n### GTK2 engines requirment\n- GTK2 engine Murrine 0.98.1.1 or later.\n- GTK2 pixbuf engine or the gtk(2)-engines package.\n\nFedora/RedHat distros:\n\n    yum install gtk-murrine-engine gtk2-engines\n\nUbuntu/Mint/Debian distros:\n\n    sudo apt-get install gtk2-engines-murrine gtk2-engines-pixbuf\n\nArchLinux:\n\n    pacman -S gtk-engine-murrine gtk-engines\n\nOther:\nSearch for the engines in your distributions repository or install the engines from source.\n\n## Installation\n\n### Flathub\n\nLight Theme\n`flatpak install flathub org.gtk.Gtk3theme.High-Sierra`\n\nDark Theme\n`flatpak install flathub org.gtk.Gtk3theme.High-Sierra-Dark`\n\n### Distributions\n\nopenSUSE [package](https://build.opensuse.org/package/show/X11:common:Factory/sierra-gtk-theme):\n\n```\nsudo zypper ar obs://X11:common:Factory/sierra-gtk-theme x11\nsudo zypper ref\nsudo zypper in sierra-gtk-theme\n```\n\nUbuntu [PPA](https://launchpad.net/~dyatlov-igor/+archive/ubuntu/sierra-theme) (maintained by [@igor-dyatlov](https://github.com/igor-dyatlov)):\n```\nsudo add-apt-repository ppa:dyatlov-igor/sierra-theme\nsudo apt update\nsudo apt install sierra-gtk-theme       # point releases\nsudo apt install sierra-gtk-theme-git   # git master branch\n```\n\n### From source\n\nRun\n\n    ./install.sh\n\n#### Install tips\n\nUsage:  `./Install`  **[OPTIONS...]**\n\n|  OPTIONS:           | |\n|:--------------------|:-------------|\n|-d,  --dest           | Specify theme destination directory (Default: $HOME/.themes)|\n|-n,  --name           | Specify theme name (Default: Sierra)|\n|-c,  --color          | Specify theme color variant(s) **[light/dark]** (Default: All variants)|\n|-o,  --opacity        | Specify theme opacity variant(s) **[standard/solid]** (Default: All variants)|\n|-f,  --flat           | Specify theme flat variant(s) **[standard/compact]** (Default: All variants)|\n|-t,  --thin           | Specify theme titlebutton variant(s) **[standard/thin]** (Default: All variants)|\n|-na, --noapple        | Specify gnome-shell to not display apple logo (Default: Display apple logo)|\n|-g,  --gdm            | Install GDM theme|\n|-h,  --help           | Show this help|\n\n## Screenshots\n\n### GNOME Shell\n\n#### Sierra\n![01](https://github.com/vinceliuice/Sierra-gtk-theme/blob/imgs/screenshot1.jpeg?raw=true)\n![02](https://github.com/vinceliuice/Sierra-gtk-theme/blob/imgs/screenshot2.jpeg?raw=true)\n![03](https://github.com/vinceliuice/Sierra-gtk-theme/blob/imgs/screenshot3.jpeg?raw=true)\n![04](https://github.com/vinceliuice/Sierra-gtk-theme/blob/imgs/screenshot4.jpeg?raw=true)\n![05](https://github.com/vinceliuice/Sierra-gtk-theme/blob/imgs/screenshot5.jpeg?raw=true)\n'
