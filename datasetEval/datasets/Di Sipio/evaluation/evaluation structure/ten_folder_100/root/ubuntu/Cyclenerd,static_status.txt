b'# status.sh\n\n[![Build Status](https://travis-ci.org/Cyclenerd/static_status.svg?branch=master)](https://travis-ci.org/Cyclenerd/static_status)\n\nSimple Bash script to generate a static status page. Displays status of websites, services (HTTP, SAP, MySQL...) and ping. Everything easy to customize. \xf0\x9f\xa4\x93\n\nYou can also easily check more complicated things with this script.\nFor example if a text is present in a web page or if a host appears in the route path (traceroute).\nChecking the route path is useful, for instance, if you have a backup mobile internet connection in addition to your cable connection.\n\n![Screenshot](https://www.nkn-it.de/static_status/Status-Page-Past-Incidents.jpg)\n\n## Installation\n\nDownload `status.sh` and configuration file:\n\n\t$ curl -f https://raw.githubusercontent.com/Cyclenerd/static_status/master/status.sh -o status.sh\n\t$ curl -f https://raw.githubusercontent.com/Cyclenerd/static_status/master/status_hostname_list.txt -o status_hostname_list.txt\n\nCustomize the `status.sh` script and the services to be monitored:\n\n\t$ vi status.sh\n\t$ vi status_hostname_list.txt\n\nRun:\n\n\t$ bash status.sh\n\n## Usage\n\n\tUsage: status.sh [OPTION]:\n\t\tOPTION is one of the following:\n\t\t\tsilent\t no output from faulty connections to stout (default: no)\n\t\t\tloud\t output from successful and faulty connections to stout (default: no)\n\t\t\thelp\t displays help (this message)\n\nExample:\n\n\t$ bash status.sh loud\n\nExecute a cron job every minute:\n\n\t$ crontab -e\n\nAdd:\n\n\t*/1 * * * * bash /path/to/status.sh silent >> /dev/null\n\n\n## Demo\n\nhttps://cyclenerd.github.io/static_status_demo/\n\n### Screenshots\n\n![Screenshot](https://www.nkn-it.de/static_status/Status-Page-Maintenance.jpg)\n![Screenshot](https://www.nkn-it.de/static_status/Status-Page-OK.jpg)\n![Screenshot](https://www.nkn-it.de/static_status/Status-Page-Outage.jpg)\n![Screenshot](https://www.nkn-it.de/static_status/Status-Page-Major_Outage.jpg)\n![Screenshot](https://www.nkn-it.de/static_status/Status-Page-Past-Incidents.jpg)\n\n## Requirements\n\nOnly `bash`, `ping`, `traceroute`, `curl` and `nc`. In many *NIX distributions (Ubuntu, macOS) the commands are already included.\nIf not, the missing packages can be installed quickly.\nOn a debian-based system (Ubuntu), just run `sudo apt-get install curl iputils-ping traceroute netcat-openbsd`.\n\n## TODO\n\n* More and better documentation\n\nHelp is welcome \xf0\x9f\x91\x8d\n\n\n## License\n\nGNU Public License version 3.\nPlease feel free to fork and modify this on GitHub (https://github.com/Cyclenerd/static_shell).'
