b"Voxelman\n========\n\n![gif](https://i.imgur.com/NxsT4AK.gif)\n![gif](https://i.imgur.com/yrpIhfk.gif)\n\n**Voxelman** is an example that shows how to use the new **[Entity Component\nSystem]** with Unity in an extreme way. Each voxel in the scene is instantiated\nas an entity, and controlled by component systems. It also utilizes the **C#\nJob System**, the **Burst Compiler** and the **asynchronous raycast** to hit\nthe maximum efficiency of multi-core processors.\n\n[Entity Component System]: https://github.com/Unity-Technologies/EntityComponentSystemSamples\n\nSystem requirements\n-------------------\n\n- Unity 2018.2 or later\n\nHow to play with this project\n-----------------------------\n\n### Submodules\n\nThis repository uses [Git submodules] to manage UPM (Unity Package Manager)\npackages. **If you know Git well and how to deal with submodules,** simply\nclone this repository and do `submodule init` & `submodule update` in it.\n\n**If you're not sure what Git submodule is,** download the contents of this\nrepository from [zip download link]. After extracting the zip file, download\nand extract the following packages into the `Packages` directory.\n\n- [jp.keijiro.cmu-mocap](https://github.com/keijiro/CMUMocap/archive/upm.zip)\n- [jp.keijiro.klak](https://github.com/keijiro/Klak/archive/upm.zip)\n- [jp.keijiro.neolowman](https://github.com/keijiro/NeoLowMan/archive/upm.zip)\n- [jp.keijiro.test-assets](https://github.com/keijiro/jp.keijiro.test-assets/archive/master.zip)\n\n[ECS repository]: https://github.com/Unity-Technologies/EntityComponentSystemSamples\n[zip download link]: https://github.com/keijiro/Voxelman/archive/master.zip\n[Git submodules]: https://git-scm.com/book/en/v2/Git-Tools-Submodules\n"
