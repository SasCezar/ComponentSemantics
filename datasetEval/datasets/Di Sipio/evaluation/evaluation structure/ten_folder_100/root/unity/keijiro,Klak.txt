b'Klak\n====\n\n![gif](http://i.imgur.com/9AfoxGo.gif)\n![gif](http://i.imgur.com/uCrvKNg.gif)\n\n![gif](http://i.imgur.com/nmvtC5d.gif)\n![gif](http://i.imgur.com/LysINFA.gif)\n\n![gif](http://i.imgur.com/MsPX8pn.gif)\n![gif](http://i.imgur.com/WjQzVWz.gif)\n\n**Klak** is a collection of scripts for creative coding with Unity.\n\nFeatures\n--------\n\n- Noise/Random number\n - Perlin noise\n - Reproducible random number series with xxHash\n - Reproducible noise generator\n\n- Interpolator\n - Frame rate independent exponential interpolation\n - Classic spring smoothing\n - Critically damped spring smoothing\n\n- Extension methods for the standard classes\n - Vector4/Quaternion converter\n - Material property setter\n\n- Klak.Motion: basic procedural motion\n - Constant motion\n - Brownian motion\n\n- Klak.Wiring: node-based patching system\n\nInstallation\n------------\n\nDownload one of the unitypackage files from the [Releases] page and import it\nto a project.\n\n[Releases]: https://github.com/keijiro/Klak/releases\n\nExtensions\n----------\n\n- [MidiKlak] - MIDI input extension\n- [OscKlak] - OSC (Open Sound Control) input extension\n- [KlakUI] - Custom UI controls\n- [KlakSpout] - Spout (inter-application video sharing) extension\n\n[MidiKlak]: https://github.com/keijiro/MidiKlak\n[OscKlak]: https://github.com/keijiro/OscKlak\n[KlakUI]: https://github.com/keijiro/KlakUI\n[KlakSpout]: https://github.com/keijiro/KlakSpout\n\nLicense\n-------\n\n[MIT](LICENSE.md)\n'
