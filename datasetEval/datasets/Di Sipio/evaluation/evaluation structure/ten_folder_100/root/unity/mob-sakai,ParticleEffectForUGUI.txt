Add your particle effect to canvas.\n2. (Option) If you want to mask particles, set a UI shader such as "UI/UIAdditive" to material for ParticleSystem.  \n![](https://user-images.githubusercontent.com/12690315/42674022-134e3a40-86a9-11e8-8f44-a110d2f14185.gif)\n3. Add `UIParticle` component to root particle system of your effect from `Add Component` in inspector.  \n![](https://user-images.githubusercontent.com/12690315/41772125-5aca69c8-7652-11e8-8442-21f6015069a1.png)\n4. If your effect consists of multiple ParticleSystems, click "Fix".  \n![](https://user-images.githubusercontent.com/12690315/49148942-1c243880-f34c-11e8-9cf5-d871d65c4dbe.png)\n5. Adjust the Scale property to change the size of the effect.  \n![](https://user-images.githubusercontent.com/12690315/49148937-19c1de80-f34c-11e8-87fc-138192777540.gif)\n6.  Enjoy!\n\n\n##### Requirement\n\n* Unity 2018.2+ (Tested in Unity 2018.2.0f2)\n* No other SDK are required\n\n\n\n\n<br><br><br><br>\n## Development Note\n\n#### Animatable material property\n\n![](https://user-images.githubusercontent.com/12690315/53286323-2d94a980-37b0-11e9-8afb-c4a207805ff2.gif)\n\nAnimation clips can change the material properties of the Renderer, such as ParticleSystemRenderer.  \nIt uses MaterialPropertyBlock so it does not create new material instances.  \nUsing material properties, you can change UV animation, scale and color etc.\n\nWell, there is a component called CanvasRenderer.  \nIt is used by all Graphic components for UI (Text, Image, Raw Image, etc.) including UIParticle.  \nHowever, It is **NOT** a Renderer.  \nTherefore, in UIParticle, changing ParticleSystemRenderer\'s MaterialPropertyBlock by animation clip is ignored.\n\nTo prevent this, Use "Animatable Material Property".  \n"Animatable Material Property" gets the necessary properties from ParticleSystemRenderer\'s MaterialPropertyBlock and sets them to the CanvasRenderer\'s material. \n\n\n\n<br><br><br><br>\n## License\n\n* MIT\n* \xc2\xa9 UTJ/UCL\n\n\n\n## Author\n\n[mob-sakai](https://github.com/mob-sakai)\n[![](https://img.shields.io/twitter/follow/mob_sakai.svg?label=Follow&style=social)](https://twitter.com/intent/follow?screen_name=mob_sakai)  \n[![become_a_patron](https://user-images.githubusercontent.com/12690315/50731629-3b18b480-11ad-11e9-8fad-4b13f27969c1.png)](https://www.patreon.com/join/2343451?)\n\n\n\n## See Also\n\n* GitHub page : https://github.com/mob-sakai/ParticleEffectForUGUI\n* Releases : https://github.com/mob-sakai/ParticleEffectForUGUI/releases\n* Issue tracker : https://github.com/mob-sakai/ParticleEffectForUGUI/issues\n* Current project : https://github.com/mob-sakai/ParticleEffectForUGUI/projects/1\n* Change log : https://github.com/mob-sakai/ParticleEffectForUGUI/blob/upm/CHANGELOG.md\n'
