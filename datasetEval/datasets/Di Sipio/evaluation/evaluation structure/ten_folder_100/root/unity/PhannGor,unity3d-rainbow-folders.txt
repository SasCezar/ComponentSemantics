This simple but colorful asset allows you to do that!\n\nWith "Rainbow Folders" you can set custom icon for any folder in unity project browser:\n\n![Browser window example](https://raw.githubusercontent.com/PhannGor/phanngor.github.io/master/stuff/rainbowfolders/images/v05/01.png)\n![Browser window example](https://raw.githubusercontent.com/PhannGor/phanngor.github.io/master/stuff/rainbowfolders/images/v05/02.png)\n![Browser window example](https://raw.githubusercontent.com/PhannGor/phanngor.github.io/master/stuff/rainbowfolders/images/v05/03.png)\n\n## Installing\n\nRainbow Folders is a standard Unity extension and should be installed like any other Unity package. Just drag the RainbowFolders.unitypackage into your current project, or in the Editor go to the drop-down menu Assets \xe2\x86\x92 Import Package \xe2\x86\x92 Custom Package and then browse to the RainbowFolders.unitypackage file.\n\nWhen downloading from the Asset Store then Download Manager will automate this process.\n\n![Browser window example](https://raw.githubusercontent.com/PhannGor/phanngor.github.io/master/stuff/rainbowfolders/images/v07/02.png)\n\nOnce the Importing dialog appears, just click the Import button.\n\n## Folder location\n\nThe package will be imported into the `Assets/Plugins/RainbowFolders` folder by default. Most users prefer to keep it here, but you can freely move it wherever you want.  Just go to **Edit \xe2\x86\x92 Preferences \xe2\x86\x92 Rainbow Folders** and update the folder location:\n\n![Browser window example](https://raw.githubusercontent.com/PhannGor/phanngor.github.io/master/stuff/rainbowfolders/images/v07/03.png) \n\n## Configuring folder icons\n\nTo apply custom icon for some folder in your project view, just hold the **Modifier key** and click on any folder icon in Unity project browser. By default, it\xe2\x80\x99s the **Alt key**, but you can change it in **Preferences**.\n\n![Alt-click example](https://raw.githubusercontent.com/PhannGor/phanngor.github.io/master/stuff/rainbowfolders/images/v05/04.png)\n\nConfiguration dialogue will appear, and you\'ll be able to assign icons to the corresponding folder.\n\n![Configuration dialog](https://raw.githubusercontent.com/PhannGor/phanngor.github.io/master/stuff/rainbowfolders/images/v07/05.png)\n\nWhat you need to configure for each folder item, are these fields:\n\n* **Folder Name** - icon will be applied to all folders with that name.\n* or **Folder Path** - icon will be applied to a single folder with specified path.\n* **Recursive** checkbox - subfolders icons will be changed automatically\n\nThen you need to specify actually icons:\n* **Small Icon** - custom icon for the left panel of the project browser (16x16 px)\n* **Large Icon** - custom icon for the right panel of the project browser (64x64 px)\n\nYour changes will be applied next time when the project browser will retrieve the focus.\n\n### Presets\n\nYou can choose icons from few dozen of presets. Simply click on the star button in configuration dialog, select one of them from the drop-down menu and apply changes.\n\n![Configuration dialog](https://raw.githubusercontent.com/PhannGor/phanngor.github.io/master/stuff/rainbowfolders/images/v07/07.png)\n![Configuration dialog](https://raw.githubusercontent.com/PhannGor/phanngor.github.io/master/stuff/rainbowfolders/images/v07/08.png)\n\n### Revert to default\n\nTo reset the folder icon to the default one, just **Alt-click** on it, then press the red cross button in configuration dialogue and apply changes.\n\n![Configuration dialog](https://raw.githubusercontent.com/PhannGor/phanngor.github.io/master/stuff/rainbowfolders/images/v07/06.png)\n\n### Multi-editing\n\nYou can also edit multiple folders at once, just select them all and **Alt-click** at one of their icons.\n\n![Configuration dialog](https://raw.githubusercontent.com/PhannGor/phanngor.github.io/master/stuff/rainbowfolders/images/v05/09.png)\n\n### Configs List\n\nTo view all existing assignments, click on the gear button in the configuration dialog, then take a look at the Inspector.\n\n![Configuration dialog](https://raw.githubusercontent.com/PhannGor/phanngor.github.io/master/stuff/rainbowfolders/images/v07/10.png)\n\nThere is reorderable list with all defined "folder" configurations. You can modify existing items, remove them using "-" button or add new ones by clicking "+" button below.\n\n![Configuration dialog](https://raw.githubusercontent.com/PhannGor/phanngor.github.io/master/stuff/rainbowfolders/images/v07/11.png)\n\nIf there is more than one config for the same folder (including recursive assignments), then **latest (lowest) item** in the list will be applied.\n\n## Context menu\n\nAll the above functionality is also available via the context menu.\n\n![Configuration dialog](https://raw.githubusercontent.com/PhannGor/phanngor.github.io/master/stuff/rainbowfolders/images/v05/14.png)\n\n**Right-click** on any folder in project view and select **Rainbow Folders \xe2\x86\x92 Apply Custom** to open configuration dialogue.\n\nThe **Rainbow Folders \xe2\x86\x92 Revert to Default** item will reset corresponding folder icon to the default one.\n\nWith **Color, Tag, Type** and **Platform** sub-items you can easily apply custom icons from few dozen of presets, right from the context menu.\n\nSelect **Rainbow Folders \xe2\x86\x92 Settings** to view all existing assignments in the inspector.\n\n## Upgrading\n\nPlease always do a clean import of the **Rainbow Folders** package (delete the old version before importing the new one). Otherwise, you may receive a number of difficult to diagnose issues.\n* Backup your settings file (optional):\n`Assets/.../RainbowFolders/Editor/Data/RainbowFoldersSettings.asset`\n* Delete the `Assets/.../RainbowFolders` folder.\n* Delete the `Assets/Editor Default Resources/RainbowFolders` folder, if exists.\n* Import the new version from package or from the Asset Store.\n* Restore your settings file.\n'
