b'# Quantum\n\n[Cron](https://en.wikipedia.org/wiki/Cron)-like job scheduler for [Elixir](http://elixir-lang.org/).\n\n[![Financial Contributors on Open Collective](https://opencollective.com/quantum/all/badge.svg?label=financial+contributors)](https://opencollective.com/quantum) [![Hex.pm Version](http://img.shields.io/hexpm/v/quantum.svg)](https://hex.pm/packages/quantum)\n[![Hex docs](http://img.shields.io/badge/hex.pm-docs-green.svg?style=flat)](https://hexdocs.pm/quantum)\n[![Build Status](https://travis-ci.org/quantum-elixir/quantum-core.svg?branch=master)](https://travis-ci.org/quantum-elixir/quantum-core)\n[![Coverage Status](https://coveralls.io/repos/quantum-elixir/quantum-core/badge.svg?branch=master)](https://coveralls.io/r/quantum-elixir/quantum-core?branch=master)\n[![Inline docs](http://inch-ci.org/github/quantum-elixir/quantum-core.svg)](http://inch-ci.org/github/quantum-elixir/quantum-core)\n[![Hex.pm](https://img.shields.io/hexpm/dt/quantum.svg)](https://hex.pm/packages/quantum)\n\n> **This README follows master, which may not be the currently published version**. Here are the\n[docs for the latest published version of Quantum](https://hexdocs.pm/quantum/readme.html).\n\n> :warning: **If you\'re using a version below `v2.2.6`, please update immediately.** :warning:\n> See [Issue #321](https://github.com/quantum-elixir/quantum-core/issues/321) for more details.\n\n## Setup\n\nTo use Quantum in your project, edit the `mix.exs` file and add Quantum to\n\n**1. the list of dependencies:**\n```elixir\ndefp deps do\n  [{:quantum, "~> 2.3"},\n   {:timex, "~> 3.0"}]\nend\n```\n\n**2. and create a scheduler for your app:**\n```elixir\ndefmodule YourApp.Scheduler do\n  use Quantum.Scheduler,\n    otp_app: :your_app\nend\n```\n\n**3. and your application\'s supervision tree:**\n```elixir\ndefmodule YourApp.Application do\n  use Application\n\n  def start(_type, _args) do\n    import Supervisor.Spec, warn: false\n\n    children = [\n      # This is the new line\n      YourApp.Scheduler\n    ]\n\n    opts = [strategy: :one_for_one, name: YourApp.Supervisor]\n    Supervisor.start_link(children, opts)\n  end\nend\n```\n\n## Troubleshooting\n\nTo see more transparently what `quantum` is doing, configure the `logger` to display `:debug` messages.\n\n```elixir\nconfig :logger,\n  level: :debug\n```\n\nIf you want do use the logger in debug-level without the messages from quantum:\n\n```elixir\nconfig :your_app, YourApp.Scheduler,\n  debug_logging: false\n```\n\nIf you encounter any problems with `quantum`, please search if there is already an\n  [open issue](https://github.com/quantum-elixir/quantum-core/issues) addressing the problem.\n\nOtherwise feel free to [open an issue](https://github.com/quantum-elixir/quantum-core/issues/new). Please include debug logs.\n\n## Migrate to V2\n\nSee the [Migration Guide](https://hexdocs.pm/quantum/migrate-v2.html).\n\n## Usage\n\nConfigure your cronjobs in your `config/config.exs` like this:\n\n```elixir\nconfig :your_app, YourApp.Scheduler,\n  jobs: [\n    # Every minute\n    {"* * * * *",      {Heartbeat, :send, []}},\n    # Every 15 minutes\n    {"*/15 * * * *",   fn -> System.cmd("rm", ["/tmp/tmp_"]) end},\n    # Runs on 18, 20, 22, 0, 2, 4, 6:\n    {"0 18-6/2 * * *", fn -> :mnesia.backup(\'/var/backup/mnesia\') end},\n    # Runs every midnight:\n    {"@daily",         {Backup, :backup, []}}\n  ]\n```\n\nMore details on the usage can be found in the [Documentation](https://hexdocs.pm/quantum/configuration.html)\n\n## Contribution\n\nThis project uses the [Collective Code Construction Contract (C4)](http://rfc.zeromq.org/spec:42/C4/) for all code changes.\n\n> "Everyone, without distinction or discrimination, SHALL have an equal right to become a Contributor under the\nterms of this contract."\n\n### tl;dr\n\n1.
