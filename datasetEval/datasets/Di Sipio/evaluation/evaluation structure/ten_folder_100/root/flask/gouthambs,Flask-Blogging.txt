b'Flask-Blogging\n==============\n\n.. image:: https://travis-ci.org/gouthambs/Flask-Blogging.svg?branch=master\n    :target: https://travis-ci.org/gouthambs/Flask-Blogging\n\n\nThis is a Flask extension for adding blog support to your site using Markdown.\nPlease see `Flask-Blogging documentation <http://flask-blogging.readthedocs.org/en/latest/>`_\nfor more details. You can extend Flask-Blogging by using `plugins from here <https://github.com/gouthambs/blogging_plugins>`_.\n\nCheck out the `Serverless Blog <https://serverlessblog.com>`_  demo running on AWS Lambda.\n\nFeatures\n--------\n\n- Integration with Markdown Editor\n- Ability to upload images for use in blog pages\n- Incorporate math formulas in LaTeX format\n- Integrates with authentication to allow multiple users\n- Plugin framework to easily extend and add new features\n    \nScreen Shots\n------------\n\n    Blog Editor\n\n.. figure:: /docs/_static/blog_editor.png\n\n    Blog Page\n\n.. figure:: /docs/_static/blog_page.png\n\n\nMinimal Example\n---------------\n\nCheck the `quickstart example <http://flask-blogging.readthedocs.org/en/latest/#quick-start-example>`_ for the latest documentation.\n\nInstallation\n------------\n\nInstall the extension with the following commands::\n\n    $ easy_install flask-blogging\n    \nor alternatively if you have pip installed::\n\n    $ pip install flask-blogging\n    \n\nDependencies\n------------\n\n- `Flask <https://github.com/mitsuhiko/flask>`_\n- `SQLAlchemy <https://github.com/zzzeek/sqlalchemy>`_\n- `Flask-Login <https://github.com/maxcountryman/flask-login>`_\n- `Flask-Principal <https://github.com/mattupstate/flask-principal>`_\n- `Flask-WTF <https://github.com/lepture/flask-wtf>`_\n- `Flask-FileUpload <https://github.com/Speedy1991/Flask-FileUpload>`_\n- `Markdown <https://pypi.org/project/Markdown/>`_\n- `Bootstrap <http://getbootstrap.com/>`_\n- `Bootstrap-Markdown <https://github.com/toopay/bootstrap-markdown>`_\n- `Markdown-js <https://github.com/evilstreak/markdown-js>`_\n\n\nLicense\n-------\n\n`MIT </LICENSE>`_\n\n.. include:: /docs/releases.rst\n\n.. include:: /docs/authors.rst\n'
