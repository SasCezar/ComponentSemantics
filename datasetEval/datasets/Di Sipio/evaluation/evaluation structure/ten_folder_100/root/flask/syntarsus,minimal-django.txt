b'# Minimal Django\nA "Hello World" for Django...Because Django can be nearly as simple as Flask.\n\nThe aim of this repository is to highlight the fact that a Django project, when simplified down to the bare essentials, can be nearly as minimal as a micro-framework application. \n\n### To run it:\n\n1. Install [Python](https://www.python.org/downloads/)\n2. Install [Django](https://docs.djangoproject.com/en/dev/topics/install/). If you have pip: `pip install Django`\n3. Clone this repository: `git clone https://github.com/rnevius/minimal-django.git`\n4. Switch to the project directory: `cd minimal-django`\n5. Run the development server: `python minimal.py runserver`\n6. Open a browser to `http://127.0.0.1:8000/`\n'
