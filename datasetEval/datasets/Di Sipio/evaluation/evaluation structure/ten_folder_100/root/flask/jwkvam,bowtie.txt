No need to know\nweb frameworks or JavaScript, focus on building functionality in Python.\nInteractively explore your data in new ways! Deploy and share with\nothers!\n\nDemo\n----\n\nSee a live `example <https://bowtie-demo.herokuapp.com/>`__ generated\nfrom this\n`code <https://github.com/jwkvam/bowtie-demo/blob/master/example.py>`__!\n\nGallery\n-------\n\nFor more examples, check out the\n`gallery <https://github.com/jwkvam/bowtie/wiki/Gallery>`__ and\n`repos <https://github.com/jwkvam/bowtie/wiki/Repos>`__.\nFeel free to add your own creations!\n\nInstallation\n------------\n\nIf you use ``conda``, you can install with::\n\n    conda install -c conda-forge bowtie-py\n\nIf you use ``pip``, you can install with::\n\n    pip install bowtie\n\nRequirements\n^^^^^^^^^^^^\n\nBowtie uses `Yarn <https://yarnpkg.com>`__ to manage node packages.\nIf you installed Bowtie through ``conda``, Yarn was also installed as a dependency.\nYarn can be installed through conda::\n\n    conda install -c conda-forge yarn\n\nOtherwise follow `install\ninstructions <https://yarnpkg.com/en/docs/install>`__ for Yarn for your\nOS.\n\nDocumentation\n-------------\n\nAvailable `here <https://bowtie-py.readthedocs.io/en/latest/>`__.\n\nJupyter Integration\n-------------------\n\nAn early integration with Jupyter has been prototyped.\nI encourage you to try it out and share feedback.\nI hope this will make it easier to make Bowtie apps.\n\nRead the\n`documentation <https://bowtie-py.readthedocs.io/en/latest/jupyter.html>`__\nfor more details.\n\nDocker\n------\n\nDocker images are provided as an alternative way to use Bowtie.
