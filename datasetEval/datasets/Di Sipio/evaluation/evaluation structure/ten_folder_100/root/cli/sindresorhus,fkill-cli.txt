Cross-platform.\n\n[![Build Status](https://travis-ci.org/sindresorhus/fkill-cli.svg?branch=master)](https://travis-ci.org/sindresorhus/fkill-cli)\n\nWorks on macOS, Linux, and Windows.\n\n\n## Install\n\n```\n$ npm install --global fkill-cli\n```\n\n\n## Usage\n\n```\n$ fkill --help\n\n  Usage\n    $ fkill [<pid|name> \xe2\x80\xa6]\n\n  Options\n    --force -f    Force kill\n    --verbose -v  Show process arguments\n    --silent -s   Silently kill and always exit with code 0\n\n  Examples\n    $ fkill 1337\n    $ fkill safari\n    $ fkill :8080\n    $ fkill 1337 safari :8080\n    $ fkill\n\n  To kill a port, prefix it with a colon. For example: :8080.\n\n  Run without arguments to use the interactive interface.\n  The process name is case insensitive.\n```\n\n\n## Interactive UI\n\nRun `fkill` without arguments to launch the interactive UI.\n\n![](screenshot.svg)\n\n\n## Related\n\n- [fkill](https://github.com/sindresorhus/fkill) - API for this module\n- [alfred-fkill](https://github.com/SamVerschueren/alfred-fkill) - Alfred workflow for this module\n\n\n## Created by\n\n- [Sindre Sorhus](https://sindresorhus.com)\n- [Daniel Baker](https://github.com/coffeedoughnuts)\n'
