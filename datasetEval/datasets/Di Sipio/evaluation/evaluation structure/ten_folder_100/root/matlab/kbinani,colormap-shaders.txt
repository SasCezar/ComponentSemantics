b'# colormap-shaders\n\nA collection of shaders to draw color map like this:\n\n![rainbow](sample/transform_rainbow.png).\n\n# usage\n\nEach `*.frag` shader sources provides a `colormap` function, which takes an `float` argument `x` (`x` should be: `0.0 <= x <= 1.0`). The `colormap` function returns a `vec4` value which represents an RGBA color.\n\n```\nvec4 colormap(float x);\n\nvoid main() {\n    gl_FragColor = colormap(gl_TexCoord[0].x);\n}\n\n/* include "shaders/rainbow.frag" here, for example. */\n```\n\n## special case\n\n### gnuplot.frag\n\nThis shader emulates `gnuplot`\'s `rgbformulae`, and the signature of `colormap` is:\n```\nvec4 colormap(float x, int red_type, int green_type, int blue_type);\n```\n\nIt takes additional 3 arguments, with same meanings to `rgbformulae`\'s 3 arguments respectively.\n\n# usage from c++\n\n```c++\n#include <colormap/colormap.h>\n#include <iostream>\n\nint main()\n{\n    using namespace colormap;\n\n    // Print RGB table of MATLAB::Jet colormap.\n    MATLAB::Jet jet;\n    std::cout << "category: " << jet.getCategory() << std::endl;\n    std::cout << "title:    " << jet.getTitle() << std::endl;\n    int const size = 256;\n    for (int i = 0; i < size; ++i) {\n        float const x = i / (float)size;\n        Color c = jet.getColor(x);\n        std::cout << x << "\\t" << c.r << "\\t" << c.g << "\\t" << c.b << std::endl;\n    }\n\n    // Dump category and title of all colormaps.\n    for (std::shared_ptr<Colormap const> const& c : ColormapList::getAll()) {\n        std::cout << c->getCategory() << " : " << c->getTitle() << std::endl;\n    }\n    return 0;\n}\n```\n\n# samples\n\n## MATLAB\n\n* http://www.mathworks.com/products/matlab/\n\n|name          |sample
