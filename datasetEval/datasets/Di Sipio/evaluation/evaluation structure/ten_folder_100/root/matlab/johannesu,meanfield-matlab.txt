b'A MATLAB wrapper for solving DenseCRF problems [1,2]. \r\nThe code uses the c++ library provided with [2].\r\n\r\nGetting started\r\n--\r\n* Setup a C++ compiler in MATLAB using `mex -setup`, you will need to have a supported compiler installed, see [https://www.mathworks.com/support/compilers.html](https://www.mathworks.com/support/compilers.html) for a list\r\n* To solve a general problem see examples/example.m\r\n* To perform segmentation on the MSRC-21 database using the unary potentials from [http://graphics.stanford.edu/projects/densecrf/unary/](http://graphics.stanford.edu/projects/densecrf/unary/) see examples/example_MSRC.m\r\n\r\n![Image and result](screenshot/screenshot.png)\r\n\r\nIncluded solvers\r\n--\r\n* Mean field approximation, using approximate filtering [2]\r\n* Mean field approximation, performing all summations explicitly (slow)\r\n* TRW-S [3]\r\n* Graph cuts [4] (only works for 2 label problems)\r\n\r\n\r\nReferences\r\n--\r\n1. __Efficient Inference in Fully Connected CRFs with Gaussian Edge Potentials__. <br />\r\nConference on Neural Information Processing Systems (NIPS), 2011. <br />\r\n_Philipp Kr\xc3\xa4henb\xc3\xbchl and Vladlen Koltun_.\r\n\r\n2. __Parameter Learning and Convergent Inference for Dense Random Fields__. <br />\r\nInternational Conference on Machine Learning (ICML), 2013. <br />\r\n_Philipp Kr\xc3\xa4henb\xc3\xbchl and Vladlen Koltun_.\r\n\r\n3. __Convergent Tree-reweighted Message Passing for Energy Minimization__. <br />\r\nIEEE Transactions on Pattern Analysis and Machine Intelligence (PAMI), 2006. <br />\r\n_Vladimir Kolmogorov_.\r\n\r\n4. __An Experimental Comparison of Min-Cut/Max-Flow Algorithms for Energy Minimization in Computer Vision__. <br />\r\nIEEE Transactions on Pattern Analysis and Machine Intelligence (PAMI), 2004 <br />\r\n_Yuri Boykov and Vladimir Kolmogorov_.\r\n'
