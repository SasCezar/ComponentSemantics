b'RobustPCA\n=========\n\nRobust PCA (Robust Principal Component Analysis) implementation and examples (Matlab).\n\n**Robust PCA** is a matrix factorization method that decomposes the input matrix X into the sum of two matrices L and S, where L is low-rank and S is sparse. This is done by solving the following optimization problem called Principal Component Pursuit (PCP):\n\n> \\min ||L||_* + \\lambda ||S||_1\n\n> s.t. L + S = X\n\nwhere ||.||_* is a nuclear norm, ||.||_1 is L1-norm. For more information on Robust PCA please refer to the original paper "Robust principal component analysis?" Emmanuel J. Cand\xc3\xa8s, Xiaodong Li, Yi Ma and John Wright, 2009. The optimization method is ADMM algorithm (Alternating Direction Method of Multipliers).\n\n**Examples:**\n\n  * Toy data example: small toy matrix decomposition into low-rank and sparse component.\n![alt text](http://img-fotki.yandex.ru/get/5107/10605357.9/0_85836_f54be680_L.png "Toy data")\n\n  * Inpainting: recovering corrupted images via low-rank representation learning.\n![alt text](http://img-fotki.yandex.ru/get/4806/10605357.9/0_85837_18223bc3_L.png "Inpainting")\n\n  * Video decomposition: separating foreground from background in the video.\n![alt text](http://img-fotki.yandex.ru/get/4806/10605357.9/0_85838_472a4c65_L.png "Video decomposition")\n\n'
