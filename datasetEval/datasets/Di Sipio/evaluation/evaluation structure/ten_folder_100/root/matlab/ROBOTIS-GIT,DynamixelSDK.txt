b'<img src="http://emanual.robotis.com/assets/images/sw/sdk/dynamixel_sdk/overview/dynamixel_sdk_concept_logo.jpg">\n\n## Dynamixel SDK\nThe ROBOTIS Dynamixel SDK is a software development kit that provides Dynamixel control functions using packet communication. The API is designed for Dynamixel actuators and Dynamixel-based platforms. For more information on Dynamixel SDK, please refer to the e-manual below.\n- [ROBOTIS e-Manual for Dynamixel SDK](http://emanual.robotis.com/docs/en/software/dynamixel/dynamixel_sdk/overview/)\n\n## Supported Programming Languages\nDynamixelSDK supports various programming languages.\n- **C**: *Dynamic library and source code of this library and examples\n- **C#** / **Java** / **MATLAB** / **LabVIEW**: Support based on dynamic library using C language\n- **C++**: *Dynamic library and source code of this library and examples\n- **Python**: Python module and examples\n(* Dynamic library (*.dll, *.so, and *.dylib files) / .dll: dynamic-link library on Windows / .so: shared object on Linux / .dylib: dynamic library on MacOS)\n\n## ROS 1 Packages for Dynamixel
