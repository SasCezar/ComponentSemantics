b"# shadedErrorBar\n\n`shadedErrorBar` is a MATLAB function that creates a continuous shaded error region around a line rather than discrete bars. \nThe error region can either be specified explicitly or calculated on the fly based upon function handles. \nThe appearance of the plot can be modified via input argumets and/or the handles of the returned plot objects.\nYou will need OpenGL to get transparency to work.\n\n\n\n# Examples\n\n### Basic example with overlaid points\n```\n% Prepare data\ny=randn(30,80)*5;\nx=(1:size(y,2))-40;\nyP = sin( linspace(-2*pi,2*pi,length(x)) )*20;\ny = bsxfun(@plus,y,yP)+60;\n\n% Make the plot\nclf\nshadedErrorBar(x,y,{@mean,@std}); \n\n% Overlay the raw data\nhold on\nplot(x,y,'.','color',[0.5,0.5,0.95])\n\ngrid on\n```\n![](./exampleImages/basic_with_overlay.png)\n\n\n\n\n### Overlaying multiple lines\n```\n% Prepare data for first line\ny=ones(30,1)*x; \ny=y+0.06*y.^2+randn(size(y))*10;\n\nclf\nshadedErrorBar(x,y,{@mean,@std},'lineprops','-b','patchSaturation',0.33)\n\n% Overlay second line\nhold on\nshadedErrorBar(x,2*y+20,{@mean,@std},'lineprops',{'-go','MarkerFaceColor','g'});\n\n%Overlay third line\ny=randn(30,80)*5; \nx=(1:size(y,2))-40;\nyP = sin( linspace(-2*pi,2*pi,length(x)) )*20;\ny = bsxfun(@plus,y,yP)+60;\n\n% Make this line non-transparent\nshadedErrorBar(x, y, {@mean,@std}, 'lineprops', '-r','transparent',false,'patchSaturation',0.075)\n\ngrid on\n```\n![](./exampleImages/multiple_lines.png)\n\n\n\n### Modifying plot properties post-hoc\n```\n% Prepare data\ny=randn(30,80)*5; \nx=(1:size(y,2));\nyP = sin( linspace(-2*pi,2*pi,length(x)) )*20;\ny = bsxfun(@plus,y,yP);\n\n% Plot\nclf\ns = shadedErrorBar(x, y, {@mean,@std}, 'lineprops', '-r');\n\n% Set face and edge properties\nset(s.edge,'LineWidth',2,'LineStyle',':')\ns.mainLine.LineWidth = 5;\ns.patch.FaceColor = [0.5,0.25,0.25];\n\n% Overlay data points post-hoc\nhold on\nplot(s.mainLine.XData, s.mainLine.YData,'or','MarkerFaceColor','w')\n\ngrid on\n\nset(gca,'XTickLabel',[],'YTickLabel',[])\n```\n\n![](./exampleImages/mod-handles.png)\n\n\n# Change Log\n\n8th Novemeber 2019\n* Merge Octave support from [JarlPed](https://github.com/JarlPed/)\n\n\n24th November 2017\n* Add `patchSaturation` as an optional argument.\n\n6th August 2017\n\n* Parameter/value pairs for input arguments greater than 3.\n* Transparency on by default.\n* No longer changes the renderer.\n\n# Contributing\nIf you wish to contribute please fork the repo, check out the `dev` branch and make your changes there, then file a pull request describing what you did. \n\n# Licensing \nThis code is licensed under GPL v3.
