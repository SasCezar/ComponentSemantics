The word has a similar meaning to _Genesis_, which is actually where the project started: Sega Genesis\xe2\x80\x94our early beginnings as humble emulator on iOS.\n\n<sup> prov\xc2\xb7e\xc2\xb7nance &nbsp; _/\xcb\x88pr\xc3\xa4v\xc9\x99n\xc9\x99ns/_ &nbsp; n. </sup><br>\n<sup> \xc2\xb9 place of origin &nbsp; \xc2\xb2 the beginning of something\'s existence </sup>\n\n## Systems Supported:\n\n- Atari\n  - 2600\n  - 5200\n  - 7800\n  - Lynx\n  - Jaguar\n- Bandai\n  - WonderSwan / WonderSwan Color\n- NEC\n  - PC Engine / TurboGrafx-16 (PCE/TG16)\n  - PC Engine Super CD-ROM\xc2\xb2 System / TurboGrafx-CD\n  - PC Engine SuperGrafx\n  - PC-FX\n- Nintendo \n  - Nintendo Entertainment System / Famicom (NES/FC)\n  - Famicom Disk System\n  - Super Nintendo Entertainment System / Super Famicom (SNES/SFC)\n  - Game Boy / Game Boy Color (GB/GBC)\n  - Virtual Boy\n  - Nintendo 64 (N64)\n  - Game Boy Advance (GBA)\n  - Pok\xc3\xa9mon mini\n- Sega\n  - SG-1000\n  - Master System\n  - Genesis / Mega Drive\n  - Game Gear\n  - CD / MegaCD\n  - 32X\n- SNK\n  - Neo Geo Pocket / Neo Geo Pocket Color\n- Sony\n  - PlayStation (PSX/PS1)\n  \n## Systems in Development:\n - Nintendo\n   - DS\n - Sega\n   - Dreamcast (reicast)\n   - Saturn (yabause)\n - Sony\n   - PSP (ppsspp)\n\n## Features\n\n- Very Accurate, Full Speed Emulation\n- Stereo Sound\n- Save States (save anywhere/_game freezing_)\n- Battery Saves (native game saves)\n- Adjustable Controller Overlay Opacity\n- Controller Support:\n  - [MFi Game Controller](https://mfigames.com/compare-mfi-controllers/) Support\n  - iCade Controller Support ([8bitdo](http://www.8bitdo.com/), etc\xe2\x80\xa6)\n- Local Multiplayer via External Controllers\n- Portrait and Landscape Orientation\n- Automatic ROM Matching (Game Title, Cover Art, Description, Genre, etc\xe2\x80\xa6) via [OpenVGDB](https://github.com/OpenVGDB/OpenVGDB)\n- ROM Customizations (Cover Art, Game Titles) ([Instructions](https://github.com/Provenance-Emu/Provenance/wiki/Customizing-ROMs))\n- Multiple Options for Managing ROMs, Game Saves and Cover Art:\n  - [Import](https://wiki.provenance-emu.com/installation-and-usage/roms/importing-roms) / [Export](https://wiki.provenance-emu.com/info/miscellaneous/restoring-files):\n    - Uploading _(via built-in web server)_\n  - [Import](https://wiki.provenance-emu.com/installation-and-usage/roms/importing-roms):\n    - Downloading _(from mobile browsers)_\n    - Copying _(from other mobile apps and AirDrop)_\n    - Injecting _(with desktop apps)_\n- Game Library Searching\n- iOS 9+ (iPhone, iPad)\n  - 3D Touch Shortcuts for Recent Games (iPhone 6s and later, except iPhone SE and iPhone X\xca\x80)\n  - Taptic Engine Button Feedback (iPhone 7 and later)\n  - Spotlight Searching ([Setup](https://wiki.provenance-emu.com/installation-and-usage/installing-provenance/building-from-source#advanced-features))\n- tvOS 9+ (Apple TV 4+)\n  - TopShelf Support ([Setup](https://wiki.provenance-emu.com/installation-and-usage/installing-provenance/building-from-source#advanced-features))\n\n<sub>Feel free to suggest/request features or report bugs by posting an issue, but please read up on our [Issues Usage](https://github.com/Provenance-Emu/Provenance/wiki/Issues-Usage) first, before posting.</sub><br><br>\n\n----\n\n### Importing ROMs\nVisit our wiki on [Importing ROMs](https://wiki.provenance-emu.com/installation-and-usage/roms/importing-roms) to learn how to load ROMs into Provenance.\n\n\n### Release Roadmap\n![GitHub Release](https://img.shields.io/github/release/provenance-emu/provenance.svg?style=flat-square) \n![GitHub Release Date](https://img.shields.io/github/release-date/provenance-emu/provenance.svg?style=flat-square)\n![Github commits (since latest release)](https://img.shields.io/github/commits-since/provenance-emu/provenance/latest.svg?style=flat-square) \n[![Travis](https://img.shields.io/travis/provenance-emu/provenance.svg?style=flat-square)](https://travis-ci.org/Provenance-Emu/Provenance)\n\nWe track upcoming releases in our GitHub [Milestones](https://github.com/Provenance-Emu/Provenance/milestones?direction=asc&sort=title&state=open).
