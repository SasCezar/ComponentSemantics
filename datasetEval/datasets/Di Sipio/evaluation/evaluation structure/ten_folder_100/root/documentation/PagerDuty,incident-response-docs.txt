b'# PagerDuty Incident Response Documentation [![Build Status](https://circleci.com/gh/PagerDuty/incident-response-docs/tree/master.svg?style=svg)](https://circleci.com/gh/PagerDuty/incident-response-docs)\nThis is a public version of the Incident Response process used at PagerDuty. It is also used to prepare new employees for on-call responsibilities, and provides information not only on preparing for an incident, but also what to do during and after. See the [about page](docs/about.md) for more information on what this documentation is and why it exists.\n\nYou can view the documentation [directly](/docs/index.md) in this repository, or rendered as a website at https://response.pagerduty.com.\n\n[![PagerDuty Incident Response Documentation](screenshot.png)](https://response.pagerduty.com)\n\n## Development\nWe use [MkDocs](http://www.mkdocs.org/) to create a static site from this repository.
