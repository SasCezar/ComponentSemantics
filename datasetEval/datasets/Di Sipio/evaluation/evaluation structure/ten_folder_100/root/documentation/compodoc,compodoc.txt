documentation generated for a simple [TodoMVC application].\n\n[demo]: https://compodoc.github.io/compodoc-demo-todomvc-angular/\n[todomvc application]: https://github.com/compodoc/compodoc-demo-todomvc-angular\n\n# Features\n\n-   **Clean, simple design** \xe2\x80\x94 Navigate the documentation on the left side, read it on the right.\n\n-   **Beautiful themes** \xe2\x80\x94 Seven themes are available from well known documentation tools like [Gitbook], [Read the Docs] and projects like [Vagrant], [Laravel], [Postmark] and [Stripe].\n\n-   **Search** \xe2\x80\x94 Includes a powerful search engine ([lunr.js]) for easily finding what you\'re looking for.\n\n-   **Automatic table of contents** \xe2\x80\x94 API table of contents is generated using elements found during files parsing.\n\n-   **JSDoc light support** \xe2\x80\x94 Support for `@param`, `@returns`, `@link`, `@ignore` and `@example` tags.\n\n-   **Documentation coverage** \xe2\x80\x94 Get the documentation coverage report of your project.\n\n-   **Angular CLI-friendly** \xe2\x80\x94 Supports [Angular CLI] projects out-of-the-box.\n\n-   **Offline** \xe2\x80\x94 No server needed, no sources uploaded online, documentation generated entirely offline.\n\n-   **Open-source and on npm** \xe2\x80\x94 Use it directly in your project using [npm] and a single script\xe2\x80\x94that\'s it!\n\n[gitbook]: https://www.gitbook.com\n[read the docs]: https://readthedocs.org/\n[vagrant]: https://www.vagrantup.com/docs/\n[laravel]:
