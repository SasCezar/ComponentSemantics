Useful for injecting API docs into project README files.\n\n## Synopsis\n\n1\\. Document your code using valid jsdoc comments.\n\n```js\n/**\n * A quite wonderful function.\n * @param {object} - privacy gown\n * @param {object} - security\n * @returns {survival}\n */\nfunction protection (cloak, dagger) {}\n```\n\n2\\. Run a command.\n\n```sh\n$ jsdoc2md example.js\n```\n\n3\\. Get markdown output.\n\n```markdown\n## protection(cloak, dagger) \xe2\x87\x92 <code>survival</code>\nA quite wonderful function.\n\n**Kind**: global function\n\n| Param  | Type                | Description  |\n| ------ | ------------------- | ------------ |\n| cloak  | <code>object</code> | privacy gown |\n| dagger | <code>object</code> | security     |\n\n```\n\n## Install\n\n```\n$ npm install --save-dev jsdoc-to-markdown\n```\n\n## See also\n\n* [API documentation](https://github.com/jsdoc2md/jsdoc-to-markdown/blob/master/docs/API.md)\n* The [wiki](https://github.com/jsdoc2md/jsdoc-to-markdown/wiki) for example output, FAQs, tutorials, plugins, use with gulp/grunt etc.\n\n* * *\n\n&copy; 2014-19 Lloyd Brookes \\<75pound@gmail.com\\>.\n'
